HSP_ACCT_CPT_CODES
Description:
This table contains hospital account CPT(R) codes from the Hospital Accounts Receivable (HAR) master file.

Primary Key
Column Name Ordinal Position
HSP_ACCOUNT_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 HSP_ACCOUNT_ID NUMERIC No
The ID number of a hospital account.
2 LINE INTEGER No
The line number in the results of a query. Since multiple CPT™ codes can be stored in one hospital account, each CPT™ code will have a unique line number.
3 CPT_CODE VARCHAR No
A CPT™ code stored in the hospital account.
4 CPT_CODE_DATE DATETIME No
A date associated with a CPT™ code stored in the hospital account.
5 CPT_PERF_PROV_ID_PROV_NAME VARCHAR No
The name of the service provider. This item may be hidden in a public view of the CLARITY_SER table.
6 CPT_EVENT_NUMBER INTEGER No
The event number associated with a CPT™ code stored in the hospital account. Event number are used to associate CPT™ codes with procedure codes.
7 CPT_MODIFIERS VARCHAR No
A modifier or modifiers associated with a CPT™ code stored in the hospital account.
8 LMRP_CODE VARCHAR No
A Local Medical Review Policy (LMRP) code associated with a CPT™ code stored in the hospital account.
9 CPT_CODE_DESC VARCHAR No
The description of a CPT® code stored in the hospital account.
10 PX_APC_FAC_RMB_AMT NUMERIC No
The monetary reimbursement amount for the procedure Ambulatory Payment Classification (APC) FAC on the hospital account.
11 PX_OCE_EDIT_CODE VARCHAR No
The procedure Outpatient Code Editor (OCE) edit code for the hospital account.
12 PX_APC_CODE VARCHAR No
The procedure Ambulatory Payment Classification (APC) code for the hospital account.
13 PX_HCFA_PAYMT_AMT NUMERIC No
The procedure Health Care Payment and Remittance Advice (HCFA) monetary payment amount for the hospital account.
14 PX_COPAY_AMT NUMERIC No
The monetary copay amount for the procedure on the hospital account.
15 PX_REV_CODE_ID NUMERIC No
The unique ID of the revenue code.
16 PX_REV_CODE_ID_REVENUE_CODE_NAME VARCHAR No
The name of the revenue code.
17 CPT_EXCLD_RPT_YN VARCHAR No
Indicates whether to exclude CPT codes from clinical reporting. This is frequently used to filter out procedures copied over during an account merge to prevent double counting.
May contain organization-specific values: No
Category Entries:
Yes
No
18 CPT_QUANTITY INTEGER No
Quantity of the CPT(R)/HCPCS code. Hospital accounts created prior to Epic 2014 have a null value in this column which implies a quantity of 1.
19 CPT_POS_TYPE_C_NAME VARCHAR No
Stores the Place of Service (POS) Type value for a coded CPT on an account..
May contain organization-specific values: Yes
