COLL_AGENCY_HX
Description:
This table contains collection agency history information.

Primary Key
Column Name Ordinal Position
TX_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 TX_ID NUMERIC No
The unique identifier for the transaction record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 COLL_HX_AGENCY_ID NUMERIC No
The ID of the collection agency which is associated with this history information.
4 COLL_HX_AGENCY_ID_COLL_AGENCY_NAME VARCHAR No
The name of the collection agency.
5 COLL_HX_SUBMIT_DATE DATETIME No
The date when account is submitted to the collection agency.
6 COLL_HX_RESOLV_DATE DATETIME No
The date when the agency resolved the submission.
7 COLL_HX_RES_TYPE_C_NAME VARCHAR No
The collection history resolution type category number for this history information.
May contain organization-specific values: No
Category Entries:
Recalled
Completed
Converted to Agency Extract Framework
8 COLL_HX_SUBMIT_DTTM DATETIME (UTC) No
The UTC instant this transaction was submitted to collections.
9 COLL_HX_RESOLV_DTTM DATETIME (UTC) No
The UTC instant this transaction was resolved with collections.
