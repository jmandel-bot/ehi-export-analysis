PLAN_DEVIATION_INFO
Description:
This table holds plan-level deviation info for oncology treatment plans, including deviation instant, deviation user, and deviation change type.

Primary Key
Column Name Ordinal Position
TREATMENT_PLAN_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 TREATMENT_PLAN_ID NUMERIC No
The unique identifier for the oncology treatment plan record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 DEVIATION_UTC_DTTM DATETIME (UTC) No
This stores the instants the plan deviated from the protocol.
4 DEVIATION_USER_ID VARCHAR No
Stores the users who made the treatment plan deviations
5 DEVIATION_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
6 DEVIATION_ACTION_TYPE_C_NAME VARCHAR No
This is the change action type of the plan deviation, for example, creating a blank plan, or creating a plan that is not from a suggested protocol, etc.
May contain organization-specific values: Yes
