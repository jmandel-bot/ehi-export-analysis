HSB_MED_SIGN_OFF
Description:
Contains information about the users involved in medication sign off
workflows, the actions they took, and when it occurred.

Primary Key
Column Name Ordinal Position
SUMMARY_BLOCK_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 SUMMARY_BLOCK_ID NUMERIC No
The unique identifier for the Episode.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 AN_MED_REVIEWER_ID VARCHAR No
Contains the ID of the user who was involved in the medication review process.
4 AN_MED_REVIEWER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
5 MED_REVIEW_STATUS_C_NAME VARCHAR No
Stores the current status of the medication review.
May contain organization-specific values: No
Category Entries:
Message Sent
Sign Off Received
Reviewed - Updates Needed
Retracted - Under Review
Sent Updates
No Signoff Needed
6 STATUS_UPDATED_DTTM DATETIME (UTC) No
Stores the instant that the status was last updated.
7 MED_RVW_MSG_ID VARCHAR No
This column contains the message ID for the message that was sent for medication sign off.
8 MED_RVW_DEV_C_NAME VARCHAR No
Stores the device used when the user signed off on the medications.
May contain organization-specific values: Yes
