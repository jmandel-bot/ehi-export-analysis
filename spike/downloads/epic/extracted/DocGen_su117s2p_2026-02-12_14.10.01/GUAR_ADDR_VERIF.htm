GUAR_ADDR_VERIF
Description:
This table contains information for guarantor address verification.

Primary Key
Column Name Ordinal Position
GUAR_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 GUAR_ID NUMERIC No
Address verification - guarantor ID
2 LINE INTEGER No
Address verification - line count for guarantors
3 G_ADDR_INST_SENT DATETIME (Local) No
The instant address verification message was sent.
4 G_ADDR_INST_RECV DATETIME (Local) No
The instant the address verification message was received.
5 G_ADDR_ID_SENT VARCHAR No
The ID of the address verification message that was sent.
6 G_ADDR_ID_RECV VARCHAR No
The ID of the address verification message that was received.
7 G_ADDR_ACTN_C_NAME VARCHAR No
Stores whether the sent or received address was kept for the guarantor.
May contain organization-specific values: No
Category Entries:
Sent Msg kept
Recvd Msg Kept
8 G_ADDR_TYP_SRCH VARCHAR No
The type of search conducted for address verification.
9 G_ADDR_REC_CR_ID VARCHAR No
This points to the note record that contains the guarantor's address verification response with the credit information.
