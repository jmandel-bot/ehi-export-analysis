MYC_ACCT_DELETION
Description:
Details about a MyChart account's impending deletion.

Primary Key
Column Name Ordinal Position
MYPT_ID 1

Column Information
Name Type Discontinued?
1 MYPT_ID VARCHAR No
The unique identifier (.1 item) for the account record.
2 DEL_ACCT_ON_DATE DATETIME No
The date this MyChart account should be deleted.
3 DEL_ACCT_DELETION_REASON_C_NAME VARCHAR No
The reason that this MyChart account should be deleted.
May contain organization-specific values: No
Category Entries:
Incomplete Setup
No Active H2G Links
Patient Requested
Missing On Cloud IdP Database
Manually Requested From Utility
4 DEL_ACCT_LST_NOTICE_UTC_DTTM DATETIME (UTC) No
The date/time we last notified the patient that we were going to delete their account.
5 DEL_ACCT_ERROR_STATUS_C_NAME VARCHAR No
This is a flag to indicate this account cannot be deleted. It will also show if the account has tried to be deleted, or if we've just checked whether or not it can.
May contain organization-specific values: No
Category Entries:
Report Failed
Delete Failed
6 DEL_ACCT_DEL_REPORT_FILE VARCHAR No
The file name/location of the report generated when we run the Patient Deletion Utility on a MyChart Account that will be deleted.
