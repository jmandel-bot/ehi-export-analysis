BED_PLAN_HX
Description:
This table contains information about edits that occurred on pending event records.

Primary Key
Column Name Ordinal Position
PEND_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 PEND_ID VARCHAR No
The unique identifier for the pending action record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 UPDATE_INST DATETIME (UTC) No
The instant when the pending action record was updated.
4 ORD_ID NUMERIC No
The linked order ID (I PND 560) at the time that the pending event was updated,
5 LOA_REASON_C_NAME VARCHAR No
The leave of absence reason category ID for the bed planning event history.
May contain organization-specific values: Yes
6 HX_BED_TYPE_C_NAME VARCHAR No
History item for requested bed type.
May contain organization-specific values: Yes
Category Entries:
Empty
Standard
Crib
Telemetry
Bariatric
7 UPDATE_INST_LOCAL_DTTM DATETIME (Local) No
The local instant when the pending event was updated. For UTC, use BED_PLAN_HX.UPDATE_INST.
