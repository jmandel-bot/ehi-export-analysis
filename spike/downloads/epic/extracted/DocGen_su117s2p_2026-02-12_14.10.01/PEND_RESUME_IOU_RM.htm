PEND_RESUME_IOU_RM
Description:
This table extracts the related multiple response Pend - Resume Indications of Use (I IEV 2090) item, which records the indications of use for pended medications in medication reconciliation

Primary Key
Column Name Ordinal Position
EVENT_ID 1
GROUP_LINE 2
VALUE_LINE 3

Column Information
Name Type Discontinued?
1 EVENT_ID VARCHAR No
The unique ID for the event record.
2 GROUP_LINE INTEGER No
The line number for the information associated with pended indications associated with this record.
3 VALUE_LINE INTEGER No
The line number of one of multiple values associated with a specific group of data within this record.
4 PEND_INDICATIONS_ID NUMERIC No
This column saves the pended indications when a discharge med is resumed.
5 PEND_INDICATIONS_ID_MEDICAL_COND_NAME VARCHAR No
This contains the name of the medical condition.
