TARGET_TASK_ASMTS
Description:
This table stores the patient encounters and assessments for target progress assessments.

Primary Key
Column Name Ordinal Position
ACTIVITY_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 ACTIVITY_ID VARCHAR No
The unique identifier (.1 item) for the task record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 TARGET_PAT_ENC_CSN_ID NUMERIC No
This item stores the patient CSNs of the target assessments.
4 TARGET_OVERALL_ASMT_C_NAME VARCHAR No
This item stores the overall assessments of the target assessments.
May contain organization-specific values: No
Category Entries:
On Track
Off Track
Unable to Assess
5 TARGET_PROG_ASMT_C_NAME VARCHAR No
This item stores the progression assessments of the target assessments.
May contain organization-specific values: No
Category Entries:
Improving
Worsening
Unchanging
Not Applicable
6 TARGET_ASSESSED_INST_UTC_DTTM DATETIME (UTC) No
This item stores the instants that the user most recently assessed the target over different encounters.
7 TARGET_ASSESSED_USER_ID VARCHAR No
This item stores the users that most recently assessed the target over different encounters.
8 TARGET_ASSESSED_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
