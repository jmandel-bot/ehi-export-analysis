HH_VO_COMPONENTS
Description:
This table contains Home Health verbal orders components information. This information is used to track changes made to verbal orders on the Home Health Remote Client.

Primary Key
Column Name Ordinal Position
VERBAL_ORDER_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 VERBAL_ORDER_ID VARCHAR No
The unique identifier for the verbal order record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 COMPON_TYPE_C_NAME VARCHAR No
Verbal orders component type. Links to category table ZC_COMP_TYPE.
May contain organization-specific values: Yes
Category Entries:
Other
Medication
Intervention
Visit
Plan of Care
DME
Admit
Hospice Plan Of Care
Certificate of Terminal Illness
Hospice Face to Face Attestation
Home Health Face to Face Attestation
Code Status
4 COMPON_OLD_MED_ID NUMERIC No
Old medication verbal order ID. Links to table ORDER_PROC.
5 COMPON_NEW_MED_ID VARCHAR No
New medication verbal order ID.
6 COMPON_VST_DISC_C_NAME VARCHAR No
Verbal order component visit discipline. Links to category table ZC_COMPON_VST_DISC.
May contain organization-specific values: Yes
7 COMPON_NOTES_ID VARCHAR No
Verbal order component note ID.
8 COMPON_CERT_PER VARCHAR No
Verbal order component certification period.
9 COMPON_EPS_CARE_ID NUMERIC No
Verbal order component episode of care ID. Links to EPISODE table.
10 COMPON_LINK_COMP VARCHAR No
Verbal order component linked component. This item is used to store the link to the components for which this verbal order component is created.
11 COMP_PENDING_FAX_YN VARCHAR No
This item stores whether the verbal order component has been pended.
May contain organization-specific values: No
Category Entries:
Yes
No
12 COMPON_TAKING_DIFF_LINE INTEGER No
This item stores the pointer to the patient taking differently line associated with a Home Health medication verbal order, if it exists.
13 DISCON_MED_TEXT VARCHAR No
Additional information on why an order's medication is discontinued.
14 MED_AUDIT_LINE INTEGER No
The line number for the medication audit trail. This is the line number found in column LINE of AUDIT_ORD_RXNOADD.
15 SUPPL_ORDER_NOT_SENT_ALL_YN VARCHAR No
Whether the order has been sent to all the providers who should be notified of changes to the plan of care.
May contain organization-specific values: No
Category Entries:
No
Yes
