SOCIAL_CARE_TRACKER
Description:
This table contains information about Social Care decisions, which are stored as tracker records.

Primary Key
Column Name Ordinal Position
FIN_ASST_TRACKER_ID 1

Column Information
Name Type Discontinued?
1 FIN_ASST_TRACKER_ID NUMERIC No
The unique identifier for the decision record.
2 RECORD_STATUS_C_NAME VARCHAR No
The record status category ID for the decision.
May contain organization-specific values: No
Category Entries:
Soft Deleted
Hidden
Hidden and Soft Deleted
3 FIN_ASST_PROGRAM_ID NUMERIC No
The unique ID of the financial assistance program for the decision.
4 FIN_ASST_PROGRAM_ID_PROGRAM_NAME VARCHAR No
The name of the financial assistance program record.
5 FIN_ASST_CASE_ID NUMERIC No
The unique ID of the financial assistance case associated with the decision.
6 STATUS_C_NAME VARCHAR No
The status category ID for the decision.
May contain organization-specific values: Yes
Category Entries:
Pending
Approved
Denied
Waiting for supervisor's approval
Pending Appeal
Overturned
Inactive
Entered in Error
Signed
Needs Review
7 START_DATE DATETIME No
The date when the decision was approved.
8 END_DATE DATETIME No
The end date for the decision.
9 DECISION_VALID_C_NAME VARCHAR No
The validity type category ID for the decision.
May contain organization-specific values: No
Category Entries:
Fixed-term
Continuous
None
10 DECISION_NOTE_ID VARCHAR No
The unique ID of the note for the decision.
11 REC_CREATION_DATE DATETIME No
The date when the decision was created.
12 INST_OF_UPDATE_1_DTTM DATETIME (Local) No
Stores the instant the record was last locked/unlocked.
13 DECISION_NEGATIVE_YN VARCHAR No
Indicates whether the decision tracker is negative, which denies application for support.
May contain organization-specific values: No
Category Entries:
No
Yes
14 PARENT_TRACKER_ID NUMERIC No
The unique ID of the service decision a related decision is linked to.
15 RESPONSIBLE_USER_ID VARCHAR No
The unique ID for the responsible user for the decision.
16 RESPONSIBLE_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
17 DECISION_DEPARTMENT_ID_EXTERNAL_NAME VARCHAR No
The external name of the department record. This is often used in patient correspondence such as reminder letters.
18 DECISION_COST_CENTER VARCHAR No
The cost center string generated for the decision.
19 REG_ENCOUNTER_CSN_ID NUMERIC No
The unique contact serial number of the special encounter that stores registration information related to the service decision.
