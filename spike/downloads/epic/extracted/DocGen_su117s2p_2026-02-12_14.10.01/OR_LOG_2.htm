Description:
The OR_LOG_2 table contains OR management system log records.

May contain organization-specific values: Yes
Category Entries:
Unrelated Case
Planned Staged Return
Planned Return Due to Complications
Unplanned Return
22 RETURN_TO_OR_ORIGINAL_CASE_ID VARCHAR No
The unique ID of the case which is responsible for this log's return to OR.
23 RETURN_TO_OR_REASON_C_NAME VARCHAR No
The category ID for the return to OR reason. It further indicates how the log relates to the past case.
May contain organization-specific values: Yes
Category Entries:
Follow up evaluation procedure
Continuation of care
Secondary procedure
Continuing care of wound
Support device maintenance/removal
Anastomotic complication
Complications post arthroplasty
Complications post device/prosthetic placement
Peritonitis

Vascular complication
Wound complications or surgical site infection
Other (comment)
24 RETURN_TO_OR_REASON_CMT VARCHAR No
The comments specific to the return to OR reason. Entered by the user who documented the log as return to OR or a user who verified the return to OR documentation.
25 RETURN_TO_OR_VERIFIED_BY_ID_PROV_NAME VARCHAR No
The name of the service provider. This item may be hidden in a public view of the CLARITY_SER table.
26 RETURN_TO_OR_ADDL_CMT VARCHAR No
Additional comments for the return to OR. Entered by the user who documented the log as return to OR or a user who verified the return to OR documentation.
27 RETURN_TO_OR_SUR_AGREE_YN VARCHAR No
Indicates whether the surgeon of the original case agrees or disagrees with the return to OR documentation.
The category values for this column were already listed for column: VIDEO_PIC_TAKEN_YN

NOT MESSAGED

ESCALATION SENT
AUTO AGREEMENT
MANUAL AGREEMENT
MANUAL DISAGREEMENT
