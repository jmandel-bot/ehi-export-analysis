HSP_ACCT_LETTERS
Description:
This table contains information about hospital billing letters.

Primary Key
Column Name Ordinal Position
NOTE_ID 1

Column Information
Name Type Discontinued?
1 NOTE_ID VARCHAR No
This column stores the unique identifier for the note record.
2 LETTER_SENT_DATE DATETIME No
The date the letter was sent.
3 LET_CREATE_USER_ID VARCHAR No
This column stores the unique identifier for the user who created the letter.
4 LET_CREATE_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
5 ACCOUNT_ID NUMERIC No
This column stores the unique identifier for the guarantor account that is associated with this letter. This is only populated for letters sent at the guarantor account level.
6 HSP_ACCOUNT_ID NUMERIC No
This column stores the unique identifier for the hospital account that is associated with this letter. This is only populated for letters sent at the hospital account level.
7 BUCKET_ID NUMERIC No
This column stores the unique identifier for the liability bucket that is associated with this letter. This is only populated for letters sent at the liability bucket level.
