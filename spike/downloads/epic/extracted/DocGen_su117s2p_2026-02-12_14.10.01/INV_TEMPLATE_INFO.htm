INV_TEMPLATE_INFO
Description:
Clarity table to extract information about intervention templates.

Primary Key
Column Name Ordinal Position
INTERVENTION_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 INTERVENTION_ID VARCHAR No
The unique identifier for the intervention record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 TASK_TEMPLATES_ID VARCHAR No
Task template ID for the intervention record.
4 TASK_TEMPLATES_ID_RECORD_NAME VARCHAR No
This column displays the name of the task template record.
5 TASK_TEMPLATES_DAT NUMERIC No
Intervention task template contact used.
6 LPI_LTI_ID NUMERIC No
The source intervention template id of the patient intervention record.
7 LPI_LTI_ID_INTERVENTION_NAME VARCHAR No
This column displays the intervention name.
8 LPI_LTI_DAT NUMERIC No
Source intervention template DAT for the patient intervention record.
9 INVTMPL_MERGE_DAT NUMERIC No
Contact where the template is created/merged.
10 DUP_ACT_APPLIED_C_NAME VARCHAR No
Duplicate action used to create/merge the template.
May contain organization-specific values: No
