File Naming Scheme:

The File Index Documents files are named using naming conventions as follows:

TransDoc_SPID_PROFILENUM_VISITNUM_TITLE_CREATEDT_CREATETM_FIDXKEY.EXTENSION, 
ClinDoc_SPID_PROFILENUM_VISITNUM_TITLE_CREATEDT_CREATETM_FIDXKEY.EXTENSION,            
PhysDoc_SPID_PROFILENUM_VISITNUM_TITLE_CREATEDT_CREATETM_FIDXKEY.EXTENSION,        
NotesDoc_SPID_PROFILENUM_VISITNUM_TITLE_CREATEDT_CREATETM_FIDXKEY.EXTENSION, 
ClinicNar_0004_SPID_PROFILENUM_VISITNUM_TITLE_CREATEDT_CREATETM_FIDXKEY.EXTENSION,  
ClinHist_SPID_PROFILENUM_VISITNUM_TITLE_CREATEDT_CREATETM_FIDXKEY.EXTENSION,        
HiddenText_SPID_PROFILENUM_VISITNUM_TITLE_CREATEDT_CREATETM_FIDXKEY.EXTENSION,  
WorkSheet_SPID_PROFILENUM_VISITNUM_TITLE_CREATEDT_CREATETM_FIDXKEY.EXTENSION.

CCDA:

CCDA_SPID_PROFILENUM_VISITNUM_TITLE_CREATEDT_CREATETM.EXTENSION

Detailed File Format:

The File Index Documents include the following formats: pdf

The CCDA Documents include the following formats: XML, HTML (human readable), and PDF (just the HTML but rendered into a PDF).
