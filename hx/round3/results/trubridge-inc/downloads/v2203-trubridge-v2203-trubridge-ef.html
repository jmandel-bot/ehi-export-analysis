The TruBridge EHR platform Electronic Forms output format.

Table of contents

1. The Electronic Forms file naming scheme

2. The detailed file format definition

File Naming Scheme:

The Electronic Forms files are named following the naming convention as follows:

ElecForms_SPID_PROFILE_VISIT_TITLE_DATE_FORMVER_SEQ.pdf

An example of this naming convention:

ElecForms_1_12345678_123456_Title_20230101_1_1.pdf

Detailed File Format:

The file output consists of Portable Document Format (pdf) files.
