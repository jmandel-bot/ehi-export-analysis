POC_EVENTS
Description:
This table contains event information for the Plan of Care (POC) master file.

Primary Key
Column Name Ordinal Position
RECORD_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 RECORD_ID NUMERIC No
The unique identifier for the plan of care record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 POC_EVENTS_C_NAME VARCHAR No
This item contains events related to creation, modification, and filing of a POC.
May contain organization-specific values: No
Category Entries:
Created
Refreshed
Saved
Completed
Deleted
Reopened
Signed
Start Date Changed after Completion
Ratified
Not Ratified
Hospice POC Converted
Deleted A Note
System Created (Hospice Admission)
System Created (Batch)
Home Health POC Converted
4 POC_EVENT_INST_DTTM DATETIME (Local) No
This item contains the POC event instant.
5 POC_EVENT_USER_ID VARCHAR No
The unique identifier for user record associated with the plan of care event.
6 POC_EVENT_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
7 POC_EVENT_LINK_CSN NUMERIC No
The contact serial number of the plan of care contact that is linked to an event in the POC.
8 EVENT_AUD_PREV_VAL VARCHAR No
As part of audit information for events on the plan of care, this item stores the changed value for items in the POC that changed in the event.
9 EVENT_AUD_CUR_VAL VARCHAR No
As part of audit information for events on the plan of care, this item stores the original value for items in the POC that changed in the event.
10 EVENT_AUD_HX_ITEM VARCHAR No
As part of audit information for events on the plan of care, this item stores items in the POC that changed in the event.
11 POC_EVENT_COMMENT VARCHAR No
Contains the comment linked to an event in the POC events log.
