CL_BEV_ALL
Description:
This table holds the no-add, single-response items from the Bed Events (BEV) master file.

Primary Key
Column Name Ordinal Position
RECORD_ID 1

Column Information
Name Type Discontinued?
1 RECORD_ID NUMERIC No
The unique ID of the bed event record.
2 EVENT_TYPE_C_NAME VARCHAR No
The bed event type category number for this bed event.
May contain organization-specific values: No
Category Entries:
Environmental Services
Bed Blocking
Bed Ahead
Bed Type
Attributes
3 EVENT_SOURCE_C_NAME VARCHAR No
The bed event source category number for this bed event.
May contain organization-specific values: Yes
Category Entries:
Discharge
Transfer
Ad-Hoc Request
Maintenance Cleaning
Leave of Absence
Auto Creation by Scheduling Non-Bed Clean
4 BED_ID VARCHAR No
The unique ID of the bed that this bed event pertains to.
5 BED_ID_BED_LABEL VARCHAR No
The name of the bed.
6 EAF_ID_LOC_NAME VARCHAR No
The name of the revenue location.
7 PRIORITY_C_NAME VARCHAR No
The priority category number for the bed event.
May contain organization-specific values: No
Category Entries:
Stat
Next
Normal
8 DEP_ID_EXTERNAL_NAME VARCHAR No
The external name of the department record. This is often used in patient correspondence such as reminder letters.
9 EPT_ID VARCHAR No
The unique ID of the patient that was associated with this bed event.
10 EPT_CSN NUMERIC No
The unique contact serial number of the patient encounter that was associated with this bed event.
11 REASN_BED_NT_ASGN_C_NAME VARCHAR No
The reason not assigned category number for the bed event (if applicable). This item is cleared when the bed is assigned for cleaning.
May contain organization-specific values: No
Category Entries:
CMJ test
12 RECORD_CREATION_DT DATETIME No
The date when this record was created.
13 CANCEL_USER_ID VARCHAR No
The unique ID of the user record for the user who canceled this bed event. This column is only populated for events that have been canceled.
14 CANCEL_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
15 CANCEL_EVENT_TM DATETIME (Local) No
The instant when the bed event was canceled. This column is only populated for events that have been canceled.
16 ADHOC_USER_ID VARCHAR No
The unique ID of the user record for the user who created this bed event. This column is only populated for cleaning requests that weren't created automatically.
17 ADHOC_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
18 LINKED_EVENT_ID NUMERIC No
The unique ID of the Admission, Transfer, Discharge, or Leave of Absence (ADT) event that is associated with this bed event.
19 BED_TYPE_C_NAME VARCHAR No
This item tracks the bed type for the bed associated with the bed event record.
May contain organization-specific values: Yes
Category Entries:
Empty
Standard
Crib
Telemetry
Bariatric
20 PREV_OUT_EVENT_ID NUMERIC No
The unique ID of the previous outgoing event of type Discharge, Transfer Out or LOA Out. If the previous event on the bed is of type incoming (Admission, Transfer In, LOA Return) instead of an outgoing type, then this item will be empty.
21 NEXT_IN_EVENT_ID NUMERIC No
The unique ID of the next incoming event of type Admission, Transfer In, or LOA Return. If the next event on the bed is of type outgoing (Discharge, Transfer Out, LOA Out) instead of an incoming type, then this item will be empty.
