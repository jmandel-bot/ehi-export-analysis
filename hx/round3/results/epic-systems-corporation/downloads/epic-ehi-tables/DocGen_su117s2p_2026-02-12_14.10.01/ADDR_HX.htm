ADDR_HX
Description:
This table extracts each patient's historical addresses.

Primary Key
Column Name Ordinal Position
PAT_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 PAT_ID VARCHAR No
The unique ID of the patient record for this row. This column is frequently used to link to the PATIENT table.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 ADDR_HX_CITY VARCHAR No
This item stores the city from a patient's historical address.
4 ADDR_HX_STATE_C_NAME VARCHAR No
This item stores the state from a patient's historical address.
May contain organization-specific values: Yes
5 ADDR_HX_ZIP VARCHAR No
This item stores the postal code from a patient's historical address.
6 ADDR_HX_COUNTY_C_NAME VARCHAR No
This item stores the county from a patient's historical address.
May contain organization-specific values: Yes
7 ADDR_HX_COUNTRY_C_NAME VARCHAR No
This item stores the country from a patient's historical address.
May contain organization-specific values: Yes
8 ADDR_HX_HOUSE_NUM VARCHAR No
This item stores the house number from a patient's historical address.
9 ADDR_HX_DISTRICT_C_NAME VARCHAR No
This item stores the district from a patient's historical address.
May contain organization-specific values: Yes
10 ADDR_HX_CONTACT VARCHAR No
This item stores the name of the contact person associated with a patient's historical address.
11 ADDR_HX_PURPOSE_C_NAME VARCHAR No
This item stores the purpose of a patient's historical address.
May contain organization-specific values: Yes
No Entries Defined
12 ADDR_HX_START_DT DATETIME No
This item stores the date on which a patient's historical address became effective.
13 ADDR_HX_END_DT DATETIME No
This item stores the date on which a patient's historical address is no longer effective.
14 ADDR_HX_PDS_IDNT VARCHAR No
This item stores the unique identifier assigned to a patient's historical address by the Personal Demographics Service.
15 ADDR_HX_TYPE_C_NAME VARCHAR No
This item stores the address type of a patient's historical address.
May contain organization-specific values: No
