OB_HSB_DATING
Description:
This table contains the associated information about the criteria for determining the estimated date of delivery for this pregnancy.

Primary Key
Column Name Ordinal Position
SUMMARY_BLOCK_ID 1
LINE 2

Column Information
Name Type Discontinued?
