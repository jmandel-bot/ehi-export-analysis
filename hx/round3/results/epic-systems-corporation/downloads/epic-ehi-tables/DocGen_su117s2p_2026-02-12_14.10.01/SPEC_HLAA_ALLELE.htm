SPEC_HLAA_ALLELE
Description:
This table stores HLA-A genotype information for a platelet product.

Primary Key
Column Name Ordinal Position
SPECIMEN_ID 1
LINE 2

Column Information
Name Type Discontinued?

3 CM_PHY_OWNER_ID VARCHAR No
The Community ID (CID) of the instance that owns this record or line. This is only populated if you use IntraConnect.

Stores HLA-A genotype information for a platelet product.
