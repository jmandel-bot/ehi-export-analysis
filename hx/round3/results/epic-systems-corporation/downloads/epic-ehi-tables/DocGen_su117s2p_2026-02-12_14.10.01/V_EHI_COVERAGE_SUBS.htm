V_EHI_COVERAGE_SUBS
Description:
This view contains information about a coverage's subscriber. Used for Electronic Health Information (EHI) export.

Primary Key
Column Name Ordinal Position
COVERAGE_ID 1

Name Type Discontinued?
1 COVERAGE_ID NUMERIC No
The unique ID assigned to the coverage record. This ID may be encrypted if you have elected to use enterprise reporting’s encryption utility.
2 SUBSCRIBER_PAT_ID VARCHAR No
If the subscriber is the same person as a patient, this item contains the patient ID.
3 SUBSCRIBER_CONFDENTIAL_NAME VARCHAR No
This item contains the confidential name of the associated patient, if it exists. The name is used to determine the confidential nature of the subscriber.
4 SUBSCRIBER_NUMBER VARCHAR No
