DOC_INFORMATION_3
Description:
The DOC_INFORMATION table contains information about documents, including scanned and electronically signed documents.

Primary Key
Column Name Ordinal Position
DOCUMENT_ID 1

Column Information
Name Type Discontinued?
1 DOCUMENT_ID VARCHAR No
The unique identifier (.1 item) for the document record.
2 CREATED_BY_SYSTEM_C_NAME VARCHAR No
The system that this document originated from. This is set on a document in a document batch that was created from directory sweep. The source system comes from the directory sweep extension.
May contain organization-specific values: Yes
3 STAFF_COUNTERSIGNED_BY VARCHAR No
The name of the person who countersigned the document.
4 DOC_DEMOG_ID NUMERIC No
The demographic set (REQ) record associated with the document. This is only populated for documents created by importing files from a legacy archive system. If a patient (EPT) record's demographics matched the data from the legacy system, the document is also linked to the patient record in I DCS 270.
5 COMM_ORIG_DOCUMENT_ID VARCHAR No
This item stores the original Document (DCS) ID when a document is converted to a PDF.
6 DOC_DISCLOSED_YN VARCHAR No
This item is set to Yes when this document is disclosed through Release of Information workflows.
May contain organization-specific values: No

8 DOC_PROXY_MYPT_ID VARCHAR No
Stores WPR ID for the proxy that the document is associated with. This version of the document will be displayed to the proxy instead of the document linked in I DCS 950. Used by Tapestry.
9 SENT_BY_FAX_NUM VARCHAR No
The fax number that sent the fax that created the batch this document came from.
10 SENT_BY_CALLER_NAME VARCHAR No
The name associated with the sender of the fax that created the batch this document came from.
11 RECEIVED_FAX_SENT_DATE DATETIME No
The date the fax vendor says the fax that created the batch this document came from was sent.
12 RECD_FAX_SENT_TM DATETIME (Local) No
The time the fax vendor says the fax that created the batch this document came from was sent.
13 PROXY_ORIG_DCS_ID VARCHAR No
Stores DCS ID for the original document that this proxy document was generated for. Used by Tapestry.
