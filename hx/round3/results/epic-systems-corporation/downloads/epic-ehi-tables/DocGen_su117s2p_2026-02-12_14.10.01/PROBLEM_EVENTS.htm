PROBLEM_EVENTS
Description:
This table contains a list of relevant events documented as the history of a patient's problem.

Primary Key
Column Name Ordinal Position
PROBLEM_LIST_ID 1
CONTACT_DATE_REAL 2
LINE 3

Column Information
Name Type Discontinued?
1 PROBLEM_LIST_ID NUMERIC No
The unique identifier for the problem record.
2 CONTACT_DATE_REAL FLOAT No
A unique contact date in decimal format. The integer portion of the number indicates the date of contact. The digits after the decimal distinguish different contacts on the same date and are unique for each contact on that date. For example, .00 is the first/only contact, .01 is the second contact, etc.
3 LINE INTEGER No
The line number for the information associated with this contact. Multiple pieces of information can be associated with this contact.
4 CONTACT_DATE DATETIME No
The date of this contact in calendar format.
5 PRB_EVENT_STATE_C_NAME VARCHAR No
Stores the state of each event (active or inactive)
May contain organization-specific values: No
Category Entries:
Active
Inactive
6 PRB_EVENT_INDEX INTEGER No
Stores the index of each event in the problem's history
7 PRB_EVENT_HX_TYPE_C_NAME VARCHAR No
Stores the history type that each event was created from
May contain organization-specific values: Yes
Category Entries:
Oncology History
8 PRB_EVENT_STDATE_DT DATETIME No
Stores the start date of each event
9 PRB_EVENT_ENDATE_DT DATETIME No
Stores the end date of each event
10 PRB_EVENT_CAT_C_NAME VARCHAR No
Stores the category/type of each event
May contain organization-specific values: Yes
Category Entries:
Initial Diagnosis
11 PRB_EVENT_DESC VARCHAR No
Stores the description/details of each event
12 PRB_EVENT_NOTE_ID VARCHAR No
Stores the problem event note ID
13 PROBLEM_EVENT_LINKED_ACTION_C_NAME VARCHAR No
This item contains the action configured in Problem List Automated Actions (I LSD 2765) that triggered the automatic creation of the oncology history event on the same line.
May contain organization-specific values: No
Category Entries:
External Event
Sign Cancer Stage
Save Treatment or Therapy Plan
Save Radiation Therapy Episode
Save BMT Episode
Save IEC Episode
14 PROBLEM_EVENT_AUTO_UPDATE_YN VARCHAR No
This item stores a yes or no to indicate whether or not an event will automatically update.
May contain organization-specific values: No
Category Entries:
No
Yes
15 CONTACT_SERIAL_NUM NUMERIC No
This column contains the contact serial number of the oncology history problem (LPL) contact.
16 PRB_EVENT_USR_DTL_AUTO_NOTE_ID VARCHAR No
This column contains the note ID for the note containing the auto-generated event's user-entered comments.
17 PRB_EVENT_AUTO_NOTE_BEHAVIOR_C_NAME VARCHAR No
This column contains whether the user-entered comments will replace or be appended to the auto-generated event details when the event is viewed in a SmartLink or print group. This setting only applies to events which are generated by the system and cannot be otherwise edited.
May contain organization-specific values: No
Category Entries:
Append
Replace
18 PRB_EVENT_EXTERNAL_UNIQUE_KEY VARCHAR No
This item contains a key to uniquely identify an oncology history event created by the external events API. If filing an automated event based on chronicles data, this item should not be used.
19 PROB_EXTERNAL_EVENT_START_DATE DATETIME No
This item contains the start date for an oncology history event filed from the external events API. It should contain the same value as LPL 8103-problem event start date unless a user has chosen to manually override the start date for the event. If filing an automated event based on chronicles data, this item should not be used.
20 PROB_EXTERNAL_EVENT_END_DATE DATETIME No
This item contains the end date for an oncology history event filed from the external events API. It should contain the same value as LPL 8104-problem event end date unless a user has chosen to manually override the end date for the event. If filing an automated event based on chronicles data, this item should not be used.
