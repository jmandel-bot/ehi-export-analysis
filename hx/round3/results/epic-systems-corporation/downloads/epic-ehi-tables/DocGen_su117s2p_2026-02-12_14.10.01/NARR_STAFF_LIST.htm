NARR_STAFF_LIST
Description:
Staff event tracking items.

Primary Key
Column Name Ordinal Position
PAT_ENC_CSN_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 PAT_ENC_CSN_ID NUMERIC No
The unique contact serial number for this contact. This number is unique across all patient encounters in your system. If you use IntraConnect, this is the Unique Contact Identifier (UCI).
2 LINE INTEGER No
The line number for the information associated with this contact. Multiple pieces of information can be associated with this contact.
3 PAT_ENC_DATE_REAL FLOAT No
A unique contact date in decimal format. The integer portion of the number indicates the date of contact. The digits after the decimal distinguish different contacts on the same date and are unique for each contact on that date. For example, .00 is the first/only contact, .01 is the second contact, etc.
4 CONTACT_DATE DATETIME No
The date of this contact in calendar format.
5 NARR_STAFF_PROV_ID_PROV_NAME VARCHAR No
The name of the service provider. This item may be hidden in a public view of the CLARITY_SER table.
6 NARR_STAFF_UNK_ID VARCHAR No
Stores the unique ID that is given to an unknown provider involved with an intervention until it is linked to an existing provider record.
7 NARR_STAFF_ROLE_C_NAME VARCHAR No
Stores the provider capacity (e.g. attending, resident, etc.) during this intervention.
May contain organization-specific values: Yes
8 NARR_STAFF_EVENT_C_NAME VARCHAR No
Indicates which type of event has been filed, for example whether a staff member has arrived or departed.
May contain organization-specific values: Yes
Category Entries:
Paged
Responded
Arrived
Departed
Deleted
9 NARR_STAFF_INST_TM DATETIME (Local) No
The instant at which the staff event is filed.
10 NARR_STAFF_COMMENT VARCHAR No
The user-entered comment for a staff related event (i.e. marking a provider as paged or arrived).
11 NARR_STAFF_LINKTO NUMERIC No
Stores the line ID of a different event, if this event was modified or deleted by it.
12 NARR_STF_MODINST_TM DATETIME (Local) No
Stores the instant of an event's modification or deletion, if applicable.
13 NARR_STAFF_DOCU_ID VARCHAR No
The unique ID of the user who documented this event.
14 NARR_STAFF_DOCU_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
15 NARR_STAFF_IS_AT_YN VARCHAR No
Stores whether or not the specified staff member is marked as an attending physician.
May contain organization-specific values: No
Category Entries:
No
Yes
16 NARR_STAFF_TYPE_ID_TANK_NAME VARCHAR No
The name of the tank record.
17 NARR_STAFF_IEV_ID VARCHAR No
Stores the ID of the event that has updated this event.
18 NARR_STAFF_IEV_LINE INTEGER No
Indicates that this event has been updated by another event and links to that event's line.
19 NARR_STAFF_DOC_SRC_C_NAME VARCHAR No
Stores the source of the staff documentation event.
May contain organization-specific values: No
Category Entries:
RTLS
RTLS - Manually Updated
