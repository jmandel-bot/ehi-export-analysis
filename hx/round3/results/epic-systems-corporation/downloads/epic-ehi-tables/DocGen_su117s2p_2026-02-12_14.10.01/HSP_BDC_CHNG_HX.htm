HSP_BDC_CHNG_HX
Description:
Change History for the Denial/Correspondence (BDC) record.

Primary Key
Column Name Ordinal Position
BDC_ID 1
LINE_COUNT 2

Column Information
Name Type Discontinued?
1 BDC_ID NUMERIC No
This column stores the unique identifier for the denial or correspondence record.
2 LINE_COUNT INTEGER No
Line number of change history that is being extracted by enterprise reporting.
3 CHNG_INSTANT DATETIME (Local) No
Instant this change occurred
4 CHNG_USER_ID VARCHAR No
User who initiated the change
5 CHNG_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
6 CHNG_TYPE_C_NAME VARCHAR No
Type of change that occurred.

7 CHNG_SOURCE_VAL VARCHAR No
This column stores the source value before change. It can be the unique identifier for the liability bucket or the denial/correspondence type (I BDC 30) depending on the type of change.
8 CHNG_TARGET_VAL VARCHAR No
This column stores the target value after change. It can be the unique identifier for the liability bucket or the denial/correspondence type (I BDC 30) depending on the type of change.
9 CHNG_FOLLOW_UP_DT DATETIME No
The follow-up date associated with the change.
10 CHNG_COMMENTS VARCHAR No
Comments for change.
11 BFH_ID NUMERIC No
Stores the Billing Follow-up History (BFH) record ID that contains the information about the billing activity that was performed on this BDC.
