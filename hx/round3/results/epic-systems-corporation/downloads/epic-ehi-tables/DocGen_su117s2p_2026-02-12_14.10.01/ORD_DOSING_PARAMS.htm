ORD_DOSING_PARAMS
Description:
This table contains dosing parameters.

Primary Key
Column Name Ordinal Position
ORDER_ID 1

Column Information
Name Type Discontinued?
1 ORDER_ID NUMERIC No
The unique identifier for the order record.
2 ORD_DOSING_WEIGHT NUMERIC No
Weight used for dosing. Always stored in kilograms.
3 ORD_DW_REC_DTTM DATETIME (Local) No
The instant at which the weight was recorded.
4 ORD_WT_SOURCE_C_NAME VARCHAR No
This column contains the source of the patient weight used for dosing patient-controlled analgesia (PCA) medication.
May contain organization-specific values: Yes
Category Entries:
Treatment plan weight
Past measured weight
Most current measured weight (actual)
Ideal body weight
Adjusted body weight
User entered weight
Lean body weight
Dosing weight
Birth Weight
Other
5 ORD_WT_COMMENTS VARCHAR No
Generated comment for dosing weight.
6 ORD_DOSING_HEIGHT NUMERIC No
This column contains the patient height used for dosing PCA medication. The value stored is in inches for all orders after weight-based dosing was turned on, or starting in Spring 2008, whichever came first. Values are stored in centimeters for treatment plan orders made prior to that.
7 ORD_HT_REC_DTTM DATETIME (Local) No
The instant at which the height was recorded.
8 ORD_HT_SOURCE_C_NAME VARCHAR No
This column contains the source of the patient height used for dosing patient-controlled analgesia (PCA) medication.
May contain organization-specific values: Yes
Category Entries:
Treatment plan height
Past measured height
Most current measured height (Actual)
User Entry
Other
9 ORD_HT_COMMENTS VARCHAR No
Generated comment for dosing height.
10 ORD_DOSING_BSA NUMERIC No
The body surface area used for dosing.
11 ORD_BSA_SRC_C_NAME VARCHAR No
This column contains the source of the body surface area used for dosing patient-controlled analgesia (PCA) medication.
May contain organization-specific values: Yes
Category Entries:
Treatment plan BSA
Based on weight and height measured in the past
Based on most recent measured weight and height (actual)
User Entry
Based on ideal body weight and current/past measured height
Based on adjusted body weight and current/past measured height
Based on dosing weight and current/past measured height
Based on normalized BSA 1.7m2
Birth weight and current height BSA
Other
12 ORD_BSA_CALC_DTL VARCHAR No
The dosing body surface area calculation details with weight, height and recorded instants.
13 ORD_BSA_COMMENTS VARCHAR No
Generated comment for dosing body surface area.
14 PAT_REPORTED_WEIGHT_SOURCE_C_NAME VARCHAR No
Indicates the source of a patient-reported dosing weight value, such as reported by a smart device or manually entered by the patient.
May contain organization-specific values: No
Category Entries:
Patient Manually Reported
Device Reported
