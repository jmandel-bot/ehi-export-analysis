ORG_DONOR_PROC_TYPES
Description:
This table contains deceased donor procedure types associated with an organ record.

Primary Key
Column Name Ordinal Position
ORG_RECORD_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 ORG_RECORD_ID NUMERIC No
The unique identifier for the organ record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 DONOR_PROC_TYPE_C_NAME VARCHAR No
The donor procedure type associated with this organ.
May contain organization-specific values: Yes
