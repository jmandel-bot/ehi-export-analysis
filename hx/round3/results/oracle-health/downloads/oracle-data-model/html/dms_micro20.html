Column Name Type Null? Definition

FILTER_END_STRING VC2(100) Y This field is used to store the ending string value for the default filter row.
FILTER_FLAG NUMBER Y This field is used to store a small numeric value for the default filter row.
FILTER_ID NUMBER N This field contains the internal identification for this reference table.
FILTER_NBR NUMBER Y This field is used to store a large numeric value for the default filter row.
PARENT_ENTITY_ID NUMBER N This field is used to store a code value or identification for the default filter row. The parent_entity_name will determine which table the value came from.
PARENT_ENTITY_NAME VC2(32) N This field is used to store the name of the table that places a value in the parent_entity_id field.
PARENT_FILTER_ID NUMBER N This field is used to store an identification for the 'parent' filter row. The parent_entity_name will determine which table the value came from.
PARENT_FILTER_NAME VC2(32) N This field is used to store the name of the 'parent filter' table. Currently, this column is only valued with 'MIC_SEQ_FILTER_DEFAULT'.
PARENT_SEQ NUMBER N This field is used in conjunction with the PARENT_FILTER_NAME and PARENT_FILTER_ID columns to identify the 'parent' filter row. When the PARENT_FILTER_NAME is valued with MIC_SEQ_FILTER_DEFAULT this field maps to the SEQUENCE column on the MIC_SEQ_FILTER_DEFAULT table.
SEQUENCE_ID NUMBER N This field contains the foreign key value used to join to sequence information stored on the mic_seq_filter and mic_sequence reference table.
UPDT_APPLCTX NUMBER N The application context number from the record info block.
