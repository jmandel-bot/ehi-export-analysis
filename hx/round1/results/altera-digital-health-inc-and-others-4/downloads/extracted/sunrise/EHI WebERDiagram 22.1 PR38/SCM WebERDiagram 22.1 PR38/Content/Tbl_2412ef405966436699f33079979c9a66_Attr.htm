SiteId 
Definition  Identifies the unique distributed database ID where the row was inserted. Distributed databases are no longer supported so this column is deprecated. 
Build 
Definition  Identifies the build number and patch level of the software that inserted/updated this row. The format is the Build number followed by 3 digits for the patch level which is typically 000. The build number is tied to specific versions of the software, e.g. Sunrise 15.3 is Build 5021, Sunrise 16.3 is Build 5503. 
TouchedBy 
Definition  Information about the user that last updated the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
TouchedWhen 
Definition  The date and time that the row was last updated and will be based on the Enterprise time zone. 
CreatedBy 
Definition  Information about the user that created the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
CreatedWhen 
Definition  The date and time that the row was created and will be based on the Enterprise time zone. 
OrderReviewPolicyDetailID 
Definition  Auto generated unique primary key value. 
OrderReviewPolicyID 
Definition  A foreign key to the OrderReviewPolicyID column in the SXAMMOrderReviewPolicy table. 
FacilityGUID 
Definition  This is the GUID of the facility assigned from the CV3Location table. It is based on records with the IsFacility column set to true 
LocationFunctionGroupGUID 
Definition  This is the GUID of the location function group from CV3LocationFunctionGroup 
LocationGroupGUID 
Definition  This is the GUID of the location group from CV3Location. It is based on records with the IsLocationGroup column set to true 
Msrepl_tran_version 
Definition  Optimistic lock identifier. 
MSReplrowguid 
Definition  Unique GUID used to support merge replication.
