dbo.SXACRTileIVDripDetail 
Default Table Name  SXACRTileIVDripDetail 
Definition  stores the configuration detail information for IV Drip tile instance. There's a one-to-one mapping between tile instance and an entry here. Stores tile specific configuration details that controls rendering of the tile.
