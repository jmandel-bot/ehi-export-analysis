dbo.SXAMMStockItemADMXref 
Default Table Name  SXAMMStockItemADMXref 
Definition  SMM to allow configuration of multiple cross references, potentially one for each communication process (Procar). The existing cross reference at the Stock Item level would be the ‘default’ and would always be used if there was no ‘override’ cross reference at the communication process (Procar) level.
