dbo.SXAMMPriceAlgorithm 
Default Table Name  SXAMMPriceAlgorithm 
Definition  This entity contains a list of Price Algorithms that are defined for the entire enterprise. A Price Algorithm defines how the Cost (determined by the Charge Algorithm) is used to calculate the Price (amount to bill). A Price Algorithm can be Active or Inactive, and can include markups, fees, a rounding option, and minimum price functionality. In addition, the Price Algorithm can be tiered, so a different set of rules can be applied based upon which range (Cost From / Cost To) the Cost falls into. Each Price Algorithm can be associated with multiple Generic Items.
