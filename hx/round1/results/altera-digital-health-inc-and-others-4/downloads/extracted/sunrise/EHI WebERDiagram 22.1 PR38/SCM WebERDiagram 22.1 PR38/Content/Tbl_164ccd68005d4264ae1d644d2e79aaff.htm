dbo.SXAMMOrderTypeLayout 
Default Table Name  SXAMMOrderTypeLayout 
Definition  This table helps define a default layout to use for dispensing details for each order type. There will be one row in this table per OrderType defined for SXAMMDispenseSubType.OrderType
