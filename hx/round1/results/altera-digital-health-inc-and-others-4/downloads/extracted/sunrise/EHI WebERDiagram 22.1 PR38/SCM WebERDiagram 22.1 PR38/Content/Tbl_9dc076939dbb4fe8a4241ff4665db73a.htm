dbo.SXAMMGINote 
Default Table Name  SXAMMGINote 
Definition  This entity contains Order Entry Default Notes specific for a Generic Item Formulary.
A Generic Item Formulary can have:
ONE Default Type = "Formulary" and
UNLIMITED Default Type = "Mnemonic"
