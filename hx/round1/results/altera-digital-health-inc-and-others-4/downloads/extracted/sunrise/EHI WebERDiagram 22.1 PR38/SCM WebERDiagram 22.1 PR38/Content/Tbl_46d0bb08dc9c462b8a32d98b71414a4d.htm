dbo.SXACDExDocPrivacyStatementPatientCareDocXRef 
Default Table Name  SXACDExDocPrivacyStatementPatientCareDocXRef 
Definition  This table stores the link between the privacy statement and the exchange document. The privacy statement is from SXACDExDocPrivacyStatement table while the document is from CV3PatientCareDocument table.

Build    int  NO  Identifies the build number and patch level of the software that inserted/updated this row. The format is the Build number followed by 3 digits for the patch level which is typically 000. The build number is tied to specific versions of the software, e.g. Sunrise 15.3 is Build 5021, Sunrise 16.3 is Build 5503.
