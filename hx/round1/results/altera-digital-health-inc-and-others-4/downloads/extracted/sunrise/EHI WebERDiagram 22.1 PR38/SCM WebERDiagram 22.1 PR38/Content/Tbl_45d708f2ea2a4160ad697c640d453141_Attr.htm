SiteID 
Definition  Identifies the unique distributed database ID where the row was inserted. Distributed databases are no longer supported so this column is deprecated. 
Build 
Definition  Identifies the build number and patch level of the software that inserted/updated this row. The format is the Build number followed by 3 digits for the patch level which is typically 000. The build number is tied to specific versions of the software, e.g. Sunrise 15.3 is Build 5021, Sunrise 16.3 is Build 5503. 
TouchedBy 
Definition  Information about the user that last updated the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
TouchedWhen 
Definition  The date and time that the row was last updated and will be based on the Enterprise time zone. 
CreatedBy 
Definition  Information about the user that created the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
CreatedWhen 
Definition  The date and time that the row was created and will be based on the Enterprise time zone. 
Active 
Definition  Denotes whether the row is active or not, 0 means the row has been soft deleted or deactivated. 
ProductID 
Definition  Auto generated unique primary key value. 
NDC 
Definition  The national drug code, unique to manufacturer, generic item, and package size. It can be used to identify the product in the drug vendor's database. 
BrandName 
Definition  The name by which the product is known 
GenericItemID 
Definition  A foreign key to the GenericItemID column in the SXARxGenericItem table. 
ManufacturerID 
Definition  A foreign key to the ManufacturerID column in the SXARxManufacturer table. 
InnerPackageSize 
Definition  inner package size 
InnerPackageSizeUOMID 
Definition  A foreign key to the RxUOMID column in the SXARxUnitOfMeasure table. 
OuterPackageSize 
Definition  outer packaging size 
IsUnitDose 
Definition  An indicator whether this product is packaged for unit dose 
IsRepackaged 
Definition  An indicator whether this product is repackaged from a product by another manufacturer 
AWP 
Definition  Average wholesale price. This usually drives what the hospital charges for the drug. 
ObsoleteDate 
Definition  Enterprise TimeZone - The date when this product is no longer available from the manufacturer 
IsMultidose 
Definition  An indicator whether this product is a multi dose package 
IsOTC 
Definition  A Indicator whether this product is available to the general public (over the counter) 
OrangeBookCode 
Definition  A foreign key to the OrangeBookCode column in the SXARxOrangeBook table. 
DrugCatalogKey 
Definition  The unique identifier for this data in the drug vendor`s database. This is the NDC code. 
MSrepl_tran_version 
Definition  Optimistic lock identifier. 
MSReplrowguid 
Definition  Unique GUID used to support merge replication.
