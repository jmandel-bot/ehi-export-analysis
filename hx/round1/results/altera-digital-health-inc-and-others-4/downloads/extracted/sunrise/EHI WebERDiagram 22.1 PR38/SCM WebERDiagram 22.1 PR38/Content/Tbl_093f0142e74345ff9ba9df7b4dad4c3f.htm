dbo.SXARCMChargeSummaryAIRConfig 
Default Table Name  SXARCMChargeSummaryAIRConfig 
Definition  This table is used when doing All Inclusive Revenue Code summarization in third party billing. The table allows the association of a visit type with a revenue code. This revenue code is used in expected reimbursement to retrieve OMB rate associated with it. The revenue code is also used in claim line population.
