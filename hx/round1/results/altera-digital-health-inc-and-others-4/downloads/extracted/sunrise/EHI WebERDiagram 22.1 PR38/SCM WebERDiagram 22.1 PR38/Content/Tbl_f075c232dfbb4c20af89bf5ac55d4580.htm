dbo.SXACommunityDataStatus 
Default Table Name  SXACommunityDataStatus 
Definition 
This table stores the status of community data import attempts. Each row represents an import attempt for a specific patient, community data provider and domain (e.g. Allergies, Health Issues, Medications, Immunizations). 

dbo.SXACommunityDataStatus Columns

The format is the Build number followed by 3 digits for the patch level which is typically 000.
