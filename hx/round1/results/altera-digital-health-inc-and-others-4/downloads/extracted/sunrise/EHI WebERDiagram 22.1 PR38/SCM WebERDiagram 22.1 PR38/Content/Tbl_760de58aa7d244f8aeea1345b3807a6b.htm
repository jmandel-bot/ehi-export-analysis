dbo.CV3OrderAvailabilityPolicy 
Default Table Name  CV3OrderAvailabilityPolicy 
Definition  Defines an availability policy that can be associated to an order catalog item, order set or pathway. This table, together with CV3OrderAvailabilityPolicyDtl, will allow the site to define the locations at which an order catalog item, pre-filled order item, order set or pathway are available. It can also define which order entry from will be used at the particular location.
