dbo.SXAAMELIncomingRelatedReferralXREF 
Default Table Name  SXAAMELIncomingRelatedReferralXREF 
Definition  This table provides the parent/child linking relationship between a Primary or Related referral and each of its Sub referrals. A Sub referral may only have a single Primary or Related referral as its parent. However, a Primary or Related referral may have many Sub referrals as children. 

dbo.SXAAMELIncomingRelatedReferralXREF Columns 
ColumnName  Domain  Datatype  NULL  Definition 
Build    int  NO  Identifies the build number and patch level of the software that inserted/updated this row. The format is the Build number followed by 3 digits for the patch level which is typically 000. The build number is tied to specific versions of the software, e.g. Sunrise 15.3 is Build 5021, Sunrise 16.3 is Build 5503. 
CreatedBy    varchar(50)  NO  Information about the user that created the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
CreatedWhenUTC    datetimeoffset(3)  NO  The date and time that the row was created including the time zone offset from UTC, for example 2015-01-30 04:55:04.372 -05:00. The date and time is based on the Enterprise Time Zone. 
RelatedReferralGroupID    bigint  NO  Identifier to provide the relationship between related referrals 
IncomingReferralID    bigint  NO  A foreign key to the IncomingReferralID column in the SXAAMREFIncomingReferral table.
