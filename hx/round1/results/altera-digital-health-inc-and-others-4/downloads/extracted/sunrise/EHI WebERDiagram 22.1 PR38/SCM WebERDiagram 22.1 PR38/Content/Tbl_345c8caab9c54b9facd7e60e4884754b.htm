dbo.SXAHMCountPreviousEventXRef 
Default Table Name  SXAHMCountPreviousEventXRef 
Definition  In the Property-Condition Group Dialog, this table tracks the property called Count Previous Events which allows the user to select 0 to N other events to count towards the series linked to the property, as a previous occurrence. For example, TD would count TDaP completed occurrences when calculating the next dose of the TD booster.
