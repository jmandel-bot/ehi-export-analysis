dbo.SXAAMBPrescriptionSampleHistory 
Default Table Name  SXAAMBPrescriptionSampleHistory 
Definition  Lot number and expiration date are stored in the SXAAMBPrescriptionSample table. During the Sample edit process the user has the ability to add or remove lot numbers and the expiration date associated with that lot number. These items are stored in this table for Auditing purposes.
