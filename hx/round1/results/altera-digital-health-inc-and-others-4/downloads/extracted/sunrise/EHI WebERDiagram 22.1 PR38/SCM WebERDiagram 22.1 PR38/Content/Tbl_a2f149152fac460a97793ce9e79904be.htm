dbo.CV3HealthIssueTypeCodingSchemeXRef 
Default Table Name  CV3HealthIssueTypeCodingSchemeXRef 
Definition  Cross-reference table that associates a Health Issue coding scheme record to an existing CV3HealthIssueType record. Used to list of set of allowable coding schemes associated with a health issue type.
