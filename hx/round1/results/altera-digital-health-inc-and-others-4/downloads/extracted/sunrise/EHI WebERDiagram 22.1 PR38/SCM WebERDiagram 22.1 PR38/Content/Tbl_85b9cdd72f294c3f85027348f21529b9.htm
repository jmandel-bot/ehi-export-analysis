dbo.CV3ClientDocDetailHistXrefCUR 
Default Table Name  CV3ClientDocDetailHistXrefCUR 
Definition  Contains current document records. This table is a cross-reference between the Client Doc History and the Client Doc Detail. It identifies all the detail objects which are valid for the document at a certain point in its history.
