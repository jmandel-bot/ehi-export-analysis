dbo.SXAMMRouteOrderTypeXref 
Default Table Name  SXAMMRouteOrderTypeXref 
Definition  Current logic in SMM identifies the Order Type for a SCM placed order by looking into the SCM Form attributes. This will be an issue in cases where the same form in SCM is used for multiple Order Types. With the new table in place, SMM can identify Order Types based on route.
