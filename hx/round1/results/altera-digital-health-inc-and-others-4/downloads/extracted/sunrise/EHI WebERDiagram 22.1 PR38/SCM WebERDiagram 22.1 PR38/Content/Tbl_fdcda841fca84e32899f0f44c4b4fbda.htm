dbo.SXACRTileOrderItemXref 
Default Table Name  SXACRTileOrderItemXref 
Definition  Defines list of electrolyte mappings for the electrolyte tile. The user maps existing order item for sodium, potassium, create, bun, co2 and ch. Each view could have multiple instance of the electrolyte tile
