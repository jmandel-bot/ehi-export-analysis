dbo.SXAHMSchedulingPriority 
Default Table Name  SXAHMSchedulingPriority 
Definition  SXAHMSchedulingPriority is a Health Management dictionary that stores all possible priorities that a schedule can be set to, which is stored in the SXAHMSchedule table's SchedulingPriorityCode column.
