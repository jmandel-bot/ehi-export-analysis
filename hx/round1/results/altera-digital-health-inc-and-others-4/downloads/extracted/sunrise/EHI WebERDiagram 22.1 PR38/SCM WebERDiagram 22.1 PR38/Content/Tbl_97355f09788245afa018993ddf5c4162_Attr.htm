Build 
Definition  Identifies the build number and patch level of the software that inserted/updated this row. The format is the Build number followed by 3 digits for the patch level which is typically 000. The build number is tied to specific versions of the software, e.g. Sunrise 15.3 is Build 5021, Sunrise 16.3 is Build 5503. 
TouchedBy 
Definition  Information about the user that last updated the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
TouchedWhenUTC 
Definition  The date and time that the row was last updated including the time zone offset from UTC, for example 2015-01-30 04:55:04.372 -05:00. The date and time is based on the Enterprise Time Zone. 
CreatedBy 
Definition  Information about the user that created the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
CreatedWhenUTC 
Definition  The date and time that the row was created including the time zone offset from UTC, for example 2015-01-30 04:55:04.372 -05:00. The date and time is based on the Enterprise Time Zone. 
DateRangeColumnConfigColumnID 
Definition  Auto generated unique primary key value. 
DateRangeColumnConfigurationID 
Definition  A foreign key to the DateRangeColumnConfigurationID column in the SXAHMDateRangeColumnConfiguration table. 
HeaderText 
Definition  Defines the display text that will appear in the header at the top of the column. It should clearly let the user know what data will be in this column. 
Description 
Definition  Provides a detailed description of the purpose of this column and the data that it is intended to contain. This information may appear as a tool-tip to let the user know more information about the intend and data of the column. 
Span 
Definition  The integer value of the elapsed time that this individual column will span. This number refers to the SpanUOM and has little meaning without the unit of measure for the value in this column. 
SpanUOM 
Definition  The unit of the time span of the column. That is, this describes how much time each column covers. For example: Year, Month, Week, Day. 
IsHiddenWhenEmpty 
Definition  This flag specifies if this column should be hidden when there is no data in the entire column. 
SubColumnSpan 
Definition  Specifies the span of the subcolumns that this column may be broken down into. 
SubColumnSpanUOM 
Definition  Specifies the unit of measure for the time covered by each subcolumn. 
SubColumnHeaderType 
Definition  Specifies a formatting type for the subcolumn header that will be dynamically generated 
MSRowVersion 
Definition  Optimistic lock identifier using rowversion datatype. 
MSReplrowguid 
Definition  Unique GUID used to support merge replication.
