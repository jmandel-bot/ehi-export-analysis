DECISION_VALUE_FLAG DOUBLE N The value of the decision. A decision value can either be Not Require Review (1) or Require Review (2).
ORDER_ID DOUBLE N The identifier of the order which is associated to the review decision.
ORDER_REVIEW_DECISION_ID DOUBLE N The primary key of this table. It is internally assigned by the system.
REVIEW_TYPE_FLAG DOUBLE N The type of review which the decision was made for. Currently, the only applicable type of review is Doctor Co-sign (2).
