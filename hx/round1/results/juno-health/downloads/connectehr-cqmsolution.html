EHI Export with Juno ConnectEHR and Juno CQMsolutions

Juno ConnectEHR and Juno CQMsolutions have been certified to (b)(10) under the limited ePHI USCDIv1 definition.

Juno ConnectEHR Electronic Health Information export

The Juno ConnectEHR UI provides the ability to export single-patient and bulk data export downloads in C-CDA 2.1 XML format.

Customers and their staff, with enhanced permissions, will be allowed to export single-patient or bulk data using the HL7 C-CDA 2.1 Standard. These exports can be scheduled or done on-demand. Configuration of the Juno ConnectEHR UI allows these functions and features to be enabled.

Juno ConnectEHR is certified to (b)(10) under the limited ePHI USCDIv1 definition. The ConnectEHR UI provides bulk data export downloads in C-CDA 2.1 XML format.

Juno FHIR API supports creating a single-patient FHIR resource DocumentReference and FHIR Bulk Data Export for patient populations.

When extracting and running a report the UI display screen will show that the data was queued and provide the completion status to proceed with the download. â€‹

Report Extraction
On-demand data export reports allow the user to select all patients, a single-patient, or a subset of patients at a specific practice.

  1. Login to Juno ConnectEHR and navigate to Administration>Data Export>>Data Export
  2. Choose a Practice from the Practice List dropdown.
  3. Click on Export specific patients and search for your patient(s)
  4. Confirm your patients then Click Run Report
  5. Navigate to Administration>Data Export>>Data Export Status to identify and download your batch.

Juno CQMsolution Electronic Health Information export

The Juno CQMsolution from Juno Health provides customers the toolset to view clinical quality information such as calculations and displays, generate the quality measures values, view patient-level results, and examine underlying data..

Juno CQMsolution is certified for the 2015 Edition Cures Update of the ONC Health IT Certification Program:

Report Outputs
Once Juno CQMsolution has calculated the numerators, denominators, etc. for all of the measures, it outputs QRDA I and QRDA III files and a QRDA III XML file which is viewable in the GUI.â€¯The QRDA I files are computable XML files containing patient-level data.

The type of report output for QRDA I (and other) files will be determined by your Juno CQMsolution license, user preferences, and the report type.

Any QRDA I program output satisfies 170.315(b)(10) EHI Export requirements. This includes HQR and Joint Commission output formats.
