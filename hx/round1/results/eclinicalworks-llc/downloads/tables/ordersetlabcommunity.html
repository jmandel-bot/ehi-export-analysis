EHI Export Schema
Table ordersetlabcommunity
Description This table contains order set linked labs for a community user.
Column DataType Description
id (Primary Key) int It is auto-generated unique primary key
Itemid int This column stores numeric type data. This column refers to itemid column in items table.
OrderSetEncid int This column stores encounterid assigned for the Orderset created by end user. This column refers to column encounterid from enc table.
Communityid int This column stores id of the Community when the Order Set is a Community Order Set.
deleteFlag tinyint A flag indicating whether the record has been deleted or not. 0 means not deleted, and 1 means deleted. Example: 0, 1.
displayindex int An integer representing the display order of the lab item within the order set. Example: 1, 2, 3.
futureFlag tinyint A flag indicating whether the lab item is for future use or not. 0 means not for future use, and 1 means for future use. Example: 0, 1.
