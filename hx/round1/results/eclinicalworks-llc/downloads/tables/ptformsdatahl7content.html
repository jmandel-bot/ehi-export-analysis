EHI Export Schema
Table ptformsdatahl7content
Description This table stores the HL7 message and the details related to it, which are received from Chadis and Rothman interfaces.
Column DataType Description
hl7msgid (Primary Key) int This is an auto incremented primary key of this column.
hl7message text This column stores the free text decrypted HL7message which is received from the lab companies.
datereceived datetime This column stores the date when result received from the lab company.
datemodified datetime This column store the date when the values last updated.
status tinyint This column stores the status of the message i.e., 3 - Preprocessing pending, 0 - Processing Pending, 1 - Pass status, 2 - Failed Status.
intid int This column stores the unique id labcompany. It refers to the id column of intlist table.
Sentid int This column stores the values of sid column of engine_intresultssent table from labhub, when result received from the hub.
HubSpokeReceivedStatus tinyint This column stores 0 or 1 based on results received from hub. When results received from the hub, it will update the status to 1.
messagesize decimal This column stores the size of the HL7message that is received.
allowedsize decimal This column stores the max allowed size of the HL7message from the hub.
maxsizeexceptiontype tinyint This column stores the type of large message exception. 0 for largemessageException and 1 for maxMessageSizeException.
