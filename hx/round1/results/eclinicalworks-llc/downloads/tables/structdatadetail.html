EHI Export Schema
Table structdatadetail
Description This table stores the detailed information about the structure of questions and their types, and various properties related to their display and usage.
Column DataType Description
id (Primary Key) int This is auto-incremented primary key in This column . This column stores unique id for each structured item created by users.
catid int This column stores id of a category for which structured item is being created. This column refers to itemid column in items table.
itemid int This column stores id of an item for which structured item/question has been created by end user. This column refers to itemid column in items table.
name varchar This column stores name or question created by end user of a structured item.
type smallint This column stores a number value which indicates what should be the type of the structured data on local . Value 0 id structured text, Value 1 is free text, Value 2 is date, Value 3 is Boolean yes and no.
mandatory smallint This column stores flags 0/1 to indicate whether question is mandatory to fill or not. 0 - question is mandatory, 1 - Item is mandatory.
displayIndex int This column stores a number to indicate in which order form questions should be listed on User Interface.
deleteFlag smallint This column stores values 0/1 to indicate whether the item has been marked deleted or not. 0 - Item is not marked deleted, 1 - Item is marked deleted.
parentid int This column stores id of a parent question under which new question has been created. This column refers to id column in structdatadetail table.
triggerflag int This column stores id of a question options for which this question has been triggered. This column refers to Optionid column in structdataoptions table.
itemType varchar This column stores a value which indicates what is the type of the structured data whether it is a category or just an Item . 'C' - is a category and 'I' - is an item.
nodeLevel smallint This column stores level of question like parent question or child question level. For parent question node level value as 0, child question node level value as 1 and for second level child question node level values as 2 etc.
tblName varchar This column stores the table name for a question to indicate category/module and item belongs to. i.e structSocialHistory , structHpi , structExam, structPreventive, structPhysicalExam etc
chartDisplay tinyint This column stores a value 0 or 1 to indicate whether the data should come up on chart or not. Value 0 indicates the structured item will not come up on chart and Value 1 indicates the structured item will come up on chart.
multiSelect tinyint This column stores a value which indicated whether the structured question us multiselect or not. Value 1 is states user can multiselect the options 0/NULL states user cannot multiselect the options
IsSearchable smallint This column stores a value which indicated whether the structured question is searchable or not. Value 1 is states the question is searchable , the value 0 states the question is not searchable.
defaultValue varchar This column stores a value which will be a default value for that question . The column value will be NULL if the question doesn't have a default value.
defaultValueid varchar This column stores an id which will be a default value for that question . This column refers to column optionid from structdataoptions table or stores blank '' if the question doesn't have a default value id.
modifynotes tinyint This column stores a value 0 or 1 to indicate whether the notes of that question can be modified or not. Value 0 indicates that the notes of the structured element cannot be modified and Value 1 indicates that the notes of the structured element can be modified
readonly tinyint This column stores a value 0 or 1 to indicate whether the question is read-only. Value 0 means the question is not readonly whereas value 1 means it is a readOnly element.
dropdown tinyint This column stores a value 0 or 1 to indicate whether the question should have a dropdown or not. Value 0 indicates that the question cannot be a dropdown and Value 1 indicates that question can be a dropdown.
PmtPlanid int This column stores id of a payment plan id . This column refers to planid column in paymentplandetail table.
imagename varchar This column stores the value of a file name of the image associated to that structured questions.
minVal varchar This column stores a value which will be a minimum value for that question . The column value will be blank if the question doesn't have a minimum value.
maxVal varchar This column stores a value which will be a maximum value for that question . The column value will be blank if the question doesn't have a maximum value.
multitrigger varchar This column stores a value which triggers multiple structured questions.
displayOnPn tinyint This column stores a value 0 or 1 to indicate whether the question should display on progress notes or not. Value 1 indicates the question should display on progress notes and Value 0 states the question should not display on progress notes.
inactive tinyint It Indicates whether the data item is inactive (1) or not (0). It default value is 0 means active.
