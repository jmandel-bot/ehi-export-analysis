EHI Export Schema
Table nt_worksheet_list
Description This table stores information about worksheets, their names, URLs, and other related properties.
Column DataType Description
id (Primary Key) int The id column is the primary key It is unique for each worksheets.
NAME varchar The name column stores the name of the worksheet with respect to the id column. Datatype is varchar
form_url text The form_url coulmn stores the name of the jsp of the respective worksheet. Datatype is text
isWorksheet int The isWorksheet column denotes the worksheet deployed on the server. isWorksheet is equal to 1 if the worksheet is deployed and present.
deleteflag int The deleteflag column stores the value as 1 / 0 for worksheets. deleteflag = 1 : The worksheet wont show up in the worksheet list deleteflag = 0 : The worksheet would show up in the worksheet list
docttype int The doctype column stores the doctype value of the worksheets. Its not in use. Holds NULL value.
displayindex int The displayindex column stores the display index with respect to worksheets
linkedids text The linkedids columns store the linked ids of the worksheet.
default_name varchar The default_name column stores the default name of the worksheets.
