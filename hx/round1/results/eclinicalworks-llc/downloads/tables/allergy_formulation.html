EHI Export Schema
Table allergy_formulation
Description This table stores information about allergy formulations, including their expiration date, review status, and associated appointment details.
Column DataType Description
id (Primary Key) int The column stores an id of the formulation of a patient. This is an auto incremental primary key of This column .
ENCid int This column stores id of an enc, for which formulation is created. It refers to column encounterid from enc table.
REVIEWED int This column stores value 0 / 1 indicating reviewed flag for formulation. 1 - Reviewed, 0 - Not Reviewed
RV_USER varchar This column stores name of user who has reviewed this formulation. If not reviewed then it will be NULL.
RV_TIME datetime This column stores time when user has reviewed this formulation. If not reviewed then it will be NULL.
FORM_EXP_DATE datetime This column stores formulation expiration date when this formulation get expire. It will be NULL if not documented.
APPT_FLAG int This column stores value 0/1/NULL for Initial appointment needed flag for this formulation. 1 - Initial Appointment Needed, 0/NULL - Not Needed.
RV_USERid int This column stores name of user id who has reviewed this formulation. If not reviewed then it will be NULL.
QUEid int This column stores linked queid for this formulation. It refers to column id from antigen_recipe_queue table.
VIAL varchar This column stores vial name associated with this formulation.
FORM_DATE datetime This column stores formulation date of the formulation. The format used is 1900-01-01 HH:MM:SS.
