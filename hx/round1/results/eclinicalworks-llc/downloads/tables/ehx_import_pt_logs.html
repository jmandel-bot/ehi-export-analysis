EHI Export Schema
Table ehx_import_pt_logs
Description This table stores logs related to the import of patient data from eEHX portal.
Column DataType Description
id int dead column not used from application, and application is not populating this column value.
pid int This column stores the unique identifier for the patient which data has been imported from eEHX
ecwmpi varchar This column stores the patient's ecwmpi id which is linked with iHX portal patient. Example values: '0d7d2e952db67305:3671a634:17217a82f90:-7ffe'
field varchar This column stores the name of the field in the patient data that has been imported from ehx. Example values: 'firstName', 'ssn', 'language'.
fieldData varchar This column stores the actual data of the field that has been imported from ehx. Example values: 'John', '8569854858', 'English'.
logTime datetime This column stores the time stamp of when the log entry was created. The default value is the current time stamp. Example values: '2022-01-01 12:34:56', '2022-01-02 14:30:00', '2022-01-03 09:15:30'.
