EHI Export Schema
Table edi_dental
Description This table stores information about dental claims, accidents, and treatments related to EDI (Electronic Data Interchange) in the dental industry.
Column DataType Description
Invid (Primary Key) int Unique identifier for the invoice related to the dental claim. This is a foreign key referring to the `edi_invoice` table's `id` column. Example: 12345
PreTreatementEstimate tinyint A boolean flag indicating if a pre-treatment estimate is required (1) or not (0). Example: 1
StmtOfActualService tinyint A boolean flag indicating if a statement of actual service is required (1) or not (0). Example: 0
MedicaidClaim tinyint A boolean flag indicating if the claim is a Medicaid claim (1) or not (0). Example: 1
EPSDT tinyint A boolean flag indicating if the claim is related to Early and Periodic Screening, Diagnostic, and Treatment (EPSDT) services (1) or not (0). Example: 0
PriorAuthNo varchar A string representing the prior authorization number for the claim. Example: 'PA123456'
PlaceOfTreatement tinyint A numeric code representing the place of treatment. Example: 1 (for office)
IsRadiograph tinyint A boolean flag indicating if a radiograph is included in the claim (1) or not (0). Example: 1
Radiograph_Many varchar A string representing the number of radiographs included in the claim. Example: '2'
IsOccupationalInjury tinyint A boolean flag indicating if the claim is related to an occupational injury (1) or not (0). Example: 0
OccupationalInjury_Desc varchar A string describing the occupational injury, if applicable. Example: 'Slip and fall at work'
IsAutoAccident tinyint A boolean flag indicating if the claim is related to an auto accident (1) or not (0). Example: 1
AutoAccident_Desc varchar A string describing the auto accident, if applicable. Example: 'Car collision with another vehicle'
IsOtherAccident tinyint A boolean flag indicating if the claim is related to another type of accident (1) or not (0). Example: 0
OtherAccident_Desc varchar A string describing the other accident, if applicable. Example: 'Bicycle accident'
IsProsthesis tinyint A boolean flag indicating if the claim involves a prosthesis (1) or not (0). Example: 1
Prosthesis_Reson varchar A string describing the reason for the prosthesis. Example: 'Missing tooth due to accident'
Prosthesis_Date varchar A string representing the date of the prosthesis placement. Example: '2021-01-01'
IsOrthDontics tinyint A boolean flag indicating if the claim involves orthodontic treatment (1) or not (0). Example: 0
OrthDontics_Desc varchar A string describing the orthodontic treatment, if applicable. Example: 'Braces for teeth alignment'
OrthDontics_Date varchar A string representing the date of the orthodontic treatment. Example: '2021-06-01'
OrthDontics_TrtmtRemaining varchar A string representing the remaining treatment duration for the orthodontic treatment. Example: '12 months'
RemarksForUnusualServices text A text field containing remarks for any unusual services provided. Example: 'Emergency extraction due to severe pain'
TeethInfo varchar A string representing information about the teeth involved in the claim. Example: 'Upper left canine, lower right molar'
