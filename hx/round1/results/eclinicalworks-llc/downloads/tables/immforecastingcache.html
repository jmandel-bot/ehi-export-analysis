EHI Export Schema
Table immforecastingcache
Description This table stores response received from Immunization Forecasting server.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key in this column.
pid int This column stores id of a patient whose immunization forecasting is being stored in This column . This column refers to UID column in user table.
immseriescvxid int This column stores the immunization series id of each series for which forecasting is shown. It refers to immseriescvx.immseriesid column.
forecastingcode varchar This column stores values 1/2/3/4 based on whether the recommendation for that series is due now, due in future, complete or not recommended. The recommendation information is sent from Immunization forecasting server. 1 - Due Now, 2 - Due In Future, 3 - Complete, 4 - Not Recommended.
forecastingdate datetime This column stores date when the immunization is recommended. The forecasting date information is sent from Immunization forecasting server.
recommendation varchar This column stores recommendation received from Immunization Forecasting server for patient's future immunizations.
interpretation varchar This column stores interpretation of recommendation mentioned under recommendation column for a respective record.
deleteflag tinyint This column stored values 0/1 to indicate whether the record has been marked as deleted or not. 0 - Record is NOT marked as deleted, 1 - Record is marked as deleted.
lastupdatedon datetime This column stores date and time the forecasting screen was loaded for the patient, thus the last time This column was modified for a patient.
lastshotgivendate datetime This column stores the given date of the last immunization ordered for the patient.
