Table vision_order_contactlensacc
Description This table stores contact lens and accessory data.
Column DataType Description
id (Primary Key) int Unique identifier for each contact lens accessory order record. (e.g., 1, 2, 3)
order_id int This column stores the reference order id from the vision_order table.
product_id int identifier for the product associated with this contact lens accessory (e.g., 1001, 1002, 1003).
unitprice decimal Unit price of the contact lens accessory (e.g., 10.0000, 15.0000, 20.0000).
noofunits int Number of units of the contact lens accessory in the order (e.g., 1, 2, 3).
promotionamount decimal Amount of promotion applied to the contact lens accessory (e.g., 0.0000, 5.0000, 10.0000).
taxratio_code_id int This column is not in use.
taxamount decimal Amount of tax applied to the contact lens accessory (e.g., 0.5000, 1.0000, 1.5000).
totalprice decimal Total price of the contact lens accessory including unit price, promotion, and tax. (e.g., 10.5000, 20.0000, 30.0000)
producttype int This column indicates which product type it is. A "1" means contact, and a "2" accessory.
upcsku varchar This column stores the stock keeping unit from the sku column of inventory_product table.
UnitQty int This columns default value is 1 always indicating unit of the quantity for this accessory.
productname varchar This column stores the name of the product. This column refers to the column name from the inventory_product table.
isdeleted tinyint This column stores a value of 0 or 1. A "1" means when this contact and accessory is deleted, and a "0" means otherwise.
createdby int This column stores the logged in user id who creates this contact and accessory. This column refers to the column UID from the user table.
createdon datetime This column stores the time stamp when this contact and accessory is created. The format used is YYYY-MM-DD HH:MM:SS.
updatedby int This column stores the logged in user id who modifies this contact and accessory. This column refers to the column UID from user table.
updatedon datetime This column stores the time stamp when this contact and accessory is modified. The format used is: YYYY-MM-DD HH:MM:SS.
discount_value decimal This column is not in use.
discount_amount decimal This column is not in use.
discount_type tinyint This column is not in use.
isreturn tinyint This column is not in use.
processorflag tinyint This column indicates where the spectacles are bought from. This column refers column id from inventory_vendorcredentials table.
contactLensid int This column indicates the exact product for the contact lens. This column refers to the column id from the vision_contactlensrx table.
employerpart decimal This column is not in use.
isreturn_sl tinyint This column is not in use.
fac_location tinyint This column indicates the facility location from this item. This column refers to the column id from the inventory_sku table.
