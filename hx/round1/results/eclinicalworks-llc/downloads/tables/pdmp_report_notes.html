EHI Export Schema
Table pdmp_report_notes
Description This table stores the notes related to PDMP (Prescription Drug Monitoring Program) reports for patients during their medical encounters.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
patientid int Unique identifier for the patient associated with the PDMP report note. This is a foreign key that refers to the `uid` column in the `users` table. Example: 101, 102, 103.
encounterid int Unique identifier for the medical encounter associated with the PDMP report note. This is a foreign key that refers to the `encounterID` column in the `enc` table. Example: 1001, 1002, 1003.
notes varchar Text notes related to the PDMP report for the patient during the medical encounter. Example: "Patient prescribed opioid medication.", "Patient has a history of prescription drug abuse."
addedby int Unique identifier for the user who added the PDMP report note. This is a foreign key that refers to the `uid` column in the `users` table. Example: 201, 202, 203.
addedon datetime Date and time when the PDMP report note was added. Example: '2022-01-01 12:34:56', '2022-01-02 14:30:00'.
modifiedby int Unique identifier for the user who last modified the PDMP report note. This is a foreign key that refers to the `uid` column in the `users` table. Example: 201, 202, 203.
modifiedon datetime Date and time when the PDMP report note was last modified. Example: '2022-01-03 15:45:30', '2022-01-04 10:20:00'.
delflag tinyint A flag indicating whether the record has been deleted (1) or not (0). Example: 0, 1.
