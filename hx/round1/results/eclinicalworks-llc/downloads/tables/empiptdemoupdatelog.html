EHI Export Schema
Table empiptdemoupdatelog
Description This table contains data about eEHX enabled patient demographics and picture updated on the EMR.
Column DataType Description
id (Primary Key) int This is the primary key column, which contains a unique integer value for each record in the table. Example: 1, 2, 3, etc.
pid int This column contains patient id. It refers to user.UID column.
LastModifiedDate datetime This column value updated when patient's EMR demographics gets updated.
picUpdatedFlag tinyint This column contains flag 0/1 when patient's picture gets upgraded. 0 - picture not updated 1 - picture updated
picUpdatedDate datetime This column contains date and time when patient picture was updated.
picExportedDate datetime This column contains date and time when patient's picture was exported to eEHX.
