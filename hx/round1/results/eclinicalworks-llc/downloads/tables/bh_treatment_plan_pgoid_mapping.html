Table bh_treatment_plan_pgoid_mapping
Description This table stores the mapping of treatment plans to their respective categories, structures, and other related information.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
tpGroupProblemid int identifier for the treatment plan group problem. Foreign key relation to `bh_treatment_plan_group_problem_mapping` table's `id` column. Example: 101, 102, 103.
categoryid int identifier for the category of the treatment plan. Foreign key relation to `items` table's `itemid` column. Example: 201, 202, 203.
structid int identifier for the structure of the treatment plan. Foreign key relation to `bh_treatment_plan_dictionary` table's `id` column. Example: 301, 302, 303.
NAME varchar Name of the treatment plan. Example: "Plan A", "Plan B", "Plan C".
goalStatus tinyint Status of the goal, represented as a tiny integer. 0 for not achieved, 1 for achieved. Example: 0, 1.
startDate datetime Start date of the treatment plan. Example: '2021-01-01', '2021-02-01', '2021-03-01'.
targetDate datetime Target date for the treatment plan. Example: '2021-12-31', '2022-01-31', '2022-02-28'.
assignedTo int identifier for the user assigned to the treatment plan. Foreign key relation to `users` table's `UID` column. Example: 401, 402, 403.
priority smallint Priority of the treatment plan, represented as a small integer. Example: 0, 1, 2.
outcomeid int identifier for the outcome of the treatment plan. Foreign key relation to `bh_outcome` table's `id` column. Example: 501, 502, 503.
deleteFlag tinyint Flag to indicate if the record is deleted, represented as a tiny integer. 0 for not deleted, 1 for deleted. Example: 0, 1.
