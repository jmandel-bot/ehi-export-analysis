EHI Export Schema
Table careplan_documentreview
Description This table stores information about document reviews related to specific care plans.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer. It uniquely identifies each document review record. Example: 1, 2, 3, etc.
CarePlanid int This is a foreign key that refers to the `id` column in the `careplan` table. It represents the care plan associated with the document review. Example: 1, 2, 3, etc.
Document text This is a text column that stores the content of the document being reviewed. Example: "This is a sample document for review."
timestamp datetime This is a datetime column that stores the date and time when the document review was created or updated. Example: '2021-09-15 14:30:00'
