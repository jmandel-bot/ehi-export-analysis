Table vision_specrxdruginfo
Description This table stores information about specific prescription drugs used in vision treatments, including the drug batch number, usage, and the date and time the drug information was recorded.
Column DataType Description
id (Primary Key) int This is the primary key, an auto-incrementing integer that uniquely identifies each record in the table. Example: 1, 2, 3, etc.
visionrxid int This is a foreign key, an integer that refers to the `id` column in visionrx table. It establishes a relationship between the prescription drug information and the specific vision prescription. Example: 101, 102, 103, etc.
batch varchar This is a varchar(50) column that stores the batch number of the drug used in the vision treatment. Example: 'B12345', 'B67890', etc.
drugused varchar This is a varchar(100) column that stores the name of the drug used in the vision treatment. Example: 'DrugA', 'DrugB', etc.
druginfodate varchar This is a varchar(15) column that stores the date when the drug information was recorded, likely in the format 'YYYY-MM-DD'. Example: '2021-09-01', '2021-09-02', etc.
druginfotime varchar This is a varchar(10) column that stores the time when the drug information was recorded, likely in the format 'HH:mm:ss'. Example: '14:30:00', '15:45:00', etc.
