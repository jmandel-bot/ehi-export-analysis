EHI Export Schema
Table phm_cohort
Description This table stores information about different cohorts in a population health management system.
Column DataType Description
id (Primary Key) int Unique identifier for each cohort (integer). Example: 1, 2, 3, etc.
description varchar A brief description of the cohort (varchar). Example: "Diabetic patients", "Elderly patients", etc.
delflag tinyint A flag to indicate if the cohort has been deleted (tinyint). 0 means not deleted, 1 means deleted. Default value is 0.
addedBy int The user ID of the person who added the cohort (integer). This is a foreign key that refers to the `uid` column in the `users` table. Example: 101, 102, 103, etc.
addedDtTime datetime The date and time when the cohort was added to the system (datetime). Example: "2021-09-01 12:34:56".
lastupdatedBy int The user ID of the person who last updated the cohort (integer). This is a foreign key that refers to the `uid` column in the `users` table. Example: 101, 102, 103, etc. This field can be NULL if the cohort has never been updated.
lastupdatedDTTime datetime The date and time when the cohort was last updated (datetime). Example: "2021-09-01 12:34:56". This field can be NULL if the cohort has never been updated.
