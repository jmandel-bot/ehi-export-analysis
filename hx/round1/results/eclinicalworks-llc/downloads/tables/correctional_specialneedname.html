Table correctional_specialneedname
Description This table stores patients' special needs.
Column DataType Description
id (Primary Key) int This column store measure_reportdata_49022 tables id value. This is one kind of foreign-key which will contain multi records for measure_reportdata_49022 table.
needtypeid int This column store patient id for respective measures.
needname varchar This column store encounter id for respective measures.
delflag tinyint This column store encounter date for respective measures.
neednamemanualindex int This column store measure signature for respective measures. With use of this column we can decide this patient record exist in IPP, Deniminator, Exclusion, Numerator, Exception, Data Missing values. x>=1 : IPP, x>=2 : Deniminator, x==3 : Exclusion, x==4 : Numerator, x==5 : Exception, x==6 : Data Missing.
