EHI Export Schema
Table advance_directives
Description This table stores information about advance directives, which are legal documents that specify a person's preferences for medical treatment in case they are unable to make decisions for themselves.
Column DataType Description
id (Primary Key) int This is the primary key column, which uniquely identifies each record in the table. It is an integer value that is automatically incremented for each new record. Example: 1, 2, 3, etc.
Code varchar This column stores a unique code associated with each advance directive. It is a variable character field with a maximum length of 25 characters. Example: AD001, AD002, AD003, etc.
Name varchar This column stores the name or description of the advance directive. It is a variable character field with a maximum length of 255 characters. Example: Do Not Resuscitate, Living Will, Healthcare Power of Attorney, etc.
