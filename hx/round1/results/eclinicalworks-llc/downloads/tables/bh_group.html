Table bh_group
Description This table stores information about various groups.
Column DataType Description
groupid (Primary Key) int This is the primary key and a unique identifier for each group. It is an integer value that is automatically incremented. Example: 1, 2, 3, etc.
groupName varchar This is a varchar column that stores the name of the group. It is a required field and cannot be NULL. Example: "Group A", "Group B", "Group C", etc.
inactive tinyint This is a tinyint column that indicates whether the group is inactive or not. It has a default value of '0', where '0' means active and '1' means inactive. Example: 0 (active), 1 (inactive).
deleteFlag tinyint This is a tinyint column that indicates whether the group has been deleted or not. It has a default value of '0', where '0' means not deleted and '1' means deleted. Example: 0 (not deleted), 1 (deleted).
downloadFlag tinyint This is a tinyint column that indicates whether the group has been downloaded or not. It has a default value of '0', where '0' means not downloaded and '1' means downloaded. Example: 0 (not downloaded), 1 (downloaded).
