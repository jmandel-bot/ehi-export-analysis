Table allergy_manufacturers
Description This table stores information about allergy manufacturers, including their names and whether they are inactive or deleted.
Column DataType Description
id (Primary Key) int This is the primary key for the table, an auto-incrementing integer that uniquely identifies each allergy manufacturer. Example: 1, 2, 3, etc.
NAME varchar This is a varchar(50) column that stores the name of the allergy manufacturer. It cannot be NULL and has a default value of an empty string. Example: 'AllergyCorp', 'AllergenTech', 'Allergy Solutions'.
inactiveflag tinyint This is a tinyint(3) unsigned column that indicates whether the allergy manufacturer is inactive (1) or active (0). It cannot be NULL and has a default value of '0'. Example: 0 (active), 1 (inactive).
deleteflag tinyint This is a tinyint(3) unsigned column that indicates whether the allergy manufacturer has been deleted (1) or not (0). It cannot be NULL and has a default value of '0'. Example: 0 (not deleted), 1 (deleted).
