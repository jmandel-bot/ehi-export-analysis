Table paymentplanistallmentpaiddetail
Description This table stores the details of payment plan installment transactions, including the amount paid and the date it was received.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3, etc.
paymentid int identifier for the payment associated with the installment. It can be a foreign key referring to the `id` column in the `payment` table. Example: 101, 102, 103, etc.
claimid int identifier for the claim associated with the payment. It can be a foreign key referring to the `id` column in the `claim` table. Example: 1001, 1002, 1003, etc.
planid int identifier for the payment plan associated with the installment. It can be a foreign key referring to the `id` column in the `paymentplan` table. Example: 501, 502, 503, etc.
paidamount decimal Amount paid for the installment. Example: 100.00, 150.50, 200.00, etc.
deleteflag tinyint A boolean flag indicating if the record has been deleted (1) or not (0). Example: 0, 1.
installmentNo int The installment number for the payment plan. Example: 1, 2, 3, etc.
recievedDate datetime Date and time when the payment was received. Example: '2022-01-01 12:30:00', '2022-01-02 15:45:00', etc.
