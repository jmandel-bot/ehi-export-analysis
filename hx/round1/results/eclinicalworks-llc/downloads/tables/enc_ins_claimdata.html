EHI Export Schema
Table enc_ins_claimdata
Description This table stores information about insurance claims related to specific encounters.
Column DataType Description
Id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
encId int This column stores encounterId which the Claim Data is documented for. It refers to column encounterId from enc table.
insId int This column stores insuranceId against which the Claim Data is documented for. It refers to column insId from insurance table.
displayIndex int This column stores sequence number of the added Insurance to be displayed in the Claim Data for the Current Visit screen in appointment.
insurancedetid int This column stores insurance detail id. It refers to column Id from insurancedetail table.
