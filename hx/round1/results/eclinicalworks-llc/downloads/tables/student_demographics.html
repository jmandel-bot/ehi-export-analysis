Table student_demographics
Description This table stores demographic information about students, including their program, college, major, dormitory, address, phone number, registration status, and other related details.
Column DataType Description
Id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
PatientId int Unique identifier for the patient associated with the student. Example: 101, 102, 103.
StudentId varchar Unique identifier for the student. Example: "S12345", "S12346", "S12347".
ProgramCode int Numeric code representing the program the student is enrolled in. Example: 1001, 1002, 1003.
ProgramType varchar Description of the program type the student is enrolled in. Example: "Undergraduate", "Graduate", "PhD".
CollegeName varchar Name of the college the student is attending. Example: "College of Arts and Sciences", "College of Engineering".
CurrYrDivMajor varchar Current year division and major code. Example: "FRCS", "SOEN", "JRBI".
MajorName varchar Name of the student's major. Example: "Computer Science", "Biology", "Mechanical Engineering".
DormName varchar Name of the dormitory the student resides in. Example: "Maple Hall", "Oak Hall", "Pine Hall".
DormAddressLine1 varchar First line of the dormitory address. Example: "123 University Drive".
DormAddressLine2 varchar Second line of the dormitory address (if applicable). Example: "Apt 4B", "Suite 200".
DormCity varchar City where the dormitory is located. Example: "New York", "Los Angeles", "Chicago".
DormState char State where the dormitory is located. Example: "NY", "CA", "IL".
DormZip varchar Zip code of the dormitory address. Example: "10001", "90001", "60601".
Phone varchar Student's phone number. Example: "555-123-4567".
WithdrawalInd char Indicator of whether the student has withdrawn from the college. Example: "Y" (Yes), "N" (No).
WithdrawalCode char Code representing the reason for withdrawal. Example: "MED", "FIN", "PER".
WithdrawalDesc varchar Description of the reason for withdrawal. Example: "Medical", "Financial", "Personal".
RegisteredForFallSem char Indicator of whether the student is registered for the fall semester. Example: "Y" (Yes), "N" (No).
RegisteredForSummerSem char Indicator of whether the student is registered for the summer semester. Example: "Y" (Yes), "N" (No).
RegisteredForSpringSem char Indicator of whether the student is registered for the spring semester. Example: "Y" (Yes), "N" (No).
RegisteredForWinterSem char Indicator of whether the student is registered for the winter semester. Example: "Y" (Yes), "N" (No).
BHSBlockInd char Indicator of whether the student has a block on their account. Example: "Y" (Yes), "N" (No).
BRSEligibile char Indicator of whether the student is eligible for BRS (a financial aid program). Example: "Y" (Yes), "N" (No).
BRSWaiveInd char Indicator of whether the student has waived their BRS eligibility. Example: "Y" (Yes), "N" (No).
EthnicCode varchar Code representing the student's ethnicity. Example: "HISP", "ASIAN", "WHITE".
EthnicCodeDesc varchar Description of the student's ethnicity. Example: "Hispanic", "Asian", "White".
AthleteInd char Indicator of whether the student is an athlete. Example: "Y" (Yes), "N" (No).
AthleteStatus char Status of the student athlete. Example: "A" (Active), "I" (Inactive), "R" (Retired).
LocalAddressLine1 varchar First line of the student's local address. Example: "456 Main Street".
LocalAddressLine2 varchar Second line of the student's local address (if applicable). Example: "Apt 3A", "Suite 100".
LocalCity varchar City where the student's local address is located. Example: "New York", "Los Angeles", "Chicago".
LocalState char State where the student's local address is located. Example: "NY", "CA", "IL".
LocalZip varchar Zip code of the student's local address. Example: "10001", "90001", "60601".
Nation varchar Country of the student's nationality. Example: "United States", "Canada", "India".
PermAddrAddInfo varchar Additional information about the student's permanent address. Example: "Care of John Smith", "PO Box 123".
