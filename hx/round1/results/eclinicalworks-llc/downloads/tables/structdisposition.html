EHI Export Schema
Table structdisposition
Description This table stores a patient's disposition for an encounter.
Column DataType Description
encounterId int This column stores the encounter id of the patient which the Disposition and Communication is associated to. This column refers to column encounterid from enc table.
catId int This column is unnecessarily added just because common API is being used. This will always store 0.
itemId int This column is unnecessarily added just because common API is being used. This will always store 0.
detailId int This column stores the id of the name of the Disposition. This column refers to column id of structdatadetail table.
value text This column stores the structured data question which refers to the column 'name' of structdataoptions table. This column also stores numeric data selected in the user interfacewhich has no reference to any other column.
valueId varchar This column stores the id of the structured question selected. This column refers column optionId from structdataoptions table.
notes text This column stores the note added by the end user for the respective structured question.
chartdisplay tinyint This column stores 0 / 1 , 0 : not to display on chart and 1 : display on chart.
displayIndex tinyint This column stores the number in order the structured data question are being displayed on user interface.
