Table allergy_extract_billing
Description This table stores information related to billing for allergy extracts, including patient details, insurance information, and billing status.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table (e.g., 1, 2, 3).
PATIENTid int Unique identifier for the patient associated with the billing record. Foreign key referring to the `id` column in the `patients` table.
QUEUEid int Unique identifier for the queue associated with the billing record.
ENCid int Unique identifier for the encounter associated with the billing record.
LABid int Unique identifier for the lab associated with the billing record.
MIXFACid int Unique identifier for the mixing facility associated with the billing record, if applicable.
MIXFACNAME varchar Name of the mixing facility associated with the billing record, if applicable (e.g., "Mix Facility A").
INSid int Unique identifier for the insurance associated with the billing record, if applicable.
INSNAME varchar Name of the insurance associated with the billing record, if applicable (e.g., "Insurance Company A").
TASKDATE datetime Date when the billing task was created (e.g., "2021-09-01").
STATUS tinyint Status of the billing record as a tiny integer (e.g., 0 for pending, 1 for completed).
BILLTYPE tinyint Type of billing as a tiny integer (e.g., 0 for standard, 1 for custom).
ASSIGNTO int Unique identifier for the user assigned to the billing task, if applicable. Foreign key referring to the `id` column in the `users` table.
ASSIGNTO_NAME varchar Name of the user assigned to the billing task, if applicable (e.g., "John Doe").
UNITBILLED int Number of units billed for the allergy extract (e.g., 5).
TOTALUNITS int Total number of units for the allergy extract (e.g., 10).
TOTALVIALS int Total number of vials for the allergy extract (e.g., 2).
CPTCODEITEMid int Unique identifier for the CPT code item associated with the billing record, if applicable.
CPTCODE varchar CPT code associated with the billing record, if applicable (e.g., "J3301").
BILLPROVidERid int Unique identifier for the billing provider associated with the billing record, if applicable.
BILLPROVidERNAME varchar Name of the billing provider associated with the billing record, if applicable (e.g., "Provider A").
FREQ int Frequency of the allergy extract administration (e.g., 3).
FREQ_UNIT tinyint Unit of frequency for the allergy extract administration as a tiny integer (e.g., 0 for days, 1 for weeks).
REC int Recommended number of administrations for the allergy extract (e.g., 6).
REC_UNIT tinyint Unit of recommendation for the allergy extract administration as a tiny integer (e.g., 0 for days, 1 for weeks).
CLAIMid int Unique identifier for the claim associated with the billing record, if applicable.
CLAIM_USER_id int Unique identifier for the user who created the claim, if applicable. Foreign key referring to the `id` column in the `users` table.
CLAIM_DATE datetime Date when the claim was created, if applicable (e.g., "2021-09-15 14:30:00").
TASK_COMP_DATE datetime Date when the billing task was completed, if applicable (e.g., "2021-09-20 16:45:00").
TASK_COMP_BY varchar Name of the user who completed the billing task, if applicable (e.g., "Jane Smith").
EXPECTED_STARTDATE datetime Expected start date for the allergy extract administration, if applicable (e.g., "2021-10-01 09:00:00").
EXTRACT_COMPL_DATE datetime Date when the allergy extract was completed, if applicable (e.g., "2021-11-01 17:00:00").
ISNONBILLABLE tinyint Flag indicating if the allergy extract is non-billable as a tiny integer (e.g., 0 for billable, 1 for non-billable).
FIRSTSHOTDATE datetime Date of the first shot for the allergy extract, if applicable (e.g., "2021-10-01").
