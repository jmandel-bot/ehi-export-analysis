EHI Export Schema
Table visitstscodes
Description This table stores information about the visit status codes, which define the status of the patients' visits.
Column DataType Description
codeId (Primary Key) int The column stores an id of the Visit Status Code. This is an auto incremented primary key of This column .
code varchar This column stores free text short name for visit status.
status varchar This column stores free text description for the visit status.
color varchar This column stores a color for the visit status code. This color is the outline color for the appointment.
NonBillable tinyint This column stores values 0/1 to mark appointments with this visit status as billable or Non Billable. 0 - Non Billable , 1 - Billable.
TriggerManFieldsCheck tinyint This column stores values 0/1 for the system to perform a validation check for patient’s demographic information, ensure it contains all mandatory fields. 0 - Non mandatory , 1 - Mandatory.
MUExclude tinyint This column stores value 0/1 to exclude any encounters in this status from Meaningful Use reporting. 0 - Not Exclude, 1 - Exclude.
triggermanappfieldscheck smallint This column stores values 0/1 for the system to perform a validation check on the patient’s appointment information to ensure it contains all mandatory fields. 0 - Non mandatory , 1 - Mandatory.
ExcludeRefFieldsCheck tinyint This column stores value 0/1 to exclude any encounters in this status from Referral visits used. 0 - Not Exclude, 1 - Exclude.
UsageRestriction tinyint This column stores integer value indicates Enable Usage Restriction. 0 Indicates Disable Usage Restriction. 1 indicates Enable Usage Restriction. Value can change only EnableUsageRestrictionForVisitStatusCode itemkey enable
CodesAfterStartTime tinyint This column stores integer value indicates Visit Status allowed only after Appointment Start Time. 1 value indicates Visit Status allowed only after Appointment Start Time. 0 value indicates feature is disable. Value can change only EnableUsageRestrictionForVisitStatusCode itemkey enable
CodesDurationAfterStartTime tinyint This column stores integer value indicates Visit Status allowed before how many hours of Appointment Start Time. Value can change only EnableUsageRestrictionForVisitStatusCode itemkey enable
includeInRotation tinyint This column stores integer value indicates Include Into Rotation entry. 0 indicates not Include Into Rotation entry 1 indicates Include Into Rotation entry This value changes only enableRotationFeature itemkey enable
ReasonRequired tinyint This column stores integer value indicates reason require or not. 0 indicates reason not require 1 indicates reason require This value can change only EnableCorrectionalFeatures itemkey enable.
excludeAsapList tinyint This column stores integer value indicates Exclude from ASAP List. 0 indicates not exclude from ASAP List 1 indicates exclude from ASAP List This value can change only EnableASAPList itemkey enable
