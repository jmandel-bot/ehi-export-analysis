EHI Export Schema
Table apptblocks
Description This table stores appointment blocks, which are periods of time reserved for a specific provider.
Column DataType Description
id (Primary Key) int Unique identifier for each appointment block. (e.g., 1, 2, 3)
Recurrence int Integer value representing the recurrence pattern of the appointment block. (e.g., 0 for no recurrence, 1 for daily, 2 for weekly)
StartDate datetime The date when the appointment block starts. (e.g., '2021-09-01')
StartTime datetime The time when the appointment block starts. (e.g., '09:00:00')
EndDate datetime The date when the appointment block ends. (e.g., '2021-09-01')
EndTime datetime The time when the appointment block ends. (e.g., '10:00:00')
AllDay int Integer value indicating if the appointment block is an all-day event. (e.g., 0 for false, 1 for true)
Description varchar A short description of the appointment block. (e.g., 'Team Meeting')
Notes text Additional notes or details about the appointment block. (e.g., 'Discuss project updates')
BlockColor int Integer value representing the color code for the appointment block. (e.g., 1 for red, 2 for blue)
blocktype tinyint Tiny integer value representing the type of the appointment block. (e.g., 0 for unavailable, 1 for meeting)
facilityid int Integer value representing the identifier of the facility where the appointment block takes place. This can be a foreign key referring to the `id` column in the `facilities` table. (e.g., 1, 2, 3)
modifiedTime datetime time stamp indicating the last time the appointment block was modified. (e.g., '2021-08-31 15:30:00')
reason text Text describing the reason for the appointment block. (e.g., 'Scheduled maintenance')
