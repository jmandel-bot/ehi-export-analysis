Table bh_treatment_plan_dictionary
Description This table stores information about treatment plan dictionaries, including their category, item, name, and status (inactive or active).
Column DataType Description
id (Primary Key) int Unique identifier for each mapping record. Example: 1, 2, 3, etc.
catid int identifier for the category of the treatment plan dictionary. Foreign key relation: `items.itemid`. Example: 101, 102, 103.
itemid int identifier for the item in the treatment plan dictionary. Foreign key relation: `items.itemid`. Example: 1001, 1002, 1003. This column can have NULL values.
name varchar The name of the treatment plan dictionary entry. Example: "Physical Therapy", "Occupational Therapy", "Speech Therapy".
inactive tinyint A flag indicating whether the treatment plan dictionary entry is inactive (1) or active (0). Example: 0 for active, 1 for inactive.
deleteFlag tinyint A flag indicating whether the treatment plan dictionary entry is deleted (1) or not deleted (0). Example: 0 for not deleted, 1 for deleted.
displayIndex int An integer representing the display order of the treatment plan dictionary entry. Example: 1, 2, 3.
