Table paymentplandetail
Description This table stores information about payment plans for patients, including details about the plan, balances, and payment terms.
Column DataType Description
planid (Primary Key) int Unique identifier for the payment plan (e.g., 1, 2, 3).
planname varchar Name of the payment plan (e.g., "Standard Plan", "Premium Plan").
createddatetime datetime Date and time when the payment plan was created (e.g., '2021-08-01 12:34:56').
createdby int Unique identifier of the user who created the payment plan (e.g., 1, 2, 3).
patientid int Unique identifier of the patient associated with the payment plan (e.g., 1, 2, 3).
plantypeid int Unique identifier of the payment plan type (e.g., 1, 2, 3).
planstatusid int Unique identifier of the payment plan status (e.g., 1 for active, 2 for inactive, 3 for completed).
currentbalance decimal Current balance of the payment plan (e.g., 100.00, 200.50).
balduefromptclaims decimal Balance due from patient claims (e.g., 50.00, 75.25).
balduefromestimates decimal Balance due from cost estimates (e.g., 25.00, 40.50).
balduefromother decimal Balance due from other sources (e.g., 10.00, 15.75).
prepaymentdeposit decimal Prepayment deposit amount (e.g., 100.00, 200.00).
initialDueAmt decimal Initial due amount for the payment plan (e.g., 500.00, 1000.00).
firstpaymentdate datetime Date of the first payment (e.g., '2021-08-01').
term varchar Term of the payment plan (e.g., "12 months", "24 months").
termmonths int Number of months in the payment plan term (e.g., 12, 24).
monthlypayment decimal Monthly payment amount (e.g., 50.00, 75.00).
monthlypaymentrounded tinyint Flag indicating if the monthly payment is rounded (e.g., 0 for false, 1 for true).
OutstandingPtClaimsFlag tinyint Flag indicating if there are outstanding patient claims (e.g., 0 for false, 1 for true).
patientcostestimateflag tinyint Flag indicating if there is a patient cost estimate (e.g., 0 for false, 1 for true).
otherAmtFlag tinyint Flag indicating if there is another amount (e.g., 0 for false, 1 for true).
otherAmt decimal Other amount value (e.g., 10.00, 20.00).
notes text Notes related to the payment plan.
deleteflag tinyint Flag indicating if the payment plan is deleted (e.g., 0 for false, 1 for true).
recPmt decimal Received payment amount (e.g., 50.00, 100.00).
recieveDate datetime Date when the payment was received (e.g., '2021-08-01 12:34:56').
statusopenon varchar Status open on date (e.g., '2021-08-01').
surgerydate varchar Date of surgery (e.g., '2021-08-01').
estduedate varchar Estimated due date for the payment (e.g., '2021-08-01').
agreementOnFile tinyint Flag indicating if there is an agreement on file (e.g., 0 for false, 1 for true).
