EHI Export Schema
Table welchallyncardiotests
Description This table stores information about cardio tests performed using Welch Allyn devices.
Column DataType Description
Id (Primary Key) int This column is used for maintaining unique ids of the cardio tests reports
RecordedAt varchar This column is used to provide the details at which the test report was recorded and saved in the system
ModifiedAt varchar This column is used to provide the details at which the test report was modified and saved in the system
EncId int This column is used to store encounter id details associated with welc allyn cardio test report. This column refers to encounterid column of enc table.
PatientId int This column is used to store the patientid details on which test was conducted in the system. It refers to uid column of user table.
labId int This column is used to store the lab Id which is linked to the lab order and based on which the Welchallyn Order is configured.
labType int Identifier for the type of lab, stored as an unsigned integer. (e.g., 1, 2, 3)
labName varchar This column stores order name details or order specific details regarding reports/tests of modality (Spiro, holter , stress, ECG) specifically for Welchallyn Device
TestType tinyint Identifier for the type of cardio test, stored as a tiny integer. (e.g., 1, 2, 3)
TestGID varchar This column stores a random unique number generated by the system and appended to each of the report file name .
MDWFile varchar Unique file name for welch allyn test
Intepretation text Text containing the interpretation of the cardio test results. (e.g., 'Normal sinus rhythm', 'Atrial fibrillation', 'Tachycardia')
IntepretationStatus varchar This column stores the status of interpretation for the given report as 'confirmed Interpretation' or 'Unconfirmed Interpretation'
DataAndComments text This column is used to store the comments for the given report saved in the system.
DeleteFlag tinyint This column stores a soft delete data. 0 : report is not deleted 1 if the report is deleted
modifiedflag int Flag indicating whether the record has been modified, stored as an integer. (e.g., 0 for not modified, 1 for modified)
TestGuidFileFormat varchar This column stores the type of file format of the test like .car , or .pdf , or .tiff
ftpdirpath varchar Path to the directory where the cardio test file is stored on an FTP server, stored as a string. (e.g., '/path/to/cardio/test/files/')
