EHI Export Schema
Table transitionofcare_comments
Description This table stores comments related to the Transition of Care records.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an integer value that uniquely identifies each comment. It is an auto-incrementing value. Example: 1, 2, 3, etc.
tocid int This is a foreign key that refers to the `id` column in the `transitionofcare` table. It is an integer value that represents the specific transition of care record that the comment is associated with. Example: 5, 12, 23, etc.
comment text This is a text column that stores the actual comment related to the transition of care record. Example: "Patient needs additional support during the transition."
modifiedtimestamp datetime This is a time stamp column that stores the date and time when the comment was last modified. It is automatically set to the current time stamp when a new comment is added or an existing comment is updated. Example: '2022-01-01 12:34:56'
