Table antigen_recipe_diluent
Description This table stores information about diluents used in antigen recipes, including their type, concentration, lot number, expiration date, and manufacturer details.
Column DataType Description
id (Primary Key) int The column stores an id of the antigen_recipe_diluent. This is an auto incremental primary key of This column .
QUEid int The column stores an queid associated with the recipe. It refers to column id from antigen_recipe_queue table.
BOTTLEid int The column stores an bottleid associated with the recipe. It refers to column bottle_id from antigen_bottle table.
DILUENT varchar The column stores the diluent value associated with the recipe for diluent.
DILUENTTYPE varchar The column stores the diluent type value associated with the recipe.
CONCENTRATION varchar The column stores the concentration value associated with the recipe for diluent.
LOTNO varchar The column stores the lot number of the recipe for diluent.
EXPDATE datetime The column stores the expiration date of the recipe for diluent.
MANUFACTURERid int identifier for the manufacturer of the diluent. Foreign key relation to the `allergy_manufacturers` table, referencing the `id` column. Example: 1, 2, 3.
ISMANUFACTURERSAVED tinyint Indicates whether the manufacturer's information is saved (1) or not (0). Example: 0, 1.
