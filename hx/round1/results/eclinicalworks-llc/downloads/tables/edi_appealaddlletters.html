EHI Export Schema
Table edi_appealaddlletters
Description This table stores information about additional letters related to appeals in the EDI system.
Column DataType Description
id (Primary Key) int This column is auto-incremented primary key in This column . Unique id is assigned to each additional supporting documents uploaded to vendor.
appealid int This column stores identifier number of a appeal, for whom changes are made. It refers to column id from edi_appealclaims table.
documentid int This column stores identifier number of a document, which is attached to appeal. It refers to column docid from documents table.
fileStatus tinyint This column stores identifier number of a appeal status code, appeal is in. It refers to column id from appealstatuscodes table.
userid int identifier for the user who created or modified the additional letter record. This is a foreign key that refers to the `id` column in the `users` table. Example: 1, 2, 3, etc.
selectedDate datetime Date and time when the additional letter was selected or processed. The default value is NULL. Example: '2022-01-01 12:34:56', '2022-01-02 14:30:00', etc.
deleteFlag tinyint Flag indicating whether the record has been deleted or not. The default value is '0'. Example: 0 (not deleted), 1 (deleted).
