EHI Export Schema
Table ip_rh_ordersetfavorites
Description This table is designed to store user-specific favorite order sets. An order set is a collection of predefined orders that a healthcare provider can use in patient care.
Column DataType Description
Id (Primary Key) int Unique identifier for each favorite order set entry. This column is the primary key of the table and does not refer to any other table. This column is used as the primary key and is essential for indexing and uniquely identifying records.
orderSetId int Foreign key that links to the order set which is marked as favorite. This column refers to the orderSet table's id column, which contains the details of the order sets. It is crucial for establishing a relationship between the favorite order set and the actual order set details.
userId int Foreign key that identifies the user who has marked the order set as a favorite. This column refers to the user's table id column, which contains user information. This column is important for personalizing the application experience by linking order sets to specific users.
