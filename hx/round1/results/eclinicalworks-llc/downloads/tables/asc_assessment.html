EHI Export Schema
Table asc_assessment
Description This table stores information about assessments related to specific encounters and properties.
Column DataType Description
id (Primary Key) int Unique identifier for each assessment record. Example: 1, 2, 3, etc.
Encounterid int identifier for the encounter associated with the assessment. NOT NULL DEFAULT '0'. Example: 101, 102, 103, etc.
catid int identifier for the category of the assessment. NOT NULL DEFAULT '0'. Example: 1, 2, 3, etc.
Propid int identifier for the property associated with the assessment. NOT NULL DEFAULT '0'. Example: 10, 20, 30, etc.
PropDetailid int identifier for the property detail associated with the assessment. NOT NULL DEFAULT '0'. Example: 100, 200, 300, etc.
Selection int Integer value representing the selection made for the assessment.DEFAULT NULL. Example: 1, 2, 3, etc.
Notes text Text field to store any additional notes related to the assessment. Example: "Patient showed improvement", "Requires further evaluation", etc.
createDate datetime Datetime value representing the date and time when the assessment record was created. Example: '2021-09-01 12:34:56'
displayseq int Unsigned integer value representing the display sequence of the assessment.UNSIGNED DEFAULT NULL. Example: 1, 2, 3, etc.
notes2 text Text field to store any secondary notes related to the assessment. Example: "Follow up in 2 weeks", "Refer to specialist", etc.
displaycatseq int Integer value representing the display sequence of the assessment category.NOT NULL DEFAULT '0'. Example: 1, 2, 3, etc.
drawingnotes text Text field to store any notes related to drawings or diagrams associated with the assessment.NOT NULL. Example: "See attached diagram", "Refer to figure 1", etc.
