EHI Export Schema
Table segvisitrounds
Description This table stores information about visit rounds for patients, including the status of the patient, visit date, and any notes or addendums.
Column DataType Description
id (Primary Key) int Unique identifier for each visit round record. Example: 1, 2, 3.
pcid int This column store the patient's correctional id. This data is coming from patients_correctional.id column.
AddressedById int Foreign key referring to the users.uid of the person who addressed the visit. Example: 1001, 1002, 1003.
patientstatus varchar The status of the patient during the visit round. Example: "stable", "critical", "discharged".
finalstatus tinyint A tinyint value representing the final status of the visit round (0 for not final, 1 for final). Example: 0, 1.
segvisitdate datetime The date and time of the visit round. Example: '2021-09-01 10:30:00', '2021-09-02 14:45:00'.
notes text Text containing any notes or observations made during the visit round. Example: "Patient is recovering well", "Patient requires further tests".
deleteflag tinyint A tinyint value representing whether the visit round record has been deleted (0 for not deleted, 1 for deleted). Example: 0, 1.
patientstatusid int Foreign key referring to the ID of the patient status from items.itemid. Example: 2001, 2002, 2003.
visitCategory bigint Foreign key referring to the ID of the visit category from items.itemid. Example: 2001, 2002, 2003.
patientid int Foreign key referring to the ID of the patient from patients.patientid. Example: 2001, 2002, 2003.
addendumNotes text Text containing any additional notes or addendums made after the visit round. Example: "Additional tests ordered", "Follow-up appointment scheduled".
