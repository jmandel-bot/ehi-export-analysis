Table vision_cdratio
Description This table stores information about the cup-to-disc ratio (CD ratio) of the optic nerve head in a patient's eyes during an encounter. The CD ratio is a measure used to assess the risk of glaucoma.
Column DataType Description
id (Primary Key) int This is the primary key for the table, an automatically incremented integer that uniquely identifies each record in the table. Example: 1, 2, 3.
encounterid int This is a foreign key that refers to the `encounterid` column in the `enc` table. It is an integer that uniquely identifies the encounter during which the vision assessment took place. Example: 101, 102, 103.
r_horizontal decimal This is a decimal (3,2) column that stores the horizontal cup-to-disc ratio for the right eye. It can be NULL if the ratio is not available. Example: 0.45, 0.60, 0.75.
l_horizontal decimal This is a decimal (3,2) column that stores the horizontal cup-to-disc ratio for the left eye. It can be NULL if the ratio is not available. Example: 0.45, 0.60, 0.75.
r_vertical decimal This is a decimal (3,2) column that stores the vertical cup-to-disc ratio for the right eye. It can be NULL if the ratio is not available. Example: 0.45, 0.60, 0.75.
l_vertical decimal This is a decimal (3,2) column that stores the vertical cup-to-disc ratio for the left eye. It can be NULL if the ratio is not available. Example: 0.45, 0.60, 0.75.
comments varchar This is a varchar (255) column that stores any additional comments or notes related to the vision assessment. It has a default value of an empty string if no comments are provided. Example: "Patient has a family history of glaucoma", "Follow up in 6 months".
