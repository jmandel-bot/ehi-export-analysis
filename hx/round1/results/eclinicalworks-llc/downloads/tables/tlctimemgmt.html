Table tlctimemgmt
Description This table stores time management records for healthcare encounters, including the time taken for each encounter and the associated user and patient information.
Column DataType Description
Id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer. It uniquely identifies each record in the table. Example: 1, 2, 3, etc.
encounterId int This is a foreign key that refers to the encounter table's primary key. It is an integer that represents the unique identifier for a specific healthcare encounter. Example: 101, 102, 103, etc.
userId int This is a foreign key that refers to the user table's primary key. It is an integer that represents the unique identifier for a specific user (such as a healthcare professional) involved in the encounter. Example: 1001, 1002, 1003, etc.
PatientId int This is a foreign key that refers to the patient table's primary key. It is an integer that represents the unique identifier for a specific patient involved in the encounter. Example: 2001, 2002, 2003, etc.
TimeTaken varchar This is a varchar(255) column that stores the time taken for the encounter in a string format. Example: "1 hour 30 minutes", "45 minutes", "2 hours", etc.
date datetime This is a date column that stores the date of the encounter. It follows the format 'YYYY-MM-DD'. Example: '2021-06-15', '2021-06-16', '2021-06-17', etc.
time datetime This is a time column that stores the specific time of the encounter. It follows the format 'HH:mm:ss'. Example: '09:30:00', '14:45:00', '16:15:00', etc.
