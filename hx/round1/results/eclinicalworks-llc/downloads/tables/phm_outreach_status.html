Table phm_outreach_status
Description This table is used as a master table to store outreach status.
Column DataType Description
id (Primary Key) tinyint This is a unique identifier for each outreach status record. It is of type tinyint(4) and is set to auto-increment. Example values: 1, 2, 3, etc.
STATUS varchar This column stores the description of the outreach status. It is of type varchar(50) and must be unique for each record. Current values: 'Pending', 'Attempted'.
delflag tinyint This column indicates whether the record has been deleted or not. It is of type tinyint(1) and has a default value of '0'. A value of '0' means the record is not deleted, and a value of '1' means the record is deleted. Example values: 0, 1.
