EHI Export Schema
Table reviewanswers
Description This table stores the answers to review questions for a given encounter, configured in the 'reviewquestions' table. Used during the Change Assigned-To function. Itemkeys 0042_UK and 0593_UK need to be enabled for this feature.
Column DataType Description
id (Primary Key) int This is the primary key column and contains a unique integer value for each answer. For example, 1, 2, 3, etc.
reviewquestion_id int This column contains an integer value representing the id of the review question that the answer is related to. For example, 1, 2, 3, etc. This column can be a foreign key referring to the `id` column in a `reviewquestions` table.
encounterid int This column contains an integer value representing the id of the patient encounter that the answer is related to. For example, 1001, 1002, 1003, etc. This column can be a foreign key referring to the `id` column in an `encounters` table.
reviewanswer varchar This column contains a string value representing the actual answer provided by the doctor for the review question. For example, "Yes", "No", or "N/A".
doctor_id int This column contains an integer value representing the id of the doctor who provided the answer. For example, 101, 102, 103, etc. This column can be a foreign key referring to the `id` column in a `doctors` table.
