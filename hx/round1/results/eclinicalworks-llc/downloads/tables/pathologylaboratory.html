Table pathologylaboratory
Description This table stores information about pathology laboratories, their associated items, and patients.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer. It uniquely identifies each laboratory record. Example: 1, 2, 3, etc.
laboratoryname varchar This is a varchar column that stores the name of the pathology laboratory. Example: LabCorp, Quest Diagnostics, etc.
itemid int This is an integer column that stores the id of the item associated with the laboratory. It could potentially be a foreign key referring to another table's item id column. Example: 101, 102, 103, etc.
displayIndex int This is an integer column that stores the display index of the laboratory record, which can be used for sorting or ordering purposes. Example: 0, 1, 2, etc.
deleteflag tinyint This is a tinyint column that stores a boolean value (0 or 1) indicating whether the laboratory record has been deleted or not. Example: 0 (not deleted), 1 (deleted).
patientid int This is an integer column that stores the id of the patient associated with the laboratory. It could be a foreign key referring to another table's patient id column. Example: 1001, 1002, 1003, etc.
