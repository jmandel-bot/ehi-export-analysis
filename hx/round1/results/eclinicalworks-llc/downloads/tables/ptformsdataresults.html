EHI Export Schema
Table ptformsdataresults
Description This table stores HL7 messages received from the Chadis and Rothman interface and processes related information.
Column DataType Description
messageid (Primary Key) int This column is an auto incremented primary key of the table.
datereceived datetime This column stores the date when result is received from outside.
datemodified datetime This column stores the date when data is last modified for that row.
messagecontent text This column stores the decrypted HL7Message.
messagestatus tinyint This column stores the digit to represent the message status. i.e., 1 - Pass, 2 - Fail.
visitnumber varchar Unique identifier for the patient's visit. Example: VN12345.
patientid int This column stores the patient's unique id. Patientid refers to the UID column of the user table.
profileid varchar This column stores the profile id (also called as Accession id) of the test received in the Results Inbound HL7 message through the interface.
ptlastname varchar This column stores the patient's Last name.
ptfirstname varchar This column stores patient's first name.
ptdob datetime This column stores patient's Date of Birth.
errorlogs text This column stores the error logs when result fails while processing/preprocessing (status = 2).
encounterid int This column stores the encounterid. Encounterid refers to the encounterid column of the enc table.
hl7msgid int This column refers to the HL7Msgid column of ptformsdatahl7content table.
obrsno int This column stores the OBR number. When more than 1 order's results are sent in same hl7message, sequence number is set to this column for each OBR.
processflow text This column stores result processing flow description.
intid int This column stores the unique id of the lab company from where result received. This column refers to the primary key of the intlist table.
providerid int This column stores the doctorid recieved from the result or fetch from the details came in the result.
docname varchar This column stores name of the doctor.
docHL7id varchar Unique identifier for the document in HL7 format. Example: DOC1234.
documentid int This column stores the document id. When there is a embedded pdf attached in the result. It will be stored in document table and primary key of that table is referred by this column.
errorStatusCode tinyint This column stores the error code according to the error occurred while processing.
patientSex varchar Sex of the patient. Example: Male, Female.
