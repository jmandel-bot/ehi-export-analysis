EHI Export Schema
Table referraldetailsreporting
Description This table stores information about referral details for reporting purposes, including the sender and receiver's information, message type, and other related data.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
referralid int Identifier for the referral, can be NULL. Example: 101, 102, 103.
msgtype tinyint Message type as a tiny integer, can be NULL. Example: 1, 2, 3.
from_npi int National Provider Identifier (NPI) of the sender, can be NULL. Example: 1234567890, 2345678901.
from_id int Identifier for the sender, can be NULL. Example: 1001, 1002, 1003.
from_firstname varchar First name of the sender, default is an empty string. Example: 'John', 'Jane'.
from_lastname varchar Last name of the sender, default is an empty string. Example: 'Doe', 'Smith'.
from_zipcode varchar Zip code of the sender, default is an empty string. Example: '12345', '67890'.
from_taxonomycode varchar Taxonomy code of the sender, default is an empty string. Example: '207Q00000X', '207R00000X'.
from_specialty varchar Specialty of the sender, default is an empty string. Example: 'Family Medicine', 'Internal Medicine'.
from_apuid int Identifier for the sender's APU (Accountable Provider Unit), can be NULL. Example: 2001, 2002, 2003.
from_practicename varchar Practice name of the sender, default is an empty string. Example: 'ABC Clinic', 'XYZ Hospital'.
to_npi int National Provider Identifier (NPI) of the receiver, can be NULL. Example: 3456789012, 4567890123.
to_id int Identifier for the receiver, can be NULL. Example: 3001, 3002, 3003.
to_firstname varchar First name of the receiver, default is an empty string. Example: 'Alice', 'Bob'.
to_lastname varchar Last name of the receiver, default is an empty string. Example: 'Johnson', 'Brown'.
to_zipcode varchar Zip code of the receiver, default is an empty string. Example: '98765', '43210'.
to_taxonomycode varchar Taxonomy code of the receiver, default is an empty string. Example: '208M00000X', '208D00000X'.
to_specialty varchar Specialty of the receiver, default is an empty string. Example: 'Pediatrics', 'Dermatology'.
innetwork tinyint Indicates if the receiver is in the network as a tiny integer, can be NULL. Example: 0 (out of network), 1 (in network).
quesAnsGuid varchar Unique identifier for the question and answer, default is an empty string. Example: 'a1b2c3d4', 'e5f6g7h8'.
syncGuidWithP2P tinyint Indicates if the GUID is synchronized with P2P (Peer-to-Peer) as a tiny integer, default is 0. Example: 0 (not synchronized), 1 (synchronized).
from_netaff varchar Network affiliation of the sender, default is an empty string. Example: 'Network A', 'Network B'.
to_netaff varchar Network affiliation of the receiver, default is an empty string. Example: 'Network C', 'Network D'.
ts_utc datetime time stamp in UTC format, default is '0000-00-00 00:00:00'. Example: '2022-01-01 12:34:56', '2022-02-02 23:45:01'.
