EHI Export Schema
Table preventive
Description This table stores information about preventive medicine prescribed to patients.
Column DataType Description
encounterid int This column stores the encounterid of the appointment for which preventive medicine is added. This column refers to encounterid column in enc table.
catid int This column stores id of the category of the preventive medicine from list of pre defined preventive medicines. This column refers to items.itemid.
itemid int This column stores id of the item listed under category of the preventive medicine. This column refers to items.itemid.
propid int This column stores id of the item listed under category of the preventive medicine. This column refers to properties.propid.
value text This column stores value of the item listed under category of the preventive medicine.
displayIndex smallint This column stores a digit to indicate order in which item listed under category of the preventive medicine should be listed in user interface.
