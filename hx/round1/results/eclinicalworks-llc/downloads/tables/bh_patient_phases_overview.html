Table bh_patient_phases_overview
Description This table stores an overview of patient phases, including pre-admission, admission, and discharge information.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table (e.g., 1, 2, 3).
ptAcoid int The identifier of the patient's ACO program, foreign key relation to `ptacoprograms` table (`id` column).
activePhase int The identifier of the patient's active phase, foreign key relation to `bh_phases` table (`id` column).
preAdmission_startDate datetime The start date and time of the pre-admission phase (e.g., '2021-09-01 00:00:00').
preAdmission_byUID int The user identifier who initiated the pre-admission, foreign key relation to `users` table (`UID` column).
admission_startDate datetime The start date and time of the admission phase (e.g., '2021-09-02 00:00:00').
admission_byUID int The user identifier who initiated the admission, foreign key relation to `users` table (`UID` column).
discharge_date datetime The date and time of the patient's discharge (e.g., '2021-09-10 00:00:00').
discharged_byUID int The user identifier who initiated the discharge, foreign key relation to `users` table (`UID` column).
delFlag tinyint A flag indicating whether the record is deleted (0 for not deleted, 1 for deleted).
createdDateTime datetime The date and time when the record was created (e.g., '2021-09-01 12:34:56').
modifiedBy int The user identifier who last modified the record, foreign key relation to `users` table (`UID` column).
modifiedDateTime datetime The date and time when the record was last modified (e.g., '2021-09-02 14:56:32').
dischargeReason varchar A text description of the reason for the patient's discharge (e.g., 'Completed treatment successfully').
isDischargeTabLocked tinyint A flag indicating whether the discharge tab is locked (0 for unlocked, 1 for locked).
