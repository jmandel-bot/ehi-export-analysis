EHI Export Schema
Table vision_dispense_rx
Description This table stores information about vision prescriptions for both the left and right eyes, including various measurements and details related to the dispense order.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key.
Enc_id int This column stores the encounter id indicating the encounter which this prescription is tied to. This column refers to column encounterid from the enc table.
Left_vision_type varchar This column stores the type of a vision for a left eye: SVD, SVN, OCC, BIF, Bal, SVi and Prog.
Right_vision_type varchar This column stores the type of a vision for a right eye: SVD,SVN,OCC,BIF,Bal, SVi and Prog.
Sphere_Left varchar This column stores the sphere measurement for the left eye. Example: "-2.00", "+1.50".
Sphere_Right varchar This column contains a sphere measurement for the right eye. For example: "-1.75", "+2.00".
Cylinder_Left varchar Cylinder measurement for the left eye. Example: "-0.50", "+0.75".
Cylinder_Right varchar Cylinder measurement for the right eye. Example: "-0.25", "+1.00".
Axis_Left varchar Axis measurement for the left eye. Example: "90", "180".
Axis_Right varchar Axis measurement for the right eye. Example: "45", "135".
Horizontal_Prism_Left varchar Horizontal prism measurement for the left eye. Example: "2", "3".
Horizontal_Prism_Right varchar Horizontal prism measurement for the right eye. Example: "1", "4".
Vertical_Prism_Left varchar Vertical prism measurement for the left eye. Example: "1.5", "2.5".
Vertical_Prism_Right varchar Vertical prism measurement for the right eye. Example: "1", "3".
Add_Left varchar Additional power measurement for the left eye. Example: "+1.00", "+2.00".
Add_Right varchar Additional power measurement for the right eye. Example: "+1.50", "+2.50".
OCS_Left decimal Ocular center space measurement for the left eye. Example: "32.5", "33.0".
OCS_Right decimal Ocular center space measurement for the right eye. Example: "32.0", "33.5".
PDS_Left varchar Pupil distance measurement for the left eye. Example: "30", "31".
PDS_Right varchar Pupil distance measurement for the right eye. Example: "29", "32".
Heights_Left varchar Heights measurement for the left eye. Example: "15", "16".
Heights_Right varchar Heights measurement for the right eye. Example: "14", "17".
Insets_Left varchar Insets measurement for the left eye. Example: "2.5", "3.0".
Insets_Right varchar Insets measurement for the right eye. Example: "2.0", "3.5".
Back_Vertex varchar Back vertex distance measurement. Example: "12", "13".
isdeleted tinyint Flag to indicate if the record is deleted. Example: 0 (not deleted), 1 (deleted).
createdby int User id of the person who created the record. Example: 1001, 1002.
createdon datetime Date and time when the record was created. Example: "2021-01-01 12:34:56".
updatedby int User id of the person who last updated the record. Example: 1001, 1002.
updatedon datetime Date and time when the record was last updated. Example: "2021-01-02 12:34:56".
modifynotes text Notes related to modifications made to the record. Example: "Updated sphere values".
NEAR_PD_Left varchar Near pupil distance measurement for the left eye. Example: "28", "29".
NEAR_PD_Right varchar Near pupil distance measurement for the right eye. Example: "27", "30".
