EHI Export Schema
Table pcf_enrollment_details
Description This table is created to store PCF attribution details such as attribution start date, attribution end date, attribution reason, attributed facility, attributed provider and so on. 
Column DataType Description
id (Primary Key) int Unique identifier for each enrollment detail record. Example: 1, 2, 3.
ptacoId int Identifier for the PTACO program the patient is enrolled in. Foreign key relation to `ptacoprograms` table, `id` column. Example: 101, 102, 103.
patientId int Identifier for the patient. Foreign key relation to `users` table, `uid` column. Example: 1001, 1002, 1003.
attributed_startDt datetime Start date and time of the attribution period. Example: '2021-01-01 00:00:00', '2021-02-01 00:00:00'.
attributed_endDt datetime End date and time of the attribution period. Example: '2021-12-31 23:59:59', '2021-11-30 23:59:59'.
attributed_facilityId int Identifier for the facility where the patient is attributed. Foreign key relation to `edi_facilities` table, `Id` column. Example: 201, 202, 203.
attributed_providerId int Identifier for the provider who attributed the patient. Foreign key relation to `users` table, `uid` column. Example: 3001, 3002, 3003.
attributed_reason varchar Reason while enrolling patient into PCF program
attrIndex int This column works as a counter which could grow with number of valid attributions ended to each PCF program enrolled patient
attrEnded tinyint Flag indicates the record of whether attribution is ended.
attrEndedBy int Contains the Primary Key column UID of users table. This column stores user Id of logged in user who ended the PCF attribution
deleteFlag tinyint Flag indicating whether the record has been deleted (1) or not (0). Example: 0, 1.
invalidAttr tinyint Flag indicates the attributed record is not valid.
addedBy int Contains the Primary Key column UID of users table. This column stores user Id of logged in user who added the PCF attribution
addedDtTime datetime Date and time when the enrollment detail was added. Example: '2021-01-01 10:00:00', '2021-02-01 11:00:00'.
modifiedBy int Contains the Primary Key column UID of users table. This column stores user Id of logged in user who modified the PCF attribution
modifiedDtTime datetime Date and time when the enrollment detail was last modified. Example: '2021-01-01 12:00:00', '2021-02-01 13:00:00'.
