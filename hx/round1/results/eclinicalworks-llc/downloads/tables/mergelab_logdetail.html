EHI Export Schema
Table mergelab_logdetail
Description This table stores the log details of changes made to items during a merge operation in a lab.
Column DataType Description
Reportid int This columns store the report id from labdata as reference
OldItemid int This column stores old item id before the merge operation is performed
OldValue varchar This column stores the old value (string) of the item before the merge operation. For example, "Old Item A", "Old Item B", etc.
NewItemid int This column stores old item id after the merge operation is performed
NewValue varchar This column stores the new value (string) of the item after the merge operation. For example, "New Item A", "New Item B", etc.
Comments text This column stores any additional comments (text) related to the merge operation. For example, "Merged due to duplicate", "Merged to update information", etc.
