EHI Export Schema
Table propertydetail
Description This table stores customized observations created for properties available in Progress Notes sections such as HPI and Examination.
Column DataType Description
propid int This column stores an id of the property for which the observation is added. This column refers column propid from properties table.
value text This column stores name (text) of the observation added which is used to display it in Progress Note.
pdetailid (Primary Key) int This column is an auto incremented Primary Key.
displayIndex smallint This column stores the sequence id of the added observation while it is added to the property.
defaultFlag int This column stores value 0 or 1. 1 - to make this observation as a default observation for the property. 0 - when this observation is not set as a default observation for the property.
deleteFlag int This column stores value 0 or 1. 1 - when observation is deleted, 0 - when observation is not deleted.
vmid varchar This column stores alphanumeric value as a unique row identifier.
color varchar This column stores decimal value of the color selected for the observation.
bold smallint This column stores value 0 or 1. 1 - when observation font is to be Bold, 0 - when observation font is not to be Bold.
underline smallint This column stores value 0 or 1. 1 - when observation font is to be Underlined, 0 - when observation font is not to be Underlined.
