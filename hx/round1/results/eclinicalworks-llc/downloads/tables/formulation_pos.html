EHI Export Schema
Table formulation_pos
Description This table is used in the Allergy Formulation tab. This table stores information about positive allergens added in the Formulation window.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
ENCid int Foreign key referring to the id of the associated encounter. Example: 100, 101, 102. (Foreign key relation: `encounter_table.id`)
ITEMid int Foreign key referring to the id of the associated item. This refers to itemid column in items table
NAME varchar Name of the Allergen
PARENTid int This column stores the id of the parent allergen
DETAILid int This column stores detailid for the allergen
QUEid int This column stores id of the recipe queue. It refers to id column in antigen_recipe_queue table
