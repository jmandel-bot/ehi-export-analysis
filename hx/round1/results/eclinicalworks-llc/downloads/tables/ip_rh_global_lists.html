EHI Export Schema
Table ip_rh_global_lists
Description This table is a master table for the global item list for the Behavioral Health Integrated Solution (BHIS) module.
Column DataType Description
Id (Primary Key) int This column is a primary key of this table. It stores auto generated integer type value.
ListName varchar This column is used to store list name which is in 32-character long string.
ListDescription varchar This column is used to store list description which defines the list name usage which is in 64-character long string.
DeleteFlag tinyint This column is used to store tinyint type value which contains either 0 or 1. The value 1 indicates inactive state of the list item and the value 0 indicates active state of the item.
isDisplayingInUI tinyint The column is added to specify whether event type is going to displayed on Reason Dictionary screen or not. By default, its value 0. This column possible value is 0, 1 or null. For older entry null will be added.
