Table appt_miscinfo_labels
Description This table stores miscellaneous information labels related to appointments, including their names, deletion status, template tags, and whether they should be added to the billing summary.
Column DataType Description
id (Primary Key) int This is a primary key which increments itself automatically.
LabelName varchar This column stores the name of the label as a string with a maximum length of 50 characters. For example: "Important Note", "Follow-up", etc.
DeleteFlag int This column indicates whether the label has been deleted or not. It is an integer value where 0 means not deleted and 1 means deleted. For example: 0, 1.
templateTag varchar This column stores the template tag associated with the label as a string with a maximum length of 100 characters. For example: "imp_note", "follow_up", etc.
addtobillingsummary int This column indicates whether the label should be added to the billing summary or not. It is an integer value where 0 means not added and 1 means added. For example: 0, 1.
