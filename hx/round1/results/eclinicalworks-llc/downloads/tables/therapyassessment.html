EHI Export Schema
Table therapyassessment
Description This table stores information about therapy assessments, including the encounter, category, item, property, and value associated with each assessment.
Column DataType Description
encounterid int This column stores id of an encounter during which patient received Physical Therapy Assessment. This column refers to encounterid column in enc table.
catid int This column stores id of a category which is being documented during patient's Physical Therapy assessment. This column refers to itemid column in items table.
itemid int This column stores id of an item/question being documented under category documented in catid column of This column during patient's Physical Therapy assessment. This column refers to itemid column in items table.
propid int This column is not being used.
value varchar This column stores information documented by staff/provider during patient's Physical Therapy assessment. Staff/Provider may select pre-configured text in the user interfaceor enter a free-text value to document this information which is why column is declared as TEXT.
