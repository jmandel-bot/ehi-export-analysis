EHI Export Schema
Table immorders
Description This table stores information about immunization orders, including their content, status, and related patient and encounter data.
Column DataType Description
Messageid (Primary Key) int This is auto incremented and Primary Key of This column .
Immid int This column is used to store Interface id. Column does not expect NULL values as it is mandatory field. Defualt value is 0.
MessageContent text This field contains HL7 message. Column does not expect NULL values as it is mandatory field. Default value is empty string.
MessageStatus int It used to store record’s status and it refers to immfiltercriteria.statusvalue. Column does not expect NULL values as it is mandatory field. Defualt value is 0.
TransmittedOn datetime A datetime field that stores the date and time when the immunization order message was transmitted. Example: '2021-09-01 10:30:00'
patientid int This field contains patient id (patients.pid). Column does not expect NULL values as it is mandatory field. Defualt value is 0.
encounterid int This field contains encoutner id (enc.encounterid). Column does not expect NULL values as it is mandatory field. Defualt value is 0.
errorlogs text This field contains error logs while processing results at registry. Column can have NULL value.
facilitycode varchar This field contains facility id (edi_facilities.id) Column can not have NULL values. Default value is empty string.
HubMsgid int This field contains reference for message id in hub table. Column does not expect NULL values as it is mandatory field. Defualt value is 0.
