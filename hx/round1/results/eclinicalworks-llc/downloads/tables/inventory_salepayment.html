EHI Export Schema
Table inventory_salepayment
Description This table stores information about payments made for sales in the inventory system.
Column DataType Description
id (Primary Key) int Unique identifier for each sale payment record. Example: 1, 2, 3.
SaleDate datetime time stamp of when the sale payment was made, automatically updated to the current time stamp. Example: '2021-09-15 12:34:56'.
Sale_id int Foreign key referring to the `id` column in the `sales` table. Represents the sale associated with the payment. Example: 101, 102, 103.
Edi_InsPayments_id int Foreign key referring to the `id` column in the `edi_ins_payments` table. Represents the insurance payment associated with the sale payment. Example: 201, 202, 203.
IsEPayment int Flag indicating if it is from epayment
IsDeleted tinyint Tiny integer flag indicating if the sale payment record has been deleted (1) or not (0). Example: 0, 1.
UpdatedBy int Foreign key referring to the `id` column in the `users` table. Represents the user who last updated the sale payment record. Example: 301, 302, 303.
UpdatedOn datetime time stamp of when the sale payment record was last updated. Example: '2021-09-15 12:34:56'.
