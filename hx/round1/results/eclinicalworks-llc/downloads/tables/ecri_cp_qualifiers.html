EHI Export Schema
Table ecri_cp_qualifiers
Description This table stores sources of the clinical notes copied, pasted or merged to an encounter.
Column DataType Description
id (Primary Key) int This is the primary key column, containing unique integer values to identify each qualifier in the table. Example values: 1, 2, 3, etc.
source varchar This column contains the source of the qualifier as a varchar (up to 300 characters). Example values: "Patient-Specific Template", "Clipboard", etc.
