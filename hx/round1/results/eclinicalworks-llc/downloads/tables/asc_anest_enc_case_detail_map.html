Table asc_anest_enc_case_detail_map
Description This table stores the mapping between anesthesia and CaseFlowDetail.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
mapid int identifier for the anesthesia encounter case mapping.NOT NULL DEFAULT '0'. Example: 101, 102, 103.
detailid int identifier for the anesthesia encounter case detail.NOT NULL DEFAULT '0'. Example: 201, 202, 203.
value varchar The value associated with the anesthesia encounter case detail.DEFAULT NULL. Example: "2.5mg", "10mg", "5mg".
valueid varchar identifier for the value associated with the anesthesia encounter case detail. NOT NULL DEFAULT '0'. Example: "301", "302", "303".
defaultValue text The default value for the anesthesia encounter case detail. Example: "1mg", "2mg", "3mg".
defaultValueid varchar identifier for the default value associated with the anesthesia encounter case detail.DEFAULT ' '. Example: "401", "402", "403".
createdBy int identifier for the user who created the record.NOT NULL DEFAULT '0'. Example: 1001, 1002, 1003.
modifiedBy int identifier for the user who last modified the record.NOT NULL DEFAULT '0'. Example: 1001, 1002, 1003.
deleteFlag int Flag to indicate if the record is deleted (1) or not (0).NOT NULL DEFAULT '0'. Example: 0, 1.
