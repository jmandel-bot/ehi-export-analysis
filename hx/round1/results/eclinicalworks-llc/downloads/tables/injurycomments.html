EHI Export Schema
Table injurycomments
Description This table stores comments related to injuries, along with information about the user who made the comment and the injury it is associated with.
Column DataType Description
id (Primary Key) int This column is an auto-increment primary key.
injuryid int This column stores the injury ID of patient. This column refers to the AtRiskid column in the ccmr_atrisk table
userid int This column will save user ID who has entered comment. This column refers to UID column in user table.
comments varchar This column will save textual data which is entered by user in form of injury comment.
deleteflag tinyint This column will define that injury comment is deleted or not. A value 1 means comment is deleted, and value 0 means comment is not deleted.
lastModifiedDate varchar This column will save last modified date or inserted date.
