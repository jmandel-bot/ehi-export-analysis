EHI Export Schema
Table ip_rh_enc_pnCoSign
Description This table store the encounter's cosign notes mapping data.
Column DataType Description
id (Primary Key) int Auto increment primary key of the table. Data type defined as: INT with size 11
encounterId int This field stores encounter id from enc table. This field stores the encounter id for which the linked cosign is mapped. This maps the encounter with the configured progress note type.
cosigneeUserId int This field stores uid from users table.This field indicates receiver of cosign notes which is being sent.
cosignerUserId int This field stores uid from users table. This field indicates sender of cosign notes.
deleteFlag tinyint Stores delete flag of the column. Default value is defined as 0 which represents the row is active and not deleted and 1 represents the row is deleted and not active.
cosignStatus tinyint Stores cosign flag of the column. Default value is defined as 0 which represents the cosign notes is in default state.1 represents that cosign notes is pending and 2 represents cosign notes is addressed.
createdBy int Column stores the id of a user by whom Cosign notes data is being Created. createdBy is reference column storing uid field value of user's table.
createdDatetime datetime Stores the timestamp of block being created.
pendingDate datetime Stores the timestamp of Pending cosign date.
addressedDate datetime Stores the timestamp of Addressed cosign date.
modifiedBy int Column stores the id of a user by whom Cosign notes data is being modified. modifiedBy is reference column storing uid field value of user's table.
modifiedDatetime datetime Stores the timestamp of block being edited.
