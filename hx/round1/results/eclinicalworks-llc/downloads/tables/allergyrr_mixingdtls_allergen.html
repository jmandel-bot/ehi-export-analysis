Table allergyrr_mixingdtls_allergen
Description This table stores information about allergens used in allergy mixing details, including their concentration, volume, and manufacturer details.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
Concentrationid int identifier for the concentration of the allergen. Foreign key relation to `antigen_concentration_patient` table's `CONCENTRATION_id` column. Example: 101, 102, 103.
Queueid int identifier for the queue in which the allergen is used. Foreign key relation to `antigen_recipe_queue` table's `id` column. Example: 201, 202, 203.
Bottleid int identifier for the bottle containing the allergen. Foreign key relation to `allergyrr_mixingdtls_bottle` table's `id` column. Example: 301, 302, 303.
Itemid int identifier for the item associated with the allergen. Foreign key relation to `items` table's `itemid` column. Example: 401, 402, 403.
Detailid int identifier for the detail associated with the allergen. Example: 501, 502, 503.
ItemName varchar Name of the allergen item. Example: 'Pollen', 'Dust Mite', 'Mold'.
Volume float Volume of the allergen in the bottle. Example: 10.5, 20.0, 30.0.
ConcPercentage varchar Concentration percentage of the allergen. Example: '10%', '20%', '30%'.
SeqNo int Sequence number for the allergen in the mixing process. Example: 1, 2, 3.
Concentration varchar Concentration of the allergen in the bottle. Example: 'Low', 'Medium', 'High'.
LotNo varchar Lot number of the allergen. Example: 'L123', 'L456', 'L789'.
ExpDate datetime Expiration date of the allergen. Example: '2022-12-31 00:00:00', '2023-01-01 00:00:00', '2023-01-02 00:00:00'.
MANUFACTURERid int identifier for the manufacturer of the allergen. Foreign key relation to `allergy_manufacturers` table's `id` column. Example: 1001, 1002, 1003.
INV_MANUFACTURERid int identifier for the inventory manufacturer of the allergen. Foreign key relation to `inventory_manufacturer` table's `id` column. Example: 1101, 1102, 1103.
MANUFACTURERNAME varchar Name of the allergen manufacturer. Example: 'AllergyCorp', 'AllergenTech', 'AllergenPro'.
INV_PRODUCTid int identifier for the inventory product associated with the allergen. Foreign key relation to `inventory_product` table's `id` column. Example: 1201, 1202, 1203.
INV_STOCKid int identifier for the inventory stock of the allergen. Foreign key relation to `inventory_stock` table's `id` column. Example: 1301, 1302, 1303.
