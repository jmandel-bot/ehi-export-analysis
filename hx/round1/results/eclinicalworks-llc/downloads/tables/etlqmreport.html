EHI Export Schema
Table etlqmreport
Description This table stores information related to quality measure reports for patients, encounters, and healthcare providers.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
QMId int Quality Measure identifier, referring to a specific quality measure. Example: 101, 102, 103.
qualitymeasurename varchar Name of the quality measure. Example: "Blood Pressure", "Diabetes Management", "Cholesterol Control".
patientid int Patient identifier, referring to a specific patient. This could be a foreign key relation to a `patients` table with a column `id`. Example: 1001, 1002, 1003.
encounterid int Encounter identifier, referring to a specific encounter between a patient and a healthcare provider. This could be a foreign key relation to an `encounters` table with a column `id`. Example: 2001, 2002, 2003.
PCPid int Primary Care Provider identifier, referring to a specific healthcare provider. This could be a foreign key relation to a `providers` table with a column `id`. Example: 3001, 3002, 3003.
PCGid int Primary Care Group identifier, referring to a specific group of healthcare providers. This could be a foreign key relation to a `provider_groups` table with a column `id`. Example: 4001, 4002, 4003.
userid int User identifier, referring to a specific user who generated the report. This could be a foreign key relation to a `users` table with a column `id`. Example: 5001, 5002, 5003.
lastruntime varchar time stamp of the last time the report was generated. Example: "2021-08-01 12:34:56", "2021-08-02 14:23:45".
numerator int Numerator value of the quality measure, representing the number of patients who meet the criteria for the measure. Example: 0, 1.
denominator int Denominator value of the quality measure, representing the total number of patients eligible for the measure. Example: 0, 1.
userqmid varchar User-defined quality measure identifier, referring to a custom quality measure created by the user. Example: "UQM101", "UQM102", "UQM103".
