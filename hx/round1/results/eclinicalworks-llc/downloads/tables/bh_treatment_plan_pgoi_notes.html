Table bh_treatment_plan_pgoi_notes
Description This table stores notes related to treatment plans, including information about the structure, user who added the note, and whether the note is locked or deleted.
Column DataType Description
id (Primary Key) int Unique identifier for each note entry in the table. Example: 1, 2, 3.
structMappingid int identifier for the structure mapping associated with the note. Foreign key relation: `bh_treatment_plan_pgoid_mapping.id`. Example: 101, 102, 103.
structid int identifier for the structure associated with the note. Foreign key relation: `bh_treatment_plan_dictionary.id`. Example: 201, 202, 203.
notes varchar Text content of the note, with a maximum length of 4000 characters. Example: "This is a sample note for the treatment plan."
addedByUserid int identifier for the user who added the note. Foreign key relation: `users.UID`. Example: 301, 302, 303.
addedDateTime datetime Date and time when the note was added, in the format of YYYY-MM-DD HH:MM:SS. Example: "2021-09-01 12:34:56".
isLocked tinyint A flag indicating whether the note is locked (1) or not (0). Example: 0, 1.
deleteFlag tinyint A flag indicating whether the note is deleted (1) or not (0). Example: 0, 1.
