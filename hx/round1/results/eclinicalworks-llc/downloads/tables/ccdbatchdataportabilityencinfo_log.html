EHI Export Schema
Table ccdbatchdataportabilityencinfo_log
Description This table stores logs related to the generation of Continuity of Care Documents (CCD) for patients in a batch. It keeps track of the batch, encounter, and patient information, as well as the time stamp of the log entry.
Column DataType Description
id (Primary Key) int Unique identifier for each log entry. It is an integer value and is the primary key for This column . Example: 1, 2, 3.
Batchid int This column stores primary batch id from ccdbatchdataportability table
Encounterid int This column stores encounter id of the transaction
CCDGenerated int This column maintains CCDA generated status for patient's encounters. Possible values - 1 - Success 3 - Failed
Patientid int This column stores patient id of the transaction
ts datetime This column stores date and time of the transaction
