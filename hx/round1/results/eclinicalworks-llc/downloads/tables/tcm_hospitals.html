EHI Export Schema
Table tcm_hospitals
Description This table stores hospital information for the Transition Care Management module.
Column DataType Description
id (Primary Key) int This column is an auto incremented Primary Key of This column .
FacilityType varchar This column stores the name of the facility type ALF,Expired,Home,Home Health,Hospice,Hospital,ICF,Left-AMA,Other,SNF/Rehab etc. from the table tcm_facilitytype
Name varchar This column stores the name of the facility.
AddressLine1 varchar This column stores the address of the facility.
AddressLine2 varchar This column stores the address of the facility.
City varchar This column stores the city of the facility.
State varchar This column stores the state of the facility.
Zip varchar This column stores the zip code of the facility address area.
Tel varchar This column stores the telephone number of the facility.
hl7id varchar This column stores the hl7id which is used to get facility when hl7message contains the same HL7id for the facility.
DeleteFlag tinyint This column stores the 0 or 1. 1 is considered as the facility is deleted.
ModifiedBy varchar This column stores the logged-in user's full name (last name, first name) who adds or update this facility.
LastModified datetime This column stores date and time when this facility was last updated or modified.
notes text This column stores the any kind of description or instruction related to facility
email varchar This column stores email of the facility.
website text This column stores website of the facility.
facilityTypeid int This column stores id column of tcm_facilitytype table.
ediFacilityid int Foreign key relation to the `edi_facilities` table, referencing the `id` column. This column stores the unique identifier of the EDI facility associated with the hospital.
