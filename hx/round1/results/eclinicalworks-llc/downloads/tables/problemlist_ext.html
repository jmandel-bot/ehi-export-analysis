EHI Export Schema
Table problemlist_ext
Description This table stores patient's no known problems data.
Column DataType Description
id (Primary Key) int This column is an auto incremented Primary Key of This column .
Patientid int This column stores id of a patient being seen by the provider for whom problem list is documented. This column refers column UID from user table.
NKPFlag tinyint This column stores a flag 0 or 1.1 - when Provider confirms there is no known problem for the patient seen. 0 - Otherwise.
NKPDate datetime This column stores a time stamp when the provider confirms there is no known for a patient. The format used is YYYY-MM-DD HH:MM:SS.
deleteFlag tinyint This column is not actively used, hence it is set to default value 0.
