Table ptprograms
Description This table stores information about different programs associated with a particular patient.
Column DataType Description
id (Primary Key) int This is the primary key for the table, which is an auto-incrementing integer value. It uniquely identifies each record in the table. Examples include 1, 2, 3, etc..
Ptid int This is a foreign key referring to the patient's id in another table (e.g., `patients` table). It is an integer value and represents the patient associated with the program. Examples include 101, 102, 103, etc..
PtProgram int This is an integer value representing the specific program associated with the patient. This is a foreign key referring to id column of referralpgms table.
StartDt varchar This is a varchar(25) column that stores the start date of the program in string format (e.g., '2021-01-01', '2021-02-15', '2021-03-20', etc.).
EndDt varchar This is a varchar(25) column that stores the end date of the program in a string format. Examples include '2021-12-31', '2021-11-30', '2021-10-15', etc..
timestamp datetime This is a timestamp column that stores the date and time when the record was last updated. It is automatically updated whenever the record is modified. Examples include '2021-01-01 12:34:56', '2021-02-15 10:20:30', '2021-03-20 18:45:22', etc..
deleteFlag tinyint This is a tinyint(2) column that indicates whether the record has been deleted or not. A value of '0' means the record is active, while a value of '1' means the record has been deleted (0 = active and 1 = inactive).
