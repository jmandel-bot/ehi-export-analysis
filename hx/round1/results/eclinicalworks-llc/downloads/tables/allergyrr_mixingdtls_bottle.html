Table allergyrr_mixingdtls_bottle
Description This table stores information about the mixing details of allergy bottles, including concentration, queue, and recipe details.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. (e.g., 1, 2, 3)
Concentrationid int identifier for the concentration of the antigen in the bottle. Foreign key relation to `antigen_concentration_patient` table's `CONCENTRATION_id` column. (e.g., 101, 102, 103)
Queueid int identifier for the queue of the antigen recipe. Foreign key relation to `antigen_recipe_queue` table's `id` column. (e.g., 201, 202, 203)
Recipeid int identifier for the antigen recipe. Foreign key relation to `antigen_recipe` table's `id` column. (e.g., 301, 302, 303)
IsNewBottle tinyint A flag indicating whether the bottle is new or not. It is stored as a tiny integer with a default value of '0'. (e.g., 0 for not new, 1 for new)
DilutionFromid int identifier for the dilution source. Default value is '0'. (e.g., 0, 401, 402)
DilutionFrom varchar Description of the dilution source. Default value is an empty string. (e.g., '', 'Source A', 'Source B')
DilutionBottleNo varchar Unique identifier for the dilution bottle number. Default value is an empty string. (e.g., '', 'B001', 'B002')
TotalVolume float Total volume of the antigen in the bottle. Stored as a floating-point number with a default value of '0'. (e.g., 0, 100.5, 200.25)
CreatedBy int identifier for the user who created the record. Foreign key relation to `users` table's `UID` column. (e.g., 501, 502, 503)
