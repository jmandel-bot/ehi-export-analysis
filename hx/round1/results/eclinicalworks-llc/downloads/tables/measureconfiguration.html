EHI Export Schema
Table measureconfiguration
Description This table stores the configuration for CDSS measures.
Column DataType Description
Measureid (Primary Key) varchar This column stores an id of the CDSS measure. This is alphanumeric and it refers column Measureid from measuredefinitions table.
Disabled tinyint This column indicates if the measure is disable or enabled. 0 - Enabled and 1 - Disabled.
Dictionary tinyint This column indicates an id of the dictionary to which this measure is linked. Possible values are - TCNY(1), DOC_NYC(2), RI(3), PQRI(7), CGHN(8), MCMP(4), LMM(5), BAYLOR(6), EHEARTS(9), MEANINGFUL_USE(10), POINT_OF_CARE(11);
NumeratorElement varchar This column indicates what codes can be used to satisfy the alert. This column refers to column Name in CDSS_codes table.
NumeratorElementType varchar This column is linked to NumeratorElement column. Possible values are Lab or DI or Immunization.
truserid int This column stores an id of the user who modifies the alert. This column refers column UID from user table.
eddate datetime This column stores the date when the alert was modified. The format used is YYYY-MM-DD 00:00:000.
