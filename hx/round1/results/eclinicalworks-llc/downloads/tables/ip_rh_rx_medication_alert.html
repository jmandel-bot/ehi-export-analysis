EHI Export Schema
Table ip_rh_rx_medication_alert
Description This table logs details of medication pertaining to subsequent ambulatory encounter after RBH active encounter.
Column DataType Description
id (Primary Key) int Unique identifier for each medication alert record.
itemid int This column stores id of the medication added to the encounter for which alert is set.This column refers column itemId from items table.
encounterId int This column is used to store encounter ID and is foreign key references encounterId column of enc table.
rxOrderNo varchar This column stores order number for a set of medication/drug added tospecific encounter. This number is generated on the application side.
medName varchar This column stores name of the medicine/drug of the id stored in itemId column which is added as part of the treatment and on which alert is set.
userId int This column stores id of the user who is treating/documenting the encounter for which alert is set. This column refers column uid from users table.
addedDateTime datetime This column stores the current timestamp when the medication is added as part of the patient's treatment for which alert is created.
isAlertActive tinyint This column indicates if the created alert is active or not. For multiple entries only the newest record is active, once the new entry is received previous entries are marked as inactive. Possible Values: 1 - alert is active, 0 - alert is inactive. Default value is 1 for medication alert record.
rxComment varchar This column stores one of the below values indicating status of the medication addedfor the patient on which alert is created. Possible values :'Unknown' , 'Discontinued', 'Not-Taking', 'Taking', 'Stop', 'Start', Continue', 'Increase', 'Decrease'
isMedDelete tinyint This column store one of the below values indicating medication is deleted or not. When the medication is marked as delete only the last active entry is updated as deleted.Possible Values: 1 = medication deleted, 0 = medication not deleted.Default value is 0 for medication alert record.
