Table inventory_sku
Description This table stores information about inventory items, their associated facilities, and their status.
Column DataType Description
id (Primary Key) int Unique identifier for each inventory item (integer). Example: 1, 2, 3.
Name varchar Name of the inventory item (string). Example: "Item A", "Item B", "Item C".
Facility_id int identifier for the facility where the inventory item is stored (integer). This is a foreign key that refers to the facility table's id column. Example: 101, 102, 103.
IsDeleted tinyint A flag to indicate if the inventory item has been deleted (boolean). 0 means not deleted, 1 means deleted. Example: 0, 1.
UpdatedBy int identifier for the user who last updated the inventory item (integer). This is a foreign key that refers to the user table's id column. Example: 1001, 1002, 1003.
UpdatedOn datetime time stamp of when the inventory item was last updated (datetime). Example: '2021-08-01 12:34:56', '2021-08-02 14:30:00'.
parent_id int identifier for the parent inventory item, if any (integer). This is a foreign key that refers to the inventory_sku table's id column. Example: 1, 2, 3.
isonhold tinyint A flag to indicate if the inventory item is on hold (boolean). 0 means not on hold, 1 means on hold. Example: 0, 1.
priorityLevel int Priority level of the inventory item (integer). Higher numbers indicate higher priority. Example: 0, 1, 2.
