EHI Export Schema
Table vision_motility_versions_deviations
Description This table stores information about vision motility deviations for patients documented during eye exams.
Column DataType Description
id (Primary Key) int This column is an auto incremental primary key.
patientid int This column refers to the UID column in the user table. It stores the patient id for patients.
encid int This column stores an encounter iid of a patient. This column refers column encounterid from enc table.
examid int This column stores the exam id and has a foreign key constraint on "id" column of "vision_motility_versions" table.
gaze_direction varchar This columns stores gaze directions for the deviation option.
horizontalOptionValue varchar This columns stores the horizontal option value for Versions exam in the Motility window. Value can be stored as text up to 22 characters. This column length is varchar(22). Default value is NULL.
verticalOptionValue varchar This columns stores the vertical option value for the Versions exam in the Motility window. Values can be stored as text up to 22 characters. This column length is varchar (22). Default value is null.
deviationOptionValue varchar This columns stores the deviation option value for the Versions exam in the Motility window. Values can be stored as text up to 22 characters. This column length is varchar (22). Default value is null.
orthoFlickValue varchar This columns stores the Ortho/Flick option value for the Versions exam in the Motility window. Values can be stored as text up to 22 characters. This column length is varchar (22). Default value is null.
deleteflag tinyint This column stores values of either 0 (not deleted) or 1 (deleted).
created_date datetime This column stores date and time when duction was created. Datetime is stored in 'YYYY-MM-DD hh:mm:ss.000' format.
last_updated_date datetime This column stores date and time when duction was last updated. Datetime is stored in 'YYYY-MM-DD hh:mm:ss.000' format.
