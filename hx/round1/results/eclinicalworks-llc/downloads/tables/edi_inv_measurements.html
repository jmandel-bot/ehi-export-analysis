EHI Export Schema
Table edi_inv_measurements
Description This table stores measurements and qualifiers related to Invoice CPT ids.
Column DataType Description
InvMeasureid (Primary Key) int Unique identifier for each inventory measurement. This is the primary key for the table. Example: 1, 2, 3.
InvCptid int identifier for the inventory control point that the measurement is associated with. This could be a foreign key referring to another table's column, such as `inventory_control_points(id)`. Example: 101, 102, 103.
RefidCode char A 2-character reference code for the inventory measurement. Example: 'A1', 'B2', 'C3'.
Qualifier char A 3-character qualifier code for the inventory measurement. Example: 'ABC', 'DEF', 'GHI'.
MeasurementValue varchar The actual value of the inventory measurement, stored as a string. Example: '10.5', '25.3', '18.0'.
deleteFlag tinyint A tiny integer flag indicating whether the record has been deleted (1) or not (0). Example: 0, 1.
