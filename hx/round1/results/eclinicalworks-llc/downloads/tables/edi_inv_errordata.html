EHI Export Schema
Table edi_inv_errordata
Description This table stores error data related to invoices like invoiceid, error code, error description, etc.
Column DataType Description
id (Primary Key) int Unique identifier for each error record. It is an integer value and is the primary key for the table. Example: 1, 2, 3.
Invid int identifier for the invoice from edi_invoice table
ErrCode varchar Error code associated with the error. It is a string of up to 10 characters. Example: 'E001', 'E002', 'E003'.
ErrDesc varchar Description of the error. It is a string of up to 255 characters. Example: 'Invalid invoice number', 'Missing customer information', 'Incorrect item price'.
EnteredBy int identifier for the user who entered the error data. It is an integer value and refers to the user id in user table
EnteredDate varchar Date when the error data was entered. It is a string of up to 25 characters. Example: '2021-08-01 12:34:56', '2021-08-02 14:23:45'.
DeletedBy int identifier for the user who deleted the error data. It is an integer value and refers to the user id in user table
DeletedDate varchar Date when the error data was deleted. It is a string of up to 25 characters. Example: '2021-08-10 10:30:45', '2021-08-11 11:45:30'.
DeleteFlag int Flag indicating whether the error data has been deleted or not. It is an integer value, where 0 means not deleted and 1 means deleted. Example: 0, 1.
