Table prisma_encounter_hospitalization_information
Description This table stores information related to hospitalization encounters, including discharge disposition, readmission details, and origin and destination details.
Column DataType Description
ID (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
ENCOUNTER_DETAIL_ID bigint Foreign key referring to the `prisma_encounter_details` table's `id` column. It represents the encounter detail associated with the hospitalization information. Example: 101, 102, 103.
DISCHARGE_DISPOSITION_CODE varchar Code representing the discharge disposition. Example: "01", "02", "03".
DISCHARGE_DISPOSITION_DISPLAY varchar Display text for the discharge disposition. Example: "Home", "Transferred to another facility", "Expired".
DISCHARGE_DISPOSITION_SYSTEM varchar System used for the discharge disposition code. Example: "http://terminology.hl7.org/CodeSystem/discharge-disposition", "http://example.com/discharge-disposition".
DISCHARGE_DISPOSITION_TEXT varchar Additional text description for the discharge disposition. Example: "Discharged to home", "Transferred to rehabilitation center", "Patient expired".
READMISSION_CODE varchar Code representing the readmission details. Example: "R1", "R2", "R3".
READMISSION_DISPLAY varchar Display text for the readmission details. Example: "Readmitted within 30 days", "Readmitted after 30 days", "Not readmitted".
READMISSION_SYSTEM varchar System used for the readmission code. Example: "http://example.com/readmission", "http://terminology.hl7.org/CodeSystem/readmission".
READMISSION_TEXT varchar Additional text description for the readmission details. Example: "Patient readmitted within 30 days due to complications", "Patient readmitted after 30 days for a different issue", "Patient was not readmitted".
ORIGIN_DISPLAY varchar Display text for the origin of the hospitalization. Example: "Emergency Department", "Clinic", "Home".
ORIGIN_ID varchar Identifier for the origin of the hospitalization. Example: "O1", "O2", "O3".
DESTINATION_DISPLAY varchar Display text for the destination after the hospitalization. Example: "Home", "Rehabilitation center", "Long-term care facility".
DESTINATION_ID varchar Identifier for the destination after the hospitalization. Example: "D1", "D2", "D3".
