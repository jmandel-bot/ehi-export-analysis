EHI Export Schema
Table deviceitems
Description This table stores information about various medical devices and their properties.
Column DataType Description
id (Primary Key) int Unique identifier for each device item. (Example: 1, 2, 3)
Itemid int identifier for the item, with a default value of 0. (Example: 101, 102, 103)
ItemType varchar Type of the device item, with possible values being 'Audio', 'Ecg', 'Holter', 'Oximetry', 'Scale', 'Spiro', 'Stress', 'MedicalImaging', 'Vision', and 'ABP'. (Example: 'Ecg', 'Holter', 'Oximetry')
DeviceName varchar Name of the device. (Example: 'CardioCheck', 'PulseOx', 'SpiroMeter')
ModelNum varchar Model number of the device. (Example: 'A123', 'B456', 'C789')
VersionNum varchar Version number of the device. (Example: '1.0', '2.0', '3.0')
isReviewable int Indicates if the device is reviewable, with 1 for yes and 0 for no. (Example: 1, 0)
hasExtTestid int Indicates if the device has an external test id, with 1 for yes and 0 for no. (Example: 1, 0)
FaxReportType varchar Type of the fax report, with possible values being 'JPG', 'PDF', 'SCP', 'DCM', 'BMP', 'GIF', 'PNG', 'DOC', and 'TIF'. Default value is 'PDF'. (Example: 'JPG', 'PDF', 'DOC')
hasOrderTest int Indicates if the device has an order test, with 1 for yes and 0 for no. Default value is 0. (Example: 1, 0)
specialityformid int identifier for the specialty form associated with the device. (Example: 1001, 1002, 1003)
IsLegacyDevice varchar Indicates if the device is a legacy device, with possible values being 'Yes' and 'no'. Default value is 'no'. (Example: 'Yes', 'no')
Labid int identifier for the lab associated with the device, with a default value of 0. (Example: 2001, 2002, 2003)
isDiscreteData varchar A comma-separated string indicating if the device has discrete data. (Example: '1,0,0', '0,1,1', '1,1,0')
docTypeid int identifier for the document type associated with the device, with a default value of 3. (Example: 3001, 3002, 3003)
docCatid varchar identifier for the document category associated with the device, with a default value of 'XRay_doc'. (Example: 'XRay_doc', 'MRI_doc', 'CT_doc')
docid int identifier for the document associated with the device, with a default value of 3. (Example: 4001, 4002, 4003)
