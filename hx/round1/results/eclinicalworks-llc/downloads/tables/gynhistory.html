EHI Export Schema
Table gynhistory
Description This table stores the gynecological history of patients during their encounters with healthcare providers.
Column DataType Description
encounterid int This column stores id of an encounter for which gyn history is being documented. This column refers to encounterid column in enc table.
itemid int This column stores itemid of items listed under Symptom section on Gyn History screen documented during the encounter. This column refers to itemid column in items table.
Notes varchar This column stores free text note documented by end user for Symptoms listed on User Interface.
hxFlag smallint This column stores values 0/1 for H/O flag in user interface. 0 - If H/O is not marked by end user, 1 - If H/O is marked by the end user
denyFlag smallint This column stores values 0/1 for denies flag in the user interface. 0 - If patient does not deny for Symptom being documented, 1 - if patient deny for a Symptom being documented.
