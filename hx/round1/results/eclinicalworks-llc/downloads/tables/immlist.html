EHI Export Schema
Table immlist
Description This table is a master table and stores immunization registry list.
Column DataType Description
id (Primary Key) int Unique identifier for each immunization registry entry. Example: 1, 2, 3.
ImmRegistryName varchar This column stores a name (string) of the State Registries configured for a practice.
deleteFlag smallint Indicates whether the registry entry is deleted or not. 0 means not deleted, 1 means deleted. Example: 0, 1.
interfaceConfigured int Indicates whether the interface for the registry is configured or not. 0 means not configured, 1 means configured. Example: 0, 1.
IsVXQConfigured tinyint Indicates whether the VXQ (Vaccine Query) configuration is enabled or not. 0 means not enabled, 1 means enabled. Example: 0, 1.
RegCode varchar Registration code for the immunization registry. Example: AB123, XY789.
adapterFlag int This column is not in use.
IsAdapter int Set 1 for template setup for VXU interfaces.
IsVaccineInterfaceConfigured tinyint Indicates whether the vaccine interface is configured or not. 0 means not configured, 1 means configured. Example: 0, 1.
