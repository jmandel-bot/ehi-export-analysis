EHI Export Schema
Table labcommunityitems
Description This table stores information about lab community items, including their unique identifier, name, deletion status, and their relationships to other elements and order sets.
Column DataType Description
Communityid (Primary Key) int This column stores the auto-incremental primary key.
CommunityName varchar This column stores the community name.
DeleteFlag int This column indicates whether the lab community item has been deleted or not. It is an integer value where 0 means not deleted and 1 means deleted. By default, it is set to 0.
CommunityElementid int This column stores the element ID. The integer 1 indicates a lab, 2 indicates lab results, and 3 indicates radiology.
OrderSetItemid int This column stores the Order Set item ID, and refers to the itemid from the items table.
