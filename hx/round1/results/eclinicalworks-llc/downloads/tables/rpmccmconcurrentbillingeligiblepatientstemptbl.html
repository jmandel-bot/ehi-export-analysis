Table rpmccmconcurrentbillingeligiblepatientstemptbl
Description This table stores temporary information related to CCM and RPM billing eligible patients for selected billing month.
Column DataType Description
id (Primary Key) int Contains auto incremented id, datatype is INT (11). This column stores integer value which could grow with number of transactions over the years hence integer(Range: 0-2,147,483,647) is selected. It is AUTO_INCREMENT column and has PRIMARY KEY index
patientid int This column stores unique id of a Patient which references the UID column of the user table. The data type chosen is INT. This column allows a NULL value.
batchid int This column stores unique id of a phm billing batch which references the id column of the ccm_batch table. The data type chosen is INT. This column allows a NULL value.
batchDate datetime This column stores date when the phm billing batch is run. The data type chosen is DATE. This column allows a NULL value.
strData varchar This column stores json object which contains details eligible patients for rpm ccm concurrent billing.
