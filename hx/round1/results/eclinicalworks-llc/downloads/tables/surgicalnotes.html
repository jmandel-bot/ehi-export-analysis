EHI Export Schema
Table surgicalnotes
Description This table stores patient's surgical notes (client specific, itemkey - SurgicalInterface).
Column DataType Description
id (Primary Key) int This column is an auto incremented Primary Key for This column .
Encounterid int identifier for the encounter associated with the surgical note. This is a foreign key that refers to the `enc` table's `encounterid` column. Example: 101, 102, 103.
Examid int This column stores category id selected while documenting the surgical notes for a patient. Foreign key relationship. items.itemid
Propid int This column stores the property id for which the surgical notes documented. Foreign key relationship - properties.propid
PropDetailid int This column is not being used anymore.
Selection int This column is not being used anymore.
Notes text Text containing the actual surgical notes. Example: "Patient tolerated the procedure well.", "Sutures were placed without complications."
createDate datetime This column is not being used anymore.
displayseq int This column is not being used anymore.
notes2 text This column stores notes documented in notes column in HTML format.
displaycatseq int This column stores the sequence number of the categories to be displayed on the Progress Note of the patient.
