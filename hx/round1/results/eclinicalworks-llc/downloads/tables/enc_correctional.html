Table enc_correctional
Description This table stores information about encounters in a correctional facility, including details about the patient's medical conditions and visit dates.
Column DataType Description
Id (Primary Key) int This is an auto incremental primary key of This column .
EncounterId int The column stores an id of the encounter of a patient. This column refers column encounterId from enc table.
BookAndCaseNo varchar This column will store BookandCaseno of patient at the time of creation of an appointment.
IsIntakeVisit tinyint This column stores value 0 / 1 indicating if the appointment is intake type of visit. 0 - Not Intake type of visit, 1 - Intake type of visit
IsAsthmatic tinyint This column is not in use.
IsDiabetic tinyint This column is not in use.
IsAsthmaticOverridden tinyint This column is not in use.
IsDiabeticOverridden tinyint This column is not in use.
IsBizarreBehavior tinyint This column is not in use.
DeleteFlag tinyint This column stores value 0 / 1 indicating if the Patient enc info is deleted, this is soft delete. 0 - Not Deleted , 1 -Deleted
CheckInDate datetime This column stores the check-in date of the patient for the appointment created.
CheckOutDate datetime This column stores the check-out date of the patient for the appointment created.
IsEncHasProvider tinyint This column stores value 0 / 1 indicating if the appointment has provider. 0 - Appointment has no provider, 1 - Appointment has provider
Patients_Correctional_Id int This column stores id of the patient correctional information. Column refers to id column in patients_correctional table.
