Table vision_cover_phoria_test
Description This table stores information about vision cover phoria tests performed on patients during their encounters.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3, etc.
Encounterid int identifier for the encounter during which the vision cover phoria test was performed. This is a foreign key that refers to the `id` column in the `encounters` table. Example: 101, 102, 103, etc.
ExamType int Integer representing the type of exam performed. Example: 1, 2, 3, etc.
CoverPhoriaFlag int Integer flag indicating the status of the cover phoria test. Example: 0 (not performed), 1 (performed).
TestName varchar Name of the vision cover phoria test. Example: "Cover Test", "Phoria Test", etc.
ODHorz varchar Horizontal measurement for the right eye (OD) in the test. Example: "2 PD", "4 PD", etc.
ODVert varchar Vertical measurement for the right eye (OD) in the test. Example: "1 PD", "3 PD", etc.
OSVert varchar Vertical measurement for the left eye (OS) in the test. Example: "1 PD", "3 PD", etc.
WWORx int Tiny integer value representing the result of the test. Example: 0 (normal), 1 (abnormal).
comments text Text field for any additional comments or observations related to the test. Example: "Patient reported discomfort during the test", "Test results inconclusive", etc.
OSHorz varchar Horizontal measurement for the left eye (OS) in the test. Example: "2 PD", "4 PD", etc.
