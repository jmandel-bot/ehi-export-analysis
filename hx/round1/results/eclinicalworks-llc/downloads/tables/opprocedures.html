EHI Export Schema
Table opprocedures
Description This table stores information about Operative Reports Procedure details.
Column DataType Description
encounterid (Primary Key) int This column stores id of an encounter for which Operative Report detail of Patient's visit is documented. This column refers column encounterid from enc table.
PostOp int This column stores values 0/1 indicating post-operative diagnosis are same as pre-operative diagnosis. 0 - post-operative diagnosis are different then pre-operative diagnosis. 1 - post-operative diagnosis are same as pre-operative diagnosis.
anesthesia text This column stores the free text Anesthesia detail.
details text This column stores the free text details of procedure.
notes text This column stores the free text note of operative report.
Assistantid int This column stores id of the Provider/Staff who marked as assistant for the procedure. This column refers to column UID from user table.
Indications text This is a text column that stores the medical indications for the operative procedure. Example: "Acute appendicitis"
Specimens text This column stores free-text specimens which accurately describe the procedure.
Complications text This column stores free-text complications which accurately describe the procedure.
Findings text This column stores free-text findings which accurately describe the procedure.
