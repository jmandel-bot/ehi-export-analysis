Table prisma_encounter_problem_codes
Description This table stores the problem codes related to medical encounters, including information about the code system, code name, and priorities.
Column DataType Description
id (Primary Key) bigint Unique identifier for each record in the table. Example: 1, 2, 3.
problemId bigint Identifier for the related problem in the `prisma_encounter_problem_list_details` table. Example: 101, 102, 103.
codeSystem varchar The code system used for the problem code. Example: "ICD-10", "SNOMED-CT", "LOINC".
code varchar The specific code within the code system for the problem. Example: "J45.9", "A41.9", "E11.9".
codeName varchar The name or description of the problem code. Example: "Asthma, unspecified", "Sepsis, unspecified", "Type 2 diabetes mellitus without complications".
sourcePriority int The priority of the source providing the problem code. Example: 0, 1, 2.
codeSystemPriority int The priority of the code system used for the problem code. Example: 0, 1, 2.
duplicatedBy int Indicates if the record is duplicated by another record. If not duplicated, the value is -1. Example: -1, 3, 5.
