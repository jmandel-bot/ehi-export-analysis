EHI Export Schema
Table vision_order_lot
Description This table is used to store contact lens order lot number information.
Column DataType Description
id (Primary Key) int Unique identifier for each lot in the vision_order_lot table. Example: 1, 2, 3.
clmiscid int Foreign key referring to the unique identifier of the related vision order in another table (e.g., vision_order). Example: 101, 102, 103.
lotno varchar A string representing the lot number. Example: 'LOT001', 'LOT002', 'LOT003'.
isdeleted tinyint A boolean flag indicating whether the lot has been deleted or not. 0 means not deleted, and 1 means deleted. Example: 0, 1.
createdby int An integer representing the unique identifier of the user who created the lot. Example: 1001, 1002, 1003.
createdon datetime A datetime value representing the date and time when the lot was created. Example: '2021-09-01 10:30:00', '2021-09-02 14:45:00'.
updatedby int An integer representing the unique identifier of the user who last updated the lot. Example: 1001, 1002, 1003.
updatedon datetime A datetime value representing the date and time when the lot was last updated. Example: '2021-09-01 10:30:00', '2021-09-02 14:45:00'.
