EHI Export Schema
Table encaddendums
Description This table stores addendums related to specific encounters, along with the user who created the addendum and the date and time it was created.
Column DataType Description
id (Primary Key) int This column is an auto incremented primary key.
encounterid int This column stores an id of the locked encounter for which an addendum is added. This column refers to column encounterid from the enc table.
userid int This column stores an id of the user who adds an addendum to the locked encounter. This column refers to the column UID from the user table.
date datetime This column stores the date on which an addendum is added for this locked encounter. The format used is: YYYY-MM-DD
time datetime This column stores the time on which an addendum is added for this locked encounter. The format used is: HH:MM:SS.
addendum text This column stores free-text information added as an addendum to the Progress Note of this locked encounter.
regionName varchar This varchar column stores the region name as a string; this aids in displaying the time zone. For example: America/New_York
