EHI Export Schema
Table correctional_specialneedtype
Description This table stores information about different types of special needs for individuals in a correctional facility.
Column DataType Description
id (Primary Key) int This column stores primary value for the table.
needtype varchar This column stores text value for the special need type.
delflag tinyint This column stores delete flag for the entry. 0 for non-deleted entry 1 for deleted entry
needtypemanualindex int This column stores index of the need type. It allow users to order special need type list.
