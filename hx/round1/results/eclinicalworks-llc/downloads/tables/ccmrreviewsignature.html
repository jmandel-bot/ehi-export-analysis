Table ccmrreviewsignature
Description This table stores information about signatures related to CCMR review records, including the signature itself, user details, and other relevant data.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table (int, e.g., 1, 2, 3).
reviewid int identifier for the associated review record (int, e.g., 101, 102, 103).
ccmrid int identifier for the associated CCMR record (int, e.g., 201, 202, 203).
userid int identifier for the user who provided the signature (int, e.g., 301, 302, 303).
programType int identifier for the type of program associated with the signature (int, e.g., 1, 2, 3).
SignTakenTime datetime The date and time when the signature was taken (datetime, e.g., '2022-01-01 12:34:56').
signPadid tinyint identifier for the sign pad used to capture the signature (tinyint, e.g., 1, 2, 3).
signatureType tinyint identifier for the type of signature (tinyint, e.g., 1 for electronic, 2 for digital).
signUserType varchar The role or type of the user who provided the signature (varchar, e.g., 'Manager', 'Employee').
Signature varbinary The binary data representing the signature (blob).
fileName varchar The name of the file where the signature is stored (varchar, e.g., 'signature_123.png').
ftpDirPath varchar The directory path on the FTP server where the signature file is stored (varchar, e.g., '/signatures/review_101/').
signPresentIn tinyint identifier for the location where the signature is present (tinyint, e.g., 1 for local, 2 for remote).
