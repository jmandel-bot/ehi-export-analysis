Table phm_decline_reasons
Description This table stores the enrollment decline reasons for different PHM programs in the system.
Column DataType Description
id (Primary Key) int Contains auto incremented id, datatype is INT. This column stores integer value which could grow with number of CCM decline reasons added. It is AUTO_INCREMENT column and has PRIMARY KEY index
acoprograms_id int Contains the acoprograms_id which refers to id column of ptacoprograms table and data type is INT. This field will always have a value hence defined as NOT NULL.
isActive tinyint Containsthe flag specifying if the decline reason is active or not and data type is TINYINT. This column stores integer value hence integer is selected. 1:Active 0: InActive. By default, the disenroll reason is considered active.
reason_name varchar Contains the name of the decline reason and the data type isVARCHAR(1000).Since the length of decline reasons would not exceed 255 characters it is selected as VARCHAR(255).This field will always have a value hence defined as NOT NULL.
deleteFlag tinyint Flag indicates reason has been deleted or not, datatype is TINYINT. This column stores integer value hence integer is selected. Datatype can be changed to tinyint - 0:NotDeleted 1: Deleted. By default reason must not be in delete status hence default 0 is chosen.
modifiedBy int Contains the user id who has modified the record last and datatype is INT. It refers the 'UID' of user table. This field will always have a value hence defined as NOT NULL.
modifiedDatetime datetime Contains date time when the record was last modified and datatype is DATETIME. This column store datetime hence time stamp is selected.
