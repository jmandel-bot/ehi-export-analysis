EHI Export Schema
Table allergy_maintenance_dose_shot_schedule
Description This table stores information about the allergy maintenance dose shot schedules, including details about the queue, template, bottle, concentration, and shot.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the allergy maintenance dose shot schedule table. Example: 1, 2, 3.
queueid int identifier for the antigen recipe queue associated with the shot schedule. Foreign key relation to `antigen_recipe_queue` table (`id` column). Example: 101, 102, 103.
templateid int identifier for the shot schedule template associated with the shot schedule. Foreign key relation to `shot_schedule_template` table (`id` column). Example: 201, 202, 203.
bottleid int identifier for the antigen bottle associated with the shot schedule. Foreign key relation to `antigen_bottle` table (`bottle_id` column). Example: 301, 302, 303.
concentrationid int identifier for the antigen concentration for the patient associated with the shot schedule. Foreign key relation to `antigen_concentration_patient` table (`CONCENTRATION_id` column). Example: 401, 402, 403.
shotid int identifier for the shot schedule detail associated with the shot schedule. Foreign key relation to `shot_schedule_dtl` table (`id` column). Example: 501, 502, 503.
createdby int identifier for the user who created the record. Foreign key relation to `users` table (`UID` column). Example: 1001, 1002, 1003.
createdon datetime time stamp when the record was created. Example: '2021-09-01 12:34:56'.
modifiedby int identifier for the user who last modified the record. Foreign key relation to `users` table (`UID` column). Example: 1001, 1002, 1003. Default value is NULL.
modifiedon datetime Date and time when the record was last modified. Example: '2021-09-02 14:56:23'. Default value is NULL.
deleteflag tinyint A flag indicating whether the record has been deleted or not. Example: 0 (not deleted), 1 (deleted). Default value is '0'.
