EHI Export Schema
Table multisatisfiedalert
Description This table stores the configuration for multi-satisfied alerts.
Column DataType Description
id (Primary Key) int Unique identifier for each multisatisfiedalert record. Example: 1, 2, 3, etc.
alertname varchar This column stores the name of the alerts (free text) added by the end user hence varchar is chosen.
alertdesc varchar This column is not used anymore.
agelower int This column stores the lower value configured for the age based alert hence int chosen. The unit is years.
ageupper int This column stores the upper value configured for the age based alert hence int chosen. The unit is years.
gender char This column stores the gender of the alert criteria. The possible values are B - Both, M- Male or F - Female.
satisby char This column is not used anymore.
deleteFlag smallint This column stores 0 or 1. 1 - when alert is marked as deleted or 0 - Otherwise.
