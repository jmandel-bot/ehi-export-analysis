EHI Export Schema
Table visionrx_fittingcomments
Description This table stores information about fitting comments related to vision prescriptions, including centration movement, toric orientation, and Nafi pattern.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer value. It uniquely identifies each record in the table. Example: 1, 2, 3, etc.
VisionRxid int Foreign key that refers to the `id` column in the `visionrx` table. It represents the unique identifier for the associated vision prescription. Example: 101, 102, 103, etc.
Encounterid int identifier for the encounter related to the vision prescription. It refers to enc table `encounterid` column. Example: 1001, 1002, 1003.
Centration_Movement text This column contains a text description of the centration movement for the vision prescription. Example: "Minimal", "Moderate", "Significant".
Toric_Orientation text This column contains a text description of the toric orientation for the vision prescription. Example: "Horizontal", "Vertical", "Oblique".
Nafi_Pattern text This column is a text description of the Nafi pattern for the vision prescription. Example: "Regular", "Irregular", "Mixed".
Fitting_Comments text This column contains additional text comments related to the fitting of the vision prescription. Example: "Patient needs further adjustments", "Fitting is perfect", "Consider different lens type".
