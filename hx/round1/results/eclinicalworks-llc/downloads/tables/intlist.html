EHI Export Schema
Table intlist
Description This table stores information about various interfaces, including their addresses, contact information, and configuration settings.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key of this column.
name varchar This column stores the free text name of the new interface company.
deleteflag smallint This column stores value 0 / 1 indicating if the Interface Company is deleted, this is soft delete. 0 - Not Deleted , 1 -Deleted.
intid int This column is currently not in use, but may be used in future.
intaddress varchar This column stores the free text address of the interface company.
intaddress2 varchar This column stores the free text address of the interface company.
intcity varchar This column stores the free text address city of the interface company.
intstate varchar This column stores the free text address state of the interface company.
intzip varchar This column stores the free text address zip code of the interface company.
intphone varchar This column stores the telephone number of the interface company.
intfax varchar This column stores the fax number of the interface company.
intemail varchar This column stores the email id of the interface company.
interfaceconfigured int This column is not in use currently, but may be used in future.
sendoutboundresult int This column is not in use currently, but may be used in future.
ResultIntid int This column stores the configured interface ID for result processing.
intcompanyType tinyint This column is not in use currently, but may be used in future.
inHouseInt int This column is not in use currently, but may be used in future.
intcode varchar This column stores the code used for part of the key to encryption/decryption for data transmission to hubs.
superhubInterfaceid int This column stores the master interface ID for this vendor at eClinicalWorks.
isAUCVendor tinyint This column stores single bit information, 1 or 0, indicating whether or not the vendor is an AUC vendor.
