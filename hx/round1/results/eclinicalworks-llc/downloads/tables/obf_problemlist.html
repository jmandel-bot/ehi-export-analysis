EHI Export Schema
Table obf_problemlist
Description This table stores information about a list of problems, their plans, and related details such as status, lock, and user interactions.
Column DataType Description
id (Primary Key) int This column stored primary key as numeric value of the table
pregid int This column stored  pregnancy id as numeric value from the obf_pregnancy table
problem varchar This column stored problems as varchar value.
plan text This column stored time stamp of user while documented the problem
createdBy varchar This column stored user name as varchar value
createdDt datetime This column stored date and time as varchar value.
modifiedBy varchar The username of the person who last modified the record. Example: "jdoe", "asmith".
modifiedDt datetime The date and time when the record was last modified. Example: "2021-09-02 14:56:23".
initials varchar This column stored initials value of problem section.
comments text This column stored comments value of problem list section
plstatus varchar This column stored status of the problem as varchar value.
pllock tinyint not in use
displayindex int This column stored display index as numeric value.
userid int This column stored userid as numeric value.
