Table prisma_document_reference_author_details
Description This table stores information about the authors associated with document references in the Prisma database.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
docRefId int The identifier of the document reference this author detail is associated with. Foreign key relation: `prisma_document_reference_details(id)`. Example: 101, 102, 103.
autRefId int The identifier of the author (practitioner) associated with this document reference. Foreign key relation: `prisma_practitioner_details(ID)`. Example: 201, 202, 203. This column can have NULL values.
authorDisplay varchar A string representing the display name of the author. Example: "Dr. John Doe", "Jane Smith, PhD". This column can have NULL values.
authorType varchar A string representing the type or role of the author. Example: "Primary", "Contributor", "Reviewer". This column can have NULL values.
