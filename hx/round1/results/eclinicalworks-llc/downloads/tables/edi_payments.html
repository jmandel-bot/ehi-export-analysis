EHI Export Schema
Table edi_payments
Description This table stores information about electronic data interchange (EDI) payments made for invoices.payments include patient payment and insurance payment.
Column DataType Description
id (Primary Key) int Unique identifier for each payment record. Example: 1, 2, 3.
Invoiceid int identifier for the invoice associated with the payment. This can be a foreign key referring to the `id` column in the `edi_invoices` table. Example: 101, 102, 103.
Date datetime The date when the payment was made. Example: '2021-09-01', '2021-09-02', '2021-09-03'.
PaymentType varchar The type of payment made, such as cash, check, or credit card. Example: 'Cash', 'Check', 'Credit Card'.
CheckNo varchar The check number, if the payment was made by check. Example: '123456', '123457', '123458'.
Payment decimal The amount of the payment made. Example: '100.00', '200.00', '300.00'.
Adjustment decimal The amount of any adjustments made to the payment, such as discounts or refunds. Example: '0.00', '10.00', '20.00'.
PayerType smallint A numeric code representing the type of payer, such as an insurance company or an individual. Example: 1, 2, 3.
Insid int identifier for the insurance company associated with the payment, if applicable. This can be a foreign key referring to the `id` column in the `edi_inv_insurance` table. Example: 1001, 1002, 1003.
Notes text Any additional notes or comments about the payment. Example: 'Payment received on time', 'Partial payment', 'Refund issued'.
DeleteFlag smallint A flag indicating whether the payment record has been deleted (1) or not (0). Example: 0, 1.
vmid varchar A unique identifier for the payment record in the form of a string. Example: 'vmid_001', 'vmid_002', 'vmid_003'.
