Table ptorders_ack
Description This table stores the acknowledgment information for each order, including the action type, who performed the action, and when the action was performed.
Column DataType Description
id (Primary Key) int This is the primary key for the table, and it is an auto-incrementing integer. It uniquely identifies each acknowledgment entry in the table. Example: 1, 2, 3, etc.
orderid int This is a foreign key that refers to the `id` column in the `orders` table. It represents the order that the acknowledgment is associated with. Example: 101, 102, 103, etc.
actionType varchar This is a varchar column that stores the type of action performed on the order. Example: 'approve', 'reject', 'cancel', etc.
performedBy int This is an integer column that stores the id of the user who performed the action. It can be a foreign key referring to the `id` column in the `users` table. Example: 1001, 1002, 1003, etc.
performedDateTime datetime This is a datetime column that stores the date and time when the action was performed. Example: '2021-09-01 12:34:56', '2021-09-02 14:23:12', etc.
deleteFlag tinyint This is a tinyint column that stores a boolean value (0 or 1) to indicate whether the acknowledgment entry has been deleted or not. Example: 0 (not deleted), 1 (deleted).
