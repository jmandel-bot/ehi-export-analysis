Table ccmr_assigngoal
Description This table stores information about assigned goals for specific encounters, including the start date, due date, and the person assigned to the goal.
Column DataType Description
Id (Primary Key) int Unique identifier for each assigned goal record. Example: 1, 2, 3.
encounterId int The identifier for the encounter to which the goal is assigned. This can be a foreign key referring to the `Id` column in the `encounters` table. Example: 101, 102, 103.
goalId int This column refers to structdatadetail.id.
startDate datetime The date and time when the goal assignment starts. Example: '2021-09-01 12:00:00', '2021-09-02 14:30:00'.
dueDate datetime The date when the goal assignment is due. Example: '2021-09-30', '2021-10-15'.
assignTo int The identifier for the person to whom the goal is assigned. This can be a foreign key referring to the `uId` column in the `users` table. Example: 1, 2, 3.
isPatient tinyint A flag indicating whether the goal is assigned to a patient (1) or not (0). Example: 0, 1.
delflag int A flag indicating whether the assigned goal record is deleted (1) or not (0). Example: 0, 1.
