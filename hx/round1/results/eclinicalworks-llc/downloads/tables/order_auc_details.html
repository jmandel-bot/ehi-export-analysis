EHI Export Schema
Table order_auc_details
Description This table stores the details of orders related to Appropriate Use Criteria (AUC) for advanced diagnostic imaging services.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table (e.g., 1, 2, 3).
reportid int Identifier for the associated report in the labdata table. Foreign key relation: labdata.Reportid.
assessmentid int Identifier for the associated assessment in the items table. Foreign key relation: items.itemid.
rwsid varchar This column stores session id of rest call initiated with AUC vendor. It is unique identifier for the RWS (Radiology Workflow System) (e.g., RWS12345).
transactionid varchar This column stores transation id, it is the id maintain by vendor for each AUC vendor request.
aucActionStatus tinyint This column stores auc action status Below are the values for each status: AUC_ACTION_PENDING = 0; AUC_ACTION_COMPLETED = 1; AUC_ACTION_FAILED = 2; AUC_ACTION_SYMPTOMS_RECIEVED = 3; AUC_ACTION_SYMPTOMS_ENTERED = 4; AUC_ACTION_RECOMMENDATION_RECEIVED = 5; AUC_ACTION_RECOMMENDATION_ADDRESSED = 6
aucOrderStatus tinyint This column stores auc order status. Below are the values for auc order status: AUC_ORDER_ORDERED = 1; AUC_ORDER_REPLACED = 2; AUC_ORDER_CANCELLED = 3; AUC_RECOMMENDATION_REJECTED = 4
gCode varchar This column stores gcode of AUC vendor.
hcpcsModifier varchar This column stores hcpc modifier details when the recommended order placed.
aucScore float This column stores auc score details.
aucSummary varchar This column stores auc summary details which later used to display read only screen.
expireDateTime datetime Date and time when the AUC expires (e.g., 2022-12-31 23:59:59).
createdDateTime datetime Date and time when the record was created (e.g., 2021-01-01 12:34:56).
updatedDateTime datetime Date and time when the record was last updated (e.g., 2021-01-02 14:56:78).
deleteflag tinyint Boolean flag (0 or 1) indicating whether the record has been deleted, where 0 means not deleted and 1 means deleted.
acceptedRejectedBy int Identifier for the user who accepted or rejected the AUC, which can be NULL if not yet accepted or rejected (e.g., 123).
acceptedRejectedTime datetime Date and time when the AUC was accepted or rejected, which can be NULL if not yet accepted or rejected (e.g., 2021-01-03 16:12:34).
encounterid int Identifier for the associated encounter in the enc table, which can be NULL if not associated with an encounter. Foreign key relation: enc.encounterid.
