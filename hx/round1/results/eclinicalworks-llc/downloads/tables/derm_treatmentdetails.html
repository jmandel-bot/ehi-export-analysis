EHI Export Schema
Table derm_treatmentdetails
Description This table stores treatment information for observation on body regions.
Column DataType Description
treatmentdetailid (Primary Key) int This is auto-incremented primary key in This column . This column stores unique id for each treatment order placed by user.
observationdetailid int This column stores observationdetailid which refers to observationdetailid column of derm_observationdetails table.
treatmenttype varchar This column stores type of treatment order. Type can be 'labs' for Lab, 'di' for Diagnostic Imaging, 'procedures' for Procedure, 'rx' for Medication, 'ti' for Therapeutic Injection/Immunization.
treatmentmasterid int This column is not in use.
itemid int This column stores itemid of treatment order which refers to itemid column of items table.
isdeleted tinyint This column stores value 0 / 1 indicating this treatment order is deleted. This is soft delete. 0 - Not Deleted , 1 -Deleted. It's default value is 0.
assessmentid int This column stores assessment id of assessment associated with treatment order. This column refers to itemid column of items table for keyName 'Assessments'.
assessmentcode varchar This is a varchar(255) column that stores the assessment code related to the treatment detail. Example: 'ACNE01', 'PSOR02', 'ECZ03', etc.
orderdate datetime This is a date column that stores the date when the treatment was ordered. Example: '2021-01-01', '2021-02-15', '2021-03-30', etc.
rxid int This column stores medication id for treatment type 'rx', imminization id for immunization/therapeutic injection. This column refers to oldrxid column of oldrxmain table for medication, and immunizationid for immunization/therapeutic injection.
