EHI Export Schema
Table edi_forms
Description This tables store CPT Form Information which is included ANSI Professional Claim in Loop 2440
Column DataType Description
id (Primary Key) int This is the primary key column that uniquely identifies each record in the table. It is an integer value that is automatically incremented with each new record. Example: 1, 2, 3, etc.
FormName varchar This column stores the name of the EDI form as a variable character string with a maximum length of 255 characters. Example: "Purchase Order Form", "Invoice Form", etc.
FormIdentificationcode char This column stores a 3-character identification code for the EDI form. Example: "POF" for Purchase Order Form, "IVF" for Invoice Form, etc.
FormIdentifier varchar This column stores a unique identifier for the EDI form as a variable character string with a maximum length of 30 characters. Example: "POF-123", "IVF-456", etc.
deleteFlag tinyint This column stores a tiny integer value (1 or 0) to indicate whether the record has been deleted or not. A value of '1' means the record is deleted, and '0' means the record is active. Example: 0, 1.
