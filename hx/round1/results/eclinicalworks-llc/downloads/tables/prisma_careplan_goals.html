Table prisma_careplan_goals
Description This table stores the goals associated with each care plan in the Prisma healthcare system.
Column DataType Description
ID (Primary Key) int Unique identifier for each care plan goal. Example: 1, 2, 3, etc.
CAREPLAN_ID int The identifier of the care plan to which the goal is associated. This is a foreign key that refers to the `prisma_careplan_details` table's `ID` column. Example: 101, 102, 103, etc.
GOAL_ID int The identifier of the specific goal associated with the care plan. Example: 1001, 1002, 1003, etc.
