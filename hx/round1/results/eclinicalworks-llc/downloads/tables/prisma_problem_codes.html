Table prisma_problem_codes
Description This table stores information about problem codes, their code systems, and their relationships with problem list details.
Column DataType Description
id (Primary Key) bigint Unique identifier for each record in the table. Example: 1, 2, 3.
problemId bigint Identifier for the associated problem in the `prisma_problemlist_details` table. Example: 101, 102, 103. Foreign key relation: `prisma_problemlist_details.id`.
codeSystem varchar The code system used for the problem code. Example: "ICD-10", "SNOMED-CT", "LOINC".
code varchar The specific code within the code system that represents the problem. Example: "A01.0", "B02.1", "C03.2".
codeName varchar The name or description of the problem code. Example: "Typhoid fever", "Varicella meningitis", "Malignant neoplasm of colon".
sourcePriority int The priority of the source providing the problem code. Example: 1, 2, 3.
codeSystemPriority int The priority of the code system used for the problem code. Example: 1, 2, 3.
duplicatedBy int Identifier for the record that duplicates this record. A value of -1 indicates no duplication. (e.g., -1, 5, 10)
isException tinyint A flag indicating if the problem code is an exception (1) or not (0). Example: 0, 1.
exceptionMessage varchar Comma separated ids, each id pointing to a message in table "prisma_ccd_preprocessor_error_messages". Empty if there is no exception.
