EHI Export Schema
Table ccmr_data
Description This table stores the data documented for patient when the problem is closed in the care plan.
Column DataType Description
id (Primary Key) int The column stores an id. This is an auto incremental primary key of This column .
ccmrId int This column stores identifier ccmrid. This column refers to ccmr_careplan table and column ccmrID.
type varchar This saves the type of the question. This is pre-defined and is a varchar field.
ItemId int This column stores question id. This column refers to items table and column itemid.
choice text This saves the documented answer and it is text datatype.
Lock_Ind int This column stores value 0 / 1 indicating this is locked. This is soft delete. 0 - unlocked , 1 -locked
DelStat int This column stores value 0 / 1 indicating this is deleted. This is soft delete. 0 - Not Deleted , 1 -Deleted
Uname varchar Username of the person who updated the record, stored as a string. Example: "jdoe", "asmith", "mjackson".
Upd_Date datetime time stamp of the last update made to the record. Example: "2021-08-01 12:34:56".
Comments text Text field for any additional comments related to the CCMR data. Example: "Customer was satisfied with the resolution", "Issue escalated to management".
atriskid int This column refers to ccmr_atrisk.AtRiskID.
