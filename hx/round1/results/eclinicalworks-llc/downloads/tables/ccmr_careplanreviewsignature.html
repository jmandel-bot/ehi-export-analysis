EHI Export Schema
Table ccmr_careplanreviewsignature
Description This table stores the information about additional care plan review signature captured on care plan review , including the signee, signature time, and other related details.
Column DataType Description
id (Primary Key) int Unique identifier for each care plan review signature. (Example: 1, 2, 3)
reviewId int This column stores the reviewid which referes from id of ccmrproblemreview table, which indeicates this signature belongs this review
signee varchar Name of the person who signed the care plan review. (Example: "John Doe", "Jane Smith")
signatureTime datetime time stamp of when the signature was made. (Example: '2021-09-15 14:30:00')
deleteFlag tinyint A boolean flag indicating if the signature has been deleted (1) or not (0). (Example: 0, 1)
signPadId tinyint Identifier for the sign pad used to capture the signature. (Example: 1, 2, 3)
signature varbinary Binary data representing the actual signature image. (Example: Binary data of a PNG or JPEG image)
fileName varchar Name of the file where the signature image is stored. (Example: "signature_123.png", "signature_456.jpg")
ftpDirPath varchar Directory path where the signature image file is stored on the FTP server. (Example: "/signatures/careplanreview/")
signPresentIn tinyint This column indicates that whether signature is present on FTP servver or in table itself 0- in table 1- FTP server
