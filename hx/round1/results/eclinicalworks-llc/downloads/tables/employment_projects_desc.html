Table employment_projects_desc
Description This table stores information about various employment projects, including their names, parent projects, and associated employers.
Column DataType Description
id (Primary Key) int This is the primary key column, which uniquely identifies each record in the table. It is an integer value that is automatically incremented as new records are added. Example: 1, 2, 3, etc.
Project_NAME varchar This column stores the name of the employment project as a variable character string with a maximum length of 60 characters. Examples include: "Project A", "Project B", "Project C", etc.
Parentid int This column stores the id of the parent project, if any, as an integer value. It is a foreign key that refers to the "id" column in the same "employment_projects_desc" table. If there is no parent project, the value will be NULL. Examples include: 1, 2, 3, etc., or NULL.
DeleteFlag int This column stores a flag indicating whether the record has been deleted or not. It is an integer value with a maximum length of 2 digits. A value of 0 indicates that the record is not deleted, while a value of 1 indicates that the record is deleted. Example: 0 or 1.
Employerid int This column stores the id of the employer associated with the project as an integer value.
