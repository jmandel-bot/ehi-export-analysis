EHI Export Schema
Table eicr_transaction_trigger_codes
Description This table stores the trigger codes associated with each eICR transaction log entry, along with the code system identifier.
Column DataType Description
id (Primary Key) int This is the primary key for the table, an auto-incrementing integer that uniquely identifies each record. Example: 1, 2, 3, etc.
eicr_transaction_log_id int This is a foreign key column that refers to the `id` column in the `eicr_transaction_logs` table. It represents the eICR transaction log entry associated with the trigger code. Example: 1, 2, 3, etc.
trigger_code varchar This is a varchar column that stores the trigger code associated with the eICR transaction log entry. Example: 'TRG123', 'TRG456', 'TRG789', etc.
code_system_id tinyint This is a tinyint column that stores the identifier for the code system used for the trigger code. Example: 1, 2, 3, etc.
