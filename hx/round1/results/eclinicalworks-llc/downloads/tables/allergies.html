EHI Export Schema
Table allergies
Description This table stores information about patients' allergies, including the drug causing the allergy, the type of allergy, and its status.
Column DataType Description
patientid int This column stores the id of a patient whose allergies are being documented. This column refers to the UID column in the user table.
drug varchar This column stores the name of the allergen causing the allergy to the patient.
allergy varchar This column stores the reaction for the corresponding allergy.
encounterid int This column stores the id of an encounter for which allergies are being documented. This column refers to the encounterid column in the enc table.
itemid int This column stores the itemid of an allergen causing the allergy to the patient. This column refers to the itemid column in the items table.
displayIndex int This column stores the order in which allergies are displayed in the user interface.
SMDamConceptid int This column stores the order in which allergies are displayed in the user interface.
SMDamConceptidDesc varchar This column is not being used.
ndc_code varchar This column stores the National Drug Code (NDC) for an allergen causing allergy to the patient.
allergytype smallint This column stores the type of allergy. Example: Allergy, Side Effects, Contraindication, Lack of Therapeutic Effect.
type char This column stores char values to indicate whether allergies are structured/coded or unstructured/uncoded. 1 - structured/coded allergy, 2 - unstructured/uncoded allergy.
status varchar This column stores information whether the patient is still allergic to an allergen. Two statuses are stored in this column 'Active' or 'Inactive'. By default, it is set to 'Active'.
criticality smallint This column stores criticality of allergy (e.g., High, Low, Medium).
onsetdate varchar This column stores the date when allergic reaction to a allergen started for a patient. Date is stored in DD/MM/YYYY format.
