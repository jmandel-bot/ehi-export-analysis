EHI Export Schema
Table emrereferralattachments
Description This table stores data/attachments received with incoming referrals.
Column DataType Description
Id (Primary Key) int Unique identifier for each attachment record. Example: 1, 2, 3.
ReferralReqId int This column stores id of am incoming referral. This column refer to refreqid column in referral table.
progressnotes text This column stores contents of progress note(s) received through an incoming referral in XML format.
medicalsummary text This column stores medical summary received through incoming referral in XML format.
labreports text This column stores content of lab reports received with incoming referral in XML format.
xrays text This column stores content of xrays received with incoming referral.
specialtyforms text This column stores specialty forms received with incoming referral in XML format.
ccr text This column stores Continuity of Care Record (CCR) received incoming referral in XML format.
reqAPUID int This column stores APU id of a sender of an incoming referral.
