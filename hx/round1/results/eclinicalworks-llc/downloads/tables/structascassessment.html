EHI Export Schema
Table structascassessment
Description This table is used for storing nursing assessment data for patients.
Column DataType Description
id int This is an auto incremental primary key of This column .
entryid int used for storing entryid,which maps to id from structascassessment_entry table.
catid int used for storing category id,which maps to itemid from items table.
itemid int used for storing itemid,which maps to itemid from items table.
detailid int used to store detailid ,which maps data from structdatadetail with id
VALUE text used for storing varchar type value.
valueid varchar used for storing value id which stores data for single-select,multi-select and boolean options.
notes text This column stores ASC assessment notes.
displayIndex tinyint Its value will always be 1, as we are not using it.
chartdisplay tinyint Its value will be always 1, as we are not using this field.
valueScore int It is used to store score for configured questions.
