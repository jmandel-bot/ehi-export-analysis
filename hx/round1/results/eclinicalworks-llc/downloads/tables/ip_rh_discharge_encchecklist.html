Tableip_rh_discharge_encchecklist
DescriptionThis table stores the encounter centric discharge checklist selected for the patient.
ColumnDataTypeDescription
id(primaryKey) int Stores the auto increment id for this table.
encounterId int Stores the encounter id FK from Enc table
categoryItemMapId int Stores the ID map from the ip_rh_discharge_checklist_map table
isChecked tinyint Stores if the item is selected for this encounter 1 - Checked, 0- Not checked
deleteFlag tinyint Soft delete flag for this entry, 1 - Deleted, 0 - Not Deleted
performedBy int User who made the selection for this cateogry, refers to uid from Users table
performedDate datetime Time stamp of when this action was performed.
