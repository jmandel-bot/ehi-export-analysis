EHI Export Schema
Table eptstmtsautopay
Description This table stores billing information to do auto-payment agianst a patient's statement thourgh the card-on-file feature.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table (int).
patientid int This column stores the patient id associtead with the payment (int).
AccountNo varchar Account number of the patient.
origencounterid int Original encounter for the claim that is recreated or split.
encdoctorid int Unique identifier for the doctor associated with the encounter (int).
encdate datetime Date of the encounter (date).
encfacilityid int Unique identifier for the facility where the encounter took place (int).
encfacilityname varchar Name of the facility where the encounter took place (varchar, 255 characters).
encfacilityphone varchar Phone number of the facility where the encounter took place (varchar, 20 characters).
encfacBillingAddressLine1 varchar First line of the facility's billing address (varchar, 30 characters).
encfacBillingAddressLine2 varchar Second line of the facility's billing address (varchar, 30 characters).
encfacBillingCity varchar City of the facility's billing address (varchar, 20 characters).
encfacBillingState varchar State of the facility's billing address (varchar, 5 characters).
encfacBillingZip varchar Zip code of the facility's billing address (varchar, 10 characters).
cardnum varchar Last four digits of the credit card number used for the payment (varchar, 10 characters).
nameoncard varchar Name on the credit card used for the payment (varchar, 255 characters).
agreementid int Agreement associated with the card on file.
transid varchar Unique identifier for the transaction (varchar, 60 characters).
ufname varchar First name of the patient.
ulname varchar Last name of the patient.
prefix varchar Prefix for the patient (e.g., Mr., Mrs., Dr.).
suffix varchar Suffix for the patient (e.g., Jr., Sr., III) (varchar, 10 characters).
uemail varchar Email for the patient.
upPhone varchar Primary phone number for the patient.
umobileno varchar Mobile phone number for the patient.
grlname varchar Last name of the guarantor (varchar, 60 characters).
grfname varchar First name of the guarantor (varchar, 60 characters).
grphone varchar Guarantor's phone number (varchar, 30 characters).
grmobile varchar Guarantor's mobile phone number (varchar, 15 characters).
gremail varchar Guarantor's email address (varchar, 255 characters).
emailserver varchar Email server used for sending notifications (varchar, 255 characters).
totalBalance decimal Total balance of related claims for this encounter.
chargeBalance decimal Charge balance for all the claims related to this encounter.
AmountPaid decimal Amount paid for this encounter.
PaymentStatus int Status of the payment (int, 2 digits; e.g., 0 for pending, 1 for successful).
paymentid int Payment id for the encounter that is made through card on file.
LANGUAGE varchar Patients preferred language.
isprocessing int Flag to indicate if the payment is processed successfully (1-success, 2 - failure, 3- response error from payment vendor, and 4- empty response from payment vendor).
inserttime datetime A time stamp when the record was inserted into the table (time stamp).
deleteflag int Flag to indicate if the record has been deleted (int, 2 digits; e.g., 0 for not deleted, 1 for deleted).
cofpayprocessingdate datetime Date and time when the payment was processed or going to process in future.
