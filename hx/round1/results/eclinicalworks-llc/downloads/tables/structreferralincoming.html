EHI Export Schema
Table structreferralincoming
Description This table stores structured data documented under the 'Structured Data' tab for incoming referrals.
Column DataType Description
referralid int This column stores id of an incoming referral for which Structured Data is being documented. This column refers to referralid column in referral table.
catid int This column is not being used for This column . Value for this column is always set to default.
itemid int This column is not being used for This column . Value for this column is always set to default.
detailid int This column stores id of an structured item which is being documented by end user. This column refers to id column in structdatadetail table.
value text This column stores option selected by end user for a to document a structured item for incoming referral. This column is declared as TEXT datatype because values can be created by end users and could be of any length.
valueid varchar This column stores id of an option selected by end user for corresponding record in value column. This column refers to optionid column in structdataoptions table.
notes text This column stores free text notes entered by end user for each of structured item. This column is declared as TEXT because user may document notes of any possible length for a structured item.
displayIndex tinyint This column is not being used.
chartdisplay tinyint This column is not being used.
