EHI Export Schema
Table tblwebencdata
Description This table stores information about Web Encounters, including the message ID, reply status, transfer status, and the last modified date and time.
Column DataType Description
encounterid (Primary Key) int This is an integer column that serves as the primary key for the table. It uniquely identifies each encounter. Example: 1, 2, 3, etc.
Msgid int This is an integer column that stores the message id associated with the encounter. Example: 101, 102, 103, etc.
ReplyStatus varchar This is a varchar column that stores the reply status of the encounter. It can have values like 'open', 'closed', or other custom statuses. Example: 'open', 'closed'.
TransferStatus varchar This is a varchar column that stores the transfer status of the encounter. It can have values like 'pending', 'completed', or other custom statuses. Example: 'pending', 'completed'.
LastModified datetime This is a datetime column that stores the date and time when the encounter was last modified. Example: '2021-09-15 12:34:56'.
