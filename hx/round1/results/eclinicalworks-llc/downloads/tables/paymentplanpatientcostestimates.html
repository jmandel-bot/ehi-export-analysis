Table paymentplanpatientcostestimates
Description This table stores information about payment plan estimates for patients, including the patient's balance and insurance balance.
Column DataType Description
id (Primary Key) int This is the primary key for the table, an integer that uniquely identifies each record. Example: 1, 2, 3, etc.
planid int This is a foreign key column, an integer that refers to the id of the payment plan in another table. Example: 101, 102, 103, etc.
estid int This is a foreign key column, an integer that refers to the id of the cost estimate in another table. Example: 201, 202, 203, etc.
deleteflag tinyint This is a tiny integer column that indicates whether the record has been deleted or not. A value of 0 means the record is not deleted, and a value of 1 means the record is deleted. Example: 0 or 1.
ptbal decimal This is a decimal column that stores the patient's balance for the payment plan. It has a precision of 10 digits and a scale of 2 decimal places. Example: 150.25, 200.00, 300.75, etc.
insbal decimal This is a decimal column that stores the insurance balance for the payment plan. It has a precision of 10 digits and a scale of 2 decimal places. Example: 100.50, 250.00, 400.25, etc.
