EHI Export Schema
Table derm_imagereviewstatus
Description This table stores image review status, date and time of when images were reviewed/unreviewed.
Column DataType Description
id (Primary Key) int Contains auto incremented id, datatype is INT. This column stores integer value which could grow with an encounter is locked. It is AUTO_INCREMENT column and has PRIMARY KEY index
encounterid int It stores an encounter Id of locked encounter. It refers to column encounterid of table enc.
userid int It stores user Id of logged in user who is locking the encounter. It refers to column uid of table users.
reviewstatus tinyint It stores images' status. It may have any of three values (1,2,3) .1 - refers to user have click on "Yes" button of confirmation popup while locking an encounter. 2 - refers to images are reviewed when locking the encounter. 3 - refers to images are unreviewed when unlocking the encounter.
entryon datetime It stored date and time of when user has confirmed to review images by clicking on "Yes" button of confirmation popup while locking an encounter.
statuschangeon datetime It stored date and time of when images are reviewed while locking an encounter.
