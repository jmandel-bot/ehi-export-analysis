EHI Export Schema
Table visitcodes
Description This table stores information about the visit type codes, which define the type of the patients' visits.
Column DataType Description
CodeId (Primary Key) int The column stores an id of the Visit Type Code. This is an auto incremented primary key of This column .
Name varchar This column stores a Name of visit type code.
Description varchar This column stores description of the visit type code.
ChartTitle varchar This column stores chart title of the visit type code.
color varchar This column stores a color for the visit type. This color is the outline color for the appointment.
phyFlag varchar Flag related to the physician (e.g., 'Y', 'N').
visitType int This column stores visit type from the Pre-defined options to determine the format and content of the Progress Notes to be used for the corresponding appointment.
obGyn smallint This column stores value 0/1 to include the OB/GYN History section in the Progress Notes for this visit type. 0 - Not Include , 1 - Include
PhysicalTherapy smallint This column stores value 0/1 to include the Physical Therapy Assessment section in the Progress Notes for this visit type. 0 - Not Include , 1 - Include
ReqClaim smallint This column stores value 0/1 to indicate the visit is billable. 0 - Not Billable , 1 - Billable.
ReqCopay smallint This column stores value 0/1 to indicate the co-payments is required for the appointment with this Visit Type. 0 - Not Required , 1 - Required.
pregVisit smallint This column stores value 0/1 to indicate any appointment with this Visit Type as a pregnancy visit. 0 - Non pregnancy visit, 1 - Pregnancy visit.
Active int This column stores value 0/1 to indicate whether the visit type code is active or inactive. 0 - Inactive, 1 - Active.
ProgDisplay int This column stores value 0/1 to indicate whether to display Appointments with this visit type in 'Progress Notes' visits Drop Down. 0 - Do not display, 1 - Display.
OrthoVisit smallint This column stores value 0/1 to indicate any appointment with this Visit Type as a orthopedic visit. 0 - Non orthopedic visit, 1 - Orthopedic visit.
VisionVisit smallint This column stores value 0/1 to include the Vision Examination section in the Progress Notes for encounters using this visit type. 0 - Not Include , 1 - Include.
webVisit tinyint This column stores value 0/1 indicating synchronize this visit type with the Patient Portal. 0 - Not synchronize , 1 - Synchronize.
DentalVisit smallint This column stores value 0/1 to indicate any appointment with this Visit Type as a dental visit. 0 - Non dental visit, 1 - Dental visit.
CarePlanVisit smallint This column stores value 0/1 to include appointments with this visit type in a care plan. 0 - Not Include , 1 - Include.
segVisit tinyint This column stores value 0/1 to indicate any appointment with this Visit Type as a patient housing. 0 - Non patient housing visit, 1 - Patient housing visit.
vmsgspanishtext text Column stores voice message in Spanish to be played for the visit type.
txtspanishtext text Column stores message in Spanish to be sent via text for the visit type.
vmsgtemplatetext text Column stores voice message in English to be played for the visit type.
txttemplatetext text Column stores message in English to be sent via text for the visit type.
MUExclude tinyint This column stores value 0/1 to exclude any encounters using this visit type from Meaningful Use reporting. 0 - Not Exclude, 1 - Exclude.
apptMachineEnglish text Stores Appointment machine template(English) set by end user.
apptMachineSpanish text Stores Appointment machine template in Spanish which is set by end user.
exclude tinyint Flag indicating if the visit is excluded (e.g., 0 for No, 1 for Yes).
NuclearTestVisit smallint This column stores value 0/1 to enable worksheets that help specialists when documenting a patient visit. 0 - Disable, 1 - Enable.
CareMgmtVisit smallint This column stores value 0/1 to to enable a Care Management visit for this visit type. 0 - Disable, 1 - Enable.
AllowCurRxReconcilationVisit smallint This column stores value 0/1 to prevent medication reconciliation from taking place for encounters using this visit type. 0 - Not reconcile, 1 - Reconcile.
CCMRVisit smallint This column stores value 0/1 to indicate any appointment with this Visit Type as a CCMR visit. 0 - Non CCMR visit, 1 - CCMR visit.
nsexclude tinyint Stores values 0/1 to indicate whether to exclude this visit code from No Show reminders.
ReferralVisit smallint This column stores value 0/1 to indicate whether require referrals for encounters using this visit type. 0 - Not Required, 1 - Required.
PhysicalTherapyVisit smallint This column stores value 0/1 to indicate any appointment with this Visit Type as a Physical Therapy visit. 0 - Non Physical Therapy visit, 1 - Physical Therapy visit.
telemedVisit smallint This column stores value 0/1 to indicate any appointment with this Visit Type as a Helow Tele visit. 0 - Non Helow Tele visit, 1 - Helow Tele visit.
DermVisit tinyint This column stores value 0/1 to mark any appointment with this visit type as a dermatology visit. 0 - Non dermatology, 1 - Dermatology.
allergyskintestvisit smallint This column stores value 0/1 to indicate any appointment with this Visit Type as a allergy visit. 0 - Non allergy visit, 1 - Allergy visit.
occHealthVisit tinyint This column stores value 0/1 to indicate any appointment with this Visit Type as a occ-health. 0 - Non occ-health visit, 1 - occ-health visit.
enablecollectubclaimdata tinyint Stores values 0/1 to indicate whether to enable collectible claim data. 0 - Do not enable collectible claim data, 1 - Enable collectible claim data
TCMVisit smallint This column stores value 0/1 to indicate if the visit code is for Transition Care Management. 0 - Not a TCM visit code , 1 - TCM visit code.
insplantype int This column stores insurance plan type code from the pre-defined list of values. 0 - Medical , 1- Dental, 4 - Vision, 5 - Behavioral Health.
ccmrGroupMaxCapacity int Column stored default max capacity for Group encounters
VascularVisit smallint This column stores value 0/1 to indicate if the visit code is for Vascular module. 0 - Not a Vascular visit, 1 - visit code is for Vascular visit.
rpmVisit tinyint Flag indicating if the visit is related to RPM (e.g., 0 for No, 1 for Yes).
wellChildVisit tinyint Flag indicating if the visit is a well-child visit (e.g., 0 for No, 1 for Yes).
rheumatologyVisit tinyint Flag indicating if the visit is related to rheumatology (e.g., 0 for No, 1 for Yes).
warfarinFlowsheetVisit tinyint Flag indicating if the visit is related to warfarin flowsheet (e.g., 0 for No, 1 for Yes).
