Table patient_miscinfo_labels
Description This table stores a label added as part of patient's miscellaneous information.
Column DataType Description
Id (Primary Key) int This column is an auto incremented Primary Key.
LabelName varchar This column stores name of the new label (tag) added as part of miscellaneous information of a patient.
DeleteFlag int This column stores value 0 or 1. 1 - When the label is deleted, 0 - Otherwise.
templateTag varchar This column stores name of the tag added while adding new label as part of miscellaneous information of a patient.
