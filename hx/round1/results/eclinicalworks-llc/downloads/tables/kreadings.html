EHI Export Schema
Table kreadings
Description This table stores keratometry readings and related information for patients' eye examinations.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the kreadings table. Example: 1, 2, 3.
Encounterid int identifier for the encounter associated with the kreading. It refers to column encounterid from enc table.
DiopterMM tinyint Diopter value in millimeters. Example: 42, 45, 48.
FlatK1OD varchar Flat K1 reading for the right eye (OD). Example: "42.5", "43.0", "43.5".
FlatK2OD varchar Flat K2 reading for the right eye (OD). Example: "44.0", "44.5", "45.0".
SteepK1OD varchar Steep K1 reading for the right eye (OD). Example: "46.0", "46.5", "47.0".
SteepK2OD varchar Steep K2 reading for the right eye (OD). Example: "48.0", "48.5", "49.0".
FlatK1OS varchar Flat K1 reading for the left eye (OS). Example: "42.5", "43.0", "43.5".
FlatK2OS varchar Flat K2 reading for the left eye (OS). Example: "44.0", "44.5", "45.0".
SteepK1OS varchar Steep K1 reading for the left eye (OS). Example: "46.0", "46.5", "47.0".
SteepK2OS varchar Steep K2 reading for the left eye (OS). Example: "48.0", "48.5", "49.0".
VertexDist varchar Vertex distance value. Example: "12.0", "12.5", "13.0".
SpecBC varchar Base curve value for the spectacle lens. Example: "8.0", "8.5", "9.0".
WorkDist varchar Working distance value. Example: "40.0", "45.0", "50.0".
WorkInter varchar Working intermediate distance value. Example: "60.0", "65.0", "70.0".
SpecificAdd varchar Specific addition value for the right eye (OD). Example: "1.0", "1.5", "2.0".
Reason varchar Reason for the keratometry reading. Example: "Routine checkup", "Post-surgery", "Vision problem".
KReadingType tinyint Type of keratometry reading. Example: 0 (manual), 1 (automated).
comments text Additional comments related to the keratometry reading. Example: "Patient reported discomfort", "Follow-up in 3 months".
krmire int Keratometry reading mire value for the right eye (OD). Example: 1, 2, 3.
klmire int Keratometry reading mire value for the left eye (OS). Example: 1, 2, 3.
SpecificAddOS varchar Specific addition value for the left eye (OS). Example: "1.0", "1.5", "2.0".
