Table vision_upc_calculation_persist
Description This table stores information related to vision UPC calculations, including amounts, adjustments, and balances for insurance and patient payments. This table is copy of vision_upc_calculation_temp and vision_upc_calculation_temp_secondary.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table.
encid int Encounter id, referring to a specific patient encounter.
ruleid int Rule id, referring to a specific rule applied to the calculation.
insid int Insurance id, referring to a specific insurance plan.
visiontype int Vision type id, representing the type of vision service provided.
facilityid int Facility id, referring to a specific facility where the service was provided.
upccode varchar Universal Product Code (UPC) for the item or service.
cptitemid int CPT (Current Procedural Terminology) item id, referring to a specific item or service.
cptcode varchar CPT code, representing the code for a specific item or service.
copayreducefromins tinyint Flag indicating whether the copay should be reduced from the insurance payment (1) or not (0).
copay decimal Copay amount for the item or service.
retailamount decimal Retail amount for the item or service before any discounts or adjustments.
retailmarkdown decimal Retail markdown amount applied to the item or service.
retailafterdisc decimal Retail amount after applying the markdown.
inspayment decimal Insurance payment amount for the item or service.
insmarkdown decimal Insurance markdown amount applied to the item or service.
remptbalance decimal Remaining patient balance for the item or service.
ptmarkdown decimal Patient markdown amount applied to the item or service.
totalptbalance decimal Total patient balance for the item or service after applying markdowns and adjustments.
adjustment decimal Adjustment amount applied to the item or service.
m1 varchar Custom field 1.
m2 varchar Custom field 2.
m3 varchar Custom field 3.
m4 varchar Custom field 4.
eyedescode int Eye description code, representing the specific eye condition or treatment.
qty int Quantity of the item or service provided.
blocked tinyint Flag indicating whether the record is blocked (1) or not (0).
transactionid int Transaction id, referring to a specific transaction related to the item or service.
upcwisesetup tinyint Flag indicating whether the UPC-wise setup is enabled (1) or not (0).
overallcoveredamt decimal Overall covered amount by insurance for the item or service.
overallptcoveredamt decimal Overall covered amount by the patient for the item or service.
cashadjustment decimal Cash adjustment amount applied to the item or service.
cmnflag tinyint Common flag, used for custom purposes.
finalinsbal decimal Final insurance balance for the item or service.
finalptbal decimal Final patient balance for the item or service.
secondaryallocation tinyint Flag indicating whether secondary allocation is enabled (1) or not (0).
insbalancedefaultallocate tinyint Flag indicating whether insurance balance default allocation is enabled (1) or not (0).
secondaryinsid int Secondary insurance id, referring to a secondary insurance plan.
secallocationretailprice decimal Secondary allocation retail price for the item or service.
secallocationcopay decimal Secondary allocation copay amount for the item or service.
secallocationinsbalance decimal Secondary allocation insurance balance for the item or service.
secallocationptbalance decimal Secondary allocation patient balance for the item or service.
addeddate datetime Date and time when the record was added to the table.
addedby int User id of the person who added the record to the table.
isbasecpt tinyint Flag indicating whether the item or service is a base CPT (1) or not (0).
adjcodeid int Adjustment code id, referring to a specific adjustment code applied to the item or service.
selfpaytype int Self-pay type id, representing the type of self-pay option for the patient.
