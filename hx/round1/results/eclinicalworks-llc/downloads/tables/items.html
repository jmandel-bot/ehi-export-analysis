EHI Export Schema
Table items
Description This table stores categories and items configured in the application.
Column DataType Description
itemid (Primary Key) int This is an auto-incremented primary key in This column .
itemName varchar This column stores name of an item provided by end user.
itemDesc varchar This column stores description of an item provided by end user.
parentid int This column stores id of a parent item under which new item has been created. This column refers to itemid column in items table.
xmlPath varchar This column stores path of the item from parent item.
xslPath varchar This column is not being used.
itemType char This column stores type of an items. C - Category , I - Item, U - Unit of Measurements, G - General.
supplierid int This column stores id of a supplier for an item. This column refers to id column in supplier table.
price decimal This column stores cost of an item.
discount decimal This column stores amount of discount for an item.
specialPrice decimal This column stores special price for an item.
startDate datetime This column is not being used.
endDate datetime This column is not being used.
createdDate datetime This column stores date and time when an item was created. Datetime is stored in 'YYYY-MM-DD hh:mm:ss.000' format.
modifiedDate datetime This column stores date and time when an item was modified. Datetime is stored in 'YYYY-MM-DD hh:mm:ss.000' format.
viewType char This column stores view type for an item.
displayIndex int This column stores number to list items in that order on User Interface.
deleteFlag int This column stores flags 0/1 to indicate whether item has been marked as deleted or not. 0 - Item is not deleted, 1 - Item is deleted.
vmid varchar Auto-generated unique string for each record inserted in This column .
keyName varchar This column stores key name for an item to indicate category/module and item belongs to. i.e HPI, Assessments, CommunityLabOrders etc.
gender varchar This column stores values M/F to indicate in which gender category item falls into. M - Male, F - Female
overdue int This column stores values 0/1/2 to indicate whether an item is Acute or Chronic. This setting is done at ICD code level. 0 - Item is NOT Acute/Chronic, 1 - Item is marked as Acute, 2 - Item is marked as Chronic.
inactive int This column stores flag 0/1 to indicate whether an item has been mark as active or inactive. 0 - Item is marked as active, 1 - Item is marked as inactive.
DrugNameid int This column stores id for drugs listed in This column for Medispan database.
showPathologyDetail tinyint This column stores flag 0/1 to indicate whether item is marked to show Pathology Details or not. 0 - Not marked to show Pathology Details screen, 1 - Marked to show Pathology Details screen.
vmsgexclude tinyint This column is not being used.
isConfidential tinyint This column stores values 0/1 to indicate whether the item is marked as confidential or not. 0 - Item is NOT marked as confidential, 1 - Item is marked as confidential.
drugGUID varchar This column stores allergen id for allergens in This column from Medispan database.
drugType varchar This column is not being used.
routeddrugid int identifier for the routed drug associated with the item (integer). Example: 1, 2, 3
dispensabledrugid int identifier for the dispensable drug associated with the item (integer). Example: 1, 2, 3
hasPackageDrugs tinyint Flag to indicate if the item has package drugs (tinyint). Example: 0 (no), 1 (yes)
