EHI Export Schema
Table asc_enc_details
Description This table contains details of ASC encounters such as discharge, arrivalTime, admitTime, etc.
Column DataType Description
id (Primary Key) int This is an auto increment primary key column of This column .
masterEncid int This column stores master encounter id
generalNotes text This column stores general notes
anestType varchar This column stores the type of Anesthesia selected for the respective appointment.
arrivalTime datetime time stamp for the patient's arrival at the facility. Example: '2021-06-01 08:00:00'.
admitTime datetime time stamp for the patient's admission to the facility. Example: '2021-06-01 08:30:00'.
prepOpStartTime datetime time stamp for the start of the pre-operation preparation. Example: '2021-06-01 09:00:00'.
prepOpEndTime datetime time stamp for the end of the pre-operation preparation. Example: '2021-06-01 09:30:00'.
inTroopStartTime datetime time stamp for the start of the intraoperative period. Example: '2021-06-01 10:00:00'.
inTroopEndTime datetime time stamp for the end of the intraoperative period. Example: '2021-06-01 11:00:00'.
anestStartTime datetime time stamp for the start of anesthesia administration. Example: '2021-06-01 10:15:00'.
anestEndTime datetime time stamp for the end of anesthesia administration. Example: '2021-06-01 10:45:00'.
postOpStartTime datetime time stamp for the start of the post-operation period. Example: '2021-06-01 11:15:00'.
postOpEndTime datetime time stamp for the end of the post-operation period. Example: '2021-06-01 12:00:00'.
discharge datetime time stamp for the patient's discharge from the facility. Example: '2021-06-01 14:00:00'.
deleteFlag int This column store delete flag. It will be set to 1 if entry is deleted else 0.
modifiedDate datetime time stamp for the last modification of the record. Example: '2021-06-01 15:00:00'.
ASAClass varchar This column stores ASAClass for the patients ASC procedure.
preop int This column stores id of the pre op Room when the patient is currently is checked in. This column refer to UID column in user table. Else it stores -1 when the patient moves out of the preop room
otroom int This column stores id of the OR Room when the patient is currently is in. This column refer to UID column in user table. Else it stores -1 when the patient moves out of the OR
pacuop int This column stores id of the post op Room when the patient is currently is checked in. This column refer to UID column in user table. Else it stores -1 when the patient moves out of the postop room
surgeryNotification varchar Notification status for the surgery. Example: "Scheduled", "Completed", "Cancelled".
