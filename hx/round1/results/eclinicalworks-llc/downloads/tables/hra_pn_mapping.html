EHI Export Schema
Table hra_pn_mapping
Description This table is to store mapped HRA from progress note.
Column DataType Description
Id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3
CPEncId int Identifier for the CP Encounters. Example: 101, 102, 103
HRAEncId int Identifier for the HRA Encounters. Example: 201, 202, 203
deleteflag tinyint A flag to indicate if the record is deleted (1) or not (0). Example: 0, 1
assmntTypeId int This stores assessment type id. This column refers to items.itemID.
assmntName varchar Name of the assessment. Example: 'Assessment A', 'Assessment B', 'Assessment C'
