EHI Export Schema
Table asc_drug_administration_header
Description This table is used to store the administered medication details.
Column DataType Description
id (Primary Key) int This is auto-incremented primary key in This column . This column stores unique id generated for each administration detail.
OldRxid int This column stores id of a prescribed medication. This column refers to oldrxid column in oldrxmain table.
Note text This column stores notes entered by provider/staff on the user interfacefor each administered medication.
CompletedFlag int Medication whose administration is completed then CompletedFlag is set to 1. Default value is 0.
Wastage decimal This column stores drug wastage mentioned by provider/staff on user interfacefor administered medication.
WastageUnit varchar This column stores drug wastage unit mentioned by provider/staff on user interfacefor administered medication.
Total decimal This column stores total administration dosage for administered medication. It is used when IV drug is being administered.
TotalUnit varchar This column stores total unit for administration dosage for administered medication. It is used when IV drug is being administered.
Patientid int This column stores identifier number of a patient, to whom the drug is administered. It refers to column UID from user table.
OrderEncounterType int This column is used to store order encounter type based on outpatient encounter or asc encounter type.
STATUS int Indicates the status of the drug administration record, where 0 represents inactive and 1 represents active. Default value is 0. (e.g., 0, 1)
AdminSource varchar This column stores administration source to identify the source of screen from where the medication administered.
ContinuousIVFlag int Indicates whether the drug is administered through a continuous intravenous (IV) infusion (1) or not (0). Default value is 0. (e.g., 0, 1)
OBSERVATION varchar This column stores observation time mentioned by provider/staff on user interfacefor administered medication.
OBSERVATIONUNIT varchar This column stores observation time unit mentioned by provider/staff on user interfacefor administered medication.
ISPTTOLWELL varchar Indicates whether the patient tolerated the drug administration well. Default value is an empty string. (e.g., "", "Yes", "No")
isAmbualnce smallint Indicates whether the drug administration occurred in an ambulance (1) or not (0). Default value is 0. (e.g., 0, 1)
isFamilyMember smallint Indicates whether a family member was present during the drug administration (1) or not (0). Default value is 0. (e.g., 0, 1)
isOwnAccord smallint Indicates whether the patient administered the drug on their own accord (1) or not (0). Default value is 0. (e.g., 0, 1)
isSignificantOther smallint This column stores discharge via significant other data when mentioned by provider/staff on user interfacefor administered medication.
IVHydration smallint Medication which is marked as IVHydration from Rx Formulary then IVHydration is set to 1. Default value is 0.
