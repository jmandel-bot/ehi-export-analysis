EHI Export Schema
Table vision_allocationcptdiscount
Description This table stores information about discounts applied to specific items in a vision allocation, including details about the discount amount, percentage, and other related item information.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
encounterid int identifier for the encounter associated with the allocation. Example: 101, 102, 103.
allocationdiscountid int identifier for the allocation discount. This is a foreign key that refers to the `id` column in the `vision_allocationdiscount` table. Example: 201, 202, 203.
allocationid int identifier for the allocation associated with the discount. Example: 301, 302, 303.
upccode varchar Universal Product Code (UPC) for the item associated with the discount. Example: '123456789012', '234567890123'.
cptitemid int identifier for the CPT (Current Procedural Terminology) item associated with the discount. Example: 401, 402, 403.
cptcode varchar CPT code for the item associated with the discount. Example: '99213', '99214', '99215'.
retailprice decimal Retail price of the item before the discount is applied. Example: 100.00, 200.00, 300.00.
ptbalance decimal Patient balance after the discount is applied. Example: 80.00, 160.00, 240.00.
discount decimal Discount amount applied to the item. Example: 20.00, 40.00, 60.00.
deleteflag tinyint Flag indicating whether the record has been deleted (1) or not (0). Example: 0, 1.
percentage decimal Discount percentage applied to the item. Example: 20.00, 30.00, 50.00.
addedby int identifier for the user who added the record. Example: 501, 502, 503.
updatedby int identifier for the user who last updated the record. Example: 504, 505, 506.
deletedby int identifier for the user who deleted the record. Example: 507, 508, 509.
addedtime datetime time stamp for when the record was added. Example: '2022-01-01 12:34:56'.
updatedtime datetime time stamp for when the record was last updated. Example: '2022-01-02 12:34:56'.
deletedtime datetime time stamp for when the record was deleted. Example: '2022-01-03 12:34:56'.
