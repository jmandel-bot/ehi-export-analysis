Table visionus_vergence
Description This table stores information about vision and vergence tests performed during an eye examination.
Column DataType Description
id (Primary Key) int This is the primary key for the table, an automatically incremented integer that uniquely identifies each record in the table. Example: 1, 2, 3, etc.
encounterid int This is a foreign key that refers to the `encounterid` column in the `enc` table. It represents the id of the encounter during which the vision and vergence tests were performed. Example: 101, 102, 103, etc.
examtype tinyint This is a tiny integer column that stores the type of eye examination performed. Example: 0, 1, 2, etc.
dist_blur smallint This is a small integer column that stores the distance at which the patient's vision starts to blur during the test. Example: 50, 100, 150, etc.
dist_break smallint This is a small integer column that stores the distance at which the patient's vision breaks during the test. Example: 60, 120, 180, etc.
dist_recovery smallint This is a small integer column that stores the distance at which the patient's vision recovers during the test. Example: 70, 140, 210, etc.
near_blur smallint This is a small integer column that stores the near distance at which the patient's vision starts to blur during the test. Example: 10, 20, 30, etc.
near_break smallint This is a small integer column that stores the near distance at which the patient's vision breaks during the test. Example: 15, 25, 35, etc.
near_recovery smallint This is a small integer column that stores the near distance at which the patient's vision recovers during the test. Example: 20, 30, 40, etc.
