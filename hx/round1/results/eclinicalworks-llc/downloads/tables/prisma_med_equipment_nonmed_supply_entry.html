Table prisma_med_equipment_nonmed_supply_entry
Description This table stores information about medical equipment and non-medical supply entries, including their details, status, and related organizations.
Column DataType Description
id (Primary Key) bigint A unique identifier for each entry in the table. It is a bigint(20) type and is the primary key. Example: 1, 2, 3.
sectionId bigint This column stores the ID of the associated section in the `prisma_ccd_document_section` table. It is a bigint value and can be NULL. Example: 10, 25, 37, etc.
medEquipId bigint A bigint(20) type representing the medical equipment ID associated with the entry. This column has a foreign key relation with the `prisma_med_equipment_details` table's `id` column. Example: 1001, 1002, 1003.
medEquipOrgId bigint A bigint(20) type representing the medical equipment organization ID associated with the entry. This column has a foreign key relation with the `prisma_med_equipment_org_detail` table's `id` column. Example: 2001, 2002, 2003.
status varchar A varchar(20) type representing the status of the entry. Example: "active", "inactive", "expired".
typeCode varchar A varchar(25) type representing the type code of the entry. Example: "TC001", "TC002", "TC003".
typeName varchar A varchar(255) type representing the type name of the entry. Example: "Type A", "Type B", "Type C".
receivedTime varchar A varchar(50) type representing the time when the entry was received. Example: "2021-09-01 12:00:00", "2021-09-02 14:30:00".
useEndTime varchar A varchar(50) type representing the end time of the entry's usage. Example: "2021-09-30 12:00:00", "2021-10-01 14:30:00".
quantity varchar A varchar(100) type representing the quantity of the supply. Example: "10", "25", "50".
quantityUnit varchar A varchar(50) type representing the unit of the quantity. Example: "pieces", "boxes", "packs".
udi varchar A varchar(255) type representing the Unique Device Identifier of the entry. Example: "UDI12345", "UDI67890", "UDI111213".
deviceCode varchar A varchar(25) type representing the device code of the entry. Example: "DC001", "DC002", "DC003".
deviceName varchar A varchar(255) type representing the device name of the entry. Example: "Device A", "Device B", "Device C".
scopingEntityId varchar A varchar(255) type representing the scoping entity ID associated with the entry. Example: "SEID123", "SEID456", "SEID789".
