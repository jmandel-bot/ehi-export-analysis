EHI Export Schema
Table tcm_facilitytypes
Description This table stores a list of facility types for the Transition Care Management module.
Column DataType Description
id (Primary Key) int This is the primary key column, which stores a unique integer value for each facility type. For example, 1, 2, 3, etc.
facilityType varchar This column stores the name of the facility type as a string. For example ALF,Expired,Home,Home Health,Hospice,Hospital,ICF,Left-AMA,Other,SNF/Rehab etc.
deleteflag varchar This column stores a flag indicating whether the facility type has been deleted or not. It stores '0' for not deleted and '1' for deleted.
description varchar This column stores the description of facility type
isNotEditable tinyint This column stores 0 or 1 values. 1 is considered as this facility type is not editable by the user.
value int This column stores an integer value associated with discharge disposition field in TCN notification message.
isFacilityType tinyint This columns store the int value with either 0 or 1. where 0 is represent that this type is facility type (i.e Hospital , Home health) and 1 represent that this type is not facility type (i.e home , expired)
