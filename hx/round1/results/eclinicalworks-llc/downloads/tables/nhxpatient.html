EHI Export Schema
Table nhxpatient
Description This table stores patient information received from incoming referrals for the pre-registration process.
Column DataType Description
id (Primary Key) int This is auto-incremented primary key in This column . This id represent patient id in pre registration queue.
nhxmpi int Column stores patient's id on NHX side.
lastname varchar This column is used to store last name of patient
firstname varchar This column is used to store first name of patient
minitial varchar This column is used to store middle initial of patient
sex varchar This column is used to store gender of patient
dob varchar This column is used to store date of birth for the patient
addressline1 varchar This column is used to store addressline 1 of the patient
addressline2 varchar This column is used to store addressline 2 of the patient
city varchar This column is used to store city of the patient
state varchar This column is used to store state of the patient
zipcode varchar This column is used to store zipcode of the patient
modifieddate datetime This column is used to store modified date of the patient
localpid int This represent patient id to which this patient has been matched.
blockid int This column stores appointment block id which created from healow appt booking.
hometel varchar This column represent telephone of patient.
refReqid int This column reqid column of nhxreferral.
insname varchar This column represents insurance name of patient.
insaddress varchar This column represents ins address of patient.
insaddress2 varchar This column is used to store insurance address line of the patient
inscity varchar This column is used to store insurance city line of the patient
insstate varchar This column is used to store insurance state of the patient
inszip varchar This column is used to store insurance zip code of the patient
insphone varchar This column is used to store insurance phone number of the patient
subscriberNO varchar This column is used to store subscriber number of the patient
patienttype smallint This column is used to store patient type
email varchar This column is used to store email of the patient
LANGUAGE varchar This column is used to store language of the patient
race varchar This column is used to store race of the patient
ethnicity varchar This column is used to store ethniciy of the patient
ethnicityCode varchar This column is used to store ethniciy code of the patient
umobileno varchar This column is used to store mobile number of the patient
