EHI Export Schema
Table encpointsnotes
Description This table stores notes related to encrypted points, along with information about the user who created the note and the time it was created.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an integer that uniquely identifies each record. It is an auto-incrementing value. Example: 1, 2, 3, etc.
Encid int This is a foreign key that refers to the encrypted points table (not provided in the query) and is an integer representing the id of the encrypted point to which the note is related. Example: 1, 5, 10, etc.
NotesTime datetime This is a datetime column that stores the date and time when the note was created. Example: 2021-09-15 14:30:00, 2021-09-16 09:45:00, etc.
Notes text This is a text column that stores the content of the note. Example: This is a sample note, Another note about the encrypted point, etc.
Notesid int This is an integer column that represents the id of the note. It has a default value of 1. Example: 1, 2, 3, etc.
deleteflag int This is an integer column that indicates whether the note has been deleted or not. A value of 0 means the note is not deleted, and a value of 1 means the note is deleted. The default value is 0. Example: 0, 1.
userid int This is a foreign key that refers to the user table (not provided in the query) and is an integer representing the id of the user who created the note. The default value is 0. Example: 1, 3, 5, etc.
