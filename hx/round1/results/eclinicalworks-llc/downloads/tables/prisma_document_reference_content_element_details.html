Table prisma_document_reference_content_element_details
Description This table stores information about content elements related to document references, including file attachments, content types, and display settings.
Column DataType Description
id (Primary Key) int Unique identifier for each content element record. Example: 1, 2, 3.
docRefId int Identifier for the related document reference. Foreign key relation with `prisma_document_reference_details` table's `id` column. Example: 101, 102, 103.
attchFilePath varchar File path for the attachment. Example: "/files/documents/attachment1.pdf", "/files/documents/attachment2.docx".
attchFileName varchar Name of the attachment file. Example: "attachment1.pdf", "attachment2.docx".
contentType varchar Type of content for the attachment. Example: "application/pdf", "application/msword".
uri varchar URI for the content element. Example: "https://example.com/content/1", "https://example.com/content/2".
attchTitle varchar Title of the attachment. Example: "Document 1", "Document 2".
attchCode varchar Code for the attachment. Example: "DOC1", "DOC2".
attchDisplay varchar Display settings for the attachment. Example: "inline", "download".
displayCategory varchar Category for displaying the attachment. Example: "Reports", "Invoices".
isErrorInFileUpload tinyint Flag to indicate if there was an error in the file upload process. 0 for no error, 1 for error. Example: 0, 1.
creationDate datetime Date and time when the content element record was created. Example: "2021-08-01 12:34:56", "2021-08-02 14:56:23".
