EHI Export Schema
Table allergy_form
Description This table stores information about various allergy forms, their creation and modification details, and related codes for different types of allergy tests.
Column DataType Description
FORMid (Primary Key) int The column stores an id of the form. This is an auto incremental primary key of This column .
FORMNAME varchar This column stores name of the form given by user.
CREATEDDATE datetime This column stores created date of the form when the form was created. The format used is YYYY-MM-DD HH:MM:SS.
MODIFIEDDATE datetime This column stores modified date of the form when the form was modified. The format used is YYYY-MM-DD HH:MM:SS.
DISPLAYINDEX int This column stores display index of the form. It stores sequence number in which the list of forms will be displayed. It is configurable by user from user interface.
DELETEFLAG int This column stores delete flag (In-active flag) of the form. It stores value 0/1/NULL. 1-Inactive, 0-Changed from Inactive to Active, NULL-Default value considered as Active.
CREATEDBY int This column stores identifier number of the provider who has created this form. It refers to column UID from user table.
MODIFIEDBY int This column stores identifier number of the provider who has modified this form. It refers to column UID from user table.
