EHI Export Schema
Table groupcodes
Description This table stores information about different group codes, their types, names, descriptions, and a flag indicating whether the record is deleted or not.
Column DataType Description
id (Primary Key) int This is a primary key which increments itself automatically.
GroupType varchar This column stores the type of the group defined by the user like ICD(for diagnosis), CPT(for CPT procedure) and IMG(for imaging) .
Name varchar This column stores the name of the group that is defined by the user.
Description varchar This column stores a brief description about the group name that is defined by the user.
deleteFlag smallint This column stores soft delete data. 1: Soft Deleted;0: Not Deleted.
