Table vision_stereopsis
Description This table stores information about vision stereopsis tests performed during patient encounters.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the vision_stereopsis table. It is an integer value that auto-increments. Example: 1, 2, 3.
encounterid int Unique identifier for the patient encounter during which the vision stereopsis test was performed. It is an integer value. Example: 101, 102, 103. This column has a foreign key relation with the `enc` table, referencing the `encounterid` column.
test_type int identifier for the type of vision stereopsis test performed. It is an integer value. It refers to items table `itemid` column Example: 1, 2, 3.
rx_type tinyint Indicates the type of prescription used during the test. It is an unsigned tinyint value. Example: 0, 1, 2.
fly varchar Stores the result of the "fly" test as a string. Example: 'Pass', 'Fail'.
animal tinyint This column stores the result of "animal" test performed.
circle tinyint This column stores the result of "circle" test performed.
distance int Stores the distance at which the test was performed, measured in millimeters. It is a mediumint value. Example: 500, 1000, 1500.
near int Stores the near distance at which the test was performed, measured in millimeters. It is a mediumint value. Example: 100, 200, 300.
comments varchar Stores any additional comments about the vision stereopsis test as a string. Example: 'Patient had difficulty with the test', 'Test results were inconclusive'.
