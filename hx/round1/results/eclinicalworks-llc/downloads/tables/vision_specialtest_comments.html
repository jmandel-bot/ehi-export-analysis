Table vision_specialtest_comments
Description This table stores comments related to special vision tests performed during an encounter.
Column DataType Description
id (Primary Key) int Stores the distance at which the test was performed, measured in millimeters. It is a mediumint value. Example: 500, 1000, 1500.
encounterid int Stores the near distance at which the test was performed, measured in millimeters. It is a mediumint value. Example: 100, 200, 300.
examtype tinyint Stores any additional comments about the vision stereopsis test as a string. Example: 'Patient had difficulty with the test', 'Test results were inconclusive'.
comments varchar This is a variable character field with a maximum length of 255 characters. It stores any comments related to the special vision test. Example: "Patient had difficulty with test A", "Test B results were inconclusive", etc.
