EHI Export Schema
Table customquestions
Description This table stores custom questions and their related information for Custom Smartform
Column DataType Description
Qsid (Primary Key) int Unique identifier for each custom question (e.g., 1, 2, 3).
Question varchar The text of the custom question (e.g., "What is your favorite color?").
QuesTip varchar A tip or hint related to the custom question (e.g., "Choose a color from the list.").
QuesCode varchar A short code representing the custom question (e.g., "Q1", "Q2", "Q3").
FieldType int An integer representing the type of input field for the custom question (e.g., 1 for text, 2 for dropdown, 3 for radio buttons).
Required tinyint A boolean value indicating whether the custom question is required (1) or not (0).
specialFormid int This column stores the smartform id . It is an integer value, and it cannot be NULL. Example: 1001, 1002, 1003.
deleteflag tinyint A boolean value indicating whether the custom question has been deleted (1) or not (0).
fieldValues varchar A string containing the possible values for the custom question's input field, separated by commas (e.g., "Red,Blue,Green" for a color dropdown).
