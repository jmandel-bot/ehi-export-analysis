EHI Export Schema
Table vision_endothelial
Description This table stores information about endothelial cell measurements for each eye during an encounter.
Column DataType Description
id (Primary Key) int This column is auto-incremented. Primary Key for This column .
encounterid int Unique identifier for each encounter, which is a foreign key that refers to the `encounterid` column in the `enc` table (e.g., 101, 102, 103).
r_ecc decimal Endothelial cell count (ECC) for the right eye, measured in cells per square millimeter (e.g., 2500.00, 2600.50, 2700.25).
l_ecc decimal Endothelial cell count (ECC) for the left eye, measured in cells per square millimeter (e.g., 2400.00, 2500.50, 2600.25).
r_std_dev decimal Standard deviation of the endothelial cell count for the right eye (e.g., 50.00, 60.50, 70.25).
l_std_dev decimal Standard deviation of the endothelial cell count for the left eye (e.g., 40.00, 50.50, 60.25).
r_ecd decimal Endothelial cell density (ECD) for the right eye, measured in cells per square millimeter (e.g., 3000.00, 3100.50, 3200.25).
l_ecd decimal Endothelial cell density (ECD) for the left eye, measured in cells per square millimeter (e.g., 2900.00, 3000.50, 3100.25).
r_ec_cof decimal Endothelial cell coefficient of variation (CV) for the right eye, expressed as a percentage (e.g., 5.00, 6.50, 7.25).
l_ec_cof decimal Endothelial cell coefficient of variation (CV) for the left eye, expressed as a percentage (e.g., 4.00, 5.50, 6.25).
r_ecp smallint Endothelial cell pleomorphism (ECP) for the right eye, expressed as a percentage (e.g., 10, 20, 30).
l_ecp smallint Endothelial cell pleomorphism (ECP) for the left eye, expressed as a percentage (e.g., 15, 25, 35).
