EHI Export Schema
Table refapptlog
Description This table stores logs for past and rescheduled appointments based on patient referral.
Column DataType Description
refid int This column stores the id of a referral created by provider/staff for which patient is visiting the referred provider. This column refers to referralid column in the referral table.
logDate datetime A date representing the date when the log was created. Example: 2021-09-15.
userid int An integer representing the unique user id associated with the appointment log. Example: 42.
username varchar A string (up to 255 characters) representing the username of the user associated with the appointment log. Example: johndoe.
prevappt varchar A string (up to 22 characters) representing the previous appointment date in a human-readable format. Example: 2021-08-25 14:30:00.
reschdate varchar A string (up to 22 characters) representing the rescheduled appointment date in a human-readable format. Example: 2021-09-10 10:00:00.
