EHI Export Schema
Table vision_colorvision
Description This table stores information about color vision tests performed during patient encounters.
Column DataType Description
id (Primary Key) int This column is an auto incremental primary key.
encounterid int This column stores encounter id and has foreign key constraint on "encounterid" column of "enc" table.
test_type int This column stores an id of the item selected in the drop down option for the color vision test in the Special Tests window. This column refers column itemid from items table.
rx_type tinyint This column stores values for rx option. Values that can be stored are 0 or 1. The default value is NULL. This column is optional and hence when user does not enter any value, NULL will be stored in them.
r_eye_v tinyint This column stores values for right eye color vision test. Default value is NULL. This column is optional and hence when user does not enter any value, NULL will be stored in them.
r_eye_total tinyint This column stores the total number of plates for the color vision test. Default value is NULL. This column is optional and hence when user does not enter any value, NULL will be stored in them.
l_eye_v tinyint This column stores values for left eye color vision test. The default value is NULL. This column is optional and hence when user does not enter any value, NULL will be stored in them.
l_eye_total tinyint This column stores the total no plates for left eye color vision test. The default value is NULL. This column is optional and hence when user does not enter any value, NULL will be stored in them.
both_eye_v tinyint This column stores values for both eye color vision test. The default value is NULL. This column is optional and hence when user does not enter any value, NULL will be stored in them.
both_eye_total tinyint This column stores the total no of plates for right eye color vision test. The default value is NULL. This column is optional and hence when user does not enter any value, NULL will be stored in them.
comments varchar This column stores the color vision test notes and varchar (255) is chosen to save comments.
