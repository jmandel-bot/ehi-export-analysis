EHI Export Schema
Table audioworkflowsfa
Description This table stores information related to audio workflows for specific tests, including the test type, description, and frequency values for each ear.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
Encid int The identifier for the related encounter. This is a foreign key that refers to the `encounterid` column in the `enc` table. Example: 101, 102, 103.
TestType int Integer value representing the type of test being performed. Example: 0, 1, 2.
TestDesc varchar A short description of the test being performed. Example: "Hearing Test", "Balance Test".
FreqEar1 varchar Frequency value for the first ear in the test. Example: "250", "500".
FreqEar2 varchar Frequency value for the second ear in the test. Example: "250", "500".
FreqEar3 varchar Frequency value for the third ear in the test. Example: "250", "500".
FreqEar4 varchar Frequency value for the fourth ear in the test. Example: "250", "500".
FreqEar5 varchar Frequency value for the fifth ear in the test. Example: "250", "500".
FreqEar6 varchar Frequency value for the sixth ear in the test. Example: "250", "500".
FreqEar7 varchar Frequency value for the seventh ear in the test. Example: "250", "500".
FreqEar8 varchar Frequency value for the eighth ear in the test. Example: "250", "500".
FreqEar9 varchar Frequency value for the ninth ear in the test. Example: "250", "500".
FreqEar10 varchar Frequency value for the tenth ear in the test. Example: "250", "500".
FreqEar11 varchar Frequency value for the eleventh ear in the test. Example: "250", "500".
