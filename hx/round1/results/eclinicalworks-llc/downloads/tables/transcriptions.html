EHI Export Schema
Table transcriptions
Description This table stores information about transcriptions, including their status, assignment, and type.
Column DataType Description
transcriptionid (Primary Key) int This is auto-incremented primary key in This column .
encounterid int This column stores id of an encounter which is being transcribed by provider. This column refers to encounterid column in enc table.
status int This column is not being used.
assignedtoid int This column stores id of a user to whom the transcription is assigned to. This column refers to UID column in user table.
deleteflag int This column stores values 0/1 to indicate whether the transcription record has been marked as deleted or not. 0 - Transcript is NOT marked as deleted, 1 - Transcript is marked as deleted.
reviewedbyid int This column stores id of a user who as reviewed the transcript. This column refers to UID column in user table.
TYPE int This column stores values 0/1 to indicate whether the transcription was created from EMR of from eCW Mobile application. 0 - Created from EMR, 1 - Created from eCW Mobile application.
readstatus int This column stores values 0/1 to indicate whether the transcript opened/read from EMR or eCW Mobile application. 0 - Read from EMR application, 1 - Read from eCW Mobile application.
