EHI Export Schema
Table ip_rh_druginteraction_details
Description This table is used to store interaction override details like interaction override reason, severity for interaction and acknowledge flag for acknowledged interaction
Column DataType Description
id (Primary Key) bigint This is the unique identifier for each interaction record.
interactionId bigint This column stores the interaction ID. This column is foreign key references Id column of ip_rh_druginteraction table.
medOrderId int This column stores the medorderId. This is a staging order ID which is associated with drug interaction and is generated when order is added in Add Order screen/Admit Reconciliation screen and is foreign key references stagingorderID column of ip_rh_ptstagingorders table.
dischargemedId int This is used to store dischargemedId associated with drug interaction. It is generated when medication is continued/started in discharge screen. This is foreign key references dismedid column of ip_rh_dischargemedication table.
