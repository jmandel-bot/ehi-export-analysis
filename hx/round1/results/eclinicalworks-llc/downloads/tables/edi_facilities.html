EHI Export Schema
Table edi_facilities
Description This table stores information about various medical facilities, including their addresses, contact information, and other related details.
Column DataType Description
Id (Primary Key) int Unique identifier for each facility (e.g., 1, 2, 3).
FacilityType char Type of facility represented by a 2-character code (e.g., 'FA').
Name varchar Name of the facility (e.g., 'General Hospital').
AddressLine1 varchar First line of the facility's address (e.g., '123 Main St').
AddressLine2 varchar Second line of the facility's address, if applicable (e.g., 'Suite 200').
City varchar City where the facility is located (e.g., 'New York').
State varchar State where the facility is located, represented by a 5-character code (e.g., 'NY').
Zip varchar Zip code of the facility's location (e.g., '10001').
Tel varchar Telephone number of the facility (e.g., '(555) 123-4567').
Fax varchar Fax number of the facility (e.g., '(555) 987-6543').
EMail varchar Email address of the facility (e.g., 'info@generalhospital.com').
BillingAddressLine1 varchar First line of the facility's billing address (e.g., '123 Billing St').
BillingAddressLine2 varchar Second line of the facility's billing address, if applicable (e.g., 'Suite 300').
BillingCity varchar City where the facility's billing address is located (e.g., 'New York').
BillingState varchar State where the facility's billing address is located, represented by a 5-character code (e.g., 'NY').
BillingZip varchar Zip code of the facility's billing address (e.g., '10002').
BillingTel varchar Telephone number for the facility's billing department (e.g., '(555) 321-6543').
BillingFax varchar Fax number for the facility's billing department (e.g., '(555) 987-1234').
BillingEMail varchar Email address for the facility's billing department (e.g., 'billing@generalhospital.com').
PracticeType char Type of medical practice represented by a 3-character code (e.g., 'MED').
PracticeOption varchar Type of medical practice, with options 'Medical', 'Chiropractic', or 'Other'.
PayableTo varchar Name of the entity to which payments should be made (e.g., 'General Hospital, Inc.').
BankAccount varchar Bank account number for the facility (e.g., '1234567890').
FederalTaxID varchar Federal tax identification number for the facility (e.g., '12-3456789').
Notes text Additional notes or comments about the facility.
PrimaryFacility int Integer value indicating if the facility is the primary facility (e.g., 1 for primary, 0 for non-primary).
CliaId varchar Clinical Laboratory Improvement Amendments (CLIA) identification number for the facility (e.g., '01D1234567').
code varchar A code associated with the facility (e.g., 'GH01').
POS int Integer value representing the place of service code for the facility (e.g., 11 for office, 21 for inpatient hospital).
MammoCertId varchar Mammography certification identification number for the facility (e.g., 'MC123456').
HPSAFlag int Integer value indicating if the facility is located in a Health Professional Shortage Area (HPSA) (e.g., 1 for yes, 0 for no).
HPSAModifier char 2-character code representing the HPSA modifier for the facility (e.g., 'H1').
DeleteFlag tinyint Tiny integer value indicating if the facility record has been deleted (e.g., 1 for deleted, 0 for active).
vmid varchar Unique identifier for the facility in the VM (virtual machine) system (e.g., 'GHVM123').
hl7id varchar Unique identifier for the facility in the HL7 (Health Level 7) system (e.g., 'HL70123').
Color int Integer value representing the color code associated with the facility (e.g., 16777215 for white).
FeeSchedule int Integer value representing the fee schedule associated with the facility (e.g., 1, 2, 3).
FacilityIdType int Integer value representing the type of facility identification used (e.g., 1 for NPI, 2 for CLIA).
PtStmtFacOption int Integer value representing the patient statement facility option (e.g., 0 for default, 1 for custom).
PtStmtFacId int Integer value representing the patient statement facility identification (e.g., 1, 2, 3).
RestrictPrintAppts tinyint Tiny integer value indicating if printing appointments is restricted for the facility (e.g., 1 for restricted, 0 for not restricted).
NPI varchar National Provider Identifier (NPI) number for the facility (e.g., '1234567890').
ImmStateId varchar State identification number for the facility's immunization records (e.g., 'IS12345').
TaxonomyCode varchar Taxonomy code for the facility's specialty (e.g., '207Q00000X' for Family Medicine).
RestrictPrintStmts tinyint Tiny integer value indicating if printing statements is restricted for the facility (e.g., 1 for restricted, 0 for not restricted).
startedOn datetime Date when the facility started operations (e.g., '2000-01-01').
RevenueCode varchar Revenue code associated with the facility (e.g., '0123').
Percent decimal Decimal value representing the percentage of revenue or fees associated with the facility (e.g., 10.000 for 10%).
OverrideSalesTax tinyint Tiny integer value indicating if sales tax should be overridden for the facility (e.g., 1 for override, 0 for no override).
MerchantID varchar Merchant identification number for the facility (e.g., 'MID12345').
ImmLocationId varchar Location identification number for the facility's immunization records (e.g., 'IL12345').
TimeZoneCode varchar Time zone code for the facility's location (e.g., 'America/New_York').
MessCallerId varchar Caller identification number for the facility's messaging system (e.g., '(555) 123-4567').
MessOperNum varchar Operator number for the facility's messaging system (e.g., 'OP123').
CreditCardOption varchar Credit card processing option for the facility (e.g., 'Visa', 'Mastercard').
CountryCode varchar Country code for the facility's location (e.g., 'US').
BillingCountryCode varchar Country code for the facility's billing address (e.g., 'US').
Immfacid varchar Immunization facility identification number (e.g., 'IF12345').
BillType varchar Billing type for the facility (e.g., 'Inpatient', 'Outpatient').
takeNewPortalAppts tinyint Tiny integer value indicating if the facility accepts new appointments through the portal (e.g., 1 for yes, 0 for no).
facilitynickname varchar Nickname for the facility (e.g., 'GenHosp').
exclude tinyint Tiny integer value indicating if the facility should be excluded from certain processes or reports (e.g., 1 for exclude, 0 for include).
portal_name varchar Name of the facility's online portal (e.g., 'GeneralHospitalPortal').
takenewportalmsg tinyint Tiny integer value indicating if the facility accepts new messages through the portal (e.g., 1 for yes, 0 for no).
showinpi tinyint Tiny integer value indicating if the facility should be displayed in the patient information system (e.g., 1 for display, 0 for hide).
modifiedDate datetime Date and time when the facility record was last modified (e.g., '2021-01-01 12:34:56').
nsexclude tinyint Tiny integer value indicating if the facility should be excluded from the no-show report (e.g., 1 for exclude, 0 for include).
facilitysmsname varchar Short name for the facility used in SMS messages (e.g., 'GenHospSMS').
birthdayexclude tinyint Tiny integer value indicating if the facility should be excluded from birthday reminders (e.g., 1 for exclude, 0 for include).
ASC_TN_FacilityType char Facility type for the ASC (Ambulatory Surgery Center) in Tennessee (e.g., 'A').
ASC_TN_FacilityId varchar Facility identification number for the ASC in Tennessee (e.g., 'TN123').
labexclude tinyint Tiny integer value indicating if the facility should be excluded from lab reports (e.g., 1 for exclude, 0 for include).
ASC_WI_FacilityType varchar Facility type for the ASC in Wisconsin (e.g., 'WIA').
rxexclude tinyint Tiny integer value indicating if the facility should be excluded from prescription reports (e.g., 1 for exclude, 0 for include).
referralExclude tinyint Tiny integer value indicating if the facility should be excluded from referral reports (e.g., 1 for exclude, 0 for include).
DoNotAllowSlide int Integer value indicating if sliding scale payments are not allowed for the facility (e.g., 1 for not allowed, 0 for allowed).
FacilityproviderId int Integer value representing the facility provider identification (e.g., 1, 2, 3).
isInfirmary tinyint Tiny integer value indicating if the facility is an infirmary (e.g., 1 for yes, 0 for no).
unitType int Integer value representing the type of unit for the facility (e.g., 1 for inpatient, 2 for outpatient).
PPL_Id int Integer value representing the PPL (Patient Payment Liability) identification for the facility (e.g., 1, 2, 3).
timezonename varchar Name of the time zone for the facility's location (e.g., 'Eastern Standard Time').
insplantype int Integer value representing the insurance plan type for the facility (e.g., -1 for none, 1 for HMO, 2 for PPO).
defaultEMServiceType int Integer value representing the default emergency medical service type for the facility (e.g., 1 for basic, 2 for advanced). Foreign key relation to `servicetypes` table (`Id` column).
RestrictPublishEstimates tinyint Tiny integer value indicating if publishing estimates is restricted for the facility (e.g., 1 for restricted, 0 for not restricted).
timezoneid int Integer value representing the time zone identification for the facility (e.g., 1, 2, 3). Foreign key relation to `timezones` table (`Id` column).
isNPIRequired bit Bit value indicating if NPI (National Provider Identifier) is required for the facility (e.g., b'1' for required, b'0' for not required).
faxDialingCountryId int Integer value representing the country identification for fax dialing for the facility (e.g., 1, 2, 3). Foreign key relation to `countrycodes` table (`id` column).
billingfaxDialingCountryId int Integer value representing the country identification for billing fax dialing for the facility (e.g., 1, 2, 3). Foreign key relation to `countrycodes` table (`id` column).
