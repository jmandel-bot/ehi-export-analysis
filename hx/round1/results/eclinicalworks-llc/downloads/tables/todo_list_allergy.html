EHI Export Schema
Table todo_list_allergy
Description This table stores information about allergies associated with items in a To Do List.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the todo_list_allergy table. Example: 1, 2, 3.
todo_list_id int identifier for the associated todo list. Foreign key relation with the id column in the todo_list table. Example: 101, 102, 103.
allergy_type smallint Numeric code representing the type of allergy. Example: 1 (food allergy), 2 (drug allergy), 3 (environmental allergy).
ccr_item_id int identifier for the associated item in the Continuity of Care Record (CCR). Example: 1001, 1002, 1003.
reaction varchar A short description of the allergic reaction. Example: hives, swelling, difficulty breathing.
onset_date datetime Date and time when the allergic reaction started. Example: 2021-09-01 14:30:00, 2021-09-02 10:15:00.
criticality smallint Numeric code representing the severity of the allergy. Foreign key relation with the id column in the allergycriticality table. Example: 1 (low), 2 (moderate), 3 (high).
allergy_status varchar A short description of the current status of the allergy. Example: active, inactive, resolved.
ccr_text varchar A detailed description of the allergy in text format. Example: Patient has a severe allergy to peanuts, causing anaphylaxis upon exposure.
