Table asc_ecw_gases_standardlist
Description This table stores information about standard gases, their units of measurement, and display preferences.
Column DataType Description
id (Primary Key) int This is the primary key column, containing a unique integer value for each gas entry. For example: 1, 2, 3, etc.
gasesName varchar This column stores the name of the gas as a string with a maximum length of 100 characters. For example: "Carbon Dioxide", "Methane", "Nitrogen", etc.
gasesUOM varchar This column stores the unit of measurement for the gas as a string with a maximum length of 20 characters. For example: "ppm", "ppb", "mg/m3", etc.
displayIndex tinyint This column stores a tiny integer value (4 bytes) representing the display order of the gas in a list or table. For example: 1, 2, 3, etc.
deleteFlag tinyint This column stores a tiny integer value (1 byte) indicating whether the gas entry has been deleted or not. A value of '0' means the entry is not deleted, and a value of '1' means the entry is deleted. For example: 0, 1.
