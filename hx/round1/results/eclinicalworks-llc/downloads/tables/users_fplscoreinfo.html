Table users_fplscoreinfo
Description This table stores information about users' FPL (Federal Poverty Level) scores, including the percentage of FPL, response file ID, and the last requested date.
Column DataType Description
fid (Primary Key) int This is the primary key column, which stores a unique integer identifier for each record in the table. Example: 1, 2, 3, etc.
gid varchar This column stores a unique varchar(11) identifier for each users table referred as guarantor id.
percentOfFPL float This column stores a float value representing the percentage of the Federal Poverty Level for the user. Example: 75.5, 120.3, etc.
responsefileid int This column stores an integer value representing the id of the response file associated with the user's FPL score. This could be a foreign key referring to fpl_responsefiles's primary key column (e.g., `responsefiles.id`). Example: 101, 102, 103, etc.
lastrequesteddate datetime This column stores a time stamp value indicating the last time the FPL score information was requested for the user. It is automatically updated to the current time stamp whenever the record is updated. Example: '2021-09-01 12:34:56', '2021-09-02 14:23:45', etc.
grid int It contains guarantors id with comma seperated
