EHI Export Schema
Table healow_data_sync_error_codes
Description This table stores information about error codes related to data synchronization in the healow system.
Column DataType Description
id (Primary Key) int Contains auto incremented id, datatype is INT. This column stores integer value which could grow with number of users added to the care team and hence integer is selected. It is AUTO_INCREMENT column and has PRIMARY KEY index.
error_code varchar Contains unique error code which represents an error occurred at healow side while syncing the readings of rpm enrolled patient devices. The data type is varchar(100). This field can have NULL value.
error_description varchar Contains description of the error code which represents an error occurred at healow side while syncing the readings of rpm enrolled patient devices. The data type is varchar(500). This field can have NULL value.
tooltipMessage varchar Contains toltip message to be displayed to the user for an error occurred at healow side while syncing the readings of rpm enrolled patient devices. The data type is varchar(500). This field can have NULL value.
deleteFlag tinyint Flag indicates the last reading received for a day is deleted, datatype is tinyint - 0 - Not Deleted, 1 - Deleted. This field will always have a value hence defined as NOT NULL. By default, this column should not be in Deleted status hence default 0 is chosen.
