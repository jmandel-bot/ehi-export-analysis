EHI Export Schema
Table allergy_shots_expired_vial_reasons
Description This table stores information about the reasons for expired allergy shot vials, including patient and encounter details, vial number, and expiration date.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
PATIENTid int Unique identifier for the patient associated with the expired vial. This is a foreign key that refers to the `UID` column in the `users` table. Example: 101, 102, 103.
ENCOUNTERid int Unique identifier for the encounter associated with the expired vial. This is a foreign key that refers to the `encounterid` column in the `enc` table. Example: 201, 202, 203.
ALLERGYSHOTid int Unique identifier for the allergy shot associated with the expired vial. This is a foreign key that refers to the `id` column in the `allergy_shots` table. Example: 301, 302, 303.
REASON varchar Description of the reason for the vial's expiration. Example: "Vial damaged", "Vial contaminated", "Vial expired".
VIALNO varchar Vial number associated with the expired vial. Example: "V001", "V002", "V003".
EXPIRATIONDATE datetime Date and time when the vial expired. Example: "2022-01-01 00:00:00", "2022-02-01 00:00:00", "2022-03-01 00:00:00".
CREATEDBY int Unique identifier for the user who created the record. This is a foreign key that refers to the `UID` column in the `users` table. Example: 401, 402, 403.
CREATEDON datetime Date and time when the record was created. Example: "2022-01-01 12:00:00", "2022-02-01 12:00:00", "2022-03-01 12:00:00".
DELETEFLAG tinyint A flag indicating whether the record has been deleted (1) or not (0). Example: 0, 1.
