EHI Export Schema
Table empiptdata
Description This table stores information related to patient consent details, including the consent options, consent signed date, opt-in practice code and name.
Column DataType Description
pid int This column represents the patient id for which consent has been saved. It refers to the user.UID column.
ecwmpi varchar This column represent eEHX enabled patient's ecwmpi id. This is gUID value.
timestamp datetime This column represents when an entry was made into this column. It is a time stamp column that defaults to the current date and time.
EnableEmpi int This column contains the enable empi value, which has a value of 1 only if the patient has opted-in with EHX, otherwise it will be 0.
eCROptInOption varchar This column contains the value of patient consent option with eEHX. Possible values include: 'OptOut', 'OptInThisFac', 'OptInAllFac,' and 'OptInThisFac'.
ReceivedConsent int This column contains the value of patient consent to eEHX. A value of '1' means the patient has consented to eEHX, and a value of '0' means the patient not consented to eEHX.
DateSigned datetime This column contains the date when a patient signed the consent form. Example: "2021-07-01"
MatchFound int This is a dead column, not used in the system, and the system does not populate this column.
delFlag int This is a tinyint (1) field that indicates whether the login record has been deleted or not. A value of '0' means the record is not deleted, and a value of '1' means the record is deleted. Example: 0, 1.
id (Primary Key) int This is the primary key column, which uniquely identifies each record in the table. It is an integer value, and it cannot be NULL. Example: 1, 2, 3, etc.
Facid varchar This is a dead column, not used from the system, and the system does not populate this column value.
optinpracticecode varchar This column represents a patient's practice code.
OptinFacilityid int This column represents a patient's optin facility id. It refers to the edi_facilities.id column.
OptinFacilityName varchar This column represents a patient's optin facility name. It refers to the edi_facilities.name column.
FacGroupid int This column is a dead column. It had been used with the MultiMaster eEHX setup, but that is no longer supported by the system.
ConsentSignedByGuarantor tinyint This is a dead column. It is not used by the system, and the system doesn't populate the column value.
IsBeneficiary int A flag indicating if the patient is a beneficiary (integer, 0 or 1). Example: 1
