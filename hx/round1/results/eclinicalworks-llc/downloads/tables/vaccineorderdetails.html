EHI Export Schema
Table vaccineorderdetails
Description This table stores information about vaccine orders, their status, and related details.
Column DataType Description
id (Primary Key) int This column is auto-generated unique primary key.
immid int This column stores immunization interface id or regid. It refers to id column of immlist table.
immunizationid int This column refers to the immunizationid column from the immunizations table.
hubOrderid int This column stores hub order id. The id is returned from hub when order is successfully placed at hub side while executing outbound job. This column refers to the column id from vaccineorder_details table at hub.
orderStatusid int This column refers to orderFlag column of the vaccine_order_status table. Order flag is a unique id that refers to specific order status, like Pending, In-Progress, etc.
orderStatusDescription varchar This column stores the order status description sent from hub for the order while PullVaccineOrderUpdate job runs. As the description is coming to EMR from vaccine provider through the hub.
MessageDetail varchar This column stores order level json which is sent to hub or received from hub.
ErrorJson varchar This column stores list of validation error for each attribute in json inbound response.
transmitDateTime datetime This column stores last updated order status date and time in YYYY-mm-dd hh:mm:ss UTC format.
responseDateTime datetime This column stores received response date and time in YYYY-mm-dd hh:mm:ss UTC format.
patientid int This column stores id of the patient. It is primary column refers to UID column of the user table.
failCount smallint This column stores int type value refers no of failed attempts for the order.
hubInboundid int This column stores primary id of vaccine_inbound table at hub.
