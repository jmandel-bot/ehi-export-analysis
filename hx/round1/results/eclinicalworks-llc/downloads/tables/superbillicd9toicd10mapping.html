EHI Export Schema
Table superbillicd9toicd10mapping
Description This table stores the mapping between ICD-9 and ICD-10 codes for medical billing purposes. It also contains information about the encounter and the group name associated with the mapping.
Column DataType Description
id (Primary Key) int Unique identifier for each mapping entry. Example: 1, 2, 3.
Encounterid int This column stores the encounter id for which data have been documented. This column refers to column encounterid from enc table.
Icd9Itemid int This column stores the itemid of the ICD9 Code. This column refers to column itemid from items table.
Icd10Itemid int This column stores the itemid of the ICD10 Code. This column refers to column itemid from items table.
GroupName varchar Name of the group associated with the mapping. Example: "Cardiology", "Neurology", "Orthopedics".
