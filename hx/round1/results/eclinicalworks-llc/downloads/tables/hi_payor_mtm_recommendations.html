Table hi_payor_mtm_recommendations
Description This table stores medication therapy management (MTM) recommendations for payors, including the recommended action, medication details, and related notification information.
Column DataType Description
id (Primary Key) int Unique identifier for each MTM recommendation record. Example: 1, 2, 3.
mtm_notification_id int The identifier for the MTM notification associated with this recommendation. Foreign key relation: references `hi_payor_mtm_notifications(id)`. Example: 101, 102, 103.
mtm_recommended_action varchar The recommended action for the MTM. Example: "Adjust dosage", "Change medication", "Discontinue medication".
mtm_medication_name varchar The name of the medication involved in the MTM recommendation. Example: "Aspirin", "Lisinopril", "Metformin".
mtm_medication_route varchar The route of administration for the medication involved in the MTM recommendation. Example: "Oral", "Intravenous", "Topical".
mtm_recommend_medication_name varchar The name of the recommended medication, if applicable. Example: "Atorvastatin", "Losartan", "Amlodipine".
mtm_recommend_medication_route varchar The route of administration for the recommended medication, if applicable. Example: "Oral", "Intravenous", "Topical".
deleteflag bit A flag indicating if the record has been deleted (1) or not (0). Example: 0, 1.
