Table transworldfilelog
Description This table stores information about file logs related to Transworld Systems, including details about the file, the number of patients and claims, and the creation date and time.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an integer that uniquely identifies each record in the table. It is an auto-incrementing value. (e.g., 1, 2, 3, etc.).
FileName varchar This is a varchar column that stores the name of the file associated with the log (e.g., file1.txt, file2.txt, etc.).
NoOfPatients int This is an integer column that stores the number of patients associated with the file (e.g., 10, 25, 50, etc.).
CreatedDate datetime This is a date column that stores the date the file was created (e.g., 2021-10-01, 2021-10-02, etc.).
CreatedTime datetime This is a time column that stores the time the file was created (e.g., 12:30:00, 14:45:00, etc.).
deleteFlag int This is an integer column with a default value of 0 that indicates whether the record has been deleted or not. A value of 0 means the record is not deleted, while a value of 1 means the record is deleted (e.g., 0, 1).
externalagencyid int This is an integer column that stores the external agency id associated with the file. The id may be a foreign key referring to the id column in billing_vendors table (e.g., 1, 2, 3, etc.).
noofclaims int This is an integer column with a default value of 0 that stores the number of claims associated with the file (e.g., 5, 10, 15, etc.).
