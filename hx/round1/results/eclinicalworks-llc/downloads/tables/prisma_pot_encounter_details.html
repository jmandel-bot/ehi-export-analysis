EHI Export Schema
Table prisma_pot_encounter_details
Description This table stores information about External encounters, including their start and end times, type codes, type names, and type texts.
Column DataType Description
id (Primary Key) bigint A unique identifier for each encounter detail record. This is the primary key for the table. Example: 1, 2, 3.
sectionId bigint A foreign key that refers to the `id` column in the `prisma_ccd_document_section` table. It represents the section in which the encounter detail is present. Example: 101, 102, 103.
encounterId varchar A unique identifier for the encounter. Example: "E123", "E124", "E125".
encounterStartTime varchar The start time of the encounter in a string format. Example: "2021-09-01T10:00:00", "2021-09-02T14:30:00".
encounterEndTime varchar The end time of the encounter in a string format. Example: "2021-09-01T11:00:00", "2021-09-02T15:30:00".
encounterTypeCode varchar A code representing the type of encounter. Example: "123", "124", "125".
encounterTypeName varchar A name representing the type of encounter. Example: "Inpatient", "Outpatient", "Emergency".
encounterTypeText varchar It contains text for encounter type.
