EHI Export Schema
Table labdata_coldetails
Description This table stores information about lab data related to specific lots, manufacturers, and expiry dates of products.
Column DataType Description
id (Primary Key) int This column stores a unique identifier for each record in the table. Example: 1, 2, 3.
Reportid int This column stores the identifier for the report associated with the lab data. This is a foreign key that refers to the ID column in another table (e.g., reports table). Example: 101, 102, 103.
Lotid int This column stores the identifier for the lot associated with the lab data. This is a foreign key that refers to the id column in another table (e.g., lots table). Example: 1001, 1002, 1003.
lotNumber varchar This column stores the lot number of the product. Example: A123, B456, C789.
Manufacturer varchar This column stores the name of the manufacturer of the product. Example: CompanyA, CompanyB, CompanyC.
ExpiryDate datetime This column stores the expiry date of the product. Example: 2022-12-31, 2023-01-31, 2023-02-28.
Dose varchar This column stores the dose information of the product. Example: 10mg, 20mg, 30mg.
Routeid int This column stores the identifier for the route of administration associated with the lab data. This is a foreign key that refers to the id column in another table (e.g., routes table). Example: 1, 2, 3.
Locationid int This column stores the identifier for the location where the lab data was collected. This is a foreign key that refers to the id column in another table (e.g., locations table). Example: 10, 20, 30.
