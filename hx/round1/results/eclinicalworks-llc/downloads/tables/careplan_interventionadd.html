Table careplan_interventionadd
Description This table stores additional information related to care plan interventions, such as addendum notes, user who added the notes, and the date the notes were created.
Column DataType Description
id (Primary Key) int This is the primary key for the table, and it is an auto-incrementing integer value that uniquely identifies each record in the table. Example: 1, 2, 3, etc.
Interventionid int This is a foreign key that refers to the `id` column in another table (likely named `interventions` or similar) and stores the unique identifier of the related intervention. Example: 101, 102, 103, etc.
AddendumNotes text This column stores the text of the addendum notes related to the intervention. Example: "Patient showed improvement after medication adjustment."
Userid int This column stores the unique identifier of the user who added the addendum notes. It could be a foreign key referring to the `id` column in a `users` table. Example: 1001, 1002, 1003, etc.
Created datetime This column stores the date and time when the addendum notes were created. Example: '2021-09-01 12:34:56'
