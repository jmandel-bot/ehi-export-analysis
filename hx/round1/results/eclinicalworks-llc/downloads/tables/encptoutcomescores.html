EHI Export Schema
Table encptoutcomescores
Description This table stores the outcome scores for different encounters, along with their associated community and local IDs, values, and publication status.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key of the table.
encounterid int This column stores the encounter id for which scores are received. This column refers to the the encounter id from the enc table.
outcomecommunityid int This column stores the community id of the questionnaire form which is the primary key (ecwPropid column) of ecwproperties table.
scorecommunityid int This column stores the community id of the results and scores of the questionnaire form, which references the ecwStructid column of the ecwstructoptions table.
VALUE text This column stores the value of the result score and option.
ispublished tinyint This column stores the status of the result score and option. A 0 means not published, and a 1 means published.
optioncommunityid int This column stores the community id of the result option of the questionnaire form which is the primary key (ecwOptionid column) of the ecwstructoptions table.
outcomelocalid int This column stores the local id of the questionnaire form which references the propid column of the ecwpropertiesmapper table.
scorelocalid int This column stores the community id of the results and scores of the questionnaire form, which references the structid column of the ecwstructmapper table.
optionlocalid int This column stores the community id of the result option of the questionnaire form which references the optionid column of the ecwstructoptionsmapper table.
