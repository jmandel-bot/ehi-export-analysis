EHI Export Schema
Table insurance_dental_insplan_category
Description This table stores information about different dental insurance plan categories, including their coverage range and deductible flags.
Column DataType Description
id (Primary Key) int • This column will store the id of the insurance_dental_insplan_category • The datatype of the column would be INT to accommodate large number of different insurance plan categories that can be added. • This would be unique and auto incremented and so PRIMARYKEY constraint is added
insid int Not Available
NAME varchar • This column stores the category name captured on UI. • The datatype of the column would be varchar as it can have upto 30 characters of category name.
range_start_code varchar • This column stores cpt range start code entered on UI. • The datatype of the column would be varchar as it is storing cpt code values ex. D0140, D2345 etc.
range_end_code varchar • This column stores cpt range end code entered on UI. • The datatype of the column would be varchar as it is storing cpt code values ex. D0140, D2345 etc. •
coverage decimal • This column stores coverage percentage value entered on UI. • The datatype of the column would be decimal as it is storing values from 0.00 to 100.
deductibleflag tinyint This column store the deductible checkbox captured on UI. • The datatype of the column would be tinyint as it can have 0 or 1 value. • 0 value mean deductible is not checked for this category. • 1 value mean deductible is checked for this category.
deleteflag tinyint • This column determines whether the insurance plan coverage category is deleted. • The datatype of the column would be tinyint as it can have 0 or 1 value. • 0 value mean insurance plan coverage category is not deleted • •1 value mean insurance plan coverage category is deleted
