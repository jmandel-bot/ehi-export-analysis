Table billingdata_ptunits_updatelog
Description This table stores the log of updates made to the billing data of patient units.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3, etc.
billingdata_id int Foreign key referring to the billing data table, representing the specific billing data record being updated. Example: 101, 102, 103, etc. Foreign key relation: `billingdata(id)`
userid int Foreign key referring to the user table, representing the user who made the update. Example: 1001, 1002, 1003, etc. Foreign key relation: `user(id)`
units_org decimal Original value of the units before the update. Stored as a decimal with 12 digits and 2 decimal places. Example: 10.50, 15.75, 20.00, etc.
units_updated decimal Updated value of the units after the update. Stored as a decimal with 12 digits and 2 decimal places. Example: 11.50, 16.75, 21.00, etc.
updatetime varchar time stamp of when the update was made, stored as a varchar(30). Example: '2021-10-01 12:34:56', '2021-10-02 14:30:00', etc.
deleteflag tinyint A tinyint(4) flag indicating whether the record has been deleted (1) or not (0). Example: 0, 1.
