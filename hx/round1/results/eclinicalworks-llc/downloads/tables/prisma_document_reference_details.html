EHI Export Schema
Table prisma_document_reference_details
Description This table stores information about document references in a healthcare system, including details about the document, its status, and related entities such as custodians.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
fhirSectionType tinyint A tiny integer representing the type of FHIR section for the document reference. Example: 0, 1, 2.
fhirId int An integer representing the unique identifier of the FHIR source. This is a foreign key that refers to the `id` column in the `fhir_source_table`.
documentReferenceId varchar A string containing the unique identifier of the document reference. Example: "DR123", "DR456".
fhirStatus varchar A string representing the status of the document reference in the FHIR system. Example: "current", "superseded".
docStatus varchar A string representing the status of the document. Example: "final", "amended".
docRefCreatedDate datetime A datetime value representing the creation date of the document reference. Example: "2021-01-01 12:34:56".
effectiveDate datetime A datetime value representing the effective date of the document reference. Example: "2021-01-02 12:34:56".
fhirDescription varchar A text field containing a description of the document reference in the FHIR system. Example: "This is a sample document reference description."
documentReferenceTypeCode varchar A string representing the code of the document reference type. Example: "L123", "L456".
documentReferenceTypeDisplay varchar A string representing the display text of the document reference type. Example: "Lab Report", "Discharge Summary".
custodianReferenceId int An integer representing the unique identifier of the custodian reference. This is a foreign key that refers to the `id` column in the `custodian_reference_table`. Example: 1, 2, 3.
custodianDisplay varchar A string representing the display text of the custodian. Example: "Hospital A", "Clinic B".
contextPeriodStartDate datetime A datetime value representing the start date of the context period for the document reference. Example: "2021-01-01 00:00:00".
contextPeriodEndDate datetime A datetime value representing the end date of the context period for the document reference. Example: "2021-01-31 23:59:59".
titleId tinyint A tiny integer representing the title identifier for the document reference. Example: 0, 1, 2.
