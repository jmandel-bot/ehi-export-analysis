Table telcc
Description This table stores information about telephone communication encounters, including details about the properties involved, whether the communication was selected, any notes taken, the duration of the call, and the sequence in which it was displayed.
Column DataType Description
Encounterid int This is an integer column that represents the unique identifier for each encounter. Example: 1, 2, 3, etc.
TelCCid int This is an integer column that represents the unique identifier for each telephone communication. Example: 101, 102, 103, etc.
Propid int This is an integer column that represents the unique identifier for each property involved in the communication. Example: 1001, 1002, 1003, etc.
Selected int This is an integer column that indicates whether the communication was selected or not. A value of 0 means not selected, and a value of 1 means selected. Example: 0 or 1.
Notes text This is a text column that stores any notes taken during the communication. Example: "Discussed property details and pricing."
Duration varchar This is a varchar(255) column that stores the duration of the telephone communication in a string format. Example: "00:15:30" (15 minutes and 30 seconds).
DisplaySeq int This is an unsigned integer column that represents the sequence in which the communication was displayed. Example: 1, 2, 3, etc.
createDate datetime This is a datetime column that stores the date and time when the record was created. Example: "2022-01-01 12:34:56" (January 1, 2022, 12:34:56 PM).
