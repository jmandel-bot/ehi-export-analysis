Table todo_source
Description This table stores information about various sources for items on a To Do List, including the source name, pathway, and unique code for each source.
Column DataType Description
id (Primary Key) tinyint This is the primary key column, which stores a unique identifier for each source. It is of type tinyint(4) and is automatically incremented. Example: 1, 2, 3, etc.
source varchar This column stores the name of the source for the to-do items. It is of type varchar(100) and cannot be NULL. Example: Work, Personal, School, etc.
pathway varchar This column stores the pathway or location of the source. It is of type varchar(255) and has a default value of an empty string. Example: /home/user/documents/work_tasks, https://example.com/personal_tasks, etc.
code varchar This column stores a unique code for each source. It is of type varchar(100) and has a default value of an empty string. Example: W1, P1, S1, etc.
