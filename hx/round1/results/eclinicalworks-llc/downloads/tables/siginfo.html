EHI Export Schema
Table siginfo
Description This table stores the signature and details related to the Signature feature such as Signed by, Signed on, etc. to be displayed on the patient's document.
Column DataType Description
UID int This column stores patient id. This column refers to UID column from user table.
content text This column stores consent form data. This column refers to value/value_spanish column based on language from kioskSetting table.
jsonImg text This column stores signature image data.
id (Primary Key) int This column stores auto incremented value and is the primary key of the table.
signedDate varchar This column stores the consent form signed date.
consentFormid int This column stores id of the signed consent form. This column refers to id column of kioskSetting table.
isFormAccepted char This column stores value as 1 or 0 (1=consent form is accepted & 0 = consent form is declined)
byteImg text This column stores signature image data.
documentid int This column stores document reference id
signedByUserFname varchar First name of the user who signed the document. Example: 'John'
signedByUserLname varchar Last name of the user who signed the document. Example: 'Doe'
signedByUserRelation varchar Relation of the user who signed the document to the main user. Example: 'Parent'
loggedInUserType varchar Enum indicating whether the user signed the document themselves ('self') or on behalf of someone else ('OBO'). Example: 'self'
