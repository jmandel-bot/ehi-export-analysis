Table ip_rh_ptStagingIntdOrders
Description This table appears to store interdisciplinary order staging information for patients. Each column serves a specific purpose in recording the details of a patient's interdisciplinary order in a staging environment before it is finalized.
Column DataType Description
id (Primary Key) int Acts as the primary key for the table, uniquely identifying each record.
stagingOrderId int Likely a reference to an order that is being placed. It has NOT NULL constraint ensures that every record must have value and column referring to table ip_rh_ptstagingorders, indicating that it is a mandatory field for the staging process.
consultingProviderId int It stores the identifier of the consulting provider associated with the intended order. If it is a foreign key, it would reference a providers table, though this is not explicitly stated.
deleteFlag int Indicates whether the intended order is marked for deletion.
