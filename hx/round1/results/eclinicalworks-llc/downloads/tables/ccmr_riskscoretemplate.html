EHI Export Schema
Table ccmr_riskscoretemplate
Description This table stores the risk score template and other attributes of that template like max score, etc.
Column DataType Description
id (Primary Key) int This is a primary key which increments itself automatically.
name varchar This column stores HRA risk template name.
isdefault tinyint This column stores default flag to indicate whether the risk template is default.1:Default;0:None.
maxscore int This column stores max score of HRA risk.
modifiedbyid int This column stores user id of a user who modified the HRA risk template.
modifiedon datetime This column stores the date on which record has been modified recently.
deleteflag tinyint This column stores soft delete flag. 1: Soft Deleted;0: Not Deleted.
structFlag tinyint This column stores struct flag of HRA risk template.
questionairetemplate int This column stores HRA questionnaire template flag.
classicFlag int An integer flag representing the classic status of the risk score template. Example: 0 (not classic), 1 (classic).
