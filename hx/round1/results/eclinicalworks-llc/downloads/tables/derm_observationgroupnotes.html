Table derm_observationgroupnotes
Description This table stores notes for a group of observations (observations of the same lesion ID) for a lesion ID and encounter ID.
Column DataType Description
id (Primary Key) int This is the primary key for the table, an integer that uniquely identifies each record in the table. It is an auto-incrementing value which grows with observation group is created and notes added to it in Dermatology Examination. Example: 1, 2, 3, etc.
encounterid int This is a foreign key column, an integer that refers to the `encounterid` column in the `enc` table. It represents the id of the encounter associated with the observation group note. Example: 101, 102, 103, etc.
lesionid int This is a foreign key column, an integer that refers to the `lesionid` column in the `derm_lesionlist` table. It represents the id of the lesion associated with the observation group note. Example: 1001, 1002, 1003, etc.
notes varchar This is a varchar column with a maximum length of 255 characters. It stores the actual notes related to the observation group for a specific encounter and lesion. Example: "Redness observed around the lesion", "Swelling has reduced since last visit", etc.
