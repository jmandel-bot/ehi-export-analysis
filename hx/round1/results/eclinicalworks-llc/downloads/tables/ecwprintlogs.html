EHI Export Schema
Table ecwprintlogs
Description This table stores information about print logs for EHR (Electronic Health Records), including details about the print job, user, and patient.
Column DataType Description
id (Primary Key) int Unique identifier for each print log entry. Example: 1, 2, 3, etc.
parameters text Text containing the parameters used for the print job. Example: copies: 2, orientation: portrait.
type varchar Type of print job, such as the document being printed. Example: prescription, lab_report, invoice.
macaddress varchar MAC address of the device that initiated the print job. Example: 00:1A:79:12:34:56.
datecreated datetime Date and time when the print log entry was created. Example: 2021-09-01 12:34:56.
dateupdated datetime Date and time when the print log entry was last updated. Example: 2021-09-02 14:23:12.
isprinted tinyint Flag indicating whether the print job was successful (1) or not (0). Example: 0 or 1.
userid int Identifier for the user who initiated the print job. This could be a foreign key referring to the id column in a user table. Example: 42.
encounterid int Identifier for the patient encounter associated with the print job. This could be a foreign key referring to the id column in an encounters table. Example: 123.
patientid int Identifier for the patient associated with the print job. This could be a foreign key referring to the id column in a patients table. Example: 789.
content text Long text containing the content of the document being printed. Example: Patient Name: John Doe\nDOB: 01/01/1980\nPrescription: Amoxicillin 500mg\n.
statusMsg varchar Status message for the print job, indicating success or failure. Example: Printed successfully or Printer error: out of paper.
