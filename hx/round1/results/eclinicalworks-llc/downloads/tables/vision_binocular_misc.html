Table vision_binocular_misc
Description This table stores information related to various vision tests and measurements for binocular vision.
Column DataType Description
id (Primary Key) int This column is a unique identifier for each record in the table (e.g., 1, 2, 3).
Encounterid int identifier for the encounter associated with the vision test, foreign key referring to the encounter table's id column (e.g., 1001, 1002, 1003).
NPC_Test varchar Description of the Near Point of Convergence test performed (e.g., "Test 1", "Test 2", "Test 3").
NPC_Blur varchar Near Point of Convergence blur measurement (e.g., "10 cm", "12 cm", "15 cm").
NPC_Break varchar Near Point of Convergence break measurement (e.g., "20 cm", "22 cm", "25 cm").
NPC_Recv varchar Near Point of Convergence recovery measurement (e.g., "30 cm", "32 cm", "35 cm").
FUSION_Test varchar Description of the fusion test performed (e.g., "Test A", "Test B", "Test C").
FUSION_Dist varchar Fusion test distance measurement (e.g., "40 cm", "42 cm", "45 cm").
FUSION_Near varchar Fusion test near measurement (e.g., "50 cm", "52 cm", "55 cm").
FUSION_Impression varchar Impression of the fusion test (e.g., "Good", "Average", "Poor").
ANISEIKONIA_ODHorz varchar Aniseikonia test horizontal measurement for the right eye (e.g., "1%", "2%", "3%").
ANISEIKONIA_ODVert varchar Aniseikonia test vertical measurement for the right eye (e.g., "1%", "2%", "3%").
ANISEIKONIA_OSHorz varchar Aniseikonia test horizontal measurement for the left eye (e.g., "1%", "2%", "3%").
ANISEIKONIA_OSVert varchar Aniseikonia test vertical measurement for the left eye (e.g., "1%", "2%", "3%").
NPA_OD varchar Near Point of Accommodation measurement for the right eye (e.g., "10 cm", "12 cm", "15 cm").
NPA_OS varchar Near Point of Accommodation measurement for the left eye (e.g., "10 cm", "12 cm", "15 cm").
NPA_OU varchar Near Point of Accommodation measurement for both eyes (e.g., "10 cm", "12 cm", "15 cm").
STEREOPSIS_Test varchar Description of the stereopsis test performed (e.g., "Test X", "Test Y", "Test Z").
STEREOPSIS_Dist varchar Stereopsis test distance measurement (e.g., "60 cm", "62 cm", "65 cm").
STEREOPSIS_Near varchar Stereopsis test near measurement (e.g., "70 cm", "72 cm", "75 cm").
Dominance varchar Description of the eye dominance test performed (e.g., "Test D1", "Test D2", "Test D3").
NRA_Blur varchar Negative Relative Accommodation blur measurement (e.g., "80 cm", "82 cm", "85 cm").
NRA_Recv varchar Negative Relative Accommodation recovery measurement (e.g., "90 cm", "92 cm", "95 cm").
PRA_Blur varchar Positive Relative Accommodation blur measurement (e.g., "100 cm", "102 cm", "105 cm").
PRA_Recv varchar Positive Relative Accommodation recovery measurement (e.g., "110 cm", "112 cm", "115 cm").
BXC varchar Base-in Cross test measurement (e.g., "5 pd", "6 pd", "7 pd").
ACARatio varchar Accommodative Convergence to Accommodation ratio measurement (e.g., "3:1", "4:1", "5:1").
NPC_WWORx int Near Point of Convergence test with or without prescription glasses (0 for without, 1 for with).
NPA_WWORx int Near Point of Accommodation test with or without prescription glasses (0 for without, 1 for with).
STEREOPSIS_WWORx int Stereopsis test with or without prescription glasses (0 for without, 1 for with).
Dominance_Result varchar Result of the eye dominance test (e.g., "Right eye dominant", "Left eye dominant", "No dominance").
ANISEIKONIA_Impression varchar Impression of the aniseikonia test (e.g., "Mild", "Moderate", "Severe").
