EHI Export Schema
Table departments
Description This table stores departments created by end users.
Column DataType Description
Deptid (Primary Key) int This stores the id for Departments Table. It is primary key. It is auto increment.
name varchar This column used to store department name.
Facilityid int This field stores associated facility id with departments. This column value is mapped with edi_facilities table -> id column value.
DeptTypeid int identifier for the type of department (integer). This is a foreign key that refers to the `DeptTypeid` column in the `department_types` table. Example: 1, 2, 3, etc.
Telephone varchar This field stores telephone number for departments table.
Fax varchar This field stores fax number for departments table.
deleteFlag int This field stores deleteflag value for department. Delete flag will update to 1 if department is deleted.
hl7id varchar This value store department id of other software's department related data. That is help to identify department with ecw software and other EMR software. This is handle by Interface.
