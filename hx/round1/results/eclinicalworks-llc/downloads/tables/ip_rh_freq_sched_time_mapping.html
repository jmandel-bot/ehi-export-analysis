Table ip_rh_freq_sched_time_mapping
Description This table is a mapping table for schedule time and frequency. It stores time mapping details for each frequency.
Column DataType Description
id (Primary Key) int This is an auto-incrementing integer column, serving as the primary key. It uniquely identifies each record in the table.
frequencyId int This column is used to store the frequency id for which time mapping is stored. It contains integer type value. This is foreign key, It references ‘frequencyId' of ‘ip_rh_frequency_list' table.
frequencyTime varchar This column stores Time type value is called frequency time configured with frequency. This column is nullable. The time configured with frequency is stored in HH:MM format.
deleteFlag tinyint This is a tinyint column with a default value of 0 and it cannot be NULL. This could be a logical delete flag, where a value of 1 indicates the record is deleted and 0 indicates it's active.
deleteFlag tinyint This will store 0 and 1 value. 0-not deleted and 1-deleted. This column is used to identify whether row is in use or not. We are using TINYINT as datatype as we just want to store 0 or 1 value. This column will always have a value hence defined as NOT NULL. By default, row will be in use, so we keep default value is 0.
