EHI Export Schema
Table oldrxmain
Description This table stores information about old prescriptions, including details about the medicine, patient, doctor, and prescription status.
Column DataType Description
OldRxId (Primary Key) int This is auto-incremented primary key in This column . This column stores unique id generated for each medication prescribed to patients.
ItemId int This column stores id associated with the medication prescribed to the patient by the end user during current visit. This column refers to itemid column in items table.
MedicineName varchar This column is not used anymore
encounterId int This column stores id of an encounter for which medications are being documented in This column . This column refers to encounterid column in enc table.
PatientsFlag int Medication when added from Current Medication screen patientsflag is set to 1. Medication when added from Treatment screen, patientsflag is set to -1.
DoctorsFlag int This column stores a status which indicates whether the patient currently taking or not taking the medications prescribed. -1 : Taking (T), -2 : Not Taking (N), -3 : Discontinued by the provider/staff (D), -4 : Status Unknown (U). In addition to some system defined statuses, user defined statuses are stored in rxcomments table. This column refers to id column in rxcomments table.
Refill smallint This column is not being used.
AssessId int This column stores id of an assessment for which the medication has been prescribed by the provider/staff. This column refers to itemid column in items table.
ndc_code varchar This column stores National Drug Code (NDC) for the medication being prescribed to the patient during the visit.
FollowUpId int This column is no longer being used.
StartDate datetime This column stores date when patient started taking prescribed medicine. This column is documented by provider/staff and does not represent the date the medication was actually prescribed.
PreparedBy varchar This column store name of the provider/staff who prescribed the medication during the visit.
DaysLeftAdjustNumber int This is retired column and not being used in the product.
awp varchar This column stores Average Wholesale Price (AWP) of the medication proscribed by provider/staff.
awup varchar This column stores Average Wholesale Unit Price (AWUP) for the drug prescribed by provider/staff.
RightPane_DLAN int This is retired column and not being used in the product. Column seem to be related DaysLeftAdjustNumber column in oldrxmain table.
StopDate datetime This column stores date when patient should stop taking prescribed medication. This column is documented by provider/staff. ---This column is of DATE type in MySQL schema.--
rxNotes varchar This column stores notes entered by provider/staff on the user interfacefor each medication.
rxSource varchar This column stores source mentioned by provider/staff on user interfacefor each medication. If medication is prescribed from Treatment screen, this column is automatically set to provider/staff name.
rxSourceId int This column stores id of the user who prescribed the medication while prescribing medication from Treatment screen. This column refers to uid column in user table. This column could also store drugname id from Medispan database.
rxkop varchar This column stores values Yes/No for the Keep On Person (KOP) status. Yes : Medication is marked as KOP, No : Medication is not marked as KOP.
rxDrugSource varchar This column stores source from where drug prescribed by provider/staff can be acquired by the patient.
rxorderno varchar This column stores order number for the set of medications prescribed during a specific encounter. This is application generated number which always begins with 'S' character.
PreparedById int This column store id of the provider/staff who prescribed the medication during the visit. This column refers to uid column in user table.
reviewflag int This column stores values 0/1 to indicate whether the medication has been reviewed by provider/staff for RxApproval functionality. 0 - Medication NOT reviewed, 1 - Medications reviewed. This flag is currently only being used in Corrections product.
reviewedby varchar This column stores name of the provider/staff who has reviewed prescribed medications.
reviewdate datetime This column stores date and time when medication was reviewed by provider/staff.
disconorstopnotes varchar This column stores free text reason documented by provider/staff for discontinuation or stopping the prescribed medication.
LastTaken datetime This column stores date when patient last took the prescribed medication.
isLongTerm tinyint This column stores values 0/1 to indicate if medication prescribed to the patient is long term medication or not. 0 - Not Log Term, 1 - Long Term medication
FillDate varchar This column stores date when prescribed medication gets filled at pharmacy. This Data is transmitted over to eCW from Pharmacy.
FillReason varchar This column stores reason if medication is not filled at the Pharmacy, if mentioned by the pharmacy. This data is transmitted over to eCW from Pharmacy.
FillStatus varchar This column stores status of the prescription at Pharmacy. Filled - If medication is successfully filled, Partially Filled - In case medication is partially filled, Not Filled - In case medication is not filled at pharmacy.
