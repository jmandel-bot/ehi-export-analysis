EHI Export Schema
Table ccmr_casemanagementtaskdetails
Description This table stores information about case management tasks and their details, including task name, risk type, and template.
Column DataType Description
id (Primary Key) int This is auto incremented and Primary Key of This column .
programid int This column store programid mapped with task detail. This column refers to id column in acoprograms table.
taskid int This column refers to taskid column in ccmr_caseManagementTask table.
taskName varchar This column stores task name of program.
riskTypeid int This column stores value 0/1 indicating risk type value of task. 0-Health Assessment,1- Care Plan
templateid int This column refers to itemid column in items table.
taskType int This column stores value 0/1 indicating task type value of task. 0-Assessment,1- Goal
delflag tinyint This column stores the value 0/1 to indicate if this case detail is deleted. 0 - Not Deleted , 1 - Deleted.
displayIndex int This column stores a digit to indicate order in which document should be listed in user interface.
