EHI Export Schema
Table dentalpriority
Description This is the dictionary table of possible priorities that may be assigned to a dental procedure. The table is currently non-configurable and shipped with the entries.
Column DataType Description
id (Primary Key) int This is the primary key column, which is an integer and auto-incremented. It uniquely identifies each record in the dentalpriority table. Example: 1, 2, 3, etc.
priorityNo int This is a secondary id column that's not auto-incremented and is used to define a number for each priority.
priorityName varchar This is a varchar(20) column with a default value of an empty string. It stores the name or description of the priority level. Example: 'High', 'Medium', 'Low', etc.
deleteflag tinyint This is a tinyint(1) column with a default value of '0'. It indicates whether a record has been deleted or not. A value of '0' means the record is not deleted, while a value of '1' means the record is deleted.
