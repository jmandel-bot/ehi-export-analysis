Table univ_demographics
Description This table stores the action logs for unprocessed lab results, including the user who performed the action and the timestamp of the action.
Column DataType Description
StuID int Unique identifier for each student, represented as an integer. Example: 12345
TERM_DESCR varchar Description of the term for which the student is enrolled, represented as a varchar(50). Example: "Fall 2021"
CTZ_CTRY_DESCR varchar Description of the student's citizenship country, represented as a varchar(50). Example: "United States"
CLASS varchar Description of the student's class level, represented as a varchar(50). Example: "Freshman"
PAID_SSF varchar Indicates whether the student has paid the Student Service Fee (SSF), represented as a varchar(50). Example: "Yes" or "No"
PAID_INS varchar Indicates whether the student has paid the insurance fee, represented as a varchar(50). Example: "Yes" or "No"
CURR_COUNTRY varchar Description of the student's current country of residence, represented as a varchar(50). Example: "United States"
OTHER_PHONE varchar The student's alternative phone number, represented as a varchar(50). Example: "+1-555-123-4567"
HOME_COUNTRY varchar Description of the student's home country, represented as a varchar(50). Example: "India"
deleteflag int Indicates whether the record has been deleted, represented as an integer (1 for deleted, 0 for not deleted). Example: 0
