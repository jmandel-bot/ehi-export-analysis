EHI Export Schema
Table asc_gases
Description This table stores information about various gases encountered during a specific event or situation.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the `asc_gases` table. Example: 1, 2, 3.
encounterid int identifier for the specific encounter or event. Foreign key relation with the `enc` table's `encounterid` column. Example: 101, 102, 103.
gasesid int identifier for the specific gas. Foreign key relation with the `asc_ecw_gases_standardlist` table's `id` column. Example: 1, 2, 3.
gasesValue varchar A string value representing the value or concentration of the gas. Example: '10.5', '25.3', '0.0'.
UpdatedBy int identifier for the user who last updated the record. Foreign key relation with the `users` table's `UID` column. Example: 1001, 1002, 1003.
UpdatedTime datetime time stamp of when the record was last updated. Example: '2021-09-01 12:34:56', '2021-09-02 14:23:45'.
notes varchar A string containing any additional notes or comments about the gas record. Example: 'Gas value is approximate', 'Measured during peak hours'.
deleteFlag tinyint A boolean flag indicating if the record has been deleted (1) or not (0). Example: 0, 1.
