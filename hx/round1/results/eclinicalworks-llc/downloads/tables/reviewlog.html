EHI Export Schema
Table reviewlog
Description This table stores information about review logs for different entity types such as Lab X-Ray, Document, Encounter, and Refund Check Print.
Column DataType Description
id (Primary Key) int This column stores primary id key for this column .
Entityid int This column stores Entity id. This column refers to enc.encounterid.
EntityType varchar This column stores type of Entity.
Reviewerid int This column stores Reviewer id. This column refers to users.UID.
ReviewedTime datetime This column stores Reviewed Time.
LockStatus tinyint This column stores Lock Status, tinyint values. It is either 0 or 1.
reason varchar Reason for the review or lock status. It is a string with a maximum length of 255 characters and has a default value of an empty string. For example: Incorrect data, Missing information, etc.
