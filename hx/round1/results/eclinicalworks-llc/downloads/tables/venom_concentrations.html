Table venom_concentrations
Description This table stores information about various venom concentrations used in tests, including their properties and test-related details.
Column DataType Description
id (Primary Key) int Unique identifier for each venom concentration entry. Example: 1, 2, 3.
ENCid int Foreign key referring to the encounter id in another table. Example: 101, 102, 103.
TESTid int Foreign key referring to the test id in another table. Example: 1001, 1002, 1003.
CONCENTRATION varchar The concentration of the venom in the test. Example: "0.1 mg/mL", "0.5 mg/mL", "1 mg/mL".
ITEMid int Foreign key referring to the item id in another table. Example: 501, 502, 503.
WHEAL varchar The size of the wheal (raised area) caused by the venom. Example: "5 mm", "10 mm", "15 mm".
FLARE varchar The size of the flare (redness) caused by the venom. Example: "20 mm", "30 mm", "40 mm".
LOCATION varchar The location on the body where the test was performed. Example: "Left arm", "Right arm", "Left leg".
SEQNO int The sequence number of the test. Example: 1, 2, 3.
POS char Indicates if the test result is positive or negative. Example: "P" for positive, "N" for negative.
DISABLED int Indicates if the test is disabled (1) or not (0). Example: 0, 1.
PSEUDO tinyint Indicates if the test is a pseudo-test (1) or a real test (0). Example: 0, 1.
