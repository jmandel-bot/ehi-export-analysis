Table labdatadetailrefrange
Description This table stores the reference range (upper and lower limit) of a patient's lab test.
Column DataType Description
reportid int This column stores ID of the lab for the patient. This column refers to the reportid column in the labdata table.
itemid int This column stores the ID of the item added to the lab/test of the patient. This column refers to the itemid column from the items table.
range varchar This column stores the reference range of the item added to the test. This data is the consolidation of the data stored in the ValueFrom, ValueTo, and ValueUnits columns in the labranges table.
UpdatedTime datetime This column stores the time stamp in YYYY-MM-DD HH:MM:SS format when a lab reference range is updated.
