Table allergy_shots_maintenance_dose_reasons
Description This table stores information about the reasons for selecting a specific maintenance dose for allergy shots, along with related details such as dilution, dose, and patient information.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table (e.g., 1, 2, 3).
reason varchar Description of the reason for selecting a specific maintenance dose (e.g., "Patient's sensitivity to allergen").
allergyShotid int identifier for the allergy shot, foreign key relation to `allergy_shots` table (`id` column).
patientid int identifier for the patient, foreign key relation to `users` table (`UID` column).
encounterid int identifier for the encounter, foreign key relation to `enc` table (`encounterid` column).
selectedDilutionid int identifier for the selected dilution, foreign key relation to `antigen_concentration_patient` table (`concentration_id` column).
selectedDilution varchar Description of the selected dilution (e.g., "1:100").
selectedDoseid int identifier for the selected dose, foreign key relation to `shot_schedule_dtl` table (`id` column).
selectedDose varchar Description of the selected dose (e.g., "0.5 ml").
maintenanceDoseid int identifier for the maintenance dose, foreign key relation to `allergy_maintenance_dose_shot_schedule` table (`id` column).
maintenanceDoseConcid int identifier for the maintenance dose concentration, foreign key relation to `antigen_concentration_patient` table (`CONCENTRATION_id` column).
maintenanceDoseShotid int identifier for the maintenance dose shot, foreign key relation to `shot_schedule_dtl` table (`id` column).
createdby int identifier for the user who created the record, foreign key relation to `users` table (`UID` column).
createdon datetime time stamp of when the record was created (e.g., "2021-09-01 12:34:56").
deleteflag tinyint Flag to indicate if the record has been deleted (0 for not deleted, 1 for deleted).
