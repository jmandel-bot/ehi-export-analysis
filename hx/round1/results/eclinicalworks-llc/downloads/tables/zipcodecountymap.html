Table zipcodecountymap
Description This table stores information about County information based on zip numbers
Column DataType Description
id (Primary Key) int This is an auto-incrementing integer column that serves as the primary key for the table. It uniquely identifies each record in the table. Example: 1, 2, 3, etc.
zipcode varchar This is a varchar(10) column that stores the zip code. It is a required field and cannot be NULL. Example: "12345", "98765", "54321", etc.
statecode char This is a char(2) column that stores the state code. It is a required field and cannot be NULL. Example: "NY", "CA", "TX", etc.
county char This is a char(3) column that stores the county code. It is a required field and cannot be NULL. Example: "001", "002", "003", etc.
deleteflag int This is an int(2) column that stores a flag indicating whether the record is deleted or not. It has a default value of '0'. Example: 0 (not deleted), 1 (deleted).
totaladdressratio decimal This is a double column that stores the total address ratio for the given zip code, state code, and county code combination. It can be NULL if the ratio is not available. Example: 0.75, 1.0, 0.5, etc.
