EHI Export Schema
Table enccosigner
Description This table stores information about a co-signed or reviewed encounter, along with the reviewer, reviewed time, and review log identifier.
Column DataType Description
id (Primary Key) int This is auto-incremented primary key in this column.
Encounterid int This column stores id of an encounter which was reviewed or cosigned by other provider than the one who documented it. This column refers to encounterid column in enc table.
ReviewLogid int This column stores the id of a review log entry which indicates a unique entry for each review of a Progress Notes. This column refers to the id column in the reviewlog table.
Reviewerid int This column stores the id of a provider who reviewed or cosigned Progress Notes for an encounter. This column refers to the UID column in the user table.
ReviewedTime datetime This column stores the date and time when a provider reviewed or cosign Progress Notes for an encounter. Date and time are stored in the format: 'YYYY-MM-DD hh:mm:ss.000'.
type tinyint This column stores values "1" or "2" to indicate whether Progress Notes were cosigned or reviewed by other provider. A "1" means they weere cosigned by another provider, a "2" means it was reviewed by another provider.
