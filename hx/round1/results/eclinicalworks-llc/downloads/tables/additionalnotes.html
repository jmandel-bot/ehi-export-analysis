EHI Export Schema
Table additionalnotes
Description This table stores additional notes related to patients, including information about the user who created the note, the date and time of the note, and any associated subject data.
Column DataType Description
id (Primary Key) int Unique identifier for each additional note. Example: 1, 2, 3.
patientid int identifier for the patient associated with the note. Example: 101, 102, 103.
LoggedInUID int identifier for the user who was logged in when the note was created. Example: 1001, 1002, 1003.
date datetime The date when the note was created. Example: 2021-09-15, 2021-09-16, 2021-09-17.
time datetime The time when the note was created. Example: 14:30:00, 15:45:00, 16:00:00.
Message text The text content of the additional note. Example: patient reported mild headache, patient's blood pressure is stable, patient is recovering well from surgery.
deleteFlag smallint Indicates whether the note has been deleted (1) or not (0). Example: 0, 1.
moduleName varchar Name of the module where the note was created. Example: PatientHistory, lLabResults, Prescriptions.
createdOn datetime The date and time when the note was created. Example: 2021-09-15 14:30:00, 2021-09-16 15:45:00, 2021-09-17 16:00:00.
createdBy int identifier for the user who created the note. Example: 1001, 1002, 1003. Foreign key relation: users (UID).
subjectData varchar Additional subject-related data for the note. Example: Migraine, Hypertension, Post-surgery.
