EHI Export Schema
Table structvisionexam
Description This table stores structured data for an eye examination in an encounter, including observations and notes for both left and right eyes.
Column DataType Description
id (Primary Key) int This is the primary key, an auto-incrementing integer that uniquely identifies each record in the table. Example: 1, 2, 3, etc.
encounterid int This column stores the encounter id for which structured vision data have been documented. This column refers to column encounterid from enc table.
fieldGroupid int This column stores the fieldGroup id for which structured vision data have been documented. This column refers to column itemid from items table.
fieldid int This column stores the field id for which structured vision data have been documented. This column refers to column propid from properties table.
detailid int This column stores the id of the structured question associated with the vision of the patient. This column refers to column id from structdatadetail table.
leftEyeObservations varchar Text containing observations for the left eye. Example: "Normal vision", "Blurry vision".
rightEyeObservations varchar Text containing observations for the right eye. Example: "Normal vision", "Blurry vision".
leftEyeNotes varchar Text containing additional notes for the left eye. Example: "Patient reports discomfort", "No issues reported".
rightEyeNotes varchar Text containing additional notes for the right eye. Example: "Patient reports discomfort", "No issues reported".
leftEyeObsid varchar This column stores the leftEyeObs id for which structured vision data have been documented. This column refers to column optionid from structdataoptions table.
rightEyeObsid varchar This column stores the rightEyeObs id for which structured vision data have been documented. This column refers to column optionid from structdataoptions table.
