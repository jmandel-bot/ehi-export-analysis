EHI Export Schema
Table physicalexam
Description This table stores patient's physical exam data for an encounter.
Column DataType Description
encounterid int Stores encounter id which the physical examination is documented for. Column refers to encounterid column in enc table.
catid int This column is to store category id which is used to display the category name in the User Interface. This column refers to column Parentid from items table.
itemid int This column stores the itemid which is associated to the category selected in the User Interface. This column refers to the column itemid in items table.
propid int This column is not actively used from This column hence it stores the assigned default value.
value text This column stores free text notes entered by provider/staff while performing physical examination. Pre defined Options could also be selected by provider/staff in the User Interface.
value2 text This column stores notes entered by provider/staff while performing physical examination in HTML format.
displayIndex smallint This column stores the sequence number of the itemids associated with the categories, which is used to display them in the User Interface.
