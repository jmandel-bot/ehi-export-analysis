EHI Export Schema
Table encdetail
Description This table stores information about specific encounters, including flags for occurrences and emergencies, as well as fee schedule details.
Column DataType Description
encounterid int This column stores the id of the encounter for which an appointment fee schedule is saved. This column refers to the column encounter id from the enc table.
occFlag int Needs further discussion.
emergencyFlag int This column stores flags "0" or "1". A "1" means it is an emergency associated encounter, and a "0" means otherwise.
FeeSchid int This column stores the id of of the fee schedule selected for the encounter. This column refers to column id from the feeschlist table.
