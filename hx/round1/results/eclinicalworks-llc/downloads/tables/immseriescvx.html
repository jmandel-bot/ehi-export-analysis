EHI Export Schema
Table immseriescvx
Description This table stores immunization series and its CVX codes. (static table). This table stores the list of Immunization Series defined by CDC and the CVXs that belong to each series. This table is utilized in Immunization Schedule tab of Immunization History screen.
Column DataType Description
ImmSeriesid int This column stores id of immunization series.
ImmSeriesName varchar This column stores Immunization Series Name available at Center for Disease Control.
ImmSeriesAlias varchar This column stores Immunization Series Name Alias that will be displayed on Immunization Schedule Screen.
membercvx varchar This column stores list of CVXs belonging to this series.
