Table inpatientvisit
Description This table stores information about inpatient visits, including the encounter ID, admission date, discharge date, and whether the patient has been discharged or not.
Column DataType Description
encounterId (Primary Key) int This is encounterId from enc table
AdmDate datetime This is a date column that stores the admission date for the inpatient visit. Example: '2021-09-01'
DischargedDate datetime This is a date column that stores the discharge date for the inpatient visit. Example: '2021-09-10'
discharged int This is an integer column that indicates whether the patient has been discharged or not. A value of '0' means the patient has not been discharged, while a value of '1' means the patient has been discharged. Example: 1
