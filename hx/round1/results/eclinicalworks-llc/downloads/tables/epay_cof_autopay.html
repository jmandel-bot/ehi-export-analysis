EHI Export Schema
Table epay_cof_autopay
Description This table stores information about automatic payments for patients in a medical facility using the ePay system.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
vmid varchar Storing the gUID for a particular COF data.
patientid int identifier for the patient associated with the card, refers to the `pid` column in the `patients` table.
autoPayModule tinyint identifier for the module handling the automatic payment. Example: 1, 2, 3.
autoPayModuleid int identifier for the specific instance of the automatic payment module. Default value is '0'. Example: 0, 1, 2.
autoPayEndDate date Date when the automatic payment is scheduled to end. Example: '2022-12-31', '2023-01-31'.
autoPayAmount decimal Amount of the automatic payment in decimal format with 9 digits and 2 decimal places. Example: 100.00, 250.50.
isActive tinyint Flag indicating if the automatic payment is active (1) or not (0). Example: 0, 1.
isDeleted tinyint Flag indicating if the record has been deleted (1) or not (0). Example: 0, 1.
createdBy int identifier for the user who created the record. Example: 1, 2, 3.
createdOn datetime Date and time when the record was created. Example: '2022-01-01 12:34:56', '2022-02-01 14:56:23'.
updatedBy int identifier for the user who last updated the record. Example: 1, 2, 3.
updatedOn datetime Date and time when the record was last updated. Example: '2022-01-15 10:23:45', '2022-02-10 16:45:12'.
