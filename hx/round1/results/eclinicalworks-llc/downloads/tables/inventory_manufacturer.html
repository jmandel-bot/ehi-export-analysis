Table inventory_manufacturer
Description This table stores information about manufacturers in the inventory system.
Column DataType Description
id (Primary Key) int Unique identifier for each manufacturer record. (Example: 1, 2, 3)
Name varchar Name of the manufacturer. (Example: "Apple Inc.", "Samsung Electronics")
Email varchar Email address of the manufacturer. (Example: "contact@apple.com", "info@samsung.com")
Phone varchar Phone number of the manufacturer. (Example: "+1-800-692-7753", "+1-800-726-7864")
Fax varchar Fax number of the manufacturer. (Example: "+1-800-123-4567", "+1-800-987-6543")
IsDeleted tinyint A flag indicating whether the manufacturer record is deleted (1) or not (0). (Example: 0, 1)
UpdatedBy int The unique identifier of the user who last updated the manufacturer record. This is a foreign key referring to the `id` column in the `users` table. (Example: 1, 2)
UpdatedOn datetime The date and time when the manufacturer record was last updated. (Example: "2021-09-01 12:34:56", "2021-09-02 14:56:23")
Code varchar A unique code assigned to the manufacturer. (Example: "APL", "SAM")
source tinyint A flag indicating the source of the manufacturer record.
hubid varchar A unique identifier in the inventory hub that associated with the manufacturer. (Example: "HUB001", "HUB002")
ManufacturerNo varchar A unique number assigned to the manufacturer. (Example: "M001", "M002")
