EHI Export Schema
Table seasoninfo
Description The table is used to maintain seasons details in athletics modules
Column DataType Description
id (Primary Key) int Unique identifier for each season record. Example: 1, 2, 3.
NAME varchar The name of the season. Example: "Spring", "Summer", "Fall", "Winter".
YEAR varchar The year of the season. Example: "2021", "2022", "2023".
startdate varchar The start date of the season in a string format. Example: "01-01-2021", "04-01-2021", "07-01-2021".
enddate varchar The end date of the season in a string format. Example: "03-31-2021", "06-30-2021", "09-30-2021".
deleteFlag tinyint A flag to indicate if the record is deleted. 0 means not deleted, and 1 means deleted. Example: 0, 1.
