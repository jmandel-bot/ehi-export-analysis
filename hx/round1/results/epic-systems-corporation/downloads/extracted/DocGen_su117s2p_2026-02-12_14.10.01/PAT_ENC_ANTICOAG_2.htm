PAT_ENC_ANTICOAG_2
Description:
This table contains information related to anticoagulation monitoring encounters. The following columns apply only when using the calendar tracking navigator section (LVN 617) for the anticoagulation encounter: AC_NO_DOSE_GIVEN_YN and AC_NO_INR_YN. There is a related table, EPI_ANTICOAG, for the most recent anticoagulation episode information.

Primary Key
Column Name Ordinal Position
PAT_ENC_CSN_ID 1

Column Information
Name Type Discontinued?
1 PAT_ENC_CSN_ID NUMERIC No
The unique contact serial number for this contact. This number is unique across all patient encounters in your system. If you use IntraConnect, this is the Unique Contact Identifier (UCI).
2 PAT_ENC_DATE_REAL FLOAT No
A unique contact date in decimal format. The integer portion of the number indicates the date of contact. The digits after the decimal distinguish different contacts on the same date and are unique for each contact on that date. For example, .00 is the first/only contact, .01 is the second contact, etc.
3 CONTACT_DATE DATETIME No
The date of this contact in calendar format.
4 AC_NO_DOSE_GIVEN_YN VARCHAR No
Tracks if the "Patient was not given dosing instructions" was selected in an anticoagulation encounter.
May contain organization-specific values: No
Category Entries:
No
Yes
5 AC_NO_INR_YN VARCHAR No
Tracks if "No INR" was selected for dosing in an anticoagulation encounter
The category values for this column were already listed for column: AC_NO_DOSE_GIVEN_YN
6 PERCENTAGE_TTR NUMERIC No
The patient's time in therapeutic range (TTR).
7 TTR_END_DATE NUMERIC No
The end date used when TTR was calculated.
8 TTR_START_DATE NUMERIC No
The start date used for TTR calculation.
9 TTR_DAYS_ON_THERAPY NUMERIC No
Number of days of therapy considered for TTR calculation.
10 AC_LINKED_ADM_PAT_ENC_CSN_ID NUMERIC No
This items contains the CSN for the linked admission whose discharge instructions we are saving this to encounter
11 EPISODE_RETURN_DATE DATETIME No
Tracks the return date set in the episode at the time of linking to the encounter
12 AC_SYNCED_ORDER_ID NUMERIC No
Tracks the order ID most recently synced to the anticoagulation tracking calendar
13 AC_DOSING_MED_SIMPLE_GENERIC_C_NAME VARCHAR No
Category ID of the simple generic vitamin K antagonist medication being used to dose the patient in an anticoagulation encounter.
May contain organization-specific values: Yes
