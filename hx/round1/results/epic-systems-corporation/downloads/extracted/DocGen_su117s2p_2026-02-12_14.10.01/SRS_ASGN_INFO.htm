SRS_ASGN_INFO
Description:
Table that stores series assignment related information as well as information on series answer record status.

Primary Key
Column Name Ordinal Position
SERIES_ANS_ID 1

Column Information
Name Type Discontinued?
1 SERIES_ANS_ID NUMERIC No
This column stores the unique identifier for the series answer record.
2 RECORD_STATUS_C_NAME VARCHAR No
This column stores the status of the series answer record if it is not active.
May contain organization-specific values: No
Category Entries:
Soft Deleted
Hidden
Hidden and Soft Deleted
3 SERIES_ID NUMERIC No
This column stores the ID of the series answer record.
4 SERIES_ID_SERIES_NAME VARCHAR No
The name for the questionnaire series record.
5 SERIES_CSN NUMERIC No
This column stores the contact date (DAT) for the series answer record.
6 SERIES_TYPE_C_NAME VARCHAR No
Stores the questionnaire series type.
May contain organization-specific values: No
Category Entries:
Patient Entered Questionnaire Series
7 SERIES_STATUS_C_NAME VARCHAR No
This column stores the status of the series answer record if it is active.
May contain organization-specific values: No
Category Entries:
Active
Completed
Disabled - Provider
Disabled - Patient Opted Out
Disabled - Found New Treatment
Disabled - Batch Job
Disabled - Patient Merge
Disabled - Contact Mover
Disabled - Procedure Pass
Disabled - Patient Expired
Disabled - Research Activity Retraction
Pending Enable
8 ORDERED_BY_USER_ID VARCHAR No
If a series is ordered through Order Entry or through an OurPractice Advisory, this column stores who ordered it for future message routing purposes.
9 ORDERED_BY_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
10 SERIES_PAT_ID VARCHAR No
To store ID (EPT id) of the patient who is answering this questionnaire series.
11 HQW_CREATION_DT DATETIME No
This column stores the date the series answer record was created.
12 INIT_ENC_CSN_ID NUMERIC No
This item stores the contact serial number (CSN) of the encounter where this series is assigned to the user. This item will be empty if the series was assigned from Reporting Workbench.
13 PARENT_SRS_ANS_ID NUMERIC No
This column stores the ID of the parent questionnaire series answer record.
14 PREV_SRS_ANS_ID NUMERIC No
This column stores the ID of the previous anchor event questionnaire series answer record.
15 LINKED_ENC_CSN_ID NUMERIC No
This item stores the encounter this questionnaire series is linked to. If the series is linked to an encounter, then the contact date (DAT) of the encounter will be used for evaluating anchor event rules and extensions.
16 LINKED_PROC_PASS_ID NUMERIC No
This item stores the ID of the Procedure Pass linked to this series. This will only be populated if the series is assigned using the Procedure Pass workflow.
17 ASSIGNING_ORD_ID NUMERIC No
This item stores the order that assigned this questionnaire series.
18 ENROLL_ID NUMERIC No
This item stores the ID of the linked patient-study association if it is linked to an association.
19 DEPARTMENT_ID_EXTERNAL_NAME VARCHAR No
The external name of the department record. This is often used in patient correspondence such as reminder letters.
