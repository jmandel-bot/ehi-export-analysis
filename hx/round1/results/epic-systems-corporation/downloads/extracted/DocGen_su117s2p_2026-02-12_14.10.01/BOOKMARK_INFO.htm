BOOKMARK_INFO
Description:
This table will extract bookmarks from the HNO database.

Primary Key
Column Name Ordinal Position
NOTE_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 NOTE_ID VARCHAR No
The unique identifier (.1 item) for the note record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 TAG_EXTERNAL_LOCAL_IDENT VARCHAR No
The locally-assigned unique identifier of the tagged external note.
4 TAG_EXTERNAL_SOURCE_DXR_CSN_ID NUMERIC No
The ID of the tagged note's source DXR contact.
5 TAG_EXTERNAL_ENC_START_DTTM DATETIME (Local) No
The start instant of the encounter associated with the tagged external note.
6 TAG_EXTERNAL_ENC_END_DTTM DATETIME (Local) No
The end instant of the encounter associated with the tagged external note.
7 TAG_EXTERNAL_ENC_TYPE_C_NAME VARCHAR No
The mapped encounter type ID of the encounter associated with the tagged external note.
May contain organization-specific values: Yes
