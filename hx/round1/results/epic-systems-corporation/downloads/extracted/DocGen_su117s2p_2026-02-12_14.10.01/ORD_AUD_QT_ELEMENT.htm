ORD_AUD_QT_ELEMENT
Description:
This table contains the auditing information about the quality tracking element.

Primary Key
Column Name Ordinal Position
ORDER_ID 1
GROUP_LINE 2
VALUE_LINE 3

Column Information
Name Type Discontinued?
1 ORDER_ID NUMERIC No
The unique identifier (.1 item) for the order record.
2 GROUP_LINE INTEGER No
The line number for the information associated with this record.
3 VALUE_LINE INTEGER No
The line number of one of the multiple values associated with a specific group of data within this record.
4 AUD_QT_ELEMENT_C_NAME VARCHAR No
The quality tracking element category ID of the audited order. The source item is located at QUALITY_TRACKING.ELEMENT_C.
May contain organization-specific values: Yes
