LNO_LAB_MAIN
Description:
Primary Lab Result Report (LNO) data.

Primary Key
Column Name Ordinal Position
RECORD_ID 1

Column Information
Name Type Discontinued?
1 RECORD_ID NUMERIC No
The unique identifier for the LNO record.
2 LAB_REP_DATA_DATE DATETIME No
The date the report data was filled into the Patient Summary Extract (LNO) record. This usually contains the report date.
3 LAB_ID_LAB_NAME VARCHAR No
The name of the lab record.
4 DEPT_ID_EXTERNAL_NAME VARCHAR No
The external name of the department record. This is often used in patient correspondence such as reminder letters.
5 LAB_REPORT_TYPE_C_NAME VARCHAR No
The result report type category number for the result report.
May contain organization-specific values: No
Category Entries:
Single Specimen Report
End of Day Report
End of Week Report
Discharge Report
STAT Report
Encounter Level Report
Multiple Encounter Report
Hold till all final Report
Any Final Report
Any Prelim or Final Report
Final Summary Report
Adhoc Report
Master Record
6 REQUISITION_ID NUMERIC No
Stores the requisition record if any for which this record was created.
7 SUBMITTER_ID NUMERIC No
This is the submitter associated with the specimen on the report.
8 SUBMITTER_ID_RECORD_NAME VARCHAR No
The name of the submitter record.
9 REQ_GRP_ID NUMERIC No
This is the reference lab patient associated with the specimen on the report.
10 CASE_ID NUMERIC No
Stores the case record for which this record was created.
