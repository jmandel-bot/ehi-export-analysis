TX_DIAG_ALT
Description:
Table lists all diagnoses matched to the charge from the alternative diagnosis code set.

Primary Key
Column Name Ordinal Position
TX_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 TX_ID NUMERIC No
The unique internal ID of the accounts receivable transaction record representing this charge.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 DX_ID_DX_NAME VARCHAR No
The name of the diagnosis.
4 DX_QUALIFIER_C_NAME VARCHAR No
The diagnosis qualifier category ID for the transaction. Indicates if diagnosis is: 1 - Active 2 - Acute 3 - Chronic.
May contain organization-specific values: Yes
