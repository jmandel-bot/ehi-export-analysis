COD_RECORD_INFO
Description:
This table stores information about Clinical Documentation Improvement (CDI) reviews.

Primary Key
Column Name Ordinal Position
CODING_RECORD_ID 1

Column Information
Name Type Discontinued?
1 CODING_RECORD_ID NUMERIC No
The unique identifier for the coding record.
2 RECORD_NAME VARCHAR No
The name of the patient associated with the coding record at the time the coding record was created.
3 RECORD_STATUS_C_NAME VARCHAR No
The record status of the coding record (hidden, soft-deleted, etc.).
May contain organization-specific values: No
Category Entries:
Soft Deleted
Hidden
Hidden and Soft Deleted
4 COD_RCD_TYPE_C_NAME VARCHAR No
The record type of the coding record.
May contain organization-specific values: No
Category Entries:
CDI Review
Doc Review Tracking
Risk Adjustment CDI Review
Coding Over Time
Health Plan Risk Adjustment Evidence
5 RECORD_CREATION_DT DATETIME No
The creation date of the coding record.
6 HSP_ACCOUNT_ID NUMERIC No
The hospital account which is linked to the coding record.
7 CDI_STATUS_C_NAME VARCHAR No
The current clinical documentation improvement (CDI) status of the account.
May contain organization-specific values: Yes
Category Entries:
Not Started
In Progress
Physician Query
All Reviews Completed
8 CDI_ASGN_USER_ID VARCHAR No
This item stores the clinical documentation improvement (CDI) user currently assigned to this account.
9 CDI_ASGN_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
10 CDI_STATUS_COMMENT VARCHAR No
The comment associated with the current clinical documentation improvement (CDI) status.
11 RA_CDI_PAT_ID VARCHAR No
The patient that this COD record is for.
