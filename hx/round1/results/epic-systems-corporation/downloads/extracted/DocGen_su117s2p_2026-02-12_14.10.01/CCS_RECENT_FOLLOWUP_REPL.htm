CCS_RECENT_FOLLOWUP_REPL
Description:
This table should not be queried directly. It is used by the CCS_RECENT_FOLLOWUP view and will ultimately be deprecated.
The table contains information about a patient's most recent outstanding follow-up actions from Cervical Cancer Screenings.

Primary Key
Column Name Ordinal Position
PAT_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 PAT_ID VARCHAR No
The unique ID of the patient record for this row. This column is frequently used to link to the PATIENT table.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 CCS_FOLLOWUP_C_NAME VARCHAR No
The specific cervical cancer screening (CCS) follow-up action.
May contain organization-specific values: Yes
