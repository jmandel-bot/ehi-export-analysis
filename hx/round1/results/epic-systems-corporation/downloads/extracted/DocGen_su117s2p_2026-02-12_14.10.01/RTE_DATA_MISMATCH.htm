RTE_DATA_MISMATCH
Description:
Contains information about real time eligibility data mismatch between coverage data in Epic and the coverage data in the payer system.

Primary Key
Column Name Ordinal Position
COVERAGE_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 COVERAGE_ID NUMERIC No
The unique identifier (.1 item) for the coverage record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 RTE_DATA_MISMATCH_C_NAME VARCHAR No
The RTE data mismatch category ID for the coverage.
May contain organization-specific values: No
