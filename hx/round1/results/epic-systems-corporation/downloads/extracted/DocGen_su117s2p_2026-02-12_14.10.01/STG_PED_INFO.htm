STG_PED_INFO
Description:
This table stores pediatric cancer staging information. Note that the only records extracted into this table are those with non-null data in the Pediatric Stage (I STG 300) item (Clarity column PEDIATRIC_STAGE_C).

Primary Key
Column Name Ordinal Position
STAGE_ID 1

Column Information
Name Type Discontinued?
1 STAGE_ID NUMERIC No
The cancer stage ID.
2 PEDIATRIC_STAGE_C_NAME VARCHAR No
The pediatric stage.
May contain organization-specific values: Yes
3 STAGE_TITLE_NM VARCHAR No
The title of this cancer stage row, usually indicative of the body site or cancer type being staged.
