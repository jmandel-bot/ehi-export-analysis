AUTH_REQ_HX_NOTIF_OVRIDE
Description:
This table contains overrides to authorization request notification milestones. The values in this table override system computer values for turnaround time.

Primary Key
Column Name Ordinal Position
AUTH_REQUEST_ID 1
CONTACT_DATE_REAL 2
LINE 3

Column Information
Name Type Discontinued?
1 AUTH_REQUEST_ID NUMERIC No
The unique identifier (.1 item) for the authorization request record.
2 CONTACT_DATE_REAL FLOAT No
A unique contact date in decimal format. The integer portion of the number indicates the date of contact. The digits after the decimal distinguish different contacts on the same date and are unique for each contact on that date. For example, .00 is the first/only contact, .01 is the second contact, etc.
3 LINE INTEGER No
The line number for the information associated with this contact. Multiple pieces of information can be associated with this contact.
4 COMM_PURPOSE_C_NAME VARCHAR No
The communication purpose category ID of the notification override.
May contain organization-specific values: No
Category Entries:
Notification of Decision
Request for Additional Information
Notification of Extension
Request to Expedite
Request for Authorization
Receipt of Additional Information
Denial of Request to Expedite
P2P Review Offer
P2P Review Completed
Notification of Approval
Notification of Denial
Notification of Dismissal
P2P Review Offer (Post-Decision)
P2P Review Completed (Post-Decision)
5 NOTIF_RECPNT_TYPE_C_NAME VARCHAR No
The recipient type category ID of the notification override.
May contain organization-specific values: No
Category Entries:
Member
Provider
Authorized Representative
6 NOTIF_METHOD_C_NAME VARCHAR No
The notification method category ID of the notification override.
May contain organization-specific values: No
Category Entries:
Oral
Written
Oral and Written
7 OVERRIDE_LOCAL_DTTM DATETIME (Local) No
The date and time (local) of the notification override.
8 OVERRIDE_UTC_DTTM DATETIME (UTC) No
The date and time (UTC) of the notification override.
