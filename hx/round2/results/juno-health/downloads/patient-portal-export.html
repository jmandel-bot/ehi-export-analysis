EHI Export with Juno Patient Portal

Juno Patient Portal has been certified to (b)(10) under the USCDIv1 definition.

Juno Patient Portal Electronic Health Information export

The Juno Patient Portal UI provides the ability to export single-patient and patient population export downloads in C-CDA standardized format.

Customers and their staff, with enhanced permissions, will be allowed to export single-patient or patient population exports using C-CDA Standards. These exports can be run on-demand. Configuration of the Juno Patient Portal UI allows these functions and features to be enabled.

When extracting and running a report the UI display screen will show that the data was queued and provide the completion status to proceed with the download. â€‹

Report Extraction Process
On-demand data export reports allow the user to select all patients, a single-patient, or a subset of patients.

  1. Login to Juno Patient Portal and navigate to Organization>Reports>Regulatory Reports.
  2. Select Electronic Health Information Request and click Run Report.
  3. Enter a 30 day report period.
  4. Select Patients and click Process Report.
  5. Navigate to Regulatory Reports>Processing Reports to view and download reports.
