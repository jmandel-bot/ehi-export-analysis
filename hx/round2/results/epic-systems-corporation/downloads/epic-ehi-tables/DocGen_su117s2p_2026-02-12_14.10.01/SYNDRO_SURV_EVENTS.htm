Column Name Ordinal Position

Column Information
Name Type Discontinued?
1 SYNDRO_SURV_EVENT_ID NUMERIC No
The unique identifier (.1 item) for the syndromic surveillance event record.

Stores the record status (hidden, soft-deleted, etc.).
May contain organization-specific values: No

The unique ID of the syndrome (HFR) for the syndromic surveillance event.

The name of the registry record.
5 PAT_ID VARCHAR No
The unique ID of the patient for the syndromic surveillance event.

The external name of the department record. This is often used in patient correspondence such as reminder letters.
8 POSTAL_CODE_RAW VARCHAR No
The raw postal code that the syndromic surveillance event contributes to in the syndrome trend.

Indicates whether the syndromic surveillance event is considered hospital onset. 'Y' indicates that the event is considered hospital onset. 'N' or NULL indicate that the event is not considered hospital onset.
