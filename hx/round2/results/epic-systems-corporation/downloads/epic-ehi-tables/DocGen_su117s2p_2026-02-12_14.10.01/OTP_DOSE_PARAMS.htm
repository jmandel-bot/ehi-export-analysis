OTP_DOSE_PARAMS
Description:
The dosing parameters for the order template.

Primary Key
Column Name Ordinal Position

Name Type Discontinued?

Past measured weight

Ideal body weight
Adjusted body weight

Lean body weight
Dosing weight
Birth Weight

The dosing height in inches for the order template in this row

The date/time in external format when the dosing height was recorded for the order template in this row.
7 OTP_HT_SOURCE_C_NAME VARCHAR No
The source of the dosing height recorded for the order template in this row.
May contain organization-specific values: Yes
Category Entries:
Treatment plan height
Past measured height
Most current measured height (Actual)
User Entry
Other
8 OTP_HT_COMMENTS VARCHAR No
The comments for the dosing height recorded for the order template in this row.

The dosing body surface area (BSA) for the order template in this row.

The source of the dosing BSA for the order template in this row.

Category Entries:

Based on most recent measured weight and height (actual)
User Entry

The dosing BSA comments recorded for the order template in this row.

Stores order-specific dosing weight in kg.
14 OTP_DOSING_BSA_ORIGINAL NUMERIC No
The original (uncapped) BSA of an order
15 RECENT_RULE_DAT NUMERIC No
If this medication has a dosing rule, this will be the contact date (DAT) of the most recent dosing rule that was checked and matched the patient.
