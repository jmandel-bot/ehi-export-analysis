FLAGGED_QUESTIONS
Description:
Questions in this answer record that have been flagged, along with reason and linked records, if applicable.

Primary Key
Column Name Ordinal Position
ANSWER_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 ANSWER_ID VARCHAR No
The unique identifier for the answer record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 FLAGGED_QUESTION_ID_QUEST_NAME VARCHAR No
The name of the question record.
4 FLAG_REASON_C_NAME VARCHAR No
Stores the reason the question was flagged.
May contain organization-specific values: No
Category Entries:
OurPractice Advisory
5 FLAG_ALT_CSN_ID NUMERIC No
Stores the unique contact serial number for the alert that caused the question to be flagged.
