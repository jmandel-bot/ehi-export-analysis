AN_HSB_CONTEXTS
Description:
This table contains information about the types of contexts that have been documented on an anesthesia record. A record can be associated with a context of Anesthesia, Sedation, or Perfusion based on the activity with which a user opened the record.

Primary Key
Column Name Ordinal Position
SUMMARY_BLOCK_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 SUMMARY_BLOCK_ID NUMERIC No
The unique identifier for the Episode.
2 LINE INTEGER No
The line number for episode context information associated with each anesthesia record. This includes the episode context, user ID, audit time for when the context was added to the record, and whether an automatic macro was attempted. There can be multiple contexts associated with each anesthesia record, with each entry corresponding to a different line.
3 AN_EPISODE_CONTEXT_C_NAME VARCHAR No
The context that has been associated with the anesthesia record. The possible contexts include Anesthesia, Sedation, and Perfusion.
May contain organization-specific values: No
Category Entries:
Anesthesia
Sedation
Perfusion
4 RECORD_AUDIT_USER_ID VARCHAR No
Stores the audit information for who caused the episode context to be added to the record.
5 RECORD_AUDIT_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
6 RECORD_AUDIT_UTC_DTTM DATETIME (UTC) No
Stores the audit information for when the episode context was added to the record.
