ABN_NOTE_CONTACT_SERVICE
Description:
This extract table contains information for items in the General Use Notes (HNO) ABN Procedures (HNO 2310) related group. These items are populated with information from the Advanced Beneficiary Notice (ABN) waiver form related to the services that triggered an ABN. A row in this table represents one failed service for which an ABN was generated.

Primary Key
Column Name Ordinal Position
NOTE_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 NOTE_ID VARCHAR No
The unique ID of the note (HNO) record that contains the Advance Beneficiary Notice (ABN) information.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 ABN_PROC_ID_PROC_NAME VARCHAR No
The name of each procedure.
4 ABN_REASON_FOR_NONCOVERAGE VARCHAR No
This item stores an additional user-specified reason for ABN. The possible reasons based on LCD rules are stored in item 2313 of the ABN record.
5 ABN_TRIGGERING_REASON VARCHAR No
Stores the reason for triggering an ABN
6 ABN_PRICE_PER_SERVICE NUMERIC No
Contains the price per procedure (orderable) or medication for an ABN waiver form. This price will take into account quantity of the procedure ordered.
7 ABN_MODIFIER_USED VARCHAR No
This item holds the modifiers used by price generation code to ensure that we are calculating the price correctly when updating the ABN record.
8 ABN_ORIGINAL_PRICE_PER_SERVICE NUMERIC No
Stores the price generated by the system for a procedure or medication that generated the ABN
9 ABN_MEDICATION_ID_MEDICATION_NAME VARCHAR No
The name of this medication record.
10 ABN_FAILED_LCD_C VARCHAR No
Indicates which LCD category values the given service failed when run through the blackbox
11 ABN_SERVICE_DUR VARCHAR No
Holds a string displayed in the ABN showing the duration for an ordered procedure or medication.
