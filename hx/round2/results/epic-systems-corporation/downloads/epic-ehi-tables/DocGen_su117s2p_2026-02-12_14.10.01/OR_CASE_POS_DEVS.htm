OR_CASE_POS_DEVS
Description:
The OR_CASE_POS_DEVS table contains the positioning devices information for the current case.

Primary Key
Column Name Ordinal Position
OR_CASE_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 OR_CASE_ID VARCHAR No
The unique ID of the case record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 POS_DEV_TYPE_C_NAME VARCHAR No
The positioning device type category number for the case record.
May contain organization-specific values: Yes
4 POS_DEV_REQ_YN_NAME VARCHAR No
Indicates whether a positioning device will be required for the case. Y indicates that the device will be required for the case. N indicates that the device will not be required. Null indicates that there is no value configured for the case.
May contain organization-specific values: No
Category Entries:
Yes
No
5 POS_DEV_START_TM INTEGER No
The time when the positioning device was started for the case.
6 POS_DEV_END_TM INTEGER No
The time when the positioning device was ended for the case.
7 POS_DEV_PANEL_NUM VARCHAR No
The panel numbers of the case in which the positioning device is used. This is a caret-delimited string of panel numbers.
8 POS_DEV_DEFAULT_YN_NAME VARCHAR No
Indicates whether a positioning device will default from the case. Y indicates that the device will default from the case. N indicates that the device will not default from the case. Null indicates that there is no value configured for the case.
May contain organization-specific values: No
Category Entries:
No
Yes
9 POS_DEV_BODYPART_C_NAME VARCHAR No
The body part type category number for the case record.
May contain organization-specific values: No
Category Entries:
Body
Head
Right Arm
Left Arm
Right Leg
Left Leg
