CL_RMT_SVC_LVL_REF
Description:
This table contains information relating to the Administrative Reference Number (REF) segment on the service line level.

Primary Key
Column Name Ordinal Position
IMAGE_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 IMAGE_ID VARCHAR No
This is the ID for the remittance image record with related remit claim references.
2 LINE INTEGER No
The line number in the results of a query. Each instance of claim remit information will have its own line.
3 REF_SVC_LN VARCHAR No
This is the service line for reference segment.
4 SVC_REF_ID_QUAL_C_NAME VARCHAR No
This is the service line level reference segment ID qualifier.
May contain organization-specific values: Yes
