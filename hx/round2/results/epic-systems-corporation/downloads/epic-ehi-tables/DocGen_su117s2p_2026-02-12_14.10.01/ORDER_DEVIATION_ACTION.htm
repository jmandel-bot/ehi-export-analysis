ORDER_DEVIATION_ACTION
Description:
This table extracts the related multiple response item Deviation Action Type (I OTP 6222), which is the Deviation Change Type of an order, in the context of an oncology treatment plan.

Primary Key
Column Name Ordinal Position
OTP_ID 1
GROUP_LINE 2
VALUE_LINE 3

Column Information
Name Type Discontinued?
1 OTP_ID NUMERIC No
The unique identifier for the patient order template record.
2 GROUP_LINE INTEGER No
The line number for the information associated with this record.
3 VALUE_LINE INTEGER No
The line number of one of the multiple values associated with a specific group of data within this record.
4 DEVIATION_ACTION_TYPE_C_NAME VARCHAR No
This column holds the change action type of the deviation (in context of oncology treatment plan deviations). For example, dose increase, dose decrease, change frequency, etc.
May contain organization-specific values: Yes
