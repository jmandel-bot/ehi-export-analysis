EAR_ID_HX
Description:
This table contains the historic Identity ID information for the guarantor.

Primary Key
Column Name Ordinal Position
ACCT_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 ACCT_ID NUMERIC No
The unique identifier for the account record.
2 LINE INTEGER No
The line number for the information associated with this row. Multiple pieces of information can be associated with this row.
3 MPI_HX_ID VARCHAR No
The historical IDs for the guarantor.
4 MPI_HX_ID_TYPE_ID NUMERIC No
The historic type of ID for the guarantor.
5 MPI_HX_ID_TYPE_ID_ID_TYPE_NAME VARCHAR No
The name of the ID Type.
6 MPI_HX_USER_ID VARCHAR No
The historic user who updated the ID for the guarantor.
7 MPI_HX_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
8 MPI_HX_AUDIT_TYPE_C_NAME VARCHAR No
The historic type of change for the guarantor ID.
May contain organization-specific values: No
Category Entries:
NEW
CHANGE
DELETED
MERGE
UNMERGE
EXPIRED
9 MPI_HX_NEW_ID VARCHAR No
The historical ID that was added for the guarantor.
10 MPI_HX_DOTONE VARCHAR No
The guarantor record ID where the ID change was made.
11 MPI_HX_FROM_DATE DATETIME No
The historical effective date for the guarantor ID.
12 MPI_HX_TO_DATE DATETIME No
The historical end date for the guarantor ID.
