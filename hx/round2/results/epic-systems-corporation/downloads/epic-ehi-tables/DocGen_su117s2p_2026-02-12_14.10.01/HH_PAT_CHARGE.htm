HH_PAT_CHARGE
Description:
This table contains information on Home Health Procedure Charges.

Primary Key
Column Name Ordinal Position
PAT_ENC_CSN_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 CONTACT_DATE_REAL FLOAT No
Unique identifier for this contact for this patient.
2 LINE INTEGER No
The line number for the information associated with this contact. Multiple pieces of information can be associated with this contact.
3 HH_CHARGE_ID_PROC_NAME VARCHAR No
The name of each procedure.
4 HH_CHARGE_QUANTITY INTEGER No
Home Health procedure charge quantity.
5 HH_CHARGE_COMMENTS VARCHAR No
Comments added to the Home Health procedure charge.
6 HH_CHARGE_USER_ID VARCHAR No
ID of the user who entered the charge. Links to table CLARITY_EMP.
7 HH_CHARGE_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
8 HH_CHARGE_EDT_INST DATETIME (Local) No
The instant at which the charge was edited.
9 HH_CHARGE_UCL_ID VARCHAR No
The Universal Charge Line record ID for the change, which is generated after the charge is routed from the clinical system to the billing system.
10 PAT_ENC_CSN_ID NUMERIC No
The unique contact serial number for this contact. This number is unique across all patient encounters in your system. If you use IntraConnect, this is the Unique Contact Identifier (UCI).
11 CONTACT_DATE DATETIME No
The date of this contact in calendar format.
12 CM_CT_OWNER_ID VARCHAR No
The Community ID (CID) of the instance that owns this contact. This is populated only if you use IntraConnect.
