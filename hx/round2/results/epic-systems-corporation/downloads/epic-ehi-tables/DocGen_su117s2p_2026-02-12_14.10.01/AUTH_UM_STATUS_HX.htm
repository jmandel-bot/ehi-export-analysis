AUTH_UM_STATUS_HX
Description:
This table contains the authorization decision history for service authorizations.

Primary Key
Column Name Ordinal Position
AUTH_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 AUTH_ID NUMERIC No
The unique identifier for the authorization record.

The line number for the information associated with this record. Multiple pieces of information can be associated with this record.

The status assigned to the service by the associated user on the specified date/time.
May contain organization-specific values: No
Category Entries:

May contain organization-specific values: Yes

The denied reason assigned to the service by the associated user on the specified date/time.
May contain organization-specific values: Yes
Category Entries:
Upheld on Appeal
7 UM_DISMISSED_RSN_C_NAME VARCHAR No

The pending reason assigned to the service by the associated user on the specified date/time.
May contain organization-specific values: Yes
Category Entries:
Appeal Under Review
Requires Appeal Effectuation
10 UM_CANCELED_RSN_C_NAME VARCHAR No
The canceled/withdrawn reason assigned to the service by the associated user on the specified date/time.

11 CHANGED_DTTM DATETIME (Local) No

12 CHANGED_BY_USER_ID VARCHAR No
The user that made the change (for ex., assigned the status and status reason) to the service.
13 CHANGED_BY_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
14 NUM_SVCS_APPROVED INTEGER No
The number of services, units, or visits approved as of the associated user's edit as of the specified date/time.
15 NUM_SVCS_REQUESTED INTEGER No
The number of services, units, or visits requested as of the associated user's edit as of the specified date/time.
16 REV_CODE_ID NUMERIC No
The procedure revenue code assigned to the authorization record as of the specified date/time. This column can be used to link to the CL_UB_REV_CODE table.
17 REV_CODE_ID_REVENUE_CODE_NAME VARCHAR No
The name of the revenue code.
18 PROC_PROV_ID_PROV_NAME VARCHAR No
The name of the service provider. This item may be hidden in a public view of the CLARITY_SER table.
19 PROC_CODE_ID_PROC_NAME VARCHAR No
The name of each procedure.
20 INTER_NUM_SVCS_APRV INTEGER No
The number of services approved for each interval as of the specified date/time.
21 INTER_NUM_SVCS_REQ INTEGER No
The number of services requested for each interval as of the specified date/time.
22 INTER_APRV_FREQ_ID VARCHAR No
The ID of the associated Frequency Record for the approved recurring authorization as of the specified date/time.
23 INTER_APRV_FREQ_ID_FREQ_NAME VARCHAR No
The name of the frequency record.
24 INTER_REQ_FREQ_ID VARCHAR No
The ID of the associated Frequency Record for the requested recurring authorization as of the specified date/time.
25 INTER_REQ_FREQ_ID_FREQ_NAME VARCHAR No
The name of the frequency record.
26 INTER_NUM_APRV INTEGER No
The number of times the interval has been approved to repeat as of the specified date/time.
27 INTER_NUM_REQ INTEGER No
The number of times the interval has been requested to repeat as of the specified date/time.

When the status changes on a service authorization, this item indicates if the change was manual or done automatically by ASA.
