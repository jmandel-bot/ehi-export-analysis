ORD_SECOND_SIGN
Description:
This table stores the information about second sign for orders.

Primary Key
Column Name Ordinal Position
ORDER_ID 1

Column Information
Name Type Discontinued?
1 ORDER_ID NUMERIC No
The unique identifier for the order record.
2 SEC_SIGN_STATUS_C_NAME VARCHAR No
This item stores the status of the second sign.
May contain organization-specific values: No
Category Entries:
No second sign needed
Needs second sign
Second sign rejected
Second sign accepted
Second sign modify
3 SEC_SIGN_FIRS_DTTM DATETIME (Local) No
This column contains the instant at which the first sign occurred.
4 SEC_SIGN_MESSAGE VARCHAR No
This is the message shown to the user.
5 SEC_SIGN_REQUIRE_C_NAME VARCHAR No
This is the second sign requirement that this order was signed with.
May contain organization-specific values: No
Category Entries:
None
Manually Set
System Set Peer Review
System Set Required
Linked Order
6 SEC_SIGN_SIGNER_ID VARCHAR No
The ID of the secondary signing user.
7 SEC_SIGN_SIGNER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
8 SEC_SIGN_SEC_DTTM DATETIME (Local) No
This column contains the instant at which the second sign occurred.
9 SEC_SIGN_REJECT_C_NAME VARCHAR No
The reason that this order was rejected.
May contain organization-specific values: Yes
Category Entries:
Medication Refill Refused
10 SEC_SIGN_COMMENT VARCHAR No
This stores the comment for the rejection.
11 SEC_SIGN_MESSAG_ID VARCHAR No
The second sign message ID.
12 SEC_SIGN_REJECT_ID VARCHAR No
This is the ID of the rejection message that is sent.
13 SEC_SIGN_INDIV_YN VARCHAR No
Whether individual review was required for this order when second signing the order.
May contain organization-specific values: No
Category Entries:
No
Yes
