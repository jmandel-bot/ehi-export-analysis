TPL_CONVERSION_SESSIONS
Description:
This table contains information about treatment plan conversion sessions.

Primary Key
Column Name Ordinal Position
TREATMENT_PLAN_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 TREATMENT_PLAN_ID NUMERIC No
The unique identifier for the plan record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 CONVSESS_FROM_DAY INTEGER No
The line in Treatment Day ID (I TPL 5000) of the first day in this conversion session.
4 CONVSESS_TO_DAY INTEGER No
The line in Treatment Day ID (I TPL 5000) of the day after the last day in this session. This can be blank if the session includes the last day in the plan.
5 CONVSESS_STATUS_C_NAME VARCHAR No
The status of a conversion session for this plan.
May contain organization-specific values: No
Category Entries:
Accepted
Discarded
Started
Active - Step 1
Active - Step 2
6 CONVSESS_TYPE_C_NAME VARCHAR No
The type of a conversion session for a plan, whether inpatient or outpatient.
May contain organization-specific values: No
Category Entries:
Inpatient
Outpatient
7 CONVSESS_ACT_UTC_DTTM DATETIME (UTC) No
The instant this conversion session action was done.
8 CONVSESS_ACT_USR_ID VARCHAR No
The user who took this conversion session action.
9 CONVSESS_ACT_USR_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
