SPEC_AP_RESULT
Description:
This table contains information related to results entered on a specimen's anatomic pathology result.

Primary Key
Column Name Ordinal Position
SPECIMEN_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 SPECIMEN_ID VARCHAR No
The unique identifier (.1 item) for the specimen record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 CM_PHY_OWNER_ID VARCHAR No
The Community ID (CID) of the instance that owns this record or line. This is only populated if you use IntraConnect.
4 AP_RES_TYPE_C_NAME VARCHAR No
The note type category ID for this AP specimen result note.
May contain organization-specific values: No
Category Entries:
Grossing
Intra-Op
Final Diagnosis
5 AP_RES_NOTE_ID VARCHAR No
The unique ID of this AP specimen result note.
6 AP_RES_STATUS_C_NAME VARCHAR No
The status category ID for this AP specimen result note.
May contain organization-specific values: No
Category Entries:
Not Started
In Progress
Complete
7 NOTIF_SENT_UTC_DTTM DATETIME (UTC) No
The first instant when a result notification was sent for this AP result in UTC.
8 ACK_COMM_ID VARCHAR No
The communication that was logged when the result is acknowledged.
9 AP_RES_PUSH_EVAL_C_NAME VARCHAR No
Stores the outcome of frozen result evaluation and push notification recipient resolution.
May contain organization-specific values: No
Category Entries:
Needs frozen follow-up
Qualified for frozen follow-up, no recipient found
Qualified for frozen follow-up, available push recipients identified
Successful push notification delivery
No routing configuration found
10 NOTIF_SENT_LOCAL_DTTM DATETIME (Local) No
The first instant when a result notification was sent for this AP result in the local time zone.
