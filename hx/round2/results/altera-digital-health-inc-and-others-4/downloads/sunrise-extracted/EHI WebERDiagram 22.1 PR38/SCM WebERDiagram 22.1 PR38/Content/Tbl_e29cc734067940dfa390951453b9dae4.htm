dbo.CV3ClientEventDeclTemplate 
Default Table Name  CV3ClientEventDeclTemplate 
Definition  Template for a significant event. This allows events to be entered from the Knowledge Tree. When entered from the Knowledge Tree (client document), this template is used to provide default values.
