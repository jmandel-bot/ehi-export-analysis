SiteID 
Definition  Identifies the unique distributed database ID where the row was inserted. Distributed databases are no longer supported so this column is deprecated. 
RepFlags 
Definition  Indicates the action in a distributed database environment. Distributed databases are no longer supported so this column is deprecated. 
Build 
Definition  Identifies the build number and patch level of the software that inserted/updated this row. The format is the Build number followed by 3 digits for the patch level which is typically 000. The build number is tied to specific versions of the software, e.g. Sunrise 15.3 is Build 5021, Sunrise 16.3 is Build 5503. 
TouchedBy 
Definition  Information about the user that last updated the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
TouchedWhen 
Definition  The date and time that the row was last updated and will be based on the Enterprise time zone. 
CreatedBy 
Definition  Information about the user that created the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
CreatedWhen 
Definition  The date and time that the row was created and will be based on the Enterprise time zone. 
Active 
Definition  Denotes whether the row is active or not, 0 means the row has been soft deleted or deactivated. 
GUID 
Definition  Unique primary key value. 
ResultAcknowledgementPolicyGUID 
Definition  A foreign key to the GUID column in the CV3ResultAcknowledgementPolicy table. 
LocnFuncGroupGUID 
Definition  GUID of location function group or null if none selected. Link to GUID column of CV3LocationFunctionGroup table. 
LocnGroupGUID 
Definition  GUID of location group or null if none selected. Link to GUID column of CV3Location table. 
FacilityGUID 
Definition  GUID of facility or 0 for . Link GUID column of CV3Location table. 
CareProviderRoleTypeGUID 
Definition  GUID of care provider role type, will be NULL if result is not distributed based on a provider role. 
IsPromptedAtOrderEntry 
Definition  Indicates whether a prompt will be displayed at order entry to confirm provider name. 
IsDistributedToOrderingProvider 
Definition  Indicates whether the result is routed to the inbox of the ordering provider. 
IsDistributedToProviderRole 
Definition  Indicates that the result is routed based on a provider role. If true, the entry of a provider type and role is mandatory. 
CanOnlyBeAcknowledgedBy 
Definition  Indicates whether only this provider or a member of his/her signature group can acknowledge the results, regardless of whether the ordering provider has the right to acknowledge results. In this case, the ordering provider will see the result in his inbox (if the Distribute to ordering provider flag is True), but cannot acknowledge it. 
MSrepl_tran_version 
Definition  Optimistic lock identifier. 
IsAllTaskRecipientsAbleToAcknowledge 
Definition  Indicates whether all result-task recipients (receiving the result-task due to result acknowledgement policy) would be able to acknowledge the task (when 1) or only one of them would be able to acknowledge (when 0) 
MSReplrowguid 
Definition  Unique GUID used to support merge replication.
