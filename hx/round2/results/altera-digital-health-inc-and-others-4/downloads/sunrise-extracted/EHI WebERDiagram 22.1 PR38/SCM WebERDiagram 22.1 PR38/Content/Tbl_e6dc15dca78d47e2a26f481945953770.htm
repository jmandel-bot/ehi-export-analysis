dbo.SXASRGSurgicalItemSubCategory 
Default Table Name  SXASRGSurgicalItemSubCategory 
Definition  This table contains the surgical item subcategory dictionary entries. This will be used to classify the surgical item by sub categories. Each subcategory will be associated to a surgical item category.
