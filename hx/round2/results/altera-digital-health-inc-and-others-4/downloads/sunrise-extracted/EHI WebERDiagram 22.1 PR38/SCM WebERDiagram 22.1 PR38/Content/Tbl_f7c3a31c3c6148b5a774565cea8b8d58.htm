dbo.CV3OrderClientDocumentXref 
Default Table Name  CV3OrderClientDocumentXref 
Definition  For the Attaching a Document to an Order feature. This new table will relate an order to one or more documents by saving the order GUID and the corresponding document GUIDS. The Document GUID will be from the GUID of the CV3ClientDocument table while the Order GUID will be from the GUID of the CV3Order table.

Build    int  NO  Identifies the build number and patch level of the software that inserted/updated this row. The format is the Build number followed by 3 digits for the patch level which is typically 000. The build number is tied to specific versions of the software, e.g. Sunrise 15.3 is Build 5021, Sunrise 16.3 is Build 5503.
