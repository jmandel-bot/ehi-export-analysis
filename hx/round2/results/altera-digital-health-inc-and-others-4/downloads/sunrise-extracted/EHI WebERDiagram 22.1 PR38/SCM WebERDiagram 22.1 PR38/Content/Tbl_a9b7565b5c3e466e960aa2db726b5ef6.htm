dbo.SXAMMChargeAlgorithm 
Default Table Name  SXAMMChargeAlgorithm 
Definition  This entity contains a list of Charge Algorithms that are defined for the entire enterprise. Each Charge Algorithm can be associated with multiple Generic Items.
For example. Cost Type: AWP, Charge Code: 102883, Charge Code Type: Hospital
