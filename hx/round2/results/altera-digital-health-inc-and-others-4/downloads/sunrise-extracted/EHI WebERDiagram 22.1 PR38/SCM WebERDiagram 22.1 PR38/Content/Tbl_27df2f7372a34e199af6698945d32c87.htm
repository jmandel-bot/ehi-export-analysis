dbo.CV3OrderReconcileComment 
Default Table Name  CV3OrderReconcileComment 
Definition  Each internal reconciliation type has several possible reconciliation actions. When an order is reconciled using a particular action, a comment can be entered to state why that action was chosen. This is done by the user during a manual reconciliation or by the system via an automated reconciliation. These comments are either user-defined or selected from a list of pre-configured comments in the system. This table records these pre-configured comments.
