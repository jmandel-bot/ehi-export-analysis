dbo.CV3CatalogSetDuplicate 
Default Table Name  CV3CatalogSetDuplicate 
Definition  Defines which sets, or which class of items, are a duplicate of this set. We need to define what sets may be duplicates of another set (or which class of items make a duplicate). This table is modeled after CV3CatalogItemDuplicate and CV3CatalogTaskDuplicate.
