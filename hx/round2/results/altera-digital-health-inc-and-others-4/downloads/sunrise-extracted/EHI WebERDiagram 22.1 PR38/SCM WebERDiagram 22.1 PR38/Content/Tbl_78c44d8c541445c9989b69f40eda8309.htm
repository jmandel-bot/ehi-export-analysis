dbo.SXAMMProductPackageJCode 
Default Table Name  SXAMMProductPackageJCode 
Definition  This Entity Links a Product Package to a J Code.There can at most one J Code linked to a Product Package for a specific Route of Administration.The same J Code can be linked to multiple Routes of administration for the Product Package.
