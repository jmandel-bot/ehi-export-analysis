dbo.SXADRClientDocumentSuppressCUR 
Default Table Name  SXADRClientDocumentSuppressCUR 
Definition  SCM will require a table to store the Community Document Suppressed information, when user clicks the Document tab and perform Suppress Community Document operation.

The format is the Build number followed by 3 digits for the patch level which is typically 000.
