dbo.SXAFindPatientResultColumnConfig 
Default Table Name  SXAFindPatientResultColumnConfig 
Definition  Stores configuration which will allow system Administrator to set the default ‘Find Patients’ search results grid behavior, including columns displayed, column order, and whether columns can be sorted, pinned, and/or moved.
