dbo.SXAAMPostalCode 
Default Table Name  SXAAMPostalCode 
Definition  This is for specification 30-4014 Zip Based Addressing. The user entered Zip Code will be validated against a list of ‘known good’ zip codes. The City/State will automatically populate in the UI.This file will contain all zip codes for imported and data entered by users
