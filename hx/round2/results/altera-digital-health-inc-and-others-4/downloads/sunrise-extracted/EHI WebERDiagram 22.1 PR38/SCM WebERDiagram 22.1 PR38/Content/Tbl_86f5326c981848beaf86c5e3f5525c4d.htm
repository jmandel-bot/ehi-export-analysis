dbo.CV3OPOFollowupComment 
Default Table Name  CV3OPOFollowupComment 
Definition 
This table is used to store user-defined follow up comments for outpatient orders. When a user is creating or editing a home medication outpatient order in Order Reconciliation Manager, a dropdown list will display all the active records in this table. The user will select from one of these comments or entering free text.

Data in this table are added or maintained by system administrators via a new config window in Dictionary Maintenance.
