dbo.SXAAMBRxAlert 
Default Table Name  SXAAMBRxAlert 
Definition  For Rx Writer, the system needs to log all alerts generated when prescriptions are written for audit purposes. These differ from CV3AlertDeclarations because these alerts will not appear anywhere else in the application and do not contain the same information as CDS alerts (ChartGUID, ClientVisitGUID, etc).
