dbo.SXAESEventCategory 
Default Table Name  SXAESEventCategory 
Definition  Events in scheduling are types of appointments that may be booked, such as "Check-Up", "CAT Scan", etc. In the 4.5 version of scheduling administrators will have the ability to add Events into Categories which will group the Events for use in various places in the application. This will allow users to select the Event Category rather then having to individually select each event in some scenarios. This table will hold the definitions of the Event Categories.
