dbo.SXAAMLocnSearchColumn 
Default Table Name  SXAAMLocnSearchColumn 
Definition  This table will contain the columns available to the user to select for displaying in the Location Searching. This table is being created to satisfy business requirements in specification 30-4026 Changes to Location Search. Previously, the columns displayed in a Location Search grid were system defined and unalterable, but now the users will be able to define which columns they would like to see in the grid. There can only be one definition per facility, and it is either active or inactive. If inactive, the default columns will be displayed. The table contains the columns available to the user to select to see in a Location Search
