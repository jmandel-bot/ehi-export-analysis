dbo.SXAMMGIPriceAlgorithm 
Default Table Name  SXAMMGIPriceAlgorithm 
Definition  This entity identifies which Price Codes apply to which Generic Items based on Location Function Group. Further properties of the link include Patient Type, and Outpatient Type combinations for which this link is valid, and these are defined in the Child Table Generic Item Price Algorithm Type.
