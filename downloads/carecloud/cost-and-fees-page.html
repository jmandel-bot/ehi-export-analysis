Accessibility Standard

WCAG 2.0 AA

170.315 (d)(13): Multi-Factor Authentication Accessibility Standard

talkEHR offers Multi-Factor Authentication (MFA) as an optional service at no additional cost. It is a user-configurable setting located on the user profile. The practice administrator can set up MFA for practice users as part of their login process.WCAG 2.0 AA

MFA adds an extra layer of security that works in conjunction with the username and password.

Once enabled by the practice administrator, MFA can be activated by either SMS/messenger or email. The user will use the email/SMS to get a one-time code, which is used in conjunction with the username and password to log in to talkEHR.MFA adds an extra layer of security that works in conjunction with the username and password.
