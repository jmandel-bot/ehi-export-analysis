IB_CALL_ACTION
Description:
The IB_CALL_ACTION table contains the actions needed to be taken to follow-up on a telephone encounter.

Primary Key
Column Name Ordinal Position
MSG_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 MSG_ID VARCHAR No
The unique ID of the In Basket Message.
2 LINE INTEGER No
Line number to identify the call action within an In Basket Message.
3 CALL_ACTION_C_NAME VARCHAR No
The category value associated with select actions that summarize the phone call and its necessary follow-up.
May contain organization-specific values: Yes
Category Entries:
Telephoned
Came to see you
Wants to see you
Returned your call
Please call
Will call again
