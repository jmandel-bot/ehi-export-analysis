COVERAGE_MISC_COMMENTS
Description:
Table to extract comments to Clarity from coverage items. This table should not be extracted unless there are attached comments on CVG-495 or CVG-10005. If those items do contain comments, this table should be extracted for EHI compliance.

Primary Key
Column Name Ordinal Position
COVERAGE_ID 1

Column Information
Name Type Discontinued?
1 COVERAGE_ID NUMERIC No
The unique identifier (.1 item) for the coverage record.
2 VERIFY_SOURCE_C_CMT VARCHAR No
Column to extract comments for CVG-495 Verification Source.
May contain organization-specific values: Yes
No Entries Defined
3 TYPE_OF_COVERAGE_C_CMT VARCHAR No
Column to extract comments from CVG-10005 Type of Coverage.
May contain organization-specific values: Yes
No Entries Defined
