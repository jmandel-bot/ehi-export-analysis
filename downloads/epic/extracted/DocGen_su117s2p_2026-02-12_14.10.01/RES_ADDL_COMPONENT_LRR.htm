RES_ADDL_COMPONENT_LRR
Description:
This item stores additional obstetric fetal measurement data. If more than one value is sent from an ultrasound machine for a given result, we will store the additional ones in this table.

Primary Key
Column Name Ordinal Position
FINDING_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 FINDING_ID NUMERIC No
The unique identifier (.1 item) for the finding record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 ADDL_READ_COMPONENT_ID NUMERIC No
If additional measurement values are provided for a component, this item will store the ID of the component (LRR).
4 ADDL_READ_COMPONENT_ID_NAME VARCHAR No
The name of the component.
5 ADDL_READ_VALUE VARCHAR No
If additional measurement values are provided for a component, this item will store the value of the LRR component.
6 ADDL_READ_UNITS VARCHAR No
If additional measurement values are provided for a component, this item will store the units of the LRR component.
7 ADDL_READ_OB_MEAS_SOURCE_C_NAME VARCHAR No
If additional measurement values are provided for a component, this item will store the source of the reading.
May contain organization-specific values: No
Category Entries:
OB Study Review
OB DICOM
OB HL7 Interface
Calculated
8 ADDL_READ_DESCR_C_NAME VARCHAR No
This item flags an OB ultrasound reading with a descriptor.
May contain organization-specific values: No
Category Entries:
None
Average
9 ADDL_READ_GA_VAL VARCHAR No
This column contains the gestational age (in days) for the associated result component reading row.
