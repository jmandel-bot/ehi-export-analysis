TPL_INFO
Description:
This table contains basic information about a treatment plan or a pathway, such as the plan/pathway name, the user who created the plan/pathway, when it was created, which protocol it was created from, the starting cycle number or step, etc.

64 IS_PLAN_DELETED_YN VARCHAR No
Virtual item that determines if a treatment plan (TPL) has been deleted or not for EHI export purpose. A treatment plan is considered deleted if it has a discontinued reason (I TPL 120) that's defined as entered in error (I LSD 77800), or if it's a BMT plan and its associated HSB record is deleted (I HSB 50). A dental treatment plan is never considered deleted.
