PROB_DIS_ST_ENTRY_WKFL_HX
Description:
This table extracts an audit trail of changes to the disease status entry workflow for a problem on a patient's problem list.

Primary Key
Column Name Ordinal Position
PROBLEM_LIST_ID 1
GROUP_LINE 2
VALUE_LINE 3

Column Information
Name Type Discontinued?
1 PROBLEM_LIST_ID NUMERIC No
The unique identifier (.1 item) for the problem record.
2 GROUP_LINE INTEGER No
The line number for the information associated with this record.
3 VALUE_LINE INTEGER No
The line number of one of the multiple values associated with a specific group of data within this record.
4 HX_DISSTAT_ENTRY_HOST_MODULE_C_NAME VARCHAR No
Stores an audit trail of the workflows in which providers have entered or changed disease status information for a problem on a patient's problem list.
May contain organization-specific values: No
