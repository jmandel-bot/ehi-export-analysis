CLARITY_DEP_4
Description:
This table extends CLARITY_DEP, which contains high-level information about departments from the Department master file.

Primary Key
Column Name Ordinal Position
DEPARTMENT_ID 1

Column Information
Name Type Discontinued?

The external name of the department record. This is often used in patient correspondence such as reminder letters.

Use OP Meds With Low Acuity Intake

The type of the department.
