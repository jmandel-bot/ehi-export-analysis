PAT_ENC_HPF_AR
Description:
This patient encounter table contains Professional Billing Hospital Professional Fee encounters. The primary key for the patient encounter table is PAT_ENC_CSN_ID.

Primary Key
Column Name Ordinal Position
PAT_ENC_CSN_ID 1

Column Information
Name Type Discontinued?
1 PAT_ENC_DATE_REAL FLOAT No
This is a numeric representation of the date of this encounter in your system. The integer portion of the number specifies the date of the encounter. Digits after the decimal point indicate multiple visits on one day.
2 PAT_ENC_CSN_ID NUMERIC No
A unique serial number for this encounter. This number is unique across all patients and encounters in the system.
3 CONTACT_DATE DATETIME No
For Hospital Professional Fee encounters, this is the admit date in calendar format.
4 HPF_ENC_STATUS_C_NAME VARCHAR No
The Hospital Professional Fee encounter status category ID for the patient.
May contain organization-specific values: Yes
Category Entries:
New
Posted
Completed
Released
Post/Billed
Comp/Billed
Deleted
5 HPF_EM_CODE_REQ_YN VARCHAR No
Indicates whether the Hospital Professional Fee encounter for the patient meets E&M code requirements.
May contain organization-specific values: No
Category Entries:
Yes
No
6 AR_HPF_ENC_VST_CSN NUMERIC No
This column contains the contact serial number which links the hospital professional fee (HPF) contact with the visit contact. This CSN is the unique contact serial number for the visit contact. This number is unique across all patient encounters in your system. If you use IntraConnect, this is the Unique Contact Identifier (UCI).
