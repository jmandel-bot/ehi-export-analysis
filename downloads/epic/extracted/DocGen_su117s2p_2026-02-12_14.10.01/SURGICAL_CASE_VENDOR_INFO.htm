SURGICAL_CASE_VENDOR_INFO
Description:
This table contains information about implant vendors documented on the surgical case.

LINE 2

Name Type Discontinued?

The unique identifier (.1 item) for the case request record.

The line number for the information associated with this record. Multiple pieces of information can be associated with this record.

Implantable device manufacturer associated with the procedure.
