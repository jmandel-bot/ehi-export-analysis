EPISODE_PAUSE
Description:
This table contains data on past and present Social Care episode pause periods that currently exist in the system.

Primary Key
Column Name Ordinal Position
SUMMARY_BLOCK_ID 1
LINE 2

Column Information
Name Type Discontinued?

The unique identifier for the episode record.

4 PAUSE_START_DATE DATETIME No
The date when the pause was started.
5 PAUSE_END_DATE DATETIME No
The date when the pause was ended.

The pause reason category ID for the pause period.
May contain organization-specific values: Yes
7 UPDATE_USER_ID VARCHAR No
The unique ID of the user who most recently updated the pause period.

The name of the user record. This name may be hidden.
9 UPDATE_UTC_DTTM DATETIME (UTC) No

10 PAUSE_KEY VARCHAR No
