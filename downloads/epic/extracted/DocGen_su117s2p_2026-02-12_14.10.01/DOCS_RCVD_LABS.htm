DOCS_RCVD_LABS
Description:
This table contains a list of name and address information for various performing sites data has been received from.

Primary Key
Column Name Ordinal Position
DOCUMENT_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 DOCUMENT_ID NUMERIC No
This item stores the Received Document record ID.
2 LINE INTEGER No
The line number for the information associated with this contact. Multiple pieces of information can be associated with this contact.
3 LAB_REFERENCE_IDENT VARCHAR No
Stores the reference ID of a performing site. The reference ID is the checksum of name, street address, city, state, postal code, phone number, and CLIA number in that order.
4 LAB_NAME VARCHAR No
Stores the name of a performing site associated with either a result component, the narrative, or the impression
5 LAB_DIRECTOR VARCHAR No
Stores the director of the performing site associated with either a result component, the narrative, or the impression
6 LAB_STREET VARCHAR No
Stores the street address of the performing site associated with either a result component, the narrative, or the impression
7 LAB_CITY VARCHAR No
Stores the city of the performing site associated with either a result component, the narrative, or the impression
8 LAB_STATE_C_NAME VARCHAR No
Stores the state of the performing site associated with either a result component, the narrative, or the impression
May contain organization-specific values: Yes
9 LAB_POSTAL_CODE VARCHAR No
Stores the postal code of the performing site associated with either a result component, the narrative, or the impression
10 LAB_PHONE_NUM VARCHAR No
Stores the phone number of the performing site associated with either a result component, the narrative, or the impression.
11 LAB_CLIA_NUM VARCHAR No
Stores the CLIA number of the performing site associated with either a result component, the narrative, or the impression
12 LAB_OBFUSCATED_ID VARCHAR No
Stores the Obfuscated ID of a resulting agency. Populated only on the Cosmos Host.
