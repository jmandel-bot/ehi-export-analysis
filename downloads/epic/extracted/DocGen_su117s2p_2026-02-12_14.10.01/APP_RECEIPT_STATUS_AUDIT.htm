APP_RECEIPT_STATUS_AUDIT
Description:
This table contains the application receipt status changes and audit information for an order.

Primary Key
Column Name Ordinal Position
ORDER_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 ORDER_ID NUMERIC No
The unique identifier (.1 item) for the order record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 APP_RECPT_STATUS_C_NAME VARCHAR No
Application receipt status for order e-messages
May contain organization-specific values: No
Category Entries:
Pending
Acknowledged
Failure
Timeout
User Done
4 APP_RECPT_USER_ID VARCHAR No
The user associated with the application receipt status update for a given order
5 APP_RECPT_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
6 APP_RECPT_STATUS_UTC_DTTM DATETIME (UTC) No
Instant of update for application receipt status of an order
