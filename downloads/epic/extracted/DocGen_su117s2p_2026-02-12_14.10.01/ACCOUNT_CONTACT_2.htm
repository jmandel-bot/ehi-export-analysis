ACCOUNT_CONTACT_2
Description:
This table contains the information recorded in billing system account contact for each account. Each row in this table contains information about one contact and is uniquely identified by the Account ID and line number combination.

Primary Key
Column Name Ordinal Position
ACCT_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 ACCT_ID NUMERIC No
The unique ID for the account record. This ID number could be encrypted if you have elected to implement enterprise reporting’s encryption security function.
2 LINE INTEGER No
Line number to identify the account contact information within the account.
3 FOL_UP_MYC_USER_ID VARCHAR No
The ID of the MyChart user who created the contact. This ID may be encrypted if you have elected to use enterprise reporting’s security utility.
4 PAY_PLAN_SOURCE_C_NAME VARCHAR No
The source workflow category ID that set up the payment plan for the guarantor.
May contain organization-specific values: No
