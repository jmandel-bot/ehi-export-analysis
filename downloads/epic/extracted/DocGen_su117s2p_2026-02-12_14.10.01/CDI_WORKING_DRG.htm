CDI_WORKING_DRG
Description:
This table contains information about the Diagnosis Related Group (DRG) data stored as part of a Clinical Documentation Improvement (CDI) review. Each contact in this table corresponds to DRG data from one CDI review.

Primary Key
Column Name Ordinal Position
CODING_RECORD_ID 1
CONTACT_DATE_REAL 2
LINE 3

Column Information
Name Type Discontinued?
1 CODING_RECORD_ID NUMERIC No
The unique identifier for the clinical documentation improvement (CDI) review record.
2 CONTACT_DATE_REAL FLOAT No
A unique contact date in decimal format. The integer portion of the number indicates the date of contact. The digits after the decimal distinguish different contacts on the same date and are unique for each contact on that date. For example, .00 is the first/only contact, .01 is the second contact, etc.
3 LINE INTEGER No
The line number for the information associated with this contact. Multiple pieces of information can be associated with this contact.
4 CONTACT_DATE DATETIME No
The date of this contact in calendar format.
5 CDI_WORKING_DRG_ID VARCHAR No
The working diagnosis-related group (DRG) for a particular clinical documentation improvement (CDI) review.
6 CDI_WORKING_DRG_ID_DRG_NAME VARCHAR No
The name of the Diagnoses Related Group name.
7 CDI_DRG_REIMBURSE NUMERIC No
The expected reimbursement for a working diagnosis-related group (DRG) on the clinical documentation improvement (CDI) review.
8 CDI_DRG_WEIGHT NUMERIC No
The weight of a working diagnosis-related group (DRG) on the clinical documentation improvement (CDI) review.
9 CDI_DRG_TYPE_ID NUMERIC No
The diagnosis-related group (DRG) type of a working diagnosis-related group on the clinical documentation improvement (CDI) review.
10 CDI_DRG_TYPE_ID_ID_TYPE_NAME VARCHAR No
The name of the ID Type.
11 CDI_DRG_SOI_C_NAME VARCHAR No
The Severity of Illness (SOI) value of a working diagnosis-related group (DRG) on the clinical documentation improvement (CDI) review.
May contain organization-specific values: No
Category Entries:
Refinement not possible
Minor
Moderate
Major
Extreme
Principal Dx Used for SOI/ROM Calc
Excluded
Duplicate Diagnosis
Excluded Complication of Care
12 CDI_DRG_ROM_C_NAME VARCHAR No
The Risk of Mortality (ROM) value for a working diagnosis-related group (DRG) on the clinical documentation improvement (CDI) review.
The category values for this column were already listed for column: CDI_DRG_SOI_C_NAME
13 CONTACT_SERIAL_NUM NUMERIC No
The unique ID of the contact that this coding record is associated with.
14 CDI_DRG_AMLOS NUMERIC No
The arithmetic mean length of stay for the working diagnosis related group in the Clinical Documentation Improvement (CDI) review.
15 CDI_DRG_GMLOS NUMERIC No
The geometric mean length of stay for the working diagnosis related group in the Clinical Documentation Improvement (CDI) review.
16 DRG_QUALIFIER_C_NAME VARCHAR No
The qualifier for the working diagnosis related group in the Clinical Documentation Improvement (CDI) review.
May contain organization-specific values: Yes
Category Entries:

Admission DRG
PPC DRG
PPC Admit DRG
17 CDI_DRG_COMP_FLAG_YN VARCHAR No
This column indicates if the diagnosis-related group (DRG) is used for comparison purposes.
May contain organization-specific values: No
Category Entries:
No
Yes
18 DRG_IS_CMI_EXCLUD_YN VARCHAR No
Indicates whether a DRG code from a DRG type is excluded from CMI calculation on CDI.
