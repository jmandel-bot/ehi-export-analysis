DERM_SKINEXAM_FINDING
Description:
Clarity table for dermatology skin exam finding no-add items.

Primary Key
Column Name Ordinal Position
FINDING_ID 1

Column Information
Name Type Discontinued?
1 FINDING_ID NUMERIC No
The unique identifier (.1 item) for the finding record.
2 COPY_FINDING_SRC_FINDING_ID NUMERIC No
This item contains the finding record that this finding was copied from. This is used for auditing purposes.
3 PALETTE_FINDING_SRC_RECORD_ID NUMERIC No
This item contains the palette row record that the finding was added from. This is used for auditing purposes.
4 PALETTE_FINDING_SRC_RECORD_ID_RECORD_NAME VARCHAR No
The name of the Scripting Template.
