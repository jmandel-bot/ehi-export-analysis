ORD_NLP_STATUS
Description:
This table contains the NLP status of the order for the corresponding model.

Primary Key
Column Name Ordinal Position
ORDER_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 ORDER_ID NUMERIC No
The unique identifier (.1 item) for the order record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 NLP_MODEL_TYPE_C_NAME VARCHAR No
Stores the NLP model type used as an index for items in this table.
May contain organization-specific values: No
Category Entries:
AFM Recommendations Model
AFM Findings Model
Lung Recommendations Model
Lung-RADS Model
External
Fact Extract Augmented Rule Evaluation
4 RIS_NLP_STATUS_C_NAME VARCHAR No
Stores the NLP evaluation status of the order for the corresponding model.
May contain organization-specific values: No
Category Entries:
Processing
Evaluated - No Results
Results Need Verification
Results Verified
Addended - Needs Confirmation
Error
Results Verified (Modified)
5 NLP_VERIFYING_USER_ID VARCHAR No
The user (EMP) that most recently verified this order's NLP-extracted results for this row's NLP model.
6 NLP_VERIFYING_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
7 NLP_VERIFIED_DTTM DATETIME (UTC) No
The instant that the user (EMP) in I ORD 52387 for this row performed the verifying action.
8 SLOWEST_NLP_MODIFY_TYPE_C_NAME VARCHAR No
Stores the way that an NLP model was modified. If it was modified multiple ways the slowest method is stored.
May contain organization-specific values: No
Category Entries:
Quick Button
Other
9 NLP_OUTCOME_C_NAME VARCHAR No
The outcome of a model generated result. Used to measure overall accuracy of NLP models.
May contain organization-specific values: No
Category Entries:
True Positive
Positive With Modification
False Positive
True Negative
False Negative
Error
10 NLP_MODEL_VERSION INTEGER No
The version of the NLP model that processed the order.
11 NLP_MODEL_REVISION INTEGER No
The Nebula code revision of the NLP model that processed the order.
12 NLP_SERVER_REVISION INTEGER No
The server code revision of the NLP model that processed the order.
13 NLP_VERIFIED_LOC_DTTM DATETIME (Local) No
The local instant that the user (EMP) in I ORD 52387 for this row performed the verifying action.
14 SENT_TO_CHARACTERISTICS_YN VARCHAR No
This item stores information on whether or not a study that was sent to the AIEF findings module was also sent to the characteristics extraction module.
May contain organization-specific values: No
Category Entries:
No
Yes
