CLAIM_REFERRAL_LK
Description:
This table contains the referrals linked to AP claims.

Primary Key
Column Name Ordinal Position
CLAIM_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 CLAIM_ID NUMERIC No
The ID number of the claim record.
2 LINE INTEGER No
The line number indicating the order of referrals associated with this claim.
3 REFERRAL_ID NUMERIC No
The ID number of the referral associated with the claim.
4 RFL_NOT_USED VARCHAR No
Stores 1 if the referral was not used during adjudication and is blank otherwise.
5 PROC_LINES_ALLOWED VARCHAR No
Stores a list of service lines allowed to count against this referral as a comma-delimited list. The lines correspond to the values in AP_CLAIM_PROC_IDS.LINE for the claim.
6 RFL_ATTACH_SOURCE_C_NAME VARCHAR No
The referral attachment source category ID for the AP claim.
May contain organization-specific values: No
