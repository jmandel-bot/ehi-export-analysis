MERCHANDISE_ORDER_INFO
Description:
This table contains the order level information of merchandise orders.

Primary Key
Column Name Ordinal Position
ORDER_ID 1

Column Information
Name Type Discontinued?
1 ORDER_ID NUMERIC No
The unique identifier for the order record.
2 MERCH_SUPPLY_ID VARCHAR No
The unique identifier for the merchandise record for the merchandise order.
3 MERCH_SUPPLY_ID_SUPPLY_NAME VARCHAR No
The name of the inventory item.
4 MERCH_ORIG_UNIT_PR NUMERIC No
This column contains the original unit price from the merchandise record when the it was sold from the outpatient pharmacy.
5 ELIGIBLE_HEALTHCARE_EXPENSE_YN VARCHAR No
Indicates whether merchandise item sold was an eligible healthcare expense at the time of sale, which qualifies it to be paid for with a healthcare expense account such as an FSA, HSA, or HRA. This is applicable only to retail merchandise sales. 'N' or NULL indicate that the merchandise is not considered an eligible healthcare expense. 'Y' indicates that the merchandise is considered an eligible healthcare expense.
May contain organization-specific values: No
Category Entries:
No
Yes
6 DISPLAY_NAME VARCHAR No
This column contains the display name for the merchandise order.
7 PHARMACY_ID NUMERIC No
The unique identifier for the pharmacy record where the merchandise was sold.
8 PHARMACY_ID_PHARMACY_NAME VARCHAR No
The name of the pharmacy.
9 FEE_TYPE_C_NAME VARCHAR No
For pharmacy fee orders, this is the fee type category ID for the order.
May contain organization-specific values: Yes
Category Entries:
Delivery Fee
