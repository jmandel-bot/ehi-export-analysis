HSP_ACCT_FINDX_ALT_HACS
Description:
The Hospital Acquired Conditions (HACs) associated with alternate code set final coded diagnoses.

Primary Key
Column Name Ordinal Position
HSP_ACCOUNT_ID 1
GROUP_LINE 2
VALUE_LINE 3

Column Information
Name Type Discontinued?
1 HSP_ACCOUNT_ID NUMERIC No
The unique identifier (.1 item) for the account record.
2 GROUP_LINE INTEGER No
The line number for the information associated with this record.
3 VALUE_LINE INTEGER No
The line number of one of the multiple values associated with a specific group of data within this record.
4 FINDX_ALT_DX_HAC_CAT_C_NAME VARCHAR No
Which Hospital-Acquired Condition (HAC) categories the diagnosis is part of for this account, if any.
May contain organization-specific values: No
