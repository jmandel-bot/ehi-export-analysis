ALTERNATIVE_PROCS
Description:
This table contains information about alternative procedures selected for patients.

Primary Key
Column Name Ordinal Position
PAT_ENC_CSN_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 PAT_ENC_CSN_ID NUMERIC No
A unique serial number for this encounter. This number is unique across all patients and encounters in the system.
2 LINE INTEGER No
The line count for alternative procedure items.
3 PAT_ENC_DATE_REAL FLOAT No
This is a numeric representation of the date of this encounter in your system. The integer portion of the number specifies the date of the encounter. The digits after the decimal point indicate multiple visits on one day.
4 CONTACT_DATE DATETIME No
The date of this contact in calendar format.
5 CM_CT_OWNER_ID VARCHAR No
ID of the deployment owner for this contact.
6 ALT_PROC_LMA_ID NUMERIC No
The identifier for the alternative procedure (LMA) record.
7 ALT_PROC_LMA_ID_ALTERNATIVE_NAME VARCHAR No
This column contains the name of the alternative medication and procedure records.
8 ALT_PROC_ORIG_ID_PROC_NAME VARCHAR No
The name of each procedure.
9 ALT_PROC_OPTION_C_NAME VARCHAR No
Alternative procedure option taken.
May contain organization-specific values: No
Category Entries:
ACCEPT
CONTINUE
CANCEL
10 ALT_PROC_USER_ID VARCHAR No
ID of user who ordered the alternative procedure.
11 ALT_PROC_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
12 ALT_PROC_TIME DATETIME (Attached) No
Instant the alternative was taken.
13 ALT_PROC_ORDER_ID_PROC_NAME VARCHAR No
The name of each procedure.
14 ALT_PROC_CLS_C_NAME VARCHAR No
Order class for alternative procedure.
May contain organization-specific values: Yes
Category Entries:
Normal
15 ALT_PROC_QUANTITY VARCHAR No
Quantity for alternative procedure.
16 ALT_PROC_PRIO_C_NAME VARCHAR No
Priority of alternative procedure.
May contain organization-specific values: Yes
17 ALT_PROC_REL_COST VARCHAR No
Relative cost of alternative procedure.
18 ALT_PROC_WHEN_C_NAME VARCHAR No
When value for alternative procedure.
May contain organization-specific values: Yes
Category Entries:
Now
AM
Today
Tomorrow
Next Draw
19 ALT_PROC_FREQ_ID VARCHAR No
Frequency of alternative procedure.
20 ALT_PROC_FREQ_ID_FREQ_NAME VARCHAR No
The name of the frequency record.
21 ALT_PROC_COUNT INTEGER No
Count value for alternative procedure.
22 ALT_PROC_CONTINUE_C_NAME VARCHAR No
The continue reason category number for the alternative procedure alert.
May contain organization-specific values: Yes
23 ALT_PROC_FB_DISC_C_NAME VARCHAR No
Captures the discrete positive/negative procedure alternatives feedback value.
May contain organization-specific values: No
Category Entries:
None
Positive
Negative
24 ALT_PROC_FB_COMMENT VARCHAR No
Captures freetext feedback comments on the procedure alternative.
