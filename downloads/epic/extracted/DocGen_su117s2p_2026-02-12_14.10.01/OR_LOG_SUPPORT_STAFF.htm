OR_LOG_SUPPORT_STAFF
Description:
This table contains information about support staff.

Primary Key
Column Name Ordinal Position
LOG_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 LOG_ID VARCHAR No
The unique identifier for the log record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 SUPPORT_STAFF_ID_PROV_NAME VARCHAR No
The name of the service provider. This item may be hidden in a public view of the CLARITY_SER table.
4 SUPPORT_STAFF_PHASE_C_NAME VARCHAR No
This category ID for the support staff phase.
May contain organization-specific values: Yes
Category Entries:
Pre-op
Holding
Intra-op
Recovery
Phase II
5 SUPPORT_STAFF_START_TM DATETIME (Local) No
This column stores the start time for the support staff.
6 SUPPORT_STAFF_END_TM DATETIME (Local) No
This column stores the end time for the support staff.
7 SUPPORT_STAFF_COMMENTS_FT VARCHAR No
This column stores free-text information about the support staff.
