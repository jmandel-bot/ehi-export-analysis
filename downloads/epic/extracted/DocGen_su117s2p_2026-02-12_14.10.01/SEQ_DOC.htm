SEQ_DOC
Description:
This table contains information about your genomics sequencing pipeline documents.

Primary Key
Column Name Ordinal Position
SEQ_DOC_ID 1

Column Information
Name Type Discontinued?
1 SEQ_DOC_ID NUMERIC No
The unique identifier for the sequencing document record.
2 CLOUD_VOL_AND_BLOB VARCHAR No
The cloud volume and blob name where the sequencing document is stored
3 PAT_ID VARCHAR No
The primary patient for this sample. Only sample types of Tumor and ProbandGL are considered for this association.
4 SEQ_DOC_FILE_CLASS_C_NAME VARCHAR No
The file class category ID for the sequencing document.
May contain organization-specific values: No
Category Entries:
Preliminary Call Set
Final Call Set
5 FILE_DESC VARCHAR No
This item holds a user facing description of the sequencing document.
