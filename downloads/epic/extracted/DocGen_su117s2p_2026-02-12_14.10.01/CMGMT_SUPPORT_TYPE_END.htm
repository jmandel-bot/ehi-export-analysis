CMGMT_SUPPORT_TYPE_END
Description:
This table extracts the related multiple response Support & Services Provided Audit Trail - End Date (I HSB 64142) item.

Primary Key
Column Name Ordinal Position
EPISODE_ID 1
GROUP_LINE 2
VALUE_LINE 3

Column Information
Name Type Discontinued?
1 EPISODE_ID NUMERIC No
The unique identifier for the episode record.
2 GROUP_LINE INTEGER No
The line number of the historical support and service type information in the episode record. Together with EPISODE_ID, this forms the foreign key for CMGMT_SUPPORT_TYPE_HX.
3 VALUE_LINE INTEGER No
The line number of the end date in the snapshot of the support and services audit trail (SI HSB 64140) and associated with the support and service type in CMGMT_SUPPORT_TYPE_HX.
4 SSP_END_DATE DATETIME No
The end date of historical support and service type information.
