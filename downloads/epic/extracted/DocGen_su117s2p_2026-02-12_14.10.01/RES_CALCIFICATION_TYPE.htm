RES_CALCIFICATION_TYPE
Description:
This table contains a result's BI-RADS calcification types.

Primary Key
Column Name Ordinal Position

LINE 2

Column Information
Name Type Discontinued?
1 FINDING_ID NUMERIC No
The unique identifier (.1 item) for the finding record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.

This column contains the BI-RADS calcification types. This column contains historical data.
