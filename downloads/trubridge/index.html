EHI Export Overview

As part of the 2015 Edition Cures Update, a Certified Health IT application that electronically stores Electronic Health Information (EHI) must certify to the Cures Update ยง 170.315(b)(10) EHI Export criterion. This criterion requires that the application provide a way to generate a set of files containing all the EHI stored in the application for a patient or patient population.

The method for exporting the EHI and the format of the patient files varies by TruBridge product.
