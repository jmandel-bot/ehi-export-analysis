EHI Export

MEDITECH's EHI Export capabilities enable the extraction of the electronic health record (EHR) from a MEDITECH system in the form of an "export zip" file which contains files in a machine-readable format in accordance with §170.315(b)(10) - Electronic Health Information export of the ONC 2015 Edition Cures Update Certification Criteria.

The content and format of the data contained within the export zip file depends on a number of factors, including:

  • The MEDITECH product version
  • The implemented MEDITECH applications
  • The healthcare organization's specific configuration
  • The healthcare organization's chosen usage of the system
Note: A consumer of the export zip is likely to encounter some variations in the output content from one healthcare organization running MEDITECH software to another.
