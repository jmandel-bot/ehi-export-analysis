GBO VC2(1) Y This field designates the product's status as a brand name product or a generic product. This status is based on Multum's internal editorial rules and cost algorithms that are used to determine whether a multi-source drug is generic or not.
GLOBAL_SECONDARY_IDENT VC2(50) N Global drug identifier
GLOBAL_TYPE_ID NUMBER N Identifies the type of Global Identifier stored in Global Secondary Id
INNER_PACKAGE_DESC_CODE NUMBER Y This field contains a code that represents units that describe the package size. Not all drug products require a package unit description for a complete representation of the packaging.
INNER_PACKAGE_SIZE FLOAT Y This field represents the number of individual units present in the innermost packaging of a drug.
MAIN_MULTUM_DRUG_CODE NUMBER Y The Main Multum Drug Code (MMDC) is Multum's designator for groups of similar drug products. An MMDC is assigned to each unique combination of the following fields: active ingredient(s), strength, route of administration, and dosage form.
NDC_CODE VC2(25) N The National Drug Code is the standard 11-digit identifier for each drug product, as recognized by HCFA, other federal and state agencies, and many commercial enterprises. It is unique to each specific product. The first 5 numbers identify the manufacturer of a product. The second 4 numbers identify the product, and the last 2 numbers identify the package size of that product.
NDC_FORMATTED VC2(25) N This field presents the National Drug Code in the format #####-####-##, where the first set of numbers identify the manufacturer of a product, the second set of numbers identify the product, and the last set identifies the package size of the product. This field differs from the ndc_code field in that it includes dashes between each set of numbers.
NDC_REGULATION VC2(10) Y National Drug Code RegulationColumn
NDC_SCHEME VC2(3) Y National Drug Code SchemeColumn
OBSOLETE_DATE DATE Y This field contains data only if a drug product is no longer being manufactured. It will contain the date when the product became obsolete. Multum uses the shelf expiration date of the last batch produced by a manufacturer, or the date of drug withdrawal, when available. When not available, Multum provides its best estimate of the obsolete date based on information provided by wholesalers and manufacturers.
ORANGE_BOOK_ID NUMBER N This field contains a unique code for each Orange Book code.
OTC_STATUS VC2(1) Y This field designates whether a product is available by prescription only (Rx) or if it is available over-the-counter (OTC).
OUTER_PACKAGE_SIZE FLOAT Y This field designates the total number of inner packages present in the final packaged product. Not all drug products require an outer package size for a complete representation of the packaging. (The package unit may suffice.)
REPACKAGED VC2(1) Y This field indicates whether another drug vendor has repackaged the original product.
SOURCE_ID NUMBER N This field contains a unique code for each drug source (manufacturer/redistributor).
UNIT_DOSE_CODE VC2(1) Y This one character designation indicates if the product is supplied in unit dose packaging. Valid choices are: U - Packaged in unit dose. N - Not packaged in unit dose.
