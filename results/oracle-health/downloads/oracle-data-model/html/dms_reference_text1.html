Column Name Type Null? Definition

TEXT_ENTITY_ID NUMBER N The id of where the text is being stored, for example may be a long_text_id.
TEXT_ENTITY_NAME VC2(32) Y The name of text type that is stored. For example could be 'LONG TEXT'.
TEXT_LOCATOR VC2(50) Y A location of the text to be displayed, could be a directory or web page address.

UPDT_TASK NUMBER N The registered (assigned) task number for the process that inserted or updated the row.
