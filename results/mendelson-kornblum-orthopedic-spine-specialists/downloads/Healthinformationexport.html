Provider Export - Single Patient

To generate a C-CDA document for a specific patient please complete the following steps.

  1. Log into the Orthoplex system using your provider credentials
  2. Search for the patient using the patient search screen (first page after login)
  3. Form patient tabs, go to the History -> CCDA
  4. You have three choices to generate a CCDA document
    1. Download a Zip file with the CCDA document included
      • It is recommended that you encrypt the download
    2. Send a secure email with the CCDA document included
    3. Send a Direct Message with the CCDA document included
      • Note you must enter a valid Direct mailbox for this to be received, this is not standard email address.
  5. The download will include the CCDA document, the format, and instructions for reading access the file.

Provider Export - Patient Population

To generate a C-CDA document for a set of patients please complete the following steps.

  1. Log into the Orthoplex system using your provider credentials
  2. From the Menu, go to the Utilities->CCDA
  3. Click "Add New Export" button
  4. First set the name of your export
  5. Second set the location you wish your export to be place on the shared server
    1. The shared server is accessible from the orthoplex scans icon on your desktop
  6. You are now able to set up an export of the patient's currently in the Orthoplex system. You have multiple options for frequency, range, and type of export.
    1. Frequency
      1. Run Once
      2. Daily
      3. Weekly
      4. Monthly
      5. Yearly
    2. Range
      1. Date start
      2. Date end
      3. Leave empty if you do not want set a boundary
    3. Type
      1. All Patients
      2. Single Patient
  7. Once you have setup your criteria for export, click the save button
  8. The system will queue your request for CCDA export
  9. Once your export is complete you CCDA file will be placed in the location you indicated on the shared server
    1. Note you must be in the MKO network to access this shared server or have connected through the Syngeryhealth VPN
  10. The download will include the CCDA documents, the format, and instructions for reading access the file.

Structure of Export

C-CDA Format


CCDA Data

Demographics
Elements
Patient (First Name LastName), Date of Birth, Sex, Race, Preferred language, Ethnicity (primary), Ethnicity (secondary), Contact Information, Patient ID.

Document
Elements
Document ID, Document Created Date, Author, Contact Info

Encounter
Elements
Encounter ID, Encounter Date, Responsible Party, Contact Info

Document Meta
Elements
Document maintained by, Contact Info

Medications
Elements
Medication, Code, Type, Strength, Dose, Route, Frequency, Date Started, Status

Problems
Elements
Problem Code, Patient Problem, Status, Date Diagnosed

Allergies, Adverse Reactions, Alerts
Elements
Allergy Code, Substance, Medication/Agent Allergy, Reaction, Date Entered

Vital Signs
Elements
Date/Time, Heigh, Weight, Blood Pressure, Pulse, Temp, Resp, Pulse Ox, 02 % BldC Ox, Inhaled Ox Conc

Care Team Information
Elements
Name, Position, Email

Immunizations
Elements
Empty (Immunizations not provided at clinics)

Social History (smoking)
Elements
Smoking Code, Smoking Status, from / to

Medical Equipment
Elements
Description, Date and Time, status

Results (Tests)
Elements
Code, Code System, Name, Date, Result, Lab, Provider

Procedures
Elements
Description, Date and Time, status

Goals
Elements
(free form)

Health Concerns
Elements
(free form)

Plan of Treatment
Elements
Plan of Treatment, Assessment

Reason for referral
Elements
(free form)
