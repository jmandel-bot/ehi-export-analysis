EHI Export Schema
Table obf_pregnancy
Description This table stores information about pregnancies, including patient details, father details, assessment values, and other related data.
Column DataType Description
Pregid (Primary Key) int This column stores unique id for each pregnancy of a patient. This is auto-incremented primary key in This column .
Patientid int This column stores id of patient who is pregnant. This column refers to UID column in user table.
Discharge_date datetime This column stores date on which patient was discharged. Date is stored in 'YYYY-MM-DD 00:00:00.000' format.
OBFStat char This column stores character Y/N to indicate status of current pregnancy. 'Y' - Pregnancy is marked as closed, 'N' - Pregnancy is still active.
LastScreen int This column is not being used.
Lock_Ind int This column is not being used.
DelStat int This column is not being used.
Uname varchar This column stores name of a provider/staff who created the pregnancy in the system from user interface.
Upd_Date datetime This column stores date and time when pregnancy was created by provider/staff from the User Interface. Date and time are stored in 'YYYY-MM-DD hh:mm:ss.sss' format.
father_id int This column is not being used.
Father_First_Name varchar This column stores First Name of the father as documented by provider/staff in the User Interface.
Father_Middle_Initial varchar This column stores Middle Name Initial of the father as documented by provider/staff in the User Interface.
Father_Last_Name varchar This column stores Last Name of the father as documented by provider/staff in the User Interface.
Asmtid int This column stores integer id of an assessment (ICD) documented by provider/staff while creating pregnancy in the system. This column refers to itemid column in items table.
AsmtValue varchar This column stores ICD code selected by provider/staff while creating pregnancy in the system.
Created_Date datetime This column stores date when pregnancy was created in the system. Date is stored in 'YYYY-MM-DD 00:00:00.000' format.
PrimaryOBProviderid int This column stores id of a primary provider of the patient. This column refers to UID column in user table.
NumberOfBabies tinyint This column stores number of babies patient is going to deliver as part of current active pregnancy.
notes varchar This column stores free text notes documented by end user on "Preg List" screen.
sticky_notes varchar This column stores free text notes documented by provider/staff under Sticky Notes section on User Interface.
