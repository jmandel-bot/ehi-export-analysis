EHI Export Schema
Table employer
Description This table stores information about employers, including their contact details, address, and related metadata.
Column DataType Description
ID (Primary Key) int This column is auto-incremented primary key in This column . Unique id is assigned to each employer
EmpName varchar This column is used to store the employer name
empAddress1 varchar This column is used to store address 1 for the employer.
empAddress2 varchar This column is used to store address 2 for the employer.
empCity varchar This column is used to store city for the employer.
empState char This column is used to store state for the employer.
empFax varchar This column is used to stores fax no for the employer.
empPhone varchar This column is used to stores phone no for the employer.
AddedBy int This column is used to store the user who added the employer
AddedAt datetime This column is used to store the date/time when the employer is added
UpdatedBy int This column is used to store the user who updated the employer
UpdatedAt datetime This column is used to store the user when the employer is updated
delFlag int This column is used to store the deleteflag for the employer
Notes text This column is used to store notes for the employer.
ContactPerson varchar This column is used to store contact person for the employer.
EmailAddress varchar This column is used to store email address for the employer.
empZip varchar This column is used to store zipcode for the employer.
CompanyCode varchar Column stores company code associated with the employer.
EnterpriseId int Column stores enterprise id associated with the employer.
faxDialingCountryId int Identifier of the country code for dialing the employer's fax number. Foreign key relation: `countrycodes(id)`. Example: 1, 2, 3.
