EHI Export Schema
Table vision_prescription
Description This table stores vision prescription information for patients, including prescription details for the right and left eyes, prescription date, and expiration date.
Column DataType Description
id (Primary Key) int Unique identifier for each vision prescription record. Example: 1, 2, 3, etc.
OrigPrescriptionid int Original prescription identifier, referring to another prescription record. Example: 0, 1, 2, etc.
PrescriptionDate datetime Date and time when the prescription was created. Example: '2021-09-15 10:30:00'.
Type varchar Type of vision prescription, such as glasses or contact lenses. Example: 'glasses', 'contacts'.
ExpDate datetime Expiration date and time of the prescription. Example: '2022-09-15 10:30:00'.
encid varchar Encounter identifier, which can be used to link this prescription to a specific patient encounter. Example: 'E12345', 'E67890'.
PrescriptionRight text Text description of the prescription for the right eye. Example: '-1.50 sphere, -0.75 cylinder, 180 axis'.
PrescriptionLeft text Text description of the prescription for the left eye. Example: '-1.25 sphere, -0.50 cylinder, 170 axis'.
deleteflag int Flag to indicate if the record has been deleted (1) or not (0). Example: 0, 1.
DateModified datetime Date and time when the record was last modified. Example: '2021-09-15 11:00:00'.
