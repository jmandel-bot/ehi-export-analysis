Table qrsdetail
Description This table stores information about lab test reports, including details about the report file, lab, and associated account.
Column DataType Description
Testid (Primary Key) int Unique identifier for each test report. Example: 1, 2, 3, etc.
Encid int identifier for the associated encounter. Example: 101, 102, 103, etc.
AcctNum int identifier for the associated account. Example: 1001, 1002, 1003, etc.
Reportid int Identifier for asssociated report
FileName varchar Name of the report file. Example: 'report1.pdf', 'report2.pdf', 'report3.pdf', etc.
ReportDate datetime Date and time when the report was generated. Example: '2021-09-01 12:34:56', '2021-09-02 14:56:23', etc.
LabType int Column is used for lab type like Labs, DI, Procedure
LabName varchar Name of the lab that conducted the test. Example: 'LabCorp', 'Quest Diagnostics', 'BioReference', etc.
delFlag int Flag to indicate if the record has been deleted (1) or not (0). Example: 0, 1.
