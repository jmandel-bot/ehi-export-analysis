Table patient_staff_signature_detail
Description This table stores the signature details of patients and staff members for a specific encounter, including the time of signature updates and deletion flags.
Column DataType Description
id (Primary Key) int This column is auto-generated unique primary key.
encounterid int identifier for the encounter associated with the signature details. Foreign key relation: `enc.encounterid`. Example: 101, 102, 103.
patientdocid int identifier for the patient's signature document. Foreign key relation: `user_signature_document.id`. Example: 1001, 1002, 1003.
staffdocid int identifier for the staff's signature document. Foreign key relation: `user_signature_document.id`. Example: 2001, 2002, 2003.
ptsignatureupdatetime datetime Date and time when the patient's signature was last updated. Example: '2022-01-01 12:34:56', '2022-01-02 14:30:00'.
staffsignatureupdatetime datetime Date and time when the staff's signature was last updated. Example: '2022-01-01 15:00:00', '2022-01-02 16:45:00'.
deleteflag tinyint Flag indicating whether the record has been deleted (1) or not (0). Example: 0, 1.
