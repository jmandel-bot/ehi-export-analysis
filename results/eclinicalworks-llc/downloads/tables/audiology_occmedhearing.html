Table audiology_occmedhearing
Description This table stores occupational medicine hearing test data for employees, including exposure levels, test dates, and related information.
Column DataType Description
OccMedid (Primary Key) int Unique identifier for each occupational medicine hearing test record. Example: 1, 2, 3.
encounterid int identifier for the encounter associated with the hearing test. Example: 101, 102, 103.
testDate datetime Date and time when the hearing test was conducted. Example: '2021-08-01 14:30:00'.
exposureLevelInHours tinyint Number of hours the employee was exposed to noise. Example: 8, 12, 16.
exposureLevelInDb tinyint Noise exposure level in decibels (dB) the employee was exposed to. Example: 85, 90, 95.
chartData varchar A string containing chart data related to the hearing test. Example: '1,2,3;4,5,6;7,8,9'.
retestDate datetime Date when a retest is scheduled, if applicable. Example: '2021-08-15'.
reviewedByid int identifier for the user who reviewed the hearing test results. Foreign key relation to `users` table, `UID` column. Example: 5, 6, 7.
examinedByid int identifier for the user who examined the employee during the hearing test. Foreign key relation to `users` table, `UID` column. Example: 8, 9, 10.
educationProvidedDate datetime Date when education was provided to the employee regarding hearing protection. Example: '2021-08-02'.
employeeTrainedDate datetime Date when the employee was trained on hearing protection. Example: '2021-08-03'.
scheduledDate datetime Date when the hearing test was scheduled. Example: '2021-07-31'.
recordUpdated tinyint A flag indicating whether the record has been updated (1) or not (0). Example: 0, 1.
recordCreatedDate datetime Date and time when the hearing test record was created. Example: '2021-07-30 10:00:00'.
recordModifiedDate datetime Date and time when the hearing test record was last modified, if applicable. Example: '2021-08-01 16:00:00'.
structureDataid int identifier for the structure data related to the hearing test. Foreign key relation to `structaudiology` table, `structureDataid` column. Example: 20, 21, 22.
calibrationdate date Date when the hearing test equipment was last calibrated. Example: '2021-07-01'.
Empid int identifier for the employee who underwent the hearing test. Foreign key relation to `employment_history` table, `id` column. Example: 1001, 1002, 1003.
