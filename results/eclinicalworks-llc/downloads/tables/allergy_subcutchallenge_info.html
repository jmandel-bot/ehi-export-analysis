EHI Export Schema
Table allergy_subcutchallenge_info
Description This table stores information about allergy subcutaneous challenge tests performed on patients, including the test details, reactions, and the staff involved in the process.
Column DataType Description
id (Primary Key) int The column stores an id of the allergy_subcutchallenge_info table. This is an auto incremental primary key of This column .
ENCOUNTERid int This column stores encounter id which is mapped with challenge test. It refers to column encounterid from enc table.
TESTid int This column stores test id which is mapped with the test. It refers to column testid from allergy_test table.
CATid int This column stores category id which is mapped with the category item. It refers to column itemid from items table.
CATNAME varchar This column stores category name which is mapped with the category item. It refers to column itemName from items table.
ITEMid int This column stores item id which is mapped with the allergen item. It refers to column itemid from items table.
ITEMNAME varchar This column stores item name which is mapped with the allergen item. It refers to column itemName from items table.
TYPE char This column stores type of test, Values stored 'C' represents Challenge Test
REACTION varchar This column stores reaction value of given dose for challenge test.
ROUTE varchar This column stores route value of given dose for challenge test.
AMOUNT varchar This column stores dose amount value of challenge test.
PARENTid int This column stores parentid of allergen in case of child allergens for challenge test. Default value of parentid is 0.
TIME datetime This column stores time of administration for challenge test.
REVIEWED_BY varchar This column stores reviewed by userid for challenge test. It refers to UID column of user table.
IS_POSITIVE int This column stores is positive flag which represents if the patient is allergic to specific allergen for challenge test.
GIVENTIME datetime This column stores given time at which allergen was given to patient for challenge test.
REVIEWED_BYNAME varchar This column stores reviewed by userName for challenge test.
DONE_BY int This column stores signed off userid for challenge test. It refers to UID column of user table.
DONE_BYNAME varchar This column stores signed off user name for challenge test.
