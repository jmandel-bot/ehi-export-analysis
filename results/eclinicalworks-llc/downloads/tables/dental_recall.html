EHI Export Schema
Table dental_recall
Description This table stores all the basic information of dental recalls which is used in the Dental Recall configuration screen, and in the CDSS > Dental tab.
Column DataType Description
id (Primary Key) int This column is auto-incremented primary key in This column . Unique id is assigned to each Dental Recall data when user create new recall(Generic or Patient Specific). It is defined as NOT NULL constraint.
NAME varchar This column stores Name of Dental Recall (Associated dental item name).
Itemid int This column stores itemid of Dental Recall (Associated itemid column value on dentalitems table).
LowerAge int This column stores Lower Age value configuration of Generic Recall.
LowerAgeFlag varchar This column stores Lower Age Flag value (Y=Year and M = Month) configuration of Generic Recall.
UpperAge int This column stores Upper Age value configuration of Generic Recall.
UpperAgeFlag varchar This column stores Upper Age Flag value (Y=Year and M = Month) configuration of Generic Recall.
Frequency int This column stores Frequency value configuration of Generic Recall.
FrequencyFlag varchar This column stores Frequency Flag value (Y=Year and M = Month) configuration of Generic Recall.
Gender varchar The gender for which the dental recall is applicable. Example: "Male", "Female", "All".
WebRef varchar A reference to a web resource related to the dental recall. Example: "https://example.com/dental_recall_101".
LastUpdated datetime time stamp of when the dental recall record was last updated. Example: "2021-08-01 12:34:56".
DeleteFlag tinyint Flag to indicate if the dental recall record has been deleted. Example: 0 (not deleted), 1 (deleted).
Patientid int This column stores information of patient's details of dental Recalls for patient specific recalls only which is used in CDSS> Dental tab screen. For Generic recalls, Patient id value=0.
ProcButtonid int This column stores identifier number of proc Button id. It refers to column procButtonNum from ProcButton table. User can set this value using Procedure button field from Dental Recall Configuration screen.
ProcButtonName varchar This column stores name of proc Button. It refers to column itemName from Definition table. User can set this value using Procedure button field from Dental Recall Configuration screen.
IsRecurrent tinyint Flag to indicate if the dental recall is recurrent. Example: 0 (not recurrent), 1 (recurrent).
DueDate datetime The date when the dental recall is due. Example: "2021-08-15", "2021-09-01".
SuppressDate datetime The date when the dental recall is suppressed. Example: "2021-08-10", "2021-09-05".
SuppressFlg tinyint Flag to indicate if the dental recall is suppressed. Example: 0 (not suppressed), 1 (suppressed).
SuppressNotes text Notes related to the suppression of the dental recall. Example: "Patient requested postponement", "Rescheduled due to dentist's unavailability".
addOneDay tinyint Flag to indicate if one day should be added to the due date. Example: 0 (do not add), 1 (add).
PermanentSuppressFlg tinyint Flag to indicate if the dental recall is permanently suppressed. Example: 0 (not permanent), 1 (permanent).
