Table bh_usertime_enc_cpt_mapping
Description This table stores the mapping between user time, encounters, and items, along with their respective durations and units.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
userTypeid int identifier for the user type associated with the record. Foreign key relation: `bh_cptusertype(id)`. Example: 101, 102, 103.
encounterid int identifier for the encounter associated with the record. Foreign key relation: `enc(encounterid)`. Example: 1001, 1002, 1003.
duration datetime The duration of the encounter in the datetime format. Example: '2022-01-01 10:30:00', '2022-01-02 12:45:00'.
itemid int identifier for the item associated with the record. Foreign key relation: `items(itemid)`. Example: 501, 502, 503.
units decimal The number of units for the item in the record, stored as a decimal with 12 digits and 2 decimal places. Default value is '0.00'. Example: 1.00, 2.50, 3.75.
delflag tinyint A tinyint(1) flag indicating whether the record is deleted or not. Default value is '0'. Example: 0 (not deleted), 1 (deleted).
