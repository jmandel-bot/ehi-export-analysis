EHI Export Schema
Table epa_rx_detail
Description This table stores detailed information about medications ordered through EPA, including medicine details, prescribing doctor, patient, and pharmacy information.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table (e.g., 1, 2, 3).
item_id int Identifier for the medicine item, foreign key relation to `items` table (`itemID` column).
medicine_name varchar Name of the prescribed medicine (e.g., "Ibuprofen", "Amoxicillin").
ncpdpid varchar National Council for Prescription Drug Programs ID, a unique identifier for pharmacies (e.g., "1234567").
encounter_id int Identifier for the patient encounter, foreign key relation to `enc` table (`encounterID` column).
doctor_id int Identifier for the prescribing doctor, foreign key relation to `doctors` table (`doctorID` column).
patient_id int Identifier for the patient, foreign key relation to `patients` table (`pid` column).
facility_id int Identifier for the facility where the prescription was made, foreign key relation to `edi_facilities` table (`Id` column).
refill smallint Number of refills allowed for the prescription (e.g., 0, 1, 2).
assess_id int Assessment identifier, used for internal purposes (e.g., 123, 456, 789).
ndc_code varchar National Drug Code, a unique identifier for the prescribed drug (e.g., "12345-6789-01").
rx_order_no varchar Prescription order number (e.g., "RX123456").
strength varchar Strength of the prescribed medicine (e.g., "200 mg", "500 mg").
formulation varchar Formulation of the prescribed medicine (e.g., "tablet", "capsule").
take varchar Instructions on how to take the medicine (e.g., "Take 1 tablet every 6 hours").
route varchar Route of administration for the medicine (e.g., "oral", "intravenous").
frequency varchar Frequency of taking the medicine (e.g., "every 6 hours", "twice daily").
duration varchar Duration of the treatment (e.g., "7 days", "2 weeks").
dispense varchar Quantity of medicine to be dispensed (e.g., "30 tablets", "10 capsules").
epa_context varchar Context of the electronic prescription authorization (e.g., "new", "refill").
assign_to_id int Identifier for the person assigned to process the prescription (e.g., 123, 456, 789).
ss_encounter_id int Identifier for the encounter in a secondary system, used for internal purposes (e.g., 123, 456, 789).
old_rx_id int Identifier for the previous prescription, if any (e.g., 123, 456, 789).
epa_transaction_id int Identifier for the electronic prescription authorization transaction, foreign key relation to `epa_transaction` table (`id` column).
daw_flag tinyint Dispense as written flag, 1 if the prescription must be dispensed as written, 0 otherwise (e.g., 0, 1).
delete_flag tinyint Flag indicating if the record has been deleted, 1 if deleted, 0 otherwise (e.g., 0, 1).
addInstructions varchar Additional instructions for the prescription (e.g., "Take with food", "Do not crush").
rx_notes varchar Notes related to the prescription (e.g., "Patient allergic to penicillin", "Monitor blood pressure").
rx_additional_notes varchar Additional notes related to the prescription (e.g., "Patient prefers brand name", "Dispense in child-resistant container").
pharmacy_id int Identifier for the pharmacy, foreign key relation to `pharmacy` table (`pmcid` column).
rx_source varchar Source of the prescription (e.g., "EHR", "Paper").
rx_source_id int Identifier for the prescription source, used for internal purposes (e.g., 123, 456, 789).
start_date date Start date of the prescription (e.g., "2021-01-01").
stop_date date Stop date of the prescription (e.g., "2021-01-08").
