Table allergy_shot_changebtlnoreason
Description This table stores the reason for changing the Bottle Number on the Shot Admin window, including details about the concentration, formulation, patient, and encounter.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table.
Concentrationid int identifier for the concentration of the allergy shot, foreign key relation to `antigen_concentration_patient` table's `CONCENTRATION_id` column.
Formulationid int identifier for the formulation of the allergy shot, foreign key relation to `allergy_formulation` table's `id` column.
Queueid int identifier for the queue of the allergy shot, foreign key relation to `antigen_recipe_queue` table's `id` column.
Patientid int identifier for the patient receiving the allergy shot, foreign key relation to `patients` table's `pid` column.
Encounterid int identifier for the encounter related to the allergy shot, foreign key relation to `enc` table's `encounterid` column.
concentrationName varchar Name of the concentration used in the allergy shot (e.g., "High", "Medium", "Low").
LotNo varchar Lot number of the allergy shot (e.g., "L12345").
ChangeBottleReason varchar Reason for changing the bottle without a reason (e.g., "Expired", "Contaminated").
ChangeBottleReasonOthers varchar Additional information or other reasons for changing the bottle without a reason (e.g., "Label damaged", "Leaking").
UpdatedByid int identifier for the user who updated the record, foreign key relation to `users` table's `UID` column.
