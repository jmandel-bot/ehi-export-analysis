EHI Export Schema
Table flowsheetitems
Description This table stores information about items in a flowsheet, including their names, types, and display properties.
Column DataType Description
id (Primary Key) int This column is an auto incremented Primary Key.
flowSheetid int This column stores flowsheet id to which the new item is added. This column refers column id from flowsheets table.
name varchar This column stores name of the item added.
type smallint This column stores one of the below ids for the item from the drop-down list to which the item is associated. {id: 1, value: 'Labs'}, {id: 2, value: 'XRays'}, {id: 3, value: 'Immunizations'}, {id: 4, value: 'Rx'}, {id: 5, value: 'Vitals'}, {id: 6, value: 'Structured', invoke: $scope.openStructured}, {id: 7, value: 'Drug Class'}, {id: 8, value: 'Rx Group'}, {id: 9, value: 'Procedures'}, {id: 10, value: 'Lab Attributes'}, {id: 0, value: 'Others'} There is no master table for this ids.
itemid int This column stores an id of the item associated. This column refers column itemid from items table.
itemName varchar This column stores the item name of the item from itemid column. This column refers column itemName from items table.
displayIndex int This column stores the sequence of the item added, this is used to display the same item in User Interface.
deleteFlag smallint This column store a value 1 - when the added item to a flowsheet is deleted or 0 - Otherwise.
Categoryid int This column is not used.
GoalLowerLimit varchar This column stores a lower limit goal for the item added.
GoalUpperLimit varchar This column stores upper limit goal for the item added.
groupName varchar This column stores name of the group for the item added. The associated item names are grouped under this group.
labAttributes text This column stores id of the attribute associated to the lab selected. This column refers column itemid from items table.
displayinheader tinyint This column stores a value 1 - when Header box for flowsheet items is checked to mark as headers or 0 - Otherwise.
