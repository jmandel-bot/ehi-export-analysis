Table prisma_providers
Description This table stores information about healthcare providers, including their personal details, address, and specialty.
Column DataType Description
id (Primary Key) int Unique identifier for each provider (e.g., 1, 2, 3). This is the primary key.
name varchar Full name of the provider (e.g., "Dr. John Doe").
speciality varchar Medical specialty of the provider (e.g., "Cardiology", "Dermatology").
npi varchar National Provider Identifier, a unique 10-digit identification number for healthcare providers in the United States (e.g., "1234567890").
providerType varchar Type of healthcare provider (e.g., "Physician", "Nurse Practitioner").
addressLine1 varchar First line of the provider's address (e.g., "123 Main St").
addressLine2 varchar Second line of the provider's address, if applicable (e.g., "Apt 4B").
addressCity varchar City of the provider's address (e.g., "New York").
addressState varchar State of the provider's address (e.g., "NY").
addressCountry varchar Country of the provider's address (e.g., "USA").
addressPostalCode varchar Postal code of the provider's address (e.g., "10001").
createdDate datetime Date and time when the provider's record was created (e.g., "2022-01-01 12:34:56").
deleteFlag tinyint A flag indicating whether the provider's record has been deleted (0 for not deleted, 1 for deleted).
prismaEncounterId int Identifier for the associated encounter in the `prisma_encounters` table. This is a foreign key that refers to the `id` column in the `prisma_encounters` table.
