EHI Export Schema
Table ip_rh_orderset_diagnosis
Description This table appears to store order mapping that belongs to icd.
Column DataType Description
Id (Primary Key) int Unique identifier for each record in the ip_rh_orderset_diagnosis table. This column is used in the UI for identifying records and is also crucial for database operations like joins.
itemId int Foreign key that references an item in another table. This column is essential for linking diagnosis to specific order items.
encounterId int Foreign key that references an encounter in another table. This column is essential for linking the diagnosis to specific encounters.
deleteflag tinyint Indicates whether the record is marked for deletion. This column is used to softly delete records, i.e., mark them as deleted without actually removing them from the database.
