EHI Export Schema
Table adjustmentposting
Description This table stores information about adjustment postings related to inventory, line items, and amounts.
Column DataType Description
id (Primary Key) int Unique identifier for each adjustment posting record. Example: 1, 2, 3.
Adjustmentid int This column is used to store the Adjustment ID.
Invid int This column is used to store the Invoice ID.
LineItemid int This column is used to store the Line Item ID.
amount decimal The amount of the adjustment posting, stored as a decimal value with 9 digits and 2 decimal places. Example: 150.25, 300.50, 450.75.
DeleteFlag int A flag indicating whether the record has been deleted or not. A value of 0 means not deleted, and a value of 1 means deleted. Example: 0, 1.
AddedBy int identifier for the user who added the adjustment posting record. This is a foreign key that refers to the id column in the user table. Example: 10, 11, 12.
AddedTime datetime The date and time when the adjustment posting record was added, stored as a date time value. Example: 2021-09-01 12:34:56, 2021-09-02 14:30:00.
UpdatedBy int identifier for the user who last updated the adjustment posting record. This is a foreign key that refers to the id` column in the user table. Example: 10, 11, 12.
UpdatedTime datetime The date and time when the adjustment posting record was last updated, stored as a date time value. Example: 2021-09-01 12:34:56, 2021-09-02 14:30:00.
closingid int This column is used to store the Closing ID.
discountType tinyint A tiny integer value representing the type of discount applied to the adjustment posting. Example: 0, 1, 2.
