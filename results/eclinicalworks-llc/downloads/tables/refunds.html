EHI Export Schema
Table refunds
Description This table stores information about refunds issued to patients or payers including the amount, check details, and related timestamps.
Column DataType Description
RefundId (Primary Key) int Unique identifier for each refund record. Example: 1, 2, 3.
ReceiverType smallint Identifier for the type of receiver (e.g., customer, vendor). Example: 0, 1.
ReceiverId int Unique identifier for the receiver of the refund. Example: 101, 102, 103.
amount decimal The amount of the refund. Example: 100.00, 50.25.
checkNo varchar The check number associated with the refund. Example: '123456', '789012'.
type varchar The type of refund. Example: 'cash', 'credit'.
date datetime The date the refund was issued. Example: '2021-08-01', '2021-08-02'.
memo text Additional notes or comments about the refund. Example: 'Refund for damaged item', 'Customer overpayment'.
PostedBy int Unique identifier for the user who posted the refund. Example: 1, 2, 3.
PostedDt datetime The date the refund was posted. Example: '2021-08-01', '2021-08-02'.
PostedTime datetime The time the refund was posted. Example: '12:30:00', '14:45:00'.
modifiedDate datetime The time stamp when the refund record was last modified. Example: '2021-08-01 12:30:00', '2021-08-02 14:45:00'.
facilityId int Unique identifier for the facility associated with the refund. Example: 1, 2, 3.
refundLock int Indicates if the refund record is locked (1) or unlocked (0). Example: 0, 1.
lockedBy int Unique identifier for the user who locked the refund record. Example: 1, 2, 3.
deleteFlag smallint Indicates if the refund record is deleted (1) or not deleted (0). Example: 0, 1.
PrintStatus int Indicates the print status of the refund (0: not printed, 1: printed). Example: 0, 1.
CheckRemitNotes text Additional notes or comments about the check remittance. Example: 'Sent via mail', 'Hand-delivered'.
PrintedBy int Unique identifier for the user who printed the refund. Example: 1, 2, 3.
PrintDate datetime The date the refund was printed. Example: '2021-08-01', '2021-08-02'.
PrintTime datetime The time the refund was printed. Example: '12:30:00', '14:45:00'.
PayeeName varchar The name of the payee for the refund. Example: 'John Doe', 'Jane Smith'.
closingId int Unique identifier for the closing associated with the refund. Example: 1, 2, 3.
unpostedrefunds decimal The amount of unposted refunds. Example: 100.00, 50.25.
UnpostedCPT decimal The amount of unposted CPT (Current Procedural Terminology) codes. Example: 100.00, 50.25.
refundType varchar The type of refund, either positive (+ve) or negative (-ve). Example: '+ve', '-ve'.
VoidRefundRefId int Unique identifier for the void refund reference. Example: 1, 2, 3.
