Table ip_rh_vitalsQualifierMaster
Description This table will be used to store qualifier name and qualifier type.
Column DataType Description
Id (Primary Key) int This is an auto-incremented primary key column in this table. It is used to uniquely identify the vitals qualifier. The data type is int as per business requirement and the column cannot be NULL.
qualifierName varchar This column stores the name of the qualifier for the vital.
qualifierType varchar This column stores the type of the qualifier. For qualifier ‘Site', the value will be ‘S'. For qualifier ‘Position', the value will be ‘P' and for qualifier ‘Type', the value will be ‘T'.
deleteFlag int A flag to indicate if the qualifier is deleted(1) or not(0). The data type is tinyint and the default value is 0.
modifiedBy int Stores the user Id of the person who last modified the qualifier. It refers to uid column in users table. Datatype is int. The default value for this column is NULL.
modifiedDateTime datetime Stores the date and time the qualifier was last modified. The datetime data type allows for precise storage of date and time values. The default value is NULL.
