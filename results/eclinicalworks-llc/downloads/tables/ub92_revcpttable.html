EHI Export Schema
Table ub92_revcpttable
Description This table stores information about revenue and CPT codes associated with invoices.
Column DataType Description
Invid int An integer representing the unique identifier of an invoice. Example: 12345 (Foreign key relation: `edi_invoice.id`)
RevCode varchar A string representing the revenue code associated with the invoice. Example: "0450"
Description varchar A string describing the service or item associated with the revenue code. Example: "Emergency Room"
CPTCode varchar A string representing the Current Procedural Terminology (CPT) code for the service or item. Example: "99281" (Foreign key relation: `cptcodes.Code`)
SrvDate varchar A string representing the date the service was provided, in the format "YYYY-MM-DD". Example: "2021-08-15"
ServUnits varchar A string representing the number of service units provided. Example: "2"
TotCharges varchar A string representing the total charges for the service or item. Example: "150.00"
NovCvrdCharges varchar A string representing the non-covered charges for the service or item. Example: "0.00"
SeqNo int An integer representing the sequence number of the revenue code and CPT code pair within the invoice. Example: 1
RateCode varchar A string representing the rate code associated with the service or item. Example: "RC1"
Itemid int An integer representing the unique identifier of the item or service. Example: 56789 (Foreign key relation: `items.itemid`)
