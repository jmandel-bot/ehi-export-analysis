EHI Export Schema
Table rpm_pn_imported_notes
Description This table stores and displays all the rpm notes of a patient imported by user.
Column DataType Description
id (Primary Key) int This column is used to identify unique records. It is an auto incremented primary key of This column . Data type is int. This field doesn't allow NULL value.
encounterid int This is a foreign key column. It refers to primary key column encounterid of enc table. The data type of this column is INT and it does not allow a NULL value.
importedNotes varchar This column stores the imported notes. This column can have large data as it stores the notes added by the user, hence the data type is varchar(MAX). It can never have a NULL value hence declared as NOT NULL.
notes varchar This column stores the additional notes. This column can have large data as it stores the notes added by the user, hence the data type is varchar(MAX). It can never have a NULL value hence declared as NOT NULL.
modifiedBy int This is a foreign key column.It refers to primary key column UID of user table. This column stores the id of user who modified the record last. The data type of this column is INT and it does not allow a NULL value.
createdDateTime datetime Contains date time when the record was created and datatype is DATETIME. This column stores datetime hence the data type is DATETIME. This column cannot have a NULL value.
modifiedDateTime datetime Contains date time when the record was last modified and datatype is DATETIME. This column stores datetime hence the data type is DATETIME. This column cannot have a NULL value.
deleteflag tinyint This flag indicates that this record has been deleted or not, datatype is TINYINT. This column stores an integer value hence tinyint is selected.  Values - 0: Not Deleted 1: Deleted.  By default, record must not be in delete status hence default 0 is chosen.
