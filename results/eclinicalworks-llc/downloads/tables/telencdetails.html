EHI Export Schema
Table telencdetails
Description This table stores patient records and details linked to specific encounters.
Column DataType Description
id (Primary Key) int This is the primary key column that uniquely identifies each record in the table. It is an integer with a maximum length of 11 digits and is automatically incremented for each new record. Example: 1, 2, 3, etc.
telencId int This column stores the telecommunication encounter ID, which is an integer with a maximum length of 11 digits. It is used to associate the details with a specific telecommunication encounter. Example: 101, 102, 103, etc.
encounterId int This column stores the encounter ID, which is an integer with a maximum length of 11 digits. It is used to associate the telecommunication encounter details with a specific encounter. This column acts as a foreign key that refers to the `id` column in the `encounters` table. Example: 201, 202, 203, etc.
