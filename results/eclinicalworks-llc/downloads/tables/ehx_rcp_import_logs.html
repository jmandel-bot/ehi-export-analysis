EHI Export Schema
Table ehx_rcp_import_logs
Description This table is used for logging data imported from the eEHX RCP/DRTLA tab to the Progress Notes.
Column DataType Description
id (Primary Key) int auto incremented id for This column . purpose of This column is to have back end logs for the data imported using eEHX/DRTLA RCP tab.
pid int This columns stores the patient id for which data import is attempted. It refers to users.UID column
encid int This column stores the encounter id to which data from right rcp imported to patient's progress note
userid int This column stores the userid who performed import operation. This refers to users.UID column
sectionid int section id which we have imported data for like allergies = 6, medication = 8, problem list = 2.
section_name varchar Name of the section associated with the `sectionid`. Example: 'Allergies', 'Medications', 'Problems'.
bulkImport tinyint This column stores 0 to indicate that single data was imported and stores 1 if the data was imported through the Import All button.
nhxcall tinyint this column have value 1 if data is imported from DRTLA workflow.
importDateTime datetime Date and time when the import was performed. Example: '2021-09-01 12:34:56', '2021-09-02 14:23:45'.
import_data text this column will contain allergy data received from ccr while attempting import. similar data is not getting populated for problem list and medication.
deleteFlag tinyint if row is marked deleted. this column is not getting populated through any application logic as of today.
