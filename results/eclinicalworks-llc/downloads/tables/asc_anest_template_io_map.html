EHI Export Schema
Table asc_anest_template_io_map
Description This table is used to store IO selected details for an anesthesia template.
Column DataType Description
id (Primary Key) int Unique identifier for each mapping entry. Example: 1, 2, 3.
templateid int This column stores ASC template id. It refers to id column of asc_anest_template_master table.
ioItemid int This column stores an io(Intake/Output) item id column. It refers to id column of asc_ioitems table.
itemType tinyint This column stores a value of item type. 0 - INTAKE,1- OUTPUT.
deleteFlag tinyint Indicates whether the mapping entry has been deleted (0 for not deleted, 1 for deleted). Example: 0, 1.
isDefault tinyint Indicates whether the input/output item is a default item for the anesthesia template (0 for not default, 1 for default). Example: 0, 1.
