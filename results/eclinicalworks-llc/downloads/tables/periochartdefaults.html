EHI Export Schema
Table periochartdefaults
Description This table stores the default settings for a patient's periodontal chart, including various measurements and color codes for different dental conditions.
Column DataType Description
id (Primary Key) bigint This column is auto-incremented primary key in This column . Unique id is assigned to each Perio defaults setting when they are created for any patient. It is defined as NOT NULL constraint.
PatNum bigint This column stores identifier number of a patient, for whom the appointment is created. It refers to column pid from patients table.
ProbingNum smallint This column stores Probing value of a patient's perio chart.
MgjNum smallint This column stores MGJ –(Muco-gingival junction) number of a patient's perio chart.
GingMargNum smallint This column stores Ging marg number (Gingival Margin value for each tooth) of a patient's perio chart.
CalculusNum smallint This column stores calculus number of a patient's perio chart.
FurcationNum smallint This column stores furcation number value of a patient's perio chart.
MobilityNum smallint This column stores Mobility number of a patient's perio chart.
PlaqueColorCode varchar This column stores Plaque color code value of a patient's perio chart.
CalculusColorCode varchar This column stores calculus color code value of a patient's perio chart.
BleedingColorCode varchar This column stores bleeding color code value of a patient's perio chart.
SuppuratioinColorCode varchar This column stores suppuratioin color code value of a patient's perio chart.
CALColorCode varchar This column stores CAL (Clinical attachment loss) color code value of a patient's perio chart to show on UI side.
FurcColorCode varchar This column stores Furc ("Furcation" for each tooth) value color code to show on UI side of a patient's perio chart.
FurcRedColorCode varchar This column stores Furc ("Furcation" for each tooth) Red color code value of a patient's perio chart.
GingMargColorCode varchar This column stores Ging Marg color code value to show font color of Ging Marg number (Gingival Margin value for each tooth) on patient's perio chart.
MGJColorCode varchar This column stores MGJ –(Muco-gingival junction) color code value to show font color of MGJ number of a patient's perio chart.
ProbingColorCode varchar This column stores PD (Probing depth) color code value of a patient's perio chart to show on UI side.
ProbingRedColorCode varchar This column stores PD (Probing depth) red color code value of a patient's perio chart to show on UI side.
