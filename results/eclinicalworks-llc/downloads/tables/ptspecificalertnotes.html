EHI Export Schema
Table ptspecificalertnotes
Description This table stores suppression data for a patient specific alert.
Column DataType Description
id int This column stores id of the patient specific alert. This column refers column id from ptspecificalert table.
suppress smallint This column stores a value 0 or 1. 1 - when the patient specific alert is suppressed, 0 - Otherwise.
lastdone datetime This column is not used anymore.
notes text This column stores additional notes for the patient specific alert from id column.
suppressedUntilDate datetime The column contains the time stamp (date/time) until which the alert is suppressed.
