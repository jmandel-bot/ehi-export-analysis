EHI Export Schema
Table icd_synonyms
Description This table stores additional details about diagnosis which are coming from third-party vendor.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
lexicalid varchar Unique identified of lexical diagnosis which is coming from IMO
lexicaldesc varchar Description of the lexical entry. Example: "Acute bronchitis", "Chronic sinusitis", "Type 2 diabetes".
createdon datetime Date and time when the record was created. Example: "2021-08-01 12:34:56".
createdby int identifier of the user who created the record. Example: 1, 2, 3.
modifiedon datetime Date and time when the record was last modified. Example: "2021-08-15 14:23:12".
modifiedby int identifier of the user who last modified the record. Example: 1, 2, 3.
vendorid int identifier of the vendor associated with the record. Example: 1, 2, 3.
ICD9_SNOMEDCT_IMO varchar ICD-9 code with SNOMED CT and IMO information. Example: "493.20|195967|Acute bronchitis".
SNOMEDid varchar Unique identifier for the SNOMED CT entry. Example: "195967", "204843", "312567".
PROBIT_SNOMEDCT_IMO varchar PROBIT code with SNOMED CT and IMO information. Example: "P123|195967|Acute bronchitis".
PROBIT_SNOMEDCT_US_IMO varchar PROBIT code with SNOMED CT and US-specific IMO information. Example: "P123|195967|Acute bronchitis (US)".
SCT_US_CONCEPT_id varchar SNOMED CT US-specific concept identifier. Example: "195967", "204843", "312567".
SNOMED_US_DESCRIPTION varchar SNOMED CT US-specific description. Example: "Acute bronchitis (US)", "Chronic sinusitis (US)", "Type 2 diabetes (US)".
SNOMED_DESCRIPTION varchar SNOMED CT description. Example: "Acute bronchitis", "Chronic sinusitis", "Type 2 diabetes".
itemid int This is a foreign key to items table.
SCT_CONCEPT_id varchar SNOMED CT concept identifier. Example: "195967", "204843", "312567".
hassec int Flag indicating whether the record has a secondary code (1 for yes, 0 for no). Example: 0, 1.
icd_data varchar star-separated details about lexical diagnosis
