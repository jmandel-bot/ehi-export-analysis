Table bh_treatment_plan_group_problem_mapping
Description This table stores the mapping between treatment plans, groups, and problems. It helps to associate a specific problem with a group within a treatment plan.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
tpid int The identifier of the treatment plan. Foreign key relation to `bh_treatment_plan` table's `id` column. Example: 101, 102, 103.
groupid int The identifier of the group. Foreign key relation to `bh_group` table's `groupid` column. Example: 201, 202, 203.
problemid int The identifier of the problem. Foreign key relation to `items` table's `itemid` column. This column can have NULL values. Example: 301, 302, NULL.
problemName varchar The name of the problem. This column can have NULL values. Example: "Problem A", "Problem B", NULL.
deleteFlag tinyint A flag to indicate if the record is deleted or not. 0 means not deleted, and 1 means deleted. Default value is 0. Example: 0, 1.
