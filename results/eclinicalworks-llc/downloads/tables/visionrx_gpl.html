Table visionrx_gpl
Description This table stores information related to vision prescriptions, including measurements and attributes for the prescribed lenses.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer value. It uniquely identifies each record in the table. Example: 1, 2, 3, etc.
VisionRxid int Foreign key that refers to the `id` column in the `visionrx` table. It represents the unique identifier for the associated vision prescription. Example: 101, 102, 103, etc.
Encounterid int identifier for the patient encounter during which the prescription was given. It refers to enc table `encounterid` column. Example: 1001, 1002, 1003.
ExamType int This column contains an identifier for the type of eye exam conducted. Example: 1, 2, 3.
Eye_OD_OS tinyint This column indicates whether the prescription is for the right eye (OD) or left eye (OS). Example: 0 (OD), 1 (OS).
OZD_0 varchar This column contains an optical zone diameter measurement for the first lens. Example: "7.5", "7.8", "8.0".
R2_0 varchar Second radial measurement for the first lens. Example: "2.5", "3.0", "3.5".
W2_0 varchar This column contains a second width measurement for the first lens. Example: "0.8", "1.0", "1.2".
R3_0 varchar This column contains a third radial measurement for the first lens. Example: "4.0", "4.5", "5.0".
W3_0 varchar This column contains a third width measurement for the first lens. Example: "1.5", "1.8", "2.0".
RPC_0 varchar This column contains a radial periphery curve measurement for the first lens. Example: "8.0", "8.5", "9.0".
WPC_0 varchar Width periphery curve measurement for the first lens. Example: "2.5", "3.0", "3.5".
BC_1 varchar Base curve measurement for the second lens. Example: "8.0", "8.5", "9.0".
DIA_1 varchar This column contains a diameter measurement for the second lens. Example: "14.0", "14.5", "15.0".
SPH_1 varchar This column contains pherical power measurement for the second lens. Example: "-2.00", "-1.50", "0.00".
CYL_1 varchar This column contains cylindrical power measurement for the second lens. Example: "-0.75", "-0.50", "0.00".
AXIS_1 varchar This column contains an axis measurement for the second lens. Example: "90", "100", "110".
OZD_1 varchar Optical zone diameter measurement for the second lens. Example: "7.5", "7.8", "8.0".
R2_1 varchar Second radial measurement for the second lens. Example: "2.5", "3.0", "3.5".
W2_1 varchar Second width measurement for the second lens. Example: "0.8", "1.0", "1.2".
R3_1 varchar Third radial measurement for the second lens. Example: "4.0", "4.5", "5.0".
W3_1 varchar Third width measurement for the second lens. Example: "1.5", "1.8", "2.0".
RPC_1 varchar Radial periphery curve measurement for the second lens. Example: "8.0", "8.5", "9.0".
WPC_1 varchar Width periphery curve measurement for the second lens. Example: "2.5", "3.0", "3.5".
Color varchar Color of the lens. Example: "Clear", "Blue", "Green".
DOT varchar Dot matrix pattern on the lens. Example: "Small", "Medium", "Large".
CT varchar Center thickness of the lens. Example: "0.08", "0.10", "0.12".
BLEND varchar Blend type of the lens. Example: "Soft", "Medium", "Hard".
AOZO varchar Aspheric or zonal optimization of the lens. Example: "Aspheric", "Zonal".
LENT varchar Lens treatment applied. Example: "UV", "Anti-Reflective", "Scratch-Resistant".
EDGE varchar Edge design of the lens. Example: "Round", "Beveled", "Polished".
EDGET varchar Edge treatment applied to the lens. Example: "Polished", "Tinted", "None".
PRISM varchar Prism correction value for the lens. Example: "0.5", "1.0", "1.5".
MATERIAL varchar Material used for the lens. Example: "Polycarbonate", "Plastic", "Glass".
