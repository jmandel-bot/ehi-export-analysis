EHI Export Schema
Table users_additionalinfo
Description This tables stores users demographic information in extension to users table for Vision module. This table is a part of dead code and will be removed. It was part of Specsavers code earlier which is not used any more.
Column DataType Description
users_addl_UID (Primary Key) int This is auto-incremented primary key in This column . This column stores unique id for each user whose additional information is being stored in This column .
users_UID int This column stores patient's actual UID in EMR database. This column refers to UID column in user table.
users_addl_address3 varchar This column stores patient's additional address.
users_addl_district varchar This column stores the name of the district patient live in.
users_addl_workphone varchar This column stores patient's work phone number.
users_addl_phoneext varchar This column stores patient's phone extension.
users_addl_PreferredName varchar This column stores patient's preferred name.
users_addl_PreferredPhoneMethod varchar This column stores patient's preferred method of communication over phone.
users_addl_preferredContactPhone tinyint This column stores patient's preferred contact number.
users_addl_preferredContactWrkPhone tinyint This column stores patient's preferred work contact number.
users_addl_preferredContactMobPhone tinyint This column stores patient's preferred mobile contact number.
users_addl_preferredContactEmail tinyint This column stores patient's preferred contact email.
users_addl_preferredContactElectronic tinyint This column stores values 0/1 to indicate patient's preferred way of communication is electronic. 0 - Patient prefer not to be contacted by electronic methods, 1 - Patient prefer to be contacted by electronic methods.
users_addl_preferredContactMail tinyint This column stores values 0/1 to indicate patient's preferred way of communication is mail. 0 - Patient prefer not to be contacted by mail, 1 - Patient prefer to be contacted by mail.
users_addl_preferredContactSMS tinyint This column stores values 0/1 to indicate patient's preferred way of communication is SMS. 0 - Patient prefer not to be contacted by SMS, 1 - Patient prefer to be contacted by SMS.
users_addl_status varchar This column stores flags 0/1 to indicate whether the user is marked as active or inactive. 0 - Active, 1 - Inactive.
users_addl_homephone varchar This column stores patient's home phone number.
users_addl_deleteflag tinyint This column stores values 0/1 to indicate whether patient is marked as deleted or not. 0 - Patient is NOT marked as deleted, 1 - Patient is marked as deleted.
users_addl_countyname varchar This column stores name of the county patient lives in.
users_addl_countycode varchar This column stores code of the county patient lives in.
