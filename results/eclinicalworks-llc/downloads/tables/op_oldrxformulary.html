EHI Export Schema
Table op_oldrxformulary
Description This table stores Rx formulary data for outpatients. The item key is: isFormularyEnabled
Column DataType Description
Rxorderno varchar A unique identifier for the prescription order, represented as a varchar(25). Example: "RX12345".
Encounterid int This column stores the encounter id to which the Rx formulary is added. This column refers column encounterid from enc table.
RxFormularyid int This column stores id of the Rx Formulary added. This column refers column id from op_drugformulary table.
Administer smallint This column stores value 0 or 1. 1 - when the medication is administered to a patient, 0 - Otherwise.
Dispense smallint This column stores value 0 or 1. 1 - when the medication is dispensed to a patient, 0 - Otherwise.
IVHydration smallint This column stores 0 or 1. 1 - when the patient opts for Intravenous medication. 0 - Otherwise.
id (Primary Key) int This is auto-incremented primary key in This column . This column stores unique id generated for each configured formulary detail.
mailOrderFlag tinyint This column stores 0 or 1. 1 - when the patient opts for mail order medication. 0 - Otherwise.
rxCardFlag tinyint This column stores 0 or 1. 1 - when the patient opts for rx order medication. 0 - Otherwise.
