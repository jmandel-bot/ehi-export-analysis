EHI Export Schema
Table hi_payor_mtm_notifications
Description This table stores notifications related to payor medication therapy management (MTM) and their associated details.
Column DataType Description
id (Primary Key) int Unique identifier for each notification record. (e.g., 1, 2, 3)
mtm_id int The identifier for the medication therapy management record. Foreign key relation to `hi_payor_mtm` table's `id` column. (e.g., 101, 102, 103)
payor_notification_id varchar Unique identifier for the payor notification. (e.g., "PN001", "PN002", "PN003")
txn_id varchar Unique identifier for the transaction. (e.g., "TXN001", "TXN002", "TXN003")
hi_txn_id varchar Unique identifier for the mtm create transaction created on healow insights cloud. Example: "HITXN123", "HITXN124", "HITXN125".
patient_id int Unique identifier for the patient. (e.g., 1001, 1002, 1003)
provider_npi varchar National Provider identifier (NPI) for the healthcare provider. (e.g., "1234567890", "0987654321")
payor_id varchar Unique identifier for the payor. (e.g., "PAY001", "PAY002", "PAY003")
notification_number int Unique number for the notification. (e.g., 1, 2, 3)
notification_type varchar Type of the notification. (e.g., "Alert", "Reminder", "Warning")
notification_name varchar Name of the notification. (e.g., "Medication Review", "Follow-up Reminder")
recommended_action_message varchar Text containing the recommended action for the notification. (e.g., "Review patient's medication list", "Schedule a follow-up appointment")
assigned_to int Unique identifier for the user assigned to the notification. (e.g., 2001, 2002, 2003)
STATUS varchar Status of the notification. Enum values include 'New', 'New(Read)', 'New(Update)', 'Addressed', 'Suppressed', and 'New(Suppressed)'.
actiondetails varchar The details of the action taken in text format.
created_by int Unique identifier for the user who created the notification record. (e.g., 3001, 3002, 3003)
createdby_name varchar Name of the user who created the notification record. (e.g., "John Doe", "Jane Smith")
created_date datetime Date and time when the notification record was created. (e.g., "2021-01-01 12:00:00")
updated_by int Unique identifier for the user who last updated the notification record. (e.g., 4001, 4002, 4003)
updatedby_name varchar Name of the user who last updated the notification record. (e.g., "John Doe", "Jane Smith")
updated_date datetime Date and time when the notification record was last updated. (e.g., "2021-01-02 12:00:00")
deleteflag tinyint Flag indicating if the notification record has been deleted. 0 for not deleted and 1 for deleted. (e.g., 0, 1)
