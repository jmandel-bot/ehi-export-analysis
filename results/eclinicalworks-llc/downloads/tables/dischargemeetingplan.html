EHI Export Schema
Table dischargemeetingplan
Description This table stores information about discharge meeting plans, including the meeting date and time, notes, and status.
Column DataType Description
id (Primary Key) int This is a primary key column.
dischargePlanId int This column contains the id from dischargeplan_corrections.id column.
meetingDate datetime This column contains the date of meeting.
meetingTime datetime This column contains the time of meeting.
notes text This column contains the meeting notes.
lockFlag tinyint This column contains the lock status of meeting.
isActive smallint This column contains the active status of meeting.
