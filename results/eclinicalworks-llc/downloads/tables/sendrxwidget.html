EHI Export Schema
Table sendrxwidget
Description This table stores information related to prescription orders, their associated encounters, and the users who added them.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
encid int Encounter identifier to which the prescription order is related. Example: 101, 102, 103.
RxOrderNo varchar Unique prescription order number. Example: 'RX12345', 'RX67890'.
AddedFrom varchar Source or location from which the prescription order was added. Example: 'Web', 'Mobile'.
ReportID int Identifier for the report associated with the prescription order. Example: 1001, 1002, 1003.
addedByID int Identifier for the user who added the prescription order. Example: 1, 2, 3.
DATETIME datetime Date and time when the prescription order was added. Example: '2021-09-01 12:34:56'.
assessmentID int Identifier for the assessment related to the prescription order. Example: 2001, 2002, 2003.
refEncID int Reference encounter identifier, if any, related to the prescription order. Example: 101, 102, 103.
providerID int Identifier for the provider associated with the prescription order. Example: 1, 2, 3.
deleteflag tinyint Flag to indicate if the record has been deleted (1) or not (0). Example: 0, 1.
