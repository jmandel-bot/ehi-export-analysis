EHI Export Schema
Table emcoder_mdm_charta
Description This table stores information about MDM (Medical Decision-Making) charts for patients, including various medical conditions and their severity levels.
Column DataType Description
id (Primary Key) int This is the primary key for the table, which is an integer value that uniquely identifies each record. It is an auto-incrementing value. Example: 1, 2, 3, etc.
encid int The encounter ID associated with the MDM chart. This is a foreign key that refers to the `enc` table's `encounterID` column. Example: 101, 102, 103.
selflim_minor int Self-limited or minor medical condition severity level. Example: 0, 1, 2.
stablechronill int Stable chronic illness severity level. Example: 0, 1, 2.
accuteuncompli int Acute uncomplicated illness or injury severity level. Example: 0, 1, 2.
chronwtepse int Chronic illness with exacerbation, progression, or side effects of treatment severity level. Example: 0, 1, 2.
undiagprblm int Undiagnosed new problem with uncertain prognosis severity level. Example: 0, 1, 2.
acutewtsystemic int Acute illness with systemic symptoms severity level. Example: 0, 1, 2.
acutecompliinjury int Acute complicated injury severity level. Example: 0, 1, 2.
chronwithsepse int Chronic illness with severe exacerbation, progression, or side effects of treatment severity level. Example: 0, 1, 2.
acutechrontol int Acute illness or injury posing a threat to life or bodily function severity level. Example: 0, 1, 2.
chartlevel int The overall level of the MDM chart. Example: 0, 1, 2.
addedby int The user ID of the person who added the record. Example: 1001, 1002, 1003.
addeddate datetime The date and time when the record was added. Example: '2021-09-01 12:34:56'.
updatedby int The user ID of the person who last updated the record. Example: 1001, 1002, 1003.
updateddate datetime The date and time when the record was last updated. Example: '2021-09-01 12:34:56'.
deleteflag tinyint This column stores value 0 / 1 indicating this appointment is deleted. This is soft delete. 0 - Not Deleted , 1 -Deleted
