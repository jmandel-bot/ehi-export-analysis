Table inventory_states
Description This table stores the list of states of in the USA.
Column DataType Description
Stateid (Primary Key) int This is the primary key for the table, and it is an integer that uniquely identifies each inventory state. It is an auto-incrementing value. Example: 1, 2, 3, etc.
Abbr varchar This is a variable character field with a maximum length of 5 characters, storing the abbreviation of the inventory state. Example: 'CA', 'NY', 'TX', etc.
StateName varchar This is a variable character field with a maximum length of 255 characters, storing the full name of the US state. Example: 'California', 'New York', 'Texas', etc.
isCredEnabled tinyint This is a tiny integer field that stores whether credit is enabled for the inventory state. It can have a value of 0 (credit not enabled) or 1 (credit enabled). Example: 0, 1.
