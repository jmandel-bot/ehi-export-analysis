Table enc_claimdata
Description This table stores information related to insurance claims for various incidents, including accident details, symptoms, and billing information.
Column DataType Description
encId (Primary Key) int It is reference key from enc table this record associate with patient's appointment
EmpRelated varchar Indicates if the incident is related to employment (varchar). Example: 'Yes' or 'No'
AccInd varchar Accident indicator, specifying if the claim is related to an accident (varchar). Example: 'Yes' or 'No'
AccState varchar State where the accident occurred (varchar). Example: 'CA' or 'NY'
SymInd varchar Symptom indicator, specifying if the claim is related to a symptom (varchar). Example: 'Yes' or 'No'
SymDate datetime Date when the symptom first appeared (date). Example: '2021-08-01'
ExtAccCause varchar External cause of the accident (varchar). Example: 'Car crash' or 'Slip and fall'
AccRespInd char Accident responsibility indicator, specifying if the claimant is responsible for the accident (char). Example: 'Y' or 'N'
AccHour varchar Hour when the accident occurred (varchar). Example: '08:00' or '14:30'
SimSymInd varchar Similar symptom indicator, specifying if the claimant had similar symptoms before (varchar). Example: 'Yes' or 'No'
SimSymDate datetime Date when the similar symptom first appeared (date). Example: '2020-06-15'
UnWorkFromDt datetime Date when the claimant became unable to work due to the incident (date). Example: '2021-08-02'
UnWorkToDt datetime Date when the claimant is expected to return to work (date). Example: '2021-08-15'
AdmDate datetime Date when the claimant was admitted to the hospital (date). Example: '2021-08-01'
DisDate datetime Date when the claimant was discharged from the hospital (date). Example: '2021-08-05'
BillToPatient int Amount billed to the patient (integer). Example: 5000
