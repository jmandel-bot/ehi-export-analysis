EHI Export Schema
Table kiosktask
Description This table stores information about tasks assigned to kiosks, including their unique identifiers, user IDs, device registration keys, encounter IDs, and timestamps for updates and creation.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key of This column .
UID int This column stores user id who associated with kiosk task. It refers to UID column from user table.
deviceRegKey varchar This is a varchar(20) column that stores the device registration key for the kiosk. Example: 'A1B2C3D4E5', 'F6G7H8I9J0', etc.
encounterid int This is an integer that represents the encounter id associated with the task.This column refers to encounterid column from enc table.
updatedTime datetime This is a datetime column that stores the time stamp of when the task was last updated. Example: '2021-09-01 12:34:56', '2021-09-02 14:30:00', etc.
createdateTime datetime This is a datetime column that stores the time stamp of when the task was created. Example: '2021-08-01 10:00:00', '2021-08-02 11:30:00', etc.
deleteFlag tinyint This is a tinyint(1) column that indicates whether the task has been deleted or not. A value of 0 means the task is not deleted, and a value of 1 means the task is deleted. Example: 0, 1.
