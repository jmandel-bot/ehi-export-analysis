Table vision_eligibility_surgery
Description This table stores information about patients' eligibility for vision-related surgeries, their surgery details, and related insurance information.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
eligid int Unique identifier for the patient's eligibility record. Example: 1001, 1002, 1003.
CataractSurgery int Indicates whether the patient has undergone cataract surgery (1) or not (0). Example: 0, 1.
DateofSurgery datetime Date and time when the surgery was performed. Example: '2021-08-15 10:30:00', '2022-01-20 14:45:00'.
eye_surgery varchar Type of eye surgery performed on the patient. Example: 'LASIK', 'PRK', 'ICL'.
ABNForm int Indicates whether an Advance Beneficiary Notice (ABN) form is required (1) or not (0). Example: 0, 1.
certificatenecessary int Indicates whether a certificate is necessary (1) or not (0) for the surgery. Example: 0, 1.
Medicareid varchar Unique identifier for the patient's Medicare insurance. Example: 'M12345678', 'M23456789'.
deleteflag int Indicates whether the record has been deleted (1) or not (0). Example: 0, 1.
autocrossplan int Indicates whether the patient has an auto cross plan (1) or not (0). Example: 0, 1.
nursingfacility int Indicates whether the patient is in a nursing facility (1) or not (0). Example: 0, 1.
firstpair int Indicates whether this is the patient's first pair of glasses after surgery (1) or not (0). Example: 0, 1.
