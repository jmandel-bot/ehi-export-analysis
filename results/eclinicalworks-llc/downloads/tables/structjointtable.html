EHI Export Schema
Table structjointtable
Description This table stores information about the relationships between different items in a structured format, such as encounters, categories, items, and details.
Column DataType Description
encounterid int This column stores the encounter id for which structured examination data have been documented. This column refers to column encounterid from enc table.
catid int This column stores id of a category (Joint Table) which is being documented by end user. This column refers to itemid column in items table.
itemid int This column stores id of an item which is being documented by end user. This column refers to itemid column in items table.
detailid int This column stores id of a symptom being documented by end user. This column refers to id column in structdatadetail table.
value text This column stores value entered by staff or provider to document pain or swelling in patient's joints during examination.
valueid varchar This column is not being used.
notes text This column stores the note added by the end user for the respective structured question.
chartdisplay tinyint This column is not being used.
displayIndex tinyint This column is not being used.
examitemid int This column stores id of a joint being documented by end user during patient's encounter. This column refers to id column in pe_muscular_exams_new table.
