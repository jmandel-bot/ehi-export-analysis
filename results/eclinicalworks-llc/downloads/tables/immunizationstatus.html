EHI Export Schema
Table immunizationstatus
Description This table stores immunization status details. This table stores all the Immunization/Th Injection statuses and codes and refers to the Status dropdown on the Immunization/Th Injection screen.
Column DataType Description
id (Primary Key) int This is an auto increment primary key column of this column.
immstatusid tinyint This column stores immunization status id.
CODE char This column contains a 3-character code associated with the immunization status. It is an optional field and can be NULL. Example: IMM (Immunized), NIM (Not Immunized), PIM (Partially Immunized).
Description varchar This column contains a short description (up to 50 characters) of the immunization status. It is a required field and must have a unique value for each record. Example: Fully Immunized, Not Immunized, Partially Immunized.
TYPE char This column contains a 3-character code representing the type of immunization status. It is an optional field and can be NULL. Example: CHD (Child), ADL (Adult), SNR (Senior).
