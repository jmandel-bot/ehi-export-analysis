Table apptrecurrenceweeklypattern
Description This table stores information about appointment recurrence patterns on a weekly basis, including start and end dates, start and end times, frequency, occurrences, and days of the week.
Column DataType Description
ID (Primary Key) int Unique identifier for each appointment recurrence weekly pattern. Example: 1, 2, 3.
StartDate datetime The start date of the appointment recurrence pattern. Example: '2021-10-01 00:00:00'.
EndDate datetime The end date of the appointment recurrence pattern. Example: '2021-12-31 00:00:00'.
StartTime datetime The start time of the appointment on each recurrence. Example: '09:00:00'.
EndTime datetime The end time of the appointment on each recurrence. Example: '10:00:00'.
Frequency int The number of weeks between each recurrence. Example: 1 (for weekly), 2 (for every two weeks).
Occurences int The total number of occurrences for the appointment recurrence pattern. Example: 10, 20, 30.
Sunday char Indicates whether the appointment recurs on Sundays. 'Y' for yes, 'N' for no. Example: 'Y'.
Monday char Indicates whether the appointment recurs on Mondays. 'Y' for yes, 'N' for no. Example: 'N'.
Tuesday char Indicates whether the appointment recurs on Tuesdays. 'Y' for yes, 'N' for no. Example: 'Y'.
Wednesday char Indicates whether the appointment recurs on Wednesdays. 'Y' for yes, 'N' for no. Example: 'N'.
Thursday char Indicates whether the appointment recurs on Thursdays. 'Y' for yes, 'N' for no. Example: 'Y'.
Friday char Indicates whether the appointment recurs on Fridays. 'Y' for yes, 'N' for no. Example: 'N'.
Saturday char Indicates whether the appointment recurs on Saturdays. 'Y' for yes, 'N' for no. Example: 'Y'.
TotalConflicts int The total number of conflicts with other appointments for this recurrence pattern. Example: 0, 1, 2.
