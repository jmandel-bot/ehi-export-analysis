EHI Export Schema
Table careplan
Description This table stores information about care plans for patients, including details about the plan, its status, and important dates.
Column DataType Description
CarePlanid (Primary Key) int This is auto-incremented primary key in This column . Column stores unique id for each care plan created in the system.
CarePlanName varchar This column stores names for the careplan which are given by users when creating care plans. (Names are generated by merging date and time when the care plan was created by end user. i.e 202004150902 - care plan was created in 2020-04-15 at 09:02. ) - Need to confirm with developer.
CaseType varchar This column stores type of careplan which are generated by end user. Values in this column are coming from Name column in case_types table.
CaseTypeCode varchar This column stores code for corresponding casetype stored in casetype column of This column . Values in this column are coming from Code column in case_types table.
Patientid int This column stores id of a patient for whom care plan has been created by end user. This column refers to UID column in user table.
CreatedBy int This column stores id of a user who created care plan. This column refers to UID column in user table.
CreatedDate datetime This column stores date when the care plan was created by end user. Date is stored in 'YYYY-MM-DD 00:00:00.000' format.
Status int This column indicates status of a careplan for a patient, whether the careplan is open or closed. 0 - Open, 1 - Closed.
deleteFlag int This column stores values 0/1 to indicate whether the care plan has been marked as deleted or not. 0 - Care plan is NOT marked as deleted, 1 - Care plan is marked as deleted. Datatype INT was selected as integer values 0 or 1 was expected in this column.
CarePlanCreatedDate datetime Date and time when the care plan was initially created (e.g., '2021-07-01 10:00:00'). This can be NULL if not available.
CarePlanReviewDate datetime This column stores date and time when Careplan was last reviewed by end user.
CarePlanPtSignOffDate datetime This column stores date and time when patient last signed off on the Careplan.
CarePlanUtilizationReviewDate datetime This column stores date and time when last Careplan was reviewed for utilization.
