Tableip_rh_pntemplate
DescriptionThis table stores the metadata for the templates created for specific progress note.
ColumnDataTypeDescription
id(primaryKey) int Stores the auto increment id for this table
tmplName varchar Stores the template name created
description varchar Stores the description for the template name created
categoryId int Stores the category for the template created referencing to ip_rh_pntemplate_categories table
facId int Stores the faility id for which the template is mapped with.
accessType tinyint Stores the access level for the tempalte. 0 - private template, 1- public template
favourite tinyint Stores flag for template marked as favorite. 0 - not favorite, 1-favourite.
forGender tinyint Stores the for gender flag. 0 - For both, 1 - Male, 2 - Female
ageRangeFrom tinyint Numeric age range from
ageRangeTo tinyint Numeric Age range to
ageIn varchar Stores age unit if Years - YRS, Months - MO
createdBy int User who created this entry in this table, refers to uid from Users table
modiBy int User who modified this entry in this table, refers to uid from Users table
modiTime datetime Time stamp of when this modify action was performed.
deleteFlag tinyint Soft delete flag for this entry, 1 - Deleted, 0 - Not Deleted
