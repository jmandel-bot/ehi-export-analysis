EHI Export Schema
Table ip_rh_eMAR_Vitals_Detail
Description This table is used to store the vital details maintained in Vitals tab of eMAR documentation
Column DataType Description
id (Primary Key) int This is the primary key and is auto incremented.
batchid int This column is used to store the ID of the batch associated with the vital details. It refers to the batch used to update vitals together. It is a foreign key referencing the ip_rh_vitals_batch table.
ipDrugAdminId int This column is used to store the ID of the drug administration. This refers to the same drug administration when vitals were filled.
medOrderId int This column is used to store the ID of the medication order associated with the vital details. It is a foreign key referencing the ip_rh_ptmedicationorders table.
