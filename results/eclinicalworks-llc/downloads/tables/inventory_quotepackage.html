EHI Export Schema
Table inventory_quotepackage
Description This table stores information about inventory quote packages, including package details, facility and anesthesia times, and effective dates.
Column DataType Description
id (Primary Key) int Unique identifier for each inventory quote package (e.g., 1, 2, 3).
PackageName varchar Name of the quote package (e.g., "Basic Package", "Premium Package").
PackageType int Integer representing the type of package (e.g., 1 for Basic, 2 for Premium).
InsProcedure tinyint Tinyint flag indicating if the package includes an insurance procedure (1 for yes, 0 for no).
FacilityTime varchar Time duration for the facility in the package (e.g., "01:30:00" for 1 hour and 30 minutes).
AnesthesiaTime varchar Time duration for anesthesia in the package (e.g., "00:45:00" for 45 minutes).
IsDeleted tinyint Tinyint flag indicating if the package is deleted (1 for yes, 0 for no).
LoginUID int Integer representing the user id of the person who added the package. This is a foreign key that refers to the `id` column in the `users` table.
AddedDateTime datetime time stamp indicating when the package was added to the database (e.g., "2021-08-01 12:34:56").
Facility_id int Integer representing the facility id associated with the package. This is a foreign key that refers to the `id` column in the `facilities` table.
Anesthesia_Facid int Integer representing the anesthesia facility id associated with the package. This is a foreign key that refers to the `id` column in the `anesthesia_facilities` table.
EffectiveFrom datetime Datetime indicating the start date when the package becomes effective (e.g., "2021-08-01 00:00:00").
EffectiveTo datetime Datetime indicating the end date when the package is no longer effective (e.g., "2021-12-31 23:59:59").
