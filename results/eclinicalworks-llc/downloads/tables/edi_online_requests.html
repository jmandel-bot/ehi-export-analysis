EHI Export Schema
Table edi_online_requests
Description This table stores information about electronic data interchange (EDI) insurance eligibility requests, including details about the request, errors, and responses. This table stores logs of transactions for insurance Eligibility.
Column DataType Description
Id (Primary Key) int Unique identifier for each EDI online request (integer).
date datetime This column stores the date of request for Insuarnce Eligibility. The format used is YYYY-MM-DDT HH:MM:SS
NSF text This column stores X12270 format data of the request.
SubFlag int Integer flag (0 or 1) indicating if the request has been submitted (default: 0).
deleteFlag int Integer flag (0 or 1) indicating if the request has been marked for deletion (default: 0).
vmid varchar A string (up to 1024 characters) representing the virtual machine ID associated with the request (default: '').
Description varchar A string (up to 1024 characters) describing the request (default: '').
Amount decimal Decimal value (11 digits, 2 decimal places) representing the amount associated with the request (default: 0.00).
ReceiverID int Integer representing the ID of the receiver for the request (default: 0).
NumClaims int Integer representing the number of claims in the request (default: 0).
BHT03 int This column stores configuration data.
ST02 int This column stores the transaction data related columns
GS06 int This column stores configuration data.
ISA15 char This column stores configuration data
ISA13 int This column stores configuration data
ISA08 varchar This column stores configuration data
ISA06 varchar This column stores configuration data
fileType varchar String (up to 4 characters) representing the file type of the request (default: '').
updateFlag int Integer flag (0 or 1) indicating if the request has been updated (default: 0).
timestamp datetime This column stores date and time when insurance Eligibility was performed. Date and time is stored in 'YYYY-MM-DD hh:mm:ss' format.
InsId int Integer representing the ID of the insurance associated with the request (default: 0).
ProviderId int Integer representing the ID of the provider associated with the request (default: 0).
PatientId int Integer representing the ID of the patient associated with the request (default: 0).
EncId int Integer representing the ID of the encounter associated with the request (default: 0).
FacilityId int Integer representing the ID of the facility associated with the request (default: 0).
Received tinyint Tiny integer flag (0 or 1) indicating if the request has been received (default: 0).
RemoteErr tinyint Tiny integer flag (0 or 1) indicating if there is a remote error in the request (default: 0).
LocalErr tinyint Tiny integer flag (0 or 1) indicating if there is a local error in the request (default: 0).
Response text Long text field containing the response data for the request.
Submitted tinyint Tiny integer flag (0 or 1) indicating if the request has been submitted (default: 0).
ErrorMessage text Long text field containing the error message associated with the request.
InsSeqNo tinyint Tiny integer representing the insurance sequence number associated with the request
RespEligibilityStatus char 1-character string representing the response eligibility status (default: 'V'). E= Eligible , N= Not Eligible,V-Not Verified
InvoiceId int Integer representing the ID of the invoice associated with the request (default: 0).
PCPMatchStatus tinyint Tiny integer flag (0 or 1) indicating the PCP match status for the request (default: 0).
isManualVerified tinyint Tiny integer flag (0 or 1) indicating if the request has been manually verified (default: 0).
