EHI Export Schema
Table ptfeeschlist
Description This table stores fee schedule and related information which acts as master data for that fee schedule to all the other tables storing various information related to the fee schedule.
Column DataType Description
PtFeeSchId (Primary Key) int This column stores the Patient Fee schedule ID
name varchar Fee Schedule name
description text Fee schedule description
MasterFeeSch int Master fee schedule id , this id is reference feeschlist.masterfeesch
AddedByUser int Job created User Id
AddedDateTime datetime Job created Date and Time
LastModifiedByUser int Last modified user id
LastModifiedDateTime datetime Last modified Date Time
timestamp datetime Current Time stamp for add/update job
deleteFlag int Delete Flag (0,1), 0 - default 1- deleted
vmid varchar Unique Id for each job
AutoAdjFlag tinyint It stores 0 and 1. If itemkey 'EnableAutoAdjustFeeSchedule' is enable then only it will show 'Enable Adjustment' checkbox under 'Setup auto claim adjustment' section in Fee Schedule record.
AdjCode varchar This column stores the Keeping Adjustment code
selfPay tinyint Self pay (0,1) default 0
custom_desc varchar Custom description for the patient fee schedule. Example: "Custom Plan A", "Custom Plan B".
show_desc tinyint Flag indicating if the description should be shown (1) or hidden (0) for the patient fee schedule. Example: 0, 1.
