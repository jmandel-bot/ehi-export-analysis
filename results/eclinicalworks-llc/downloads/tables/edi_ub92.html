Table edi_ub92
Description This table stores information related to electronic data interchange (EDI) UB-92 forms, which are used for billing purposes in the healthcare industry. The table contains various details about the invoice, billing types, admission and discharge details, condition codes, occurrence codes, value codes, and other relevant information.
