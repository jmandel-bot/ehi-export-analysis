Table prisma_patient_address
Description This table stores the address information of patients from document received from different external sources
Column DataType Description
ID (Primary Key) int Unique identifier for each address record. Example: 1, 2, 3, etc.
PATIENT_DATA_ID bigint The identifier of the patient associated with this address. This is a foreign key that refers to the `id` column in the `prisma_patient_data` table. Example: 101, 102, 103, etc.
ADDRESS_LINE1 varchar The first line of the patient's address. Example: "123 Main St."
ADDRESS_LINE2 varchar The second line of the patient's address (optional). Example: "Apt 4B"
ADDRESS_LINE3 varchar The third line of the patient's address (optional). Example: "Building 7"
CITY varchar The city of the patient's address. Example: "New York"
ADDRESS_STATE varchar The state of the patient's address. Example: "NY"
POSTAL_CODE varchar The postal code of the patient's address. Example: "10001"
COUNTRY varchar The country of the patient's address. Example: "United States"
POSTAL_ADDRESS_TYPE varchar The type of postal address, such as "home" or "work". Example: "home"
ADDRESS_START_DATE datetime The start date of the patient's address record. Example: "2021-01-01 00:00:00"
ADDRESS_END_DATE datetime The end date of the patient's address record (optional). Example: "2021-12-31 00:00:00"
