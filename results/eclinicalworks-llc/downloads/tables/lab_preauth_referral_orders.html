EHI Export Schema
Table lab_preauth_referral_orders
Description This table stores information about lab pre-authorization referral orders, including their relationships with referrals, reports, and users who created them.
Column DataType Description
id (Primary Key) int This is Primary Key and store numeric value.
referralid int This column stores referralid from referral table. 
reportid int This column stores reportid from labdata table. 
deleteFlag tinyint This stores 0 and 1 to indicate a row is deleted or not where 0 means not deleted and 1 means deleted.
createdBy int This column stores UID from user table. 
createdDate datetime This column stores created date time.
