EHI Export Schema
Table empiptdatalog
Description This table contains logging of patient's eEHX consent. Schema wise, it is a replica of empiptdata tables.
Column DataType Description
id (Primary Key) int This column represent s an auto incremented id of the row.
pid int This column represents which patient this consent log belongs to. It refers to the user.UID column.
ecwmpi varchar This column represents eEHX enabled patient's ecwmpi id. This is gUID value.
timestamp datetime This column represents the oldest date and time consent was modified.
EnableEmpi int This column represents a patient's old empi enabled status before the row was updated in the empiptdata table.
eCROptInOption varchar This column represents a patient's old opt-in status before the row was updated in the empiptdata table.
ReceivedConsent int This column represents a patient's old Consent Received check box value before the row was updated in the empiptdata table.
DateSigned datetime This column represents a patient's old Date Signed before the row was updated in the empiptdata table.
delFlag int This column represents the rows deletition status. A "0" means the row is not deleted, and a "1" means the row is deleted.
Facid varchar This is a dead column. It is not used by the system, and the system doesn't populate the column value.
MatchFound int This is a dead column. It is not used by the system, and the system doesn't populate the column value.
optinpracticecode varchar This column represents a patient's old practice code before the row was updated in the empiptdata table.
OptinFacilityid int This column represents a patient's old optin facility id before the row was updated in the empiptdata table.
OptinFacilityName varchar This column represents a patient's old optin facility name before the row was updated in the empiptdata table.
FacGroupid int This is a dead column. It is not used by the system, and the system doesn't populate the column value.
ConsentSignedByGuarantor tinyint This is a dead column. It is not used by the system, and the system doesn't populate the column value.
IsBeneficiary int This column represents a patient's old beneficiary flag value before the row was updated in the empiptdata table.
