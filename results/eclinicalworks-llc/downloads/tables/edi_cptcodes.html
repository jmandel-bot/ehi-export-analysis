EHI Export Schema
Table edi_cptcodes
Description This table contains information regarding CPT codes for labs along with fee and cost.
Column DataType Description
Id (Primary Key) int This is an auto incremental primary key of This column .
ItemId int This column stores the Item Id.
ShortDesc varchar This column stores the short Description of CPT codes.
MedDesc varchar This column stores the medium description of CPT codes.
LongDesc text This column stores the long description of CPT codes.
Fee decimal This column stores the Fee of decimal type.
Cost decimal This column stores the cost of decimal type.
pos varchar This column stores place of service codes of CPT code.
TOS varchar This column stores Type of Service codes for CPT Code.
BillToPt smallint Indicates whether the service should be billed to the patient (1) or not (0).
PtPortion decimal The portion of the fee that the patient is responsible for paying.
AllowedFee decimal The maximum fee allowed by insurance for the service associated with the CPT code.
ReqCliaId int This column stores Integer value indicate is this CPT Code require Clia Id or not. 1 indicate it is require clia id 0 indicate not require clia Id
ReqMammographyId int This column value stores integer value which indicates Requires Mammography Certification Number. 1 value indicate Requires Mammography Certification Number 0 value indicate not Requires Mammography Certification Number
Mod1 varchar This column value stores modifier 1. It indicates mapping of modifier 1 with CPT Code.
Mod2 varchar This column value stores modifier 2. It indicates mapping of modifier 2 with CPT Code.
Mod3 varchar This column value stores modifier 3. It indicates mapping of modifier 3 with CPT Code.
HPSAFlag int This column stores integer value indicates Qualifies for HPSA Incentive. 1 value indicates Qualifies for HPSA Incentive 0 value indicates not qualifies for HPSA Incentive
ExcludeCPT int Not Available
revcode varchar This column indicates revenue code.
NDCCode varchar This column stores NDC Code of CPT Code.
GlobalBillingDays int This column stores integer value indicates Global Billing Days
ReqOrdPrInfo tinyint Not Available
bActive char This column indicates CPT Code is Active or Inactive. Y indicates Active N indicates Inactive
minunits int This column stores integer value and indicates minimum unit of CPT Code
maxunits int This column stores integer value and indicates maximum unit of CPT Code
chargecode varchar This column stores the charge code for CPT.
Mod4 varchar This column value stores modifier 4. It indicates mapping of modifier 4 with CPT Code.
AnesthesiaBaseUnitValue int This column stores integer value indicate Anesthesia Base Units.
CPSPMaxUnitsAllowed int This column stores integer value indicates CPSP maximum units allowed.
CPSPMinsPerUnit int This column stores integer value indicates CPSP minutes per unit.
Instruction varchar Not Available
ServiceIdQualifier varchar This column stores string value indicates Service Id Qualifier. If it is blank then default set as 'HC'
NDCUnitPrice decimal This column stores decimal value indicates NDC Unit Price
NDCUom char This column stores string value indicates Unit Standard designates units of measure to describe all drugs.
NDCUnits varchar This column indicates NDC units
documentMinutes tinyint This column stores integer value indicates document minutes 0 indicates CPT code not set for document minutes 1 indicates CPT code set for document minutes
timedCode tinyint This column stores integer value indicates timed code 0 indicates CPT code not set for timed code 1 indicates CPT code set for timed code
feeclaimedit tinyint This column indicates Suppress Claim Edit that warns if Billed fee is < = $0
nocdescription varchar This column indicates Non-specific Procedure (NOC) Code Description
servicetype int This column indicates CPT Service Type. 0 indicates no service type map 1 indicates global service type map 2 indicates professional service type map 3 indicates facility service type map
HCPCSCodeRange varchar This column indicates condition of HCPCS Code. Possible values 1) Equal To 2) Up To
HCPCSCodeType varchar This column indicates measures of HCPCS code dosage type.
HCPCSCodeUnit decimal This column indicates unit of HCPCS code
additionalinfo varchar This column indicates Additional information of CPT Code
