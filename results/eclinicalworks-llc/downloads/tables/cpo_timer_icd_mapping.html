EHI Export Schema
Table cpo_timer_icd_mapping
Description This table stores information regarding timer details added for cpo certification including ICD codes selected, cpo services selected, time added for each cpo service.
Column DataType Description
id (Primary Key) int Contains auto incremented id, datatype is INT (11). This column stores integer value which could grow with number of transactions over the years hence integer(Range: 0-2,147,483,647) is selected. It is AUTO_INCREMENT column and has PRIMARY KEY index.
Timerid int Contains unique id of phm timer table. This field references to id column of ccmr_ccm_timer table. This field cannot have a NULL value.
itemid varchar This column stores the Primary Key of items table where the ICD Codes are stored. The value of this field cannot be NULL. This field references to itemid column of Items table. The data type chosen is VARCHAR(500).
icdCode varchar Contains the international classification of disease code and datatype is VARCHAR(500). This field will always have a value hence defined as NOT NULL.
cpo_service_id int Contains unique id of cpo services. This field references id column of cpo_services table. The data type is INT. This field cannot have a NULL value.
cpo_service_time int Contains number of minutes allocated to specific cpo service. The data type is INT and this field cannot have a NULL value.
ptacoid int Contains Primary Key column of ptacoprograms table to store PTACO id . The datatype chosen is an INT The value of this field cannot be NULL.
delflag tinyint Flag indicates if this data is deleted or not, datatype is TINYINT. This column stores binary flag i.e 0/1. 0: not deleted 1: deleted - By default the record must not be in deleted status hence default 0 is chosen.
createdBy int Contains the user id which is referenced by UID column of user table for tracking the record added by which user and the datatype is INT. This column stores high range numeric value hence integer is selected and This field will always have a value hence defined as NOT NULL. also, it has default value is 0.
createdDtTime datetime This column stores the date time when record is added its datatype is DATETIME. This column stores datetime hence Datetime is selected. This field can have a NULL value.
modifiedDtTime datetime This column stores the date time when record is modified last . its datatype is DATETIME. This column stores datetime hence Datetime is selected. This field can have a NULL value.
