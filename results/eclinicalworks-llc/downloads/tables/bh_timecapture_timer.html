EHI Export Schema
Table bh_timecapture_timer
Description This table stores information about time capture timers related to encounters, including start and end times, duration, notes, and user details.
Column DataType Description
id (Primary Key) int Unique identifier for each time capture timer record. (Example: 1, 2, 3)
Encounterid int identifier for the encounter associated with the time capture timer. Foreign key relation with `enc` table's `encounterid` column. (Example: 1001, 1002, 1003)
UserType int identifier for the user type associated with the time capture timer. Foreign key relation with `bh_cptusertype` table's `id` column. (Example: 1, 2, 3)
StartTimer datetime Start time of the time capture timer. (Example: '2022-01-01 12:00:00', '2022-01-02 14:30:00')
EndTimer datetime End time of the time capture timer. (Example: '2022-01-01 12:30:00', '2022-01-02 15:00:00')
duration datetime Duration of the time capture timer. (Example: '00:30:00', '00:45:00')
Notes varchar Notes related to the time capture timer. (Example: 'Initial consultation', 'Follow-up appointment')
CreatedBy int identifier for the user who created the time capture timer record. Foreign key relation with `users` table's `UID` column. (Example: 101, 102, 103)
CreatedDateTime datetime Date and time when the time capture timer record was created. (Example: '2022-01-01 11:59:00', '2022-01-02 14:29:00')
ModifiedBy int identifier for the user who last modified the time capture timer record. Foreign key relation with `users` table's `UID` column. (Example: 101, 102, 103)
ModifiedDateTime datetime Date and time when the time capture timer record was last modified. (Example: '2022-01-01 12:01:00', '2022-01-02 14:31:00')
CapturedBy int identifier for the user who captured the time. Foreign key relation with `users` table's `UID` column. (Example: 101, 102, 103)
RecordedManually tinyint Flag indicating whether the time capture timer was recorded manually (1) or not (0). (Example: 0, 1)
TimerStatus tinyint Status of the time capture timer, where 0 is inactive and 1 is active. (Example: 0, 1)
IsResetTimer tinyint Flag indicating whether the time capture timer has been reset (1) or not (0). (Example: 0, 1)
IsCptAssigned tinyint Flag indicating whether a CPT code has been assigned to the time capture timer (1) or not (0). (Example: 0, 1)
DeleteFlag tinyint Flag indicating whether the time capture timer record has been deleted (1) or not (0). (Example: 0, 1)
