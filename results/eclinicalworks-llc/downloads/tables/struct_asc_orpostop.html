EHI Export Schema
Table struct_asc_orpostop
Description This table used for storing structured type questions for postoperative details.
Column DataType Description
encounterid int used for storing patient's encounter id from enc master table.
catid int used for storing category id,which maps to itemid from items table.
itemid int used for storing item id,which maps data from properties table with propid.
detailid int used to store detailid ,which maps data from structdatadetail with id
value text This column stores value of an answer to the structured question documented by end user. This column refers to column name from structdataoptions table.
valueid varchar used for storing value id,which maps data from structdataoptions table with optionid.
notes text A text field containing any additional notes related to the detail. Example: "Patient recovered well after surgery."
displayIndex tinyint A tinyint(1) field representing the display order of the detail within the item. Example: 1
chartdisplay tinyint Its value will be always 1, as we are not using this field. This column is not in use anymore.
