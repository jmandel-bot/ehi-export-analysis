EHI Export Schema
Table oa_encounters
Description This table stores information about encounters created through healow, including healow id,their payment status and the time they were last updated.
Column DataType Description
id (Primary Key) int This is auto-incremented primary key in This column .
encounterid int This column stores an encounter id when appointments are created from healow through openaccess. This value refers to the encounterid column of the enc table.
healow_uid int This column stores the Healow unique identifier of a patient who is enrolled for Open Access Campaign. This is an int(11) type. And the default value is 0. The actual value is sent from healow and there are no other references of this value in the EMR DB
payment_status varchar This is an enumeration column that stores the payment status of the encounter. It can have one of the following values: 'N/A', 'Pending', 'Success', or 'Failed'.
updated_ts_utc datetime This is a datetime column that stores the time stamp of when the encounter payment status was last updated in Coordinated Universal Time (UTC). Example: '2022-01-01 12:34:56'
