Table prisma_pot_encounter_problem
Description This table stores information about problems encountered during a patient's visit, including details about the problem type, problem code, author, and reason.
Column DataType Description
id (Primary Key) bigint A unique identifier for each record in the table. Example: 1, 2, 3.
encounterDetailId bigint A foreign key that refers to the `id` column in the `prisma_pot_encounter_details` table. It represents the encounter detail associated with the problem. Example: 101, 102, 103.
problemTypeCode varchar A short code representing the type of problem. Example: "PTC1", "PTC2", "PTC3".
problemType varchar A more descriptive name for the problem type. Example: "Medical", "Surgical", "Dental".
problemCode varchar A short code representing the specific problem.
problemname varchar A more descriptive name for the specific problem. Example: "Headache", "Toothache", "Fracture".
problemText varchar A textual description of the problem. Example: "Severe headache", "Mild toothache", "Fractured arm".
authorId varchar A unique identifier for the author who recorded the problem. Example: "A1", "A2", "A3".
authorName varchar The full name of the author who recorded the problem. Example: "Dr. John Smith", "Dr. Jane Doe", "Dr. Alice Johnson".
reasonTypeCode varchar A short code representing the reason for the problem.
reasonType varchar A more descriptive name for the reason type. Example: "Allergy", "Injury", "Infection".
reasonCode varchar A short code representing the specific reason.
reasonName varchar A more descriptive name for the specific reason. Example: "Peanut allergy", "Car accident", "Bacterial infection".
