Table ccmrproblemreview
Description This table stores information related to patient care plan review.
Column DataType Description
id (Primary Key) int Unique identifier for each problem review record. (Example: 1, 2, 3)
ccmrid int identifier for the Chronic Care Management record associated with the problem review. (Example: 101, 102, 103)
Patientlabel varchar Label for the patient associated with the problem review. (Example: "John Doe", "Jane Smith")
Providerlabel varchar Label for the provider associated with the problem review. (Example: "Dr. Brown", "Dr. Johnson")
notes text Text notes related to the problem review. (Example: "Patient needs follow-up appointment.")
reviewType int Integer representing the type of review (0 or 1). (Example: 0)
reviewDate datetime Date and time when the review was conducted. (Example: '2021-09-01 10:30:00')
deleteFlag int Integer indicating whether the record is deleted (1) or not (0). (Example: 0)
userid int identifier for the user who created the problem review. (Example: 1001, 1002, 1003)
patientid int identifier for the patient associated with the problem review. (Example: 2001, 2002, 2003)
Programid int identifier for the Chronic Care Management program associated with the problem review. (Example: 3001, 3002, 3003)
isCompleted int Integer indicating whether the review is completed (1) or not (0). (Example: 1)
completedBy int identifier for the user who completed the problem review. (Example: 1001, 1002, 1003)
completedOn datetime Date and time when the review was completed. (Example: '2021-09-01 11:30:00')
ismigrated tinyint Tiny integer indicating whether the record is migrated (1) or not (0). (Example: 0)
problemDetails varbinary Binary data (longblob) containing details of the problem.
adhocReview tinyint Tiny integer indicating whether the review is ad hoc (1) or not (0). (Example: 1)
isNewCpr tinyint Tiny integer indicating whether the review is a new CPR (1) or not (0). (Example: 0)
patientsign varbinary Binary data (longblob) containing the patient's signature.
providerSign varbinary Binary data (longblob) containing the provider's signature.
isMerged tinyint Tiny integer indicating whether the record is merged (1) or not (0). (Example: 0)
newProblemDetails varbinary Binary data (longblob) containing details of the new problem.
ptAcoid int identifier for the ACO program associated with the patient. Foreign key relation to `ptacoprograms` table's `id` column. (Example: 4001, 4002, 4003)
isEnded tinyint Tiny integer indicating whether the review is ended (1) or not (0). (Example: 1)
reviewTypeNew tinyint Tiny integer representing the new review type (0, 1, 2, or 3). (Example: 2)
lockedData varchar Text data containing locked information related to the problem review. (Example: "Locked due to privacy concerns.")
