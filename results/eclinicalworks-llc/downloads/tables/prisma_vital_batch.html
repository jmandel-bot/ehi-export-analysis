Table prisma_vital_batch
Description This table stores information about vital batches, including details about the author and the associated section.
Column DataType Description
id (Primary Key) bigint A unique identifier for each record in the table. It is a bigint(20) data type and is auto-incremented. For example, 1, 2, 3, etc.
sectionId bigint A foreign key that refers to the `id` column in the `prisma_ccd_document_section` table. It is a bigint(20) data type and is used to associate the vital batch with a specific section. For example, 101, 102, 103, etc.
batchId varchar A unique identifier for each batch. It is a varchar(200) data type and can store alphanumeric values. For example, 'batch_001', 'batch_002', 'batch_003', etc.
batchDate varchar The date associated with the batch. It is a varchar(50) data type and can store date values as strings. For example, '2021-09-01', '2021-09-02', '2021-09-03', etc.
authorId varchar A unique identifier for the author of the batch. It is a varchar(50) data type and can store alphanumeric values. For example, 'author_001', 'author_002', 'author_003', etc.
authorName varchar The full name of the author. It is a varchar(100) data type and can store text values. For example, 'John Doe', 'Jane Smith', 'Alice Johnson', etc.
authorAddressLine1 varchar The first line of the author's address. It is a varchar(100) data type and can store text values. For example, '123 Main St', '456 Elm St', '789 Oak St', etc.
authorAddressLine2 varchar The second line of the author's address, if applicable. It is a varchar(100) data type and can store text values. For example, 'Apt 4B', 'Suite 300', 'Building 2', etc.
authorCity varchar The city where the author is located. It is a varchar(25) data type and can store text values. For example, 'New York', 'Los Angeles', 'Chicago', etc.
authorState varchar The state where the author is located. It is a varchar(10) data type and can store text values. For example, 'NY', 'CA', 'IL', etc.
authorPostalCode varchar The postal code of the author's address. It is a varchar(15) data type and can store alphanumeric values. For example, '10001', '90001', '60601', etc.
authorCountry varchar The country where the author is located. It is a varchar(10) data type and can store text values. For example, 'US', 'CA', 'UK', etc.
