EHI Export Schema
Table bulk_resources_file_details
Description This table stores information about the files containing Fhir R4 bulk resources, including their names, record counts, and statuses.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
bulk_batch_master_id int Identifier for the associated bulk batch master record. This is a foreign key that refers to the `id` column in the `bulk_batch_master` table. Example: 101, 102, 103.
fhir_resources_id tinyint Identifier for the associated FHIR resource. Example: 1, 2, 3.
file_name varchar Name of the file containing the bulk resources.
record_count int Number of records in the file. Default value is '0'. Example: 100, 200, 300.
resource_file_status tinyint Status of the resource file. Default value is '0'. Example: 0 (Open), 1 (Closed), 2 (Uploaded), 3 (Failed).
