Table bh_intervention_frequency
Description This table stores information about different intervention frequencies, including their names, statuses, display indexes, and deletion flags.
Column DataType Description
frequencyid (Primary Key) int This is the primary key and an auto-incrementing integer that uniquely identifies each intervention frequency. Example: 1, 2, 3, etc.
frequencyName varchar This is a varchar(2000) column that stores the name of the intervention frequency. Example: "Daily", "Weekly", "Monthly", etc.
frequencyStatus tinyint This is a tinyint(1) column that stores the status of the intervention frequency. A value of '0' indicates inactive, and '1' indicates active. Example: 0 or 1.
displayIndex int This is an int(11) column that stores the display order of the intervention frequencies. Lower values will be displayed first. Example: 0, 1, 2, etc.
deleteFlag tinyint This is a tinyint(1) column that stores the deletion status of the intervention frequency. A value of '0' indicates not deleted, and '1' indicates deleted. Example: 0 or 1.
