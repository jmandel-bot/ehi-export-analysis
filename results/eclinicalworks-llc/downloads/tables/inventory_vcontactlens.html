Table inventory_vcontactlens
Description This table stores information about various contact lenses in the inventory, including their specifications, pricing, and availability.
Column DataType Description
id (Primary Key) int Unique identifier for each contact lens entry in the table. (e.g., 1, 2, 3)
Product_id int identifier for the product, possibly referring to a product in another table. (e.g., 101, 102, 103)
UPC varchar Universal Product Code for the contact lens. (e.g., '123456789012', '234567890123')
BaseCurve varchar Base curve value of the contact lens. (e.g., '8.6', '8.7', '8.8')
Diameter varchar Diameter value of the contact lens. (e.g., '14.0', '14.2', '14.5')
Cyl varchar Cylinder value for the contact lens, related to astigmatism correction. (e.g., '-0.75', '-1.25', '-1.75')
Axis varchar Axis value for the contact lens, related to astigmatism correction. (e.g., '10', '20', '30')
Colour int identifier for the color of the contact lens, possibly referring to a color in another table. (e.g., 1, 2, 3)
ReadingAdd varchar Reading addition value for the contact lens, related to presbyopia correction. (e.g., '+1.00', '+1.50', '+2.00')
AddType varchar Type of addition for the contact lens, such as bifocal or multifocal. (e.g., 'Bifocal', 'Multifocal')
isdeleted tinyint Flag to indicate if the contact lens entry has been deleted (1) or not (0). (e.g., 0, 1)
createdby int identifier for the user who created the contact lens entry. (e.g., 9095, 9096, 9097)
createdon datetime time stamp for when the contact lens entry was created. (e.g., '2021-08-01 12:34:56')
updatedby int identifier for the user who last updated the contact lens entry. (e.g., 9095, 9096, 9097)
updatedon datetime time stamp for when the contact lens entry was last updated. (e.g., '2021-08-02 14:56:23')
sku int Stock Keeping Unit identifier for the contact lens. (e.g., 1001, 1002, 1003)
lensid varchar Unique identifier for the contact lens, possibly used by the manufacturer. (e.g., 'LEN123', 'LEN124', 'LEN125')
description varchar Description of the contact lens. (e.g., 'Daily disposable contact lens', 'Monthly disposable contact lens')
lensconvert varchar Conversion value for the contact lens, possibly related to prescription conversion. (e.g., '1.00', '1.50', '2.00')
POWER int Power value for the contact lens, related to vision correction. (e.g., 0, -1, -2)
productFamily varchar Name of the product family the contact lens belongs to. (e.g., 'Acuvue Oasys', 'Biofinity', 'Air Optix')
lprice decimal Price of the contact lens. (e.g., '10.0000', '15.0000', '20.0000')
CLSphere varchar Sphere value for the contact lens, related to vision correction. (e.g., '-1.00', '-1.50', '-2.00')
