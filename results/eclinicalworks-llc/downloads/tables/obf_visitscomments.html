EHI Export Schema
Table obf_visitscomments
Description This table stores information about visits and comments related to a specific pregnancy case.
Column DataType Description
Visitsid (Primary Key) int This column stores primary key of the table as numeric value, This column stored  Notes section data based on Pregnancy id
Pregid int identifier for the pregnancy case associated with the visit/comment. This is a foreign key referring to the primary key of the related pregnancy table. Example: 101, 102, 103.
Notes text This column stores notes data as varchar value.
CommentDate datetime This column stores date and time of documented notes
Lock_Ind int This column stores 0 or 1 value as numeric (if value is 1 then notes is lock , if 0 then notes is unlock)
DelStat int This column stores 0 value as numeric
Uname varchar This column stores uname of logged user name.
Upd_Date datetime This column stores date and time of updated documented notes
Initials varchar This column stored initital value of notes section as varchar format
userid int This column stores user id as numeric value from user table
