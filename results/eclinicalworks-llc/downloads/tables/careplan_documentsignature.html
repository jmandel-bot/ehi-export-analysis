Table careplan_documentsignature
Description This table stores information about document signatures related to care plans, including the file name of the signature, the time stamp, and the device used for signing.
Column DataType Description
id (Primary Key) int Unique identifier for each document signature entry. (Example: 1, 2, 3)
DocumentReviewid int identifier for the document review associated with the signature. This is a foreign key that refers to the `id` column in another table (e.g., `document_review` table). (Example: 101, 102, 103)
SignatureFileName varchar The file name of the signature image. (Example: 'signature_123.png', 'signature_456.jpg')
timestamp datetime The date and time when the signature was created. (Example: '2021-09-15 14:30:00', '2021-09-16 10:15:00')
PrintLabel varchar An optional label for the signature, which can be used for printing purposes. (Example: 'Dr. John Smith', 'Nurse Jane Doe')
Device tinyint A boolean value (0 or 1) indicating whether the signature was created using a device (1) or not (0). (Example: 0, 1)
