EHI Export Schema
Table formcompletionlogs
Description This table stores information about form completion logs, including details about the user, encounter, policy, survey, and message types.
Column DataType Description
id (Primary Key) int This is an auto increment id and primary key of table.
initialNotificationid int This is id of message sent to a patient
UID int The identifier for the user who completed the form. Example: 1001, 1002, 1003.
encounterid int This is an encounter id for which survey has been sent to a patient.
apUID int This column stores apUID
policyid int This is a id of campaign thru which Survey has been sent to patient.
surveyid int id of survey which patient has responded to.
surveyCompletionStatus varchar The completion status of the survey. It can be either 'Completed' or 'Expired'.
messageType varchar The type of message sent to the user. It can be 'Initial', 'First', or 'Second'.
messageSentDate_UTC datetime The date and time when the message was sent to the user, in UTC format. Example: '2021-09-01 12:34:56'.
SurveyResponseLastModifiedOrCompletedOn_UTC datetime The date and time when the survey response was last modified or completed, in UTC format. Example: '2021-09-02 14:56:32'.
inserttime datetime The date and time when the form completion log entry was inserted into the table, in UTC format. This value is automatically updated when the record is modified. Example: '2021-09-03 10:23:45'.
