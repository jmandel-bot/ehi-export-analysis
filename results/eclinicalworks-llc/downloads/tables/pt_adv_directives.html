EHI Export Schema
Table pt_adv_directives
Description This table stores patient advanced directives information, which are legal documents that allow patients to convey their decisions about end-of-life care ahead of time.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key of This column .
Ptid int This column stores id of the patient for which special directive is added. This column refers to column UID from user table.
Code varchar This column stores code of the patient’s advance directive.
Name varchar This column stores name of the patient’s advance directive.
MDate varchar This column stores the created date of the advance directive of the patient.
delflag tinyint This column stores value 0 / 1 indicating if the patient’s advance directive is deleted, this is soft delete. 0 - Not Deleted , 1 -Deleted
displayindex tinyint This column stores a digit to indicate order in which patient's advance directive should be listed in user interface.
