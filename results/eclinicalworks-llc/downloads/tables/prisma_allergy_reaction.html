Table prisma_allergy_reaction
Description This table stores information about allergy reactions, their severity, and related details.
Column DataType Description
id (Primary Key) bigint Unique identifier for each allergy reaction record. (e.g., 1, 2, 3)
allergyId bigint Identifier for the allergy associated with the reaction. This is a foreign key that refers to the `id` column in the `prisma_allergy_details` table. (e.g., 101, 102, 103)
reactionCode varchar Code representing the type of reaction. (e.g., "R001", "R002", "R003")
reactionName varchar Name of the reaction. (e.g., "Hives", "Swelling", "Rash")
reactionText varchar Description of the reaction. (e.g., "Red, itchy hives", "Swelling of the face and lips", "Red, itchy rash")
severityCode varchar Code representing the severity of the reaction. (e.g., "S001", "S002", "S003")
severityname varchar Name of the severity level. (e.g., "Mild", "Moderate", "Severe")
severityText varchar Description of the severity level. (e.g., "Mild discomfort", "Moderate pain and discomfort", "Severe pain and potential risk")
sourcePriority int Priority of the source providing the allergy reaction information. (e.g., 0, 1, 2)
codeSystemPriority int Priority of the code system used for the reaction and severity codes. (e.g., 0, 1, 2)
duplicatedBy int Identifier for the record that duplicates this record. A value of -1 indicates no duplication. (e.g., -1, 5, 10)
allergen varchar Name or description of the allergen causing the reaction. (e.g., "Peanuts", "Shellfish", "Pollen")
isException tinyint Flag indicating if there is an exception in the record. 0 means no exception, and 1 means there is an exception. (e.g., 0, 1)
exceptionMessage varchar Message describing the exception, if any. Empty if there is no exception.
