EHI Export Schema
Table dischargemeetingattendees
Description This table stores information about the attendees of discharge meetings, including their user IDs and the meetings they are attending.
Column DataType Description
id (Primary Key) int This is primary key column.
dischargePlanId int This column store the id of discharge plan. It's value is coming from dischargeplan_corrections.id column.
meetingId int This column store the id of discharge plan meeting. It's value is coming from dischargemeetingplan.id column.
userId int This column store the user id of user who are added to discharge plan meeting.
deleteflag tinyint This column store the delete value discharge plan meeting.
