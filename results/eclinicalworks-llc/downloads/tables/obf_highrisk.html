Table obf_highrisk
Description This table stores information about high-risk pregnancies and their associated items.
Column DataType Description
HighRiskid (Primary Key) int This column stored primary key of the Table, This column stored Risks section data for High Risk based on Pregnancy id
Pregid int This column stored Pregnancy id of the obf_pregnancy table.
Itemid int This column stored number value as Item id of the items table.
Choice varchar This column stored varchar value of the items documented.
Lock_Ind int This column stored numeric value as 0 or 1 ( if value is 0 respective section is lock , if 1 then section is lock)
DelStat int This column stored numeric value as 0 or 1
Uname varchar This column stored vachar value as logged in uname of the user
Upd_Date datetime This column stored vachar value as date and time at the time of documented
