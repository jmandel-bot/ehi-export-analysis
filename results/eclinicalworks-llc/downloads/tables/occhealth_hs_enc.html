EHI Export Schema
Table occhealth_hs_enc
Description This table stores information about occupational health surveillance encounters, including details about the employer, surveillance, results, and suggested screenings.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
encounterid int identifier for the encounter associated with the record. This column referr to enc table's encounter id column. Example: 101, 102, 103.
employerid int identifier for the employer associated with the encounter. This column refers to insurance table's insid column. Example: 1001, 1002, 1003.
surveillanceid int identifier for the surveillance associated with the encounter. This column refers to occhealth_healthsurveillance table's hsid column. Example: 2001, 2002, 2003.
resultid int identifier for the result associated with the encounter. This column referrs to occhealth_hs_result_names table's id column. Example: 3001, 3002, 3003.
nextSuggScreeningid int identifier for the next suggested screening associated with the encounter. This column refers to occhealth_hs_screenings table's screening id column. Example: 4001, 4002, 4003.
nextSuggDueDate datetime Date and time when the next suggested screening is due. Example: '2022-01-01 00:00:00', '2022-02-01 00:00:00'.
hsGroupid int identifier for the health surveillance group id associated with the encounter. Example: 5001, 5002, 5003.
actionDate datetime Date and time when an action was taken for the encounter. Example: '2021-12-01 00:00:00', '2021-12-02 00:00:00'.
resultBy int identifier for the user who recorded the result for the encounter. This may be a foreign key referring to another table's user id column. Example: 6001, 6002, 6003.
screeningid int identifier for the screening associated with the encounter. This column refer to occhealth_hs_screenings table's screening id column. Example: 7001, 7002, 7003.
delFlag tinyint A flag indicating whether the record has been deleted (1) or not (0). Example: 0, 1.
nextvisitdue tinyint A flag indicating whether the next visit is due (1) or not (0). Example: 0, 1.
