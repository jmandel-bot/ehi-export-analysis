EHI Export Schema
Table aftercarestatushistory
Description This table stores aftercare history status data for the encounter.
Column DataType Description
statusHistoryid (Primary Key) int This column is an auto incremented primary key in this column. Unique status history id is assigned to each record when they are created using the user interface.
encounterid int This column stores the id of an encounter for which aftercare status history related to the patient's visit is documented. This column refers to the column encounterid from the enc table.
code varchar This column stores the aftercare history status code. This column refers to the column code from the aftercare status codes table.
date datetime This column stores the aftercare history status created date.
time datetime This column stores the aftercare history status created time.
userid int This column stores identifier number of the user who has created the aftercare status history. It refers to the column UID from the user table.
substatuscode varchar This column stores the aftercare history sub status code. This column refers to the column code from the aftercare sub status codes table.
