EHI Export Schema
Table asc_ioorder_detail
Description This table is used to store intake/output order details associated to an encounter.
Column DataType Description
detailid (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
intakeOutputid int This is a intakeOutputid column of asc_ioorders table.
routeid int This column stores route id, primary key of ASC_IORoute_List table.
givenAmount varchar The amount of intake or output given to the patient. Example: 500ml, 2L, 1.5L.
givenRate varchar The rate at which the intake or output was given. Example: 100ml/h, 50ml/h, 200ml/h.
startdateTime datetime The date and time when the intake or output started. Example: 2021-06-01 10:30:00, 2021-06-02 14:00:00.
enddateTime datetime The date and time when the intake or output ended. Example: 2021-06-01 12:30:00, 2021-06-02 16:00:00.
givenByid int identifier for the staff member who gave the intake or output. Example: 1001, 1002, 1003.
givenDateTime datetime The date and time when the intake or output was given. Example: 2021-06-01 10:30:00, 2021-06-02 14:00:00.
deleteFlag smallint A flag indicating if the record has been deleted (1) or not (0). Example: 0, 1.
notes text Additional notes or comments about the intake or output order detail captured by the nurse. Example: Patient tolerated well, Rate adjusted due to patient discomfort.
orderTypeid tinyint identifier for the type of order (intake or output). Foreign key relation to asc_io_ordertype table's id column. Example: 1, 2, 3.
