Table prisma_goal_outcome
Description This table stores the outcome details of specific goals in the Prisma system.
Column DataType Description
ID (Primary Key) int This is the primary key for the table and is an integer that uniquely identifies each goal outcome. For example, 1, 2, 3, etc.
GOAL_DETAILS_ID int This is a foreign key that refers to the `ID` column in the `prisma_goal_details` table. It is an integer that represents the specific goal that the outcome is associated with. For example, 1, 5, 10, etc.
OUTCOME_CODE varchar This is a varchar column that stores a code representing the outcome of the goal. For example, 'OC_001', 'OC_002', 'OC_003', etc.
OUTCOME_DISPLAY varchar This is a varchar column that stores a human-readable description of the outcome. For example, 'Outcome Achieved', 'Outcome Not Achieved', 'Outcome Partially Achieved', etc.
