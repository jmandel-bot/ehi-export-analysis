Table ip_rh_order_icd_map
Description This table appears to store order mapping that belongs to icd.
Column DataType Description
Id (Primary Key) int Unique identifier for each record in the table. This column is used as a primary key and is essential for uniquely identifying a record.
orderId int Foreign key that links to the order associated with the ICD mapping. This column likely refers to an ip_rh_ptorders table's orderId column.
assessmentId int Foreign key that links to the assessment associated with the ICD mapping.
deleteflag tinyint Indicates whether the record is logically deleted. This column is not used on UI but is important for logical deletion, which helps in soft deleting records without removing them from the database.
