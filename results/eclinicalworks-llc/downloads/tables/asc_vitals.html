EHI Export Schema
Table asc_vitals
Description This table is used to save vitals data for a patient.
Column DataType Description
encounterid int used for storing patient's encounter id from enc master table.
vitalid int used for storing vitalid,which maps to itemid from items table.
VALUE varchar Used for saving value of vitals.
id (Primary Key) int used to store Autoincrement id for the table.
UpdatedBy int Used to store modified user's id, which maps UID from user table
unitid int used for storing unit id,which maps to itemid from items table.
batchid int used for storing batchid ,which maps to btachid from asc_vitals_batch table.
value2 varchar Used for saving value of vitals.
qualifierSite int used for storing qualifiersite,which maps to id from asc_vitalsqualifiermaster table.
qualifierPosition int used for storing qualifier position,which maps to id from asc_vitalsqualifiermaster table.
qualifierType int used for storing qualifier type,which maps to id from asc_vitalsqualifiermaster table.
UpdatedTime datetime used to store updated date and time.
deleteFlag int its value can be 0 or 1, which indicates if item is deleted or not.1 indicates the item is deleted.0 indicates the item exists.
notes varchar Any additional notes or comments related to the vital sign. Example: "Patient was feeling dizzy."
