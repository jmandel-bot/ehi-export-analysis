EHI Export Schema
Table occhealth_options
Description This table stores the options related to occupational health, grouped by different categories.
Column DataType Description
optionid (Primary Key) int This is the primary key, an integer that uniquely identifies each option in the table. It is auto-incremented, meaning that each new option added to the table will have an `optionid` value one greater than the previous option. Example: 1, 2, 3, etc.
optionName varchar This is a variable character field with a maximum length of 150 characters. It stores the name of the option. Example: 'Health Insurance', 'Safety Training', 'Ergonomic Assessment', etc.
optionGroup varchar This is a variable character field with a maximum length of 150 characters. It stores the group or category to which the option belongs. Example: 'Insurance', 'Training', 'Assessment', etc.
deleteFlag tinyint This is a tiny integer field that indicates whether the option has been deleted or not. It has a default value of '0', which means the option is not deleted. If the value is '1', the option is considered deleted. Example: 0 (not deleted), 1 (deleted).
