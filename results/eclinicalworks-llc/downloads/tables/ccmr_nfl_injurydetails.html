Table ccmr_nfl_injurydetails
Description This table stores information about injuries sustained by NFL players, including details about the body part affected, the severity of the injury, and the activity during which the injury occurred.
Column DataType Description
id (Primary Key) int This is the primary key for the table, which is an integer value that uniquely identifies each record. It is an auto-incrementing value. Example: 1, 2, 3, etc.
Injuryid int This column will save injuryid of player which is reference key of ccmr_atrisk table of column AtRiskid
BodyArea varchar This column will save body area such as leg, arm, headandneck, lowertorso etc
BodyPart varchar This column will save body part such as Forehead, Hand, Wrist, Knee, Hand etc
BodySite varchar This column will save body site such as Medial Collateral Ligament, Anterior Cruciate Ligament, Meniscus etc
BodySeverity varchar This column will save body Severity such as Grade 1, Grade 2, Moderate etc
BodySide varchar This column will save body side such as Right, Midline, Left etc
BodyInjury varchar This column will save body Injury such as Sprain, Concussion, Tendinitis, Displacement, Bruising etc
Activityid int This column will save activityid, at time of injury saving value will be 0.
ActivityTitle varchar This column will save activity title with date and time in textual data format
InjuryType int This column will save injurytype value, possible values are 0 and 1, 0 means Sports and 1 means Non-Sports
ClosedBy varchar This will save username who have closed injury
ClosedByid int This column will save userid who have closed injury. This column refers to UID column in users table.
ClosedOn datetime This column will save date and time when the injury was closed
ReturnToPlay datetime This column will save date and time when player was returned to play
cptcodes text This column stores the CptCode and description separated with colon(:)
