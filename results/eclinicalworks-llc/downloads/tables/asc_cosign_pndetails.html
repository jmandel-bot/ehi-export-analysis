Table asc_cosign_pndetails
Description This table stores information about co-sign details for a specific encounter, including the co-sign flag, the person who co-signed, and the date of co-signing.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
encounterid int The identifier for the encounter that this co-sign detail is associated with. This could be a foreign key referring to the id column in an encounters table. Example: 101, 102, 103
cosignFlag varchar This column is used to store co-sign flag to check whether order is co-signed or not.
cosignBy int The identifier for the person who cosigned the encounter. This could be a foreign key referring to the id column in a user or staff table. Example: 1001, 1002, 1003.
cosignDate datetime The date and time when the cosign was made. Example: 2021-09-01 12:30:00, 2021-09-02 14:45:00.
deleteFlag tinyint A boolean flag indicating whether the record has been deleted or not. 0 means not deleted, 1 means deleted. Example: 0, 1.
