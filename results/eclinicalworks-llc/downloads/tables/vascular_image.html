EHI Export Schema
Table vascular_image
Description This table stores the documents for the Vascular module.
Column DataType Description
id (Primary Key) int This column is auto-incremented primary key in This column .
document varbinary This column stores Image-file (Binary/File).
encounterid int Stores encounter id which the document is associated to, if it is not associated to any encounter, id stored in this column is 0. Column refers to encounterid column in enc table.
patientid int This column stores patient identifier for a patient, whom the document is associated to. It refers to column UID from users and pid from patients.
docid int This column stores document identifier. It refers to column docid from document Table.
observationid int This column stores Vascular Observation identifier. It refers to column id from vascular_observation Table.
filename varchar This column stores the complete file name of the document along with the extension.
displayindex smallint This column stores display Index for Image. This index will be used to sort Images while displaying Image List.
notes varchar This column stores notes associated with Image.
deleteflag tinyint This column stores value 0 / 1 indicating if the document is marked deleted using the User Interface, this is soft delete. 0 - Not Deleted , 1 -Deleted
catid int This column stores an id of the category selected to display in patient documents. This column refers column itemid from items table.
itemid int Stores Order selected by user to upload Images. List of Orders (Lab, DI, Procedure) are stored in items table. Column refers to itemid column in items table.
