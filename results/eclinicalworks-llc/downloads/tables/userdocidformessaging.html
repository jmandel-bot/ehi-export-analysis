Table userdocidformessaging
Description This table stores information related to user document IDs for messaging purposes, including the user's last login time.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer value. It uniquely identifies each record in the table. Example: 1, 2, 3, etc.
userid int This is an integer value that represents the user's ID. It is a foreign key that refers to the `id` column in the `users` table (not provided in the query). Example: 101, 102, 103, etc.
docid int This is an integer value that represents the document ID associated with the user for messaging purposes. It is a foreign key that refers to the `id` column in the `documents` table (not provided in the query). Example: 1001, 1002, 1003, etc.
lastlogintime datetime This is a datetime value that stores the user's last login time. Example: '2021-09-01 12:34:56', '2021-09-02 08:15:30', etc.
