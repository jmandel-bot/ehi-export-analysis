Table milestoneencdetail
Description This table stores details of milestones related to specific encounters, including their outcomes, triggers, statuses, and interventions.
Column DataType Description
MileStoneDetailid (Primary Key) int The unique identifier for each milestone detail entry. (Examples include: 1, 2, 3).
MileStoneid int identifier for the milestone associated with the encounter. (Foreign key relation: `milestone.milestoneid`) (Example: 101, 102, 103)
Encounterid int identifier for the encounter associated with the milestone. (Foreign key relation: `enc.encounterid`) (Example: 1001, 1002, 1003)
Outcome varchar The description of the outcome of the milestone. (Examples include: 'Successful', 'Failed', 'In Progress').
MileStoneTrigger char A single character representing the trigger for the milestone. (Examples include: 'A', 'B', 'C').
ActiveDate datetime The date when the milestone became active. (Examples include: '2022-01-01', '2022-01-15', '2022-02-01').
Category varchar The category of the milestone (e.g.,'Health', 'Education', 'Social').
Status int The integer representing the status of the milestone with '0' for inactive and '1' for active).
TargetScore1 int The first target score for the milestone (e.g., 0, 1, 2).
TargetScore2 int The second target score for the milestone (e.g., 0, 10, 20).
CurrentScore int The current score of the milestone (e.g., 0, 5, 15).
Intervention1 varchar The description of the first intervention related to the milestone. (Examples include: 'Counseling', 'Medication', 'Support Group', etc.).
Intervention2 varchar The description of the second intervention related to the milestone. (Examples include: 'Counseling', 'Medication', 'Support Group', etc.).
TargetDate datetime The target date for achieving the milestone. (Examples include: '2022-03-01', '2022-03-15', '2022-04-01', etc.).
ResolvedDate datetime The date when the milestone was resolved. (Examples include: '2022-03-01', '2022-03-15', '2022-04-01', etc.).
Detail text Additional text details about the milestone. (Examples include: 'Patient showed improvement after intervention', 'Milestone not achieved due to lack of resources', etc.).
