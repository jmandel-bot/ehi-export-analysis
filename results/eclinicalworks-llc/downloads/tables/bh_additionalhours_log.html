Table bh_additionalhours_log
Description This table stores logs of additional hours for patients, including information about the patient, the CPT item, bonus hours, the user who modified the log, and the authorization number.
Column DataType Description
id (Primary Key) int Unique identifier for each log entry. It is an integer value and is the primary key for the table. Example: 1, 2, 3.
patientid int identifier for the patient associated with the log entry. It is a foreign key that refers to the `id` column in the `patients` table. Example: 101, 102, 103.
cptitemid varchar It is a foreign key that refers to the id column in the items table. Example: 101, 102, 103.
bonushoursid int identifier for the bonus hours associated with the log entry. It is an integer value. Example: 1, 2, 3.
User_id varchar It is a foreign key that refers to the id column in the user table. Example: 101, 102, 103.
ModifyDate datetime Date and time when the log entry was last modified. It is a datetime value. Example: '2021-09-01 12:34:56', '2021-09-02 14:30:00'.
Authorization_No varchar Authorization number associated with the log entry. It is a string value. Example: 'AUTH001', 'AUTH002', 'AUTH003'.
deleteflag tinyint Flag to indicate if the log entry has been deleted. It is a tinyint value with 0 representing not deleted and 1 representing deleted. Default value is '0'. Example: 0, 1.
