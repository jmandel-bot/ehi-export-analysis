Table patients_residential
Description This table stores information about the residential details of patients in a healthcare facility.
Column DataType Description
PatientId (Primary Key) int This is the primary key for the table and represents a unique identifier for each patient. It is an integer value and cannot be NULL. Example: 1, 2, 3, etc.
FacilityId int This column represents the identifier of the healthcare facility where the patient is staying. It is an integer value and can be NULL if the patient is not currently in a facility. Example: 101, 102, 103, etc.
Room varchar This column stores the room number or name where the patient is staying within the facility. It is a varchar(20) and can be NULL if the patient is not currently in a room. Example: 'A101', 'B203', 'C305', etc.
Bed varchar This column stores the bed number or name assigned to the patient within the room. It is a varchar(100) and can be NULL if the patient is not currently assigned a bed. Example: 'Bed 1', 'Bed 2', 'Bed 3', etc.
deleteflag tinyint This column is a tinyint(4) and indicates whether the record has been deleted or not. A value of '0' means the record is active, while a value of '1' means the record has been deleted. Example: 0, 1.
