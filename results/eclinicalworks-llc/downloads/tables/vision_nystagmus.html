Table vision_nystagmus
Description This table stores information about nystagmus, a vision condition characterized by involuntary eye movements, for each patient encounter.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer value. It uniquely identifies each record in the table. Example: 1, 2, 3, etc.
encounterid int Unique identifier for each patient encounter. This column has a foreign key relation with the `enc` table and the `encounterid` column. Example: 101, 102, 103.
direction varchar The direction of the nystagmus eye movement. Example: 'horizontal', 'vertical', 'rotary'.
frequency varchar The frequency of the nystagmus eye movement. Example: 'low', 'medium', 'high'.
amplitude varchar The amplitude of the nystagmus eye movement. Example: 'small', 'medium', 'large'.
NULL_point varchar The position of the eyes where the nystagmus movement is minimized or absent. Example: '10 degrees left', '5 degrees up'.
head_posture varchar The head posture adopted by the patient to minimize the nystagmus movement. Example: 'tilted left', 'chin down'.
comments varchar Additional comments or observations related to the nystagmus condition. Example: 'worsens with fatigue', 'improves with glasses'.
