EHI Export Schema
Table claimrefund
Description This table stores information about claim refunds, including the refund amount, claim number, and related encounter and closing IDs.
Column DataType Description
Id (Primary Key) int Unique identifier for each claim refund record. Example: 1, 2, 3.
RefundId int This is the identifier for the refund associated with this claim refund. This column has a foreign key reference to 'refunds' table, 'RefundId' column
EncounterId int This column stores the encounter Id associated with this claimrefund. This column has a foreign key reference to 'enc' table, 'encounterid' column
ClaimNo int This column stores the Invoice Id associated with this claimrefund. This column has a foreign key reference to 'edi_invoice' table, 'Id' column
Amount decimal The refund amount for the claim refund, stored as a decimal value with 9 digits and 2 decimal places. Example: 100.00, 150.50, 200.25.
DeleteFlag int A flag indicating whether the claim refund record has been deleted (1) or not (0). Example: 0, 1.
parentdelete int A flag indicating whether the parent record of the claim refund has been deleted (1) or not (0). Example: 0, 1.
ClosingId int This column denotes the hard close Id associated with this claim refund if the refund was hard closed. It has a foreign key reference of 'closingtransactions' table, 'closingId' column
