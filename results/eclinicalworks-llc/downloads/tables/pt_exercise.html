EHI Export Schema
Table pt_exercise
Description This table stores information about different physical therapy exercises.
Column DataType Description
id (Primary Key) int This is the primary key column, which uniquely identifies each exercise record in the table. It is an integer value that is automatically incremented when a new record is added. Example: 1, 2, 3, etc.
Name varchar This column stores the name of the exercise. It is a required field and cannot be NULL. It is a variable character string with a maximum length of 255 characters. Example: "Push-ups", "Squats", "Lunges", etc.
Description varchar This column not in use.
deleteFlag int This column stores values 0/1 to indicate whether the record has been marked as deleted or not. 0 - If record is NOT marked as deleted, 1 - If record is marked deleted.
CptItemid int This column stores itemid used for the Billing. This column refers to column itemid from billingdata table.
