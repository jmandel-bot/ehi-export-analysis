EHI Export Schema
Table ethnicity
Description This table stores information about different ethnicities, including their codes, names, and other related details; this is a meta table.
Column DataType Description
Code varchar This column stores a code for a specific ethnicity. Ethnicity code is a string value so it's a varchar.
Name varchar This column stores ethnicity name. The ethnicity name is a string value so it is a varchar.
Ethid (Primary Key) int This column is an auto incremented primary key.
OBMMapping varchar This column stores OBM mapping of the ethnicity. OBM mapping is a string value so it is a varchar.
Favorite tinyint This column stores a value of 1 if ethnicity is set as favorite, all else is 0. 0 is for unfavourite and 1 for favourite so a tynyint(1) is used.
