EHI Export Schema
Table asc_anesthesia_billing
Description This table contains ASC anesthesia billing-related information, such as ID, anesthesia billing ID, maxCostValue, baseUnit, etc.
Column DataType Description
anesthesiaBillingid (Primary Key) int This is an auto incremented primary key column of this column.
id int This column stores id. It refers to id column of billingdata table.
procedureItemid int This column stores procedure item id. It refers to item id column of items table.
cptItemid int This column stores cpt item id. It refers to item id column of items table.
maxCostValue decimal This column stores the maximum cost value for the anesthesia billing. It is a decimal value with 10 digits and no decimal places, with a default of 0. Example: 1000, 2000, 3000, etc.
deleteFlag tinyint This column stores a flag indicating whether the anesthesia billing record has been deleted or not. It is a tiny integer value (1 byte) with a default of 0, where 0 means not deleted and 1 means deleted. Example: 0 or 1.
BaseUnit int This column stores decimal type base unit value entered by user.
