EHI Export Schema
Table claimlevelinfo
Description This table stores information about insurance claims, their amounts, statuses, and related details.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. (e.g., 1, 2, 3)
Fileid int identifier for the claim status. (Foreign key relation: `claimstatus.FileId`)
Payorid varchar identifier for the payor (insurance company) responsible for the claim. (e.g., 'A123', 'B456', 'C789')
Providerid int identifier for the healthcare provider associated with the claim. (Foreign key relation: `Users.UID`)
Claimid int identifier for the claim itself. (Foreign key relation: `edi_invoice.id`)
ClaimAmt decimal Total amount of the claim. (e.g., 1000.00, 500.00, 750.00)
AcceptedClaimAmt decimal Amount of the claim that has been accepted. (e.g., 900.00, 450.00, 700.00)
RejectedClaimAmt decimal Amount of the claim that has been rejected. (e.g., 100.00, 50.00, 50.00)
FinalizedClaimAmt decimal Final amount of the claim after processing. (e.g., 900.00, 450.00, 700.00)
PandingClaimAmt decimal Amount of the claim that is still pending. (e.g., 0.00, 0.00, 0.00)
Status varchar Status of the claim (e.g., 'Accepted', 'Rejected', 'Pending')
StatusCode varchar Code representing the status of the claim. (e.g., 'A', 'R', 'P')
Code varchar Code associated with the claim. (e.g., 'C1', 'C2', 'C3')
Message varchar Message related to the claim status. (e.g., 'Accepted', 'Rejected due to error', 'Pending review')
ErrorCodeMessage varchar Error message associated with the claim, if any. (e.g., '', 'Invalid code', '')
ClaimType varchar Type of the claim. (e.g., 'Medical', 'Dental', 'Vision')
ClaimInfo varchar Additional information about the claim. (e.g., 'Inpatient', 'Outpatient', 'Routine checkup')
CreatedDate datetime Date when the claim record was created. (e.g., '2021-01-01', '2021-02-01', '2021-03-01')
DeleteFlag tinyint Flag indicating if the record has been deleted (1) or not (0). (e.g., 0, 1)
DeletedBy int identifier for the user who deleted the record, if applicable. (e.g., 0, 3001, 3002)
DeletedDate datetime time stamp when the record was deleted, if applicable. (e.g., '2021-01-15 12:34:56', '0000-00-00 00:00:00')
DeletedReason text Reason for deleting the record, if applicable. (e.g., '', 'Duplicate entry', 'Incorrect data')
ErrorClaimAmt decimal Amount of the claim with errors, if any. (e.g., 0.00, 50.00, 0.00)
ClaimAmt997 decimal Amount of the claim related to the 997 code, if any. (e.g., 0.00, 0.00, 50.00)
ClaimAmtPNS decimal Amount of the claim related to the PNS code, if any. (e.g., 0.00, 50.00, 0.00)
batchid varchar identifier for the batch the claim is associated with. (e.g., 'B1', 'B2', 'B3')
response text Text response related to the claim. (e.g., 'Accepted', 'Rejected due to error', 'Pending review')
