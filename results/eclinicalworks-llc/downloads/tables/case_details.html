Table case_details
Description This table stores information about different cases, including details about the patient, provider, guarantor, attorney, case manager, insurance, and other related information.
Column DataType Description
Caseid (Primary Key) int Unique identifier for each case (e.g., 1, 2, 3).
Patientid int Unique identifier for the patient associated with the case, foreign key referring to the `patients` table (e.g., 101, 102, 103).
Providerid int Unique identifier for the provider associated with the case, foreign key referring to the `Doctors` table (e.g., 201, 202, 203).
Guarantorid int Unique identifier for the guarantor associated with the case, foreign key referring to the `patients` table (e.g., 301, 302, 303).
Attorneyid int Unique identifier for the attorney associated with the case, foreign key referring to the `patient_attorney` table (e.g., 401, 402, 403).
CaseManagerid int Unique identifier for the case manager associated with the case, foreign key referring to the `casemanager` table (e.g., 501, 502, 503).
Insuranceid int Unique identifier for the insurance associated with the case, foreign key referring to the `insurance` table (e.g., 601, 602, 603).
CaseType int Integer representing the type of case, foreign key referring to the `case_types` table (e.g., 1 for personal injury, 2 for workers' compensation).
BillToPatient int Integer indicating whether the bill should be sent to the patient (1 for yes, 0 for no).
PreAuth varchar Pre-authorization information for the case (e.g., 'PA12345').
Referral varchar Referral information for the case (e.g., 'Dr. Smith').
CourtName varchar Name of the court associated with the case (e.g., 'Superior Court').
CourtLocation varchar Location of the court associated with the case (e.g., 'Los Angeles, CA').
CourtCase varchar Court case number (e.g., 'LA123456').
CourtDate datetime Date of the court hearing (e.g., '2022-01-01').
DateOfInjury datetime Date of the injury related to the case (e.g., '2021-12-01').
WorkRelated int Integer indicating whether the case is work-related (1 for yes, 0 for no).
Accident int Integer indicating whether the case involves an accident (1 for yes, 0 for no).
AccidentState char Two-letter state abbreviation where the accident occurred (e.g., 'CA').
AccidentTime varchar Time of the accident (e.g., '14:30').
AccidentCause varchar Cause of the accident (e.g., 'Slip and fall').
MissedWorkFrom datetime Start date of missed work due to the injury (e.g., '2021-12-02').
MissedWorkTo datetime End date of missed work due to the injury (e.g., '2021-12-15').
HospitalizationFrom datetime Start date of hospitalization due to the injury (e.g., '2021-12-01').
HospitalizationTo datetime End date of hospitalization due to the injury (e.g., '2021-12-05').
city varchar City where the case is located (e.g., 'Los Angeles').
county varchar County where the case is located (e.g., 'Los Angeles County').
AddressNoStNo text Address number and street number of the case location (e.g., '123 Main St').
DescAccOccur text Description of how the accident occurred (e.g., 'Slipped on wet floor').
externalCaseNo varchar External case number, if applicable (e.g., 'EXT12345').
empDetailid int Unique identifier for the employee detail associated with the case, foreign key referring to the `employee_details` table (e.g., 701, 702, 703).
addressline1 varchar First line of the case address (e.g., '123 Main St').
addressline2 varchar Second line of the case address, if applicable (e.g., 'Apt 4B').
zipcode varchar Zip code of the case location (e.g., '90001').
