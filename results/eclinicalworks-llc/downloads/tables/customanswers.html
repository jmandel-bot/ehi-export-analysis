EHI Export Schema
Table customanswers
Description This table stores custom answers provided by patients for a particulatr question of Custom smartform.
Column DataType Description
Qsid int This column stores the unique identifier for each question. It is an integer value, and it cannot be NULL. Example: 1, 2, 3.
encounterid int This column stores the encounter id . It is an integer value, and it cannot be NULL. Example: 1001, 1002, 1003.
answer varchar This column stores the custom answer provided for the patient for a specific question on custom smartorm. It is a string value with a maximum length of 100 characters. Example: "Yes", "No", "Sometimes".
patientid int This column stores the patient id . It is an integer value, and it cannot be NULL. Example: 1001, 1002, 1003.
DateModified datetime This column stores the time stamp of when the record was last modified. It is a time stamp value and cannot be NULL. Example: '2021-09-01 12:34:56'.
User int This column stores the user id of the user using the Custom SMartform. It is an integer value, and it cannot be NULL. Example: 1001, 1002, 1003.
