EHI Export Schema
Table edi_form_question
Description This tables store CPT Form Question Information which is included ANSI Professional Claim in Loop 2440
Column DataType Description
id (Primary Key) int This is the primary key column, containing a unique integer value for each question in the table. It is automatically incremented as new questions are added. Example: 1, 2, 3, etc.
AssignedId varchar This column contains a varchar value (up to 20 characters) that represents an assigned identifier for the question. It can be NULL if no identifier is assigned. Example: "Q1", "Q2", "Q3", etc.
Question text This column contains a longtext value that represents the text of the question. Example: "What is your name?", "What is your age?", etc.
Questiontype tinyint This column contains a tinyint value (3 digits) that represents the type of question. It has a default value of '0'. Example: 1(Yes/No Response), 2 (Text/Uncodified response), 3 (Date), 4 (Percentage).
forms_id int This column contains an integer value that represents the ID of the form the question belongs to. It has a default value of '0'. This is a foreign key that refers to the `id` column in the `edi_forms` table. Example: 1, 2, 3, etc.
deleteFlag tinyint This column contains a tinyint value (1 digit) that indicates whether the question has been deleted or not. It has a default value of '0'. Example: 0 (not deleted), 1 (deleted).
