EHI Export Schema
Table claim_assignedto_history
Description This table stores the history of claims assigned to different users, along with their status, priority, and other related information.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
ClaimNo int reference of edi_invoice table of id unique column
assignedByid int Unique identifier for the user who assigned the claim. Example: 1001, 1002, 1003.
assignedToid int Unique identifier for the user to whom the claim is assigned. Example: 2001, 2002, 2003.
assignedTo varchar Name of the user to whom the claim is assigned. Example: 'John Doe', 'Jane Smith'.
Date varchar Date when the claim was assigned in a string format. Example: '2021-10-01', '2021-10-02'.
Time varchar Time when the claim was assigned in a string format. Example: '12:30:00', '14:45:00'.
startDate datetime Date when the assigned user started working on the claim. Example: '2021-10-03', '2021-10-04'.
status tinyint Status of the claim assignment, where 0 represents not started, 1 represents in progress, and 2 represents completed. Example: 0, 1, 2.
DueDate datetime Due date for the claim to be completed. Example: '2021-10-10', '2021-10-11'.
completedDt datetime Date when the claim was completed. Example: '2021-10-09', '2021-10-10'.
priority tinyint Priority level of the claim, where 0 represents low priority, 1 represents medium priority, and 2 represents high priority. Example: 0, 1, 2.
startTime datetime Date and time when the assigned user started working on the claim. Example: '2021-10-03 09:00:00', '2021-10-04 10:30:00'.
endTime datetime Date and time when the assigned user completed the claim. Example: '2021-10-09 17:00:00', '2021-10-10 16:30:00'.
totalTime datetime Date and time representing the total time spent on the claim. Example: '0000-00-00 08:00:00', '0000-00-00 06:30:00'.
queueid int reference of workqueueitems table of id primary column
isactive smallint A flag indicating whether the claim assignment is active or not, where 0 represents inactive and 1 represents active. Example: 0, 1.
