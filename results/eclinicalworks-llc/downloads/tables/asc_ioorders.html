EHI Export Schema
Table asc_ioorders
Description This table stores information about intake and output orders for patients during their medical encounters.
Column DataType Description
intakeOutputid (Primary Key) int Unique identifier for each intake/output order. Example: 1, 2, 3.
encounterid int identifier for the medical encounter associated with the intake/output order. This could be a foreign key referring to the encounters table's id column. Example: 1001, 1002, 1003.
itemid int Identifier for the specific item associated with the intake/output order. Example: 101, 102, 103.
itemType int Integer representing the type of item in the intake/output order. Example: 0, 1, 2.
routeid int This column contains routeid which is primary key of ASC_IORoute_List.
itemTableName varchar This column contains item table name as ASC_IOITEMS.
notes text This column is not in use.
displayIndex int This column is not in use.
deleteFlag smallint A small integer indicating whether the intake/output order has been deleted (1) or not (0). Example: 0, 1.
isIOOrderEditable int This column is not in use.
