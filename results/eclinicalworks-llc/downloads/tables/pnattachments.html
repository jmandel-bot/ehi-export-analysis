EHI Export Schema
Table pnattachments
Description This table stores information about attachments added for appointments on the Progress Notes.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key of This column .
encounterid int This column stores the encounterid of the appointment for which attachment is added. This column refers to encounterid column in enc table.
catid int This column is not in use. It will always store value 0.
itemid int This column is not in use. It will always store value 0.
document varbinary This column stores document detail attached for the appointment.
filename varchar This column stores auto generated file name attached for the appointment.
docid int This column stores documentid of the document attached for the appointment. This column refers to docid column in document table.
displayindex tinyint This column stores a digit to indicate order in which document should be listed in progress note.
deleteflag tinyint This column stores value 0 / 1 indicating if the document is deleted or not, this is soft delete. 0 - Not Deleted , 1 -Deleted.
