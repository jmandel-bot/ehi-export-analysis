EHI Export Schema
Table op_drugformulary
Description This table is a master table and stores Rx Formulary. The item key is: isFormularyEnabled
Column DataType Description
id (Primary Key) int This column is an auto incremented Primary Key.
itemid int This column stores itemid of the drug added in this Rx Formulary. This column refers to the column itemid from items table.
itemName varchar This column stores the drug name. This column refers column itemName from items table.
drugNameid int This column stores drug name id from column drugNameid from items table.
strength varchar Strength of the drug (e.g., "500mg", "200mg", "100mg").
route varchar Route of administration for the drug (e.g., "Oral", "IV", "IM").
formulation varchar Formulation of the drug (e.g., "Tablet", "Capsule", "Injection").
take varchar Instructions on how to take the drug (e.g., "Take 1 tablet with water", "Inject 1ml intramuscularly").
frequency varchar Frequency of drug administration (e.g., "Every 6 hours", "Twice a day", "Once a day").
duration varchar Duration of drug treatment (e.g., "For 7 days", "For 3 days", "As needed").
dispense varchar Instructions on how the drug should be dispensed (e.g., "Dispense 30 tablets", "Dispense 10ml").
refill varchar Refill instructions for the drug (e.g., "Refill 2 times", "No refills allowed").
ndc varchar National Drug Code for the drug (e.g., "12345-6789", "98765-4321").
cpt varchar Current Procedural Terminology code for the drug (e.g., "J1234", "J5678").
STATUS smallint This column stores a value 0 or 1. 1 - when this rx formulary is inactive or 0 - when this rx formulary is active.
createdBy int This column stores id of the Admin/Staff who creates this rx formulary. This column refers column UID from user table.
createdDate datetime This column stores a date when this Rx Formulary was created by the Admin/Staff who has access to it. The format used is YYYY-MM-DD HH:MM:SS.
modifiedBy int This column stores id of the Admin/Staff who has access to this rx formulary and modifies it. This column refers column UID from user table.
modifiedDate datetime This column stores a date when this Rx Formulary was modified by the Admin/Staff who has access to it. The format used is YYYY-MM-DD HH:MM:SS.
AdministerFlag smallint This column stores a value 0 or 1. 1 - when the drug can be administered, 0 - Otherwise.
DispenseFlag smallint This column stores a value 0 or 1. 1 - when the drug can be dispensed, 0 - Otherwise.
ContinuousIVFlag smallint This column stores a value 0 or 1. 1 - when the drug is marked as IntraVenous drug, 0 - Otherwise.
cptItemid int This column stores itemid associated with the CPT code in CPT column. This column refers column itemid from items table.
isDeleted smallint This column stores a flag 0 or 1. 1 - when this Rx Formulary is marked as deleted. 0 - Otherwise.
IVHydration smallint This column stores a value 0 or 1. 1 - when the drug is marked for IV hydration being a IntraVenous drug, 0 - Otherwise.
originalNDC varchar This column stores original National Drug Code for the drug in this Rx Formulary.
rxCardFlag tinyint Not Available
mailOrderFlag tinyint Not Available
inActiveDate date Date when the drug formulary record became inactive, if applicable (e.g., "2021-03-01").
