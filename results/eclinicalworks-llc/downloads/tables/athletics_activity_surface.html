Table athletics_activity_surface
Description This table stores information about different surfaces used in athletic activities, along with their subcategories and status.
Column DataType Description
CODE (Primary Key) int This is an integer column that serves as the primary key and is auto-incremented. It uniquely identifies each athletic activity surface. Example: 1, 2, 3, etc.
description varchar This is a varchar(80) column that stores the description of the athletic activity surface. Example: "Grass", "Synthetic Turf", "Clay", etc.
subCat varchar This is a varchar(50) column that stores the subcategory of the athletic activity surface. It has a default value of an empty string. Example: "Indoor", "Outdoor", "All-Weather", etc.
subCatCode int This is an int(25) column that stores the unique code associated with the subcategory of the athletic activity surface. It has a default value of 0. Example: 1, 2, 3, etc.
deleteFlag bit This is a bit(1) column that stores a boolean value indicating whether the record has been deleted or not. It has a default value of b'0' (false). Example: b'0' (not deleted), b'1' (deleted).
