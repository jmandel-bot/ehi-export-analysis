EHI Export Schema
Table enc_cdc_countries
Description This table stores information about patient travel histories(visited countries, their health alerts, and location data). Note: Travel history feature has been discontinued.
Column DataType Description
id (Primary Key) bigint Unique identifier for each record in the table. Example: 1, 2, 3
enc_id bigint Foreign key referring to the `id` column in another table (not specified). Example: 101, 102, 103
visiting_date datetime The date when the country was visited. Example: '2021-08-15', '2021-08-16', '2021-08-17'
country varchar The name of the country. Example: 'United States', 'Canada', 'Mexico'
alert varchar The alert level or status for the country. Example: 'Level 1', 'Level 2', 'Level 3'
alert_desc text A description of the alert. Example: 'Practice usual precautions', 'Exercise increased caution', 'Reconsider travel'
link varchar A URL related to the country or alert. Example: 'https://www.example.com/country1', 'https://www.example.com/country2'
alert_link varchar A URL specifically for the alert information. Example: 'https://www.example.com/alert1', 'https://www.example.com/alert2'
suffix varchar Additional information or a suffix for the country name. Example: 'USA', 'CAN', 'MEX'
coordinates varchar The geographical coordinates of the country. Example: '37.7749,-122.4194', '45.4215,-75.6972'
bounds varchar The geographical bounds of the country. Example: '24.396308,-124.848974,49.384358,-66.885444', '41.676555,-141.00275,83.336212,-52.323198'
displayname varchar The display name of the country, which may include additional information. Example: 'United States of America', 'Canada (CAN)'
location_coordinates varchar The geographical coordinates of a specific location within the country. Example: '37.7749,-122.4194', '45.4215,-75.6972'
location_bounds varchar The geographical bounds of a specific location within the country. Example: '37.63983,-123.173825,38.78614,-121.88629', '45.245,-76.246,45.534,-75.246'
location_name varchar The name of the specific location within the country. Example: 'San Francisco', 'Ottawa'
crawling_date datetime The date when the data was crawled or collected from Scribe. Example: '2021-08-01', '2021-08-02'
