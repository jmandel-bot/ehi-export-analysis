EHI Export Schema
Table attachedpastorders
Description This table stores past lab reports pulled into Progress Notes from the RCP (Right Chart Panel).
Column DataType Description
id (Primary Key) int This is the primary key for the table, which is an auto-incrementing integer value. It uniquely identifies each record in the table. Example: 1, 2, 3, etc.
Reportid int This column stores id of past lab ordered for patient. This column refers to reportid column in labdata table.
AttachedToEncid int This column stores id of an encounter to which patient's past lab report is pulled to from Right Chart Panel. This column refers to encounterid column in enc table.
deleteFlag tinyint This column stores values 0/1 to indicate whether the past report is marked as deleted or not. 0 - Past order is not marked as deleted, 1 - Past order is marked as deleted.
labAttributes varchar This column stores date and time when lab result is merged to Progress Notes from Right Chart Panel. Date and time are stores in YYYY-MM-DD hh:mm:ss.000 format.
performedDate datetime This column stores date and time when lab result is merged to Progress Notes from Right Chart Panel. Date and time are stores in YYYY-MM-DD hh:mm:ss.000 format.
