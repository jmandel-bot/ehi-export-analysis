EHI Export Schema
Table prisma_patient_data
Description This table stores Patient demographic and contact information received from external sources
Column DataType Description
id (Primary Key) bigint Unique identifier for each patient record (e.g., 1, 2, 3).
sectionId bigint Identifier for the section this patient data belongs to. Foreign key relation to `prisma_ccd_document_section.id`.
patientId varchar Unique identifier for the patient (e.g., "P12345").
firstName varchar Patient's first name (e.g., "John").
lastName varchar Patient's last name (e.g., "Doe").
middleName varchar Patient's middle name (e.g., "Michael").
previousName varchar Patient's previous name, if any (e.g., "Jane Smith").
suffix varchar Patient's name suffix, if any (e.g., "Jr.").
dateOfBirth datetime Patient's date of birth (e.g., "1990-01-01 00:00:00").
dateOfDeath datetime Patient's date of death, if applicable (e.g., "2020-01-01 00:00:00").
raceCodes varchar Patient's race code(s) (e.g., "2106-3").
raceName varchar Patient's race name (e.g., "White").
genderCode varchar Patient's gender code (e.g., "M").
genderName varchar Patient's gender name (e.g., "Male").
maritalStatusCode varchar Patient's marital status code (e.g., "S").
maritalStatusName varchar Patient's marital status name (e.g., "Single").
ethnicityCode varchar Patient's ethnicity code (e.g., "2186-5").
ethnicityName varchar Patient's ethnicity name (e.g., "Not Hispanic or Latino").
religiousAffiliationCode varchar Patient's religious affiliation code, if any (e.g., "1013").
religiousAffiliationName varchar Patient's religious affiliation name, if any (e.g., "Christian").
sex varchar Patient's sex (e.g., "M" or "F").
birthPlace varchar Patient's place of birth (e.g., "New York, NY").
homephone varchar Patient's home phone number (e.g., "555-123-4567").
workphone varchar Patient's work phone number (e.g., "555-987-6543").
mobilephone varchar Patient's mobile phone number (e.g., "555-555-5555").
fax varchar Patient's fax number, if any (e.g., "555-111-2222").
lastModifiedTime datetime Date and time when the patient record was last modified (e.g., "2021-01-01 00:00:00").
patient_prefix varchar Patient's name prefix, if any (e.g., "Mr.").
deceasedFlag tinyint Indicator of whether the patient is deceased (e.g., 0 for alive, 1 for deceased).
