EHI Export Schema
Table globalalertdetails
Description This table stores the Global Alerts associated to each patient.
Column DataType Description
id (Primary Key) int This column is auto-incremented primary key.
Patientid int This column stores the ID of a patient to whom the alert is linked to. This column refers to UID column in user table.
Priority int This columns stores values 0 or 1 which indicates the priority of the alert. 0 is normal priority, and 1 is high priority.
Alertid int This column stores an ID for an alert which is being associated with the patient. This column refers to the ID column in the globalalerts table.
Notes text This column stores notes written by provider and staff while creating new global alert or while adding alert for the patient.
deleteflag int This column stores values 0 or 1 for soft delete flag for an alert associated with the patient. 0 - Alert is not deleted, 1 - Alert is deleted.
ExpiryDate datetime This column stores date after which the global alert expires automatically and should not be visible in Right Chart Panel. This value is set by end user while creating a global alert.
