Table inventory_salereturn
Description This table stores information about sales returns in the inventory system, including the charges, discounts, taxes, and total amount of the return, as well as the currency used and the user who created and updated the record.
Column DataType Description
id (Primary Key) int Unique identifier for each sale return record. Example: 1, 2, 3
Sale_id int identifier for the sale associated with the return. Example: 101, 102, 103
ChargesAmt decimal The amount of charges associated with the sale return. Example: 10.0000, 15.2500, 20.5000
DiscountAmt decimal The amount of discount applied to the sale return. Example: 5.0000, 7.5000, 10.0000
TaxesAmt decimal The amount of taxes applied to the sale return. Example: 1.5000, 2.2500, 3.0000
TotalAmt decimal The total amount of the sale return, including charges, discounts, and taxes. Example: 25.0000, 30.0000, 35.0000
IsDeleted tinyint A flag indicating whether the sale return record has been deleted (1) or not (0). Example: 0, 1
UpdatedBy int identifier for the user who last updated the sale return record. Example: 1001, 1002, 1003
UpdatedOn datetime The date and time when the sale return record was last updated. Example: '2021-10-01 12:34:56', '2021-10-02 14:30:00'
CreatedBy int identifier for the user who created the sale return record. Example: 1001, 1002, 1003
CreatedOn datetime The date and time when the sale return record was created. Example: '2021-09-01 12:34:56', '2021-09-02 14:30:00'
SoldBy int identifier for the user who sold the item that is being returned. Example: 1001, 1002, 1003
invoiceid int identifier for the invoice associated with the sale return. Example: 2001, 2002, 2003
Enc_id int Encrypted identifier for the sale return record. Example: 3001, 3002, 3003
Currency varchar The currency used for the sale return amounts. Example: 'USD', 'EUR', 'GBP'
