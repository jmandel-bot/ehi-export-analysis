Table prisma_careplan_activity_goal
Description This table stores the relationship between care plan activities and their associated goals.
Column DataType Description
ID (Primary Key) int This is the primary key for the table, which uniquely identifies each record. It is an integer value that is automatically incremented when a new record is added. Example: 1, 2, 3, etc.
CAREPLAN_ACTIVITY_ID int This column stores the ID of the care plan activity that the goal is associated with. It is a foreign key that refers to the `ID` column in the `prisma_careplan_activity_details` table. Example: 101, 102, 103, etc.
GOAL_ID int This column stores the ID of the goal associated with the care plan activity. It is an integer value and can be NULL if there is no specific goal associated with the activity. Example: 201, 202, 203, etc.
