EHI Export Schema
Table rpm_device_review_billing_details
Description This table stores the Claims / Out of office encounter details related to remote patient monitoring (RPM) monthly device reviews for enrolled patients.
Column DataType Description
id (Primary Key) int Contains auto incremented id, datatype is INT. This column stores integer value which could grow with number of users added to the care team and hence integer is selected. It is AUTO_INCREMENT column and has PRIMARY KEY index.
rpm_enrollment_id int Contains Primary Key column of rpm_enrollment_details table to store RPM Enrollment id. The value of this field cannot be NULL. There is a Foreign Key constraint associated with this field FK_rpm_device_review_billing_details_rpm_enrollment_id referencing  id column rpm_enrollment_details table.
ptacoid int Contains Primary Key column of ptacoprograms table to store PTACO id . The datatype chosen is an INT The value of this field cannot be NULL.  There is a Foreign Key constraint associated with this field FK_rpm_device_review_billing_details_ptacoid referencing  id column ptacoprograms table.
patientid int Contains Primary Key column of user table to store Patient id. The value of this field cannot be NULL.  There is a Foreign Key constraint associated with this field FK_rpm_device_review_billing_details_patientid referencing  UID column of user table.
device_review_claimid int Contains Claim id for Device Review details and datatype chosen is INT. The value of this field can be NULL. There is Foreign Key constraint associated with this field FK_rpm_device_review_billing_details_device_review_claimid referencing id column edi_invoice table.
device_review_encounterid int Contains Encounter id for Device Review details and datatype chosen is INT. The value of this field can be NULL. There is Foreign Key constraint associated with this field FK_rpm_device_review_billing_details_device_review_encounterid referencing encounterid column enc table.
device_review_claimDate datetime Contains Claim Date for Device Review when the claim got generated. The value of this field can be NULL. The default value of this field is '1901-01-01 00:00:00' for MsSQL and '0000-00-00 00:00:00' for MySQL.
device_review_billingDate datetime Contains Billing Date for Device Review for specific month. The value of this field can be NULL. The default value of this field is '1901-01-01' for MsSQL and '0000-00-00' for MySQL.
device_review_claimFailureReason varchar Device Review Claim Failure reason in case of failed claims. The value of this field can be NULL.  The default value of this field is not provided.
deleteFlag tinyint Flag indicates the record added is deleted, datatype is tinyint - 0: Not Deleted 1: Deleted. This field will always have a value hence defined as NOT NULL. By default, this column should not be in Deleted status hence default 0 is chosen.
