Table bh_treatment_plan_category_mapping
Description This table stores the mapping between treatment plans and their categories, with additional information about each mapping.
Column DataType Description
id (Primary Key) int Unique identifier for each mapping record. Example: 1, 2, 3, etc.
tpid int The identifier of the treatment plan. This is a foreign key that refers to the `id` column in the `bh_treatment_plan` table. Example: 101, 102, 103, etc.
structid int The identifier of the treatment plan category. This is a foreign key that refers to the `id` column in the `bh_treatment_plan_dictionary` table. Example: 201, 202, 203, etc.
NAME varchar The name of the treatment plan category mapping. Example: "Category A - Plan 1", "Category B - Plan 2", etc.
deleteFlag tinyint A flag indicating whether the mapping record has been deleted or not. A value of 0 means the record is not deleted, and a value of 1 means the record is deleted. Example: 0, 1.
