EHI Export Schema
Table refxchangelogs
Description This table stores the information about referral exchange logs(print,fax,etransmit), including user and referral details, log type, number of attachments, and modification time.
Column DataType Description
id (Primary Key) int This is auto-incremented primary key in This column .
userid int This column stores id of a user who is interacting with a referral which is prompting entry into this log table. The user is usually a provider or a staff. This column refers to UID column in user table.
referralid int This column stores id of a referral created by provider/staff for which patient is visiting the referred provider. This column refers to referralid column in referral table.
logType tinyint This column stores a number which for an action performed by end user on a referral mentioned in respective referralid column of This column . eg. PRINTED(0),PRINTED WITH ATTACHMENT(1),FAXED(2),FAXED WITH ATTACHMENT(3),SEND(5)
numattachments tinyint This column stores number of attachment count sent via referral
modifiedtime datetime This column stores date and time if any referral exchange is made on the referral by end user. Date and time is stored in "YYYY-MM-DD hh:mm:ss.000" format.
