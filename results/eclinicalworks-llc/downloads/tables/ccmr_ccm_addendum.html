EHI Export Schema
Table ccmr_ccm_addendum
Description This table stores information about the addendum notes added to specific timer log including timerId, provider who added notes and date time when it is added.
Column DataType Description
id (Primary Key) int Contains auto incremented id, datatype is INT. This column stores integer value which could grow with number of users added to the care team and hence integer is selected. It is AUTO_INCREMENT column and has PRIMARY KEY index.
timerid int Contains unique id of phm timer table. This field references to id column of ccmr_ccm_timer table. This field cannot have a NULL value.
providerid int Contains the userid which is referenced by UID column of user table for addendum added by which user and the datatype is INT. This field cannot have a NULL value.
addendumDate datetime Contains date time value when a user has added the addendum and datatype is DATETIME. This field can have a NULL value.
notes text Contains the notes added by the user for specific timer entry. The data type is text and this field can have NULL value.
