Table allergy_test_options
Description This table stores the options for allergy tests, including the types of tests and their results.
Column DataType Description
id (Primary Key) int The column stores an id of the allergy test options table. This is an auto incremental primary key of This column .
FORMid int This column stores identifier number of the form, which is associated with the test. It refers to column formid from allergy_form table.
TESTid int This column stores the id of the parent table (allergy_test), for the test. It refers to column id from allergy_test table.
ENCid int This column stores the encounterid of associated with the test. It refers to column encounterid from enc table.
