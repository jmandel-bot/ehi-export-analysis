EHI Export Schema
Table pt_exercisevalues
Description This table stores information about physical therapy exercise values which define the information related to exercise performed in a visit.
Column DataType Description
id (Primary Key) int This column is auto-incremented primary key in This column . Unique id is assigned to each record when they are created using User Interface.
Encid int This column stores encounterid which the physical therapy is documented for. This column refers column encounterid from enc table.
Exerid int This column stores identifier number of the exercise . It refers to column id from pt_exercise table.
Set_Repetition varchar This column stores Set Repetition values for the exercise. This column size is changed from 100 to 256 because: [the clinician is allowed to document and save up to 256 characters in the Sets & Repetitions field of the Preventions & Contradictions log section. Hence, the column size is increased from 100 to 256]
deleteFlag int This column stores values 0/1 to indicate whether the record has been marked as deleted or not. 0 - If record is NOT marked as deleted, 1 - If record is marked deleted.
Time int This column stores exercise time.
