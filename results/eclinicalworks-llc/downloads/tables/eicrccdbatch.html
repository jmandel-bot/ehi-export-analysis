EHI Export Schema
Table eicrccdbatch
Description This table stores information about batches of electronic health records in the form of C-CDA (Consolidated Clinical Document Architecture) documents, including their start and end times, status, C-CDA type, and transmission mode.
Column DataType Description
batch_id (Primary Key) int This is the primary key column, containing a unique integer identifier for each batch. Example: 1, 2, 3, etc.
batch_start_time datetime This column stores the start time of the batch as a datetime value. Example: '2022-01-01 12:00:00'
batch_end_time datetime This column stores the end time of the batch as a datetime value. Example: '2022-01-01 12:30:00'
batch_status tinyint This column stores the status of the batch as a tinyint value, where 1 represents 'Success', 2 represents 'Failed'
ccda_type varchar This column stores the type of C-CDA document as a varchar(255) value.
