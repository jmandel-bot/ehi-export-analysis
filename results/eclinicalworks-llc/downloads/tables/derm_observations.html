EHI Export Schema
Table derm_observations
Description This table stores observation data on body regions unique for each encounter. Detailed information on observation is tabled in derm_observationdetails referring to the column observationID.
Column DataType Description
observationid (Primary Key) int This is auto-incremented primary key in This column . This is unique for each encounter. This refers to overall dermatology examination documentation for this encounter.
diagramid int This column stored constant 1. This is column is not in use.
encounterid int This column stores encounter id and refers to column encounterid from table enc.
patientid int This column stores patient id and refers to column UID from table users.
thumbnail text This column is not in use.
createdby int This column stores identifier number of the provider who has created this observation. It refers to column UID from user table.
createdon datetime This column stores date and time of when this observation is created.
modifiedby int This column stores identifier number of the provider who has modified this observation. It refers to column UID from user table.
modifiedon datetime This column stores date and time of when this observation is modified.
data_3d text This column stores graphical information which display annotation on biodigital image of body.
xpoints text This column is not in use.
ypoints text This column is not in use.
notes text This column stores general notes for dermatology examination.
