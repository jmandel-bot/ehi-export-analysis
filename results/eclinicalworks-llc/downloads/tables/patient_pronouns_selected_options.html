Table patient_pronouns_selected_options
Description This table stores the selected pronoun options for patients along with the reason, if selected, for 'Another Pronoun', and the status of the record.
Column DataType Description
id (Primary Key) int This column is the primary key column, which uniquely identifies each record in the table. It is an integer value that is automatically incremented with each new entry. Example: 1, 2, 3, etc.
patientid int The unique identifier of the patient associated with the selected pronoun option. It is a foreign key that refers to the `users` table's `UID` column.
modified_by int The unique identifier of the user who last modified the record. It is a foreign key that refers to the `users` table's `UID` column.
pp_id int The unique identifier of the selected pronoun option. It is a foreign key that refers to the `pronouns_options` table's `id` column.
pp_reason varchar This column is a 20-character length column that specifies free-text pronouns if the user selected the Another Pronoun option.
isactive tinyint A flag indicating whether the record is active (1) or inactive (0).
deleteflag tinyint A flag indicating whether the record has been deleted (1) or not (0).
added_on datetime The date and time when the record was added to the table. (e.g., '2021-09-01 12:34:56')
inactive_date datetime The date and time when the record was marked as inactive, if applicable. (e.g., '2021-10-01 12:34:56' or NULL)
action_performed varchar An enumeration of the last action performed on the record. Possible values are 'added', 'created', 'modified', 'deleted', 'active', and 'inactive'.
modality varchar A short description of the modality used for the action performed on the record. (e.g., 'web', 'mobile', 'API')
