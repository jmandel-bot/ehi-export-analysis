EHI Export Schema
Tableip_rh_pntypes
DescriptionThis table stores the different standard and custom progress notes stored and used in the system.
ColumnDataTypeDescription
id(primaryKey) int Stores the auto increment id for this table
NAME varchar Stores the Name of the note type
groupId int Stores the groupid id of the master groupid name from ip_rh_pnsectiongroups table
encType int Stores the type of encounter which system identifyes the note type with
pnType varchar Stores the standard or custom note type text, Standard - For standard note, Custom - for custom note
disciplineId int Stores the Discipline id of the
departmentId int This stores the id from department mapping table ip_rh_loc_department
deleteFlag tinyint Soft delete flag for this entry, 1 - Deleted, 0 - Not Deleted
createdBy int User who created this entry in this table, refers to uid from Users table
createdDatetime datetime Time stamp of when this created action was performed.
modifiedBy int User who modified this entry in this table, refers to uid from Users table
modifiedDatetime datetime Time stamp of when this modify action was performed.
published tinyint Flag to specify if the note is published to be used on the patient side. 0 - Not published, 1 - Published.
instanceLimit tinyint Used for Summary tab only to specify how many instance of structured type links should be shown in the patient summary tab.
contentTypeId int Used to map the progress note to system understandable content type. Internal to the system
isHidden tinyint Used to hide the note from the front end. 0 - Not hidden, 1 - Hidden
