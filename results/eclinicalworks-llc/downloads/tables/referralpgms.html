EHI Export Schema
Table referralpgms
Description This table stores information about referral programs.
Column DataType Description
id (Primary Key) int This is the primary key column, which uniquely identifies each referral program. It is an integer value that is automatically incremented when a new referral program is added. Example: 1, 2, 3, etc.
RefPgm varchar This column stores the name of the referral program. It is a variable character string with a maximum length of 25 characters and cannot be NULL. The default value is an empty string. Example: Referral Program A, Referral Program B, etc.
deleteFlag tinyint This column indicates whether a referral program has been deleted or not. It is a tiny integer value with a default value of 0. A value of 0 means the referral program is not deleted, and a value of 1 means the referral program is deleted. Example: 0, 1.
