EHI Export Schema
Table pathologydetail
Description This table stores pathology details related to patients, including information about the pathology item, patient, provider, specimen, insurance, and results.
Column DataType Description
id (Primary Key) int Unique identifier for each pathology detail record. Example: 1, 2, 3.
itemid int Unique identifier for the pathology item. Example: 101, 102, 103.
location text Text description of the pathology item's location. Example: Left breast, Right lung.
ruleout text Text description of the conditions to be ruled out. Example: Infection, Inflammation.
comments text Additional text comments related to the pathology detail. Example: Requires further examination, Follow up in 6 months.
itemname varchar Name of the pathology item. Example: Biopsy, Cytology.
patientid int Unique identifier for the patient. Example: 1001, 1002, 1003.
collection varchar Description of the specimen collection method. Example: Fine needle aspiration, Core needle biopsy.
orderdate datetime Date when the pathology order was placed. Example: 2021-06-15, 2021-07-20.
providerid int Unique identifier for the healthcare provider. Example: 2001, 2002, 2003.
specimenid int Unique identifier for the specimen. Example: 3001, 3002, 3003.
facilityid int Unique identifier for the facility where the pathology was performed. Example: 4001, 4002, 4003.
laboratory varchar Name of the laboratory where the pathology was performed. Example: LabCorp, Quest Diagnostics.
Prinsid int Unique identifier for the primary insurance. Example: 5001, 5002, 5003.
PrGrid int Unique identifier for the primary insurance group. Example: 6001, 6002, 6003.
Secinsid int Unique identifier for the secondary insurance. Example: 7001, 7002, 7003.
SecGrid int Unique identifier for the secondary insurance group. Example: 8001, 8002, 8003.
InsuranceText text Text description of the insurance coverage. Example: PPO, HMO.
primaryinsurance varchar Name of the primary insurance company. Example: Blue Cross Blue Shield, Aetna.
secondaryinsurance varchar Name of the secondary insurance company. Example: Cigna, UnitedHealthcare.
specimenno int Unique identifier for the specimen number. Example: 9001, 9002, 9003.
reportid int Unique identifier for the pathology report. Example: 10001, 10002, 10003.
assessmentid int Unique identifier for the pathology assessment. Example: 11001, 11002, 11003.
TxDate datetime Date when the pathology treatment was performed. Example: 2021-08-15, 2021-09-20.
STATUS varchar Status of the pathology detail, either Open or Addressed.
result varchar Result of the pathology, either Benign, or Malignant.
