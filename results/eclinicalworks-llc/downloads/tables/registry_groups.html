Table registry_groups
Description This table stores information about different registry groups, including their names and associated colors.
Column DataType Description
regrp_id (Primary Key) int This is an integer column that serves as the primary key for the table. It is an auto-incrementing value that it will automatically increase for each new entry. Example values: 1, 2, 3, etc.
group_name varchar This is a varchar(100) column that stores the name of the registry group. It has a NOT NULL constraint, meaning that a value must be provided for this column. It also has a UNIQUE constraint, ensuring that no two groups can have the same name. Example values: Admin, Marketing, Sales, etc.
group_color varchar This is a varchar(50) column that stores the color associated with the registry group. This column does not have any constraints, so it can be left empty or have a default value. Example values: Red, Blue, Green, etc.
