Table smartnote_prompt
Description This table stores smart notes prompt details
Column DataType Description
id (Primary Key) int Primary Key for this table hence it is defined by int(11) in Mysql and int in SQL Server. This column is not expected to be NULL as it is primary key.
categoryid int Represents unique id for HPI and Examination section (Foreign key to itemID column in items table). Hence, defined as int. This column is mandatory as smart notes are created from HPI and Examination section and hence is NOT NULL. 
name varchar Represents the name of the smart note prompt. Hence, defined as varchar (255). This column is mandatory as there always exists the name of the prompt and hence is NOT NULL. Unique constraint on categoryid, name.
type tinyint Represents the type of prompt. The value can be 1-6 , 1- Boolean, 2 Date, 3 Free Text, 4 Multi Select, 5 Numeric , 6- Single Select. Hence, defined as tinyint. This column is mandatory as there always exists a type for the prompt and hence is NOT NULL. (Foreign key to smartnote_prompt_type(typeId).
defaultValue varchar This column is used to capture the default value of a prompt. The value is of type varchar(255) and can be NULL. For prompt types Boolean, Date, Numeric default value is stored in this column.
