EHI Export Schema
Table nt_worksheet
Description This table stores the entries of saved and updated worksheets.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
encounterid int This column stores the encounter id of the worksheet encounter.
worksheetid int This column stores the id of the worksheet. The worksheetid column corresponds to the id column in the nt_worksheet_list table.
islocked tinyint This column hold the value of the locked status of the worksheet. If islocked = 1, then the worksheet is locked If islocked = 0, then the worksheet is unlocked.
data text This column stores the data that is filled in the worksheet in an xml format. As part of data-validation, the xml tags are encrypted while saving the data. It is decrypted when the data is loaded.
deleteflag int A boolean flag indicating whether the record is deleted or not. 1 means deleted, and 0 means not deleted. Example: 0, 1.
created_date datetime The date and time when the record was created. Example: '2021-09-01 12:34:56'.
modified_date datetime The date and time when the record was last modified. Example: '2021-09-02 14:56:32'.
print_data text This column stores the entire worksheet data in an xml format. The data from this column is fetched while printing / faxing.
sf_data text This column stores the structured data details in the XML tags format.
modified_by int This column stores the user id of the logged in provider.
print_dataNo_JS text This column stores the entire worksheet data in an xml format without the Javascript code.
PRINT_PDF_DATA text This column stores the entire worksheet data in an xml format. The data from this column is fetched when the PDF format for the worksheet is enabled.
