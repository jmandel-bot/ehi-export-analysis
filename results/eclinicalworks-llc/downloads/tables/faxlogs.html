EHI Export Schema
Table faxlogs
Description This table stores information about outgoing faxes, including details about the sender, recipient, fax content, and status.
Column DataType Description
LogID (Primary Key) int The column stores an id of the outgoing faxes. This is an auto incremental primary key of This column .
LoginID int The column stores an id of the logged in user who has sent the fax. This column refers to the uid column in user table. This can not have NULL value.
DoctorID int This column stores the ID of the provider that had auto-populated or updated in From Provider column Fax preview window. This column refers to the uid column in user table.
PatientID int The column stores an id of the patient of which the information is being sent in the fax . This column refers to the uid column in user table.
ToID int This column stores the id of the recipient. e.g. When the fax is sent to provider, it will store the id of the provider. When the fax is sent to pharmacy, it will store the id of the pharmacy selected etc
ToName varchar Name of the recipient of the fax. (Example: "John Doe", "Jane Smith")
ToFax varchar Fax number of the recipient. (Example: "123-456-7890", "987-654-3210")
tocompany varchar Name of the company the recipient belongs to. (Example: "ABC Hospital", "XYZ Clinic")
EncounterID int Identifier for the patient encounter associated with the fax. (Example: 4001, 4002, 4003)
FaxInitTime varchar time stamp when the fax was initiated. (Example: "2022-01-01 12:34:56")
Retry int Number of retries for sending the fax. (Example: 0, 1, 2)
Status int This is not being used anymore.
faxdata text Long text containing the content of the fax. (Example: "Prescription details...", "Lab report...")
FaxStatus varchar Enumerated status of the fax (Logged, Completed, Failed, Pending, Query, Received, Sending, Deleted, UserCancelled).
SendDate datetime Date and time when the fax was sent. (Example: "2022-01-01 12:45:00")
FaxType varchar Enumerated type of fax content (Prescription, ProgressNotes, ConsultNotes, LabReport, LabOrder, TiffDocument, TxtDocument, CustomForms, WebURL).
FaxManLogID varchar Unique identifier for the fax manager log entry. (Example: "a1b2c3d4", "e5f6g7h8")
FaxPrefix varchar Prefix for the fax number. (Example: "1", "01")
FaxFile varchar FaxFile column is used to store the name of the file being faxed in case of non-HTML faxes. e.g. 4860-9193-122-b4b844a3-62a1-4aa5-fc22-d9ec1a3afc95.tif For HTML faxes initially it will be blank. Fax server will update it with the tiff file name after it converts HTML fax into Tiff file.
FaxTransaction varchar This is not longer being used.
RxIds text Column stores name of Rx being filled through Fax.
FaxLogStatus varchar This is no longer being used.
FaxLogBatch int This column is not being used anymore.
FacilityID int This column stores the id of the facility that was selected in the Facility field while sending the fax. This refers to the id column in edi_facilities table.
FacilityName varchar This column stores the name of the facility that was selected in the Facility field while sending the fax. This refers to the name column in edi_facilities table.
reminded int This is no longer being used.
remindedDate datetime This is no longer being used.
FaxCoverPage int This is no longer being used. This was part of the VB based fax server.
FaxCoverPageID int This is no longer being used. This was part of the VB based fax server.
FaxStatusMsg varchar This column is used to store provide the more details on the current status of the fax. e.g. In case of Fax which is in 'Failed' status it will have value 'Fax [4859] - Failed'.
faxsubject varchar This column is being used to store the subject of the fax. Short description of the content being sent in the Fax which user enters while sending the fax.
LoginName varchar Name of the user who initiated the fax. (Example: "Alice", "Bob")
scheduledDate datetime This contains future date and time of the fax, when fax is scheduled for future time and not to be sent immediately. For regular Fax, it will have same value as of FaxInitTime column.
sentnotification tinyint This is no longer being used.
OriginalFaxDoc varchar This is no longer being used.
FromName varchar This stores the name of the sender that is sending the fax.
FromVoice varchar This column stores the Telephone number of the facility thathas auto-populated in the facility field. This will also contain the telephone number that is free-texted in the voice field.
fromcompany varchar This column stores the Name of the facility that has auto-populated in the Company field. This will also contain the value that is free-texted in the Company field.
FaxInitTime2 datetime This is not being used.
isMsgSent tinyint This is not being used.
isTextMsgSent tinyint This is not being used.
externaldocid int This column stores ID of document/fax received from outside of organization for Smart Referral Management feature.
isAddressed tinyint This columns conveys whether user has marked the outgoing fax as addressed or not. By default value is set to 0 and it updated to 1 when addressed.
addressedComment text Text containing comments about the addressed fax. (Example: "Reviewed and approved", "Needs further review")
OriginalFaxFile varchar This column contains the name of the document which is Faxed in case of non-HTML faxes. In case of HTML faxes it is kept blank.
dialInCode smallint Dial-in code for the fax. (Example: 1234, 5678)
suffix varchar Suffix for the fax number. (Example: "A", "B")
splitFaxesBatchId varchar Identifier for the batch of split faxes. (Example: "SPLIT123", "SPLIT456")
