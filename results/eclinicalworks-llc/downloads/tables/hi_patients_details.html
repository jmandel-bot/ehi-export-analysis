EHI Export Schema
Table hi_patients_details
Description This table stores the details of patients' extracted encounters through healow Insights, with encounter information and extraction date.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
transactiondetailsid int Unique identifier for the transaction details associated with the patient encounter. Example: 101, 102, 103.
patientid int Unique identifier for the patient involved in the encounter. Example: 1001, 1002, 1003.
patientaccountno varchar Unique account number associated with the patient. Example: 'PA12345', 'PA67890'.
encounterid int Unique identifier for the patient encounter. Example: 5001, 5002, 5003.
doctorid int Unique identifier for the doctor involved in the encounter. Example: 2001, 2002, 2003.
DATE datetime Date and time of the patient encounter. Example: '2022-01-01 10:00:00', '2022-01-02 14:30:00'.
startTime datetime Start time of the patient encounter. Example: '10:00:00', '14:30:00'.
endTime datetime End time of the patient encounter. Example: '10:30:00', '15:00:00'.
deleteflag tinyint Indicates whether the record has been deleted (1) or not (0). Example: 0, 1.
extractiondate datetime Date and time when the record was extracted from the source system. Example: '2022-01-01 12:00:00', '2022-01-02 16:00:00'.
inserteddate datetime Date and time when the record was inserted into the table. Example: '2022-01-01 12:05:00', '2022-01-02 16:05:00'.
