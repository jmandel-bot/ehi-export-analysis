Table prisma_goal_details
Description This table stores information about various goals, their details, and their relationships with other tables.
Column DataType Description
ID (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
GOAL_ID varchar Unique identifier for each goal. Example: "G1", "G2", "G3".
SECTION_ID bigint Identifier for the section the goal belongs to. Foreign key relation with `prisma_ccd_document_section` table's `id` column. Example: 101, 102, 103.
SOURCE_ID int Identifier for the source of the goal. Example: 1, 2, 3.
LIFE_CYCLE_STATUS varchar Status of the goal's life cycle. Example: "Active", "Inactive", "Completed".
DESCRIPTION_CODE varchar Code representing the description of the goal. Example: "D1", "D2", "D3".
DESCRIPTION_DISPLAY varchar Display text for the description of the goal. Example: "Improve health", "Increase productivity", "Reduce stress".
ACHIEVEMENT_STATUS varchar Status of the goal's achievement. Example: "Achieved", "Not Achieved", "In Progress".
EXPRESSED_BY varchar Information about who expressed the goal. Example: "John Doe", "Jane Smith", "Alice Johnson".
GOAL_PRIORITY varchar Priority level of the goal. Example: "High", "Medium", "Low".
STATUS_DATE datetime Date and time when the status of the goal was last updated. Example: "2021-09-01 12:34:56".
STATUS_REASON varchar Reason for the current status of the goal. Example: "Completed tasks", "Lack of resources", "Changed priorities".
isException tinyint Flag indicating if there is an exception (1) or not (0). Example: 0, 1.
exceptionMessage varchar Message describing the exception, if any. Empty if there is no exception.
sourcePriority int Priority of the source. Example: 1, 2, 3.
codeSystemPriority int Priority of the code system. Example: 1, 2, 3.
duplicatedBy int Identifier for the record that duplicates this record. Example: -1, 2, 3.
