EHI Export Schema
Table structdataoptions
Description This table stores the options for structured data along with their display order, deletion status, and other related information.
Column DataType Description
optionid (Primary Key) int This is auto-incremented primary key in This column . It is unique identifier for each option.
id int This column stores id of a question for which options are configured in This column . This column refers to id column in structdatadetail table.
deleteFlag smallint This column stores values 0/1 to indicate whether the option is marked as deleted or not. 0 - Option is not marked as deleted, 1 - Option is marked as deleted.
displayIndex int This column stores a number to indicate in which order option should be listed on User Interface.
name varchar This column stores value of an option.
measureid varchar This column stores value of the Quality measures measure id used for MU.
inactive tinyint This column stores values 0/1 to indicate whether the option is marked as active or not. 0 - Option is not marked as active, 1 - Option is marked as inactive.
