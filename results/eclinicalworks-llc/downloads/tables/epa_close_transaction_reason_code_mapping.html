EHI Export Schema
Table epa_close_transaction_reason_code_mapping
Description This table stores the mapping between EPA transactions and their corresponding reason codes, along with a delete flag to indicate if the mapping is active or deleted.
Column DataType Description
id (Primary Key) int This is the primary key for the table, an auto-incrementing integer that uniquely identifies each record in the table. Example: 1, 2, 3, etc.
epa_transaction_id int This is a foreign key that refers to the `id` column in the `epa_transaction` table. It represents the ID of the EPA transaction associated with the reason code. Example: 101, 102, 103, etc.
reason_code varchar This is a varchar column that stores the reason code associated with the EPA transaction. Example: 'RC001', 'RC002', 'RC003', etc.
delete_flag tinyint This is a tinyint column that indicates whether the mapping record is active or deleted. A value of '0' means the record is active, while a value of '1' means the record is deleted. Example: 0, 1
