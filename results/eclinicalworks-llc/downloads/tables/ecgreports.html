EHI Export Schema
Table ecgreports
Description This table stores information about ECG (Electrocardiogram) reports for patients, including file details, report dates, and review information.
Column DataType Description
reportId (Primary Key) int This is an auto incremented primary key of This column .
encId int This column stores ID of an encounter for which ECG report information of Patient's visit is documented. This column refers column encounterId from enc table.
patientId int This column stores ID of Patient for whom information is documented. This column refers column pid from patients table.
labId int This column stores id of the lab which has ECG reports associated with it. This column refers column reportId from labdata table.
labType int This column stores values 0 / 1 / 3 to indicate what type of order is placed. 0 - if a Lab is ordered , 1 - if a Diagnostic Imaging is ordered, 3 - if a Procedure is ordered.
filePath varchar This column stores report local file path.
fileName1 varchar This column stores File name for Pre ECG report.
fileName2 varchar Name of the second ECG report file, if applicable. (e.g., "ecg_report_2.pdf")
reportDate datetime Date when the ECG report was generated. (e.g., "2021-10-01")
reviewDate datetime Date when the ECG report was reviewed by a medical professional, if applicable. (e.g., "2021-10-02")
reviewedBy varchar Name of the medical professional who reviewed the ECG report. (e.g., "Dr. John Doe")
technician varchar Name of the technician who performed the ECG. (e.g., "Jane Smith")
delFlag int Integer indicating whether the ECG report has been deleted (1) or not (0). (e.g., 0)
FaxReportFileName1 varchar This column stores Fax File name for Pre ECG report.
FaxReportFileName2 varchar This column stores Fax File name for Post ECG report.
comments text Text containing any additional comments or notes about the ECG report. (e.g., "Patient experienced mild chest pain during the test.")
ftpdirpath varchar File path where the ECG report is stored on the FTP server, if applicable. (e.g., "/ftp/ecg_reports/patient_1001/")
pdfuploadsuccess int This column stores value 0/1 indicating FTP file upload status. 0 - Fail , 1 - Success.
sdkversion varchar This column stores information about sdk version when performing test.
