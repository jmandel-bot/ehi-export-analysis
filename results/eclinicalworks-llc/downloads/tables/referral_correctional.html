Table referral_correctional
Description This table is used as a mapping table for referrals created for patients in a corrections facility.
Column DataType Description
Id (Primary Key) int Unique identifier for each record in the referral_correctional table. Example: 1, 2, 3.
ReferralId int Unique identifier for each referral. This column is a foreign key that refers to the `Id` column in another table (e.g., `referrals` table). Example: 101, 102, 103.
BookAndCaseNo varchar This column stores the BookAndCaseNo of the patient. Its has reference to patients_correctional.BookAndCaseNo column.
DeleteFlag tinyint This column stores the value 1 or 0. Its used to determine if record is deleted.
Patients_Correctional_Id int This column stores reference to column patients_correctional.Id
