EHI Export Schema
Table struct_asc_anest_genexam
Description This table stores general examination data related to anesthesia for different encounters, categories, and items.
Column DataType Description
encounterid int An integer representing the unique identifier for an encounter.NOT NULL DEFAULT '0'. Example: 10234
catid int An integer representing the unique identifier for a category.NOT NULL DEFAULT '0' Example: 5
itemid int An integer representing the unique identifier for an item within a category. NOT NULL DEFAULT '0'.Example: 23
detailid int An integer representing the unique identifier for a specific detail within an item.NOT NULL DEFAULT '0'. Example: 67
VALUE text A text field storing the value associated with the specific detail. Example: "Normal"
valueid varchar A varchar(255) field storing a unique identifier for the value.NOT NULL DEFAULT ' '. Example: "val_123"
notes text A text field storing any additional notes related to the specific detail. Example: "Patient showed no signs of distress."
displayIndex tinyint A tinyint(1) field representing the display order of the details within an item.NOT NULL DEFAULT '0'. Example: 2
chartdisplay tinyint A tinyint(2) field indicating whether the detail should be displayed on a chart (1) or not (0).DEFAULT '1'. Example: 1
