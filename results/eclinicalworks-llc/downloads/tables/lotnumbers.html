EHI Export Schema
Table lotnumbers
Description This table stores lot numbers and other related information about immunizations.
Column DataType Description
Id (Primary Key) int This is an auto incremental primary key of This column .
lotnumber varchar This column is used to store lot number
manufacturer varchar It is used to store manufacturer's name
VISDate datetime This column stores Immunization VIS date in 'yyyy-mm-dd' format
ExpiryDate datetime This column stores lot number expiry date in 'yyyy-mm-dd' format
deleteflag char This column stores value 0 / 1 indicating whether entry is deleted. This is soft delete. 0 - Not Deleted , 1 -Deleted
NoOfDoses int This column is used to store number of doses a lot has
VaccineId int This column stores Vaccine Id. It refers to vaccineid of lot_immunizations table
dosage varchar It is used to store dosage for a lot
GovSupplied char This flag is used to indicate whether a lot is government supplied. Possible values for this are n, y, u etc
DosesRemaining decimal This is used to store remaining doses of the lot
Facility int This column stores facility id, primary key of edi_facilities table
inactive tinyint It is used to store a flag to indicate whether a lot is inactive
odhVac varchar This field is dependent on ODHFields itemkey (Ohio Dept of Health). If itemkey is set to yes, displays odhVac textbox on UI to place value for the field from user.
misc varchar This field is dependent on ODHFields itemkey (Ohio Dept of Health). If itemkey is set to yes, displays misc textbox on UI to place value for the transaction misc from user.
dosesGiven decimal Its data type is int and is used to store given doses
NdcCode varchar This column is used to store NDC code, Refers to the NDC chosen at the time of Lot Number creation
routeofAdminId int It is used to store route of administration
hl7id varchar This column stores External Control number of a patient that is received via interface
isSingleUse tinyint This flag is used to check if immunization is for single use
immInjType tinyint This column store immunization type as 1 for immunization or 2 for injection
Dose decimal The dose amount of the vaccine (e.g., 0.5).
unitOfMeasure int The unique identifier for the unit of measure of the vaccine dose (e.g., 1 for milliliters, 2 for grams). Foreign key relation: `unitOfMeasure` references `immunizationunitofmeasure(id)`.
