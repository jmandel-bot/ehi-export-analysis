Table occhealth_hs_status
Description This table stores information about the health screening status of patients in an occupational health context, including details about the patient, health screening, employer, and related dates.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
patientid int identifier for the patient associated with the health surveillance. Example: 101, 102, 103.
hsid int identifier for the health surveillance associated with the patient and employer. Example: 201, 202, 203.
employerid int identifier for the employer associated with the patient and health surveillance. Example: 301, 302, 303.
status tinyint A boolean value representing the status of the health screening (0 for incomplete, 1 for complete). Example: 0, 1.
nextScreeningid int identifier for the next health screening scheduled for the patient. Example: 204, 205, 206.
nextDueDate datetime The date when the next health screening is due for the patient. Example: '2022-12-01', '2023-01-15'.
actionDate datetime The date when the health screening action was taken or is planned to be taken. Example: '2022-11-01', '2022-11-15'.
resultid int identifier for the result associated with the health screening. Example: 401, 402, 403.
delFlag tinyint A boolean value representing whether the record is deleted (0 for not deleted, 1 for deleted). Example: 0, 1.
encid int identifier for the encounter associated with the health screening. Example: 501, 502, 503.
