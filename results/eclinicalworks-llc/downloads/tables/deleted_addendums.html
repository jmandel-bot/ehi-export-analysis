EHI Export Schema
Table deleted_addendums
Description This table stores information about addendums that have been deleted from encounters.
Column DataType Description
encounterid int The unique identifier of the encounter associated with the deleted addendum. It is an integer value and cannot be NULL. Example: 12345
userid int The unique identifier of the user who deleted the addendum. It is an integer value and cannot be NULL. Example: 6789
createddate datetime The date when the addendum was created. It is a date value and can be NULL. Example: '2021-09-01'
createdtime datetime The time when the addendum was created. It is a time value and can be NULL. Example: '14:30:00'
addendum text The content of the deleted addendum. It is a text value and cannot be NULL. Example: 'This is a deleted addendum.'
deleteddate datetime The date when the addendum was deleted. It is a date value and can be NULL. Example: '2021-09-02'
deletedtime datetime The time when the addendum was deleted. It is a time value and can be NULL. Example: '15:45:00'
