Table dischargeplan_corrections
Description This table stores information about discharge plans for patients in a correctional facility, including details about the planned discharge date, status, and facilities involved in the discharge process.
Column DataType Description
id (Primary Key) int This column is a Primary key.
patientsCorrectionalId int This column stores the patient's correctional id from patients_correctional.id column.
dischargeOpenDate datetime This column store the date of creation of Discharge plan.
plannedDischargeDate datetime This column store the planned date of Discharge plan.
dischargeStatus tinyint This column store the status of Discharge plan.
assignedTo int This column store the user id to whom the Discharge plan is assigned.
internalNotes text This column store the notes added in Discharge plan.
finalNotes text This column store the final notes of Discharge plan.
deleteflag tinyint This column store the delete flag value of Discharge plan.
dischargeType int This column store the type of Discharge plan. 0 - Facility Transfe, 1- Final Discharge.
facilityfrom int This column store the current facility id of patient's Discharge plan.
facilityto int This column store the facility id of patient's to which patient is going to be transferred in Discharge plan.
FinalBookoutDate datetime This column store the date and time of patient's final discharge in Discharge plan.
eidflag tinyint This column store the EID flag coming through interface.
plannedDischargeTime datetime This column store the planned time of Discharge plan.
