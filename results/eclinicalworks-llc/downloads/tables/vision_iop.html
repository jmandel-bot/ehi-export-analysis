EHI Export Schema
Table vision_iop
Description This table stores the values related to intraocular pressure tests for eyes.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key.
Encounterid int This column stores the encounter id for which IOP data is being documented. This column refers to the column encounterid from the enc table.
OD varchar Test result for the right eye (Oculus Dexter). Example: "20/20", "20/40"
OS varchar Test result for the left eye (Oculus Sinister). Example: "20/20", "20/40"
Test varchar Name or description of the vision test. Example: "Visual Acuity", "Color Vision"
TestTime varchar Time when the test was conducted. Example: "10:30", "14:45"
Category varchar Category of the vision test. Example: "Routine", "Special"
DisplayIndex int Index for displaying the record in a list or table. Example: 1, 2, 3
AOD varchar Additional test result for the right eye. Example: "20/30", "20/50"
AOS varchar Additional test result for the left eye. Example: "20/30", "20/50"
OD2 varchar This field is not in use.
OS2 varchar This field is not in use.
OD3 varchar This field is not in use.
OS3 varchar This field is not in use.
OD4 varchar This field is not in use.
OS4 varchar This field is not in use.
avgod varchar This field is not in use.
avgos varchar This field is not in use.
comments text Text field for additional comments or notes about the test. Example: "Patient had difficulty with color vision test."
rejectedflagOD tinyint This field is not in use.
rejectedflagOS tinyint This field is not in use.
rejectedflagOD2 tinyint This field is not in use.
rejectedflagOS2 tinyint This field is not in use.
rejectedflagOD3 tinyint This field is not in use.
rejectedflagOS3 tinyint This field is not in use.
rejectedflagOD4 tinyint This field is not in use.
rejectedflagOS4 tinyint This field is not in use.
testtype varchar Type of test conducted, such as "NC" for non-contact. Example: "NC"
drug_time varchar Time when the drug was administered. Example: "09:45", "13:30"
drug_batch varchar Batch number of the drug used. Example: "B12345", "B67890"
drug_name varchar Name of the drug used. Example: "Tropicamide", "Phenylephrine"
drug_expirydate varchar Expiry date of the drug used. Example: "2023-12-31", "2024-06-30"
deleteFlag tinyint Flag indicating if the record has been deleted (1) or not (0). Example: 0, 1
targetod tinyint Target value for the right eye test result. Example: 20, 30.
targetos tinyint Target value for the left eye test result. Example: 20, 30.
userid int identifier for the user who conducted the test. Foreign key relation: `users.UID`. Example: 1001, 1002, 1003
