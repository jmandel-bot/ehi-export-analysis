EHI Export Schema
Table validcpts
Description This table stores the valid cpt's details.
Column DataType Description
id (Primary Key) int This is the primary key column, containing a unique integer identifier for each record in the table. Example: 1, 2, 3, etc.
cptcode varchar This column stores the CPT code as a string of up to 5 characters. Example: "12345", "A1234".
cptName varchar This column stores the name or description of the CPT code as a string of up to 255 characters. Example: "Office Visit", "Surgical Procedure".
year int This column stores the year associated with the CPT code as an integer of 4 digits. Example: 2006, 2021.
type varchar This column stores the type of the CPT code as a string of up to 10 characters. Example: "surgery", "diagnostic".
deleteFlag tinyint This column stores flags 0/1 to indicate whether the CPT is marked as active or inactive. 0 - Active, 1 - Inactive.
ValidFrom datetime This column stores the starting date of the validity period for the CPT code as a date. Example: '2021-01-01', '2022-07-01'.
ValidTo datetime This column stores the ending date of the validity period for the CPT code as a date. Example: '2021-12-31', '2023-06-30'.
