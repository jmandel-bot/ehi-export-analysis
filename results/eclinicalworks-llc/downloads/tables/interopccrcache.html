EHI Export Schema
Table interopccrcache
Description This table contains merged CCR received from ihub to emr when patient request data from commonweal/care equality network using patient arrival or advanced trigger planning.
Column DataType Description
id (Primary Key) int This is the primary key for the table, an auto-incrementing integer that uniquely identifies each record in the table. Example: 1, 2, 3.
doctorid int This column represent the user id who requested data from care quality/commonwell. It refers to user.UID column. But application is not populating the column value and not using anywhere.
patientid int This contains patient id for whom data was requested from ihub. It refers to user.UID column.
ecwmpi varchar This column is not in use. Dead column.
mergedccr text This column contains merged ccr received from ihub.
timestamp datetime This column contains date and time when entry was made into This column .
delflag tinyint This is a tinyint(2) column with a default value of '0'. It is used to mark a record as deleted without actually removing it from the table. A value of '1' indicates that the record has been deleted. Example: 0, 1.
