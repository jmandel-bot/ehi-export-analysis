EHI Export Schema
Table asc_enc_custom_tpl_notes
Description This table stores custom template notes for specific encounters, along with their categories, properties, and versions. It also includes information about the creation and modification of the notes, as well as flags for deletion and answer strings.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
encounterid int Stores integer values.
catid int Stores integer values. Default "0" if no value provided
propid int Stores integer values.
versionid int Stores integer values with Not NULL condition
notes text Text field containing the content of the custom template note. Example: "Patient showed improvement in symptoms."
deleteFlag tinyint A boolean flag indicating whether the record has been deleted (1) or not (0). Example: 0, 1.
createdBy int Stores integer values with Not NULL condition
createdDateTime datetime Date and time when the custom template note was created. Example: '2021-09-01 12:34:56'.
modifiedBy int Stores integer values with Not NULL condition
modifiedDateTime datetime time stamp of the last modification to the custom template note. Example: '2021-09-02 14:56:32'.
isAnswerString tinyint A boolean flag indicating whether the record contains an answer string (1) or not (0). Example: 0, 1.
answerString text Text field containing the answer string associated with the custom template note, if applicable. Example: "Yes, the patient is improving."
