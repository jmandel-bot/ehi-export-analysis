EHI Export Schema
Table claimstatus
Description This table stores information about the status of claims, including details about the file, import, and deletion.
Column DataType Description
Fileid (Primary Key) int Unique identifier for each claim status record. Example: 1, 2, 3.
ImportedBy int identifier for the user who imported the claim status. Example: 101, 102, 103.
ImportedDate datetime time stamp of when the claim status was imported. Example: '2021-08-01 12:34:56'.
RawFileName varchar The original file name of the claim status. Example: 'claim_status_20210801.txt'.
XML varchar The XML file name associated with the claim status. Example: 'claim_status_20210801.xml'.
ISA13 varchar The ISA13 value from the claim status file. Example: '000000123'.
ISA06 varchar The ISA06 value from the claim status file. Example: '987654321'.
ClearingName varchar The name of the clearinghouse associated with the claim status. Example: 'Clearinghouse A'.
DeleteFlag tinyint A flag indicating if the claim status has been deleted (1) or not (0). Example: 0, 1.
DeletedBy int identifier for the user who deleted the claim status. Example: 201, 202, 203.
DeletedDate datetime time stamp of when the claim status was deleted. Example: '2021-08-15 10:20:30'.
DeletedReason text Text description of the reason for deleting the claim status. Example: 'Duplicate entry'.
GS06 varchar The GS06 value from the claim status file. Example: '123456789'.
BHT03 varchar The BHT03 value from the claim status file. Example: '20210801000001'.
ST02 varchar The ST02 value from the claim status file. Example: '0001'.
FilePosted tinyint A flag indicating if the claim status file has been posted (1) or not (0). Example: 0, 1.
GS08 varchar The GS08 value from the claim status file. Example: '005010X221A1'.
