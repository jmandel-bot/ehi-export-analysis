Table todo_list_comment
Description This table stores comments related to a patient's To Do List, including information about the patient, the comment itself, and the user who documented the comment.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer. It uniquely identifies each comment in the table. Example: 1, 2, 3, etc.
patient_id int This is a foreign key that refers to the UID column in the user table. It represents the patient associated with the comment. Example: 101, 102, 103, etc.
notes varchar This is a text column that stores the actual content of the comment. Example: Patient needs to schedule a follow-up appointment.
documented_on datetime This is a date time column that stores the date and time when the comment was documented. Example: 2021-09-15 14:30:00.
documented_by int This is a foreign key that refers to the UID column in the user table. It represents the user who documented the comment. Example: 201, 202, 203, etc.
deleteflag tinyint This is a tinyint column that indicates whether the comment has been deleted or not. A value of 0 means the comment is not deleted, while a value of 1 means the comment is deleted. Example: 0, 1
