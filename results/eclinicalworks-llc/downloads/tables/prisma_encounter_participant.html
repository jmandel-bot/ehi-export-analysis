EHI Export Schema
Table prisma_encounter_participant
Description This table stores information about participants involved in a specific encounter, including their type, period of participation, and individual details.
Column DataType Description
ID (Primary Key) int Unique identifier for each record in the prisma_encounter_participant table. Example: 1, 2, 3.
ENCOUNTER_DETAIL_ID bigint Foreign key referring to the `prisma_encounter_details` table, representing the specific encounter the participant is involved in. Example: 101, 102, 103. Foreign Key Relation: `prisma_encounter_details.id`
PARTICIPANT_TYPE_CODE varchar A code representing the type of participant. Example: "doctor", "nurse", "patient".
PARTICIPANT_TYPE_DISPLAY varchar A human-readable display of the participant type. Example: "Doctor", "Nurse", "Patient".
PARTICIPANT_TYPE_SYSTEM varchar The system used to define the participant type. Example: "http://hl7.org/fhir/participant-type".
PARTICIPANT_TYPE_TEXT varchar A textual description of the participant type. Example: "A medical doctor", "A registered nurse", "An individual receiving care".
PARTICIPANT_PERIOD_START datetime The start date and time of the participant's involvement in the encounter. Example: "2021-09-01 09:00:00".
PARTICIPANT_PERIOD_END datetime The end date and time of the participant's involvement in the encounter. Example: "2021-09-01 17:00:00".
PARTICIPANT_INDIVIDUAL_DISPLAY varchar A human-readable display of the participant's individual information. Example: "Dr. John Doe", "Nurse Jane Smith", "Patient Alice Johnson".
PARTICIPANT_INDIVIDUAL_ID varchar A unique identifier for the participant's individual information. Example: "12345", "67890", "24680".
