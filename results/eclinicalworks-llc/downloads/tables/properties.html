EHI Export Schema
Table properties
Description Stores properties which are used in area of EMR like it's acting as line items for HPI and Examination sections in Progress Notes
Column DataType Description
propID (Primary Key) int This is an auto incremented primary key of This column .
NAME varchar This column stores free text property name.
description varchar This column stores free text description of the property name.
type varchar This column stores value 1 / 2 / 3 / 4 property type from pre-defined values in user interface. 1 - Integer , 2 -Text, 3 - Date , 4 -Money.
viewtype varchar This column stores view type of the property. Like TextBox, Dropdown, Listbox, Checkbox.
size varchar This column stores length (size) of the property based on view type.
displayName varchar This column stores display name of the property.
pdetailID int This column stores the property detail id. This column refers to pdetailid column in propertydetail table.
displayIndex smallint This column stores a digit to indicate order in which properties should be listed in user interface.
delPropFlag int This column stores value 0 / 1 indicating if the property is deleted, this is soft delete. 0 - Not Deleted , 1 -Deleted
vmid varchar This column stores a string as unique identifier for each row.
isConfidential tinyint This column stores value 0 / 1 indicating if the property is configured as confidential. 0 - Property is not confidential , 1 -Property is confidential
