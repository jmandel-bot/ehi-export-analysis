EHI Export Schema
Table apptrecurrencemapping
Description This table stores the mapping between appointment encounters and their recurrence patterns.
Column DataType Description
ID (Primary Key) int This is the primary key for the table, which is an integer that uniquely identifies each record in the table. It is auto-incremented, meaning that its value is automatically increased for each new record. Example: 1, 2, 3, etc.
encounterid int This column stores the integer ID of the appointment encounter. It is a foreign key that refers to the `ID` column in the `encounters` table. Example: 101, 102, 103, etc.
recurrencepatternid int This column stores the integer ID of the recurrence pattern. It is a foreign key that refers to the `ID` column in the `ApptRecurrenceWeeklyPattern` table. Example: 1, 2, 3, etc.
recurrencepattern int This column stores the integer value representing the type of recurrence pattern for the appointment. Example: 2 (could represent a specific recurrence pattern, such as daily or weekly).
