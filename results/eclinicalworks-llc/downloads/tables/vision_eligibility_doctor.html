Table vision_eligibility_doctor
Description This table stores information about doctors who are eligible for vision-related services, including their personal and professional details, and contact information.
Column DataType Description
id (Primary Key) int Unique identifier for each doctor in the table. Example: 1, 2, 3.
eligid int Unique identifier for the doctor's eligibility record. Example: 1001, 1002, 1003.
LastName varchar Last name of the doctor. Example: "Smith", "Johnson", "Doe".
FirstName varchar First name of the doctor. Example: "John", "Jane", "Michael".
NPI varchar National Provider identifier of the doctor. Example: "1234567890", "0987654321".
Taxid varchar Tax identification number of the doctor. Example: "123456789", "987654321".
DocProvider varchar Doctor's provider information. Example: "Provider A", "Provider B", "Provider C".
ExamAuthorization varchar Authorization code for the doctor to perform vision exams. Example: "AUTH123", "AUTH456", "AUTH789".
AddressLine1 varchar First line of the doctor's address. Example: "123 Main St", "456 Elm St".
AddressLine2 varchar Second line of the doctor's address (optional). Example: "Apt 4B", "Suite 200".
City varchar City where the doctor is located. Example: "New York", "Los Angeles", "Chicago".
State varchar State where the doctor is located. Example: "NY", "CA", "IL".
ZipCode varchar Zip code of the doctor's location. Example: "10001", "90001", "60601".
Country varchar Country where the doctor is located. Example: "USA", "Canada", "UK".
PhoneNumber varchar Contact phone number of the doctor. Example: "(555) 123-4567", "(555) 987-6543".
deleteflag int Flag to indicate if the record has been deleted (0 for active, 1 for deleted). Example: 0, 1.
