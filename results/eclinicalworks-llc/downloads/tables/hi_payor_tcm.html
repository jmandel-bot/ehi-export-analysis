EHI Export Schema
Table hi_payor_tcm
Description This table stores information about payor transactions related to TCM (Transitional Care Management) services for patients.
Column DataType Description
id (Primary Key) bigint A unique identifier for each record in the table. Example: 1, 2, 3.
patient_id int The identifier of the patient associated with the TCM service. This is a foreign key that refers to the `users` table's `UID` column. Example: 101, 102, 103.
txn_id varchar A unique identifier for the transaction. Example: "TXN123", "TXN124", "TXN125".
hi_txn_id varchar A unique identifier for the health insurance transaction. Example: "HI_TXN12345", "HI_TXN12346", "HI_TXN12347".
tcm_UID varchar A unique identifier for the TCM service. Example: "TCM_UID001", "TCM_UID002", "TCM_UID003".
tcm_dob date The date of birth of the patient receiving the TCM service. Example: "1990-01-01", "1985-05-15", "1978-12-20".
payor_tcm_id varchar A unique identifier for the payor related to the TCM service. Example: "PAYOR_TCM001", "PAYOR_TCM002", "PAYOR_TCM003".
tcm_adt_message varchar The ADT (Admission, Discharge, and Transfer) message related to the TCM service. Example: "Admitted to hospital", "Discharged from hospital", "Transferred to another facility".
document_id varchar An optional identifier for a document related to the TCM service. Example: "DOC001", "DOC002", "DOC003".
tcm_ref_id int A reference identifier for the TCM service. Example: 1001, 1002, 1003.
payor_id varchar The identifier for the payor associated with the TCM service. Example: "PAYOR001", "PAYOR002", "PAYOR003".
created_date datetime The date and time when the record was created. Example: "2021-01-01 12:00:00", "2021-02-15 14:30:00", "2021-03-20 10:45:00".
updated_date datetime The date and time when the record was last updated. Example: "2021-01-02 12:00:00", "2021-02-16 14:30:00", "2021-03-21 10:45:00".
deleteflag bit A flag indicating whether the record has been deleted (1) or not (0). Example: 0, 1.
