EHI Export Schema
Table ccdbatchdataportability
Description This table stores information about CCD batches of data portability requests, including their status, file names and date ranges.
Column DataType Description
Batchid (Primary Key) int This column stores primary batch id of Data Export.
BatchRequestedOn datetime This column stores requested date and time of the batch.
batchstatus smallint This column maintains batch status for batch. Possible values - 0 - Pending 1 - Success 2 - SAAS failed 3 - Failed
BatchFolderName varchar This column stores FTP folder name where CCD's zip file will be available.
BatchFileName varchar This column stores CCD's zipped file name
ccdatype int It's depenedent on this itemkey "DataPortabilityDefaultCCDAType"
cheferror varchar This column stores chef's error information
apUID varchar This column stores apUID of the emr
userid int This column stores userid(UID of user table) of the user requested the batch.
allPatientFlag int This column is set when all patients is selected on the patient selection screen. 0 - Some Patients selected, 1 - All Patients Selected.
rangeFromdate datetime The starting date of the date range for the batch, stored as a datetime value. For example: '2021-08-01 00:00:00'.
rangeTodate datetime The ending date of the date range for the batch, stored as a datetime value. For example: '2021-08-31 23:59:59'.
scheduleText varchar This column stores schedule text information(Run now, Recurring or Run Once)
encFromDate datetime This column stores the encounter from date of the ccd data portability batch
encToDate datetime This column stores the encounter to date of the ccd data portability batch
jobSchDate datetime This column stores job scheduled date time
