Table allergy_structdata
Description This table stores structured data related to the Allergy module, including names, values, types, and whether they are active or inactive.
Column DataType Description
id (Primary Key) int The column stores an id of the allergy structure data. This is an auto incremental primary key of This column .
NAME varchar This column stores the name of the structure data.
VALUE varchar This column stores the value of the structure data.
TYPE varchar This column stores the type of the structure data.
MANUFACTURERid int This is an integer column that stores the id of the manufacturer related to the allergy. It can be NULL. This column is a foreign key that refers to the `id` column in the `allergy_manufacturers` table. Example: 1, 2, 3, etc.
INACTIVEFLAG tinyint This is a tinyint(3) unsigned column that stores a flag indicating whether the allergy is inactive (1) or active (0). It cannot be NULL and has a default value of '0'. Example: 0 (active), 1 (inactive).
