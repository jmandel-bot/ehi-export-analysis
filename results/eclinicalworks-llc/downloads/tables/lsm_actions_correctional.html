Table lsm_actions_correctional
Description This table stores actions associated with patients for the corrections module.
Column DataType Description
id (Primary Key) int This is the primary key column, which uniquely identifies each record in the table. It is an integer value that is automatically incremented with each new entry. Example: 1, 2, 3, etc.
actionid int This column stores id of action.
patient_correctional_id int This column stores contains patient correctional id.
