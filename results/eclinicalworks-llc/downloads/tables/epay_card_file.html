EHI Export Schema
Table epay_card_file
Description This table stores information about token information for COF data and cards masked information by patients in a healthcare facility.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table.
patientid int identifier for the patient associated with the card, refers to the `pid` column in the `patients` table.
facilityid int identifier for the facility where the card is used, refers to the `id` column in the `edi_facilities` table.
cardnumber varchar This column stores the first digit and last 4 digits of the card number. Example 4xxxxxxxxxxx1234
nameoncard varchar The name of the cardholder as it appears on the card (e.g., John Doe).
expmonth varchar The expiration month of the card (e.g., 12 for December).
expyear varchar The expiration year of the card (e.g., 2025).
address varchar The billing address associated with the card (e.g., 123 Main St).
zipcode varchar The billing zip code associated with the card (e.g., 12345).
recurringid varchar identifier for recurring payments associated with the card.
updateduser int identifier for the user who last updated the record, refers to the `id` column in the `users` table.
datecreated datetime The date and time when the record was created (e.g., 2021-01-01 12:34:56).
primecard varchar Indicates if the card is the primary card for the patient (e.g., 'yes' or 'no').
verify varchar Verification status of the card (e.g., 'yes' or 'no').
referenceid varchar Reference identifier for the card, used for external systems.
maxamount decimal The maximum amount that can be charged to the card (e.g., 500.00).
maxdate varchar The maximum date until which the card can be charged (e.g., 2023-12-31).
isdeleted int Indicates if the record has been deleted (e.g., 0 for not deleted, 1 for deleted).
maxyears varchar The maximum number of years the card can be charged (e.g., 2).
encounterid int identifier for the encounter associated with the card, refers to the `id` column in the `encounters` table.
cardtype varchar The type of the card (e.g., 'VISA', 'MC', 'AMEX', 'DISC').
emailid varchar The email address associated with the card (e.g., johndoe@example.com).
reasonid int identifier for the reason associated with the card, refers to the `id` column in the `reasons` table.
docid int identifier for the document associated with the card, refers to the `docid` column in the `documents` table.
ecliniformid int identifier for the electronic clinical form associated with the card, refers to the `id` column in the `ecliniforms` table.
dateupdated datetime The date when the record was last updated (e.g., 2021-02-01).
datedeleted datetime The date when the record was deleted (e.g., 2021-03-01).
autobillcarddays int The number of days between automatic billing cycles for the card (e.g., 30).
datedeleteUID int identifier for the user who deleted the record, refers to the `UID` column in the `users` table.
dateupdatedUID int identifier for the user who updated the record, refers to the `UID` column in the `users` table.
