EHI Export Schema
Table derm_diagramregions
Description This table stores information about dermatology diagram regions, including the region's name, gender, and any custom region names.
Column DataType Description
id int This column stores unique id for smaller region on body parts.
regionid int This column stores region id for location on human body
regionname text This column stores region name for location on human body. Example values: 'Left Medial Knee', 'Left Lower Quadrant of Abdomen', 'Mid Upper Abdomen'.
diagramid int This column stores id for human body parts. This refers to column diagramid from table derm_diagrams.
gender char This column stores the gender for which the diagram region is applicable. It can have a maximum of 3 characters. Example values: 'M', 'F', 'U' (where 'M' stands for Male, 'F' for Female, and 'U' for Unisex).
custom_regionname varchar This column stores region name for location on human body. Example values: 'Left Medial Knee', 'Left Lower Quadrant of Abdomen', 'Mid Upper Abdomen'.
