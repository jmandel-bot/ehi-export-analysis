EHI Export Schema
Table ccdbatchdataportabilityencinfo
Description This table stores the most recent encounter ID, CCDA generation status and generation time for every patient.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
Batchid int This column stores primary batch id from ccdbatchdataportability table.
Encounterid int This column stores encounter id of the transaction.
CCDGenerated int This column maintains CCDA generated status for a patient's encounters. Possible values: 0 - Processing 1 - Success 3 - Failed
Patientid int This column stores patient id of the transaction.
ts datetime This column stores date and time of the transaction.
