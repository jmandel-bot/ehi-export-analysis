Table surescript_changerxoptions
Description This table stores data related to a changed medication requested from a pharmacy as well as information about payer benefits used by the pharmacy.
Column DataType Description
Reqid (Primary Key) int Unique identifier for each change request record (e.g., 1, 2, 3).
Seqid int Sequence identifier for the alternatives requested as part of the change request
Messageid varchar Acts as a foreign key to associate alternatives requested and the change request for which it has been received and stored in surescript_requestlog.
ChangeRxType char Type of change request (e.g., 'A1', 'B2').
DrugDescription varchar Description of the requested alternative
ProductCode varchar Product identifiers for drugs
ProductCodeQualifier varchar Qualifier for the product code (e.g., 'NDC', 'UPC').
DosageForm varchar Form of the drug dosage (e.g., 'Tablet', 'Capsule').
Strength varchar Strength of the drug (e.g., '200mg', '500mg').
StrengthUnits varchar Units of the drug strength (e.g., 'mg', 'mcg').
DrugDBCode varchar This column stores id of a medication in the associated drug database sent for the sent eprescription.
DrugDBCodeQualifier varchar This column stores the qualifier for Drug database associated. MD- Medispan, - Multum
duration varchar This column stores information denoting how long the prescribed medication has to be taken by the patient.
Quantity varchar Quantity of the drug prescribed (e.g., '30', '60').
QuantityQualifier varchar Qualifier for the quantity (e.g., 'TAB', 'CAP').
Take varchar Instructions for taking the drug (e.g., 'Take 1 tablet every 4 hours', 'Take 2 capsules daily').
Refills int Number of refills allowed for the prescription (e.g., 1, 2).
RefillsQualifier char Qualifier for the refills (e.g., 'PRN', 'NRF').
SubsMedAllowed int Indicates if substitution of medication is allowed (1 for yes, 0 for no).
Comments varchar Additional comments related to the change request (e.g., 'Patient prefers brand name', 'Generic not available').
WrittenDate varchar Date the prescription was written (e.g., '2021-01-01', '2021-02-15').
LastFillDate varchar Date of the last prescription fill (e.g., '2021-01-30', '2021-02-28').
PriRxDiagnosisQualifier varchar Qualifier for the primary prescription diagnosis (e.g., 'ICD10', 'SNOMED').
PriRxDiagnosisValue varchar Value of the primary prescription diagnosis (e.g., 'M54.5', 'G43.1').
ClinicalInformationQualifier varchar Qualifier for the clinical information (e.g., 'ICD10', 'SNOMED').
PriorAuthQualifier char This column stores prior authorization qualifier. (No information on data, data type, size and Default value)
PriorAuthValue varchar This column stores prior authorization value. (No information on data type, size and Default value)
DEASchedule varchar This column stores type of a schedule of the corresponding prescribed control substance.
PayerName varchar This column stores Payer Name of Insurance. (No information on data type, size and Default value)
Cardholderid varchar Unique identifier for the cardholder (e.g., 'CH12345', 'CH67890').
Groupid varchar This column stores Insurance Group id. (No information on data type, size and Default value)
CardholderName varchar Name of the cardholder (e.g., 'John Doe', 'Jane Smith').
Payorid varchar This column stores Payor id of Insurance. (No information on data type, size and Default value)
PlanNetidNo varchar This column stores Plan id of Insurance. (No information on data type, size and Default value)
EffectiveDate datetime Effective date of the change request (e.g., '2021-03-01', '2021-04-01').
