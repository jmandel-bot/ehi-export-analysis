EHI Export Schema
Table allergy_extracts_icdcodes
Description This table stores information about allergy extracts and their associated ICD codes.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
EXTid int Unique identifier for the allergy extract. Example: 101, 102, 103.
BOTTLEid int Unique identifier for the bottle containing the allergy extract. Default value is '0'. Example: 1, 2, 3.
BOTTLE_NAME varchar Name of the bottle containing the allergy extract. Example: 'Bottle A', 'Bottle B', 'Bottle C'.
RECIPE_NAME varchar Name of the recipe for the allergy extract. Example: 'Recipe 1', 'Recipe 2', 'Recipe 3'.
ICDCODE1 varchar First ICD code associated with the allergy extract. Example: 'J30.1', 'J30.2', 'J30.3'.
ICDCODE1ITEMid int Unique identifier for the first ICD code item. Example: 1001, 1002, 1003.
ICDCODE2 varchar Second ICD code associated with the allergy extract. Example: 'J30.4', 'J30.5', 'J30.6'.
ICDCODE2ITEMid int Unique identifier for the second ICD code item. Example: 2001, 2002, 2003.
ICDCODE3 varchar Third ICD code associated with the allergy extract. Example: 'J30.7', 'J30.8', 'J30.9'.
ICDCODE3ITEMid int Unique identifier for the third ICD code item. Example: 3001, 3002, 3003.
ICDCODE4 varchar Fourth ICD code associated with the allergy extract. Example: 'J31.0', 'J31.1', 'J31.2'.
ICDCODE4ITEMid int Unique identifier for the fourth ICD code item. Example: 4001, 4002, 4003.
