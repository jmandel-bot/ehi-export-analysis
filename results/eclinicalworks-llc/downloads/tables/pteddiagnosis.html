EHI Export Schema
Table pteddiagnosis
Description This table stores information about patient education diagnosis, including details about the diagnosis, content, and related patient and encounter data and also have url of the Patient education that was publish to portal.
Column DataType Description
id (Primary Key) int Primary key of This column and Unique identifier for each record in the table. (Example: 1, 2, 3).
encid int This column stores id of the encounter on which the patient education was published by the end user. This column refers to encounterid column in enc table.
patientid int This column stores id of the patient for which patient education is logged by the end user. This column refers to UID column in user table.
encDate datetime This column stores the date of the encounter where the patient education was published. Date is stored in YYYY-MM-DD format.
reason text Reason defined on the encounter where the patient education was published.  This column refers to reason column in enc table.
description varchar Title of the patient education material that was published to the portal
contentid varchar Contentid for the patient education published to the portal defined by the vendor
delflag tinyint This column stores value 0 / 1 indicating if the added logged item is deleted. 
websync tinyint This column stores value 0 / 1 indicating if the logged item has been synced with the web portal. 1="Yes", 0="No"
url text URL for the patient education material that was published to the portal (Example: 'https://example.com/content1', 'https://example.com/content2')
edType tinyint Stores the patient education vendor id for the patient education that was printed: 0="ADAM Quicksheets" (No longer in use), 1 = "Exit-Writer" (No longer in use), 2="Krames" (No longer in use), 3="Healthwise", 4="Custom Education" (only available in the exe), 5="Krames Staywell", 6="ADAM Smartcare", 7="Elsevier Exitcare", 8="Santovia"
contentTypeid varchar Not currently being used by active patient education integrations. Previously was used by Pt Ed type 1 and 2
publishdate datetime This column stores the date & time that the patient education was published.  The format used is YYYY-MM-DD HH:mm:ss
