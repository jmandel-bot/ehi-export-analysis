EHI Export Schema
Table enctaxids
Description This table stores information about tax IDs associated with different encounters.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the enctaxids table. Example: 1, 2, 3.
enccounterid int The identifier for the encounter associated with the tax ID. This is a foreign key that refers to the `id` column in the `encounters` table. Example: 101, 102, 103.
taxid varchar The tax ID value. Example: "123-45-6789", "987-65-4321".
taxidtype varchar The type of tax ID. Example: "SSN", "EIN".
deleteflag tinyint A flag indicating whether the record is deleted or not. A value of 0 means the record is not deleted, and a value of 1 means the record is deleted. Example: 0, 1.
modifieddate datetime The date and time when the record was last modified. Example: "2021-09-15 10:30:00", "2021-09-16 14:45:00".
enctype int The type of encounter associated with the tax ID. Example: 1, 2, 3.
