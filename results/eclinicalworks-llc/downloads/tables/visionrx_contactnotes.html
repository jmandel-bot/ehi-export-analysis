Table visionrx_contactnotes
Description This table stores information related to contact lens prescriptions, including wear schedules, replenishment details, care regimens, and other relevant notes.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer value. It uniquely identifies each record in the table. Example: 1, 2, 3, etc.
VisionRxid int This column contains a foreign key that refers to the `id` column in the `visionrx` table. It represents the unique identifier for the associated vision prescription. Example: 101, 102, 103, etc.
Encounterid int Foreign key that refers to the `encounterid` column in the `enc` table. It represents the unique identifier for the associated patient encounter. Example: 201, 202, 203, etc.
WearSchedule text This column contains a text description of the contact lens wear schedule. Example: "Daily wear", "Extended wear", "Continuous wear", etc.
Replenishment text This column is a text description of the contact lens replenishment schedule. Example: "Monthly", "Bi-weekly", "Quarterly", etc.
DOT text This column is a text description of the dispensing optician or technician's notes. Example: "Patient prefers daily disposable lenses", "Patient has dry eyes", etc.
CareRegiment text This column contains a text description of the recommended care regimen for the contact lenses. Example: "Use multipurpose solution", "Use hydrogen peroxide solution", "Use daily cleaner and conditioning solution", etc.
Notes text Additional text notes related to the contact lens prescription. Example: "Patient has astigmatism", "Patient is new to contact lenses", etc.
lenticular varchar A varchar (25) column that stores information about the lenticular design of the contact lens. Example: "Standard", "Custom", "Aspheric", etc.
edgelift varchar A varchar(25) column that stores information about the edge lift of the contact lens. Example: "Low", "Medium", "High", etc.
