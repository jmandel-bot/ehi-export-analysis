EHI Export Schema
Table vision_order_pair
Description This table stores information about vision order pairs (spectacles), including details about pricing, promotions, taxes, and other related data.
Column DataType Description
id (Primary Key) int Unique identifier for each vision order pair.
order_id int This column stores the reference order ID from the vision_order table.
dispense_pair varchar This column stores an identifier for this spectacle order. This value is a combination of encounter ID and pair value. If this precedes with 'R' that indicates it is a remake order otherwise it's a new order.
pair varchar This columns stores spectacle order number denoted by any letter from A to Z.
unitprice decimal This column stores the unit amount in decimal format for this pair order.
totalprice decimal This column stores the final amount in decimal format for this pair order.
isdeleted tinyint A tiny integer indicating whether the vision order pair has been deleted (1) or not (0).
createdby int Identifier for the user who created the vision order pair.
createdon datetime This column stores the time stamp when this pair order is created. The format used is YYYY-MM-DD HH:MM:SS.
updatedby int This column stores the logged in user id who modifies this pair order. This column refers column UID from user table.
updatedon datetime This column stores the time stamp when this pair order is modified. The format used is YYYY-MM-DD HH:MM:SS.
dispense_type varchar This column stores the type of dispense: New or Remake.
promotionamount decimal This column stores the discount amount in decimal format for this pair order.
taxamount decimal This column stores the tax amount in decimal format for this pair order.
inspectedby int This column stores the ID of the logged in user who performs the inspection on this pair order. This column refers to column UID from the user table.
inspecteddate datetime This column stores the time stamp when this pair order was inspected. The format used is YYYY-MM-DD HH:MM:SS.
collectedby int This column stores the ID of the patient who collects this pair order. This column refers to column UID from the user table. If this repair order is collected by any other person apart from patient then value 0 is inserted to this column.
collectednotes text Text notes related to the collection of the vision order pair.
status int An integer representing the status of the vision order pair, e.g., 0.
dispense_id int Identifier for the dispense associated with the vision order pair.
deliveredby int Identifier for the user who delivered the vision order pair.
delivereddate datetime The date and time when the vision order pair was delivered.
notes text Text notes related to the vision order pair.
collecteddate datetime This column stores the time stamp when this pair order was collected by the customer. The format used is YYYY-MM-DD HH:MM:SS.
oldpairid int This column stores a reference pair ID (parent pair ID) in case of a remake. This column refers to column id of this column.
cancelreasonid int This column indicates the reason why this pair order is cancelled. This column refers column to id from vision_cancelorder_reasons table.
cancelleddate datetime This column stores the time stamp when the pair order was cancelled. The format used is: YYYY-MM-DD HH:MM:SS.
processorflag tinyint This column indicates where the spectacles are bought from. This column refers column to id from the inventory_vendorcredentials table.
labid int This column indicates who is the supplier for this pair order. This column refers to column id from the inventory_supplier table.
labName varchar This column stores name of the supplier from whom the spectacles are bought. This column refers to the column name from inventory_supplier table.
prescribeid int This column indicates the prescription to which this pair order is linked. This column refers column id from visionrx table.
instruction varchar This column stores any additional notes (free text) added by the dispenser for internal purposes while this pair order is being processed.
accountid int This column stores lab company account number set for spectacle order.
noLensFlag bit This column stores bit value and by default is set to 0. This column will refer to as No Lens.
addSalesTaxForFrame bit This column stores bit value and by default is set to 0. This column value when set to 1 inditcates to honor the sales tax for a frame order when no lens option is selected.
