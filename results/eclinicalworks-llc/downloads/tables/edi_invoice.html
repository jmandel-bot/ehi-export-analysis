EHI Export Schema
Table edi_invoice
Description This table stores information about EDI (Electronic Data Interchange) invoices for medical services provided to patients.
Column DataType Description
id (Primary Key) int The column stores an id of the claim for a patient with encounter or without encounter. This is an auto incremental primary key of This column .
Encounterid int The column stores identifier number of the encounterid of a patient , for whom the claim is created. It refers to column encounterid from enc table.
Patientid int identifier for the patient associated with this invoice, refers to the `id` column in the `patient` table.
InvoiceDt datetime This column stores Date of the Invoice created.
ServiceDt datetime Date when the medical service was provided.
InvoiceNo varchar Unique invoice number.
InvoiceAmount decimal This Column stores Invoice Amount
Payment decimal Total payment received for the invoice.
copay decimal This column stores co-pay amount set for the claim. It is usually a fixed percentage of the claim amount, which the patient pays at the time of a claim creation.
uncoveredAmount decimal Amount not covered by insurance.
PtResp decimal Patient's responsibility for payment.
PtPayment decimal Payment made by the patient.
PtBalance decimal This column stores patient portion balance.
Balance decimal Remaining balance for the invoice.
SuperBillNo varchar Unique number for the superbill associated with the invoice.
DOSProviderid int This column stores the doctor id who provided the health services to the patient(the actual date on which service provided).
PayToProviderid int This column store Providerid to whom the payment will be made
RenProviderid int Rendering Provider id -Third party Provider
Supervisorid int This column stores the supervisor id ,who supervise and coordinate third party billing and proper application of patient ,insurance and other payments.
Guarantorid int This column stores the id of a patient who responsible to pay the bill, the guarantor is always the patient unless the patient is an minor or an incapacitated adult.
FileStatus varchar This column stores the code related to claim status like : PEN,CMS,CMC,PCM,PAT .It refers column code from claimstatuscodes table.
Submissionid int identifier for the submission of the invoice.
BilledToid int This column stores the either patient id or insurance id if BilledToidType is 1 then insurance id otherwise patient id.
BilledToidType smallint This column stores the value 1 / 2 which indicating 1 - Insurance , 2 - Patient.
PtStmtMsg text Patient statement message.
PayorType smallint This column stores the value 1 / 2 which indicating 1 - Insurance , 2 - Patient.
FeeSchid int This column stores PtFedSchid value. It refers column PtFedSchid from PtFeeSchList table.
DeleteFlag tinyint This column stores value 0 / 1 indicating this claim is deleted or not. This is soft delete. 0 - Not Deleted , 1 -Deleted.
vmid varchar This column stores a string as unique identifier for each row.
ModifyDate datetime time stamp for the last modification of the invoice record.
Type tinyint This column stores the value 0/1/2 .It refers 0 - HCFA Claim , 1 - UB Claim , 2 - Dental Claim.
ClaimLock varchar This column stores L/U value , indicating this claim is locked or UnLock. 'L' - Lock Claim , 'U' - UnLock claim.
IncidentTo int This column stores value 0/1. 0 - not in used , 1 - Services performed by a nurse practitioner 'Incident To' a physician. If it is 'Incident To' claim, Supervising physician provider number will be put in box 24K. For this itemkeys 'EDI_EnableIncidentToClaims' should be enabled otherwise it stores 0.
InvFacilityid int This column stores identifier number of the facility id which is assigned at the time of claim creation.
AddlProvider1 int This column stores 1st additional Provider id which is optional and they are used only for ecw reports and will be submitted to insurance.
AddlProvider2 int Additional Provider id
GivenToCollection tinyint Flag indicating if the invoice is given to a collection agency (0 for not given, 1 for given).
CollectionCode varchar Code for the collection agency handling the invoice.
TransferDate datetime This column stores the transfer date when claim is transfer from insurance to patient or vice versa.
SubmittedToid int This column is stores BilledToid value, used only for log when claim is assignedTo Ins/Patient and the claim is Submitted for the same.
SubmittedToidType int this column is stores BilledToidType value, used only for log when claim is assignedTo Ins/Patient and the claim is Submitted for the same.
SubmittedDate datetime This column is stores today's date , the day claim is submitted, it will update whenever you submitted claim successfully.
CurrentCollectionStatus tinyint Current status of the invoice in the collection process (e.g., 0 for not in collection, 1 for in collection).
MovedToCollectionDate varchar Date when the invoice was moved to the collection process.
RemovedFromCollectionDate varchar Date when the invoice was removed from the collection process.
FirstSubmittedDate datetime This column store date when claim is submitted successfully first time, It will not updated.
CreatedBy int This column stores the user id when claim is created. It refers column UID from user table.
SplitClaimid int identifier for the split claim associated with the invoice.
NetPayment decimal Net payment received for the invoice.
LineItemPayersEnabled int This column stores 0/1. 0 - when itemkeys EnableLineItemPayers is not enabled for LineItemSubmission , 1 - when itemkeys EnableLineItemPayers is enabled for LineItemSubmission.
LineItemTransitionStage int This column stores 0/1. 0 - when no submission for LineItemPayer. 1- when submission for LineItemPayer.
OrgBilledToid int This column stores value of BilledToid when claim PreProcess For LineItemSubmission.
OrgBilledToidType int This column stores value of BilledToidType when claim PreProcess For LineItemSubmission.
OrgFileStatus varchar This column stores value of FileStatus when claim PreProcess For LineItemSubmission.
AllowedFee decimal This column is stores the sum of (AllowedFee).It refers column AllowedFee from edi_inv_cpt table.
invoicedeptid int This column stores the claim department id ,when EnableDepartments itemkeys is enabled otherwise by default 0.
closingid int This column stores 0/1 value. 0 - Claim is not hard closed , 1 - Claim is hard closed.
collectionstatus varchar This column not in used.
collcodechgdate varchar This column stores the date the day user transfer to collection. The format used is MM/dd/yyyy .
collcode varchar This column stores the collection status code like 'CO','C1','C2'.
ApptStartTime datetime This column stores the appointment created time if claim have encounter id otherwise NULL. The format used is HH:MM:SS
InvPOS int This column stores the claim Place of service code.
PrTransitionStage int This column stores 0/1 value. 0 - when PostProcess for claim is required. 1 - when PreProcess for claim is required.
TransformProviderType varchar This cloumn stores enum('PayTo','Rendering','Supervising','None') value. 'None' - when PostProcess for claim is required , ('PayTo','Rendering','Supervising') - when PreProcess for claim is required.
PayToProvideridOrg int This column stores PayToProviderid value when PreProcess for a claim is required.
RenProvideridOrg int This column stores RenProviderid value when PreProcess for a claim is required.
SupervisoridOrg int This column stores Supervisor id when PreProcess for a claim is required.
payrejectionstatus int This column stores the id of erareasongroups table , from claim UI user can see payment rejection status. If itemkeys 'EnableRSS' is yes then it come into the picture.
assignedToid int This column stores the provider/staff id from User Interface.
assignedTo varchar this column stores the provider/staff name from User Interface.
FinanceChrgFlag tinyint This column stores 0/1. 0 - by default , 1 - when claim is Finance Charge claim(For creating FinanceChargeClaim itemkeys 'EnableFinanceChargeCalculation' should be enabled and it will created from Accountslookup screen.)
FinanceChrgAddedDt datetime This column stores the date whenever user created Finance Charge Claim.
AnesthesiaFlag tinyint This column stores 0/1. 0 - by default , 1 - when user select Anesthesia claim while creating claim.
AssignmentType char Type of assignment for the invoice (e.g., 'AR' for accounts receivable).
VoidFlag tinyint This column stores the value 0/1/2 , where 0 - created claim , 1 - original claim , 2- Voided claim
queueid int This column stores id for reference to work queue items queue. It refers column id from table workqueueitems.
queueName varchar This column not in used.
ChargeFeeSchid int This column stores id of feesch when ED FeeScheduled is Enabled.
PrimaryInsid int This column stores identifier number of the insurance who is assigned the patient as a Primary Ins id.
SecondaryInsid int This column stores identifier number of the insurance who is assigned the patient as a Secondary Ins id.
TertiaryInsid int This column stores identifier number of the insurance who is assigned the patient as a Tertiary Ins id.
Practiceid int This column stores an id of the practice at which claim is created. This column refers column id from practice table.
InvRespPartyid int This column stores the id of responsible party. It should be patient/guarantor id.
InvRespPartyRel int This column stores the id of responsible party relation.
InvResourceid int This column stores id of resource for a claim such as anyone on the staff (except a physician), devices, or even rooms.
IsGrPt int This column holds flag of Is Patient has Guarantor
neano varchar This column stores the Nea Receipt number for claim.
episode_id int identifier for the episode of care associated with the invoice.
collbalance decimal Balance of the invoice in the collection process.
demandtype varchar This column not in used.
collectiondemanddt datetime This column not in used.
stmtholdtype varchar Type of statement hold for the invoice (e.g., 'A' for automatic, 'M' for manual).
ExcludeFromStmt int This column stores 0/1. 0 - when claim is not exclude from statement, 1 - when claim is excluded from statement and It come into picture only when Itemkeys 'EnableExcludeClmFromStmt' is enabled.
clmstmtholdreasonid int This column holds Claim statement hold reason id. This id reference with statementholdreasons.id
clmstmthelddate datetime Date when the claim was put on statement hold.
collectionstagedate datetime Date when the invoice entered the current collection stage.
isClaimAppealed tinyint Flag indicating if the claim has been appealed (0 for not appealed, 1 for appealed).
OperatingProviderid int This column is used when IPBilling itemkeys is enabled.
OtherOperatingProviderid int This column is used when IPBilling itemkeys is enabled.
AttendingProviderid int identifier for the attending provider associated with the invoice, refers to the `id` column in the `provider` table.
isopticalclaim tinyint Flag indicating if the invoice is for optical services (0 for not optical, 1 for optical).
