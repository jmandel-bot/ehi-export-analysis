EHI Export Schema
Table edi_inv_cpt
Description This table stores information about invoice line items, including details about the service provided, billing information, and related codes.
Column DataType Description
Id (Primary Key) int Unique identifier for each record in the table.
InvoiceId int Identifier for the invoice this line item belongs to.
ItemId int Identifier for the item associated with this line item.
Code varchar CPT (Current Procedural Terminology) code for the service provided.
PrimaryCode tinyint Indicates if the code is primary (1) or not (0).
Mod1 varchar First CPT modifier code associated with the service.
Mod2 varchar Second CPT modifier code associated with the service.
Mod3 varchar Third CPT modifier code associated with the service.
Mod4 varchar Fourth CPT modifier code associated with the service.
SDOS datetime Start date of service.
FromTime datetime Start time of service.
EDOS datetime End date of service.
ToTime datetime End time of service.
TOS varchar Type of service code.
pos varchar Place of service code.
UnitType char Unit type for the billed service.
BilledUnitFee decimal Fee per unit billed for the service.
Units decimal Number of units billed for the service.
BilledFee decimal Total fee billed for the service.
NonCoveredCharges decimal Charges not covered by insurance.
ICD1 varchar First ICD (International Classification of Diseases) diagnosis code associated with the service.
ICD2 varchar Second ICD diagnosis code associated with the service.
ICD3 varchar Third ICD diagnosis code associated with the service.
ICD4 varchar Fourth ICD diagnosis code associated with the service.
AllowedUnitFee decimal Allowed fee per unit for the service.
AllowedFee decimal Total allowed fee for the service.
PtPortion decimal Patient portion of the fee.
TotalPtPortion decimal Total patient portion of the fee.
EPSDT char Early and Periodic Screening, Diagnostic, and Treatment indicator.
EMG char Emergency indicator.
COB char Coordination of Benefits indicator.
ReviewByCode char Review by code indicator.
OrdPrUserId int Ordering provider user ID.
OrdPrNPI varchar Ordering provider NPI (National Provider Identifier) number.
RefPrUserId int Referring provider user ID.
RefPrNPI varchar Referring provider NPI number.
FacilityId int Facility identifier where the service was provided.
RendPrUserId int Rendering provider user ID.
RendPrNPI varchar Rendering provider NPI number.
SupPrUserId int Supervising provider user ID.
SupPrNPI varchar Supervising provider NPI number.
CliaID varchar Clinical Laboratory Improvement Amendments (CLIA) ID.
MammographCertNo varchar Mammography certification number.
AnesOxMinutes int Anesthesia oxygen minutes.
Notes text Notes related to the line item.
BillToIns int Indicates if the line item is billed to insurance (1) or not (0).
BillToPt smallint Indicates if the line item is billed to the patient (1) or not (0).
RevCodeID varchar Revenue code identifier.
RevCode varchar Revenue code description.
PrescriptionNumber varchar Prescription number associated with the service.
ServiceCodeSlNo int Service code serial number.
ServiceIDType char Service ID type.
ServiceCode varchar Service code.
cptOrder smallint CPT order number.
deleteFlag int Indicates if the record is deleted (1) or not (0).
displayIndex int Display index for the line item.
NDCCode varchar National Drug Code associated with the service.
NDCUnits varchar National Drug Code units.
InitialTreatmentDate varchar Initial treatment date for the service.
HospiceProvInd varchar Hospice provider indicator.
BilledToIdType int Billed to ID type.
BilledToId int Billed to ID.
BilledToIdSeqNo int Billed to ID sequence number.
FileStatus varchar File status of the line item.
OrgBillToIns int Original bill to insurance.
OrgOrdPrNPI varchar Original ordering provider NPI number.
OrgRefPrNPI varchar Original referring provider NPI number.
OrgRendPrNPI varchar Original rendering provider NPI number.
OrgSupPrNPI varchar Original supervising provider NPI number.
ToothNumber varchar Tooth number associated with dental services.
Surface varchar Surface code associated with dental services.
closingId int Closing identifier.
NdcUnitPrice decimal National Drug Code unit price.
NdcUom varchar National Drug Code unit of measure.
RxOrderNumber varchar Prescription order number.
cptBalance decimal CPT balance.
OrgItemId int Original item ID.
OrgCode varchar Original CPT code.
OrgBilledUnitFee decimal Original billed unit fee.
OrgUnits decimal Original number of units.
OrgBilledFee decimal Original billed fee.
CPTReplacementStage tinyint CPT replacement stage.
AnesthesiaStartTime datetime Anesthesia start time.
AnesthesiaEndTime datetime Anesthesia end time.
AnesthesiaBaseUnitVal int Anesthesia base unit value.
AnesBaseUnitVal int Anesthesia base unit value (duplicate).
OrgDispIndex int Original display index.
T1015TransitionStage tinyint T1015 transition stage.
HPSAModifierAddedColumn varchar HPSA (Health Professional Shortage Area) modifier added column.
OrgBillToInsT1015 int Original bill to insurance T1015.
IsSedationBilling tinyint Indicates if sedation billing is applicable (1) or not (0).
NOCDescription varchar Not Otherwise Classified (NOC) description.
vinvcptid int Vendor invoice CPT ID.
dmerccertificationtypecode char Durable Medical Equipment Regional Carrier (DMERC) certification type code.
dmercdurationmonths int DMERC duration in months.
dmerccertificationindicator char DMERC certification indicator.
dmercconditionindicator1 varchar DMERC condition indicator 1.
dmercconditionindicator2 varchar DMERC condition indicator 2.
referringcliaid varchar Referring CLIA ID.
cmnflag tinyint Certificate of Medical Necessity (CMN) flag.
documentedTime int Documented time for the service.
OperatingProviderId int Operating provider ID.
OperatingProviderNPI varchar Operating provider NPI number.
OtherOperatingProviderId int Other operating provider ID.
OtherOperatingProviderNPI varchar Other operating provider NPI number.
timeCaptureType varchar Time capture type for the service.
