Table antigen_recipe
Description This table stores information about antigen recipes, including details about the diluants, volumes, and concentrations used in the recipe.
Column DataType Description
id (Primary Key) int The column stores an id of the recipe. This is an auto incremental primary key of This column .
BOTTLEid int The column stores the bottleid associated with the recipe. It refers to column bottle_id from antigen_bottle table.
QUEid int The column stores the queid associated with the recipe. It refers to column id from antigen_recipe_queue table.
DILUANT varchar The column stores the diluant value associated with the recipe.
DILUANT_PCT varchar The column stores the diluant concentration value associated with the recipe.
TOTAL_VOLUME varchar The column stores the total volume of the recipe.
BOTTLE_NAME varchar The column stores the bottle name of the recipe. It refers bottle_name from antigen_bottle table.
RECIPE_NAME varchar The column stores the recipe name given by the user of the recipe.
ICDCODES varchar The column stores the icd codes associated with the recipe (bottle).
RECIPE_NOTE text The column stores the note for the recipe.
REVIEWED int The column stores the reviewed flag for the recipe.
RV_USER varchar The column stores the diluant type for the recipe.
RV_TIME datetime The column stores the reviewed time of the recipe.
CONCENTRATION varchar The column stores the concentration value for the recipe.
MAINT_DOSE varchar The column stores the maintenance dose of the recipe.
