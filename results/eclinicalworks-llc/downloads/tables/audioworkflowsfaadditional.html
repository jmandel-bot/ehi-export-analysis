EHI Export Schema
Table audioworkflowsfaadditional
Description This table stores additional information related to audio workflows, such as various settings and parameters.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
Encid int Foreign key that refers to the id of the associated audio workflow in another table. Example: 101, 102, 103.
SAT varchar A string representing the SAT (Speech-to-Text) setting for the audio workflow. Example: "SAT1", "SAT2", "SAT3".
SRT varchar A string representing the SRT (SubRip Text) setting for the audio workflow. Example: "SRT1", "SRT2", "SRT3".
WH varchar A string representing the WH (Watermark Hash) setting for the audio workflow. Example: "WH1", "WH2", "WH3".
Warble int An integer representing the Warble setting for the audio workflow. Example: 0, 1, 2.
NBN int An integer representing the NBN (Noise Background Noise) setting for the audio workflow. Example: 0, 1, 2.
PTN tinyint A tinyint representing the PTN (Priority Task Number) setting for the audio workflow. Example: 0, 1.
PN tinyint A tinyint representing the PN (Priority Number) setting for the audio workflow. Example: 0, 1.
