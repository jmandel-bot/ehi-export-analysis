Table careplan_team
Description This table stores information about the team members involved in a specific care plan, including their roles and responsibilities.
Column DataType Description
Id (Primary Key) int Unique identifier for each record in the careplan_team table. Example: 1, 2, 3, etc.
CarePlanId int Identifier for the care plan that the team member is associated with. This is a foreign key that refers to the `Id` column in the `careplan` table. Example: 101, 102, 103, etc.
StaffId int Identifier for the staff member who is part of the care plan team. This is a foreign key that refers to the `Id` column in the `staff` table. Example: 1001, 1002, 1003, etc.
isLeadClinician int A flag indicating whether the staff member is the lead clinician for the care plan. This is a binary value, where 0 means the staff member is not the lead clinician, and 1 means the staff member is the lead clinician. Example: 0 or 1.
