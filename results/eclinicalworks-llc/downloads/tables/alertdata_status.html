EHI Export Schema
Table alertdata_status
Description This table is used to handle concurrency while CDSS alerts are being calculated for a patient.
Column DataType Description
patientid (Primary Key) int This column stores the id of a patient who qualifies for the CDSS alerts. This column refers to the column UID from the user table.
isInsertInProgress bit This column stores a flag 0 or 1. 1- while CDSS alerts are being calculated, and 0 - when CDSS alerts calculation finished.
lockTakenAt datetime This column stores the current time stamp in YYYY-MM-DD HH:MM:SS format.
