EHI Export Schema
Table prisma_device_property
Description This table stores properties of devices, such as code, display, code system, and code text, and associates them with a specific device through a foreign key.
Column DataType Description
id (Primary Key) bigint A unique identifier for each device property entry. It is of type bigint(20) and is the primary key for the table. Example: 1, 2, 3, etc.
deviceId bigint A foreign key reference to the `prisma_device_details` table, specifically the `id` column. It is of type bigint(20) and is used to associate the device property with a specific device. Example: 101, 102, 103, etc.
code varchar A code representing the device property. It is of type varchar(50) and can be NULL. Example: "PROP01", "PROP02", "PROP03", etc.
display varchar A human-readable display name for the device property. It is of type varchar(50) and can be NULL.
codeSystem varchar A code system used to categorize the device property. It is of type varchar(50) and can be NULL. Example: "SNOMED-CT", "LOINC", "FHIR", etc.
codeText varchar A textual description of the device property code. It is of type varchar(255) and can be NULL.
