EHI Export Schema
Table asc_anesthesia_plan_enc
Description This table stores the records of selected anesthesia plans and options based on ASC encounters.
Column DataType Description
id (Primary Key) int This is a primary key which increments itself automatically.
encid int This column stores encounter identifier. It refers to encounterid column of enc table.
anestPlanDetailid int This column stores the Anesthesia plan detail identifier. It refers to id column from asc_anesthesia_plan_details table.
anestPlanOptionid int This column stores the Anesthesia plan option identifier. It refers to id column from asc_anesthesia_plan_options table.
userid int This column stores the user id or provider id. It refers to the userid column from user table.
