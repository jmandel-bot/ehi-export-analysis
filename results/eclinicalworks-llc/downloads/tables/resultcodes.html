EHI Export Schema
Table resultcodes
Description This table stores information about various result codes, their associated action codes, and related templates.
Column DataType Description
id (Primary Key) int Unique identifier for each result code entry. Example: 1, 2, 3.
resultCode varchar A string representing the result code. Example: "Success", "Error", "Warning".
actionCodeid int An integer representing the associated action code id. Example: 101, 102, 103.
PoupupTemplate tinyint A tiny integer indicating whether a popup template is used (1) or not (0). Example: 0, 1.
Templateid int An integer representing the id of the associated template. Example: 1001, 1002, 1003.
TemplateName varchar A string representing the name of the associated template. Example: "Success Template", "Error Template", "Warning Template".
Queueid int An integer representing the id of the associated queue. Example: 201, 202, 203.
deleteflag tinyint A tiny integer indicating whether the result code entry is deleted (1) or not (0). Example: 0, 1.
scoring int An integer representing the scoring value associated with the result code. Example: 10, 20, 30.
DueDays int An integer representing the number of days until the result code is due. Example: 5, 10, 15.
type tinyint A tiny integer representing the type of the result code. Example: 0, 1.
