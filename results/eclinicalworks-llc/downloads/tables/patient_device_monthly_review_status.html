Table patient_device_monthly_review_status
Description This table stores the monthly reading review status of rpm enrolled patient device. This includes information related to review status and review month.
Column DataType Description
id (Primary Key) int Contains auto incremented id, datatype is INT. This column stores integer value which could grow with number of transactions added and hence integer is selected . It is AUTO_INCREMENT column and has PRIMARY KEY index
patientid int Contains the Primary Key column UID of user table. It stores the unique id of a Patient and its data type is INT. This field will always have a value hence defined as NOT NULL.
rpm_enrollment_id int Contains enrollmentid value referenced from id column of the rpm_enrollment_details table, datatype is INT. This field will always have a value hence defined as NOT NULL.
reading_month datetime Contains reading date of when the device readings were received. This column stores date hence Date is selected. This field will always have a value hence defined as NOT NULL. By default '1901-01-01'/ '0000-00-00' for MSSQL/MYSQL accordingly is chosen.
review_status tinyint This column is introduced to maintain if the review for 16 consecutive readings is completed for any of the eligible devices for a patient. Contains review status with values 0 and 1,(0-Pending,1-Complete) datatype is TINYINT. This field will always have a value hence defined as NOT NULL.
deleteFlag tinyint Flag indicates the last reading received for the month is deleted, datatype is TINYINT - 0 - Not Deleted, 1 - Deleted. This field will always have a value hence defined as NOT NULL. By default, this column should not be in Deleted status hence default 0 is chosen.
modifiedBy int Contains the user id who has modified the record last. The data type chosen is INT. This field is expected to have a value always and has a value referencing the id column of the user table
modifiedDatetime datetime Contains "modifiedDatetime " value when a user is added/updated reading on a monthly basis for a particular enrollment and datatype is DATETIME.
