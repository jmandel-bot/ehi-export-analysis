EHI Export Schema
Table vision_order_repair
Description This table stores information about vision order repairs, including details about the product, pricing, tax, discount, and status updates throughout the repair process.
Column DataType Description
id (Primary Key) int This column is an auto incremented primary key.
order_id int This column stores the reference order id from vision_order table.
product_id int This column stores the product id from inventory_product table.
notes text This column stores the additional notes (free text) added by the end user for this repair order.
sku varchar This column stores the stock keeping unit from sku column of inventory_product table .
unitprice decimal This column stores the unit amount in decimal format for this repair order.
taxratio_code_id int This column is not in use.
taxamount decimal This column stores the tax amount in decimal format for this repair order.
totalprice decimal This column stores the final amount in decimal format for this repair order.
isdeleted tinyint This column stores a value 0 or 1. 1 - when this repair order is deleted, 0 - Otherwise. This is not inserted / modified by application.
createdby int This column stores the logged in user id who creates this repair order. This column refers column UID from user table.
createdon datetime This column stores the time stamp when this repair order is created. The format used is YYYY-MM-DD HH:MM:SS.
updatedby int This column stores the logged in user id who modifies this repair order. This column refers column UID from user table.
updatedon datetime This column stores the time stamp when this repair order is modified. The format used is YYYY-MM-DD HH:MM:SS.
addtocollection tinyint This column stores a value 0 or 1. 1 - This repair order will be collected later, 0 - Otherwise.
duedate datetime This column is not in use.
collecteddate datetime This column stores the time stamp when this repair order was collected by the customer. The format used is YYYY-MM-DD HH:MM:SS.
inspecteddate datetime This column stores the time stamp when this repair order was inspected. The format used is YYYY-MM-DD HH:MM:SS.
inspectedby int This column stores logged in user id who performs the inspection on this repair order. This column refers column UID from user table.
deliveredby int This column stores logged in user id who delivers this repair order to the customer. This column refers column UID from user table.
delivereddate datetime This column stores the time stamp when this repair order was delivered. The format used is YYYY-MM-DD HH:MM:SS.
collection_notes text This column stores additional notes (free text) specific to collection.
status int This column indicates status of this repair order placed. This column refers column id from vision_master table with type = 1.
collectedby int This column stores the ID of the patient who collects this repair order. This column refers column UID from user table. If this repair order is collected by any other person apart from patient then a value of 0 is inserted to this column.
collectednotes text This column stores additional notes pertaining to the value '0' in the collectedBy column.
facilityid int This column indicates the facility where the patient and customer is administered. This column refers column id from edi_facilities table.
