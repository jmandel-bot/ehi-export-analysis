Table ccmr_visits_addldata
Description This table stores the players injury visit related details such as injuryid, ccmrid, visitId, visitDate etc along with primary reason and timing.
Column DataType Description
id (Primary Key) int The column stores an id of the care plan of a patient. This is an auto incremental primary key of This column .
ccmrid int The column stores identifier ccmrid of the patient. This refers to column ccmrid from ccmr_careplan table.
visitid int This column is used to save injury visitid
Problemid int The column stores identifier problemid of the ccmr_atrisk table. This refers to column atRiskid from ccmr_atrisk table.
Problem varchar This column is used to save injury name
VisitDate datetime This column is used to save visit date when it was created
deleteFlag tinyint Flag to store whether the record has been marked deleted or not. 0 - Not deleted, 1 - deleted
encounterid int Column stores encounterid from enc table created for care plan.
