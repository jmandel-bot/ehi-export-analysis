EHI Export Schema
Table asc_enc_or_proc
Description This table contains operation procedure details for ASC encounter, such as Procedure Type, primarySurgeonId, etc.
Column DataType Description
id (Primary Key) int This is an auto increment primary key column of This column .
procEncid int This column stores procedure encounter id
procType int Integer representing the type of procedure being performed. Example: 1 (surgery), 2 (diagnostic), 3 (therapeutic).
primarySurgeonid int This column stores primary surgeon id
attendingSurgeonid int Foreign key referring to the attending surgeon's id from another table. Example: 2001, 2002, 2003.
primaryAnestid int This column stores primary anest id
anestSupervisorid int This column stores anest supervisor id
firstAsstid int Foreign key referring to the first assistant's id from another table. Example: 5001, 5002, 5003.
secondAsstid int This column stores second assistant id
deleteFlag int This column store delete flag. It will be set to 1 if entry is deleted else 0.
modifybyid int This column stores current user logged-in user id. It is primary id user table.
modifiedDate datetime This column is used to capture date when details is updated.
mergeDefaultTemplateAction tinyint Tiny integer indicating the action to be taken when merging default templates. Example: 1 (merge), 2 (overwrite), 3 (ignore).
