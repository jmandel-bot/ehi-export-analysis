Table asc_drug_administration_details
Description This table stores the details of drug administration events, including information about the drug, the administration process, and the patient's consent.
Column DataType Description
id (Primary Key) int Unique identifier for each drug administration detail record. (e.g., 1, 2, 3)
ADMINISTRATION_HEADER_id int This column stores id of a administered drug. This column refers to id column in ASC_Drug_Administration_Header table.
ACTION int This field stores the action for administration like given, not given
NotGivenReason varchar This field stores the not given reason for administration
Rate decimal The fields store the rate of administration for IV drug.
RateUnit varchar This field store the rate unit for IV drug administration like ml/hr.
VolumeGiven varchar This field stores the administration volume for given drug
VolumeGivenUnit varchar This field store the volume unit for administered IV drug.
GivenBy varchar This column store name of the provider/staff who administered the medication during the visit.
GivenByid int This column store id of the provider/staff who administered the medication during the visit. This column refers to UID column in user table.
Site int This column stores the site id where the drug is administered.
StartTime datetime This field stores the start time for IV drug administration.
EndTime datetime This field stores the end time for IV drug administration.
CreatedBy int This column store id of the provider/staff who administered the medication during the visit. This column refers to UID column in user table.
CreatedDate datetime This column stores date when provider added the administered details.
ModifiedBy int Integer representing the unique identifier of the user who last modified the record. (e.g., 2001, 2002)
ModifiedDate datetime This column stores date when provider updated the administered details.
STATUS int Integer representing the status of drug administration. The value will be 1 when drug administration would get deleted. Default value is 0.
PatientVerified smallint A boolean value (0 or 1) indicating whether the patient's identity was verified during the drug administration event. (e.g., 0, 1)
MedicationVerified smallint A boolean value (0 or 1) indicating whether the medication was verified during the drug administration event. (e.g., 0, 1)
RiskVerified smallint A boolean value (0 or 1) indicating whether the risks were verified during the drug administration event. (e.g., 0, 1)
WrittenConsent smallint A boolean value (0 or 1) indicating whether the patient provided written consent for the drug administration. (e.g., 0, 1)
VerbalConsent smallint A boolean value (0 or 1) indicating whether the patient provided verbal consent for the drug administration. (e.g., 0, 1)
IsSequence smallint This field stores the value for is sequence drug administration. The value can be 0 or 1. Default value is 0
parentid int This column is used for sequential IV drug mapping. which stores value of id column from same table for IV in progress.
IVPush smallint A boolean value (0 or 1) indicating whether the drug was administered via IV push. (e.g., 0, 1)
deleteFlag int Medication whose administration is deleted then deleteFlag is set to 1. Default value is 0.
