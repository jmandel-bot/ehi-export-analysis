Table prisma_allergy_details
Description This table stores information about patients' allergy details, including allergens, allergy types, reactions, and related clinical information.
Column DataType Description
id (Primary Key) bigint Unique identifier for each allergy detail record (bigint). Example: 1, 2, 3.
sectionId bigint Foreign key referring to the `prisma_ccd_document_section` table's `id` column, representing the section in which the allergy detail is recorded (bigint). Example: 101, 102, 103.
entryId varchar Identifier for the entry in the document where the allergy detail is recorded (varchar). Example: "entry_001", "entry_002", "entry_003".
allergen varchar Description of the allergen causing the allergy (varchar). Example: "Peanuts", "Dust", "Penicillin".
moodCode varchar Code representing the mood of the allergy record (varchar). Example: "EVN", "INT", "RQO".
statusCode varchar Code representing the status of the allergy record (varchar). Example: "active", "inactive", "resolved".
recordedDate varchar Date when the allergy record was created (varchar). Example: "2021-08-01", "2021-08-02", "2021-08-03".
allergyTypeCode varchar Code representing the type of allergy (varchar). Example: "food", "drug", "environment".
allergyType varchar Description of the allergy type (varchar). Example: "Food Allergy", "Drug Allergy", "Environmental Allergy".
rxNormCode varchar Code representing the RxNorm identifier for the allergen (varchar). Example: "12345", "67890", "11121".
rxNormName varchar Name of the RxNorm identifier for the allergen (varchar). Example: "Peanut", "Dust Mite", "Penicillin G".
comments text Additional comments about the allergy record (longtext). Example: "Patient experienced severe reaction", "Mild symptoms only", "Carries EpiPen".
isException tinyint Flag indicating if the record is an exception (tinyint). Example: 0 (false), 1 (true).
allergyReactionType varchar Type of reaction caused by the allergen (varchar). Example: "Anaphylaxis", "Rash", "Swelling".
allergyCodingSystem varchar Coding system used for the allergy record (varchar). Example: "ICD-10", "SNOMED CT", "LOINC".
verificationStatusCode varchar Code representing the verification status of the allergy record (varchar). Example: "confirmed", "unconfirmed", "refuted".
verificationStatusDisplay varchar Description of the verification status (varchar). Example: "Confirmed", "Unconfirmed", "Refuted".
clinicalStatusDisplay varchar Description of the clinical status of the allergy (varchar). Example: "Active", "Inactive", "Resolved".
allergyIntoleranceCriticality varchar Criticality of the allergy intolerance (varchar). Example: "low", "high", "unable-to-assess".
onsetDate datetime Date when the allergy symptoms first appeared (datetime). Example: "2021-08-01 12:00:00", "2021-08-02 14:30:00", "2021-08-03 16:45:00".
onsetEndDate datetime Date when the allergy symptoms ended (datetime). Example: "2021-08-01 14:00:00", "2021-08-02 16:30:00", "2021-08-03 18:45:00".
allergenCode varchar Code representing the allergen (varchar). Example: "A001", "A002", "A003".
allergenName varchar Name of the allergen (varchar). Example: "Peanut", "Dust Mite", "Penicillin G".
encounterId bigint Identifier for the encounter related to the allergy record (bigint). Example: 1001, 1002, 1003.
recorderId int Identifier for the person who recorded the allergy information (int). Example: 1, 2, 3.
