Table visionrx_spectaclenotes
Description This table stores information about spectacle notes and lens add-ons related to a specific vision prescription and encounter.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer value. It uniquely identifies each record in the table. Example: 1, 2, 3, etc.
VisionRxid int This is a foreign key that refers to the `id` column in the `visionrx` table. It represents the vision prescription associated with the spectacle notes and lens addons. Example: 101, 102, 103, etc.
Encounterid int This is a foreign key that refers to the `encounterid` column in the `enc` table. It represents the encounter associated with the spectacle notes and lens addons. Example: 201, 202, 203, etc.
LensAddons text This is a text column that stores information about any lens addons for the spectacle prescription. Example: "Anti-glare coating, Blue light filter".
Notes text This is a text column that stores any additional notes related to the spectacle prescription. Example: "Patient prefers lightweight frames, Adjust nose pads for comfort".
