EHI Export Schema
Table kioskepaytrxlogs
Description This table stores information about payment transactions made at kiosks.
Column DataType Description
id (Primary Key) int Unique identifier for each transaction record. This is the primary key for the table. Example: 1, 2, 3, etc.
ptid int The identifier for the patient associated with the transaction. This could be a foreign key referring to the `id` column in a `patients` table. Example: 101, 102, 103, etc.
paymentid int The identifier for the payment method used in the transaction. This could be a foreign key referring to the `id` column in a `paymentmethods` table. Example: 1, 2, 3, etc.
encid int The identifier for the encounter associated with the transaction. This is foreign key referring to the `encounterid` column in an `enc` table
amountPaid decimal The amount paid in the transaction. This is a double value. Example: 50.00, 100.00, 150.00, etc.
copay int A flag indicating whether the transaction is a copay (1) or not (0). This is an integer value. Example: 0 or 1.
