EHI Export Schema
Table ccmr_bhnotes
Description This table contains notes information for Goal, Objective or Intervention.
Column DataType Description
Id (Primary Key) int Unique identifier for each note entry in the table. Example: 1, 2, 3.
goalMappingId int This column stores unique id of a goal/objective/intervention for which notes are being documented by end user. This column refers to id column in ccmr_goalmapping table.
TYPE char This column store G/O/I to indicate whether the data is for Goal, Objective or Intervention. G - Goal, O - Objective, I - Intervention.
bhId int Column refers to structdatadetail.Id
notes text This column stores notes being documented by end user.
delflag int This column stores flag to indicate whether the record is deleted or not. 0 - Not deleted, 1 - deleted
userId int This column stores id of the end user who is adding notes. This column refers to uid column in user table.
notesDateTime datetime This column stores date and time when notes were added to the application by end user.
lockNotes int A flag indicating whether the note is locked (1) or unlocked (0). Example: 0, 1.
