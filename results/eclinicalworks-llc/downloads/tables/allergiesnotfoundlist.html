EHI Export Schema
Table allergiesnotfoundlist
Description This table stores data imported from the EHX server when a user imports allergies for a patient. This table is populated when the local database does not have allergies.
Column DataType Description
id (Primary Key) int This is an auto-incremented primary key in this column.
encounterid int This column stores the id of a patient's encounter from which data is being imported. This column refers to the encounterid column in the enc table.
trUserid int This column stores the id of a user who imported allergies related data from the EHX server. This column refer to the UID column in the user table.
AllergyName varchar This column stores the name of the allergan being imported from the EHX server.
reaction varchar This column stores the patient's reaction to the allergy imported from the EHX server.
MappedAllergyName varchar When imported allergies are mapped using itemid, this column stores the name of the equivalent allergy mapped with the allergies from the items table. The mapping is performed by the end user.
mtimestamp datetime This column stores the date and time when the record was imported from the EHX server.
hasMatched tinyint This column stores values 0/1 to indicate whether the imported allergies has been matched with a local database or not. 0 - has matched, or 1 - not matched.
delflag tinyint This column stores values 0/1 to indicate whether the record has been marked as deleted or not. 0 - Record is NOT marked as deleted, or 1 - Record is marked as deleted.
CCRDataObjectid int This column stores an id received in CCR document for each allergy listed in the CCR. This column stores a unique number for each allergy imported in a single call from the EHX server.
refReqid int This column stores the id of a referral in case allergy related information has been imported using the RCP > DRTLA tab. This column refers to the Referralid column in the referral table.
allergyType int This column stores the type of allergy whether it is coded or uncoded.
mappedAllergyItemid int This column stores the id of the equivalent item mapped with the items table. It is the itemid from the items table.
