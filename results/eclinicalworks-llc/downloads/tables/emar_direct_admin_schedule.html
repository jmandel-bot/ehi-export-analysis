EHI Export Schema
Table emar_direct_admin_schedule
Description This table stores information about the administration schedule of medications, including details about the medication, dosage, and administration status for Correctional Clients
Column DataType Description
Id (Primary Key) int This column is the primary key for This column .
Emar_Medication_Id int This column store the id from emar_medications table.
DueDate datetime This column store the date on which administration is due.
DueTime datetime This column store the time on which administration is due.
Given int This column store the given status of an administration. 0 - Not Given, 1 - Given
GivenStrength varchar This column store the Strength of Rx at the time of Rx administration.
GivenAmount varchar This column store the Amount of Rx at the time of Rx administration.
GivenDate datetime This column store the Date of Rx administration.
GivenBy varchar This column store User's name who did Rx administration.
NDC varchar This column store NDC code of Rx administration.
Notes text This column store Notes entered by user at the time of Rx administration.
Stat int This column store Stat status of Rx administration. 0 - Not stat , 1- Stat
First int This column store First status of Rx administration. 0 - Not first , 1- First
Stop int This column store Stop status of Rx administration. 0 - Not Stop , 1- Stop
AdminFailed int This column store Failed status of Rx administration. 0 - Not Failed , 1- Failed
AdminNotes text This column store notes added by Admin at the time of Rx administration.
Createdtimestamp datetime This column store time stamp when Rx administration was created in the system.
DeleteFlag int This column store delete status of Rx administration. 0 - Not Deleted , 1 - Deleted
Onhold tinyint This column store On Hold status of Rx administration. 0 - Not Onhold , 1 - Onhold.
OnholdReasonId int This column store reason associated with On Hold status of Rx administration. This data is coming from items.itemid
StatusReasonId int This column store reason associated with Given/Not given status of Rx administration. This data is coming from items.itemid
LateEarlyReasonId int This column is not in use.
ActualPillPassTime datetime This column store the user input time of Rx administration while administering the drug.
OnholdBy varchar This column store the user's id who put Rx administration status on hold. This data is coming from users.uid.
LateEarlyEntry tinyint This column is not in use
OnholdDate datetime This column store the data and time when Rx administration status is put on hold.
wasted tinyint This column is not in use
ingredient varchar This column is not in use
Facility int This column store the facility id in which patient is at the time of administration schedule creation. This data is coming from edi_facilities.id.
HousingArea varchar This column store the housing area in which patient is at the time of administration schedule creation.
FilledMedname varchar This column store the Filled Med name for Rx administration. This is pharmacy data that is returned for Rx administration.
FilledDose varchar This column store the Filled Dose for Rx administration. This is pharmacy data that is returned for Rx administration.
FilledAmount varchar This column store the Filled Amount for Rx administration. This is pharmacy data that is returned for Rx administration.
FollowupNotes text This column store the Followup notes for Rx administration.
NotNeeded tinyint This column is not in use
isVitalTaken smallint This column store the status of Vital Taken at the time of Rx administration. 0 - Vitals not taken, 1 - Vitals taken.
VitalDenialReason text This column store the reason for denial for taking vitals at the time of Rx administration.
