Table ccmr_groupsignatures
Description This table stores information about group signatures, including the user who signed, their credentials, and the time the signature was taken.
Column DataType Description
id (Primary Key) int Unique identifier for each group signature entry. (Example: 1, 2, 3)
groupEncid int identifier for the group associated with the signature. (Example: 101, 102, 103)
groupSignUserType tinyint Type of user who signed the group (0 or 1, where 0 could represent a regular user and 1 an admin user). (Example: 0, 1)
signUserid int identifier for the user who signed the group. (Example: 1001, 1002, 1003)
signUserName varchar Name of the user who signed the group. (Example: "John Doe", "Jane Smith")
degreeCredential varchar Degree or credential of the user who signed the group. (Example: "Ph.D. in Computer Science", "MBA in Finance")
signTakenTime datetime Date and time when the signature was taken. (Example: '2022-01-01 12:34:56')
deleteFlag tinyint Indicates whether the group signature entry has been deleted (0 for not deleted, 1 for deleted). (Example: 0, 1)
userSignature varbinary Binary data representing the user's signature. (Example: Binary data of an image file)
