Table dental_savedtpdiscounts
Description This table stores information about discounts applied to dental treatment plans.
Column DataType Description
id (Primary Key) int • This column will store the id of the dental_savedtpdiscounts • The datatype of the column would be INT to accommodate many different procedures discount data that can be added. • This would be unique and auto incremented and so PRIMARYKEY constraint is added
procTPNum int • This column stores the id of the procedure added in saved dental treatment plan. • This is a reference to the column proctpnum of table proctp.
practiceDiscount decimal • This column stores practice discounts applied on procedures of treatment plans. • The datatype of the column would be decimal as it can have 0 to 9999999.99 values
slideDiscount decimal • This column stores slide discounts applied on procedures of treatment plans. • The datatype of the column would be decimal as it can have 0 to 9999999.99 values.
deleteFlag tinyint • This column determines whether the procedure discount record is deleted or not. • The datatype of the column would be tinyint as it can have 0 or 1 value. • 0 value means the procedure discount record is not deleted. • 1 value means the procedure discount record is deleted
