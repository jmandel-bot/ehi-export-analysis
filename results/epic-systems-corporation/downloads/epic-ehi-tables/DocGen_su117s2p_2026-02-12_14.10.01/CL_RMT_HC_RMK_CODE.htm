CL_RMT_HC_RMK_CODE
Description:
Contains health care remark code information from the service line level of an electronic remittance file. This information is sent in the LQ segment of an ANSI 835 Health Care Claim Payment/Advice file. This information is stored in the remittance image record.

Primary Key
Column Name Ordinal Position
IMAGE_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 IMAGE_ID VARCHAR No
ID for the remittance image record containing the health care remark code information.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 LQ_SERVICE_LINE VARCHAR No
The service line associated with the health care remark code.
4 CODE_LST_QUAL_C_NAME VARCHAR No
The code list qualifier code. This is a standard code which indicates which code set the remark code belongs to.
May contain organization-specific values: No
Category Entries:
Claim payment remark codes
National council for prescription drug programs reject/payment codes
5 INDUSTRY_CODE VARCHAR No
The specific health care remark code.
