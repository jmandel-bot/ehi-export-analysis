EPA_INFO_2
Description:
This table holds information about electronic prior authorization requests.

Primary Key
Column Name Ordinal Position
REFERRAL_ID 1

Column Information
Name Type Discontinued?
1 REFERRAL_ID NUMERIC No
The unique identifier for the referral record.
2 PREV_AUTH_EXP_DATE DATETIME No
For prior authorization requests that were created because the existing authorization is about to expire, this item holds the date that the existing authorization will expire.
3 ELIG_PAT_ENC_CSN_ID NUMERIC No
This item holds the CSN of the encounter to use for pharmacy benefits for a prior authorization request.
4 TOTAL_DOC_SECONDS INTEGER No
This item tracks how many total seconds users spent with ePA documentation activities open.
5 PA_PRIORITY_SETBY_C_NAME VARCHAR No
The prior authorization priority change source category ID for the source of the priority of medication prior authorization request.
