COMPILED_ATTRIBUTION_DATA
Description:
This table holds the External Identifier Bundles and attributed providers for which attribution data was compiled.

Primary Key
Column Name Ordinal Position
ASGN_DATA_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 ASGN_DATA_ID NUMERIC No
The unique identifier (.1 item) for the assignment data record record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 CM_PHY_OWNER_ID VARCHAR No
The Community ID (CID) of the instance that owns this record or line. This is only populated if you use IntraConnect.
4 CM_LOG_OWNER_ID VARCHAR No
The Community ID (CID) of the instance from which this record or line was extracted. This is only populated if you use IntraConnect.
5 ATTR_PROV_TYPE_C_NAME VARCHAR No
The type of the attributed provider for this compiled attribution period.
May contain organization-specific values: Yes
6 ATTRIBUTED_PROV_ID_PROV_NAME VARCHAR No
The name of the service provider. This item may be hidden in a public view of the CLARITY_SER table.
7 ATTRIBUTION_PERIOD_EFF_DATE DATETIME No
The start date of the compiled provider attribution period.
8 ATTRIBUTION_PERIOD_TERM_DATE DATETIME No
The end date of the compiled provider attribution period.
