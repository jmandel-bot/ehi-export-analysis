DOCS_RCVD_RES_TEXT
Description:
The DOCS_RCVD_RES_TEXT table contains textual information about a result.

Primary Key
Column Name Ordinal Position
DOCUMENT_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 DOCUMENT_ID NUMERIC No
Stores the Received Document record ID.
2 LINE INTEGER No
The line number for the information associated with this contact. Multiple pieces of information can be associated with this contact.
3 RES_TEXT_KEY_CMP VARCHAR No
The key which links the result text with the order.
4 RES_TEXT_COMPILED VARCHAR No
A line of text for the result.
5 RES_TEXT_CAT_CMP_C_NAME VARCHAR No
The internal category identifying what kind of textual information is recorded in this line.
May contain organization-specific values: No
Category Entries:
Component Text
Result Impression
Result Narrative
Other Result Text
Result Transcription
Addendum
Result Comment
Procedure Note
Specimen Comment
Resulting Agency Comment
Image Note
Pathologist Signature
6 RES_TEXT_SUBID_CMP VARCHAR No
A key used to identify different entries for multiple results of the same type. Lines may match the same result key but will have different sub IDs to identify them as different records.
