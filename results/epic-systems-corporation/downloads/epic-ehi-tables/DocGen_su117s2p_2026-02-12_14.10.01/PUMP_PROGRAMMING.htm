PUMP_PROGRAMMING
Description:
This table contains compliance information about infusion pumps. The records included in this table are Alerts (ALT) records that track information about pump programming attempts.

Primary Key
Column Name Ordinal Position
ALERT_ID 1
LINE 2

4 INF_ERROR_TITLE VARCHAR No
The title of the message from the pump that is displayed to the user.
5 INF_ERROR_TEXT VARCHAR No
The message text from the pump that is displayed to the user.
6 INF_INT_STAT_C_NAME VARCHAR No

7 INF_SEND_INST_DTTM DATETIME (UTC) No
The instant when the pump program was sent to the infusion pump.
8 INF_RESPONSE_I_DTTM DATETIME (UTC) No
The instant when the response was received from the infusion pump.
9 INF_MED_ID_MEDICATION_NAME VARCHAR No
The name of this medication record.
10 INF_ORDER_ID NUMERIC No
The unique ID of the order that is associated with this message from the pump.
11 INF_USER_ID VARCHAR No
The unique ID of the user that is associated with this message from the pump. This column is frequently used to link to the CLARITY_EMP table.
12 INF_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
13 INF_DEPARTMENT_ID_EXTERNAL_NAME VARCHAR No
The external name of the department record. This is often used in patient correspondence such as reminder letters.
14 INF_DEVICE_ID VARCHAR No
The unique ID of the device that is associated with this message from the pump.
15 INF_DEVICE_ID_DEVICE_NAME VARCHAR No
The name for this device.
16 INF_LINE_NUMBER INTEGER No
The line number for the most recent prior message sent to the pump for the same order for the same patient encounter.
17 VOLUME_TO_INFUSE_IN_ML NUMERIC No
The volume to infuse in mL that was sent to the pump.
18 INF_ATTEMPT_UTC_DTTM DATETIME (UTC) No
