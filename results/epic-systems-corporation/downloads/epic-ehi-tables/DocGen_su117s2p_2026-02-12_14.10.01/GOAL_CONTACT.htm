GOAL_CONTACT
Description:
This table contains contact specific data for the IGO (Discrete Goals) master file.

Primary Key
Column Name Ordinal Position
GOAL_ID 1
CONTACT_DATE_REAL 2

Column Information
Name Type Discontinued?
1 GOAL_ID VARCHAR No
The unique ID for the goal record.
2 CONTACT_DATE_REAL FLOAT No
The DTE contact date for the goal record.
3 CONTACT_DATE DATETIME No
The date of this contact in calendar format.
4 CP_PRIORITY_C_NAME VARCHAR No
The priority assigned to this goal.
May contain organization-specific values: Yes
5 START_DATE DATETIME No
The date to begin work on this goal.
6 END_DATE DATETIME No
The date work on this goal was completed.
7 EXP_END_DATE DATETIME No
The date work on this goal is expected to be completed.
8 GOAL_OUTCOME_C_NAME VARCHAR No
The outcome(s) associated with a care plan goal.
May contain organization-specific values: Yes
9 EDIT_USER_ID VARCHAR No
This item holds edited user ID information.
10 EDIT_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
11 PATH_OUTCOME_VAL_C_NAME VARCHAR No
The category number for the documentation value assigned to this pathway outcome for this contact.
May contain organization-specific values: No
Category Entries:
Met
Not Met
12 PATH_OUTCOME_VAR_ID VARCHAR No
The unique ID of the variance that was documented on this pathway outcome for this contact.
13 CARE_PLAN_CSN_ID NUMERIC No
Links to the care plan contact. Used to recreate historic versions of the care plan.
14 INCLUDE_DESC_YN VARCHAR No
If set to Yes, the goal description will be included on the plan of care. If set to No or left blank, the goal description will be omitted.
May contain organization-specific values: No
Category Entries:
No
Yes
15 HAS_NOTE_EDIT_YN VARCHAR No
Y means that, for this reading, the default smarttext has been pulled in at least once.
The category values for this column were already listed for column: INCLUDE_DESC_YN
16 READING_UTC_DTTM DATETIME (UTC) No
Documents when the original reading contact was documented. Used to order active documentation readings in the order they clinically happened.
17 UPDATED_GOAL_ID VARCHAR No
This item links to the new goal ID that was created when this original goal was resolved and updated.
18 REASON_FOR_UPDATE VARCHAR No
This item contains the reason as to why this goal was completed and updated.
