TREATMENT_PATTERN_DAYS
Description:
Treatment day patterns in a treatment plan.

Primary Key
Column Name Ordinal Position
TREATMENT_PLAN_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 TREATMENT_PLAN_ID NUMERIC No
The unique identifier (.1 item) for the plan record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 PATTERN_DAY_TRG_CSN_ID NUMERIC No
The unique contact serial number of the TRG that this treatment day uses.
4 PATTERN_DAY_SORT_VALUE INTEGER No
Stores the Id to identify a day in a pattern cycle. Note that multiple days can have the same Id if they are part of different pattern cycles.
5 PATTERN_DAY_CYCLE_LINE INTEGER No
The cycle pattern that this day is part of. Stored as a line in SI TPL 12000.
6 PATTERN_DAY_DATE_IN_CYCLE INTEGER No
The day in the cycle that this treatment day falls on. Will be 0 if the day falls on the cycle start date, 1 if it is the second day of the cycle, etc.
7 PATTERN_DAY_STATUS_C_NAME VARCHAR No
The treatment day status category Id for the pattern treatment day.
May contain organization-specific values: No
Category Entries:
Planned
Scheduled
Completed
Deleted
Started
Deferred
Canceled
Given Externally
Unscheduled
Non-Treatment Orders
8 PATTERN_DAY_NAME VARCHAR No
Tha name for this pattern day. This will be set as the name of the treatment day that will be created from this pattern day.
9 PATTERN_DAY_TYPE_C_NAME VARCHAR No
The inpatient or outpatient category Id for the pattern treatment day.
May contain organization-specific values: No
Category Entries:
Inpatient
Outpatient
10 PATTERN_DAY_NUM INTEGER No
That day number in the cycle of a pattern day.
11 PATTERN_DAY_LENGTH INTEGER No
The length of a treatment day. Always 1 for outpatient days but can be 1 or more for inpatient days.
12 PATTERN_DAY_SRC_TYPE_C_NAME VARCHAR No
The type of day this pattern day was created from.
May contain organization-specific values: No
Category Entries:
Created From Normal Plan Day
Created By Copying a Normal Plan Day
Created By Copying a Pattern Plan Day
13 PATTERN_DAY_SOURCE_LINE INTEGER No
The line in the source where this day was created from.
