HSP_BKT_SRCHG_HX
Description:
Surcharge history information. Stores all the data that can go into HSP_BKT_SURCHARGE by the date and time that it happened.

Primary Key
Column Name Ordinal Position
BUCKET_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 BUCKET_ID NUMERIC No
This column stores the unique identifier for the bucket record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 SURCHARGE_CHAN_DTTM DATETIME (Local) No
This item stores the instant that the surcharge changed.
4 SURCHARGE_COMMENT VARCHAR No
This item stores the comment entered by the user when the surcharge is overridden or the comment generated by the system when the surcharge is assigned.
5 SURCHARGE_UPDATE_C_NAME VARCHAR No
This item stores whether the user or the system updated the surcharge
May contain organization-specific values: No
Category Entries:
System
User
Ins Pmt
6 SURCHARGE_EFF_DT DATETIME No
This item stores the effective date of the surcharge as of the instant of the surcharge change.
7 SURCHARGEABLE_AMT NUMERIC No
This item stores the surchargeable amount on the bucket as of the instant of the surcharge change.
8 SURCHARGE_PERCENT NUMERIC No
This item stores the percent of the surchargeable that applies to the surcharge as of the instant of the surcharge change.
9 SURCHARGE_HX_AMOUNT NUMERIC No
This item stores the amount of the surcharge as of the instant of the surcharge change.
10 SRCHG_HX_RULE_ID NUMERIC No
This item stores the rule that was last used to determine the surcharge percent as of the instant of the surcharge change.
11 SRCHG_HX_RULE_ID_RULE_NAME VARCHAR No
This column stores the name of the hospital billing workqueue rule.
12 HSP_ACCOUNT_ID NUMERIC No
This column stores the unique identifier for the hospital account with which the liability bucket is associated.
13 SURCHARGE_USER_ID VARCHAR No
This item stores the user who changed the surcharge.
14 SURCHARGE_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
15 TAX_HAR_CER_RULE_ID VARCHAR No
The unique Id of the general rule that was last used to generate tax charges.
16 TAX_HAR_CER_RULE_ID_RULE_NAME VARCHAR No
The name of the rule.
