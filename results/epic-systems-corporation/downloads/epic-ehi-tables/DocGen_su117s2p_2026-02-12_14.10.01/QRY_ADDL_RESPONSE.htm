QRY_ADDL_RESPONSE
Description:
This table extracts the response status (examples: whether the query was read, reply type, and timestamp) of the secondary recipients of the query.

Primary Key
Column Name Ordinal Position
QUERY_CSN_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 QUERY_ID NUMERIC No
The unique identifier for the query record.
2 LINE INTEGER No
The line number for the information associated with this contact. Multiple pieces of information can be associated with this contact.
3 CONTACT_DATE DATETIME No
The date of this contact in calendar format.
4 ADDL_RESP_USER_ID VARCHAR No
Stores the non-primary responding users of the query.
5 ADDL_RESP_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
6 ADDL_RESPONSE_C_NAME VARCHAR No
Stores the non-most recent responses of the query.
May contain organization-specific values: No

7 ADDL_RESP_UTC_DTTM DATETIME (UTC) No
Stores the non-most recent response instant of the query in UTC.
8 QUERY_CSN_ID NUMERIC No
The contact serial number (CSN) of the contact.
9 ADDL_RESP_USR_WRKFLW_C_NAME VARCHAR No
This item describes what workflows additional users responded to QRYs in.
