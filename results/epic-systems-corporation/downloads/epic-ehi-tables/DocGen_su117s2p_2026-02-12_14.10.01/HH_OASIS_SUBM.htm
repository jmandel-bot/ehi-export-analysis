HH_OASIS_SUBM
Description:
Contains information on the submission of Home Health Outcome and Assessment Information Set (OASIS) data sets.

Primary Key
Column Name Ordinal Position
OASIS_SET_ID 1
CONTACT_DATE_REAL 2
LINE 3

Column Information
Name Type Discontinued?
1 OASIS_SET_ID NUMERIC No
Unique identifier for the OASIS data set.
2 CONTACT_DATE_REAL FLOAT No
Unique identifier for this contact for this patient.
3 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
4 OASIS_SUBM_TYPE_C_NAME VARCHAR No
OASIS data set submission type. Links to category list ZC_OASIS_SUBM_TYPE. Submission, correction, resubmission, inactivation.
May contain organization-specific values: Yes
Category Entries:
Submission
Correction
Resubmission
Inactivation
5 OASIS_SUBM_INST_MK DATETIME (Attached) No
Instant at which OASIS data set is marked submitted.
6 OAS_SUBM_USER_ID VARCHAR No
User ID of user who submitted OASIS data set. Links to table CLARITY_EMP.
7 OAS_SUBM_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
8 OAS_SUBM_FILE_INST DATETIME (Local) No
Instant at which OASIS submission was filed.
9 OAS_SUBM_CRRCT_NUM VARCHAR No
OASIS submission correction number.
10 OASIS_SUBMIT_QUE_C_NAME VARCHAR No
OASIS data set's position in submission queue. Links to category list ZC_OASIS_SUBMIT_Q.
May contain organization-specific values: Yes
Category Entries:
Default
11 QUEUE_HISTORY_C_NAME VARCHAR No
OASIS data set submission queue history. Links to category table ZC_OASIS_SUBMIT_Q.
The category values for this column were already listed for column: OASIS_SUBMIT_QUE_C_NAME
12 SUBM_QUEUE_DT_INDX VARCHAR No
OASIS submission queue index item.
13 HISTORY_INDEX VARCHAR No
OASIS submission queue history index.
