PAT_ENC_2
Description:
This table supplements the PAT_ENC table. It contains additional information related to patient encounters or appointments.

Primary Key
Column Name Ordinal Position
PAT_ENC_CSN_ID 1

Column Information
Name Type Discontinued?

25 SYNC_IP_DATA_C_NAME VARCHAR No
This is a contact-specific flag to tell Chart Sync whether inpatient clinical data for this visit (such as medication administration or documentation flowsheet data) should be synchronized between deployments. A value of 1 (or Yes) means that inpatient clinical data has been, and will continue to be, synchronized for this visit. Note that this item can only be set for a contact whose type is (or was at some point) an inpatient admission.
May contain organization-specific values: No
Category Entries:
No
