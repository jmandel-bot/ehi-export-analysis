NOTE_EXT_SIGNERS
Description:
Note signer information for auto-reconciled external notes.

Primary Key
Column Name Ordinal Position
NOTE_CSN_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 NOTE_CSN_ID NUMERIC No
The contact serial number (CSN) of the contact.
2 LINE INTEGER No
The line number for the information associated with this contact. Multiple pieces of information can be associated with this contact.
3 NOTE_ID VARCHAR No
The unique identifier (.1 item) for the note record.
4 CONTACT_DATE DATETIME No
The date of this contact in calendar format.
5 EXT_NOTE_SIGNER_NAME VARCHAR No
An auto-reconciled external note signer's name.
6 EXT_NOTE_SIGNING_UTC_DTTM DATETIME (UTC) No
An auto-reconciled external note signing instant for a note signer.
7 EXT_NOTE_SIGNER_ROLE_C_NAME VARCHAR No
External note signer's role in authenticating the content of an auto-reconciled external note.
May contain organization-specific values: No
