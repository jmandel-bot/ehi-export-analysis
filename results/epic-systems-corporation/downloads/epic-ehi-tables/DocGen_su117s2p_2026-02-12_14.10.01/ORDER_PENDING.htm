ORDER_PENDING
Description:
This table contains information on pending orders.

Primary Key
Column Name Ordinal Position
ORDER_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 ORDER_ID NUMERIC No
The unique identifier for the pended order record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record
3 USER_ID VARCHAR No
The unique ID of the user pending the order.
4 USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
5 PENDED_TIME DATETIME (Local) No
The time an order was pended.
6 PENDED_FOR VARCHAR No
The reason an order was pended.
7 RELEASED_USER_ID VARCHAR No
The unique ID of the user releasing a pended medication.
8 RELEASED_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
9 PENDING_COMMENTS VARCHAR No
Order pending comments.
10 PEND_REASON_C_NAME VARCHAR No
Order pending reason.
