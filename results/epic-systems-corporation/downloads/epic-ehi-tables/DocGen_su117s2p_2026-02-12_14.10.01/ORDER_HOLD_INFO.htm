ORDER_HOLD_INFO
Description:
This table contains hold related information for orders that were held/unheld.

Primary Key
Column Name Ordinal Position
ORDER_ID 1
LINE 2

Name Type Discontinued?

The date on which the patient should restart taking this med - that is, the date of the first dose to take after the pause.
21 RESTART_TAKING_TIME_OF_DAY_C_NAME VARCHAR No
The category ID for the time of day (morning, bedtime, etc.) when the patient should restart taking this med - that is, the time of the first dose to take after the pause.

22 PAT_COMMENTS VARCHAR No
Comments directed to the patient about an instruction to pause taking this medication.

This stores the CSN of the encounter where this hold action was taken.
24 HOLD_START_STATUS_C_NAME VARCHAR No
Determines how the hold start instant should be interpreted. If this is blank or set to 0 then HOLD_START_UTC_DTTM should be interpreted as is. If this is anything else, HOLD_START_UTC_DTTM will be interpreted differently.
May contain organization-specific values: No
Category Entries:
