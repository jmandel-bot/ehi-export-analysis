REI_RES_PERSONNEL
Description:
The REI_RES_PERSONNEL table contains information about an embryology result's personnel and each person's responsibilities.

Primary Key
Column Name Ordinal Position
RESULT_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 RESULT_ID VARCHAR No
The unique identifier (.1 item) for the result record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 REI_EMB_RESPONSIBILITY_C_NAME VARCHAR No
The responsibility of this embryology result that the associated person was responsible for.
May contain organization-specific values: Yes
Category Entries:
Performed Procedure
Witnessed Procedure
Checked Patient ID
Checked Specimen IDs
Checked Plate IDs
Performed Tubing
Witnessed Tubing
Checked Sperm IDs
Performing Embryologist
4 RESPONSIBLE_USER_ID VARCHAR No
The responsible user for the associated responsibility of this embryology result.
5 RESPONSIBLE_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
6 RESPONSIBLE_PERSON_TEXT VARCHAR No
The non-user responsible person for the associated responsibility of this embryology result.
7 RESPONSIBLE_PERSON_AUTH_YN VARCHAR No
Whether the responsible person for this embryology procedure responsibility has authenticated.
May contain organization-specific values: No

No
