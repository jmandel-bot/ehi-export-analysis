CLP_NDC_RELATED
Description:
This table holds the National Drug Code (NDC) information on the claim.

Primary Key
Column Name Ordinal Position
CLAIM_PRINT_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 CLAIM_PRINT_ID NUMERIC No
This is the claim record ID.
2 LINE INTEGER No
The line number of one of the multiple values associated with a specific group of data within this record.
3 NDC_CODE_ID VARCHAR No
This is the National Drug Code ID.
4 NDC_CODE_ID_NDC_CODE VARCHAR No
The external code for the National Drug Code (NDC). An NDC represents packages of medications.
5 NDC_NUMBER VARCHAR No
This is the National Drug Code number.
6 NDC_QTY NUMERIC No
This is the National Drug Code quantity.
7 NDC_UNITS_C_NAME VARCHAR No
This holds the units of the drug. The units are billing appropriate.
May contain organization-specific values: No
Category Entries:
Milliliters
Milligrams
Grams
International Units
Units
8 CLAIM_LINE_PTR NUMERIC No
This field points to a claim line.
9 ERX_ID_MEDICATION_NAME VARCHAR No
The name of this medication record.
10 RX_NUM VARCHAR No
This holds the prescription number.
11 COMPOUND_LINK_NUM VARCHAR No
This holds the Link Sequence Number used to associate ingredients of a compound drug.
