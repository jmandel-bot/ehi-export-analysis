CL_ORD_FST_LST_SCH
Description:
This table stores an order's first and last scheduled date and time, along with the type of review notice for the expire items (i.e. let expire or review).

Primary Key
Column Name Ordinal Position
ORDER_ID 1

Column Information
Name Type Discontinued?
1 ORDER_ID NUMERIC No
The unique ID of the order record associated with this order.
2 DATE_FIRST_ORDEREN DATETIME No
Date of the first schedule.
3 TIME_FIRST_ORDEREN DATETIME (Local) No
Time of the first schedule.
4 DATE_LAST_ORDERENT DATETIME No
Date of the last schedule.
5 TIME_LAST_ORDERENT DATETIME (Local) No
Time of the last schedule.
6 LET_EXPIRE_TYPE_C_NAME VARCHAR No
Determines the type of review notice for the former "let expire" items
May contain organization-specific values: No
Category Entries:
Let Expire
Review
