MEMBER_USAGE_HIST
Description:
This table contains the history of different activities associated with the member.

Primary Key
Column Name Ordinal Position
BENEFIT_USAGE_ID 1
CONTACT_DATE_REAL 2

Column Information
Name Type Discontinued?
1 BENEFIT_USAGE_ID VARCHAR No
The unique ID of the benefit usage record.
2 CONTACT_DATE_REAL FLOAT No
The date when benefit usage is recorded.
3 BENEFIT_USG_TYP_C VARCHAR No
The type of usage record.
May contain organization-specific values: No
Category Entries:
New Member
Renew Member
Deactivate Membership
Event
Benefit Usage
Card Request
Card Issued
4 BENEFIT_DATE DATETIME No
The date when the benefit is used.
5 BENEFIT_ID_MEMBERSHIP_NAME VARCHAR No
The name for membership/benefit record.
6 START_DATE DATETIME No
The start date for the benefit usage.
7 EXP_DATE DATETIME No
The end date for the benefit usage.
8 EVENT_ATTENDEES INTEGER No
The number of benefits used.
9 DEACT_RSN_C VARCHAR No
The reason for deactivating the membership/benefit.
May contain organization-specific values: Yes
10 CARD_REQ_RSN_C VARCHAR No
The reason for requesting new card.
