RES_REPEAT_COMP_SNOMED
Description:
This table contains the SNOMED codes for component repeats as reported by the external resulting agency.

Primary Key
Column Name Ordinal Position
RESULT_ID 1
GROUP_LINE 2
VALUE_LINE 3

Column Information
Name Type Discontinued?
1 RESULT_ID VARCHAR No
The unique identifier (.1 item) for the result record.
2 GROUP_LINE INTEGER No
The line number for the specific repeat of a component on the result from the RES_REPEAT_COMP table.
3 VALUE_LINE INTEGER No
The line number of one of the multiple SNOMED codes for this specific repeat of the component on the result.
4 SNOMED_CODE VARCHAR No
The SNOMED code associated with the value for the specific repeat of the component on this result as reported by the external resulting agency.
