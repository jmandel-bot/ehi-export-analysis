PAT_ENC_BILLING_ENC
Description:
This table contains encounter-specific data related to Billing Encounters.

Primary Key
Column Name Ordinal Position
PAT_ENC_CSN_ID 1

Column Information
Name Type Discontinued?
1 PAT_ENC_CSN_ID NUMERIC No
The unique contact serial number for this contact. This number is unique across all patient encounters in your system. If you use IntraConnect, this is the Unique Contact Identifier (UCI).
2 CONTACT_DATE DATETIME No
The date of this contact in calendar format.
3 CM_CT_OWNER_ID VARCHAR No
The Community ID (CID) of the instance that owns this contact. This is only populated if you use IntraConnect.
4 POS_TYPE_OVRIDE_C_NAME VARCHAR No
Place of service type override for Billing Encounters. If empty, the place of service type is determined by the type set in the facility record (EAF) contained in the Visit Place of Service (I EPT 5617).

Indicates whether a Billing Encounter has been marked as voided. For example, if a user creates a Billing Encounter for a patient in error, the user can correct that error by marking the Billing Encounter as voided.
