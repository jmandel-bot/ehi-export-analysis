HH_INTVTN_ENC
Description:
Contains Home Health care plan intervention overtime single items information.

Primary Key
Column Name Ordinal Position
INTERVENTION_ID 1
CONTACT_DATE_REAL 2

Column Information
Name Type Discontinued?
1 INTERVENTION_ID VARCHAR No
Unique identifier for the care plan intervention.
2 CONTACT_DATE_REAL FLOAT No
Unique identifier for this contact for this patient.
3 CONTACT_NUM VARCHAR No
Contact number.
4 CONTACT_DATE DATETIME No
The date of this contact in calendar format.
5 CM_CT_OWNER_ID VARCHAR No
ID of the contact owner deployment for this record. Links to table CL_COMMUNTY_INSTNC.
6 INT_START_DT DATETIME No
Care plan intervention start date.
7 INT_RESOLV_DT DATETIME No
Care plan intervention resolved date.
8 EDIT_USER_ID VARCHAR No
User ID of person who edited care plan intervention data. Links to table CLARITY_EMP.
9 EDIT_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
10 EDIT_INSTANT DATETIME (Local) No
Instant at which care plan intervention was edited.
11 INTV_FREQ_ID VARCHAR No
Care plan intervention frequency ID. Links to table IP_FREQUENCY.
12 INTV_FREQ_ID_FREQ_NAME VARCHAR No
The name of the frequency record.
13 SCHEDULED_VISITS VARCHAR No
Care plan intervention scheduled visits.
14 CARE_PLAN_CSN VARCHAR No
Link to care plan contact. Used to recreate historic versions of care plan.
15 HAS_PENDED_ORDER_YN VARCHAR No
Determines whether the intervention order was created but not sent. This is set to Yes if any user created an intervention order without sending it; No if an order is not created or it was created and already sent.
