SPEC_TASK_LIST_SUB
Description:
This is sub container information for Anatomic Pathology specimens.

Primary Key
Column Name Ordinal Position
SPECIMEN_ID 1
GROUP_LINE 2
VALUE_LINE 3

Column Information
Name Type Discontinued?
1 SPECIMEN_ID VARCHAR No
The unique ID of the specimen record
2 GROUP_LINE INTEGER No
The line number of the sub container information for anatomic pathology (AP) specimens. Together with SPECIMEN_ID, this forms the foreign key for this table, SPEC_TASK_LIST.
3 VALUE_LINE INTEGER No
The line number of specific sub container information for anatomic pathology (AP) specimens associated with the different tasks on the specimen from the SPEC_TASK_LIST table.
4 TASK_LINKED_SCTR_ID VARCHAR No
Stores a container designation that may be created in association with a specific container and task, as in a slide for a given block designation with a specific stain.
