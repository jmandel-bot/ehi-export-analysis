LOG_CLAVIEN_SCORE
Description:
This table contains Surgical Log (ORL) items related to Clavien Scores for procedural complications.

Primary Key
Column Name Ordinal Position
LOG_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 LOG_ID VARCHAR No
The unique identifier for the log record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 CLAVIEN_SCORE_C_NAME VARCHAR No
This item contains the documented Clavien score that defines complication severity for a procedure on a log.
May contain organization-specific values: Yes
