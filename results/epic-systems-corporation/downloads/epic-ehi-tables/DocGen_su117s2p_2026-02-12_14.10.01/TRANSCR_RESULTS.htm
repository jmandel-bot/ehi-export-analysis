TRANSCR_RESULTS
Description:
This table contains transcribed result types and their values.

Primary Key
Column Name Ordinal Position
ORDER_ID 1
CONTACT_DATE_REAL 2
LINE 3

Column Information
Name Type Discontinued?
1 ORDER_ID NUMERIC No
The unique identifier (.1 item) for the order record.
2 CONTACT_DATE_REAL FLOAT No
A unique contact date in decimal format. The integer portion of the number indicates the date of contact. The digits after the decimal distinguish different contacts on the same date and are unique for each contact on that date. For example, .00 is the first/only contact, .01 is the second contact, etc.
3 LINE INTEGER No
The line number for the information associated with this contact. Multiple pieces of information can be associated with this contact.
4 TRANSCR_RES_TYPE_C_NAME VARCHAR No
List of the different transcribed result types for this order.
May contain organization-specific values: No
Category Entries:
Pap Smear
HPV
Transformation Zone
Colposcopy
Excision
Ablation
Endometrial Biopsy
Endocervical Curettage
5 TRANSCR_FINDING INTEGER No
Discrete result for each finding type
6 TRANSCR_ABNORM_C_NAME VARCHAR No
The current result flag value for each associated result finding type. It is based on the Order Result Flag (I ORD 2020).
May contain organization-specific values: Yes
Category Entries:
(NONE)
