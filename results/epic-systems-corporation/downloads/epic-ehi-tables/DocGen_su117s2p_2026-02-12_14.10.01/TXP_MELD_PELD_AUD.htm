TXP_MELD_PELD_AUD
Description:
The audit trail for the Model for End-Stage Liver Disease (MELD), Pediatric End-Stage Liver Disease (PELD), and liver status of the patient.

Primary Key
Column Name Ordinal Position
SUMMARY_BLOCK_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 SUMMARY_BLOCK_ID NUMERIC No
The unique identifier for the episode record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 MELD_AUDIT INTEGER No
Stores the audit information for Model for End-Stage Liver Disease (MELD) values for the transplant episode.
4 PELD_AUDIT INTEGER No
Stores the audit information for Pediatric Model for End-Stage Liver Disease (PELD) values for the transplant episode.
5 MELD_DATE_AUDIT_DT DATETIME No
Stores the audit information for the Model for End-Stage Liver Disease (MELD) dates for the transplant episode.
6 PELD_DATE_AUDIT_DT DATETIME No
Stores the audit information for Pediatric Model for End-Stage Liver Disease (PELD) dates for the transplant episode.
7 MELDPELD_OTHR_AUDIT VARCHAR No
Stores the audit information for the Model for End-Stage Liver Disease (MELD), Pediatric Model for End-Stage Liver Disease (PELD), and liver status free-text exceptions for the transplant episode.
8 LIVER_ST_AUDIT_C_NAME VARCHAR No
Stores the audit trail information for the liver status for the transplant episode.

One B

Stores the user responsible for the corresponding audit event for the transplant episode.

11 MELD_PELD_UPD_INST_DTTM DATETIME (UTC) No
