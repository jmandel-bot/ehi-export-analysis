TASK_LOG
Description:
This table extracts the related table Task Log: Instant (I LTK 81900), which contains the log of actions that happened on a task.

Primary Key
Column Name Ordinal Position
ACTIVITY_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 ACTIVITY_ID VARCHAR No
The unique identifier (.1 item) for the task record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 TASK_LOG_INST_DTTM DATETIME (UTC) No
The instant in UTC the action was done.
4 TASK_LOG_ACTION_TYPE_C_NAME VARCHAR No
The type of action logged.
May contain organization-specific values: No
Category Entries:
System Action
User Action
Patient Action
5 TASK_LOG_ACTION_C_NAME VARCHAR No
The action that was done on the task.
May contain organization-specific values: No
Category Entries:
Added
Removed
Forward Scheduled
Paused
Resumed
Details Edited
Mark Resume After Discharge
Mark Not Resumed After Discharge
Schedule Updated
Instances Cleaned Up
Time Zone Changed
Added via Medication Reminder Pause Anchor Event
Readded
6 MYPT_ID VARCHAR No
The MyChart user (WPR) who did the action.
7 DISPLAY_NAME VARCHAR No
The new display when the display name (LTK-41) is changed.
8 TASK_DESC VARCHAR No
The new description when the description (LTK-42) is changed.
9 FREQ_ID VARCHAR No
The new frequency record when the frequency (LTK-300) is changed.
10 FREQ_ID_FREQ_NAME VARCHAR No
The name of the frequency record.
11 TASK_START_UTC_DTTM DATETIME (UTC) No
The new start instant when the start date (LTK-540) is changed.
12 TASK_END_UTC_DTTM DATETIME (UTC) No
The new end instant when the end date (LTK-550) or duration (LTK-556) is changed.
13 GOAL_ID VARCHAR No
The new goal record when the goal (LTK-220) is changed.
14 TASK_START_TIME_C_NAME VARCHAR No
The new start type when the start type (LTK-270) is changed.

May contain organization-specific values: No
Category Entries:

After
17 DURATION_DAYS INTEGER No
The new duration in days when the duration (LTK-556) is changed.
18 DAYS_AFTER_DISCHARGE INTEGER No
The new days after discharge when the days after discharge (LTK-81051) is changed.
19 SCHEDULE_STATUS_C_NAME VARCHAR No

Category Entries:
