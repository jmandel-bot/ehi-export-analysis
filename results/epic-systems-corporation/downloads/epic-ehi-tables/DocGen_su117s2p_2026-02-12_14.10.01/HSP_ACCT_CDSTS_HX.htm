HSP_ACCT_CDSTS_HX
Description:
This table contains the history of coding status changes for each account coded.

Primary Key
Column Name Ordinal Position
HSP_ACCOUNT_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 HSP_ACCOUNT_ID NUMERIC No
The hospital account ID with related coding status history information.
2 LINE INTEGER No
The line number in the results of a query. Each coding history will have its own line number.
3 CDSTS_HX_UTC_DTTM DATETIME (UTC) No
Coding status changed instant - stored in UTC time. Data will not exist in this column for coding statuses created before UTC instant tracking was enabled for coding history.
4 CDSTS_HX_SVC_STS_C_NAME VARCHAR No
Holds the simple visit coding (SVC) completion status when coding status is set by simple visit coding.
May contain organization-specific values: No
Category Entries:
Success
Fail
5 CDSTS_HX_ASGN_USER_ID VARCHAR No
Item to store coding assigned user changes
6 CDSTS_HX_ASGN_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
