CL_RMT_OPT_ADJ_INF
Description:
This table contains outpatient adjudication information from the Image Database (IMD) master file.

Primary Key
Column Name Ordinal Position
IMAGE_ID 1

Name Type Discontinued?

The reimbursement rate percentage expressed as a decimal for outpatient adjudication info.

4 OUTPAT_REMARK_1 VARCHAR No
Outpatient adjudication remark code 1.
5 OUTPAT_REMARK_2 VARCHAR No
Outpatient adjudication remark code 2.
6 OUTPAT_REMARK_3 VARCHAR No
Outpatient adjudication remark code 3.
7 OUTPAT_REMARK_4 VARCHAR No
Outpatient adjudication remark code 4.
8 OUTPAT_REMARK_5 VARCHAR No
Outpatient adjudication remark code 5.
9 CLAIM_ESRD_PAY_AMT NUMERIC No
End stage renal disease (ESRD) payment amount for outpatient adjudication info.
10 OUT_NONPAY_PRF_AMT NUMERIC No
Monetary amount for outpatient adjudication nonpayable professional component.
