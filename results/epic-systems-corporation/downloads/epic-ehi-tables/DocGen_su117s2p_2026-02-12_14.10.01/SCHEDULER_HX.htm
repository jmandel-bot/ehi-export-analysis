SCHEDULER_HX
Description:
Inpatient scheduler history table.

Primary Key
Column Name Ordinal Position
ORDER_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 ORDER_ID NUMERIC No
The unique ID for the order record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 SCH_EXTRA_ACTION_C_NAME VARCHAR No
This item stores what special scheduler action, if any, was performed on scheduling.
May contain organization-specific values: No
Category Entries:
Skip first occurrence per off-schedule occurrence
Extra occurrence scheduled at start point
Skip first occurrence per existing MAR time
First occurrence elapsed, extra occurrence scheduled at start point
4 SCH_SKIPPED_DTTM DATETIME (Local) No
The instant an occurrence was skipped by the scheduler because an off schedule occurrence was too close to a standard time.
5 SCHEDULE_EXTRA_DTTM DATETIME (Local) No
Time that an extra occurrence was scheduled since a standard specified time was just missed.
