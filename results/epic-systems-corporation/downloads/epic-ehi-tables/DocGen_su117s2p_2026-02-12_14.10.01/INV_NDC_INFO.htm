INV_NDC_INFO
Description:
This table holds the National Drug Code (NDC) information for the invoice (INV) record.

Primary Key
Column Name Ordinal Position
INVOICE_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 INVOICE_ID NUMERIC No
The invoice record ID.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 NDC_ID VARCHAR No
The national drug code record ID.
4 NDC_ID_NDC_CODE VARCHAR No
The external code for the National Drug Code (NDC). An NDC represents packages of medications.
5 NDC_NUM VARCHAR No
The national drug code number.
6 NDC_QTY NUMERIC No
The national drug code quantity.
7 NDC_UNITS_C_NAME VARCHAR No
The billable units for the national drug code.
May contain organization-specific values: No
Category Entries:
Milliliters
Milligrams
Grams
International Units
Units
8 LINE_PTR NUMERIC No
The invoice line for the national drug code.
9 MEDICATION_ID_MEDICATION_NAME VARCHAR No
The name of this medication record.
10 RX_NUM VARCHAR No
The prescription number.
11 COMPOUND_LINK_NUM VARCHAR No
The link sequence number for an associated ingredients in a compound drug.
