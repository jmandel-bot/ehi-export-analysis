CLARITY_HVR
Description:
This table contains information about the generic verification (HVR) record.

Primary Key
Column Name Ordinal Position
VERIFY_ID 1

Column Information
Name Type Discontinued?
1 VERIFY_ID NUMERIC No
The unique ID of the verification record
2 REC_STAT_C_NAME VARCHAR No
Record status flag special item. Indicates whether the verification record is active, deleted, hidden, or a combination of statuses.
May contain organization-specific values: No
Category Entries:
Active
Inactive
Deleted
Inactive and Deleted
Hidden
Inactive and Hidden
Deleted and Hidden
Inactive Deleted and Hidden
3 VERIFY_TYPE_C_NAME VARCHAR No
Category value representing the type of verification performed.
May contain organization-specific values: Yes
Category Entries:
Anesthesia
4 PAT_ID VARCHAR No
The unique ID of the patient associated with the verification record.
5 PAT_CSN_ID NUMERIC No
The Contact Serial Number of the patient encounter the verification was performed within.
6 LOG_ID VARCHAR No
Surgical log ID for this verification record.
7 VERIFY_USER_ID VARCHAR No
The unique ID of the user who performed the verification
8 VERIFY_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
9 VERIFY_INSTANT DATETIME (Attached) No
The instant the verification was performed
