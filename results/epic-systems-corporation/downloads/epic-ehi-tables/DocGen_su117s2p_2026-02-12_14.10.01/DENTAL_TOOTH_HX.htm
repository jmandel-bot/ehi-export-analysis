DENTAL_TOOTH_HX
Description:
This table contains the history of edits for a dental tooth record.

Primary Key
Column Name Ordinal Position
FINDING_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 FINDING_ID NUMERIC No
The unique identifier for the finding record
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 DENTAL_PATIENT_CSN NUMERIC No
Stores the patient encounter in which a particular contact of the record was modified if it was available.
4 DENTAL_MOD_USER_ID VARCHAR No
Stores the user who modified the tooth record.
5 DENTAL_MOD_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
6 DENTAL_MOD_INSTANT_DTTM DATETIME (Attached) No
Stores the date and time when the tooth record was modified.
7 DEN_TOO_STAT_HIST_C_NAME VARCHAR No
Keeps track of how status of a tooth changes over time.
May contain organization-specific values: No
Category Entries:
Missing
Erupted
Unerupted
Partially Erupted
Bridged
Implant
Denture
Other
8 DENT_ACTIVE_HIST_C_NAME VARCHAR No
This column keeps track of whether a tooth was shown in the tooth chart over time.
May contain organization-specific values: No
Category Entries:
Inactive
Active
9 DENT_CHRON_INSTANT_DTTM DATETIME (Attached) No
This is the date and time the documentation was made for, such as when documenting past historical data.
