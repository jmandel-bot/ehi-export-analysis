HH_DATASET_EXPORT
Description:
This stores data for Outcome and Assessment Information Set (OASIS) or Hospice Item Set (HIS) data set exports.

Primary Key
Column Name Ordinal Position
DATASET_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 DATASET_ID NUMERIC No
The unique identifier for the data set record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 EXPORT_TYPE_C_NAME VARCHAR No
This item stores the type of export in the export history of an OASIS or HIS record.
May contain organization-specific values: Yes
Category Entries:
Export
Export to 3rd Party
Test Export
Test Export to 3rd Party
Export to 3rd Party (Web)
4 EXPORT_USER_ID VARCHAR No
This item stores the exporting user in the export history of an OASIS or HIS record.
5 EXPORT_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
6 EXPORT_INSTANT_DTTM DATETIME (UTC) No
This item stores the instant of export in the export history of an OASIS or HIS record.
7 EXPORT_SNAPSHOT_DAT VARCHAR No
The contact pointer to the OASIS snapshot data.
