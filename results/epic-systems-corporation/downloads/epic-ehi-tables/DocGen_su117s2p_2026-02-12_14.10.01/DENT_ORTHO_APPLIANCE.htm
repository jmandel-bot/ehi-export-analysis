DENT_ORTHO_APPLIANCE
Description:
Includes data for orthodontic appliances.

Primary Key
Column Name Ordinal Position
FINDING_ID 1
CONTACT_DATE_REAL 2

Column Information
Name Type Discontinued?
1 FINDING_ID NUMERIC No
The unique identifier (.1 item) for the finding record.
2 CONTACT_DATE_REAL FLOAT No
A unique contact date in decimal format. The integer portion of the number indicates the date of contact. The digits after the decimal distinguish different contacts on the same date and are unique for each contact on that date. For example, .00 is the first/only contact, .01 is the second contact, etc.
3 CONTACT_DATE DATETIME No
The date of this contact in calendar format.
4 APPLIANCE_STATUS_C_NAME VARCHAR No
This item stores the status for an orthodontics appliance.
May contain organization-specific values: No
Category Entries:
Ongoing
Completed
Discontinued
Canceled
Planned
5 APPL_START_DATE DATETIME No
This item stores the start date for an orthodontics appliance.
6 EST_DURATION INTEGER No
This item stores the estimated duration (in months) for an orthodontics appliance.
7 REMOVAL_DATE DATETIME No
This item stores the removal date for an orthodontics appliance.
8 MODIFIED_INST_DTTM DATETIME (UTC) No
History item that tracks every instant an orthodontic appliance has been modified.
9 MODIFIED_USER_ID VARCHAR No
Stores the user that modified orthodontics appliance data for this history line.
10 MODIFIED_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
11 MODIFIED_PAT_ENC_CSN_ID NUMERIC No
Stores the patient CSN where data was modified for this history line.
