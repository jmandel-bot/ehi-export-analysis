OR_CASE_SCHED_HIST
Description:
The OR_CASE_SCHED_HIST table contains OR management system case scheduling history.

Primary Key
Column Name Ordinal Position
OR_CASE_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 OR_CASE_ID VARCHAR No
The unique ID of the case record.
2 LINE INTEGER No
The number of the line of the OR scheduling history information.
3 ROOM_ID_PROV_NAME VARCHAR No
The name of the service provider. This item may be hidden in a public view of the CLARITY_SER table.
4 HIST_DATE DATETIME (Local) No
The date on which the case was scheduled.
5 HIST_SCHED_USER_ID VARCHAR No
The ID of the user that scheduled the case.
6 HIST_SCHED_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
7 HIST_SCHED_TIME DATETIME (Local) No
The time at which the user scheduled the case.
8 MATCH_SRVC_BLOCK_C_NAME VARCHAR No
Stores the matching service block ID of the schedule instance if the match type is a service block.
May contain organization-specific values: Yes
9 SCHEDULING_SOURCE_C_NAME VARCHAR No
The source activity or workflow associated with this entry in the scheduling history.
May contain organization-specific values: No
