CLM_VALUES_6
Description:
All values associated with a claim are stored in the Claim External Value record. The CLM_VALUES_6 table holds claim-level values set by the system during claims processing or by user edits.

Primary Key
Column Name Ordinal Position
RECORD_ID 1

Column Information
Name Type Discontinued?
1 RECORD_ID NUMERIC No
The unique identifier (.1 item) for the claim value record.
2 RESUBMISSION_COMMENT VARCHAR No
This item stores the resubmission comments for the claim.
3 SUBSCR_EMPLOYER_PHONE VARCHAR No
This item holds the subscriber's employer's phone number.
4 PAT_SEX_ASSIGNED_AT_BIRTH VARCHAR No
Patient Sex Assigned at Birth
5 ENC_START_DATE DATETIME No
This item stores the encounter start date for the claim.
6 ENC_START_TIME_TM DATETIME (Local) No
This item stores the encounter start time for the claim.

This item stores the encounter end date for the claim.

This item stores the encounter end time for the claim.
9 ENC_END_TYPE VARCHAR No
