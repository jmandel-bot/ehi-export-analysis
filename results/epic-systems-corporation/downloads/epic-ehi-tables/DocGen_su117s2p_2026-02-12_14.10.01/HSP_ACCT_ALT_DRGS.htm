HSP_ACCT_ALT_DRGS
Description:
This table stores the multiple Diagnosis Related Group (DRG) information for the alternate ICD code set when dual coding a hospital account.

Primary Key
Column Name Ordinal Position
HSP_ACCOUNT_ID 1
LINE 2

Column Information
Name Type Discontinued?
