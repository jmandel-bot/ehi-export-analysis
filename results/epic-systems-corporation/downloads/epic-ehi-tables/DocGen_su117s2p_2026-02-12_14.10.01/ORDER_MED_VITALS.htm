ORDER_MED_VITALS
Description:
This table stores historical patient vitals information for each medication order at the time the order was released. It should only be used for reporting on whether or not vitals information had been entered at that point in time.

Primary Key
Column Name Ordinal Position
ORDER_ID 1

Column Information
Name Type Discontinued?
1 ORDER_ID NUMERIC No
The unique identifier for the order record.
2 WEIGHT_AT_RELEASE NUMERIC No
The patient's recorded weight in kilograms at the time the order was released.
3 WEIGHT_REL_SOURCE_C_NAME VARCHAR No
The source category ID for the patient's recorded weight at the time the order was released.
May contain organization-specific values: Yes
Category Entries:
Treatment plan weight
Past measured weight
Most current measured weight (actual)
Ideal body weight
Adjusted body weight
User entered weight
Lean body weight
Dosing weight
Birth Weight
Other
4 HEIGHT_AT_RELEASE NUMERIC No
The patient's recorded height in centimeters at the time the order was released.
5 HEIGHT_REL_SOURCE_C_NAME VARCHAR No
The source category ID for the patient's recorded height at the time the order was released.
May contain organization-specific values: Yes
Category Entries:
Treatment plan height
Past measured height
Most current measured height (Actual)
User Entry
Other
6 BSA_AT_RELEASE NUMERIC No
The patient's calculated body surface area (BSA) in meters squared at the time the order was released.
7 BSA_REL_SOURCE_C_NAME VARCHAR No
The source category ID for the patient's recorded body surface area (BSA) at the time the order was released.
May contain organization-specific values: Yes
Category Entries:
Treatment plan BSA
Based on weight and height measured in the past
Based on most recent measured weight and height (actual)
User Entry
Based on ideal body weight and current/past measured height
Based on adjusted body weight and current/past measured height
Based on dosing weight and current/past measured height
Based on normalized BSA 1.7m2
Birth weight and current height BSA
Other
