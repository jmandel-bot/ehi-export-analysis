GUAR_PAT_BILL_INV_HX
Description:
This table contains information about pending authorized user invitations to a guarantor account.

Primary Key
Column Name Ordinal Position
ACCT_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 ACCT_ID NUMERIC No
The unique identifier (.1 item) for the guarantor record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 BILL_ACSS_INV_CODE VARCHAR No
This item stores a unique code used to invite a user to become an authorized user on this guarantor.
4 BILL_ACCS_INV_UTC_DTTM DATETIME (UTC) No
This item stores the instant when an authorized user invite was sent.
5 BIL_ACS_INV_MYPT_ID VARCHAR No
This item stores the WPR of the authorized user that is invited, if applicable.
6 BILL_ACCS_ATMPT_CNT INTEGER No
This items stores the number of times a user attempts to confirm an authorized user invite offer.
7 BILL_ACSS_INV_NAME VARCHAR No
This item stores the name of the person the guarantor invited to become an authorized user.
8 BILL_ACSS_INV_EMAIL VARCHAR No
This item stores the email of the person the guarantor invited to become an authorized user.
9 BILL_ACSS_INV_PHONE VARCHAR No
This item stores the phone number of the person the guarantor invited to become an authorized user.
10 BILL_ACSS_INV_SYS_C_NAME VARCHAR No
Stores the billing system of the guarantor account that the authorized user is invited to.
May contain organization-specific values: No
Category Entries:
Professional Billing
Hospital Billing
Hospital & Professional Billing
External
11 BILL_ACCS_INV_REC_STMT_YN VARCHAR No
This items stores whether the MyChart account with billing access to the guarantor account should receive paper copies of statements once the invited authorized user accepts the authorized user invite offer.
May contain organization-specific values: No
Category Entries:
No
Yes
