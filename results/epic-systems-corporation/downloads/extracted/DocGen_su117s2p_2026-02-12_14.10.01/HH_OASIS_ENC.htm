HH_OASIS_ENC
Description:
Contains Home Health Outcome and Assessment Information Set (OASIS) and Hospice Item Set (HIS) overtime single items.

Primary Key
Column Name Ordinal Position
OASIS_SET_ID 1
CONTACT_DATE_REAL 2

Column Information
Name Type Discontinued?
1 OASIS_SET_ID NUMERIC No
Unique identifier for the OASIS data set.
2 CONTACT_DATE_REAL FLOAT No
Unique identifier for this contact for this patient.
3 CONTACT_NUM INTEGER No
Contact number.
4 CONTACT_DATE DATETIME No
The date of this contact in calendar format.
5 INSTANT_OF_ENTRY DATETIME (Local) No
Instant of OASIS data set data entry.
6 DATA_ENTRY_PERSON VARCHAR No
Name of user who entered data in OASIS data set.
7 DATASET_CONTACT_TYPE_C_NAME VARCHAR No
This item stores the OASIS/HIS dataset type.
May contain organization-specific values: No
