HSPC_BENEFIT_PRDS
Description:
This table contains hospice benefit period information.

Primary Key
Column Name Ordinal Position
CVG_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 CVG_ID NUMERIC No
The unique identifier for the coverage record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 HSPC_START_DT DATETIME No
This item stores the start date for any hospice benefit periods.
4 HSPC_END_DT DATETIME No
This item stores the end date for any hospice benefit periods.
5 HSPC_CTI_STATUS_C_NAME VARCHAR No
This item stores the signature status of the certificate of terminal illness for the hospice benefit period.
May contain organization-specific values: Yes
Category Entries:
Signed
Not signed
6 HSPC_CTI_SIGN_DT DATETIME No
This item stores the date that the certificate of terminal illness (CTI) was signed for each hospice benefit period.
7 HSPC_COMMENTS VARCHAR No
Free text comments about hospice benefit periods. Could be used to indicate where the patient received care for this benefit period, if at an outside agency.
