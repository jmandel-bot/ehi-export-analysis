ORDER_CASE_PANEL_INFO
Description:
This table stores data about panels for case request orders.

Primary Key
Column Name Ordinal Position
ORDER_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 ORDER_ID NUMERIC No
The unique identifier (.1 item) for the order record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 REQUESTED_PROC_MINUTES INTEGER No
This item contains the surgeon's requested total procedure length in minutes for a panel in a case request order.
4 PROC_LENGTH_MODIFIED_YN VARCHAR No
This item stores whether the requested panel length is a user-modified value.
May contain organization-specific values: No
Category Entries:
No
Yes
5 REQUESTED_PROC_COMMENTS VARCHAR No
This item contains the surgeon's comments about a requested total procedure length for a panel in a case request order.
6 PANEL_CONFIDENCE_C_NAME VARCHAR No
Confidence score for the estimated time for a panel generated by Epic
May contain organization-specific values: No
Category Entries:
No Data
Modified
Low Confidence
Medium Confidence
High Confidence
Low - Preference Card
Low - Individual Procedures
Low - No Cases
Low - High Variability
Low - Few Stored Cases
Low - Possible Procedures
Low - Estimates Used All Lateralities
Medium - Moderate Variability
High - Low Variability
7 PAN_DURATION_ESTIMATE_MINUTES INTEGER No
Estimated panel duration generated by Epic
8 PANEL_REQ_LEN_USER_ID VARCHAR No
This item tracks the last user to modify the requested total procedure length in minutes for a panel in a case request order so that the default length is not used.
9 PANEL_REQ_LEN_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
