CHART_REVIEW_BOOKMARKS
Description:
This table contains bookmarked content.

Primary Key
Column Name Ordinal Position
BOOKMARK_LIST_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 BOOKMARK_LIST_ID NUMERIC No
The unique identifier for the bookmark list record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 CMT VARCHAR No
The user-entered comment for each bookmark.
4 ENC_CSN NUMERIC No
The contact serial number (CSN) for each bookmarked encounter.
5 LETTER_LN INTEGER No
The line number in the item for letter status for each bookmarked letter record.
6 NOTE_ID VARCHAR No
The note identification number for every bookmarked note.
7 ORDER_ID NUMERIC No
The order identification number for each bookmarked order.
8 COMPONENT_IDX INTEGER No
The index of each bookmarked result component.
9 DOC_INFO_ID VARCHAR No
The document identification number for every bookmarked piece of media.
10 EPISODE_ID NUMERIC No
The episode identification number for every bookmarked episode.
11 LDA_ID VARCHAR No
The LDA identification number for each bookmarked LDA (Lines, Drains, and Airways) record.
12 REFERRAL_ID NUMERIC No
The referral ID for every bookmarked referral.
13 ADDED_UTC_DTTM DATETIME (UTC) No
The instant at which an item was bookmarked.
14 OUTSIDE_ENC VARCHAR No
The base Care Everywhere reference ID of each outside encounter that has been bookmarked.
15 OUTSIDE_NOTE_LOCAL_IDENT VARCHAR No
The locally-assigned unique identifier of the bookmarked external note.
16 OUTSIDE_NOTE_DOCUMENT_CSN_ID NUMERIC No
The ID of the bookmarked note's source DXR contact.
17 OUTSIDE_EPISODE_IDENT VARCHAR No
The episode ID of the bookmarked external episode.
