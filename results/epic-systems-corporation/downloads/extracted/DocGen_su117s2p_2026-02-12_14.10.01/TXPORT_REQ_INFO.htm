TXPORT_REQ_INFO
Description:
This table contains information about transport requests.

Primary Key
Column Name Ordinal Position
TRANSPORT_ID 1

Column Information
Name Type Discontinued?
1 TRANSPORT_ID NUMERIC No
The unique ID of the transport request record.
2 TRANSPORT_NAME VARCHAR No
The name of the transport request record.
3 PAT_ID VARCHAR No
The unique ID of the patient to be transported.
4 TXPORT_PAT_CSN NUMERIC No
The unique contact serial number of the patient associated with this transport request. This is optional as there may not be a particular patient contact that the transport request is linked to.
5 TXPORT_TYPE_C_NAME VARCHAR No
The transport type category number for the transport request.
May contain organization-specific values: No
Category Entries:
Patient
Non-patient
Cleaning Inspection
Non-patient Bundle
6 TXPORT_DATE DATETIME No
The date on which the transport request needs to be completed.
7 TXPORT_TIME DATETIME (Local) No
The time at which the transport request needs to be completed.
8 TXPORT_FROM_PLF_ID_RECORD_NAME VARCHAR No
The name of the PLF record.
9 TXPORT_TO_PLF_ID_RECORD_NAME VARCHAR No
The name of the PLF record.
10 ROUND_TRIP_YN VARCHAR No
Flag to indicate if this is a round trip request.
May contain organization-specific values: No
Category Entries:
No
Yes
11 TXPORT_MODE_C_NAME VARCHAR No
The transport mode category number for the transport request.
May contain organization-specific values: Yes
12 REQUEST_SOURCE_C_NAME VARCHAR No
The source of the transport request.
May contain organization-specific values: Yes
Category Entries:
Ad-Hoc
Transfer
Discharge
Admission
Scheduling
System generated
Leave of Absence
13 REQ_USER_ID VARCHAR No
The unique ID of the user who created the transport request.
14 REQ_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
15 LOC_ID_LOC_NAME VARCHAR No
The name of the revenue location.
16 CANCEL_USER_ID VARCHAR No
The unique ID of the user who canceled the transport request.
17 CANCEL_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
18 CANCEL_EVENT_DTTM DATETIME (UTC) No
The date and time when a transport request was canceled.
19 PRIORITY_C_NAME VARCHAR No
The priority of the transport request.
May contain organization-specific values: No
Category Entries:
Stat
High
Normal
Low
20 RECORD_CREATION_DT DATETIME No
The date on which the transport request was created.
21 RND_TRIP_RETURN_ID NUMERIC No
This item contains the linked return transport request for a round trip request.
22 RND_TRIP_ORIG_ID NUMERIC No
This item contains the linked original transport request for a round trip request.
23 TXPORT_FROM_TYPE_C_NAME VARCHAR No
Indicates if the patient is to be picked up at their current location or an override location specified.
May contain organization-specific values: No
Category Entries:
Use current location
Use override location
Use current bed
24 TXPORT_TO_TYPE_C_NAME VARCHAR No
Indicates if the patient is to be dropped off at their currently occupied bed or to a specific location.
May contain organization-specific values: No
Category Entries:
Use specified location
Back to bed
25 CURRENT_STATUS_C_NAME VARCHAR No
This item indicates the current status of the request.
May contain organization-specific values: No
Category Entries:
Pending
Assigned
In Progress
Delayed
Completed
Canceled
Future
Postponed
Acknowledged
26 TXPORT_ADMIT_CSN NUMERIC No
This item contains the contact serial number (CSN) of the admission contact. This is optional as there may not be a current admission contact for the patient.
27 PARENT_REQUEST_ID NUMERIC No
This item contains the parent transporter request of a multiple transporter request.
28 ASSOCIATED_BEV_ID NUMERIC No
This item contains the linked bed clean request (if any).
29 REGION_ID NUMERIC No
The unique ID of the region record that the transport request is for.
30 REGION_ID_RECORD_NAME VARCHAR No
The name of this cleaning sector.
31 TXPORT_PAT_OUT_CSN NUMERIC No
This item contains the contact serial number (CSN) of the patient contact that is linked to this transport request if the request is an outgoing request from an appointment. This may not always be populated if the request is not linked to an appointment contact.
32 SECTOR_ID NUMERIC No
This item contains the sector that the transport request is in.
33 SECTOR_ID_RECORD_NAME VARCHAR No
The name of this cleaning sector.
34 COMP_ACTION_RSLT_C_NAME VARCHAR No
Holds the result of the attempt to transfer or discharge a patient.
May contain organization-specific values: No
Category Entries:
Completed Successfully
Failed - Patient Locked
Failed - Validation Errors
Transported Patient Successful - Linked Patients Failed
Transported Patient Failed - Linked Successful
Failed - Primary and Linked Patients
Failed - Patient has Earlier Pending Transfers
35 REQ_DEP_ID_EXTERNAL_NAME VARCHAR No
The external name of the department record. This is often used in patient correspondence such as reminder letters.
36 TXPORT_PHONE_NUM VARCHAR No
This item contains the phone number associated with the transport request, stored as a short free-text string.
37 TNP_CREATE_SRC_C_NAME VARCHAR No
Specifies what made the request be created
May contain organization-specific values: No
Category Entries:
Manual
Appointment
Ready To Move
38 BED_TYPE_C_NAME VARCHAR No
This item tracks the bed type associated with this transport request.
May contain organization-specific values: Yes
Category Entries:
Empty
Standard
Crib
Telemetry
Bariatric

40 FOLLOW_UP_SOURCE_TRANSPORT_ID NUMERIC No
Holds the transport request record that triggered the creation of this follow-up task.
