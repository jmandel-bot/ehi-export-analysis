OR_LOG_ALL_PROC
Description:
The OR_LOG_ALL_PROC table contains OR management system log procedures.

Primary Key
Column Name Ordinal Position
LOG_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 LOG_ID VARCHAR No
The unique ID of the log referred to by this procedure.
2 LINE INTEGER No
The number of the line of procedure in the surgical log.
3 OR_PROC_ID VARCHAR No
The unique ID of the procedure.
4 OR_PROC_ID_PROC_NAME VARCHAR No
The name of the surgical procedure record.
5 POS_C_NAME VARCHAR No
The category value which represents the patient’s body position during the surgery in the surgical log.
May contain organization-specific values: Yes
6 ANES_TYPE_C_NAME VARCHAR No
The category value which represents the types of anesthesia needed for the surgery in the surgical log.
May contain organization-specific values: Yes
7 LRB_C_NAME VARCHAR No
The category value which represents what section of the body the surgery in the surgical log was performed on.
May contain organization-specific values: Yes
Category Entries:
Left
Right
Bilateral
N/A
Default
8 OP_REG_C_NAME VARCHAR No
The category value which represents what region of the body will be operated on during the surgery in the surgical log.
May contain organization-specific values: Yes
9 WND_CLS_C_NAME VARCHAR No
The category value which represents the wound class of the patient in the surgical log.
May contain organization-specific values: Yes
10 WND_LOC_C_NAME VARCHAR No
The category value which represents the body location of the wound on the patient in the surgical log.
The category values for this column were already listed for column: OP_REG_C_NAME
11 ALL_APPROACH_C_NAME VARCHAR No
The category value which represents the approach to the wound.
May contain organization-specific values: Yes
12 ALL_PROCS_TOT_TIME INTEGER No
The total time of all procedures in the log.
13 ALL_PROCS_PANEL INTEGER No
The number of the panel in which this procedure was performed.
14 COMMENTS VARCHAR No
The free text comments for this procedure.
15 RESOURCE_PREF_ID VARCHAR No
Stores the procedure/preference id which was used to default resources (staff, equipment, etc.) corresponding to this procedure.
16 RESOURCE_PREF_ID_PROC_NAME VARCHAR No
The name of the surgical procedure record.
17 PICKLIST_PREF_ID VARCHAR No
Stores the procedure/preference id whose pick list was copied to create the corresponding pick list for this procedure in the case.
18 PICKLIST_PREF_ID_PROC_NAME VARCHAR No
The name of the surgical procedure record.
19 MATCHED_PREF_ID VARCHAR No
Stores the preference card that has been selected to override the defaulted preference card. NOTE: Starting in the Epic 2012 release, this column will display the ID of the selected preference card even though the column name says Matched Pref. The ID of the preference card that is matched or defaulted will be stored in I ORL 2018.
20 MATCHED_PREF_ID_PROC_NAME VARCHAR No
The name of the surgical procedure record.
21 DBC_EPISODE_ID NUMERIC No
Stores the Diagnosis Behandling Combinatie (DBC) episode associated with the current procedure. Used in billing.
22 PROC_EAP_ID_PROC_NAME VARCHAR No
The name of each procedure.
23 ALL_PROC_AS_ORDERED VARCHAR No
Denotes the procedure name as exactly ordered by the surgeon/provider.
24 ALL_DEF_OR_PROC_ID VARCHAR No
Stores the defaulted preference card.
25 ALL_DEF_OR_PROC_ID_PROC_NAME VARCHAR No
The name of the surgical procedure record.
26 ALL_PROC_CODE_ID_PROC_NAME VARCHAR No
The name of each procedure.
27 ALL_PROCS_INCISION_CLOSURE_C_NAME VARCHAR No
The category value of the incision closure associated with the procedure.
May contain organization-specific values: No
Category Entries:
Deep and Superficial Layers
Superficial Layers Only
Deep Layers Only
No Layers Closed / Incision Left Open
No Incision / NA
28 ALL_PANEL_ADDL_ID NUMERIC No
Stores the pointer to the procedure additional data record for all panels.
29 ALL_PANEL_ORDER_ID NUMERIC No
This item stores the order ID for the procedure request associated with the log created for the procedure plan case.
