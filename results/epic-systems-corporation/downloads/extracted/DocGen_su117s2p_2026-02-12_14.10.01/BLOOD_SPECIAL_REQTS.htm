BLOOD_SPECIAL_REQTS
Description:
This table contains the list of the patient's current blood special requirements.

Primary Key
Column Name Ordinal Position
PAT_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 PAT_ID VARCHAR No
The unique ID of the patient record for this row. This column is frequently used to link to the PATIENT table.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 BLOOD_SPECIAL_REQTS_C_NAME VARCHAR No
The special requirements category ID for a patient who has special requirements for blood transfusions.
May contain organization-specific values: Yes
Category Entries:
Irradiated
CMV Negative
Leukoreduced
Sickle Cell (Hgb S) Negative
Concentrate
HLA Match
Solid-Phase Red-Cell Adherence
PLA-1 Negative
Cryoreduced
Washed
PAS
Plasma Compatible
4 BLOOD_SPECIAL_REQTS_DEGREE_C_NAME VARCHAR No
The category ID of the degree to which a blood special requirement is critical for patient care.
May contain organization-specific values: Yes
No Entries Defined
5 BLOOD_SPECIAL_REQTS_ADD_INFO VARCHAR No
This item stores additional information to describe a blood special requirement in cases where the name does not adequately describe the requirement. This item can also be used to describe why the requirement is necessary.
