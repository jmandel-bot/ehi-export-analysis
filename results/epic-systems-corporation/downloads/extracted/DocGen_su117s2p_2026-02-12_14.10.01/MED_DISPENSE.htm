MED_DISPENSE
Description:
This table contains information about a patient's med dispense history from a third-party interface. This information can be helpful for reviewing whether a patient is getting their prescriptions filled at the correct intervals.

Primary Key
Column Name Ordinal Position
DOCUMENT_ID 1
CONTACT_DATE_REAL 2
LINE 3

Column Information
Name Type Discontinued?
1 DOCUMENT_ID NUMERIC No
This item stores the Received Document record ID.
2 CONTACT_DATE_REAL FLOAT No
A unique contact date in decimal format. The integer portion of the number indicates the date of contact. The digits after the decimal distinguish different contacts on the same date and are unique for each contact on that date. For example, .00 is the first/only contact, .01 is the second contact, etc.
3 LINE INTEGER No
The line number for the information associated with this contact. Multiple pieces of information can be associated with this contact.
4 CONTACT_DATE DATETIME No
The date of this contact in calendar format.
5 EXT_DRUG_DESP VARCHAR No
This column stores the drug description in free text.
6 EXT_MED_REF_ID VARCHAR No
This column stores a reference identifier associated with each medication dispense.
7 EXT_DRUD_ID_STR VARCHAR No
This column stores the identifier of the drug.
8 EXT_MED_ERX_ID_MEDICATION_NAME VARCHAR No
The name of this medication record.
9 EXT_DRUG_CODE_SYS VARCHAR No
This column stores the coding system used for the drug identifier.
10 EXT_DRUG_DISP_INST_DTTM DATETIME (Local) No
This column stores the dispense instant.
11 EXT_DRUG_DISP_AMT INTEGER No
This column stores the actual dispense amount of the drug.
12 EXT_DRUG_DISP_UNIT VARCHAR No
This column stores the drug dispense unit ID.
13 EXT_DRUG_UNIT_TXT VARCHAR No
This column stores the free text name of the dispense unit.
14 EXT_DRUG_DOSE_FORM VARCHAR No
This column stores the drug dosage form when dispensed.
15 EXT_DRUG_RX_NUM VARCHAR No
This column stores the prescription number of the dispense.
16 EXT_DRUG_DSPPROV_ID NUMERIC No
This column stores the dispensing provider ID.
17 EXT_DRUG_DSPPROV_ID_PHARMACY_NAME VARCHAR No
The name of the pharmacy.
18 EXT_DRUG_PROV_NAME VARCHAR No
The dispensing provider name for the external dispense.
19 EXT_DRUG_UNIT_STR VARCHAR No
This column stores the actual unit strength.
20 EXT_DRUG_PHAR_PHON VARCHAR No
The pharmacy phone (or other contact) number associated with the external dispense.
21 EXT_MED_DAY_SUPPLY INTEGER No
This column stores the number of days the dispense is written for.
22 EXT_MED_ORDPROV_ID_PROV_NAME VARCHAR No
The name of the service provider. This item may be hidden in a public view of the CLARITY_SER table.
23 EXT_MED_ORD_PROVNAM VARCHAR No
The name of the ordering provider for this medication.
24 EXT_MED_ENT_ORG_NAM VARCHAR No
This column stores the entering organization name of the dispense data.
25 EXT_MED_ORD_ID NUMERIC No
This column stores the linked order ID for the corresponding order record in Epic, if one exists.
26 EXT_MED_DISP_UNT_C_NAME VARCHAR No
The mapped dose unit category ID for the external dispense.
May contain organization-specific values: Yes
