PATIENT_CONF_ADDR
Description:
This table contains the patient's confidential address information.

Primary Key
Column Name Ordinal Position
PAT_ID 1

Column Information
Name Type Discontinued?
1 PAT_ID VARCHAR No
The unique ID for the patient record.
2 CONF_CITY VARCHAR No
The city associated with the patient's confidential address.
3 CONF_STATE_C_NAME VARCHAR No
The category value corresponding to the state that is associated with the patient's confidential address.
May contain organization-specific values: Yes
4 CONF_COUNTRY_C_NAME VARCHAR No
The category value corresponding to the country that is associated with the patient's confidential address.
May contain organization-specific values: Yes
5 CONF_ZIP_CODE VARCHAR No
The Zip Code that is associated with the patient's confidential address.
6 CONF_CONTACT_PERSON VARCHAR No
The name of the person who should be contacted at the patient's confidential address.
7 CONF_EMAIL_ADDRESS VARCHAR No
The patient’s confidential e-mail address.
8 CONF_START_DATE DATETIME No
The date in which the patient's confidential address should take effect for the patient.
9 CONF_END_DATE DATETIME No
The date in which the patient's confidential address should expire for the patient.
10 CONF_COUNTY_C_NAME VARCHAR No
The category value corresponding to the county that is associated with the patient's confidential address.
May contain organization-specific values: Yes
11 CONF_HOUSE_NUM VARCHAR No
The house number associated with the patient's confidential address.
12 CONF_DISTRICT_C_NAME VARCHAR No
The category value corresponding to the district that is associated with the patient's confidential address.
May contain organization-specific values: Yes
