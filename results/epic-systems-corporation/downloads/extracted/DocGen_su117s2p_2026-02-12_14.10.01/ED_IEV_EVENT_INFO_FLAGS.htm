ED_IEV_EVENT_INFO_FLAGS
Description:
Holds the Event Value Flags for Events Database (IEV) Events.

Primary Key
Column Name Ordinal Position
EVENT_ID 1
GROUP_LINE 2
VALUE_LINE 3

Column Information
Name Type Discontinued?
1 EVENT_ID VARCHAR No
The unique identifier for the event record.
2 GROUP_LINE INTEGER No
The line number for the information associated with this event.
3 VALUE_LINE INTEGER No
The line number of one of the multiple flags associated with a specific group of data within this record.
4 EVENT_FLAGS_C_NAME VARCHAR No
The flags for a value that was added as an ED course comment, such as "abnormal" or "critical" for a lab result value.
May contain organization-specific values: No
