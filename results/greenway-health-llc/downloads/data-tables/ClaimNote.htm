Table Definition

Description: This table stores notes for patient claims. The Note will remain even if the claim is undone.

Field Datatype Default Null Option Comment
ClaimNoteSID INTEGER ?   MANDATORY System generated unique note id using the sequence ClaimNote.
Class CHARACTER(6)   MANDATORY System set note class. Note class codes are user defined
CCWorklistSID INTEGER ?   OPTIONAL FK - from CCWorklist table
PlanID INTEGER ?   OPTIONAL FK - from PlanClaim table
PolicyNumber CHARACTER(20)   OPTIONAL FK - from PlanClaim table
ClaimSID INTEGER ?   OPTIONAL FK - from PlanClaim table
Body CHAR(32000)   OPTIONAL User entered body of the note.
TimeStamp CHARACTER(14)   OPTIONAL System generated date/time stamp record was last written.
LastUser INTEGER ?   OPTIONAL User ID of user that last wrote the record.
CreateStamp CHARACTER(14)   OPTIONAL System generated date/time stamp record was created.
CreateUser INTEGER ?   OPTIONAL User ID of user that created the record.
