Table Definition

Description: This table stores one or more patient reference IDs for each PBM eligibility response.

Field Datatype Default Null Option Comment
RxEligPBMResponseSID INTEGER ?   MANDATORY System generated value from sequence RxEligibility.
PBMPatientIDQual CHAR(3)   MANDATORY Code to identify the type of ID
RxEligRequestSID INTEGER ?   MANDATORY The RxEligRequestSID at the time the record was created. No RFI to RxEligRequest.
RxEligPBMSID INTEGER ?   MANDATORY The RxEligPBMSID at the time the record was created. No RFI to RxEligPBM.
PBMPatientID CHAR(50)   OPTIONAL Some type of ID
PBMPatientIDDesc CHAR(80)   OPTIONAL A description for the PBMPatientID
TimeStamp CHARACTER(14)   OPTIONAL System generated date/time stamp record was last written.
LastUser INTEGER ?   OPTIONAL User ID of user that last wrote the record.
CreateStamp CHARACTER(14)   OPTIONAL System generated date/time stamp record was created.
CreateUser INTEGER ?   OPTIONAL User ID of user that created the record.
