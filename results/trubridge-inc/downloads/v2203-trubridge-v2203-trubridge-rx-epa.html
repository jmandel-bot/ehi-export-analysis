Escribe ePA

The TruBridge EHR platform Electronic Prior Authorization file output format.

Table of contents

1. The ePA file naming scheme

2. The detailed file format definition

File Naming Scheme:

The ePA files are named following the naming convention as follows:

Rx_spid_profile_visit_xmlFileName_CreationDate_CreationTime

An example of this naming convention:

Rx_1_12345678_123456_SSEPA00004940_1_2_sent_20230101_1200000000

Detailed File Format:

The file output consists of XML data in the NCPDP format(NCPDP SCRIPT Version 2013101).
