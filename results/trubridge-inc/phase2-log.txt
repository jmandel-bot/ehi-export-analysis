       <h1>TruBridge EHR v2203 Overview</h1>
<p>The TruBridge EHR Electronic Health Information (EHI) Export feature allows users to generate a data extract containing all Protected Health Information (PHI) stored in the TruBridge EHR for a single patient or a population of patients. The export generates four main categories of data:</p>
<ul>
<li>Clinical Data</li>
<li>Full Database Export</li>
<li>Images and Documents</li>
<li>Insurance and Eligibility</li>
</ul>

<th>Category</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td>Clinical Data</td>
<td>The clinical data export consists of ancillary transcriptions, medical records transcriptions, electronic forms, medication history, point of care documentation and prescribing messages.</td>
</tr>
<tr>
<td>Full Database Export</td>
<td>The complete TruBridge EHR database is exported into a JSON formatted file including table, field, description and data. This format is described in depth.</td>
