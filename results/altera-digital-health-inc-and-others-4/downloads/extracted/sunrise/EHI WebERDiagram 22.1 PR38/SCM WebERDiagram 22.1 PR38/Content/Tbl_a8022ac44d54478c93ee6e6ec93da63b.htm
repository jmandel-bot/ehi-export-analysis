dbo.CV3AllergyDeclTemplate 
Default Table Name  CV3AllergyDeclTemplate 
Definition  The template of an allergy. This allows allergies to be entered from the Knowledge Tree. When entered from the Knowledge Tree (client document), this template is used to provide default values.
