dbo.CV3HealthIssueTypeStatusXRef 
Default Table Name  CV3HealthIssueTypeStatusXRef 
Definition  Cross-reference table that associates a Health Issue status record to an existing CV3HealthIssueType record. Used to list of set of allowable health issues statuses associated with a health issue type.
