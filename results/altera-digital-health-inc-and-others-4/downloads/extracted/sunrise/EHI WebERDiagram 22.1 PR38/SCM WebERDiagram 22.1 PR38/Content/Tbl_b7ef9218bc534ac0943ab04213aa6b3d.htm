dbo.SXAHMWorkingViewColumnConfiguration 
Default Table Name  SXAHMWorkingViewColumnConfiguration 
Definition  This Health Management configuration table stores column configurations for the working view to determine which columns will display, and in what sequence. 

dbo.SXAHMWorkingViewColumnConfiguration Columns 
ColumnName  Domain  Datatype  NULL  Definition 
Build    int  NO  Identifies the build number and patch level of the software that inserted/updated this row. The format is the Build number followed by 3 digits for the patch level which is typically 000. The build number is tied to specific versions of the software, e.g. Sunrise 15.3 is Build 5021, Sunrise 16.3 is Build 5503. 
TouchedBy    varchar(50)  NO  Information about the user that last updated the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
TouchedWhenUTC    datetimeoffset(3)  NO  The date and time that the row was last updated including the time zone offset from UTC, for example 2015-01-30 04:55:04.372 -05:00. The date and time is based on the Enterprise Time Zone. 
CreatedBy    varchar(50)  NO  Information about the user that created the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
CreatedWhenUTC    datetimeoffset(3)  NO  The date and time that the row was created including the time zone offset from UTC, for example 2015-01-30 04:55:04.372 -05:00. The date and time is based on the Enterprise Time Zone. 
Active    bit  NO  Denotes whether the row is active or not, 0 means the row has been soft deleted or deactivated. 
WorkingViewColumnConfigurationID    int  NO  Auto generated unique primary key value. 
Name    varchar(100)  NO  This is the name of this column configuration, which must specify the intended purpose of this column configuration to the user. 
Description    varchar(2000)  YES  The detailed description of the purpose of this column configuration. This is an optional field in which the user can specify as much information as they require. 
IsSXADefined    bit  NO  This flag identifies if this column is system defined, and therefore read only. 
UserID    HVCIDdt  NO  Links the the user that created this custom column configuration.
