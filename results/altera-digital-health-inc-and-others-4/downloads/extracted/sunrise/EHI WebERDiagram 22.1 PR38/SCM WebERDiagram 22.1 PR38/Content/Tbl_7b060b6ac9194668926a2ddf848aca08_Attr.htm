SiteID 
Definition  Identifies the unique distributed database ID where the row was inserted. Distributed databases are no longer supported so this column is deprecated. 
Build 
Definition  Identifies the build number and patch level of the software that inserted/updated this row. The format is the Build number followed by 3 digits for the patch level which is typically 000. The build number is tied to specific versions of the software, e.g. Sunrise 15.3 is Build 5021, Sunrise 16.3 is Build 5503. 
TouchedBy 
Definition  Information about the user that last updated the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
TouchedWhen 
Definition  The date and time that the row was last updated and will be based on the Enterprise time zone. 
CreatedBy 
Definition  Information about the user that created the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
CreatedWhen 
Definition  The date and time that the row was created and will be based on the Enterprise time zone. 
PharmacyLocationConfigID 
Definition  Auto generated unique primary key value. 
PharmacyID 
Definition  A foreign key to the PharmacyID column in the SXAMMPharmacy table. 
PharmacyLocationName 
Definition  Name of the location in the pharmacy (shelf A, shelf B, drawer 101, etc) 
ParentPharmacyLocationConfigID 
Definition  A foreign key to the PharmacyLocationConfigID column in the SXAMMPharmacyLocationConfig table. 
MSrepl_tran_version 
Definition  Optimistic lock identifier. 
PharmacyLocationPath 
Definition  Identifies the full path for this pharmacy location. It is stored as a pipe separated string. 
IsRobot 
Definition  Identifies if the Pharmacy Location is a Robot Location. 
RobotCommunicationProcessGUID 
Definition  Identifies the Communication Process that is to be used when sending the dispense request to the Robot.  Links to the CV3CommunicationProcess table. 
RobotManagesBilling 
Definition  Identifies if the Robot responsible for billing the patient for the items that are dispensed via the Robot. 
RobotManagesInitialDose 
Definition  Indicates if this robot also manages initial doses 
PharmacyPrintRobotLabels 
Definition  if true, pharmacy will print medication labels for robot orders requiring initial fills 
PharmacyPrintRobotTransferLabels 
Definition  true if pharmacy should print medication labels for robot orders requiring fills as a result of patient transfer 
RobotManagesTransferDose 
Definition  true if Patient Transfer generated initial fill HL7 messages should be dispatched to the robot 
PrintRobotFillLabels 
Definition  if true then fill service will print robot labels 
MSReplrowguid 
Definition  Unique GUID used to support merge replication.
