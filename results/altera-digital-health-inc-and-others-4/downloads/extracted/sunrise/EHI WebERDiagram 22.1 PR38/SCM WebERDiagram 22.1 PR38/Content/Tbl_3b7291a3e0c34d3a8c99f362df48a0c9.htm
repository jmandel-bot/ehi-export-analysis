dbo.SXAMMMnemonicParameter 
Default Table Name  SXAMMMnemonicParameter 
Definition  The SXAMMMnemonicParameter table will contain configuration data for Mnemonics that have a Start Time that is Event-Based or a SIG that is either Event-Based, User Schedule, Variable Interval, QxH, or QxM
