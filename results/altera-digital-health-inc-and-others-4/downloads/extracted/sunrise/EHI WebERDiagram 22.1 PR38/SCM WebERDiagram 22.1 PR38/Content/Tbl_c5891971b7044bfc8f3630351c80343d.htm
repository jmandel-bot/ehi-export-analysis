dbo.SXARCMNonReceivableTransactionCode 
Default Table Name  SXARCMNonReceivableTransactionCode 
Definition  This table is used to store non AR Transaction Type (charge, payment, adjustment) which is used to record a Adjustment which does not decrease the value of the Account’s Receivable for the enterprise
