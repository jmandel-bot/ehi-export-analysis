dbo.SXAMMMessageStatus 
Default Table Name  SXAMMMessageStatus 
Definition  This entity contains the facility defined Statuses for a Message Type for each Department. For example; for a MessageStatusType = 0 (New) the MessageDeptType = 1 (Nursing) may want to call their 'New' message status 'Initial'.
