dbo.SXARCMPPS3MDictionaryDefaultValuesXREF 
Default Table Name  SXARCMPPS3MDictionaryDefaultValuesXREF 
Definition  This table identifies which row within the SXARCMPPS3MDictionaryValues table stores the default value of a dictionary field for a particular 3M Reimbursement Scheme. The data for the table is loaded by a system defined data load script.
