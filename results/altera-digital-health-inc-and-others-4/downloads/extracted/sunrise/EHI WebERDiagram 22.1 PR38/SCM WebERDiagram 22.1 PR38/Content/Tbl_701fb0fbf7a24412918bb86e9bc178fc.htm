dbo.CV3ObservationDocumentCUR 
Default Table Name  CV3ObservationDocumentCUR 
Definition  Contains current observation records. This table stores extended information for the Observation table and it has multiple records per observation when fanout has occurred. XREF relating CV3Observation and CV3ClientDocument. Observations are recorded as part of a client document.
