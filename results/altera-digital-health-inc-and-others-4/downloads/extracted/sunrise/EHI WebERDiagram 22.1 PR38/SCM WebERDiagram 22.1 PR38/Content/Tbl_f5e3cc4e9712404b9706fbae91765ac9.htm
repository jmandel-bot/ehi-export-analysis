dbo.SXAAMVisitTypePrefix 
Default Table Name  SXAAMVisitTypePrefix 
Definition  In Patient ID Type config, for each ID Code Class = Visit, we are allowing the user to specify a prefix for each Visit Type via another dialog. The existing CV3ClientIDType.Prefix column will be treated as a default for Visit Types not associated with specific prefixes.
