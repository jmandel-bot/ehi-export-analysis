dbo.SXAAMTransportLocation 
Default Table Name  SXAAMTransportLocation 
Definition  This table maintains transpiration locations. Transportation locations have associated address and can be configured through config tools or through the application. Those configured through the application have the IsFreeText flag set to true, which will prevent them from showing up in config tools.
