dbo.SXAAMBillCode 
Default Table Name  SXAAMBillCode 
Definition  This table is a new restricted dictionary for 4 types of Billing Codes; Occurrence Codes, Occurrence Span Codes, Condition Codes, and Value Codes. These will be enumerated in CV3EnumReference.The codes are used to determine liability, coordinate benefits and administer clauses in benefit programs. Some will be predefined and unchangeable. The user will be able to add, modify, and delete user defined codes, but not system codes
