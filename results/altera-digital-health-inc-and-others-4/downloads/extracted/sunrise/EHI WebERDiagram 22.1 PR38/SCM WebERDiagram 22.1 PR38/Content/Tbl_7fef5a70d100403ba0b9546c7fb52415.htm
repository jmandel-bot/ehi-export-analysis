dbo.CV3ClientDocProviderXrefARC 
Default Table Name  CV3ClientDocProviderXrefARC 
Definition  Contains older document records. This table is a cross-reference table used to link a CV3ClientDocument with one or more CV3CareProviders used as authors of a document.
