dbo.SXAESEventPrePostRequisite 
Default Table Name  SXAESEventPrePostRequisite 
Definition  Events now allow a pre or post requisite to be defined. A pre/post requisite is another event that is required to be scheduled before or after the event it is defined for. This table will store these relationships.
