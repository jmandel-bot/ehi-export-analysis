dbo.CV3UserAlertDeclaration 
Default Table Name  CV3UserAlertDeclaration 
Definition  This is a parallel table to the CV3AlertDeclaration, required to avoid conflicts arising with existing reports that run against the CV3AlertDeclaration table..
