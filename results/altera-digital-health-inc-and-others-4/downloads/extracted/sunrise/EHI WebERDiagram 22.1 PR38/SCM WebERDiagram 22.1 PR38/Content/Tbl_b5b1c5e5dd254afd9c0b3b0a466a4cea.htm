dbo.SXACCHierarchicalConditionsCategory 
Default Table Name  SXACCHierarchicalConditionsCategory 
Definition  The table will store HCC ( Hierarchical Category codes ) compliant problems, the type of insurance linked to these problems. It will be updated via the TermSync process, which is responsible for syncing the ATP content with SCM health issue related tables.
