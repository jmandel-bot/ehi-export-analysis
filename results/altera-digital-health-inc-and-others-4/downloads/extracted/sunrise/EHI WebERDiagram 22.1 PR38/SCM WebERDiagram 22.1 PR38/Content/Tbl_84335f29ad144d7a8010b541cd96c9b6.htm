dbo.SXAMMGIDoseConversion 
Default Table Name  SXAMMGIDoseConversion 
Definition  This entity contains a list of Dose Conversions for the Generic Item.The Units of Measure used for the Dose Conversions must all be equal to each other. For example; The Dose Conversions for Ibuprofen 200mg Oral Tablet could be 1 TAB = 200 MG.
