dbo.SXAAMEDRegistrationDefault 
Default Table Name  SXAAMEDRegistrationDefault 
Definition  This table will contain the default values that are used in an ED registration.This is for specification 03-4003: ED Visit Registration and Express Registration Changes. The goal of these functions is to allow Emergency Department registration to occur as rapidly and efficiently as possible with minimum information.
