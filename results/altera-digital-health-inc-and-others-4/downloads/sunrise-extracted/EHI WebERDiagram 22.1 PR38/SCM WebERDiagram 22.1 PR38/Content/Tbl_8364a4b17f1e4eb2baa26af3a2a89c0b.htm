dbo.SXAGNMessageSeverity 
Default Table Name  SXAGNMessageSeverity 
Definition  Stores information regarding messages that are part of AM’s new Message Management feature. The information stored here contains the messages severity level.

The format is the Build number followed by 3 digits for the patch level which is typically 000. The build number is tied to specific versions of the software, e.g. Sunrise 15.3 is Build 5021, Sunrise 16.3 is Build 5503.
