dbo.SXAMMMessageDefaultText 
Default Table Name  SXAMMMessageDefaultText 
Definition  This entity contains Default Message Text for a Department. The Default Message Text provides an easy way to create a new message. For example; the MessageDeptType = 1 (Nursing) may want to have a MessageDefaultText = ‘Please send 1 additional IV bag – IV bag contaminated.’
