dbo.SXAMMMnemonic 
Default Table Name  SXAMMMnemonic 
Definition  This entity contains order entry defaults for a formulary Generic Item.
A Generic Item Formulary can have:
ONE Default Type = 'Formulary' and UNLIMITED Default Type = 'Mnemonic'.
