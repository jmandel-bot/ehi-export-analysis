dbo.SXASCLSiteMapping 
Default Table Name  SXASCLSiteMapping 
Definition  This table stores the mapping definition between the CV3Location and the SCL Sites(Hospitals in the SCL Db) for Sunrise Charge Logic. It is a Many-to-One mapping. So multiple locations from CV3Location can be mapped to a single Hospital in SCL.
