dbo.SXAHMClientEventDefinition 
Default Table Name  SXAHMClientEventDefinition 
Definition  This Health Management patient data table holds the information that determines which series definition will be used for each event on the patient's schedule of events. Only one ClientEventDefinition row will be active for each EventID. The table also stores the event-level status.
