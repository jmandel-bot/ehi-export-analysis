dbo.SXARCMRelationshipTranslation 
Default Table Name  SXARCMRelationshipTranslation 
Definition  This table holds the Relationship Translations set up by the user using the Relationship Translation screen (SFM Config --> Dictionary Maintanence --> Billing / Receivable - General --> Claim Population --> Relationship Translation).
This translation translates SCM Relationship Dictionary Codes to codes acceptable by 837 claims
