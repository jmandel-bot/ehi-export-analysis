dbo.SXASNSectionProviderXref 
Default Table Name  SXASNSectionProviderXref 
Definition  This table is a xref between CareProvider and SN Sections. All providers that are required to sign a given SN will have an entry in this table. This table is similar to the CV3ClientDocProviderXref table except that Structured Notes save data in BOTH these tables as the CV3ClientDocProviderXref table does not have a way to track signatures per section
