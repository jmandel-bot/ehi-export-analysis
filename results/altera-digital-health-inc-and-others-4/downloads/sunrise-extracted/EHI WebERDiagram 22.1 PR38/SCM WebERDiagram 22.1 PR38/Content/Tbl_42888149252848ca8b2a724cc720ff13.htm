dbo.SXACRCompositeTileXref 
Default Table Name  SXACRCompositeTileXref 
Definition  save the instance information of composite tiles. When a composite tile is created in the Configuration tool, the instance is stored here. This table stores the parent-child relationship to indicate which tiles are part of the composite tile and it's display order.
