dbo.SXAAMBAllergyCategory 
Default Table Name  SXAAMBAllergyCategory 
Definition  For the Ambulatory Rx Writer, allergy/drug interaction checking will need to be performed when a new drug is prescribed for a patient. This table will be populated by the external drug information provider (Multum, First Data Bank, etc).Specifically, it will provide a list of the allergy categories provided by the vendor.
