dbo.SXACRCompositeTileDefinition 
Default Table Name  SXACRCompositeTileDefinition 
Definition  Contains the default definition of Composite Tiles. This table is used to provide composite tile templates. At this point, the table holds only Eclipsys supplied templates. The table will have a parent tile category (linked to the Tile Category Template Table, which is SXACRTileDefault) containing child tile categories (again child tiles are linked to SXACRTileDefault)
