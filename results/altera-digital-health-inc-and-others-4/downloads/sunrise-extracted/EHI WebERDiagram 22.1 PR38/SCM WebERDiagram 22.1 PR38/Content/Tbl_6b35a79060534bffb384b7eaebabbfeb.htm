dbo.SXASHParameterValue 
Default Table Name  SXASHParameterValue 
Definition  This table contains user configured values associated with each parameter. The values for the parameter are set and configured 

dbo.SXASHParameterValue Columns 
ColumnName  Domain  Datatype  NULL  Definition 
SiteID    smallint  NO  Identifies the unique distributed database ID where the row was inserted. Distributed databases are no longer supported so this column is deprecated. 
Build    int  NO  Identifies the build number and patch level of the software that inserted/updated this row. The format is the Build number followed by 3 digits for the patch level which is typically 000. The build number is tied to specific versions of the software, e.g. Sunrise 15.3 is Build 5021, Sunrise 16.3 is Build 5503. 
TouchedBy    varchar(128)  NO  Information about the user that last updated the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
TouchedWhen    datetime  NO  The date and time that the row was last updated and will be based on the Enterprise time zone. 
CreatedBy    varchar(128)  NO  Information about the user that created the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
CreatedWhen    datetime  NO  The date and time that the row was created and will be based on the Enterprise time zone. 
ParameterValueID    bigint  NO  Auto generated unique primary key value. 
UserID    bigint  NO  A foreign key to the UserID column in the SXASHUser table. 
ParameterID    int  NO  A foreign key to the ParameterID column in the SXASHParameter table. 
ParameterValue    varchar(2000)  NO  The value that is assigned to the parameter. 
MSrepl_tran_version    uniqueidentifier  NO  Optimistic lock identifier. 
MSReplrowguid    uniqueidentifier  NO  Unique GUID used to support merge replication.
