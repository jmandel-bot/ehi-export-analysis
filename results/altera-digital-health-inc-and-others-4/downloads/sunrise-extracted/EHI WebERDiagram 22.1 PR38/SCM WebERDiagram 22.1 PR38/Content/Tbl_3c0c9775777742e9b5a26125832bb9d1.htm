dbo.SXAAMPredefinedComment 
Default Table Name  SXAAMPredefinedComment 
Definition  This new table is required to support Predefined comments. Specifically, it will serve as the datastore for a new standard dictionary entitled “Predefined Comments”. The table will contain user-defined comments that can later be selected for quick selection and submission in the comments dialog as well as any other applicable dialogs
