dbo.SXAAMVirtualEncounterDefault 
Default Table Name  SXAAMVirtualEncounterDefault 
Definition  Default values for Visit Type, Care Level, Service, Assigned Location and Attending Provider are set up for various workstation locations so visits of virtual encounter types may be created automatically from those locations with these default values
