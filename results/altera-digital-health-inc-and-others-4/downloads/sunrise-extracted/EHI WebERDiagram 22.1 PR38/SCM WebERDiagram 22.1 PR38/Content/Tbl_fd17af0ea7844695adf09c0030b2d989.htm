dbo.SXAMMStockItemLocationFormulary 
Default Table Name  SXAMMStockItemLocationFormulary 
Definition  This entity contains a list of Stock Items that can be dispensed for a location. The definition of multi-formulary is; that a stock item can be dispensed to a certain location. A formulary is location specific, within each formulary (location) the user defines which stock items can be dispensed.
