dbo.SXACDDocExSectionSourceMap 
Default Table Name  SXACDDocExSectionSourceMap 
Definition  Configuration table for ARRA Document Exchange section source mapping. This will be populated via a new config tool dialog in Dictionary Maintenance/Patient Care Documentation.
