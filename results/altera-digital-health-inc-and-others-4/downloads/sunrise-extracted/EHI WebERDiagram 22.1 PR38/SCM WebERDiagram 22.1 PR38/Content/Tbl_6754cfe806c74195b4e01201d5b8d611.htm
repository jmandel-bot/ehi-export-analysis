dbo.SXAEISEntityUniquenessConstraintXref 
Default Table Name  SXAEISEntityUniquenessConstraintXref 
Definition  Meta Data table that associates a query with an entity. The query determines unique records within the entity, thus functioning as a uniqueness constraint for the entity

The build number is tied to specific versions of the software, e.g. Sunrise 15.3 is Build 5021, Sunrise 16.3 is Build 5503.
