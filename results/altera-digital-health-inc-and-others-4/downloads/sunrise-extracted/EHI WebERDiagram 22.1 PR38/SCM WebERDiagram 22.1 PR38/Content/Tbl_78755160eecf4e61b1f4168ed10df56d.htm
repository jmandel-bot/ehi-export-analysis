dbo.CV3OrderReconcileOrderXRef 
Default Table Name  CV3OrderReconcileOrderXRef 
Definition  Cross reference items that were actioned in an order reconciliation with the items(s) they were reconciled with. One ‘From’ item can be reconciled with multiple ‘To’ items Multiple ‘From’ items can be reconciled with the same ‘To’ iteam.
