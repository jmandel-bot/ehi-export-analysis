dbo.SXARCMTemplateSource 
Default Table Name  SXARCMTemplateSource 
Definition  Stores the valid sources for a template. Templates provided by Eclipsys which be marked as Eclipsys and the ones which will be created by end users will be saved as ‘Enterprise’
