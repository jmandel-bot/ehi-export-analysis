dbo.SXAAMVisitDisplayColumn 
Default Table Name  SXAAMVisitDisplayColumn 
Definition  This table will contain the columns that will be used to display in the Visit Display grid.Provide the ability to show a listing of the patient’s visits and allow the user to select and place the visit into context. This will solve the workflow problem in which a user had to go to Find Patient again in order to get access to the patient’s other visits
