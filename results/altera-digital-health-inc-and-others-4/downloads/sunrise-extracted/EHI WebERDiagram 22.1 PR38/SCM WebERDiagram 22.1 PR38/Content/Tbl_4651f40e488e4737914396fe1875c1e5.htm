dbo.SXADMTnmgChoice 
Default Table Name  SXADMTnmgChoice 
Definition  Stores the different choices that are available for the user to select from in order to assist in classifying the stage of the disease. For cancerous diseases, there are four metrics used: T-size, N-lympth nodes, M-metasis, G-grade. This table holds the possible values that should be displayed to the user for each of these metrics.
