dbo.CV3AlertGroup 
Default Table Name  CV3AlertGroup 
Definition  Table that holds the values for Rule Group user dictionary. This table allows for rule group and a rule subgroup to be associated with a functional area. This later on can be used as a filter on queried for alerts.
