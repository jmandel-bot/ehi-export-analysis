dbo.SXAMMOrderSetMnemonicXREF 
Default Table Name  SXAMMOrderSetMnemonicXREF 
Definition  This entity contains details of items comprising an order set. An order set can contain a collection of either
- existing mnemonics
- order set specific mnemonics (identified by the "Is From Order Set" flag)
Existing mnemonics may be included in multiple order sets but are not editable (within the context of the order set maintenance).
Order set specific mnemonics are created as part of the order set definition and can only be included in one order set, and they are editable (only within the context of the order set maintenance).
