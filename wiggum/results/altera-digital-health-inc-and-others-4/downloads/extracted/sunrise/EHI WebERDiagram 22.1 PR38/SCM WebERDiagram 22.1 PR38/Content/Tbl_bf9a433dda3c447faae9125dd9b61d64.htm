dbo.SXACRTileColumnDisplay 
Default Table Name  SXACRTileColumnDisplay 
Definition  Clinical Review Tile Column Display names and widths. This table will have a master list of columns for each tile and the column widths. For example, for an allergy tile, this table would have 3 columns defined with column widths for the grid display
