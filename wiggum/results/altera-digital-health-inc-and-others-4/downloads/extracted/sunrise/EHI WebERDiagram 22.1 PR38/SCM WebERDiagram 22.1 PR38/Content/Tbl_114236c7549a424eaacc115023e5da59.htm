dbo.SXARCMRevenueCycleDetailDischargeDispositionXREF 
Default Table Name  SXARCMRevenueCycleDetailDischargeDispositionXREF 
Definition  The information contained in this table allows the billing process to determine the correct length of stay value to use in expected reimbursement calculation and when reporting value code 80 on a claim. Some payers allow for the discharge date to be counted in the length of stay if the patient has expired. The expiration flag is on the discharge disposition and thus this table is here to allow greater control on which discharge dispositions are to be considered for this calculation.
