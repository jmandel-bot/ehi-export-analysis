SiteID 
Definition  Identifies the unique distributed database ID where the row was inserted. Distributed databases are no longer supported so this column is deprecated. 
Build 
Definition  Identifies the build number and patch level of the software that inserted/updated this row. The format is the Build number followed by 3 digits for the patch level which is typically 000. The build number is tied to specific versions of the software, e.g. Sunrise 15.3 is Build 5021, Sunrise 16.3 is Build 5503. 
TouchedBy 
Definition  Information about the user that last updated the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
TouchedWhen 
Definition  The date and time that the row was last updated and will be based on the Enterprise time zone. 
CreatedBy 
Definition  Information about the user that created the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
CreatedWhen 
Definition  The date and time that the row was created and will be based on the Enterprise time zone. 
Active 
Definition  Denotes whether the row is active or not, 0 means the row has been soft deleted or deactivated. 
PharmacyID 
Definition  Auto generated unique primary key value. 
PharmacyName 
Definition  Identifies the name for the Pharmacy 
FacilityGUID 
Definition  Identifies the Links to CV3 Location linking the pharmacy to a facility. 
PharmacyLocationGUID 
Definition  Identifies physical location within the facility that the Pharmacy is located in ie. "Basement | Room A". 
UsesPrePrintedLabels 
Definition  Indicates if the pharmacy uses labels with a preprinted header 
OutpatientLabelCopiesNum 
Definition  Identifies the default number of label copies to be printed 
OutpatientLabelPrinterGUID 
Definition  Identifies the Label Printer that is to be used for the Outpatient Labels. Links to the CV3LogicalPrinter table. 
OutpatientMonographCopiesNum 
Definition  Identifies the default number of monograph copies to be printed 
OutpatientMonographPrinterGUID 
Definition  Identifies the Monograph Printer that is to be used for the Outpatient Monograph. Links to the CV3LogicalPrinter table. 
OutpatientReceiptCopiesNum 
Definition  Identifies the default number of receipt copies to be printed 
OutpatientReceiptPrinterGUID 
Definition  Identifies the Receipt Printer that is to be used for the Outpatient Receipts. Links to the CV3LogicalPrinter table. 
AutoTransferLabelGeneration 
Definition  Indicates that when a patient is transferred, labels will automatically be generated. 
AutoPatientTransferReport 
Definition  Indicates that each time a patient is transferred a report should be printed. 
AutoPatientMedTransferList 
Definition  Indicates that a report (with all the patient's meds) will generate each time a patient is transferred from one unit location to another unit location. 
MSrepl_tran_version 
Definition  Optimistic lock identifier. 
NonFirstDoseFillPharmacyID 
Definition  A foreign key to the PharmacyID column in the SXAMMPharmacy table. 
PrintRobotLabels 
Definition  Flag to indicate whether robot labels should be printed 
PrintFloorstockLabels 
Definition  Flag to indicate whether floorstock labels should be printed 
MSReplrowguid 
Definition  Unique GUID used to support merge replication.
