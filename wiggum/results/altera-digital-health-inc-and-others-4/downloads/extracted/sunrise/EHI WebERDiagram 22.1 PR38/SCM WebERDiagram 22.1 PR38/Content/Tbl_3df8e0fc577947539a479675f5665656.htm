dbo.CV3ObsValidValue 
Default Table Name  CV3ObsValidValue 
Definition  Defines the possible pre-defined values for an observation catalog item. CV3ObsValidValue has a 1-1 relationship with CV3UserDictionary, for user dictionary non-restricted items.
