dbo.SXAAMBSigShortcut 
Default Table Name  SXAAMBSigShortcut 
Definition  Reference table for “sig” shortcuts used when writing prescription instructions. “Sig” shortcuts are used as shorthand for the main parts of a prescription (1 tab twice a day x 10 days).

The build number is tied to specific versions of the software, e.g. Sunrise 15.3 is Build 5021, Sunrise 16.3 is Build 5503.
