dbo.CV3UserAlertRepository 
Default Table Name  CV3UserAlertRepository 
Definition  This is a parallel table to the CV3AlertRepository, to avoid conflicts arising with existing reports that run against the CV3AlertRepository table. The UserAlert’s stored in the CV3UserAlertRepository table often have NULL Client/Chart/Visit guids.
