dbo.CV3ClientDocDetailHistXrefARC 
Default Table Name  CV3ClientDocDetailHistXrefARC 
Definition  Contains older document records. This table is a cross-reference between the Client Doc History and the Client Doc Detail. It identifies all the detail objects which are valid for the document at a certain point in its history.
