dbo.SXAMMKBAUsabilityPolicy 
Default Table Name  SXAMMKBAUsabilityPolicy 
Definition  This entity contains Usability Policies for KBA. Each Location Function Group (which is made up of multiple locations) can have it's own Usability Policy - which defines the rules around how KBA will function for that location group.
