dbo.SXAESLocationDefaultDisplayColumn 
Default Table Name  SXAESLocationDefaultDisplayColumn 
Definition  One of the primary features of the Location Defaults functionality is to store the columns that should appear in the calendar and worksheet views. This table stores this settings by storing a row for every column that is configured to appear in the calendar or worksheet views.
