dbo.SXAMMPharmacyLocationConfig 
Default Table Name  SXAMMPharmacyLocationConfig 
Definition  This entity contains the configuration description of the pharmacy. It describes the locations that are used to store the inventory (to the lowest level that the user wishes to define).

Example location names are: shelf, bin, drawer, cupboard, fridge, etc.
