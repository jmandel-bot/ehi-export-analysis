dbo.SXAAMLocnSearchColumnFacilityXREF 
Default Table Name  SXAAMLocnSearchColumnFacilityXREF 
Definition  This table will maintain the relationship between Facilities and the Location Search columns defined for them. This table is being created to satisfy business requirements in specification 30-4026 Changes to Location Search. Previously, the columns displayed in a Location Search grid were system defined and unalterable, but now the users will be able to define which columns they would like to see in the grid. There can only be one definition per facility, and it is either active or inactive. If inactive, the default columns will be displayed. This table maintains the relationship between a facility and the selected columns
