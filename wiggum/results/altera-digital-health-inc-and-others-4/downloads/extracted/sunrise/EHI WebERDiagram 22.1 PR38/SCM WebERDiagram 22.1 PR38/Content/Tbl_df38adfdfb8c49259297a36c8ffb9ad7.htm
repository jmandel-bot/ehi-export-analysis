dbo.CV3LocationFunctionGroupXref 
Default Table Name  CV3LocationFunctionGroupXref 
Definition  Defines the location groups that belong to a function group. A “function group” simply allows the site to group locations together that share business rules with respect to a function or policy. For example, if we have 3 pediatric units that all share the same order availability rules, then the site would define a function group called “Pediatric Availability” and all 3 units would be associated to that group. Furthermore, they could share the same collection schedule, so a function group of the type “collection schedule” would also be defined. This table defines the location groups that belong to the function group.
