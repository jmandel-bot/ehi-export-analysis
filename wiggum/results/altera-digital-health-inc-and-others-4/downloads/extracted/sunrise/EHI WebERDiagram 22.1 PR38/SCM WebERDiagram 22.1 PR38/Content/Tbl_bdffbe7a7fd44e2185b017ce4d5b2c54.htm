dbo.SXAMMFillOccurrenceDetail 
Default Table Name  SXAMMFillOccurrenceDetail 
Definition  This entity contains a row for each item that has been selected to:
a) go on the cart, OR
b) go up to the floor in an IV
Current dispense quantities are stored in this table.
