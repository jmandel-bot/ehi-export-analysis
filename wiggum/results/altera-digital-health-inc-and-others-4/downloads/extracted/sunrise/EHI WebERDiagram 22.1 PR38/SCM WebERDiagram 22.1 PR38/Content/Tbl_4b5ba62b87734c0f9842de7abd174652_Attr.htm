SiteID 
Definition  Identifies the unique distributed database ID where the row was inserted. Distributed databases are no longer supported so this column is deprecated. 
Build 
Definition  Identifies the build number and patch level of the software that inserted/updated this row. The format is the Build number followed by 3 digits for the patch level which is typically 000. The build number is tied to specific versions of the software, e.g. Sunrise 15.3 is Build 5021, Sunrise 16.3 is Build 5503. 
TouchedBy 
Definition  Information about the user that last updated the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
TouchedWhen 
Definition  The date and time that the row was last updated and will be based on the Enterprise time zone. 
CreatedBy 
Definition  Information about the user that created the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
CreatedWhen 
Definition  The date and time that the row was created and will be based on the Enterprise time zone. 
InstanceReviewRequestID 
Definition  Auto generated unique primary key value. 
DataModelID 
Definition  A foreign key to the DataModelID column in the SXAEISDataModel table. 
EntityDefinitionID 
Definition  A foreign key to the EntityDefinitionID column in the SXAEISEntityDefinition table. 
PrimaryInstanceID 
Definition  The InstanceID of the first suspected duplicate instance 
SecondaryInstanceID 
Definition  The InstanceID of the second suspected duplicate instance 
RequestingClientApplication 
Definition  The name of the application being used when the pair of records were found to be possible duplicates 
RequestedReviewType 
Definition  Type of review needed to be performed on this instance 
FirstRequestedWhen 
Definition  Enterprise TimeZone - Date the duplicate pair was first identified 
FirstRequestedBy 
Definition  User that first requested the duplicate pair 
LastRequestedWhen 
Definition  Enterprise TimeZone - Date the duplicate pair was last reviewed 
RequestCount 
Definition  Number of times this duplicate pair was identified 
ReviewingClientApplication 
Definition  Application that was used to review the duplicate pair 
ReviewedWhen 
Definition  Enterprise TimeZone - Date the duplicate pair was reviewed 
ReviewedBy 
Definition  PHI: User that reviewed the duplicate pair 
ResolvingAction 
Definition  Action taken to resolve the possible duplicate pair 
MSrepl_tran_version 
Definition  Optimistic lock identifier. 
MSReplrowguid 
Definition  Unique GUID used to support merge replication.
