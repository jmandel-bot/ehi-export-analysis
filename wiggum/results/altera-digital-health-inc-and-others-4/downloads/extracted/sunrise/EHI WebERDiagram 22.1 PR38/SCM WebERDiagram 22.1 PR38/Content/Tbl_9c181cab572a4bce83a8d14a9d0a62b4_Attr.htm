Build 
Definition  Identifies the build number and patch level of the software that inserted/updated this row. The format is the Build number followed by 3 digits for the patch level which is typically 000. The build number is tied to specific versions of the software, e.g. Sunrise 15.3 is Build 5021, Sunrise 16.3 is Build 5503. 
CreatedBy 
Definition  Information about the user that created the row. This will typically be the UserID (IDCode from CV3User table) and may be prefixed with an environment identifier followed by an underscore, e.g. b96d2bf5_jongmore. 
CreatedWhenUTC 
Definition  The date and time that the row was created including the time zone offset from UTC, for example 2015-01-30 04:55:04.372 -05:00. The date and time is based on the Enterprise Time Zone. 
HistoryEventDetailID 
Definition  Auto generated unique primary key value. 
HistoryEventID 
Definition  A foreign key to the HistoryEventID column in the SXAMMHistoryEvent table. 
AuditType 
Definition  This column holds enum that specifies the auditing type - Added, Removed, Modified 
ComponentName 
Definition  This column indicates the auditing component - Ingredient Name, Ingredient Name/Detail, Main, Label 
AuditField 
Definition  This column holds the Field/Colume of the table on auditing 
OldValue 
Definition  This column holds the old value of the Field/Colume of the table on auditing 
NewValue 
Definition  This column holds the new value of the Field/Colume of the table on auditing 
TableName 
Definition  This column holds the name of the child table on auditing 
ParentPKValue 
Definition  This column holds the PK ref to a parent item on auditing 
OldValueInByte 
Definition  This column holds the old value of the Field/Colume of the table on auditing in array of byte 
NewValueInByte 
Definition  This column holds the new value of the Field/Colume of the table on auditing in array of byte
