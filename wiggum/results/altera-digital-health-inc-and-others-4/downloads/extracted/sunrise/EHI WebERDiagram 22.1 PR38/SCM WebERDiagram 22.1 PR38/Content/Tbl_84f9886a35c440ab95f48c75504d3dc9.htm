dbo.SXAMMInventoryTransactionReason 
Default Table Name  SXAMMInventoryTransactionReason 
Definition  This table stores the list of valid reasons for changes in the inventory (ex: manual adjustment transfer of inventory, data entry error, order received, initial inventory stocked, etc)
