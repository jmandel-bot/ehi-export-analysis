dbo.SXAAMBarcodeDefinition 
Default Table Name  SXAAMBarcodeDefinition 
Definition  This table is to store the barcode definitions. Barode definition will be used while creating a GS1 barcode as well as scanning a GS1 barcode.

The format is the Build number followed by 3 digits for the patch level which is typically 000.
