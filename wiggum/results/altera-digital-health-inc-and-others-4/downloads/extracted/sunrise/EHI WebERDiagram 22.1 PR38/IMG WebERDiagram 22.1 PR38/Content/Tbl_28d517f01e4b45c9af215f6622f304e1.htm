dbo.SXAIMGDocumentType 
Default Table Name  SXAIMGDocumentType 
Definition  Contains Document Types, which are categories for classifying Documents. Example Document Types are such things as Drivers License, Insurance Card, etc. A Document Type may be assigned to many Document Type Sets
