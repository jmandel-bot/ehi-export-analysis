EHI Export Schema
Table asc_ptorders
Description This table stores information about patient orders, including order details, status, and related entities.
Column DataType Description
orderid (Primary Key) int This column is an auto generated unique primary key.
encounterid int The column stores an id of the encounter of a patient.
userid int This column stores UID which is primary id of user table.
orderedByid int This column stores UID who has placed the order. It refers to UID from user table.
orderItemid int Identifier for the item being ordered, foreign key referring to the orderItems table (e.g., 100, 200, 300).
parentid int Identifier for the parent order, if applicable, foreign key referring to the asc_ptorders table (e.g., 0, 1, 2).
orderDateTime datetime This column stores time stamp of order time.
startDateTime datetime Order start time. Can be defined when Oder is placed.
endDateTime datetime Order end time. Can be defined when Order is expired.
isRecurringOrder int This column stores tinyint value, It can be either 0 or 1. Set 1 if order is of recurring type else 0.
orderFrequencyid int Column value is no longer used. Default value is 0.
orderStatus int This column have reference in java file but it's not used in update flow. This column is not in use.
orderPriority int This column stores order priority: for STAT it stores 1 for ASAP it stores 2 for ROUTINE it stores 3
orderInstructions varchar Detailed instructions for the order (e.g., Administer 500mg of medication every 8 hours).
orderReason varchar Reason for the order (e.g., Patient experiencing severe pain).
orderNotes varchar Additional notes related to the order (e.g., Monitor patient closely for side effects).
orderEntryMethodid int This column stores order entry method. 1 - Per-Protocol 2 - Telephone Order Read Back 3 - Verbal Order Read Back 4 - Written 5 - Nurse Initiated Orders 6 - Standing Order
cosignedBy int This column contains User id of the user who co-signed the order.
cosignDateTime datetime This column contains time stamp when co-signed.
inactive int This column stores tinyint value, It can be either 0 or 1. If 1 then order is inactive else active.
prevOrderid int This column contains previous order id .
orderCreatedDate datetime This column contains time stamp when order is created.
deleteFlag int This column store delete flag. It will be set to 1 if entry is deleted else 0.
