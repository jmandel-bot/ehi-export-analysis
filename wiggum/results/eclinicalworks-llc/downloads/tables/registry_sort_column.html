EHI Export Schema
Table registry_sort_column
Description This table stores the results of Registry reports saved via the Analyze Data workflow.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table (e.g., 1, 2, 3).
ptid int This column stores the id of the patient for registry sort column. This column refers to pid column in patients table.
plname varchar Patient's last name (e.g., Smith, Johnson, Williams).
pfname varchar Patient's first name (e.g., John, Jane, Michael).
dob datetime Patient's date of birth (e.g., 1990-01-01, 1985-06-15, 1978-12-20).
labvalue varchar Lab value for a specific test (e.g., 4.5, 120, 98).
icdcode varchar This column stores Icd codes attached to the patient in any particular encounter either problem list or assessment
rxname varchar Name of the prescribed medication (e.g., Aspirin, Metformin, Albuterol).
age int Patient's age in years (e.g., 30, 45, 60).
rxstrength varchar Strength of the prescribed medication (e.g., 100mg, 50mg, 25mg).
labname varchar Name of the lab test (e.g., Hemoglobin A1c, Blood Pressure, Cholesterol).
lab_date datetime This column stores lab ordered date
icd_date datetime This column stores Icd date attached to the patient in any particular encounter either problem list or assessment
rx_date datetime Date and time when the prescription was issued (e.g., 2021-01-01 10:00:00, 2021-06-15 14:30:00, 2021-12-20 09:15:00).
allergic_to varchar Allergens that the patient is allergic to (e.g., Penicillin, Peanuts, Pollen).
allergy_date datetime This column stores date when allergic reaction to a allergen started for patient
demo_date datetime Date and time when the demographic information was recorded (e.g., 2021-01-01 10:00:00, 2021-06-15 14:30:00, 2021-12-20 09:15:00).
comm_pref varchar This column stores value is comma separated multiple value is as follows: 1. Voice enabled: 1 2. Letters enabled: 2 3. Text enabled: 3 4. Web enabled: 4 5. Email enabled: 5
ReportName varchar Name of the report associated with the registry sort column (e.g., Diabetes Report, Hypertension Report, Asthma Report).
sortColumn varchar Name of the column used for sorting the report (e.g., age, labvalue, rxname).
sortType varchar Type of sorting applied to the column (e.g., ASC, DESC).
deleteflag tinyint Flag to indicate if the record is deleted (0 for not deleted, 1 for deleted).
