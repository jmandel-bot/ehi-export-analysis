Table vision_order_shippingaddress
Description This table stores shipping addresses for contact lens orders.
Column DataType Description
id (Primary Key) int Unique identifier for each shipping address record. (Example: 1, 2, 3)
order_id int The identifier for the order that this shipping address is associated with. This is a foreign key that refers to the `id` column in the `orders` table. (Example: 101, 102, 103)
addressline1 varchar The first line of the shipping address. (Example: "123 Main St")
addressline2 varchar The second line of the shipping address, typically used for apartment or suite numbers. This field can be empty if not applicable. (Example: "Apt 4B", "")
city varchar The city of the shipping address. (Example: "New York", "Los Angeles")
state varchar The state or province of the shipping address. (Example: "NY", "CA")
phone varchar The phone number associated with the shipping address. (Example: "555-123-4567", "555-987-6543")
zip varchar The postal code of the shipping address. (Example: "10001", "90210".)
shipto tinyint This column stores the flag for what option the user has chosen to ship the order after completion, a "1" means office and a "2" means patient.
