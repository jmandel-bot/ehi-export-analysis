EHI Export Schema
Table cre_immunization_message
Description This table stores the association between the patient and the immunization messages to be shown.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an integer that uniquely identifies each record in the table. It is an auto-incrementing value, meaning it will automatically increase by 1 for each new record added to the table. Example: 1, 2, 3, etc.
Patientid int Identifier for the patient associated with the immunization message (e.g., 101, 102, 103). This is a foreign key that refers to the `pid` column in the `patients` table.
Eventid int Identifier for the event associated with the immunization message (e.g., 1001, 1002, 1003). This is a foreign key that refers to the `id` column in the `cre_vfc_event` table.
