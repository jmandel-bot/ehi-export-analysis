EHI Export Schema
Table encdashboard
Description This table stores information about patient encounters, including the encounter date and time.
Column DataType Description
id (Primary Key) int This is the primary key for the table, and it is an auto-incrementing integer. It uniquely identifies each row in the table. Example: 1, 2, 3, etc.
Encounterid int This is an integer that represents the unique identifier for a specific encounter. Example: 101, 102, 103, etc.
Patientid int This is an integer that represents the unique identifier for a specific patient. It can be used as a foreign key to reference the `Patient` table (if it exists) and the corresponding `Patientid` column. Example: 1001, 1002, 1003, etc.
EncDate datetime This is a date column that stores the date of the encounter. It is in the format 'YYYY-MM-DD'. For example: '2021-10-01', '2021-10-02', '2021-10-03', etc.
EncTime datetime This is a time column that stores the time of the encounter. It is in the format 'HH:mm:ss'. Example: '09:00:00', '14:30:00', '18:15:00', etc.
