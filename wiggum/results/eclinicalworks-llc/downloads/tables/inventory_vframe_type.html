EHI Export Schema
Table inventory_vframe_type
Description This table stores information about different types of inventory frames, including their names and whether they are marked for deletion.
Column DataType Description
id (Primary Key) int Is the primary key column and contains a unique integer value for each inventory frame type. For example: 1, 2, 3, etc.
name varchar The name of the frame type. Reference table inventory_vframe_type.
deleteFlag smallint An indicator to mark the frame bucked as deleted. If the value is 1, then the frame bucked will be marked as deleted.
typeid int The frame type id from the inventory_vframe_type table. Used to create buckets with combinations of frame materials.
createdBy int The user id value from the user table that is mapped to the id of the user who created the frame type.
createdOn datetime The created date for the frame type entry. For example : 2019-08-30 17:54:04
