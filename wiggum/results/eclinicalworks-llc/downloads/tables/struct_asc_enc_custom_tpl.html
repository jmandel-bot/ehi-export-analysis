EHI Export Schema
Table struct_asc_enc_custom_tpl
Description This table is used to save smart macro answers.
Column DataType Description
encounterid int This column stores enc id value. Map to encounterid column of enc table. Column does not expect NULL . Default value is '0'.
catid int This column stores category item id. Map to itemid column of items table. Column does not expect NULL . Default value is '0'.
itemid int This column stores property id. Map to propid column of properties table. Column does not expect NULL . Default value is '0'.
detailid int This column represent id value. Maps to detailid column for a table struct_asc_keyword_map. Column does not expect NULL . Default value is '0'.
value text This column store answer text value. Column does not expect NULL . Default value is ''.
valueid varchar The unique identifier for the value associated with the custom template. Example: "val_123"
notes text This column stores free text note for macro answer.
displayIndex tinyint This column is used to represent the display order of the record when grouped by certain record identifier.
chartdisplay tinyint A tiny integer value (1 or 2) indicating the display preference for the custom template entry on a chart. Example: 1
questionNotes text Additional notes related to the question in the custom template entry. Example: "This is a sample question note."
sentenceid int This column represent sentence id. Map to id column of asc_pnsmartmacrostnotes table.
createdBy int This column stores id to represent who created the record. Maps to the UID column in user table.
modiBy int This column stores id to represent who modified the record. Maps to the UID column in user table.
modiTime datetime The time stamp for when the custom template entry was last modified. Example: "2021-09-01 12:34:56"
createdDatetime datetime The datetime for when the custom template entry was created. Example: "2021-08-01 12:34:56"
deleteFlag tinyint This column stores value 0 / 1 indicating if the record is deleted or not, this is soft delete. 0 - Not Deleted , 1 -Deleted.
id (Primary Key) int This is an auto incremented primary key of This column .
versionid int This column represent version id. Maps to id column of table asc_enc_custom_tpl_version.
