EHI Export Schema
Table vascular_anatomy
Description This table stores male and female anatomy metadata.
Column DataType Description
id (Primary Key) int This column is auto-incremented primary key in This column .
objectid varchar This column stores anatomy id.
objectname varchar This column stores anatomy name.
viewtype tinyint This column stores View type value 1(Vein) or 2(Skin).
gender char This column stores gender value M(Male) or F(Female).
leg char This column stores leg identifier L(Left) or R(Right).
deleteflag tinyint This column stores values 0/1 to indicate whether the anatomy has been marked as deleted or not. 0 - If anatomy is NOT marked as deleted, 1 - If anatomy is marked deleted.
abbreviation varchar This column stores abbreviation for anatomy name.
simplifiedvein tinyint This column stores value of simplifiedvein attribute for HumanAPI.
