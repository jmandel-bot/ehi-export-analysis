EHI Export Schema
Table asc_enc_or_proc_details
Description This table stores mapping for ASC encounters and ASC procedures.
Column DataType Description
id (Primary Key) int This is an auto increment primary key column of This column .
encounterid int The column stores an id of the encounter of a patient.
itemid int This column stores numeric type data. This column refers to itemid column in items table.
displayIndex int This columns stores numeric type values which maintains the order of the data
deleteFlag int This column store delete flag. It will be set to 1 if entry is deleted else 0.
modifiedDate datetime This column is used to capture date when details is updated.
modifiedByid int This column stores current user logged-in user id. It is primary id user table.
