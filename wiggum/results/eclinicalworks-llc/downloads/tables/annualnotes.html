EHI Export Schema
Table annualnotes
Description This table stores General (section level) notes for diffrent sections of a set of Progress Notes.
Column DataType Description
encounterid int This column stores id of an encounter for which notes are being documented. This column refers to encounterid column in enc table.
notes text This column stores free text notes documented by provider/staff from modules mentioned in type column.
type varchar This column stores name of the module for which notes have been documented by provider/staff during the encounter. 'Procedure' - Notes form Procedure screen, 'Family' - Notes from Family History screen, 'Therapeutic' - Notes from Therapeutic Intervention screen, 'TherapyAssessment' - Notes from Physical Therapy Assessment Screen, 'PhyExam' - Notes from Physical Examination screen, 'Billing' - Notes from Billing screen, 'Social' - Notes from Social History Screen, 'PrevMed' - Notes from Preventive Medicine screen, 'Vitals' - Notes from Vitals screen, 'ROS'- Notes from Review Of Systems screen
notesHTML varchar This column stores HTML formatted free text notes documented by provider/staff from modules mentioned in type column.
