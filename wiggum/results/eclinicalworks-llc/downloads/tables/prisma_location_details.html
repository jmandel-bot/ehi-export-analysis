Table prisma_location_details
Description This table stores information about various locations, including their names, addresses, and managing organizations.
Column DataType Description
ID (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
LOCATION_ID varchar Unique identifier for the location. Example: "LOC001", "LOC002".
SECTION_ID bigint Identifier for the section the location belongs to. Foreign key relation: `prisma_ccd_document_section.id`. Example: 101, 102.
SOURCE_ID int Identifier for the source of the location data. Example: 1, 2, 3.
LOCATION_NAME varchar Name of the location. Example: "Main Office", "Warehouse".
ALIAS_NAME varchar Alternative name for the location. Example: "Headquarters", "Storage Facility".
CONTACT_NUMBER varchar Contact phone number for the location. Example: "+1-555-123-4567", "+44-20-7123-4567".
PART_OF_LOCATION varchar Identifier for the parent location, if any. Example: "LOC001", "LOC002".
MANAGING_ORGANIZATION varchar Identifier for the organization managing the location. Example: "ORG001", "ORG002".
MANAGING_ORGANIZATION_DISPLAY varchar Display name of the managing organization. Example: "Acme Corp.", "Global Industries".
ADDRESS_LINE1 varchar First line of the location's address. Example: "123 Main St.", "456 Market St.".
ADDRESS_LINE2 varchar Second line of the location's address. Example: "Suite 100", "Building 2".
ADDRESS_LINE3 varchar Third line of the location's address, if any. Example: "Floor 3", "Room 12".
CITY varchar City where the location is situated. Example: "New York", "London".
LOCATION_STATE varchar State or province where the location is situated. Example: "NY", "ON".
POSTAL_CODE varchar Postal code for the location. Example: "10001", "M5V 3K1".
COUNTRY varchar Country where the location is situated. Example: "USA", "Canada".
LOCATION_TYPE_DISPLAY varchar Display name for the type of location. Example: "Office", "Warehouse".
LOCATION_STATUS varchar Status of the location. Example: "Active", "Inactive".
LOCATION_DESCRIPTION varchar Brief description of the location. Example: "Main office for the company", "Primary storage facility".
ENDPOINT_REF varchar Identifier for the endpoint reference, if any. Example: "EP001", "EP002".
LOCATION_MODE varchar Mode of the location. Example: "Physical", "Virtual".
