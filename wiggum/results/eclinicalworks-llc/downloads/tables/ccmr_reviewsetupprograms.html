Table ccmr_reviewsetupprograms
Description This table stores the information regarding review setup added on user facing side which includes intial, recurring frequencies, signature required for provider
Column DataType Description
Id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3
ccmrId int This column stores ccmrid which refers id of ccmr_careplan
patientId int Identifier for the patient associated with the review setup program. Example: 1001, 1002, 1003
programId int This column stores program id for which review setup is stored for signature mandatory which refers id of acoprograms
contactId int This column stores the id of the user whose signature is mandatory for the review from review setup screen
contactType varchar This column stores the role of the user whose signature is mandotory selected from review setup screen
custProgramId int This column stores the program id for the user who is selected for signature mandatory from review setup screen for comprehensive type program
deleteFlag tinyint A flag indicating whether the record has been deleted or not. A value of '0' means not deleted, and a value of '1' means deleted. Example: 0, 1
