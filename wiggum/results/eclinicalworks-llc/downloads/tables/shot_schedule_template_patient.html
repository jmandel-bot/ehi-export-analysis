EHI Export Schema
Table shot_schedule_template_patient
Description This table stores templates selected on a formulation shot schedule. It stores the relationship between the shot template and the recipe queue.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3, etc.
QUEUE_id int The identifier of the queue to which the patient belongs. This is a foreign key that refers to the `id` column in the `queue` table. Example: 101, 102, 103, etc.
TEMPLATE_id int The identifier of the shot schedule template assigned to the patient. This is a foreign key that refers to the `id` column in the `shot_schedule_template` table. Example: 201, 202, 203, etc.
