Table antigen_recipe_dtl
Description This table stores detailed information about antigen recipes, including the items used, their volume, percentage, and other related data.
Column DataType Description
id (Primary Key) int The column stores an id of the recipe detail table. This is an auto incremental primary key of This column .
RECIPEid int This column stores the recipeid associated with the recipe detail entry. It refers to column id from antigen_recipe table.
ITEMid int This column stores the itemid associated with the recipe detail entry. It refers to column itemid from items table.
DETAILid int This column stores the detailid associated with the recipe detail entry.
ITEMNAME varchar This column stores the itemName associated with the recipe detail entry.
VOLUME varchar This column stores the volume of the recipe detail entry.
PERCENTAGE varchar This column stores the percentage (concentration) of the recipe detail entry.
SEQ_NO int This column is not in use.
CONCENTRATION varchar This column stores the concentration of the recipe detail entry.
LOTNO varchar This column stores the Lot No of the recipe detail entry.
EXPDATE datetime This column stores the expiration date of the recipe detail entry.
MANUFACTURERid int identifier for the manufacturer of the item. Example: 1, 2, 3. Foreign key relation: `allergy_manufacturers.id`.
ISMANUFACTURERSAVED tinyint Flag indicating whether the manufacturer information is saved or not. Example: 0 (not saved), 1 (saved).
