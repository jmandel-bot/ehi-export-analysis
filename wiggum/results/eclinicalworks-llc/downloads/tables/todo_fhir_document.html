EHI Export Schema
Table todo_fhir_document
Description This table is used to store FHIR documents related to the todo items.
Column DataType Description
id (Primary Key) int An auto-incremented primary key field
documentName varchar This field to store the name of the FHIR document.
appName varchar This field to store the name of the application.
mimeType varchar This field to store the MIME type of the FHIR document.
addedOn datetime This field to store the date and time when the document was added.
patientId int A foreign key field referencing the users table's uid column (the patient associated with the document).
status tinyint This field to store the status of the document (default 0) and 1 when document imported from UI.
deleteflag tinyint This field to store the deletion flag of the document (default 0) and 1 When user delete from UI.
vendor_id int This column is like a label that helps identify different vendors. It stores a specific number or code that corresponds to the vendor's information in reference from 'fhirappvendors.id'
