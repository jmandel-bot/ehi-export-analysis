EHI Export Schema
Table rx_dispense_summary
Description This table stores information about prescription dispense summaries, including details about the prescription order, item, status, and dispensing and cancellation events of medications ordered from Inventory.
Column DataType Description
id (Primary Key) int This is the auto-incremented primary key in This column .
RxOrderNo varchar This column stores the order number of medications prescribed from inventory. It refers to rxOrderNo from oldrxmain table.
RxItemId int This column stores the item id for medications prescribed from Inventory. This refers to itemid of items table.
RxNDCCode varchar This column stores the National Drug Code (NDC) of the documented drug.
RxStatus tinyint This column stores the status of medications i.e. Start, Stop etc. This refers to DoctorsFlag column of OldRxMain table.
DispensedDate datetime This column stores the medication dispensed date. The format used is MM/DD/YYYY.
DispensedBy varchar This column stores the details of user who dispensed the medication from Inventory.
DeleteFlag tinyint This column stores the value 0/1 indicating if a medication ordered from inventory is deleted in treatment section. 0-Not Deleted. 1-Deleted.
UserId int This column stores the userid of the user who orders a medication from Inventory. This corresponds to userid of user table.
inv_SaleId int This column stores the saleId for a medication ordered from Inventory. This refers to 'Id' column of 'inventory_sale' table and is updated once the claim is created successfully.
RxEncId int This column stores the encounter where medication is added from inventory. It corresponds to encounterID from enc table.
CancelledDate datetime Date and time when the prescription was cancelled. Example: '2021-08-03 16:00:00', '2021-08-04 11:15:00'.
CancelledBy varchar Name of the person who cancelled the prescription. Example: 'John Doe', 'Jane Smith'.
CancelReasonId int Unique identifier for the reason the prescription was cancelled, referring to the primary key in the cancel reasons table (not provided). Example: 1 (Out of Stock), 2 (Prescription Error), 3 (Patient Request).
