Table ip_rh_clinical_sd_template_formula
Description This table stores the formula and its description for the given template.
Column DataType Description
Id (Primary Key) int This is an auto-incremented primary key column in this table. It is used to uniquely identify the template formula. The data type is int as per business requirement and the column cannot be NULL.
tempId int This column stores the template id. It refers to id column in ip_rh_clinical_sd_template table. The data type is INT and the column cannot be NULL.
NAME varchar This column stores the name of the formula.
Description varchar This column stores the description of the formula. The data type is varchar(max) . As the column could store huge data as value and we have free text, therefore we have defined it as varchar(max) and the default value is NULL.
Formula varchar This column stores the formula created for the template which will populate the score based on this formula on the user facing screen.
modiBy int Stores the user Id of the person who last modified the formula. It refers to uid column in users table. The datatype is int. The default value for this column is NULL.
modiTime datetime Stores the date and time the template formula was last modified. The datetime data type allows for precise storage of date and time values. The default value is NULL.
deleteFlag tinyint A flag indicating if the template formula is deleted (1) or not (0). The tinyint data type allows for efficient storage of binary values. The default value is 0, indicating the formula is not deleted.
