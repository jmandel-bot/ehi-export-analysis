EHI Export Schema
Table ssnreasonlist
Description This is a metatable which stores SSN reason list.
Column DataType Description
reasonid (Primary Key) int This is an integer column that serves as the primary key for the table. It is automatically incremented and uniquely identifies each reason in the table. Example values: 1, 2, 3, etc.
reasonName varchar This is a varchar column with a maximum length of 100 characters. It stores the name of the reason and must be unique. Example values: "identity Theft", "Lost Card", "Name Change", etc.
deleteflag tinyint This is a tinyint column that indicates whether the reason has been deleted or not. It has a default value of '0'. A value of '0' means the reason is not deleted, and a value of '1' means the reason is deleted. Example values: 0, 1.
