Table obf_atrisk
Description This table stores information about items that are considered at risk during pregnancy.
Column DataType Description
AtRiskid (Primary Key) int  This column stored Primary Key of the Table
Pregid int identifier for a pregnancy record. It is an integer value and serves as a foreign key referring to the `Pregid` column in another table (e.g., `pregnancy` table). Example: 101, 102, 103, etc.
Itemid int identifier for an item that is considered at risk. It is an integer value and serves as a foreign key referring to the `Itemid` column in another table (e.g., `items` table). Example: 1001, 1002, 1003, etc.
Choice varchar This column stored value as varchar, documented for the risks section
Lock_Ind int This column used stored numeric value where the respective section is lock/unlock ( if lock then 1 , unlock then 0)
DelStat int An integer value indicating the deletion status of the record. A value of 0 means the record is not deleted, while a value of 1 means the record is deleted. Example: 0, 1.
Uname varchar This column used stored varchar value of logged uname of the user.
Upd_Date datetime This column used stored varchar value items documented date and time
