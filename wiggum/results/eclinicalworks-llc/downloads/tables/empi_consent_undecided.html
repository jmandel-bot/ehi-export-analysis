Table empi_consent_undecided
Description This table contains all consents and when to remind users for patients' consents when creating appointments.
Column DataType Description
id (Primary Key) int This column is an auto increment id value for each undecided consent entry.
pid int This column contains the patient id which opted for undecided consent.
decisionDate datetime This column contains the date and time of a patient id which has opted for undecided consent.
remindDuration varchar This column contains the duration of when to remind and when to ask for consent for an undecided consent patient.
remindDate datetime This column contains the date and time of a patient which has opted for undecided consent.
ctimestamp datetime This column contains the current time stamp while inserting the record into the table.
delFlag tinyint This column stores values 0 or 1 to indicate whether a record has been marked as deleted or not. A "0" means a record is not marked as deleted, and a "1" means a record is marked as deleted.
