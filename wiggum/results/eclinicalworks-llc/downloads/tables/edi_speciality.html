EHI Export Schema
Table edi_speciality
Description This table stores a master list of specialties available from a drop-down list.
Column DataType Description
id (Primary Key) int This column denotes id of the speciality entry in the table and is primary key.
Speciality varchar This column stores the name of the specialty referred in EMR billing modules.
Code char This column stores the code corresponding to the specialty.
deleteFlag tinyint This column indicates delete status of row. 1 indicates row is deleted and 0 indicates it is active.
approvalReq tinyint This column hold 1 for approval required and 0 when approval is not required.
