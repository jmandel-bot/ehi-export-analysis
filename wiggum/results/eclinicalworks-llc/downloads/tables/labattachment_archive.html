EHI Export Schema
Table labattachment_archive
Description This is a log table that stores the copy of the Pink Paper Clip xml for every result processed through the interface. This includes the older result copies.
Column DataType Description
id (Primary Key) int This column stores an auto-incremented ID.
ProcessedResult text This column stores the pink paper clip XML copy. This references the processedresult column from the labdataex table.
SeqNo int This column stores the incremented ID (incremented by code and not auto-incremented) per the reportid entry.
DateArchived datetime This columns stores the current time stamp.
DateReceived datetime This column stores the date and time when the result was electronically received through the interface. It references the resultrcvddatetime column of the labdataex table.
Reportid int This column stores the report ID of the result processed through the interface and references the Reportid column of the labdata table.
