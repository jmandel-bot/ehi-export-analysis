Table patient_cpt_units
Description This table stores information about the units of Current Procedural Terminology (CPT) codes associated with each patient. Add the CPSP Max Unit and Minutes/Unit by navigating to Billing > CPT.
Column DataType Description
id (Primary Key) int The unique identifier for each record in the table. Examples include: 1, 2, 3, etc.
patientid int The unique identifier for the patient associated with the CPT units. This could be a foreign key referring to the `id` column in a `patients` table. Examples include: 101, 102, 103, etc.
itemid int The unique identifier for the item associated with the CPT units, which could be a foreign key referring to the `id` column in an `items` table. Examples include: 1001, 1002, 1003, etc.
cpt varchar The CPT code associated with the item. Examples include: '99213', '99214', '99215'.
unitBal int The current balance of CPT units for the patient and item. Examples include: 10, 20, 30, etc.
newUnit int The new CPT units added for the patient and item. Examples include: 5, 10, 15, etc.
controlno varchar A control number associated with the record. Examples include: 'CN001', 'CN002', 'CN003', etc.
deleteFlag tinyint A flag indicating whether the record has been deleted (1) or not (0).
flag tinyint A flag that can be used mark as enties are reset if it is 1.
