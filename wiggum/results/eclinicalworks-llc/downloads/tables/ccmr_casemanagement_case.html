EHI Export Schema
Table ccmr_casemanagement_case
Description This table contains information related to a particular case of a patient and mapping of patient encounter ID and program ID along with other details.
Column DataType Description
caseid (Primary Key) int This is a primary key which is a case id of a specific case of a patient and increments itself automatically.
encid int This column stores the encounter id of a patient encounter. It saves reference to encounterid column of enc table.
programid int This column stores the program id under which patient has been enrolled . It saves reference to id column of acoprograms table.
createddate datetime This column stores the date on which case has been created for a patient.
receiveddate datetime This column stores the date on which case has been received for a patient.
completiondate datetime This column stores the date on which case has been completed for a patient.
refentity int This column stores the id of user to whom case has been referred. It saves reference to UID column of user table
ptacoid int This is foreign key. It saves reference to id column of users ptacoprograms table
deleteflag tinyint This column stores soft delete data. 0: Not deleted; 1: Soft Deleted
