Table bh_intervention_provider
Description This table stores information about intervention providers, including their names, statuses, display indexes, and deletion flags.
Column DataType Description
providerid (Primary Key) int This is the primary key and an auto-incrementing integer that uniquely identifies each intervention provider. Example: 1, 2, 3, etc.
providerName varchar This is a varchar(2000) column that stores the name of the intervention provider. Example: "Provider A", "Provider B", "Provider C", etc.
providerStatus tinyint This is a tinyint(1) column that stores the status of the intervention provider. A value of 0 indicates that the provider is inactive, while a value of 1 indicates that the provider is active. Example: 0 (inactive), 1 (active).
displayIndex int This is an int(11) column that stores the display index of the intervention provider. This value is used to determine the order in which providers are displayed in a list or user interface. Example: 0, 1, 2, 3, etc.
deleteFlag tinyint This is a tinyint(1) column that stores a flag indicating whether the intervention provider has been deleted. A value of 0 indicates that the provider has not been deleted, while a value of 1 indicates that the provider has been deleted. Example: 0 (not deleted), 1 (deleted).
