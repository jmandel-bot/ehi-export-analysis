EHI Export Schema
Table statements_pdfs
Description This table stores information about PDF statements, including their file names, creation details, and flags for deletion and statement types.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
Batchid int identifier for the batch to which the PDF statement belongs. Example: 101, 102, 103.
Insid int identifier for the institution associated with the PDF statement. Example: 1001, 1002, 1003.
fileName varchar Name of the PDF statement file. Example: 'statement_2021_01.pdf', 'statement_2021_02.pdf'.
CreatedByid int identifier for the user who created the PDF statement. Example: 10001, 10002, 10003.
CreatedDateTime datetime Date and time when the PDF statement was created. Example: '2021-01-01 12:34:56', '2021-02-01 14:56:23'.
deleteflag tinyint Flag indicating whether the PDF statement has been deleted (1) or not (0). Example: 0, 1.
Ptid int identifier for the patient associated with the PDF statement. Example: 2001, 2002, 2003.
StmtFlag int Flag indicating the type of statement (e.g., 0 for regular, 1 for summary, 2 for detailed). Example: 0, 1, 2.
