EHI Export Schema
Table treatmentnotes
Description This table stores treatment notes related to specific encounters and assessments.
Column DataType Description
Encounterid int This column store id of the encounter for which treatment of patient is documented. This column refers to column encounterid from enc table.
Asmtid int This column stores id of the assessment for which notes are documented. This column refers to column itemid from items table.
Notes text This column stores notes in free text format entered by end user under Notes section. Pre-defined keywords could also be selected as part of notes.
NotesHTML text This column stores notes with HTML tags in case end user has used text formats like Bold, Italic or Underline from the user interface.
