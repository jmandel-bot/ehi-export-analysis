EHI Export Schema
Table prisma_med_equipment_details
Description This table stores information about medical equipment details associated with a specific section in a document.
Column DataType Description
id (Primary Key) bigint This is the primary key column that uniquely identifies each record in the table. It is an auto-incrementing bigint(20) value. Example: 1, 2, 3, etc.
sectionId bigint This column stores the ID of the associated section in the `prisma_ccd_document_section` table. It is a bigint(20) value and can be NULL. This column has a foreign key relation with the `prisma_ccd_document_section` table's `id` column. Example: 10, 25, 37, etc. Foreign key relation: `prisma_ccd_document_section.id`.
