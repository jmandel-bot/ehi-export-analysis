EHI Export Schema
Table edi_documentationtype
Description This table stores information about different types of EDI (Electronic Data Interchange) documentation.
Column DataType Description
Code varchar A unique identifier for each EDI documentation type. This is a varchar(50) field and cannot be NULL. Example values: "INV", "PO", "ASN".
Name varchar A descriptive name for the EDI documentation type. This is a varchar(255) field and cannot be NULL. Example values: "Invoice", "Purchase Order", "Advance Ship Notice".
