Table phmpatientwiseallcombinationstbl
Description This table stores information about patient wise all cpt combinations for ccm / rpm programs in phm billing.
Column DataType Description
id (Primary Key) int Contains auto incremented id, datatype is INT (11). This column stores integer value which could grow with number of transactions over the years hence integer(Range: 0-2,147,483,647) is selected. It is AUTO_INCREMENT column and has PRIMARY KEY index.
patientid int This column stores unique id of a Patient which references the UID column of the user table. The data type chosen is INT. This field will always have a value hence defined as NOT NULL.
batchid int This column stores unique id of a ccm batch which references the id column of the ccm_batch table. It refers to the batch of specific billing month. The data type chosen is INT. This column does not allow a NULL value.
batchDate datetime Contains the month for which the batch is run. The datatype of this field is datetime. This field does not allow NULL values.
strData varchar Contains json object containing all the cpt combinations for billing eligible patients. The data type of this field is text.
