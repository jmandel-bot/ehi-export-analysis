EHI Export Schema
Table shot_schedule_template
Description This table stores information about different shot schedule templates, including their names, statuses, and billing types.
Column DataType Description
id (Primary Key) int This is the primary key column, which stores a unique integer value for each shot schedule template. Example: 1, 2, 3, etc.
TEMPLATE_NAME varchar This column stores the name of the shot schedule template as a string with a maximum length of 50 characters. Example: "Weekly Schedule", "Monthly Schedule", etc.
STATUS varchar This column stores the status of the shot schedule template as an enumeration with two possible values: 'A' for active and 'I' for inactive. Example: 'A' or 'I'.
DELETE_FLAG varchar This column stores a flag to indicate if the shot schedule template has been deleted or not. It is an enumeration with two possible values: 'Y' for yes (deleted) and 'N' for no (not deleted). Example: 'Y' or 'N'.
BILLING_TYPE tinyint This column stores the billing type associated with the shot schedule template as a tiny integer. Example: 0 for free, 1 for paid, etc.
