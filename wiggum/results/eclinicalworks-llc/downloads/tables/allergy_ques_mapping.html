EHI Export Schema
Table allergy_ques_mapping
Description This table stores the template selected on the Patient Questionnaire window.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
QUEid int This column stores the queUID from antigen_recipe_queue table
TEMPLATEid int identifier for the template associated. This refers to itemid from items table
ENCid int identifier for the encounter associated with the allergy question. Example: 301, 302, 303.
MAP_ENCid int identifier for the mapped encounter associated with the allergy question. Example: 401, 402, 403.
RECORDDATE datetime Date and time when the record was created. Example: '2021-08-01 12:34:56'.
PROVidERNAME varchar Name of the provider associated with the allergy question. Example: 'Dr. John Doe', 'Dr. Jane Smith'.
PROVidERid int identifier for the provider associated with the allergy question. Example: 501, 502, 503.
TEMPLATENAME varchar Name of the template associated with the allergy question. Example: 'Allergy Template 1', 'Allergy Template 2'.
LOCKFLAG tinyint Flag indicating whether the record is locked (1) or not (0). Example: 0, 1.
DELETEFLAG tinyint Flag indicating whether the record is deleted (1) or not (0). Example: 0, 1.
PARENTid int identifier for the parent record in the table. Example: 1, 2, 3.
