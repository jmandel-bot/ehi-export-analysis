EHI Export Schema
Table bh_discharge_addendums
Description This table stores information about discharge addendums for patients in the ACO programs.
Column DataType Description
id (Primary Key) int Unique identifier for each discharge addendum record. Example: 1, 2, 3.
ptAcoid int The identifier for the patient's ACO program. Foreign key relation: `ptacoprograms.id`. Example: 101, 102, 103.
createdBy int The identifier for the user who created the discharge addendum. Foreign key relation: `users.UID`. Example: 1001, 1002, 1003.
createdDate datetime The date and time when the discharge addendum was created. Example: '2021-09-01 12:34:56', '2021-09-02 14:30:00'.
addendum varchar The text content of the discharge addendum, with a maximum length of 4000 characters. Example: 'Patient needs follow-up appointment in 2 weeks', 'Patient prescribed new medication'.
deleteflag tinyint A boolean flag indicating whether the discharge addendum has been deleted (1) or not (0). Example: 0, 1.
