Table asc_timecapture_reporting_field_mapping
Description This table stores the mapping between reporting fields and time capture entries, along with information about the creation and modification of these mappings.
Column DataType Description
id (Primary Key) int This is the primary key for the table, which uniquely identifies each mapping entry. It is an integer value that is automatically incremented. Example: 1, 2, 3, etc.
reportingFieldid int This column stores the id of the reporting field that is being mapped. NOT NULL DEFAULT '0'. Example: 101, 102, 103, etc.
timeCaptureid int This column stores the id of the time capture entry that is being mapped. NOT NULL DEFAULT '0'. Example: 201, 202, 203, etc.
deleteFlag tinyint This is a tiny integer column that indicates whether the mapping entry has been deleted or not. A value of 0 means the entry is not deleted, and a value of 1 means the entry is deleted. Example: 0 or 1.
modifiedBy int This column stores the id of the user who last modified the mapping entry. It is an integer value.NOT NULL DEFAULT '0'. Example: 1001, 1002, 1003, etc.
createdBy int This column stores the id of the user who created the mapping entry. It is an integer value.NOT NULL DEFAULT '0'. Example: 1001, 1002, 1003, etc.
