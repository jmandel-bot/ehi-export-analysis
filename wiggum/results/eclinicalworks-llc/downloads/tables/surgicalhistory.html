EHI Export Schema
Table surgicalhistory
Description This table stores information about patients' surgical history, including the encounter ID, reason for surgery, date of surgery, display index, CPT code, and a flag for whether the surgery is an occupational surgery.
Column DataType Description
encounterid int This column stores encounter id which the Social History is documented for. This column refers column encounterid from enc table.
reason varchar This column stores the history (free text) or predefined options displayed using the search field on the user interface.
date varchar This column stores free text date when surgical procedure mentioned in the reason column was performed on the patient.
displayIndex int This column stores sequence of the surgical histories of the patient documented in the reason column.
cptcode varchar This column stores Current Procedural Terminology (CPT) code of the surgical procedures documented for the patient. This column refers to column cptCode in VALidCPTS table.
occsurgflag tinyint This column was introduced to differentiate the Surgical history data from Ocular surgical history data. It can take a value of 0 for Surgical History and 1 for Ocular Surgical History. Note: Feature has been redesigned. This column is no longer being used and will be deleted in future versions.
