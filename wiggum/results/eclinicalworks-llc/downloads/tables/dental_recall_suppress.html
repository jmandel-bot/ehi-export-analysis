Table dental_recall_suppress
Description This table suppresses information of dental recalls and this information is used to display in the Progress Notes > CDSS > Dental tab.
Column DataType Description
id (Primary Key) int Unique identifier for each dental recall suppression record. Example: 1, 2, 3.
DentalRecallid int The identifier of the dental recall record that this suppression is associated with. This is a foreign key that refers to the `id` column in the `dental_recall` table. Example: 101, 102, 103.
SuppressDate datetime The date when the dental recall suppression was created. Example: '2021-09-01', '2021-09-02', '2021-09-03'.
SuppressFlg tinyint A flag indicating whether the dental recall is suppressed (1) or not (0). Example: 0, 1.
SuppressNotes text Text notes related to the dental recall suppression. Example: 'Patient requested a temporary hold on recalls', 'Moved out of state'.
Patientid int This column stores the patient ID for patient-specific dental recalls that are used on the CDSS > Dental window.
SuppressBy int The identifier of the user who created the dental recall suppression. This is a foreign key that refers to the `id` column in the `user` table. Example: 2001, 2002, 2003.
CreatedDate datetime The date when the dental recall suppression record was created. Example: '2021-09-01', '2021-09-02', '2021-09-03'.
DeleteFlag tinyint A flag indicating whether the dental recall suppression record is deleted (1) or not (0). Example: 0, 1.
PermanentSuppressFlg tinyint A flag indicating whether the dental recall suppression is permanent (1) or temporary (0). Example: 0, 1.
