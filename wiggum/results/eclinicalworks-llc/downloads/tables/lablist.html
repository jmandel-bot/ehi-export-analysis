EHI Export Schema
Table lablist
Description This table stores information about various labs, their addresses, contact details, and configuration settings.
Column DataType Description
ID (Primary Key) int This is an auto incremented primary key of This column .
name varchar This column stores the free text name of the new lab company from User Interface.
deleteFlag smallint This column stores value 0 / 1 indicating if the Lab Company is deleted, this is soft delete. 0 - Not Deleted , 1 -Deleted.
LabId int This column stores information of configured interface ID which will decide on transmission of order/results. This column refers to interfaceID column in labgateway table.
labaddress varchar This column stores the free text address of the lab company.
labaddress2 varchar This column stores the free text address of the lab company.
labcity varchar This column stores the free text address city of the lab company.
labstate varchar This column stores the free text address state of the lab company.
labzip varchar This column stores the free text address zip code of the lab company.
labphone varchar This column stores the telephone number of the lab company.
labfax varchar This column stores the fax number of the lab company.
labemail varchar This column stores the email ID of the lab company.
interfaceConfigured int This column stores value 0/1/2 indicating electronic interface configuration type. 0 - No electronic interface , 1 - Only inbound result interface, 2 - Bidirectional interface.
sendoutboundresult int This column stores value 0/1 indicating send outbound result. 0 - Not send outbound result, 1- Send outbound result.
ResultLabId int This column stores configured interface ID for result processing. This column refers to interfaceID column in labgateway table.
labcompanyType tinyint This column stores value 0/1 indicating lab company type. 0 - Lab , 1 - DI.
inHouseLab int This column stores value 0 / 1 indicating whether the lab company is In-House or an outside lab company. 0 - Outside , 1 -In-House.
labCode varchar This column stores the code used for part of the key to encryption / decryption for data transmission to hubs.
superhubInterfaceID int This column stores master interface id for this vendor at eclinicalworks.
RepoBuildStatus tinyint This column stores Repo Build status
defaultTransmitMode tinyint This column stores default transmit mode for an order
cirrusPractId int Unique identifier for the Cirrus practice associated with the lab (e.g., 500, 600, 700).
faxDialingCountryId int Unique identifier for the country code used for fax dialing (e.g., 1, 44, 61). Foreign key relation: References `countrycodes` table's `id` column.
