EHI Export Schema
Table keywords
Description This table stores information about various keywords associated with items.
Column DataType Description
id (Primary Key) int This is an auto-incremented primary key in This column .
itemid int This column stores the value of itemid from items table against which details of keyword are stored. It refers to column itemid from items table.
name varchar This column stores the value present in name field in keywords pop up.
details text This column stores the description present in notes field in keywords pop up
keyname varchar It is a unique key which is used in the screens that implement keywords functionality
deleteflag int This column stores flags 0/1 to indicate whether keyword has been marked as deleted or not. 0 - keyword is not deleted, 1 - keyword is deleted.
displayindex int This column stores order in which keywords are listed on the UI.
