EHI Export Schema
Table vision_pachymetry
Description This table stores information about vision pachymetry measurements for each encounter.
Column DataType Description
id (Primary Key) int This is the primary key, an auto-incrementing integer that uniquely identifies each record in the vision_pachymetry table. Example: 1, 2, 3, ...
encounterid int This is a foreign key that refers to the `encounterid` column in the `enc` table. It is an integer that uniquely identifies the encounter associated with the vision pachymetry measurement. Example: 101, 102, 103, ...
r_eye smallint This is a small integer column that stores the pachymetry measurement for the right eye. It can be NULL if the measurement is not available. Example: 550, 560, 570, ...
l_eye smallint This is a small integer column that stores the pachymetry measurement for the left eye. It can be NULL if the measurement is not available. Example: 540, 550, 560, ...
comments varchar This is a varchar (255) column that stores any additional comments or notes related to the vision pachymetry measurement. It has a default value of an empty string (''). Example: 'Measurement taken after LASIK surgery', 'Patient has dry eyes', ...
