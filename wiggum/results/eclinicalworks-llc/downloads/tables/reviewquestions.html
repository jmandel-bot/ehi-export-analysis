EHI Export Schema
Table reviewquestions
Description This table stores questions used during the Change Assigned-To function. Itemkeys 0042_UK and 0593_UK need to be enabled for this feature.
Column DataType Description
id (Primary Key) int This column is an integer with a maximum length of 5 digits and serves as the primary key for the table. It is an auto-incrementing value that uniquely identifies each review question. Example: 1, 2, 3, etc.
Question text This column is of the text data type and stores the actual text of the review question. It cannot be NULL. Example: "What did you like most about the product?"
Type int This column stores 0/1 to indicate the type of review question. 1 - Yes/No Question, 0 - Numeric question (0 - 9)
isDeleted tinyint This column is a tiny integer with a maximum length of 1 digit and is used to indicate whether the review question has been deleted or not. It has a default value of '0' (not deleted). Example: 0 (not deleted), 1 (deleted).
