EHI Export Schema
Table enterprisedirectory
Description This table stores the information on enterprise units, which is used to form enterprise hierarchy, such as organization unit, facility or practice. Users are assigned access to certain levels of enterprise hierarchy based on their role.
Column DataType Description
id (Primary Key) int The column stores an id of the enterprise directory. This column is an auto incremented primary key.
Orgid int This column is generally a practice id or facility id. It holds the int value.
Parentid int This column represents the parent of any node present in the Enterprise Directory Hierarchy. It refers to the enterprisedirectory id.
OrgName varchar This column stores the name Org Unit while creating Org Unit/building enterprise hierarchy. It holds the varchar values.
OrgDesc text This column stores the name Org Unit Description while creating Org Unit building enterprise hierarchy. It holds the text values.
OrgType varchar This column stores the name Org Type while creating Org Type. Fix Org types are 'Root', 'OU', Practice' and 'Facility' for root, UnAssigned, Practice and Facility Org Unit respectively. It holds varchar values. The default value is OU which represents the unassigned org unit.
DeleteFlag tinyint It represents that the value is deleted or not. It holds the tinyint value of 1 or 0. A value of 1 means the entry is deleted. The default value is 0.
modifiedDate datetime This column stores the time stamp on value inset/update operation.
