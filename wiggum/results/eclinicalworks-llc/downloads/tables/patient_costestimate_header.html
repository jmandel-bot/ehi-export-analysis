Table patient_costestimate_header
Description This table stores data for patients cost estimation. Like (patient detail, facility, insurance, deductibles etc. )
Column DataType Description
estid (Primary Key) int Unique identifier for the cost estimate record.
estdate datetime Date and time when the cost estimate was created.
createdby int Identifier of the user who created the cost estimate.
patientid int Identifier of the patient associated with the cost estimate.
providerid int Identifier of the provider associated with the cost estimate.
npid int National Provider Identifier (optional).
facid int Identifier of the facility associated with the cost estimate.
anesthesia_min smallint Number of anesthesia minutes for the procedure.
posid int Identifier of the place of service associated with the cost estimate.
selfpay char Indicates whether the patient is self-paying (1) or not (0).
insid int Identifier of the insurance associated with the cost estimate (optional).
ins_override char Indicates whether the insurance information is overridden (1) or not (0).
est_deduction decimal Estimated deduction amount for the patient.
est_coinsurance decimal Estimated coinsurance percentage for the patient.
est_copay decimal Estimated copay amount for the patient.
esttotchgall decimal Total estimated charges for all services.
esttotptresp decimal Total estimated patient responsibility amount.
esttotinsresp decimal Total estimated insurance responsibility amount.
discid int Identifier of the discount associated with the cost estimate (optional).
comments text Additional comments or notes about the cost estimate (optional).
msgid int Identifier of the message associated with the cost estimate (optional).
void char Indicates whether the cost estimate is voided (Y) or not (N).
voiduserid int Identifier of the user who voided the cost estimate (optional).
voiddt datetime Date and time when the cost estimate was voided (optional).
esttotchgprov decimal Total estimated charges for the provider.
esttotchgfac decimal Total estimated charges for the facility.
esttotchganes decimal Total estimated charges for anesthesia.
estptrespprov decimal Estimated patient responsibility amount for the provider.
estptrespfac decimal Estimated patient responsibility amount for the facility.
estptrespanes decimal Estimated patient responsibility amount for anesthesia.
dtsaved datetime Date and time when the cost estimate was saved.
isprinted tinyint Indicates whether the cost estimate has been printed (1) or not (0).
servicetype varchar Type of service associated with the cost estimate (optional).
deleteflag tinyint Indicates whether the cost estimate is deleted (1) or not (0).
estimatedetail text Detailed information about the cost estimate (optional).
publishToPortal tinyint Indicates whether the cost estimate is published to the patient portal (1) or not (0).
fileName varchar Name of the file associated with the cost estimate (optional).
proffeeschid int Identifier of the professional fee schedule associated with the cost estimate (optional, default -1).
facfeeschid int Identifier of the facility fee schedule associated with the cost estimate (optional, default -1).
statusid tinyint Identifier of the status of the cost estimate (default 1).
encounterID int Identifier of the encounter associated with the cost estimate (optional). Foreign key relation to `enc` table, column `encounterID`.
