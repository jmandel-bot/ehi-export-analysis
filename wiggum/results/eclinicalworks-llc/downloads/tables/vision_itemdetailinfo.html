Table vision_itemdetailinfo
Description This table stores information about the details of vision items, including taxes, cash statuses, and deletion flags.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
encid int Foreign key referring to the id of the related entity in another table. Example: 101, 102, 103.
itemFileObj text Binary Large Object (BLOB) column that stores the item file object, such as an image or document related to the vision item. Example: (Binary data).
salestax decimal The sales tax associated with the vision item as a double-precision floating-point number. Example: 7.25, 8.5, 6.75.
instax decimal The tax associated with the vision item for institutions as a double-precision floating-point number. Example: 5.5, 6.25, 4.75.
patienttax decimal The tax associated with the vision item for patients as a double-precision floating-point number. Example: 4.5, 5.25, 3.75.
isCash tinyint A tiny integer (0 or 1) indicating whether the vision item is a cash item or not. Example: 0 (not cash), 1 (cash).
deleteflag tinyint A tiny integer (0 or 1) indicating whether the record has been deleted or not. Example: 0 (not deleted), 1 (deleted).
DateCreated datetime The date and time when the record was created, stored as a datetime value. Example: '2021-09-01 12:34:56', '2021-10-15 09:45:30'.
