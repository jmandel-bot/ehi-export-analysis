EHI Export Schema
Table patient_athletics
Description This table is used to store athletics data of a specifc player.
Column DataType Description
id (Primary Key) int This is the primary key -auto incremented value of This column
patientid int stores the player id. This refers to UID column of user table
currentstatus varchar used to store current status code of the player. This refers to code column of playersatuses table
projectedstatus varchar used to store projected status code of the player. This refers to code column of playersatuses table
gamedaystatus varchar used to store gameday status code of the player. This refers to code column of playersatuses table
deleteflag tinyint used to delete player record from This column when the flag is set to 1
nickname varchar stores the nick name of the player
