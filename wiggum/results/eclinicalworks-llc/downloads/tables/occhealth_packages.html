EHI Export Schema
Table occhealth_packages
Description This table stores information about occupational health packages, including their details, availability, and status.
Column DataType Description
id (Primary Key) int This is the primary key for the table, which uniquely identifies each collection note. It is an integer value that is automatically incremented. Example: 1, 2, 3, etc.
packageName varchar This columns stores package name value
isActive tinyint This column Stores flag value for the package is active or inactive
isGeneric tinyint This column Stores flag value for the package is generic or not
cptCode varchar This column Stores cpt code added for package.
notes text This column Stores notes for the packages
isAvailableOnline tinyint This column Stores flag value for the package is available online
delflag tinyint This column Stores delete flag for packages. 0 : Not deleted, 1 : Deleted
cptItemid int This column Stores cpt item id.
cptFeeSchid int This column Stores cpt fee schedule id.
packageStartDate varchar This column Stores start date for packages
packageEndDate varchar This column Stores end date for packages
serviceType tinyint This column Stores package service type
