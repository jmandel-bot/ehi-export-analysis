Table laceration_proccpt
Description This table stores information about laceration procedures and their associated billing codes.
Column DataType Description
encounterid (Primary Key) int This column stores the unique identifier for the encounter (e.g., 12345) during which the laceration procedure took place. It is an integer and serves as a part of the primary key for the table. It may have a foreign key relation to the `encounter` table, specifically the `id` column.
billingid (Primary Key) int This column stores the unique identifier for the billing record (e.g., 67890) associated with the laceration procedure. It is an integer and serves as a part of the primary key for the table. It has foreign key relation to the `billingdata` table, specifically the `id` column.
cptcode varchar This column stores the Current Procedural Terminology (CPT) code (e.g., "12001") for the laceration procedure. It is a variable character string with a maximum length of 30 characters.
cptid int This column stores the unique identifier for the CPT code (e.g., 98765) associated with the laceration procedure. It is an integer and may have a foreign key relation to the `edi_cptcodes` table, specifically the `itemid` column.
