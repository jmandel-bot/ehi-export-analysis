EHI Export Schema
Table cmpgn_htmlformpatientresponsedetails
Description This table stores the details of patient responses including information about the survey, message, and response status.
Column DataType Description
id (Primary Key) int This is an auto increment id and primary key of table.
UID int User identifier for the patient who responded to the HTML form. Foreign key relation to `users` table (`UID` column). Example: 101, 102, 103.
apUID int This column stores apUID
surveyid int id of survey which has been sent to patient
policyid int This is a id of campaign thru which Survey has been sent to patient and to which patient has responsed.
encounterid int This is an encounter id for which survey has been sent to a patient.
doctorid int Doctor's id who sent a survey to patient. Foreign key relation to `users` table (`UID` column).
resourceid int Resource's id who sent a survey to patient. Foreign key relation to `users` table (`UID` column).
facilityid int facility's id who sent a survey to patient. Foreign key relation to `edi_facilities` table (`id` column).
initMessageid int This is id of message sent to a patient
msgViewed tinyint Indicates whether the message has been viewed by the patient (1) or not (0). Example: 0, 1.
msgSentDate datetime Date and time when the message was sent to the patient. Example: '2021-08-01 12:34:56'.
msgUniqueidentifier varchar Unique identifier for the message sent to the patient. Example: 'abc123', 'def456', 'ghi789'.
responseStatus smallint Status of the patient's response to the HTML form (0: not started, 1: in progress, 2: completed). Example: 0, 1, 2.
responseLanguage varchar Language used in the patient's response. Example: 'en', 'es', 'fr'.
htmlFormResponseStartedOn datetime Date and time when the patient started responding to the Survey. Example: '2021-08-02 13:45:00'.
htmlFormResponseModifiedOn datetime Date and time when the patient last modified their response to the Survey. Example: '2021-08-03 14:55:00'.
htmlFormResponseCompletedOn datetime Date and time when the patient completed their response to the Survey. Example: '2021-08-04 15:00:00'.
responseUniqueidentifier varchar Unique identifier for the patient's response to survey. Example: 'xyz123', 'uvw456', 'rst789'.
isResponseAnonymous tinyint Indicates whether the patient's response is anonymous (1) or not (0). Example: 0, 1.
inserttime datetime Date and time when the record was inserted into the table. Example: '2021-08-05 16:00:00'.
loggedinuserid int id of user who is sending healow survey to a patient
