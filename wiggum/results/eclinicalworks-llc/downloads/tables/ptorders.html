EHI Export Schema
Table ptorders
Description This table stores information about patient orders, including order details, status, and scheduling.
Column DataType Description
orderid (Primary Key) int Unique identifier for each order (e.g., 1, 2, 3).
encounterid int Identifier for the patient encounter associated with the order (e.g., 100, 101, 102).
userid int Identifier for the user who created the order (e.g., 10, 11, 12).
orderedByid int Identifier for the user who authorized the order (e.g., 20, 21, 22).
orderItemid int Identifier for the specific item being ordered (e.g., 1000, 1001, 1002).
orderDateTime datetime Date and time when the order was created (e.g., 2021-08-01 14:30:00).
startDateTime datetime Date and time when the order should start (e.g., 2021-08-01 15:00:00).
endDateTime datetime Date and time when the order should end (e.g., 2021-08-01 16:00:00).
isRecurringOrder tinyint Indicates if the order is recurring (1) or not (0) (e.g., 0, 1).
orderStatus int Numeric code representing the status of the order (e.g., 0, 1, 2).
orderFrequencyid int Identifier for the frequency of the order (e.g., 30, 31, 32).
orderPriority tinyint Indicates the priority of the order, with 1 being high priority and 0 being normal priority (e.g., 0, 1).
orderNotes text Additional notes about the order (e.g., Patient requires extra monitoring).
orderInstructions varchar Specific instructions for the order (e.g., Administer medication with food).
orderReason varchar Reason for the order (e.g., Patient experiencing pain).
orderEntryMethodid int Identifier for the method used to enter the order (e.g., 40, 41, 42).
includeNow tinyint Indicates if the order should be included immediately (1) or not (0) (e.g., 0, 1).
lastScheduleGenerateDate datetime Date and time when the last schedule for the order was generated (e.g., 2021-08-01 14:45:00).
discontinueReasonid int Identifier for the reason the order was discontinued (e.g., 50, 51, 52).
discontinuedDate datetime Date and time when the order was discontinued (e.g., 2021-08-01 17:00:00).
discontinuedBy int Identifier for the user who discontinued the order (e.g., 60, 61, 62).
deleteFlag tinyint Indicates if the order has been deleted (1) or not (0) (e.g., 0, 1).
