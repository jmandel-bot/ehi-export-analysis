Table ip_rh_orderstatus_master
Description The ip_rh_orderstatus_master table contains the status of orders like New, Modified etc.
Column DataType Description
id (Primary Key) int This is an auto-incrementing integer column, serving as the primary key.
statusGroupId int This is an integer column that can not store NULL values. It relates to the ID of the status group associated with the order status. It references the statusGroupId in the ip_rh_orderstatus_groupmaster table.
statusName varchar This is a string (varchar) column with a maximum length of 100 characters. It cannot be NULL and has a default value of empty string. This column represents the name of the order status. The NOT NULL constraint ensures that this column must always have a value, which makes sense as every order status should have a name.
statusText varchar This is a string (varchar) column with a maximum length of 100 characters. It cannot be NULL.This column could be used to store a short text description or label for the order status.
description varchar This is a string (varchar) column with a maximum length of 255 characters. It cannot be null. This column could be used to store a longer description of the order status.
activeStatus int This is tinyint column with a default value of 0 and it cannot be NULL. This column might be used to indicate whether the order status is active mark as 1 or inactive mark as 0.
displayIndex int This is an tinyint column with a default value of 0 and it cannot be NULL. This column might be used to determine the order in which the order statuses are displayed in a user interface, with lower values appearing first.
deleteFlag int This is a tinyint, a smaller range integer. It cannot be NULL and has a default value of 0. This column indicates whether an order status has been marked for deletion or not. deleteFlag value 0 mark as not deleted and value 1 mark as deleted.
