EHI Export Schema
Table commonwellnetlink
Description This table stores information about the network link records in the CommonWell Health Alliance network.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
pid int Patient identifier, referring to the patient's id in another table (foreign key). Example: 101, 102, 103.
trid varchar Transaction identifier, a unique string representing the transaction related to the patient's record. Example: 'TR123', 'TR124', 'TR125'.
dtCreated datetime Date and time when the record was created. Example: '2021-09-01 12:34:56'.
dtModified datetime time stamp of the last modification made to the record. Example: '2021-09-02 14:23:12'.
ptNetStatus int Integer representing the status of the patient's record in the network. Example: 0 (inactive), 1 (active), 2 (pending).
updatedby int User identifier, referring to the user who last updated the record (foreign key). Example: 1001, 1002, 1003.
netLinkData text Binary data containing information about the link between the patient and their record in the network. Example: (binary data).
reviewed int Integer indicating whether the record has been reviewed or not. Example: 0 (not reviewed), 1 (reviewed).
