EHI Export Schema
Table bh_importedtreatmentplan
Description This table stores information about imported treatment plans, including their creation and modification details, as well as the imported data and a flag indicating that the record has been deleted.
Column DataType Description
id (Primary Key) int Unique identifier for each imported treatment plan record. Example: 1, 2, 3.
encounterid int identifier for the encounter associated with the imported treatment plan. Foreign key relation: `enc.encounterid`. Example: 101, 102, 103.
treatmentPlanid int identifier for the treatment plan associated with the imported data. Foreign key relation: `bh_treatment_plan.id`. Example: 1001, 1002, 1003.
createdDate datetime Date and time when the imported treatment plan record was created. Example: '2021-07-01 12:34:56'.
createdBy int identifier for the user who created the imported treatment plan record. Foreign key relation: `users.UID`. Example: 501, 502, 503.
modifiedDate datetime Date and time when the imported treatment plan record was last modified. Example: '2021-07-02 14:56:32'.
modifiedBy int identifier for the user who last modified the imported treatment plan record. Foreign key relation: `users.UID`. Example: 504, 505, 506.
importedData varchar Text containing the imported data for the treatment plan. Example: '{"treatment": "chemotherapy", "duration": "6 months"}'.
deleteflag tinyint A flag indicating if the imported treatment plan record has been deleted. 0 means not deleted, 1 means deleted. Example: 0, 1.
