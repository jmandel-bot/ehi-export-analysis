Table occhealth_employer_claimmapping
Description This table stores employer claim and invoice information for occupational health encounter. It will also stores the information if the claim is splited or not. If yes, it also stores the splited claim and encounter information.
Column DataType Description
id int This is the primary key for the table, which is an integer value that uniquely identifies each record. It is an auto-incrementing value. Example: 1, 2, 3, etc.
EmployerClaimId int Stores the claimid
EmployerEncId int Stores the encounter id
SplitedClaimId int Stores the claim for split claim if claim is split if not then it stores -1
SplitedEncId int Stores the encounter id for split claim if claim is split if not then it stores -1
SplitedClaimBilledToType int Stores the bill type of the claim
SplitedClaimBilledToId int Stores the billed to id of split claim
deleteflag tinyint This column stores value 0 / 1 indicating this row is deleted. This is soft delete. 0 - Not Deleted , 1 -Deleted
isInvoiced tinyint Stores flag value for invoice is created or not.
invoiceNumber int Stores invoice number for employer claim
