EHI Export Schema
Table ip_rh_ptstagingmedicationorders
Description This table stores patient staging orders information which includes details like route, frequency of medications that are placed for a patient during their encounter in a staged environment before being finalized.
Column DataType Description
medOrderId (Primary Key) int This is the unique identifier for each interaction record.
ptOrderId int This column is used to store ptOrder id and is foreign key references orderId column of ip_rh_ptorders table.
ipitemid int This column is used to store item id and is foreign key references itemID column of items table.
medSigId int This is used to store Medication signature ID, this value is stored for reference and is foreign key references medSigId column of ip_rh_medsignature table.
order_dose varchar This column is used to store order dosages like 50mg, 50ml. This value is used from the of doseSize of ip_rh_drugformulary table
order_dose_UOM varchar This column is used to store order dosage units like ml, mg.This value is used from the of doseUoM of ip_rh_drugformulary table
RouteId int This column is used to store route id and is foreign key references routeId column of ip_rh_drugformulary_routes table.
FrequencyId int This column is used to store frequency id and is foreign key references id column of ip_rh_drugformulary_frequency table.
order_duration int This column is used to store order duration.
order_duration_type varchar This column is used to store order duration type like days, hours.
isHomeMed tinyint This column is used to store a home medication flag to show whether it is converted from home medication or not. 1 - represents it is converted from home medication 0 - represents it is not converted from home medication.Default value is 0 ensures not home med.
isPatientSupplied tinyint This column is used to store patient supplied flag and shows whether patient is supplied or not.1 - represents patient supplied 0 - represents patient is not supplied Default value is 0 ensures patient not supplied.
isInFormulary tinyint This column is used to store formulary flag and shows whether it is in formulary or not. 1 - represents is in formulary 0 - represents not in formulary Default value is 0 ensures item not in formulary
rxNotes varchar This column is used to store rx notes.
PRN tinyint This column is used to store PRN if frequency is PRN.1 - represents PRN frequency 0 - represents Not PRN Frequency Default value of PRN is 0
PRNIndicationId int This column is used to store prn indication id and is foreign key references id column of ip_rh_drugformulary_prnindication table.
deleteflag tinyint This column is used to store in delete flag and shows whether order is deleted or not. 1 - represents order is deleted 0 - represents order is not deleted Default value is 0 ensures order is not deleted.
CreatedBy int This column is used to store id of user who created order. referencing the uid column of users table.
CreatedDateTime datetime This column is used to store create date and time in yyyy-mm-dd hh:mm:ss format.
ModifiedBy int This column is used to store the id of the user who modified order.referencing the uid column of users table.
ModifiedDateTime datetime This column is used to store modified date and time in yyyy-mm-dd hh:mm:ss format.
orderType tinyint This column is used to store orderType value.1 - medication order 2 - IV order 3 - complex order Default Value 1 ensures this type is medication.
isSubstitute tinyint This column is used to store substitute values like 1, 2. Default value is 0 if substitute is not specified. 0 - Substitute is not allowed 1 - Substitute is allowed
rxOrderNo varchar This column is used to store Rx order numbers generated from staging order ID.
homeMedId int This column is used to store home medication ID. referencing the homemedid from the ip_homemedication table.
pomFlag tinyint This column is used to store in pom flag. 1 - represents is pom 0 - represents is not pom Default value is 0 ensure order is not pom.
disMedStatus smallint This column is used to store dispense medication status. The default value is NULL. NULL - No Action 0 - Stop, 1 - Start, 2 - Continue, -2 - Stopped, -3 - Started, -4 - Continued
eMarEntryInstructions varchar This column is used to store eMAR Entry Instructions
rphInstructions varchar This column is used to resident pharmacist instructions.
orderEntryInstructions varchar This column is used to store order entry Instructions.
ivRate varchar This column is used to store IV rate defined in formulary for IV and complex order type.
ivRateUom varchar This column is used to store units like ml/hr, ml/min.
isRenewInExpiringTab tinyint This column is used to store renew in expiring flag.1 - represents is eligible for renew in expiring 0 - represents not eligible for renew in expiring Default value is 0 ensure order is not eligible for renew.
episodeEncId int This column is used to store episode encounter ID.The column has a foreign key constraint referencing the encounterID from the enc table.
totalVolume decimal This is used to store total volume defined during order entry.
totalDose decimal This is used to store total dose defined during order entry.
isTitrationAllowed tinyint This column is used to store is titration or not. 1 - represents titration allowed 0 - represents titration not allowed. Default value is 0 ensure order is not titration allowed
internalNotes varchar This is used to store internal notes while adding the order.
issFlag tinyint This column is used to store issflag enabled or not. 1 - represents enbled ISS 0 - represents no ISS flag. Default value is 0 ensure order is not ISS.
configuredISSFlag tinyint This column is used to store configured ISS flag or not.1 - represents enbled ISS 0 - represents no ISS flag. Default value is 0 ensure order is not ISS.
minDose varchar This is used to store min Dose value like 1ml, 200MG.
maxDose varchar This is used to store max Dose value like 5ml, 500MG.
rxIncludeNow tinyint This column is used to store Rx include now or not.1 - represents first dose of Rx administration is at now 0 - represents first dose of Rx administration is at calculated datetime
infuseOverMinutes smallint This column is used to store infused over minutes.
infuseOverDisplayUOM varchar This column is used to store infuse over display UOM like Hour, Minutes
infuseOverDisplayValue varchar This column is used to store infuse over display value like 1 Hour, 1 Minute(s)
bulkflag tinyint This column is used to store bulk or not. 1 - represents order in bulk 0 - represents order not in bulk Default value is 0 ensure order is not in bulk.
