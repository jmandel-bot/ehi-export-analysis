EHI Export Schema
Table race_codes
Description This table stores information about race codes, their names, CDC race values, and flags for favorites and user interface visibility.
Column DataType Description
id (Primary Key) int This is the primary key column, which stores a unique integer identifier for each race code entry. It is an auto-incrementing value. For example: 1, 2, 3, etc.
Code varchar This column stores a short string (up to 6 characters) representing the race code. For example: AA, HISP, ASIAN.
NAME varchar This column stores the full name of the race code, with a maximum length of 100 characters. For example: African American, Hispanic, Asian.
CDCRace varchar This column stores the CDCRace value associated with the race code, with a maximum length of 100 characters. For example: Black or African American, "Hispanic or Latino, Asian.
Favorite tinyint This is a tinyint(1) column that stores a boolean value (0 or 1) indicating whether the race code is marked as a favorite. For example: 0 (not a favorite), 1 (favorite).
UIFlag tinyint This is a tinyint(1) column that stores a boolean value (0 or 1) indicating whether the race code is visible in the user interface. For example: 0 (not visible), 1 (visible).
