Table prisma_careplan_addresses
Description This table stores the relationship between care plans and their associated conditions.
Column DataType Description
ID (Primary Key) int This is the primary key column, which uniquely identifies each record in the table. It is an integer value that is automatically incremented when a new record is added. Example: 1, 2, 3, etc.
CAREPLAN_ID int This column stores the care plan ID associated with a specific condition. It is an integer value and is a foreign key that refers to the `ID` column in the `prisma_careplan_details` table. Example: 101, 102, 103, etc.
CONDITION_ID int This column stores the condition ID associated with a specific care plan. It is an integer value and can be NULL if there is no condition associated with the care plan. Example: 201, 202, 203, etc.
