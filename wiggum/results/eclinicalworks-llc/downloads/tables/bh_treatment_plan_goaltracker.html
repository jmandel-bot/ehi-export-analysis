Table bh_treatment_plan_goaltracker
Description This table stores information about the progress of users' treatment plan goals, including the structure mapping, progress value, and tracking date.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
structMappingid int identifier for the structure mapping associated with the goal. This is a foreign key that refers to the `id` column in the `bh_treatment_plan_pgoid_mapping` table. Example: 101, 102, 103.
structid int identifier for the structure associated with the goal. This is a foreign key that refers to the `id` column in the `bh_treatment_plan_dictionary` table. Example: 201, 202, 203.
progressValue tinyint A tiny integer value representing the progress of the goal, with a default value of '0'. Example: 0, 1, 2.
userid int identifier for the user associated with the goal. This is a foreign key that refers to the `UID` column in the `users` table. Example: 301, 302, 303.
trackDate datetime The date and time when the goal progress was tracked, with a default value of the current time stamp. Example: '2022-01-01 12:34:56', '2022-01-02 14:30:00'.
deleteFlag tinyint A tiny integer value representing whether the record is deleted or not, with a default value of '0'. A value of '1' indicates the record is deleted. Example: 0, 1.
