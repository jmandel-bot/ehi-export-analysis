EHI Export Schema
Table audioworkflowpt
Description This table stores information about audio workflow tests, including test types, descriptions, and frequency values for both right and left ears.
Column DataType Description
id (Primary Key) int Unique identifier for each audio workflow test (e.g., 1, 2, 3).
Encid int identifier for the encounter associated with the audio workflow test, which is a foreign key referring to the `encounterid` column in the `enc` table.
TestType int Integer representing the type of audio test (e.g., 0, 1, 2).
TestDesc varchar Description of the audio test (e.g., "Hearing Test", "Tinnitus Test").
FreqREar1 varchar Frequency values for the right ear, stored as strings (e.g., "250", "500", "1000").
FreqREar11 varchar Frequency values for the right ear, stored as strings (e.g., "250", "500", "1000").
FreqLEar1 varchar Frequency values for the left ear, stored as strings (e.g., "250", "500", "1000").
FreqLEar11 varchar Frequency values for the left ear, stored as strings (e.g., "250", "500", "1000").
BaseLine tinyint A tinyint value (0 or 1) indicating whether the test is a baseline test or not (e.g., 0 for not a baseline test, 1 for a baseline test).
