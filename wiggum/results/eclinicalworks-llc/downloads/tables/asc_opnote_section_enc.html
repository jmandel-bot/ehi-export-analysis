Table asc_opnote_section_enc
Description This table stores information about the association between operation notes, sections, and encounters, including display order and deletion status.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
opNoteid int identifier for the operation note associated with the record. This can be a foreign key referring to the operation notes table. Example: 101, 102, 103.
sectionid int Stores integer values. If value not provided : DEFAULT '0'
encounterid int Stores integer values. If value not provided : DEFAULT '0'
displayIndex int Stores integer values. If value not provided : DEFAULT '0'
deleteFlag tinyint A tinyint value (0 or 1) indicating whether the record has been deleted (1) or not (0). Example: 0, 1.
modifiedBy int Stores integer values. If value not provided : DEFAULT '0'
createdDateTime datetime time stamp for when the record was created. Example: '2021-09-01 12:34:56'.
modifiedDateTime datetime time stamp for when the record was last modified. Example: '2021-09-02 14:23:45'.
createdBy int identifier for the user who created the record. This can be a foreign key referring to the user table. Example: 401, 402, 403.
