Table inventory_saleproduct
Description This table stores information about the sale products in the inventory, including their prices, quantities, taxes, discounts, and other related details.
Column DataType Description
id (Primary Key) int Unique identifier for each sale product record. (e.g., 1, 2, 3)
Product_id int identifier for the product associated with the sale. (e.g., 101, 102, 103)
Sale_id int identifier for the sale associated with the product. (e.g., 1001, 1002, 1003)
Price decimal Price of the product in the sale. (e.g., 10.0000, 15.5000, 20.2500)
Qty decimal Quantity of the product sold. (e.g., 1.0000, 2.0000, 3.0000)
TaxAmt decimal Total tax amount applied to the product. (e.g., 0.5000, 1.0000, 1.5000)
UniqueString varchar Unique string associated with the sale product record. (e.g., "abc123", "def456", "ghi789")
IsDeleted tinyint Flag indicating if the record is deleted (1) or not (0). (e.g., 0, 1)
UpdatedBy int identifier for the user who last updated the record. (e.g., 1, 2, 3)
UpdatedOn datetime Date and time when the record was last updated. (e.g., '2021-09-01 12:34:56', '2021-09-02 14:56:23')
DiscountPerLine decimal Discount amount applied per line item. (e.g., 0.0000, 1.0000, 2.0000)
DiscReason_id int identifier for the discount reason associated with the sale product. (e.g., 1, 2, 3)
DiscountType int identifier for the type of discount applied to the sale product. (e.g., 1, 2, 3)
DiscValue decimal Discount value applied to the sale product. (e.g., 0.0000, 1.0000, 2.0000)
Stock_id int identifier for the stock associated with the sale product. (e.g., 10001, 10002, 10003)
encid int Encounter identifier associated with the sale product record. (e.g., 123, 456, 789)
CountryTax decimal Tax amount applied at the country level. (e.g., 0.0000, 0.5000, 1.0000)
CityTax decimal Tax amount applied at the city level. (e.g., 0.0000, 0.2500, 0.5000)
CosmeticTax decimal Tax amount applied for cosmetic products. (e.g., 0.0000, 0.1000, 0.2000)
OtherTax decimal Tax amount applied for other types of products. (e.g., 0.0000, 0.0500, 0.1000)
CountryTaxCode varchar Tax code associated with the country tax. (e.g., "CT001", "CT002", "CT003")
CityTaxCode varchar Tax code associated with the city tax. (e.g., "CIT001", "CIT002", "CIT003")
CosmeticTaxCode varchar Tax code associated with the cosmetic tax. (e.g., "COS001", "COS002", "COS003")
OtherTaxCode varchar Tax code associated with other types of taxes. (e.g., "OTH001", "OTH002", "OTH003")
Currency varchar Currency used for the sale product prices and taxes. (e.g., "USD", "EUR", "GBP")
patientid int identifier for the patient associated with the sale product. (e.g., 1001, 1002, 1003)
providerid int identifier for the provider associated with the sale product. (e.g., 2001, 2002, 2003)
bundle_id int identifier for the bundle associated with the sale product. (e.g., 3001, 3002, 3003)
bundleqty decimal Quantity of the bundle associated with the sale product. (e.g., 0.0000, 1.0000, 2.0000)
isdiscarded tinyint Flag indicating if the sale product is discarded (1) or not (0). (e.g., 0, 1)
itemid int identifier for the item associated with the sale product. (e.g., 4001, 4002, 4003)
dispenseQty decimal Quantity of the product dispensed. (e.g., 0.0000, 1.0000, 2.0000)
adjstock_id int identifier for the adjusted stock associated with the sale product. (e.g., 5001, 5002, 5003)
adjexpirydate datetime Date and time when the adjusted stock expires. (e.g., '2022-09-01 12:34:56', '2022-09-02 14:56:23')
invoiceunlocked int Flag indicating if the invoice is unlocked (1) or locked (0). (e.g., 1, 0)
