EHI Export Schema
Table extract_billing_units
Description This table stores information related to billing units for each patient's extracted bottle.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer. It uniquely identifies each record in the `extract_billing_units` table. Example: 1, 2, 3, etc.
PATIENTid int This is a foreign key that refers to the `id` column in the `patients` table. It represents the patient associated with the billing units. Example: 101, 102, 103, etc.
QUEUEid int This is a foreign key that refers to the `id` column in the `queues` table. It represents the queue in which the patient's billing units are processed. Example: 201, 202, 203, etc.
BOTTLE_id int This is a foreign key that refers to the `id` column in the `bottles` table. It represents the specific bottle from which the units are extracted and billed. Example: 301, 302, 303, etc.
UNITS_BILLED int This is an integer column that stores the number of units billed for the extracted bottle. It has a default value of 0. Example: 0, 5, 10, 15, etc.
