EHI Export Schema
Table vision_fittingcomments
Description This table stores the contact lens related fitting comments, such as toric orientation, centration coverage, movement, etc., for the right and left eyes.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key.
visionrxid int This column stores the vision Rx id for which fitting comments are getting documented. This column refers to the id from the visionrx table.
encounterid int This column stores the encounter id for fitting comments which are being documented. This column refers to the encounterid from enc table.
centrationcoverageOD text This field is used to store comments for centration coverage indicating whether the cornea is completely covered or partially covered for the right eye.
centrationcoverageOS text This field is used to store comments centration coverage indicating whether the cornea is completely covered or partially covered for the left eye.
movementOD text This field is used to store measurements of contact lens periphery, which lies out of the cornea in movements, like up gaze, down gaze, and horizontal gaze for the right eye.
movementOS text This field is used to store measurements of contact lens periphery, which lies out of the cornea in movements, like up gaze, down gaze, and horizontal gaze for the left eye.
toricorientationOD text This field is used to store details on horizontal or vertical orientation for toric lenses for the right eye.
toricorientationOS text This field is used to store details on horizontal or vertical orientation for toric lenses for the left eye.
nafipatternOD text This field is used to store comments for nafi patterns for the right eye.
nafipatternOS text This field is used to store comments for nafi patterns for left eye.
Fitting_Comments text This column contains comments for contact lens Rx will be stored in this field.
Fitting_CommentsOS text This field is used to store overall fitting comments for left eye are stored in this field.
Fitting_CommentsOD text This field is used to store overall fitting comments for right eye are stored in this field.
