EHI Export Schema
Table assignedpnenc
Description This tables stores information regarding Progress Notes assigned to a provider for review.
Column DataType Description
id (Primary Key) int This is auto-incremented primary key in This column .
Encounterid int This column stores id of an encounter which is assigned for review to provider or staff. This column refers to encounterid column in enc table.
AssignedToid int This column stores id of a user to whom a Progress Note is assigned to for review. This column refers to UID column in user table.
LockStatus smallint This column stores values 0/1 to indicate whether the Progress Notes being assigned to other provider/staff for review is locked or not. 0 - Progress Note being assigned is not locked, 1 - Progress Notes being assigned is locked.
Notes text This column stores free text notes documented by provider/staff while assigning Progress Notes for review to other provider/staff.
randomselection tinyint This columns stores values 0/1 to indicate whether the progress note was selected in a random selection algorithm for the respective month. This is custom column added for HealthStat.
assigntosupprid int This column stores is of supervising provider id for a Progress Notes. This column refers to UID column in user table. This is custom column added for HealthStat.
