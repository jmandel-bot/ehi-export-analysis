EHI Export Schema
Table surescript_reqotherinfo
Description This table stores additional information related to prescription requests, including supervisor and diagnosis details.
Column DataType Description
ReqInfoid int This column stores the auto generated id
RxType varchar It stores type of eprescription request
Messageid varchar Unique identifier for the message related to the prescription request. Example: "MSG12345", "MSG67890".
RelatedMessageid varchar Unique identifier for a related message, if applicable. Example: "MSG54321", "MSG09876".
Supvr_SPI varchar This column stores SPI of the supervising provider. (No information on data type, size and default value).
Supvr_PriorAuth varchar This column stores supervising provider's prior authorization. (No information on data type, size and default value).
Supvr_Prefix varchar This column stores supervising provider's prefix. (No information on data type, size and default value).
Supvr_FName varchar This column stores supervising provider's first name. (No information on data type, size and default value).
Supvr_MName varchar This column stores supervising provider's middle name. (No information on data type, size and default value).
Supvr_LName varchar This column stores supervising provider's last name. (No information on data type, size and default value).
Supvr_Suffix varchar This column stores supervising provider's suffix. (No information on data type, size and default value).
Supvr_AddressLine1 varchar This column stores supervising provider's address. (No information on data type, size and default value).
Supvr_AddressLine2 varchar This column stores supervising provider's additional address. (No information on data type, size and default value).
Supvr_City varchar This column stores supervising provider's city. (No information on data type, size and default value).
Supvr_State char This column stores supervising provider's state. (No information on data type, size and default value).
Supvr_ZipCode varchar This column stores supervising provider's zip code. (No information on data type, size and default value).
Supvr_Tel varchar This column stores supervising provider's telephone number. (No information on data type, size and default value).
Supvr_Mobile varchar This column stores supervising provider's mobile number. (No information on data type, size and default value).
Supvr_Pager varchar This column stores supervising provider's pager number. (No information on data type, size and default value).
PriRxDiagnosisQualifier varchar Primary prescription diagnosis qualifier. Example: "ICD-10", "ICD-9".
PriRxDiagnosisValue varchar Prescription primary diagnosis code
ClinicalInformationQualifier varchar This column stores Clinical Information Qualifier. (No information on data type, size and default value).
PriorAuthQualifier char This column stores Prior Authorization Qualifier. (No information on data type, size and default value).
PriorAuthValue varchar This column stores Prior Authorization value. (No information on data type, size and default value).
MD_PriRxDiagnosisQualifier varchar This column stores Primary Rx Diagnosis Qualifier for Medication Dispensed. (No information on data type, size and default value).
MD_PriRxDiagnosisValue varchar This column stores Primary Rx Diagnosis Value for Medication Dispensed (No information on data type, size and default value).
MD_ClinicalInformationQualifier varchar This column stores Clinical Information Qualifier for Medication Dispensed (No information on data type, size and default value).
MD_PriorAuthQualifier char This column stores Prior Auth Qualifier for Medication Dispensed (No information on data type, size and default value).
MD_PriorAuthValuevar varchar This column stores Prior Auth Value for Medication Dispensed (No information on data type, size and default value).
MD_PriorAuthValue varchar This column stores Prior Auth Value for Medication Dispensed (No information on data type, size and default value).
