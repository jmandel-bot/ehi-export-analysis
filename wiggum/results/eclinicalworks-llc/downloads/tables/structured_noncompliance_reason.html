Table structured_noncompliance_reason
Description This table stores the reasons for noncompliance in a structured format, along with information about non-structured reason and if the record is deleted.
Column DataType Description
reasonId (Primary Key) int This is the primary key and an auto-incrementing integer that uniquely identifies each reason for noncompliance. Example: 1, 2, 3, etc.
reason varchar This is a varchar(255) column that stores the description of the noncompliance reason. Example: "Late submission", "Incomplete documentation", etc.
deleteflag tinyint This is a tinyint(4) column that indicates whether the record is deleted or not. A value of '0' means the record is not deleted, and a value of '1' means the record is deleted. Example: 0, 1.
nonStructuredReason tinyint This is a tinyint(4) column that indicates whether the reason is non-structured or not. A value of '0' means the reason is structured, and a value of '1' means the reason is non-structured. Example: 0, 1.
