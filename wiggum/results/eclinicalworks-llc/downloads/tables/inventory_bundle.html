Table inventory_bundle
Description This table stores information about inventory bundles, including their names, barcodes, descriptions, categories, charges, discounts, taxes, and other related details.
Column DataType Description
id (Primary Key) int Unique identifier for each inventory bundle. Example: 1, 2, 3.
name varchar The name of the inventory bundle. Example: "Electronics Bundle", "Kitchen Appliances Bundle".
barcode varchar The barcode associated with the inventory bundle. Example: "123456789012", "987654321098".
Description varchar A brief description of the inventory bundle. Example: "A bundle of electronic gadgets", "A set of essential kitchen appliances".
bundlecategory_id int The identifier for the category that the inventory bundle belongs to. This is a foreign key that refers to the `id` column in another table (e.g., `bundle_category`). Example: 1, 2, 3.
ChargesAmt decimal The total charges amount for the inventory bundle. Example: 100.0000, 250.0000.
DiscountType int The type of discount applied to the inventory bundle. Example: 1 (percentage), 2 (fixed amount).
DiscValue decimal The value of the discount applied to the inventory bundle. Example: 10.0000, 25.0000.
DiscountAmt decimal The total discount amount for the inventory bundle. Example: 10.0000, 50.0000.
DiscReason_id int The identifier for the reason of the discount. This is a foreign key that refers to the `id` column in another table (e.g., `discount_reason`). Example: 1, 2, 3.
TaxesAmt decimal The total taxes amount for the inventory bundle. Example: 8.0000, 20.0000.
TotalAmt decimal The total amount for the inventory bundle after applying discounts and taxes. Example: 98.0000, 220.0000.
IsDiscontinued tinyint A flag indicating whether the inventory bundle is discontinued or not. Example: 0 (not discontinued), 1 (discontinued).
IsDeleted tinyint A flag indicating whether the inventory bundle is deleted or not. Example: 0 (not deleted), 1 (deleted).
UpdatedBy int The identifier for the user who last updated the inventory bundle. This is a foreign key that refers to the `id` column in another table (e.g., `user`). Example: 1, 2, 3.
UpdatedOn datetime The date and time when the inventory bundle was last updated. Example: '2021-10-01 12:34:56', '2021-10-02 14:30:00'.
CreatedBy int The identifier for the user who created the inventory bundle. This is a foreign key that refers to the `id` column in another table (e.g., `user`). Example: 1, 2, 3.
CreatedOn datetime The date and time when the inventory bundle was created. Example: '2021-09-01 10:00:00', '2021-09-02 11:30:00'.
