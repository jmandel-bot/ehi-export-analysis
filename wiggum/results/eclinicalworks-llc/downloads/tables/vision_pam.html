EHI Export Schema
Table vision_pam
Description This table stores information about patients' eye measurements during an encounter.
Column DataType Description
id (Primary Key) int This is the primary key and an auto-incrementing integer that uniquely identifies each record in the vision_pam table. Example: 1, 2, 3, etc.
encounterid int This is a foreign key that refers to the `encounterid` column in the `enc` table. It is an integer that uniquely identifies the encounter associated with the eye measurements. Example: 101, 102, 103, etc.
r_eye varchar This is a varchar column that stores the measurement of the right eye during the encounter. It has a maximum length of 10 characters. Example: '20/20', '20/40', etc.
l_eye varchar This is a varchar column that stores the measurement of the left eye during the encounter. It has a maximum length of 10 characters. Example: '20/20', '20/40', etc.
