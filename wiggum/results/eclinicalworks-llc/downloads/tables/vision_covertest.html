EHI Export Schema
Table vision_covertest
Description This table stores information about vision tests performed during a patient's encounter.
Column DataType Description
id (Primary Key) int This column is an auto incremental primary key.
encounterid int This column stores encounterid and has foreign key constraint on "encounterid" column of "enc" table.
rx_type tinyint This column stores values for rx option. Values that can be stored are 0 or 1. Default value is NULL. This column is optional and hence when user does not enter any value, NULL will be stored in them.
uct_dist_test int Integer value representing the uncorrected distance test result. Example: 20, 25, 30.
uct_near_test int Integer value representing the uncorrected near test result. Example: 15, 20, 25.
act_dist_test int Integer value representing the corrected distance test result. Example: 10, 15, 20.
act_near_test int Integer value representing the corrected near test result. Example: 5, 10, 15.
uct_dist smallint Small integer value representing the uncorrected distance vision. Example: 100, 200, 300.
uct_near smallint Small integer value representing the uncorrected near vision. Example: 50, 100, 150.
act_dist smallint Small integer value representing the corrected distance vision. Example: 20, 40, 60.
act_near smallint Small integer value representing the corrected near vision. Example: 10, 20, 30.
comments varchar A string (up to 255 characters) containing any additional comments about the vision test. Example: "Patient needs new glasses", "Vision improved since last visit".
