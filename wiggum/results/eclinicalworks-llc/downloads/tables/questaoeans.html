EHI Export Schema
Table questaoeans
Description This table stores information about the answer to the question associated with Ask at Order Entry (AOE) with a lab or diagnostic imaging order.
Column DataType Description
Qsid int This column indicates a question id. This column refers to the column Qsid from the question table.
reportid int This column stores the id of the lab order. Column refers to the reportid column in the labdata table.
answer varchar This column stores the answer of the question associated with Ask at Order Entry (AOE), associate with lab or diagnostic imaging order.
Answercode varchar This columns stores the answercode of the answer selected for an AOE question at the time of placing the order. This column refers to the column optioncode from the aoeoptions table.
AnsweredBy int This column stores the user id of the Provider/Staff who answered the question of Ask at Order Entry (AOE). This column refers to the column UID from the user table.
AnsweredDate datetime This column stores the date on which the answer of Ask at Order Entry (AOE) is given.
AnsweredTime datetime This column stores the time on which the answer of Ask at Order Entry (AOE) is given.
labid int This column stores the id of the lab for which Ask at Order Entry (AOE) is set.
