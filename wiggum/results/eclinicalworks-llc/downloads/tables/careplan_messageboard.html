Table careplan_messageboard
Description This table stores messages related to care plans, including information about the user, message type, and staff involved.
Column DataType Description
CarePlanId int An integer representing the unique identifier of the care plan associated with the message. Example: 1024
UserId int An integer representing the unique identifier of the user who posted the message. Example: 5678
MsgType varchar A varchar(20) representing the type of message being posted. Example: 'Update', 'Reminder', or 'Alert'
Msg text A text column containing the actual content of the message. Example: 'Patient has completed physical therapy session.'
ModifiedTime datetime A datetime column representing the last modified time of the message. Example: '2021-09-15 14:30:00'
StaffId int An integer representing the unique identifier of the staff member who posted or modified the message. Example: 1234
EventId int An integer representing the unique identifier of the event associated with the message, if applicable. Example: 7890
Id (Primary Key) int An integer representing the unique identifier of the message, which is the primary key for the table. Example: 1
