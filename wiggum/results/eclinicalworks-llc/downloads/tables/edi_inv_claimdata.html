EHI Export Schema
Table edi_inv_claimdata
Description This table stores information related to Claims, including details about the claim, provider, and various dates and codes associated with the claim.
Column DataType Description
InvId (Primary Key) int This column stores the invoice Id for claim. This has reference to Id column of edi_invoice table
EmpRelated varchar Indicates if the claim is employment-related (varchar). Example: 'Y', 'N'
AccInd varchar Accident indicator (varchar). Example: 'A', 'O', 'N'
AccState varchar State where the accident occurred (varchar). Example: 'CA', 'NY'
SymInd varchar Symptom indicator (varchar). Example: '1', '0'
SymDate datetime Date when the symptoms first appeared (date). Example: '2021-01-01'
ExtAccCause varchar External cause of the accident (varchar). Example: 'Car accident', 'Fall'
AccRespInd char This column stores information about Accident Responsibility Indicator. Possible values are A - Auto Accident, W -Worker's Compensation, O-Other Accident
AccHour varchar Hour when the accident occurred (varchar). Example: '08:00', '14:30'
SimSymInd varchar Similar symptom indicator (varchar). Example: 'Y', 'N'
SimSymDate datetime Date when similar symptoms first appeared (date). Example: '2021-01-01'
UnWorkFromDt datetime Date when the patient became unable to work (date). Example: '2021-01-10'
UnWorkToDt datetime Date when the patient is expected to return to work (date). Example: '2021-02-01'
RefProvId int Unique identifier for the referring provider (integer). Example: 98765
RefProvNPI varchar National Provider Identifier (NPI) of the referring provider (varchar). Example: '1234567890'
RefProvIdType char Type of identifier used for the referring provider (char). Example: '01', '02'
AdmDate datetime Date of admission (date). Example: '2021-01-05'
DisDate datetime Date of discharge (date). Example: '2021-01-10'
LabInd varchar This column stores the information about the tests are done in outside lab or not. Y indicates tests are done in Outside lab. N indicates that outside lab is not involved.
LabCharges decimal Charges for laboratory services (decimal). Example: 150.00
DocInd varchar This column stores Documentation indicator. For e.g. 1- In email, 2-Fax, NS-Not specified
RelInfoInd varchar This column stores Release of Information indicator. Possible values are Y-Yes Release Allowed, N-No Release, M-Modified or Restricted Release, I-ANSI 5010 Informed Consent
RelInfoDate datetime This column stores date when Release information was signed and provided
DateLastSeen datetime Date when the patient was last seen by the provider (date). Example: '2021-01-20'
initialTrmtDate datetime Date of the initial treatment (date). Example: '2021-01-01'
PtWeight char Patient's weight (char). Example: '150', '200'
AmbTnsCode varchar Ambulance transportation code (enum). Example: 'I', 'R', 'T', 'X'
AmbTnsReasonCode varchar Ambulance transportation reason code (enum). Example: 'A', 'B', 'C', 'D', 'E'
AmbTnsDistance varchar Distance traveled for ambulance transportation (varchar). Example: '10', '25'
RoundTripPurposeDescription varchar Description of the purpose for the round trip (varchar). Example: 'Medical appointment', 'Emergency room visit'
StretcherPurposeDescription varchar Description of the purpose for using a stretcher (varchar). Example: 'Patient immobility', 'Severe injury'
AmbCondCode varchar Ambulance condition code (enum). Example: 'Y', 'N'
AmbCondId1 varchar Ambulance condition identifier 1 (enum). Example: '01', '02', '03', '04', '05', '06', '07', '08', '09', '60'
AmbCondId2 varchar Ambulance condition identifier 2 (enum). Example: '01', '02', '03', '04', '05', '06', '07', '08', '09', '60'
AmbCondId3 varchar Ambulance condition identifier 3 (enum). Example: '01', '02', '03', '04', '05', '06', '07', '08', '09', '60'
AmbCondId4 varchar Ambulance condition identifier 4 (enum). Example: '01', '02', '03', '04', '05', '06', '07', '08', '09', '60'
AmbCondId5 varchar Ambulance condition identifier 5 (enum). Example: '01', '02', '03', '04', '05', '06', '07', '08', '09', '60'
HomeBoundCondCode varchar Homebound condition code (enum). Example: 'Y'
HomeBoundCondId1 varchar Homebound condition identifier 1 (enum). Example: 'IH'
ideNumber varchar IDE (Investigational Device Exemption) number (varchar). Example: '123456'
SplPgmId varchar Special program identifier (enum). Example: '01', '02', '03', '05', '07', '08', '09'
EPSDTReferral varchar EPSDT (Early and Periodic Screening, Diagnostic, and Treatment) referral (enum). Example: 'N', 'Y'
EPSDTRefSts varchar EPSDT referral status (enum). Example: 'AV', 'NU', 'S2', 'ST'
AdjustVoidClaimCode varchar Adjustment or void claim code (varchar). Example: '123', '456'
DXCode varchar Diagnosis code (varchar). Example: 'A1234', 'B5678'
CrimeRelated varchar Indicates if the claim is crime-related (varchar). Example: 'Yes', 'No'
DocIdType char Type of identifier used for the documentation (char). Example: '01', '02'
DocSentDate datetime Date when the documentation was sent (date). Example: '2021-01-15'
OrgRefProvNPI varchar Original referring provider's NPI (varchar). Example: '1234567890'
OrgRefProvIdType char Type of identifier used for the original referring provider (char). Example: '01', '02'
PWK06 varchar Additional claim information (varchar). Example: 'Additional notes', 'Special instructions'
ConditionCode char Condition code (char). Example: '01', '02'
YesOrNoResponseCode char Yes or no response code (char). Example: 'Y', 'N'
PregIndctor char Pregnancy indicator (char). Example: 'Y', 'N'
AcuteManifestationDate datetime Date of acute manifestation (date). Example: '2021-01-05'
ConditionInformationCode varchar Condition information code (varchar). Example: 'A123', 'B456'
LastXrayDate datetime Date of the last X-ray (date). Example: '2021-01-10'
HearingVisionPrescriptionDt datetime Date of the hearing or vision prescription (date). Example: '2021-01-15'
LastWorkedDate datetime Date when the patient last worked (date). Example: '2021-01-05'
AuthReturnToworkDt datetime Date when the patient is authorized to return to work (date). Example: '2021-02-01'
RelinquishedCareDt datetime Date when care was relinquished (date). Example: '2021-01-20'
PropCasualityFirstDt datetime This column stores information about Property and Casualty date of first contact(First Visit or Consultation)
RepricerReceivedDt datetime This column stores information about Repricer Receiver Date
AssumedCareDate datetime Date when care was assumed (date). Example: '2021-01-01'
InvestigationalExemptionNo varchar This column stores information about Investigational Device Exemption Number
claimIdForTrans varchar This column stores information about Claim Identifier For Transmission Intermediaries
demonstrationprojectid varchar Demonstration project identifier (varchar). Example: '123', '456'
careplanoversight varchar This column stores information about Care Plan Oversight(Care Plan Oversight Number)
isInpatient tinyint Indicates if the claim is for an inpatient service (tinyint). Example: 0 (No), 1 (Yes)
isFacility tinyint Indicates if the claim is for a facility service (tinyint). Example: 0 (No), 1 (Yes)
