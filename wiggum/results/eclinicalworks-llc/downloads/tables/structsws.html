EHI Export Schema
Table structsws
Description This table stores the answers for structured questions [with tblname=structsws in structdatadetail table] when data is saved from CQW. This table is used when the itemkey 3493_VAA is disabled.
Column DataType Description
encounterid int An integer representing the unique identifier for an encounter. Example: 1023
catid int An integer representing the unique identifier for a category within the encounter. Example: 5
itemid int An integer representing the unique identifier for an item within the category. Example: 12
detailid int An integer representing the unique identifier for a specific detail within the item. Example: 45
VALUE text A text field storing the value associated with the detail. Example: "Sample value"
valueid varchar A varchar(255) field storing the unique identifier for the value. Example: "val_123"
notes text A text field storing any additional notes or comments related to the detail. Example: "This is a sample note."
chartdisplay tinyint A tinyint(2) field indicating whether the detail should be displayed on a chart (1) or not (0). Example: 1
displayIndex tinyint A tinyint(1) field representing the display order of the detail within the item. Example: 3
modifieddatetime datetime A datetime field storing the date and time when the record was last modified. Example: "2021-09-15 14:30:00"
