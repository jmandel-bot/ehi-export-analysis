Table ccmr_groupdocrecurrentpatients
Description This table stores information about recurrent patients in a group, including their encounter and patient IDs, and whether the patient encounter has been created or not.
Column DataType Description
id (Primary Key) int This is the primary key for the table, and it is an auto-incrementing integer. It uniquely identifies each record in the table. Example: 1, 2, 3, etc.
recGroupEncid int This is an integer that stores the recurrent group encounter id. Example: 101, 102, 103, etc.
recGroupPatientid int This is an integer that stores the recurrent group patient id. Example: 1001, 1002, 1003, etc.
IsPatientEncCreated int This is an integer that indicates whether the patient encounter has been created or not. It has a default value of '0'. A value of '1' means the patient encounter has been created, while '0' means it has not been created. Example: 0 or 1.
recGroupPatientEncid int This is an integer that stores the recurrent group patient encounter id. Example: 2001, 2002, 2003, etc.
deleteFlag int This is an integer with a default value of '0'. It is used to mark a record as deleted without actually removing it from the table. A value of '1' means the record is deleted, while '0' means it is not deleted. Example: 0 or 1.
