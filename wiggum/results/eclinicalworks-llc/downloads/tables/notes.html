EHI Export Schema
Table notes
Description This table stores free text notes from the HPI and Examination windows from the Progress Notes.
Column DataType Description
encounterid int This column stores id of an encounter for which notes are being stored in This column . This column refers to encounterid column in enc table.
catid int This column store itemid of a category for which notes have been added by the end user. This column refers to itemid column in items table.
notes text This column stores free text or pre-configured notes provided by end user for each category on user interface.
type char This column stores a character indicating screen/module the notes has been docuemnted on. H and G - For notes from HPI screen, E - For notes from Examination screen.
notesHTML varchar This column stores free text or pre-configured notes provided by end user for each category on user interfacein HTML format.
