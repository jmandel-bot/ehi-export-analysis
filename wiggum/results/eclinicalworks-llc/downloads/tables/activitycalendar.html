EHI Export Schema
Table activitycalendar
Description This table stores information about activities and events in a calendar, including their start and end times, recurrence patterns, and related metadata.
Column DataType Description
id (Primary Key) int This is the primary key of This column and is an auto incremented value.
Title varchar The title or name of the activity or event. Example: team meeting, birthday party.
StartDate datetime The date when the activity or event starts. Example: 2022-01-01.
StartTime datetime The time when the activity or event starts. Example: 14:00:00.
EndDate datetime The date when the activity or event ends. Example: 2022-01-01.
EndTime datetime The time when the activity or event ends. Example: 16:00:00.
DeleteFlag tinyint A flag indicating if the activity or event has been deleted (1) or not (0). Example: 0, 1.
AllDay tinyint A flag indicating if the activity or event is an all-day event (1) or not (0). Example: 0, 1.
CreatedDate datetime The time stamp when the activity or event was created. Example: 2022-01-01 12:00:00.
ModifiedDate datetime The time stamp when the activity or event was last modified. Example: 2022-01-02 12:00:00.
RecFlag tinyint This column is not in use anymore
RecStartDate datetime This column is not in use anymore
RecEndInfo varchar This column is not in use anymore
RecPattern tinyint This column is not in use anymore
RecPatternEvery varchar This column is not in use anymore
createuserid int This column stores the user id of the user who created the event. This refers to the UID column of the user table.
moduserid int The unique identifier of the user who last modified the activity or event. Example: 1, 2, 3.
facilityid int This column stores the facility id of the event where it is created. This refers to the id column of the edi_facilities table.
