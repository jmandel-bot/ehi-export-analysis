EHI Export Schema
Table user_signature_document
Description This table stores information about user signature documents, including the file path, file name, user ID, signed date and time, and a delete flag.
Column DataType Description
id (Primary Key) int Unique identifier for each user signature document record. Example: 1, 2, 3.
filepath varchar The file path where the user signature document is stored. Example: "/documents/signatures/user1/", "/documents/signatures/user2/".
filename varchar The name of the user signature document file. Example: "signature1.pdf", "signature2.pdf".
uid int The unique identifier of the user associated with the signature document. This is a foreign key that refers to the `uid` column in the `users` table. Example: 101, 102, 103.
signeddatetime datetime The date and time when the user signed the document. Example: "2021-08-01 12:34:56", "2021-08-02 14:30:00".
deleteflag tinyint A flag indicating whether the user signature document has been deleted (1) or not (0). Example: 0, 1.
