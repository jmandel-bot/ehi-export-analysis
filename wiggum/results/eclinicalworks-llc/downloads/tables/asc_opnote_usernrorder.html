EHI Export Schema
Table asc_opnote_usernrorder
Description This table is used to sort macros in operative notes on the user side.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key of This column .
encounterid int This column stores enc id value. Map to encounterid column of enc table. Column does not expect NULL . Default value is '0'.
sectionItemid int This column stores section item id. Map to itemid column of items table. Column does not expect NULL . Default value is '0'.
propid int This column stores prop id for macro. Maps to propid column of table properties.
versionid int This column store version id for macro userside. Maps to id column of table asc_enc_custom_tpl_version.
displayIndex smallint This column is used to represent the display order of the record when grouped by certain record identifier.
userid int This column is used to store the Userid from Users Master Table. Can be logged in user id or provider id for all available usertypes.
createdDateTime datetime This column stores Date and Time when this record was created.
modifiedDateTime datetime This column stores Date and Time when this record was modified.
deleteFlag tinyint This column stores value 0 / 1 indicating if the record is deleted or not, this is soft delete. 0 - Not Deleted , 1 -Deleted.
