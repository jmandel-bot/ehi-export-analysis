EHI Export Schema
Table asc_enc_mappings
Description This table contains asc_enc_mappings-related information such as masterEncId, AssocEncId, and opNoteId.
Column DataType Description
id (Primary Key) int This is an auto increment primary key column of This column .
masterEncid int Refers to encid in enc table
assocEncid int The identifier for the associated encounter. Example: 2001, 2002, 2003.
deleteFlag int This column store delete flag. It will be set to 1 if entry is deleted else 0.
modifybyid int This column stores current user logged-in user id. It is primary id user table.
modifiedDate datetime This column is used to capture date time when details is updated.
isLegacy tinyint This column stores tinyint value, It can be either 0 or 1.
opNoteid int This column stores operative note id
