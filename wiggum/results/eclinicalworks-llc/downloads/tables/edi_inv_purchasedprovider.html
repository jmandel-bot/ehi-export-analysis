EHI Export Schema
Table edi_inv_purchasedprovider
Description This table stores purchased service providers and related information for specific invoices and invoice line items.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3
Invoiceid int identifier for the invoice associated with the purchased service. This can be a foreign key referring to the `id` column in the `edi_invoice` table. Example: 101, 102, 103
InvoiceCptid int identifier for the invoice line item (CPT code) associated with the purchased service. This can be a foreign key referring to the `id` column in the `invoice_cpt` table. Example: 1001, 1002, 1003
PurchasedServiceProviderType tinyint Numeric code representing the type of purchased service provider. Example: 1 (Physician), 2 (Hospital), 3 (Laboratory)
PurchasedServiceProviderNpi varchar National Provider identifier (NPI) of the purchased service provider. Example: '1234567890', '0987654321'
PurchasedServiceProviderid int identifier for the purchased service provider.
PurchasedServiceAmount decimal Monetary amount of the purchased service. Example: 100.00, 250.50, 75.25
PurchasedServiceProvideridentifier varchar Unique identifier for the purchased service provider. Example: 'PSP123', 'PSP456', 'PSP789'
SecondaryidQualifier char Two-character code representing the type of secondary identifier for the purchased service provider. Example: '1G', 'G2', 'LU'
Secondaryid varchar Secondary identifier for the purchased service provider. Example: 'S123', 'S456', 'S789'
deleteFlag tinyint A flag indicating if the record has been deleted (1) or not (0). Example: 0, 1
