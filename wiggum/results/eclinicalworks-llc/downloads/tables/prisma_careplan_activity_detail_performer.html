Table prisma_careplan_activity_detail_performer
Description This table stores information about the performers involved in care plan activities, such as practitioners and care teams.
Column DataType Description
ID (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3, etc.
CAREPLAN_ACTIVITY_ID int The identifier of the care plan activity that the performer is associated with. This is a foreign key that refers to the `ID` column in the `prisma_careplan_activity_details` table. Example: 101, 102, 103, etc.
PERFORMER_PRACTITIONER_ID int The identifier of the practitioner who performed the care plan activity. This can be NULL if the performer is a care team instead of an individual practitioner. Example: 1001, 1002, 1003, etc.
PERFORMER_CARETEAM_ID int The identifier of the care team that performed the care plan activity. This can be NULL if the performer is an individual practitioner instead of a care team. Example: 2001, 2002, 2003, etc.
