EHI Export Schema
Table playerextract
Description This table stores player information replicated from the NFL production database.
Column DataType Description
UID int This column stores the user table UID for the player that's created in ecw.
Playerid int This column stores the player's id. NFL database maintains playerid to uniquely identify the player
dtLoadDate datetime This column tells the date when the player row was loaded in NFL database.
Season int This column stores the NFL season represents the year when the league happened.
FootballName varchar the column represent the player's football name
College varchar This column represent the college that the player played for, represented.
CollegeConference varchar This column stores the zonal conference that the player/college represented.
CollegeConferenceAbbr varchar This column stores the college conference abbreviation. ex. SEC
SSNumber varchar This column stores the player's SSN.
Height varchar this column stores the player's height.
Weight int this column stores the player's weight as logged in NFL database.
PositionAbbr varchar This column stores the player position where he plays in the league. ex. QB, WR, RB
PositionDescription varchar tells the player position description. QB - quarterback, WR- wide receiver, RB - running back
JerseyNumber int stores the player jersey number he wears while playing the league. ex. Tom brady - 12
EntryYear int stores the info of year the player entered the league.
DraftRound int this column stores the draft round. at what round the player got picked in the draft process.
DraftNumber int Stores the overall draft number of the player as picked for a club in the draft process.
DraftClub varchar stores the club that the player got drafted/picked.
RookieYear int This stores the year of when the player first played for a club in NFL leage.
EnterUnderClassman char This stores the class info of the player. This data is replicated from NFL database and is not modified in ecw.
SupplementalDraft char This stores the supplemental draft of the player. This data is replicated from NFL database and is not modified in ecw.
CurrentClub varchar This stores the info of the club where the player is associated with.
CurrentClubFullName varchar stores the full name of the club that the player belongs to.
StatusDescriptionAbbr varchar current status of the player if he is a free agent or other playing status.
StatusShortDescription varchar short status description. FA for free agent. This data is replicated from NFL database and is not modified in ecw.
strStatusDescription varchar This stores the current status of the player. This data is replicated from NFL database and is not modified in ecw.
PracticeSquadEligible char This stores the practice squad eligibility info of the player. This data is replicated from NFL database and is not originated in ecw.
TotalPensionCredits int This stores the total pension credits of the player. This data is replicated from NFL database and is not modified in ecw.
AccruedSeasons int This stores the total number of seasons accrued by the player. This data is replicated from NFL database and is not modified in ecw.
MinSalaryCredits int This stores the minimum salary credits of the player. This data is replicated from NFL database and is not modified in ecw.
NFLExperience int This stores the total NFL experience of the player. This data is replicated from NFL database and is not modified in ecw.
EliasPlayerid varchar this column stores the alternate playerid if applicable.
Agentid int This stores the agent id of the player. This data is replicated from NFL database and is not modified in ecw.
Clubid int This stores the club id of the player. This data is replicated from NFL database and is not originated in ecw.
EntryClub varchar This stores the entry club of the player that he had association with. This data is replicated from NFL database and is not modified in ecw.
DraftPick int This stores the draft pick of the player. This data is replicated from NFL database and is not modified in ecw.
GSISid varchar This stores the GSIS id of the player. This data is replicated from NFL database and is not modified in ecw.
PracticeSquadCredit int This stores the PracticeSquadCredit of the player. This data is replicated from NFL database and is not modified in ecw.
NFSid int This stores the NFSid of the player. This data is replicated from NFL database and is not modified in ecw.
Sportid int this column is with value 1 when creating, updating player.
datemodified datetime Date and time when the record was last modified. Example: '2021-09-01 12:30:00'.
portalFlag int Flag indicating if the player's information is available in the portal. Example: 0 (Not available), 1 (Available).
