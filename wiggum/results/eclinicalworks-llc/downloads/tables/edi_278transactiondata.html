EHI Export Schema
Table edi_278transactiondata
Description This table stores information related to EDI 278 transactions, including payload, response, and other related data.
Column DataType Description
id (Primary Key) int Unique identifier for each transaction record. (e.g., 1, 2, 3)
said int Foreign key referring to the id of the related service agreement. (e.g., 101, 102, 103)
Payload278 text Text containing the EDI 278 transaction payload. (e.g., "ISA*00*...IEA*1*000000001~")
Response278 text Text containing the EDI 278 transaction response. (e.g., "ISA*00*...IEA*1*000000002~")
ResXml text Text containing the XML format of the EDI 278 transaction response. (e.g., "
...
...")
deleteFlag tinyint A tiny integer indicating if the record is deleted (1) or not (0). (e.g., 0, 1)
timestamp datetime Datetime value representing the date and time when the transaction was created. (e.g., '2021-08-25 14:35:12')
TypeOfRequest varchar A short string indicating the type of request for the transaction. (e.g., 'Auth', 'Ref')
Encounterid int Foreign key referring to the id of the related encounter. (e.g., 201, 202, 203)
Refid int Foreign key referring to the id of the related reference. (e.g., 301, 302, 303)
Userid int Foreign key referring to the id of the user who initiated the transaction. (e.g., 401, 402, 403)
STATUS int Integer value representing the status of the transaction. (e.g., 0 for pending, 1 for approved, 2 for denied)
Patientid int Foreign key referring to the id of the related patient. (e.g., 501, 502, 503)
Scheduled int Integer value representing whether the transaction is scheduled (1) or not (0). (e.g., 0, 1)
