Table hi_payor_mtm_medications
Description This table stores information about medications associated with a specific MTM (Medication Therapy Management) record.
Column DataType Description
id (Primary Key) int Unique identifier for each medication record. (Example: 1, 2, 3)
mtm_id int identifier for the MTM record that the medication is associated with. Foreign key relation to `hi_payor_mtm` table's `id` column. (Example: 101, 102, 103)
name varchar Name of the medication. (Example: "Aspirin", "Lisinopril", "Metformin")
medication_condition varchar The medical condition that the medication is prescribed for. (Example: "Headache", "High blood pressure", "Type 2 diabetes")
prescriber varchar Name of the healthcare professional who prescribed the medication. (Example: "Dr. John Smith", "Dr. Jane Doe", "Dr. Alice Johnson")
listdate datetime Date when the medication was added to the list. (Example: "2021-01-01", "2021-02-15", "2021-03-30")
deleteflag bit A flag indicating if the medication record has been deleted. '0' means not deleted, and '1' means deleted. (Example: 0, 1)
