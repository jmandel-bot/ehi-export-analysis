Table bh_treatment_plan
Description This table stores information about treatment plans for patients, including details about the plan's start and end dates, status, type, and associated program.
Column DataType Description
id (Primary Key) int Unique identifier for each treatment plan (e.g., 1, 2, 3).
patientid int identifier for the patient associated with the treatment plan, foreign key relation to `users` table, `UID` column (e.g., 101, 102, 103).
programid int identifier for the program associated with the treatment plan, foreign key relation to `acoprograms` table, `id` column (e.g., 1, 2, 3).
ptAcoid int identifier for the patient's ACO program, foreign key relation to `ptacoprograms` table, `id` column (e.g., 10, 20, 30).
tpStartDate datetime Start date and time of the treatment plan (e.g., '2022-01-01 00:00:00').
tpEndDate datetime End date and time of the treatment plan (e.g., '2022-01-31 00:00:00').
tpStatus tinyint Status of the treatment plan as a tiny integer (e.g., 0 for inactive, 1 for active).
tpType int identifier for the type of treatment plan, foreign key relation to `bh_treatment_plan_type` table, `id` column (e.g., 1 for inpatient, 2 for outpatient).
isLocked tinyint Indicates if the treatment plan is locked (1) or unlocked (0).
isLockedBy int identifier for the user who locked the treatment plan, foreign key relation to `users` table, `UID` column (e.g., 1001, 1002, 1003).
tpEndedBy int identifier for the user who ended the treatment plan, foreign key relation to `users` table, `UID` column (e.g., 1001, 1002, 1003).
unlockedBy int identifier for the user who unlocked the treatment plan, foreign key relation to `users` table, `UID` column (e.g., 1001, 1002, 1003).
lockedOn datetime Date and time when the treatment plan was locked (e.g., '2022-01-15 12:00:00').
unlockedOn datetime Date and time when the treatment plan was unlocked (e.g., '2022-01-20 12:00:00').
delflag tinyint Indicates if the treatment plan is deleted (1) or not deleted (0).
addedBy int identifier for the user who added the treatment plan, foreign key relation to `users` table, `UID` column (e.g., 1001, 1002, 1003).
addedDtTime datetime Date and time when the treatment plan was added (e.g., '2022-01-01 00:00:00').
modifiedBy int identifier for the user who modified the treatment plan, foreign key relation to `users` table, `UID` column (e.g., 1001, 1002, 1003).
modifiedDtTime datetime Date and time when the treatment plan was modified (e.g., '2022-01-10 12:00:00').
discharge_plan_notes varchar Text containing notes about the patient's discharge plan (e.g., "Patient needs follow-up appointment with primary care physician.").
isRecurringTp tinyint Indicates if the treatment plan is recurring (1) or not recurring (0).
dischargeNoteDateTime datetime Date and time when the discharge note was created (e.g., '2022-01-31 12:00:00').
