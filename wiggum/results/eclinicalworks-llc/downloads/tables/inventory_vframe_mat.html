Table inventory_vframe_mat
Description This table stores information about various materials used in inventory frames, including their names, descriptions, and precious metal content.
Column DataType Description
id (Primary Key) int A unique identifier for each material record. For example: 1, 2, 3, etc.
NAME varchar The name of the material. Example: 'Gold', 'Silver', 'Platinum', etc.
deleteFlag smallint An indicator of whether the material record is deleted (1) or not (0).
description varchar A brief description of the material. For example: 'Yellow gold 18k', 'Sterling silver 925', etc.
preciousMetal varchar The type of precious metal used in the material. For example: 'Gold', 'Silver', 'Platinum', etc.
preciusMetalDesc varchar A description of the precious metal content in the material. For example: '18k gold', '925 silver', etc.
matid int A foreign key that refers to the id column in another table (not specified in the provided SQL) that contains additional information about the material.
createdBy int A foreign key that refers to the id column in a users table (not specified in the provided SQL) that stores information about the user who created the material record.
createdOn datetime The date and time when the material record was created. For example: '2021-09-01 12:34:56'.
