EHI Export Schema
Table asc_ioitems
Description This table is used to store a master list of intake outputs.
Column DataType Description
itemid (Primary Key) int This is the primary key and an auto incrementing integer that uniquely identifies each item in the table. Example: 1, 2, 3, etc.
itemName varchar This is a unique varchar column that stores the name of the item. It cannot be NULL and has a default value of an empty string. Example: Item A, Item B, Item C, etc.
description varchar This is a varchar column that stores a brief description of the item. It cannot be NULL and has a default value of an empty string. Example: Item A is used for..., Item B is a component of..., etc.
itemType smallint This is a smallint column that stores the type of the item. It can be NULL. Example: 1 (Raw Material), 2 (Finished Product), 3 (Semi-Finished Product), etc.
unit varchar This column is not in use.
deleteFlag smallint This is a smallint column that indicates whether the item has been deleted or not. It cannot be NULL and has a default value of 0. A value of 0 means the item is not deleted, and a value of 1 means the item is deleted.
isInactive smallint This is a smallint column that indicates whether the item is inactive or not. It cannot be NULL and has a default value of 0. A value of 0 means the item is active, and a value of 1 means the item is inactive.
