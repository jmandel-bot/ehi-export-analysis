Table student_immunizations
Description This table stores information about the immunization status of students.
Column DataType Description
Student_Immid (Primary Key) int This column is auto-generated unique primary key.
Patientid int identifier for the student, which can be used to link to other tables containing student information. This can be a foreign key referring to the students table, for example. Example: 101, 102, 103.
Imm_Mumps tinyint Indicates whether the student has been immunized against mumps (1) or not (0). Example: 1, 0.
Imm_Measles tinyint Indicates whether the student has been immunized against measles (1) or not (0). Example: 1, 0.
Imm_Rubella tinyint Indicates whether the student has been immunized against rubella (1) or not (0). Example: 1, 0.
Imm_DT tinyint Indicates whether the student has been immunized against diphtheria and tetanus (1) or not (0). Example: 1, 0.
Imm_Meningococcal tinyint Indicates whether the student has been immunized against meningococcal disease (1) or not (0). Example: 1, 0.
Imm_HepB tinyint Indicates whether the student has been immunized against hepatitis B (1) or not (0). Example: 1, 0.
Imm_Varicella tinyint Indicates whether the student has been immunized against varicella (chicken pox) (1) or not (0). Example: 1, 0.
Imm_PPD tinyint Indicates whether the student has been tested for tuberculosis using the purified protein derivative (PPD) test (1) or not (0). Example: 1, 0.
ImmunizationClearance tinyint Indicates whether the student has received clearance for all required immunizations (1) or not (0). Example: 1, 0.
ClinicalClearance tinyint Indicates whether the student has received clearance for clinical practice (1) or not (0). Example: 1, 0.
Imm_HepBTitre tinyint Indicates whether the student has a positive hepatitis B antibody titre (1) or not (0). Example: 1, 0.
