Table derm_observationdetails
Description This table stores graphical and documented information on each observation marked on body regions for selected lesion.
Column DataType Description
observationdetailid (Primary Key) int This is auto-incremented primary key in This column . This is unique for each annotation marked on body regions.
observationid int This is unique for dermatology examinaton documented for each encounter. This refers to column observationid from table derm_observations.
lesionid int This column stores lesion id of lesion for which annotation is marked on body. It is a foreign key refers to column lesionid of table derm_lesionlist.
regionid int This column stores region id of body location on which annotation is marked. It is foreign key refers to column regionid from table derm_diagramregions.
height float Height of the lesion in the specified units. (e.g., 1.5, 2.3, 4.1)
width float Width of the lesion in the specified units. (e.g., 1.2, 2.4, 3.6)
color varchar Color of the lesion. (e.g., "red", "brown", "black")
units float This column is not in use.
counts int This column stores total count of marks comes under single observation.
notes text Additional textual notes about the observation detail. (e.g., "Slightly raised", "Irregular border")
lesionruleoutid tinyint This column is not in use.
observelesion tinyint This column is not in use.
isgroup tinyint Flag indicating whether this observation detail is part of a group (1) or not (0). (e.g., 0, 1)
isdeleted tinyint Flag indicating whether this observation detail has been deleted (1) or not (0). (e.g., 0, 1)
createdby int identifier for the user who created this observation detail. (e.g., 1001, 1002, 1003)
createdon datetime Date and time when this observation detail was created. (e.g., "2022-01-01 12:34:56")
modifiedby int identifier for the user who last modified this observation detail. (e.g., 1001, 1002, 1003)
modifiedon datetime Date and time when this observation detail was last modified. (e.g., "2022-01-02 12:34:56")
x2d float This column is not in use.
y2d float This column is not in use.
regionname text This column stored region name on which observation/annotation is marked. This stored region name corresponding to column regionid in This column .
annotationid int This column stored unique number assigned to annotation marked on biodigital image of body for this observation.
displayindex int This column stores order in which observations are listed in the UI.
worldPos varchar This column stores coordinates of annotation on biodigital image. It's value is used within data_3d column from table derm_observations.
objectid varchar This column stores region id of location marked on biodigital image. It's value is used within data_3d column from table derm_observations.
copiedfrom int This column stores observation id from where this record is copied from past encounter. This column refers to observationdetailid from This column .
encounterid int This column stores encounter id and refers to column encounterid from table enc.
serialno varchar This column stores order in which observations are displayed on screen. It also stores order of items within group observations.
camera text This column stores coordinates of camera position to capture biodigital image along with annotations. For example, head view, torso view, side view, Foot view.
lesiondescription text This column stores lesion description for this observation. It could be from table derm_lesiondescription or modified from observation documentation screen.
colorname varchar This column stores color of lesion on patient's skin.
largeRegion tinyint Flag indicating whether the lesion is in a large region (1) or not (0). (e.g., 0, 1)
