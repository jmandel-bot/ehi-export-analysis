EHI Export Schema
Table todo_list_checklist
Description This table stores the checklist items associated with a specific To Do List.
Column DataType Description
todo_list_id (Primary Key) int This is an integer column that stores the unique identifier for a todo list. It is the primary key for This column and is a foreign key that refers to the id column in the todo_list table. Example: 1, 2, 3.
phmMeasureid varchar This is a varchar(30) column that stores the unique identifier for a specific measure within the checklist. Example: PHM001, PHM002, PHM003.
measureCode varchar This is a varchar(30) column that stores the code associated with the specific measure in the checklist. Example: M001, M002, M003.
