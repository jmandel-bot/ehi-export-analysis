EHI Export Schema
Table imagingstudies
Description This table stores information about imaging studies related to medical encounters.
Column DataType Description
id (Primary Key) int This column is a Primary Key and auto-incremented.
encounterid int This column stores id of the encounter for which Diagnostic Study / Test is done. This column refers to column encounterid from enc table.
notes text This is a text column that stores any notes or comments related to the imaging study. Example: "Patient has a fracture in the left femur."
notesHtml varchar This is a text column that stores the notes or comments related to the imaging study in HTML format. Example: "

Patient has a fracture in the left femur.

"
