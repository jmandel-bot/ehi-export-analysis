EHI Export Schema
Table edi_837contractinfo
Description This table store about Claim level Contract Info
Column DataType Description
Id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
contractcode varchar A 4-character code representing the contract. Example: 'A123', 'B456'.
MonetaryAmt varchar A monetary amount associated with the contract, stored as a string. Example: '1000.00', '500.50'.
DecFormatPercent varchar A decimal formatted percentage value, stored as a string. Example: '0.10', '0.25'.
RefIdentification varchar A reference identification number for the contract, stored as a string. Example: 'REF12345', 'REF67890'.
TermsDiscPercent varchar A decimal formatted percentage value representing the terms discount percentage, stored as a string. Example: '0.05', '0.15'.
VersionIdentifier varchar A version identifier for the contract, stored as a string. Example: 'V1.0', 'V2.0'.
ClaimorCptFlag tinyint A boolean flag (0 or 1) indicating whether the record is for a claim (0) or a CPT (1). Example: 0, 1.
InvId int An integer representing the foreign key relation to the `Inv` table's `Id` column. Example: 10, 20.
InvCptId int An integer representing the foreign key relation to the `InvCpt` table's `Id` column. Example: 30, 40.
deleteFlag tinyint A boolean flag (0 or 1) indicating whether the record is deleted (1) or not (0). Example: 0, 1.
