EHI Export Schema
Table ecwstructoptions
Description This tables stores options to structured questions stored in the ecwstructitems table.
Column DataType Description
ecwOptionid (Primary Key) int This is the primary key for the table, an integer that uniquely identifies each option record. It is auto-incremented. Example: 1, 2, 3.
ecwStructid int This column stores the ecw structured question id for which the option belongs to. Foreign key relation : ecwstructid column of ecwstructitems table.
deleteFlag tinyint This column stores values 0/1 to indicate whether the option is marked as deleted or not. 0 - Option is not marked as deleted, 1 - Option is marked as deleted.
displayIndex smallint This column stores a number to indicate in which order option should be listed on User Interface.
name varchar This is a varchar(255) column that stores the name of the option. Example: 'Option A', 'Option B', 'Option C'.
isCDSS tinyint This column stores value 0 or 1 which states whether the structured option is also used in CDSS alerts. Value 0 states structured option is not used in CDSS alert where as Value 1 states structured option is used in CDSS alert .
diagnosisid int This is an optional integer column that stores the id of the related diagnosis, if any. Example: 5, 15, 20.
