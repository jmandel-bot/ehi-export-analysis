Table case_insurances
Description This table stores information about insurance cases, including their associated case and insurance IDs, display index, and insurance details ID.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer value. It uniquely identifies each record in the `case_insurances` table. Example: 1, 2, 3, etc.
Caseid int This is a foreign key that refers to the `id` column in another table (e.g., `cases` table) and represents the case associated with the insurance record. It is an integer value and is set to 0 by default. Example: 101, 102, 103, etc.
Insid int This column stores the id of the insurance patient has. This column refers to column insid from insurance table.
displayIndex int This column stores a digit to indicate order in which insurance records should be listed in user interface.
insurancedetid int This column stores the id of the insurance detail of the patient's insurance. This column refers to column id from insurancedetail table.
