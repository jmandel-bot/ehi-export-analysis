EHI Export Schema
Table interactionnotes
Description This table stores Notes entered in the Drug Interaction window by the user.
Column DataType Description
Id (Primary Key) int This is auto-incremented primary key in This column .
EncounterId int The column stores an id of the encounter of a patient.
Notes text This column contains Notes in free text form
ProviderAction varchar This column stores Provider action. Possible values are Cancel, Created, Ignore, Override, Viewed
patient_id int This column stores the patient id. This corresponds to userid of user table.
erx_req_id int If drug interaction is loaded from refill or change request, the request id is logged in this column
