Table edi_837k3_claimlevel
Description This table store about Claim Level Additional Data invoice order with display order and other related Info
Column DataType Description
Id (Primary Key) int This is the primary key for the table and is an integer that uniquely identifies each record in the table. Example: 1, 2, 3, etc.
InvId int This is a foreign key that refers to the invoice ID in another table. It is an integer that represents the ID of the invoice associated with the claim level. Example: 101, 102, 103, etc. Foreign key relation: `invoiceTable.invoiceId`
DisplayOrder int This is an integer that represents the display order of the claim level in the user interface. Example: 1, 2, 3, etc.
k3Notes varchar This is a varchar(255) column that stores any notes or comments related to the claim level. Example: "Additional information about the claim", "Follow up needed", etc.
deleteFlag tinyint This is a tinyint(1) column that indicates whether the record has been deleted or not. A value of 0 means the record is not deleted, and a value of 1 means the record is deleted. Example: 0, 1.
