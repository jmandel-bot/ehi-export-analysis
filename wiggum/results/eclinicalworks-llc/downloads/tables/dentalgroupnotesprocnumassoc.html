Table dentalgroupnotesprocnumassoc
Description This table establishes the relationship between a group note and associated procedures. Each group note ID can be linked to two or more procNums, and there by, linking procedures to the group note. A group note is a single note that applies to multiple procedures.
Column DataType Description
groupNoteid bigint This column is a reference to ProcNum column of the table procedurehistory. It identifies an entry in procedurehistory that's flagged as a group note.
procNum bigint This column is a reference to ProcNum column of the table procedurehistory. It identifies an entry in procedurehistory that's associated to the group note identified by the column groupNoteid. For each groupNoteid, there are two or more entries for procNum. This links the group note to the procedures.
