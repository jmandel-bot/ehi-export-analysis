EHI Export Schema
Table questaoe
Description This table stores Ask at Order Entry (AOE) questions associated with a lab or diagnostic imaging order for a lab company.
Column DataType Description
Qsid (Primary Key) int Unique identifier for each question with an integer value. For example: 1, 2, 3.
OBRCode varchar This column stores a lab code associated with the AOE question. This column refers to the column code from the labcodes table.
OBXCode varchar This column stores a question (observation or result) code associated with the AOE question added by the end user.
Question varchar The text of the question stored as a varchar(255). For example: What is your favorite color?, How old are you?.
Questip varchar Additional information or tips related to the question stored as a varchar(255). For example: Choose one color, Enter your age in years.
ABN char A flag indicating if the question is related to an abnormal result stored as a char(1). For example: Y for yes, N for no.
labid int This column stores the lab id of the company selected for the lab order associated with this AOE question.
FieldType varchar An enumeration representing the type of input field for the question, with possible values FT (Free Text), DD (Drop-down), ST (Single Text), RB (Radio Button), and CB (Check Box). For example: FT, DD.
Required tinyint This column stores a value 0 or 1. 1 - when this question is marked as mandatory, 0- otherwise.
itemid int An integer identifier for the item associated with the question, with a default value of 0. For example: 1, 2, 3.
MaxChar int The maximum number of characters allowed for the answer stored as an int(2) with a default value of 0. For example: 50, 100.
defaultvalue varchar The default value for the answer stored as a varchar(255) with a default value of an empty string. For example: N/A, 18.
deleteflag tinyint This column stores a flag 0 or 1. 1 - when this AOE question is marked as deleted. 0 - otherwise.
groupid int This column stores the group id to which the answer options are associated to for this AOE question.
inactive tinyint This column stores the flag value where 0 means active record and 1 means inactive record.
