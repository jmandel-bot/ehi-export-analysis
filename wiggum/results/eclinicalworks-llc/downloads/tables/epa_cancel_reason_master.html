EHI Export Schema
Table epa_cancel_reason_master
Description This table stores the cancellation reasons for a specific vendor along with their respective codes.
Column DataType Description
id (Primary Key) smallint Unique identifier for each cancellation reason entry. It is a small integer with a maximum value of 65535 and is auto-incremented. Example: 1, 2, 3, etc.
reason_code varchar A short code representing the cancellation reason. It is a string with a maximum length of 10 characters. Example: "RC001", "RC002", "RC003", etc.
reason_name varchar A detailed description of the cancellation reason. It is a string with a maximum length of 500 characters. Example: "Customer requested cancellation", "Product out of stock", "Shipping delay", etc.
vendor_id tinyint Identifier for the vendor associated with the cancellation reason. It is a tiny integer with a maximum value of 255 and a default value of 0. Example: 1, 2, 3, etc. This column can be a foreign key referring to the `id` column in a `vendors` table.
delete_flag tinyint A flag indicating whether the cancellation reason entry has been deleted or not. It is a tiny integer with a value of either 0 (not deleted) or 1 (deleted). Default value is 0. Example: 0, 1.
