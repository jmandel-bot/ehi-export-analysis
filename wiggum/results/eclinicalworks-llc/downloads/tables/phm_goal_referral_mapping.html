Table phm_goal_referral_mapping
Description This table stores the mapping between goals and referrals, along with the associated encounter and problem encounter information.
Column DataType Description
id (Primary Key) int Unique identifier for each mapping record. Example: 1, 2, 3.
encId int This column stores careplan and Patient Problems encounterId. Column refers to encounterid column in enc table.
problemEncId int This column stores careplan and Patient Problems encounterId. Column refers to encounterid column in enc table. This can be NULL if there is no specific problem encounter
goalId int This is a foreign key that refers to the `Id` column in the `structdatadetail` table.
referralId int This is a foreign key that refers to the `ReferralId` column in the `referral` table.
deleteflag tinyint A tinyint flag indicating if the record is deleted or not. Default value is '0'. Example: 0 (not deleted), 1 (deleted).
