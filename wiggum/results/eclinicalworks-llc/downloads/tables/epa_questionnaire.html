EHI Export Schema
Table epa_questionnaire
Description This table stores information about questionnaires related to EPA transactions, including their content, status, and deadlines.
Column DataType Description
id (Primary Key) int Unique identifier for each questionnaire (e.g., 1, 2, 3).
epa_transaction_id int Identifier for the related EPA transaction; foreign key relation to `epa_transaction` table's `id` column.
epa_case_id varchar Unique identifier for the EPA case (e.g., "EPA12345").
pa_reference_id varchar Unique identifier for the PA reference (e.g., "PA12345").
epa_message_id varchar Unique identifier for the EPA message (e.g., "MSG12345").
question_set_id varchar Unique identifier for the question set (e.g., "QS12345").
question_set_no tinyint Number representing the order of the question set (e.g., 1, 2, 3).
question_set_title varchar Title of the question set (e.g., "Environmental Impact Assessment").
question_set_desc varchar Description of the question set (e.g., "This question set covers the environmental impact assessment of the project.").
content varchar Content of the questionnaire in medium text format.
deadline_for_reply datetime Deadline for replying to the questionnaire in datetime format (e.g., "2022-12-31 23:59:59").
questionnaire_status varchar Status of the questionnaire (e.g., "Pending", "Completed", "Expired").
created_by int Identifier for the user who created the questionnaire; foreign key relation to `users` table's `uid` column.
created_datetime datetime Datetime when the questionnaire was created (e.g., "2022-01-01 12:34:56").
updated_by int Identifier for the user who last updated the questionnaire; foreign key relation to `users` table's `uid` column.
updated_datetime datetime Datetime when the questionnaire was last updated (e.g., "2022-01-02 12:34:56").
pa_note varchar Notes from the PA regarding the questionnaire (e.g., "Please provide additional information on the environmental impact.").
is_active tinyint Flag indicating if the questionnaire is active (1) or not (0).
delete_flag tinyint Flag indicating if the questionnaire is deleted (1) or not (0).
