EHI Export Schema
Table prisma_device_details
Description This table stores information about various devices, their details.
Column DataType Description
id (Primary Key) bigint Unique identifier for each device detail entry (bigint). Example: 1, 2, 3.
externalDeviceId varchar Unique identifier for the external device (varchar). Example: "device123", "device456".
sectionId bigint Identifier for the section the device belongs to (bigint). Foreign key relation with `prisma_ccd_document_section` table's `id` column. Example: 101, 102, 103.
sourceId int Identifier for the source of the device information (int). Foreign key relation with `fhir_source_table` table's `id` column. Example: 1, 2, 3.
deviceStatus varchar Status of the device (varchar). Example: "active", "inactive", "unknown".
lotNumber varchar Lot number of the device (varchar). Example: "L123", "L456".
serialNumber varchar Serial number of the device (varchar). Example: "S123", "S456".
manufacturer varchar Manufacturer of the device (varchar).
modelNumber varchar Model number of the device (varchar).
manufacturerDate datetime Date when the device was manufactured (datetime). Example: "2021-01-01 00:00:00", "2020-12-31 00:00:00".
expirationDate datetime Expiration date of the device (datetime). Example: "2025-01-01 00:00:00", "2024-12-31 00:00:00".
distinctIdentifier varchar Distinct identifier for the device (varchar). Example: "DID123", "DID456".
organizationId int Identifier for the organization the device belongs to (int). Foreign key relation with `prisma_organization_details` table's `ID` column. Example: 1, 2, 3.
deviceVersion varchar Version of the device (varchar). Example: "1.0", "2.0".
locationId int Identifier for the location of the device (int). Foreign key relation with `prisma_location_details` table's `ID` column. Example: 1, 2, 3.
typeCode varchar Type code of the device (varchar). Example: "TC123", "TC456".
typeDisplay varchar Display text for the device type (varchar).
typeSystem varchar System for the device type (varchar).
typeText varchar Text description of the device type (varchar).
udiCarrierDeviceIdentifier varchar Unique device identifier carrier device identifier (varchar). Example: "UDI123", "UDI456".
udiCarrierHRF varchar Unique device identifier carrier human-readable format (varchar). Example: "UDI-HRF123", "UDI-HRF456".
udiCarrierEntryType varchar Unique device identifier carrier entry type (varchar). Example: "manual", "barcode".
isException tinyint Indicates if there is an exception (tinyint). Example: 0 (false), 1 (true).
exceptionMessage varchar Message describing the exception, if any. Empty if there is no exception.
sourcePriority int Priority of the source (int). Example: 1, 2, 3.
codeSystemPriority int Priority of the code system (int). Example: 1, 2, 3.
duplicatedBy int Identifier for the duplicated entry (int). Example: -1 (not duplicated), 1, 2, 3.
