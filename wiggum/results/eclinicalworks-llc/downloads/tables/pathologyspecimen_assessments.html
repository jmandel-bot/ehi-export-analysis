Table pathologyspecimen_assessments
Description This table stores information about assessments related to pathology specimens, including their association with specific reports and whether they have been deleted or not.
Column DataType Description
id (Primary Key) int This is the primary key for the table, which is an integer that uniquely identifies each pathology specimen assessment. Example values: 1, 2, 3, etc.
assessmentid int This column stores primary assessment id.
reportid int This column stores lab report id of labdata table.
deleteflag tinyint This is a tinyint(1) column that indicates whether the assessment has been deleted or not. A value of 0 means the assessment is not deleted, while a value of 1 means it is deleted. Example values: 0, 1.
