EHI Export Schema
Table ccmr_nfl_injury_notes
Description This table stores information about injury notes related to NFL players, including details about the injury, user who added the note, and any modifications. Notes can be enter from Injury Tab, Rehab Tab, Modalities Tab, Notes Tab of NFL injury screen.
Column DataType Description
id (Primary Key) int Auto-increment primary key in This column .
Injuryid int This column defined the injuryid for which note is created, it is reference of ccmr_atrisk table
Userid int This column will save userid who have saved notes. This column refers to UID column in user table.
UserName varchar This column will save Username who have saved notes. This column refers to uname column in user table.
Notes text This field will save actual notes entered by provider.
ModifiedTime datetime This column will save time when note was entered/modified.
soarNote int This column will save single digit integer value. Possible values are 0 and 1. 0 means Note is saved from Injury Initial Impression Field and 1 means Notes is saved from other places like Rehab Tab, Modalities Tab, Notes Tab
deleteflag int This column defines that note is deleted or not. It will have 2 possible values, 0 and 1. 0 means note is not deleted. 1 means note is deleted.
ccmrid int This field will save ccmrid of user who has saved note. each patient has one ccmrid so it is used to identify patients.
visitid int This column will save visitid which defined the visit of particular injury. Injury can have multiple visits so each visits can be separated using visitid and ccmrid
Providerid int Not in use
