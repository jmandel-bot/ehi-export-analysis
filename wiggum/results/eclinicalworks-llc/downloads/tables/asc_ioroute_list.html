EHI Export Schema
Table asc_ioroute_list
Description This table is used to store a master list of IO routes.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key of this column.
ioRoute varchar This column stores name of Input/Output route.
deleteFlag smallint This is a smallint(1) column that indicates whether the I/O route has been deleted or not. A value of 0 means the route is active, and a value of 1 means the route has been deleted. Example: 0, 1.
