EHI Export Schema
Table reminderdata
Description This table stores the suppression status of Generic, Rx, & Dx Alerts for patients.
Column DataType Description
patientid int This column stores patient's id for whom alert is suppressed. This column refers column UID from user table.
itemid int This column stores an id of the item associated with the alert. This column refers column itemid from items table.
suppress smallint This column stores a value 0 or 1. 1 - when the alert is suppressed, 0 - Otherwise.
dueDate datetime This column stores date which indicates the alert is suppressed up to. The format used is YYYY-MM-DD HH:MM:SS.
notes text This column stores the additional notes (free text) documented by the end user.
alert_status varchar This column is not used anymore.
