EHI Export Schema
Table careplan_encounters
Description This table stores the relationship between care plans and encounters, linking each care plan to its corresponding encounter.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer value that uniquely identifies each row in the table. Example: 1, 2, 3, etc.
CarePlanid int This is a foreign key that refers to the `id` column in the `careplans` table. It represents the unique identifier of a care plan associated with an encounter. Example: 101, 102, 103, etc.
Encounterid int This is a foreign key that refers to the `id` column in the `encounters` table. It represents the unique identifier of an encounter associated with a care plan. Example: 201, 202, 203, etc.
