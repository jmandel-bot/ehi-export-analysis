EHI Export Schema
Table hl7labnotes
Description This table stores the attribute or test level notes (OBR and OBX notes) coming from the hl7 message.
Column DataType Description
reportid int This column stores id of a Lab/DI or procedure ordered for a patient. This column refers to Reportid column in labdata table.
notes text This column stores the notes that are received in the nte segments. This column stores string values received from lab interface. Hence, this column is declared as TEXT as the length of the notes coming over interface can not be determined.
serialNo int This column stores the sequence in which the notes are received for each test and attribute level. The first note will have serial number 1 and the following will have sequence number incremented by 1.
obrcode varchar This column is not used anymore.
hl7id varchar This column Stores the attribute id to distinguish the notes for each attribute. For test level notes this column is blank.
corrreportno int This column stores the number of times corrected report came for a specific attribute.
hl7itemid int This stores the itemid for the lab attribute. It refers to itemid column in items table.
hl7SrNo int This column stores the serial number of the attribute. This column refers to serialNo column in hl7labdatadetail table.
