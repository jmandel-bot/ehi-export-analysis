EHI Export Schema
Table nhxp2precordstatus
Description This table stores the information of transmission status of referral / patient record / P2P message. it also stores ccr parse info for record
Column DataType Description
id (Primary Key) int This column used to strore primary key of table
refid int This column used to strore referral id or patient record id for each transaction done through P2P
msgTxid int This column stores message transaction id received from P2P against referral / patient record
STATUS tinyint This column stores status for each transaction send to P2P
RecordType tinyint This column stores record type it can have valueset as 0 or 1
toPrName varchar This column stores to provider name whom referral / record is being sent
modifiedDate datetime This column stores time stamp of when transaction was sent / updated
recipientReferralStatus int This columns stores the status of recipient against the transaction
ccrparsedtags varchar This column used to store the name of CCR parsed tags on P2P
ccrparsedtagvalues varchar This column used to store the name of CCR parsed values on P2P
ccrparsedflag tinyint This column used to store the flag whether CCR was parsed on P2P or not
ccrparsedmiscinfo varchar This column used to store the parsed info in json format
txsenttimestamp datetime This column used to store the transaction sent time stamo
synccount tinyint This column used to store the transaction sync count
PAMSSyncCount tinyint This column used to store the PAMS sync count
