Table referral_addendum_reason
Description This table stores the reasons for addendums in referrals, linking a referral to a specific reason for the addendum.
Column DataType Description
id (Primary Key) int This is the primary key for the table, which is an auto-incrementing integer value. It uniquely identifies each record in the table. Example: 1, 2, 3, etc.
referralid int This is a foreign key column that refers to the `referral` table's `Referralid` column. It stores the id of the referral associated with the addendum reason. Example: 101, 102, 103, etc.
reasonid int This is a foreign key column that refers to the `structured_addendum_reason` table's `reasonid` column. It stores the id of the reason for the addendum in the referral. Example: 1, 2, 3, etc.
