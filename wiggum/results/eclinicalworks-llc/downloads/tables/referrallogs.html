EHI Export Schema
Table referrallogs
Description This table stores information about referral logs, including details about the sender, receiver, patient, and referral status.
Column DataType Description
LogID (Primary Key) int Unique identifier for each referral log entry. (Example: 1, 2, 3)
LoginID int Identifier for the user who initiated the referral. (Example: 101, 102, 103)
DoctorID int Identifier for the doctor associated with the referral. (Example: 1001, 1002, 1003)
PatientID int Identifier for the patient associated with the referral. (Example: 2001, 2002, 2003)
ToID int Identifier for the recipient of the referral. (Example: 3001, 3002, 3003)
ToName varchar Name of the recipient of the referral. (Example: "Dr. John Doe", "Dr. Jane Smith")
ToEmail text Email address of the recipient of the referral, stored as a blob. (Example: "johndoe@example.com", "janesmith@example.com")
ToCompany varchar Company name of the recipient of the referral. (Example: "ABC Hospital", "XYZ Clinic")
EncounterID int Identifier for the patient encounter associated with the referral. (Example: 4001, 4002, 4003)
EmailInitTime varchar time stamp of when the referral email was initiated. (Example: "2022-01-01 12:34:56", "2022-01-02 14:56:23")
Retry int Number of retries for sending the referral email. (Example: 0, 1, 2)
Status int Status code for the referral. (Example: 0, 1, 2)
EmailData text Text data of the referral email. (Example: "Dear Dr. Doe, Please find the attached referral...", "Hello Dr. Smith, I am referring a patient to you...")
Subject varchar Subject of the referral email. (Example: "Referral", "Urgent Referral")
EmailStatus varchar Status of the referral email, with possible values: Logged, Completed, Failed, Pending, Query, Received, Sending, Deleted. (Example: "Logged", "Completed")
SendDate datetime Date and time when the referral email was sent. (Example: "2022-01-01 12:35:00", "2022-01-02 14:57:00")
EmailType varchar Type of the referral email, with possible values: Prescription, ProgressNotes, ConsultNotes, LabReport, LabOrder, TiffDocument. (Example: "Prescription", "LabReport")
EmailManLogID int Identifier for the email management log associated with the referral. (Example: 5001, 5002, 5003)
EmailPrefix varchar Prefix for the referral email. (Example: "REF", "URG")
EmailFile varchar File name of the attachment in the referral email. (Example: "referral_123.pdf", "lab_report_456.pdf")
EmailTransaction varchar Transaction identifier for the referral email. (Example: "TX123", "TX456")
RxIds text Text containing prescription identifiers associated with the referral. (Example: "1001,1002", "1003,1004")
EmailLogStatus varchar Status of the email log, with possible values: Logged, Reviewed. (Example: "Logged", "Reviewed")
EmailLogBatch int Batch number for the email log. (Example: 1, 2, 3)
FacilityID int Identifier for the facility associated with the referral. (Example: 6001, 6002, 6003)
FacilityName varchar Name of the facility associated with the referral. (Example: "Main Hospital", "Downtown Clinic")
WebSync int Flag indicating whether the referral data has been synchronized with the web. (Example: 0, 1)
DataSent int Flag indicating whether the referral data has been sent. (Example: 0, 1)
RndNumber varchar Random number associated with the referral. (Example: "100", "200")
FromID int Identifier for the sender of the referral. (Example: 7001, 7002, 7003)
FromName varchar Name of the sender of the referral. (Example: "Dr. Alice Johnson", "Dr. Bob Williams")
FromEmpiID int Identifier for the sender's EMPI (Enterprise Master Patient Index). (Example: 8001, 8002, 8003)
PatientEmrID int Identifier for the patient's EMR (Electronic Medical Record). (Example: 9001, 9002, 9003)
PatientFname varchar First name of the patient. (Example: "John", "Jane")
PatientMname varchar Middle name of the patient. (Example: "A.", "B.")
PatientLname varchar Last name of the patient. (Example: "Doe", "Smith")
PatientDOB datetime Date of birth of the patient. (Example: "1990-01-01", "1985-12-31")
PatientAddress varchar Address of the patient. (Example: "123 Main St", "456 Elm St")
PatientCity varchar City of the patient's address. (Example: "New York", "Los Angeles")
PatientState varchar State of the patient's address. (Example: "NY", "CA")
PatientPhone varchar Phone number of the patient. (Example: "555-123-4567", "555-987-6543")
PatientZipcode varchar Zip code of the patient's address. (Example: "10001", "90001")
PatientEmail varchar Email address of the patient. (Example: "johndoe@example.com", "janesmith@example.com")
ToPracticeCode varchar Practice code of the recipient of the referral. (Example: "PRACT01", "PRACT02")
FromPracticeName varchar Name of the sender's practice. (Example: "ABC Medical Group", "XYZ Family Practice")
FromPracticeCode varchar Practice code of the sender of the referral. (Example: "PRACT03", "PRACT04")
AssociatedReferralID int Identifier for the associated referral. (Example: 10001, 10002, 10003)
RefType varchar Type of the referral, with possible values: InReferral, InResponse, OutReferral, OutResponse. (Example: "InReferral", "OutReferral")
AdditionalInfo text Additional information about the referral, stored as a blob.
ReferralStatus varchar Status of the referral, with possible values: Addressed, Open, Received, Responded, In-Process. (Example: "Addressed", "Open")
EmpiReferralInfoId int Identifier for the EMPI referral information. (Example: 11001, 11002, 11003)
refThreadId varchar Thread identifier for the referral. (Example: "REF123", "REF456")
AssignedToId int Identifier for the user assigned to handle the referral. (Example: 12001, 12002, 12003)
delFlag tinyint Flag indicating whether the referral log entry has been deleted. (Example: 0, 1)
timestamp datetime time stamp of when the referral log entry was created. (Example: "2022-01-01 12:34:56", "2022-01-02 14:56:23")
ReadTime varchar time stamp of when the referral log entry was read. (Example: "2022-01-01 12:35:00", "2022-01-02 14:57:00")
Priority tinyint Priority level of the referral. (Example: 0, 1)
ccr text Continuity of Care Record (CCR) data, stored as mediumtext.
