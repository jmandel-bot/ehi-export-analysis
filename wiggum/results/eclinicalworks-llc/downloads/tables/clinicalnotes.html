EHI Export Schema
Table clinicalnotes
Description This table stores clinical notes created by healthcare providers for their patients.
Column DataType Description
id (Primary Key) int This column stores an id of clinical notes. This is an auto incremental primary key of this table and is not expected to have NULL in it.
patientid int This column stores provider id. Column is not expected to have NULL in it.
providerId int This column stores Patient id. Column is not expected to have NULL in it.
notes varchar This column stores notes for the clinical notes. Column has been declared as VARCHAR because this column may store notes value up to 3000 character length and is not expected to have NULL in it. By default, it will be empty.
createdtimestamp varchar This column stores Timestamp of the note created. Column has been declared as VARCHAR because this column may store timestamp value up to 50 character length and is not expected to have NULL in it. By default, it will be empty.
