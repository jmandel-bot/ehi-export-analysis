Table audioworkflowar
Description This table stores information related to audio workflow, including various measurements and parameters for both right and left ears.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table (e.g., 1, 2, 3).
Encid int The encounter id associated with the audio workflow record, which is a foreign key referring to the `encounterid` column in the `enc` table.
PR500Ipsi varchar Pure tone audiometry result for the right ear at 500 Hz, ipsilateral (e.g., "25 dB", "30 dB").
PR500Contra varchar Pure tone audiometry result for the right ear at 500 Hz, contralateral (e.g., "25 dB", "30 dB").
PR1000Ipsi varchar Pure tone audiometry result for the right ear at 1000 Hz, ipsilateral (e.g., "20 dB", "25 dB").
PR1000Contra varchar Pure tone audiometry result for the right ear at 1000 Hz, contralateral (e.g., "20 dB", "25 dB").
PR2000Ipsi varchar Pure tone audiometry result for the right ear at 2000 Hz, ipsilateral (e.g., "15 dB", "20 dB").
PR2000Contra varchar Pure tone audiometry result for the right ear at 2000 Hz, contralateral (e.g., "15 dB", "20 dB").
PL500Ipsi varchar Pure tone audiometry result for the left ear at 500 Hz, ipsilateral (e.g., "25 dB", "30 dB").
PL500Contra varchar Pure tone audiometry result for the left ear at 500 Hz, contralateral (e.g., "25 dB", "30 dB").
PL1000Ipsi varchar Pure tone audiometry result for the left ear at 1000 Hz, ipsilateral (e.g., "20 dB", "25 dB").
PL1000Contra varchar Pure tone audiometry result for the left ear at 1000 Hz, contralateral (e.g., "20 dB", "25 dB").
PL2000Ipsi varchar Pure tone audiometry result for the left ear at 2000 Hz, ipsilateral (e.g., "15 dB", "20 dB").
PL2000Contra varchar Pure tone audiometry result for the left ear at 2000 Hz, contralateral (e.g., "15 dB", "20 dB").
TympRType varchar Tympanometry type for the right ear (e.g., "Type A", "Type B").
TympLType varchar Tympanometry type for the left ear (e.g., "Type A", "Type B").
TympRPP varchar Tympanometry peak pressure for the right ear (e.g., "-50 daPa", "0 daPa").
TympLPP varchar Tympanometry peak pressure for the left ear (e.g., "-50 daPa", "0 daPa").
TympRECV varchar Tympanometry equivalent ear canal volume for the right ear (e.g., "1.5 ml", "2.0 ml").
TympLECV varchar Tympanometry equivalent ear canal volume for the left ear (e.g., "1.5 ml", "2.0 ml").
TympRSC varchar Tympanometry static compliance for the right ear (e.g., "0.6 ml", "0.8 ml").
TympLSC varchar Tympanometry static compliance for the left ear (e.g., "0.6 ml", "0.8 ml").
TympRPH varchar Tympanometry peak height for the right ear (e.g., "0.5 ml", "0.7 ml").
TympLPH varchar Tympanometry peak height for the left ear (e.g., "0.5 ml", "0.7 ml").
TympRGr varchar Tympanometry gradient for the right ear (e.g., "0.2 ml", "0.3 ml").
TympLGr varchar Tympanometry gradient for the left ear (e.g., "0.2 ml", "0.3 ml").
TympPrToneFreq varchar Tympanometry probe tone frequency (e.g., "226 Hz", "1000 Hz").
PL4000Ipsi varchar Pure tone audiometry result for the left ear at 4000 Hz, ipsilateral (e.g., "10 dB", "15 dB").
PR4000Ipsi varchar Pure tone audiometry result for the right ear at 4000 Hz, ipsilateral (e.g., "10 dB", "15 dB").
PR4000Contra varchar Pure tone audiometry result for the right ear at 4000 Hz, contralateral (e.g., "10 dB", "15 dB").
PL4000Contra varchar Pure tone audiometry result for the left ear at 4000 Hz, contralateral (e.g., "10 dB", "15 dB").
