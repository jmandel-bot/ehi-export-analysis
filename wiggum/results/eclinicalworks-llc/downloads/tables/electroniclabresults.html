EHI Export Schema
Table electroniclabresults
Description This table stores lab HL7 results pulled from an external vendor.
Column DataType Description
Messageid (Primary Key) int This is an auto incremental primary key of This column .
Labid int This column stores lab id, primary key of lablist table.
DateReceived datetime This column stores date on which results are received.
DateModified datetime This column stores date on which results are modified.
MessageContent text This column stores HL7 message content
Messagestatus int This column stores message status codes.
sentid int This column stores the id from the LabHub for that messages for Hub & Spoke Interfaces. This column was being used for non-Generic Interfaces. As all Interfaces are now using the Generic workflow, so this column is not being used any more.
patientid varchar This column stores patient id, primary key of patients table.
providerid int This column stores provider id, primary key of doctor's table.
profileid varchar Profile id will store unique profile id received in HL7Message.
PtLastName varchar This column stores patient's lastname.
PtFirstName varchar This column stores patient's firstname.
DocName varchar This column will store doctor's firstname and last name.
OrderName varchar This column will store order name.
messagetype varchar This column will store the type of the Message (i.e. LAB)
ptdob datetime This column will store patient's date of birth.
errorlogs text This column will store the errorlogs if any occured while processing the hl7message.
reportname varchar Column stores name of the lab/report.
colldate varchar This column will store collection Date from hl7message.
colltime varchar Time when the sample was collected for the test (e.g., '09:00:00', '14:30:00').
encounterid varchar This column will store the encounterid.
hl7Msgid int This column will store the HL7messageid from electronicHL7content table.
OBRSno int This column will store sequence of result message. it will be 1 for single OBR result but incase of multiple it will store in increasing order for each OBR.
itemid int This column store the item id. It refers to itemid field in items table.
testcode varchar This column store the conducted test code.
docHL7id varchar This column store the HL7 id of the doctor.
BookCase varchar This column store the book case for correction interface.
NYSid varchar This column stores the New York state id.
reportid int This column will store Reportid (id from LabData) for result.
processflow text Text content of the process flow associated with the lab result message (e.g., 'Received > Processed > Sent').
XMLid int This column stores id from InterfaceXMl table and which is used for particular interface.
MigrateFlag tinyint This column used (one time use) for migrating data from This column to electronichl7content.
UpgradeMigrateFlag int This column used (one time use) for migrating data from This column to electronichl7content.
facilityid int This column will store the facilityid.
documentid int This column stores the document id. It refers to the id from the document table.
errorStatusCode tinyint This column store specific flag to type of preprocessing exception.
examCode varchar Examcode column will store value of examType lab parameter from hl7 message for Rad results and for other result by default it will be empty.
