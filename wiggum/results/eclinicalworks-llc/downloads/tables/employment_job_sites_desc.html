Table employment_job_sites_desc
Description This table stores information about various job sites, including their geographical location and associated employer.
Column DataType Description
id (Primary Key) int This is the primary key for the table, which is an auto-incrementing integer value. It uniquely identifies each record in the table. Example: 1, 2, 3, etc.
NAME varchar This column stores the name of the job site.
Latitude varchar This column stores latitude coordinate of the job site.
Longitude varchar This column stores longitude coordinate of the job site.
Parentid int This column stores the identifier for the parent job site, if any.
DeleteFlag int This column Indicates whether the job site has been deleted (1) or not (0). Example: 0, 1.
Employerid int This column stores the employer id .This column refers to the insid column in the insurance table.
