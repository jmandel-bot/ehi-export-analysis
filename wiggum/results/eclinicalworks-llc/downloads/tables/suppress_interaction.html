EHI Export Schema
Table suppress_interaction
Description This table stores data regarding suppressed medication interaction prompts.
Column DataType Description
id (Primary Key) int This auto-incremented primary key in This column . This column stores unique id for each suppressed interaction.
doctorid int This column stores id of a provider/staff who suppressed the interaction prompt. This column refers to UID column in user table.
Patientid int This column stores id of a patient for whom drug interaction warning was prompted. This column refers to UID column in user table.
InteractionType char This column stores abbreviation indicating type of interaction prompted in a drug interaction warning. DA - Drug to Allergy Interaction , DI - Drug to Drug Interaction.
Drugid1 int This column stores id of a drug which has interaction with other prescribed drug. ids stored in this column comes from Medispan API or Multum drug database.
Drugid2 int This column stores id of a drug which has interaction with other prescribed drug. ids stored in this column comes from Medispan API or Multum drug database.
OriginalSeverity varchar This column stores system assigned severity of interaction. In case of Drug to Drug interaction (IntercationType DI), 1 - Major , 2 - Moderate , 3 - Minor. In case of Drug to Allergy Interaction (IntercationType DA), reaction to the drug may be mentioned in this column.
AdjustedSeverity tinyint This column stores severity adjusted by provider/staff for certain interaction. 1 - Major , 2 - Moderate , 3 - Minor.
AdjustedBy int This column stores id of the provider/staff who adjusted severity for certain interaction. This column refers to UID column in user table.
AdjustedDate datetime This column stores date and time when provider/staff adjusted severity for certain interaction. Date and time is stored in 'YYYY-MM-DD hh:mm:ss.000' format.
encounter_id int This column stores the encounter id. This corresponds to encounterid of enc table.
erx_req_id int If drug interaction is loaded from refill or change request, the request id is logged in this column
