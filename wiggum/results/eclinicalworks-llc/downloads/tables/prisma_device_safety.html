Table prisma_device_safety
Description This table stores safety information related to devices, including their codes, display values, code systems, and code texts.
Column DataType Description
id (Primary Key) bigint A unique identifier for each record in the table. It is a bigint(20) type and is auto-incremented. Example: 1, 2, 3.
deviceId bigint A foreign key that refers to the `id` column in the `prisma_device_details` table. It is a bigint(20) type and is used to associate the safety information with a specific device. Example: 101, 102, 103.
code varchar A varchar(50) type column that stores the code related to the device safety information. It can be NULL. Example: "SAFETY01", "SAFETY02".
display varchar A varchar(50) type column that stores the display value associated with the device safety code. It can be NULL. Example: "Safety Warning", "Safety Alert".
codeSystem varchar A varchar(50) type column that stores the code system used for the device safety information. It can be NULL. Example: "SNOMED", "LOINC".
codeText varchar A varchar(255) type column that stores a textual description of the device safety code. It can be NULL. Example: "Device requires regular maintenance", "Device should not be used near water".
