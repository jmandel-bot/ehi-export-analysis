EHI Export Schema
Table derm_regionobservation
Description This table stores observations added to watchlist along with whether the observation has been addressed. If yes, then address reason.
Column DataType Description
id (Primary Key) int This is an auto incremental primary key of This column .
patientid int This column stores patient id for whom observations are documented and moved to watchlist. This is a foreign key that refers to the `pid` column in the `patients` table.
observationdetailid int The column stores id for observation added to watchlist. This refers to observationdetailid of table derm_observationdetails.
notes text This column is not in use.
startdate datetime This column stores date when this item is moved to watchlist.
enddate datetime This column stores date when this item is addressed and moved to watchlist history.
starttime datetime This column stores time when this item is moved to watchlist.
endtime datetime This column stores time when this item is addressed and moved to watchlist history.
isaddressed int This column stores value 0 / 1 indicating this item is addressed. 1 - Addressed; 0 - Not addressed.
observed_by int This column stores user id for who has moved this observation to watchlist. This column refers to UID column of user table.
addressed_reason text This column stores reason for which this item is addressed and moved to watchlist.
addressed_by int This column stores user id for who has addressed and moved this observation to watchlist history. This column refers to UID column of user table.
