EHI Export Schema
Table cascodes
Description This table stores information about various codes, their names, and their associated alphanumeric and numeric identifiers.
Column DataType Description
Code (Primary Key) varchar This is the primary key column and contains a unique varchar(15) code value. For example, 'A123B'.
Name text This column stores the text description or name associated with the code. For example, 'Sample Code Name'.
Alphaid varchar This column stores the value which may be NULL or 1st Character of Code i.e. if code is number then default value NULL or code is A1,A2,A3,A4 then stores A.
Numid int This column stores the default value 0 or the code i.e. if code is numeric value then stores the same numeric value otherwise if code is A0,A1 then 0,1.
