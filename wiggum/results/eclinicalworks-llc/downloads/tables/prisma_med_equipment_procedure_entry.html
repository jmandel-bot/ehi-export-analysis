Table prisma_med_equipment_procedure_entry
Description This table stores information about medical equipment used in various procedures, along with details about the procedure entry, target site, device, and author.
Column DataType Description
id (Primary Key) bigint A unique identifier for each record in the table (bigint). Example: 1, 2, 3.
sectionId bigint This column stores the ID of the associated section in the `prisma_ccd_document_section` table. It is a bigint value and can be NULL. Example: 10, 25, 37, etc.
procedureEntryId varchar A unique identifier for the procedure entry (varchar). Example: "PE-001", "PE-002", "PE-003".
medEquipId bigint An identifier for the medical equipment used in the procedure (bigint). This is a foreign key that refers to the `id` column in the `prisma_med_equipment_details` table. Example: 201, 202, 203.
codeid varchar An identifier for the code associated with the procedure entry (varchar). Example: "CID-001", "CID-002", "CID-003".
codeValue varchar The value of the code associated with the procedure entry (varchar). Example: "CV-001", "CV-002", "CV-003".
statusCode varchar The status code of the procedure entry (varchar). Example: "SC-001", "SC-002", "SC-003".
recordedDate varchar The date when the procedure entry was recorded (varchar). Example: "2021-09-01", "2021-09-02", "2021-09-03".
TargetSiteCode varchar An identifier for the target site of the procedure (varchar). Example: "TSC-001", "TSC-002", "TSC-003".
TargetSiteValue varchar The value of the target site of the procedure (varchar). Example: "TSV-001", "TSV-002", "TSV-003".
deviceCode varchar An identifier for the device used in the procedure (varchar). Example: "DC-001", "DC-002", "DC-003".
deviceName varchar The name of the device used in the procedure (varchar). Example: "Device A", "Device B", "Device C".
referenceText text A long text field containing reference information about the procedure entry (longtext). Example: "This is a reference text for procedure entry PE-001."
authorId varchar An identifier for the author of the procedure entry (varchar). Example: "AID-001", "AID-002", "AID-003".
authorName varchar The name of the author of the procedure entry (varchar). Example: "John Doe", "Jane Smith", "Alice Johnson".
