Table claim_actions
Description This table stores information about different claim actions, their scoring, and related properties.
Column DataType Description
Id (Primary Key) int A varchar(10) representing the type of the street (e.g., Ave, Blvd, St). Example: 'St'
Action varchar A varchar(10) representing whether the house numbers are odd, even, or both. Example: 'Both'
Scoring int A varchar(20) representing the place or area where the street is located. Example: 'Downtown'
deleteflag tinyint An integer representing a code for the census voting tabulation. Example: 5678
DueDays int A varchar(200) representing the city associated with the mailing address. Example: 'New York'
queueId int An integer representing the zip code of the area. Example: 10001
