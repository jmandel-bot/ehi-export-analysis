EHI Export Schema
Table clinicalnotifications
Description This table is the Transition Care Management program specific table and stores patient enrolled in this program.
Column DataType Description
id (Primary Key) int This column is an auto generated Primary Key.
patientname varchar This column stores last name, first name of the patient enrolled in the TCM program. (No information on data type size and default value)
providername varchar This column stores Primary Care Provider's last name, first name for the enrolled patient. (No information on data type size and default value)
assignedToid int This column stores id of the provider / staff to whom this patient is assigned. This column refers column UID from user table. (No information on default value)
assignedto varchar This column stores last, first name of the provider / staff to whom this patient is assigned. (No information on data type size)
admittype varchar This column stores enrolled patient's admit status and it can have one of the following: IP,ED,Admit,ER Visit,Discharge,Observation. Admit type cannot be a big length word/s hence with buffer varchar (30) selected. (No information on default value)
dischargedate datetime This column stores date on which the enrolled patient is discharged. The format used is YYYY-MM-DD HH:MM:SS hence dateTime data type selected.
reason varchar This column stores free text reason in brief for hospitalization for this patient. Hence varchar (150) is selected.
hospitalsourceid int This column stores id of the Hospital selected while enrolling this patient. This column refers column id from TCM_Hospitals table. (No information on default value)
hospitalsourcename varchar This column stores name of the Hospital for the hospital in hospitalsourceid column. This column refers column name from TCM_Hospitals table. (No information on data type size and default value)
status int This column indicates the status of the enrolled patient. It can be on of the below values: 0 - ALL (default value) , 1- OPEN, 2- ADDRESSED. (No information on data type size)
subscriptionname varchar Not in use.
serviceDescription text Not in use.
action varchar Not in use.
inputsource int This column indicates the source from where this patient is enrolled into this TCM program. It can be one of the below values: 0 - TCM Notes Screen, 1- Manual Enrolment,2- Admin Section 3- File Import. (No information on data type size and default value)
documentid int This column stores id of the document uploaded while enrolling this patient, if any. This column refers column docid from document table.
patientid int This column stores id of this enrolled patient. This column refers column UID from user table. (No information on default value)
link text Not in use.
notificationid int Not in use.
SOURCEtimestamp datetime Not in use.
notificationComments varchar This column stores the brief comment added while enrolling this patient. (No information data type size)
providerid int This column stores id of the Provider selected while enrolling this patient. This column refers column UID from user table. (No information on default value)
VisitNumber varchar This column stores unique identifier assigned by health care facility(hospital,clinic etc) and given to a patient for that particular visit to the facility. (No information on data type size and default value)
admitstatus varchar Not in use.
cptcode varchar This column stores a code indicating what was done to the patient during the consultation, including diagnostic, laboratory, radiology, and surgical procedures. (No information on data type size and default value)
diagnosisCode varchar This column stores a list of ICD code(s) for Secondary Diagnosis selected for this patient while enrolling. (No information on data type size)
itemid varchar This column stores a list of item id(s) of the corresponding code(s) in diagnosisCode column. (No information on data type, size and default value)
PrimaryDiagnosisCode varchar This column stores the ICD code for Primary Diagnosis selected fro this patient while enrolling. This column refers column value from ITEMDETAIL table. (No information on data type size and default value)
primaryDxItemid varchar This column stores the item id of the corresponding code in PrimaryDiagnosisCode column. This column refers column itemid from items table. (No information on data type, size and default value)
admnotificationdate datetime This column stores date on which the enrolled patient is admit. The format used is YYYY-MM-DD HH:MM:SS hence dateTime data type selected.
patIndicator tinyint Not in use.
pmid int This column stores id of the pmlist table. This column have value when the notification comes through an interface.
messageid int This column stores messageid of the electronicpmresults table. This column has an integer value when the notification comes through an interface.
dischargeFacilityid int This column stores id of the tcm_hospitals table.
dischargeType varchar This column stores id of the TCM_facilityTypes table.
ptRisk int This column store the integer value based on the patient primary diagnosis as follows. 1 - High Risk 2 - Moderate Risk 3 - Low Risk
ptAttemptsNo int This column stores value for how many attempts are made by the hospital to contact the patient after dicharge.
encid int This column stores the enconterid of the enc table. This column has an integer value when encounter is linked with tcm from TCM Notes.
pcpid int This column stores userid of PCP . This column has an integer value when patient enroll in TCM module
