Table todo_list_immunization
Description This table stores information about immunization items in a To Do List, including details about the immunization, status, and associated notes.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
todo_list_id int identifier for the to-do list that the immunization item belongs to. Foreign key relation: References todo_list table's id column. Example: 101, 102, 103.
item_id int identifier for the immunization item. Foreign key relation: References items table's itemid column. Example: 1001, 1002, 1003.
cvx char A 3-character code representing the specific vaccine or immunization. Example: 110, 120, 130.
notes varchar Text field for any additional information or notes related to the immunization item. Example: First dose, Second dose, Booster shot.
hm_flag tinyint A tiny integer (0 or 1) indicating if the immunization item is marked as high priority (1) or not (0). Example: 0, 1.
given_date datetime The date when the immunization was given, if applicable. Example: 2021-09-01, 2021-10-15, 2021-11-30.
