EHI Export Schema
Table ccmr_careplansignature
Description This table stores the information about the signature captured on care plan visit screen for that visit, including the signee, signature time, and other related details.
Column DataType Description
id (Primary Key) int Unique identifier for each care plan signature entry. Example: 1, 2, 3.
carePlanEncId int Foreign key referring to the care plan encounter ID. Example: 101, 102, 103. (Foreign key relation: `carePlanEncId` refers to the `id` column in the `carePlanEncounter` table)
signee varchar The name of the person who signed the care plan. Example: "John Doe", "Jane Smith".
signatureTime datetime The date and time when the care plan was signed. Example: '2021-09-01 14:30:00', '2021-09-02 10:15:00'.
deleteFlag tinyint A flag indicating whether the care plan signature entry has been deleted (1) or not (0). Example: 0, 1.
vrRandomNo varchar A random number associated with the care plan signature. Example: "12345", "67890".
signPadId tinyint Identifier for the signature pad used to capture the signature. Example: 1, 2, 3.
signature varbinary The binary data of the care plan signature image. Example: (Binary data).
fileName varchar The name of the file containing the care plan signature image. Example: "signature_1.png", "signature_2.jpg".
ftpDirPath varchar The directory path where the care plan signature image file is stored on the FTP server. Example: "/signatures/careplan/1/", "/signatures/careplan/2/".
signPresentIn tinyint A flag indicating the presence of the signature in the care plan (1) or not (0). Example: 0, 1.
