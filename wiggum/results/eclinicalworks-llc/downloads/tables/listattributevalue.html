EHI Export Schema
Table listattributevalue
Description This table stores information about various attributes and their values for patients and their encounters.
Column DataType Description
Id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
patientid int Unique identifier for the patient. Example: 101, 102, 103.
encounterId int Unique identifier for the patient's encounter. Example: 1001, 1002, 1003.
encounterdate datetime Date of the encounter. Example: '2021-09-01', '2021-09-02', '2021-09-03'.
ptage int Age of the patient at the time of the encounter. Example: 25, 30, 35.
Attributid varchar Identifier for the attribute. Example: 'height', 'weight', 'smoking_status'.
Value varchar Value of the attribute as a string. Example: '5ft 6in', '150 lbs', 'non-smoker'.
numericvalue int Numeric value of the attribute, if applicable. Example: 5, 10, 15.
bpsystolicvalue int Systolic blood pressure value, if applicable. Example: 120, 130, 140.
bpdystolicvalue int Diastolic blood pressure value, if applicable. Example: 80, 85, 90.
pcpid int Unique identifier for the primary care provider. Example: 201, 202, 203.
pcgid int Unique identifier for the primary care group. Example: 301, 302, 303.
gender varchar Gender of the patient. Example: 'male', 'female', 'other'.
race varchar Race of the patient. Example: 'Caucasian', 'African American', 'Asian'.
ethnicity varchar Ethnicity of the patient. Example: 'Hispanic', 'Non-Hispanic', 'Unknown'.
lang varchar Language spoken by the patient. Example: 'English', 'Spanish', 'Chinese'.
scheduledJobFlag int Flag to indicate if the record is part of a scheduled job (1) or not (0). Example: 1, 0.
