EHI Export Schema
Table patientinfo
Description This table stores patient's additional information captured from the Patient Demographics module.
Column DataType Description
pid int This column stores ID of a patient whose additional information is being recorded in this column. This column refers to UID column in user table.
birthOrder tinyint This column stores birth order of a patient.
advdirective text This column is not being used.
RegistryEnabled tinyint This column stores values 0 or 1 to indicate whether the patient is registry enabled or not. 0 - Patient is NOT Registry Enabled, 1 - Patient is Registry Enabled.
deceaseddate datetime This column is not being used.
registryenableddate datetime This column stores date when patient was enabled for Registry. Date is stored in 'YYYY-MM-DD 00:00:00.000' format.
inactivedate datetime This column stores date when patient was marked as inactive for registry search. Date is stored in 'YYYY-MM-DD 00:00:00.000' format.
ImmConsent char This column is not being used.
pcpStartDate datetime This column stores date when a PCP for the patient was selected. Date is stores in 'YYYY-MM-DD 00:00:00.000' format.
FinanceChrgAddedDate datetime This column stores date when financial charge was added for the patient. Date is stores in 'YYYY-MM-DD 00:00:00.000' format.
IgnoreFinanceChrg tinyint This column stores values 0 or 1 to indicate whether financial changes can be ignored or not. 0 - Financial charges should NOT be ignored, 1 - Financial charges should be ignored.
createdbymobile int This column stores values 0 or 1 to indicate whether the patient was created using eClinicalMobile application or not. 0 - patient was not created using eClinicalMobile, 1 - patient was created using eClinicalMobile application.
datemobilecreated datetime This column stores date when patient was created using eClinicalMobile application. Date is stores in 'YYYY-MM-DD 00:00:00.000' format.
homemsgtype varchar This column stores an option selected by end user for a message a practice can leave on a patient's home phone number. Values in this column can be 'Brief' or 'Extended'.
cellmsgtype varchar This column stores an option selected by end user for a message a practice can leave on a patient's cell phone number. Values in this column can be 'Brief' or 'Extended'.
workmsgtype varchar This column stores an option selected by end user for a message a practice can leave on a patient's work phone number. Values in this column can be 'Brief' or 'Extended'.
createdby int This column stores id of a user who created the patient only when a patient is created from eClinicalMobile application. This column refers to UID column from user table.
publicityCode int This column stores Publicity Code for a patient which is used to send patient's voice notifications.
PublicityCode_UpdatedTime datetime This column stores date and time when patient's publicity code is updated.
phytherapynotes varchar This column stores notes documented by provider or staff during Physical Therapy visit from Physical Therapy module.
ssnReason int This column stores ID for a reason why patient did not provide Social Security Number. 1 - Patient Refused, 2 - SSN unknown, 3 - Child SSN Unknown, 4 - Patient does not have SSN, 5 - Other Reason, 6 - Duplicate SSN.
ssnReasonNotes varchar If 'Other Reason' option is selected for ssnReason column, this column stores free text reason typed in by end user for patient not providing their SSN.
