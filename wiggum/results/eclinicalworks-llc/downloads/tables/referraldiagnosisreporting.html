EHI Export Schema
Table referraldiagnosisreporting
Description This table stores information about the diagnosis reports associated with each referral.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the referraldiagnosisreporting table. Example: 1, 2, 3.
referralid int The identifier of the referral associated with the diagnosis report. This is a foreign key that refers to the `ReferralId` column in the `referral` table. Example: 101, 102, 103.
diagnosiscode varchar A code representing the specific diagnosis for the referral. Example: 'A01.0', 'B02.1', 'C03.2'.
diagnosisname varchar The name of the diagnosis associated with the diagnosis code. Example: 'Typhoid fever', 'Zoster encephalitis', 'Malignant neoplasm of upper lobe, bronchus or lung'.
faxlogid int The identifier of the fax log associated with the diagnosis report. Example: 1001, 1002, 1003.
ts_utc datetime The time stamp in UTC when the record was created or updated. Example: '2022-01-01 12:34:56', '2022-02-02 23:45:01'.
