Table asc_macro_specimenorder_mapping
Description This table stores the mapping information between macro, report, and specimen entities, along with their respective answers and notes.
Column DataType Description
id (Primary Key) int Unique identifier for each mapping record. Example: 1, 2, 3.
macroid int Identifier for the macro associated with this mapping. Example: 101, 102, 103.
reportid int Identifier for the report associated with this mapping. Example: 201, 202, 203.
specimenid int Identifier for the specimen associated with this mapping. Example: 301, 302, 303.
macroversionid int Identifier for the macro version associated with this mapping. Example: 401, 402, 403.
locationAnswerid varchar Identifier for the location answer associated with this mapping. Example: LA1, LA2, LA3.
collectionAnswerid varchar Identifier for the collection answer associated with this mapping. (Example: CA1, CA2, CA3.
locationAnswer text Text description of the location answer. Example: Location A, Location B, Location C.
collectionAnswer text Text description of the collection answer. Example: Collection 1, Collection 2, Collection 3.
Notes text Additional notes or comments related to the mapping. Example: Note 1, Note 2, Note 3.
deleteFlag int Flag to indicate if the mapping record is deleted (1) or not (0). Example: 0, 1.
ruleoutAnswerid int Identifier for the rule out answer associated with this mapping. Example: 501, 502, 503.
ruleoutAnswer varchar Text description of the rule out answer. Example: Rule Out A, Rule Out B, Rule Out C.
