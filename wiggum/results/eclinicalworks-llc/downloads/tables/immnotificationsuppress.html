EHI Export Schema
Table immnotificationsuppress
Description This table stores suppressed immunizations. This table stores the Suppress information including suppress date and status for a given Immunization/Th Injection that is marked as Not Administered.
Column DataType Description
id int This column stores an identifier for the row.
Immid int This column stores Immunization id of an Immunization that is/was suppressed. This column refers to column immunizationid from immunizations.
Itemid int This column stores Item id of an Immunization that is/was suppressed. This column refers column itemid from items table.
SuppressDate datetime This column stores the Date when Immunization is suppressed. The format used is YYYY-MM-DD HH:MM:SS.
SuppressStatus tinyint This column stores suppress status of an Immunization that is/was suppressed .entry WITH suppressstatus = 1 - UNTIL a certain date, entry WITH suppressstatus=2 - forever, entry WITH suppressstatus = 0 - was Suppressed initially later status was changed to Administered/Pending/etc.
deleteflag tinyint This column stores value 0 or 1. 1- when the suppressed immunization is deleted. 0 - Otherwise.
EffectiveDate datetime This column stores the Effective Date when Immunization is suppressed. The format used is YYYY-MM-DD.
ExpirationDate datetime This column stores the Expiration Date when Immunization is suppressed. The format used is YYYY-MM-DD.
