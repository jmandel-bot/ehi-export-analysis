Table prisma_pot_encounter_performer
Description This table stores information about the performers involved in a specific External encounter, including their personal and organizational details.
Column DataType Description
id (Primary Key) bigint Unique identifier for each record in the table. Example: 1, 2, 3.
encounterDetailId bigint Foreign key referring to the `prisma_pot_encounter_details` table's `id` column, representing the specific encounter the performer is associated with. Example: 101, 102, 103.
userId varchar Unique identifier for the user associated with the performer. Example: "U123", "U124", "U125".
addressLine1 varchar First line of the performer's address. Example: "123 Main St", "456 Oak St".
addressLine2 varchar Second line of the performer's address (optional). Example: "Apt 4B", "Suite 300".
city varchar City where the performer is located. Example: "New York", "Los Angeles".
county varchar County where the performer is located. Example: "Los Angeles County", "Kings County".
state varchar State abbreviation where the performer is located. Example: "CA", "NY".
postalcode varchar Postal code for the performer's address. Example: "90210", "10001".
country varchar Country where the performer is located. Example: "USA", "Canada".
homePhone varchar Home phone number of the performer. Example: "555-123-4567", "555-987-6543".
workPhone varchar Work phone number of the performer. Example: "555-321-7654", "555-678-9876".
faxnumber varchar Fax number of the performer. Example: "555-111-2222", "555-333-4444".
personFName varchar First name of the performer. Example: "John", "Jane".
personLName varchar Last name of the performer. Example: "Doe", "Smith".
personSufix varchar Suffix for the performer's name (optional). Example: "Jr.", "III".
organizationId varchar Unique identifier for the organization the performer is associated with. Example: "O123", "O124".
organizationName varchar Name of the organization the performer is associated with. Example: "Acme Corp", "Globex Inc".
organizationPhone varchar Phone number of the organization. Example: "555-222-3333", "555-444-5555".
organizationFax varchar Fax number of the organization. Example: "555-666-7777", "555-888-9999".
organizationAddLine1 varchar First line of the organization's address. Example: "789 Pine St", "246 Maple St".
organizationAddLine2 varchar Second line of the organization's address (optional). Example: "Floor 2", "Building B".
organizationCity varchar City where the organization is located. Example: "San Francisco", "Chicago".
organizationState varchar State abbreviation where the organization is located. Example: "IL", "TX".
organizationpostalcode varchar Postal code for the organization's address. Example: "60601", "78701".
organizationCountry varchar Country where the organization is located. Example: "USA", "Mexico".
