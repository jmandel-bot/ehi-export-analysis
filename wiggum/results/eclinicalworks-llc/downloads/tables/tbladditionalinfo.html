Table tbladditionalinfo
Description This table stores additional information about patients, their contact details, address, employment, and preferred pharmacy.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
pid int Patient identifier, referring to the patient's unique id in another table. Foreign key relation: `patientTable(patientid)`. Example: 101, 102, 103.
mobileno varchar Mobile phone number of the patient. Example: '123-456-7890', '987-654-3210'.
workPhone varchar Work phone number of the patient. Example: '555-123-4567', '555-987-6543'.
Pcpid int Primary care provider identifier, referring to the provider's unique id in another table. Foreign key relation: `providerTable(providerid)`. Example: 201, 202, 203.
RefPrid int Referring provider identifier, referring to the provider's unique id in another table. Foreign key relation: `providerTable(providerid)`. Example: 301, 302, 303.
RendPrid int Rendering provider identifier, referring to the provider's unique id in another table. Foreign key relation: `providerTable(providerid)`. Example: 401, 402, 403.
maritalStatus varchar Marital status of the patient. Example: 'Single', 'Married', 'Divorced'.
strAddress1 varchar Street address line 1 of the patient. Example: '123 Main St', '456 Elm St'.
strAddress2 varchar Street address line 2 of the patient (optional). Example: 'Apt 4B', 'Suite 300'.
city varchar City of the patient's address. Example: 'New York', 'Los Angeles'.
state varchar State of the patient's address. Example: 'NY', 'CA'.
zip varchar Zip code of the patient's address. Example: '10001', '90001'.
language varchar Preferred language of the patient. Example: 'English', 'Spanish'.
race varchar Race of the patient. Example: 'White', 'Black or African American'.
restype varchar Residence type of the patient. Example: 'House', 'Apartment'.
empName varchar Name of the patient's employer. Example: 'Acme Corp', 'Tech Solutions'.
empAddress1 varchar Street address line 1 of the patient's employer. Example: '789 Market St', '123 Corporate Dr'.
empAddress2 varchar Street address line 2 of the patient's employer (optional). Example: 'Floor 5', 'Building 2'.
empCity varchar City of the patient's employer's address. Example: 'San Francisco', 'Chicago'.
empState varchar State of the patient's employer's address. Example: 'CA', 'IL'.
empZipCode varchar Zip code of the patient's employer's address. Example: '94103', '60601'.
pharmacyid int Preferred pharmacy identifier, referring to the pharmacy's unique id in another table. Foreign key relation: `pharmacyTable(pharmacyid)`. Example: 501, 502, 503.
ethnicity varchar Ethnicity of the patient. Example: 'Hispanic', 'Non-Hispanic'.
uparea varchar Area of the Patient(e.g., 'Floor 3', 'Building 5')
