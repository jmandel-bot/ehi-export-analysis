EHI Export Schema
Table enc_billingdata
Description This table stores billing data related to medical encounters.
Column DataType Description
encounterid (Primary Key) int An integer representing the unique identifier for the encounter. This is the primary key for the table. Example: 12345
hcfa19 varchar A varchar (83) field that stores information related to the HCFA-19 form. Example: "Sample HCFA-19 data"
claimnote varchar A varchar (103) field that stores any additional notes related to the claim. Example: "Patient has a pre-existing condition"
spprgcode varchar An enum field that stores the special program code. Allowed values are ', '01', '02', '03', '05', '07', '08', '09'. Example: '03'
epsdtrefpt varchar An enum field that stores the Early and Periodic Screening, Diagnostic, and Treatment (EPSDT) referral indicator. Allowed values are '', 'N', 'Y'. Example: 'Y'
epsdtrefcode varchar An enum field that stores the EPSDT referral code. Allowed values are '', 'AV', 'NU', 'S2', 'ST'. Example: 'AV'
symptomdatetype varchar A varchar (5) field that stores the type of date related to the symptom or accident. Example: "ACC"
accidentorsymptomdt datetime A date field that stores the date of the accident or symptom. Example: '2021-09-01'
hcfaformstartdt datetime A date field that stores the start date of the HCFA form. Example: '2021-09-02'
hcfaformenddt datetime A date field that stores the end date of the HCFA form. Example: '2021-09-30'
