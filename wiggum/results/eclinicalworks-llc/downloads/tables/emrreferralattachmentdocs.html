EHI Export Schema
Table emrreferralattachmentdocs
Description This table stores documents attached to incoming or outgoing referrals.
Column DataType Description
Id (Primary Key) int Unique identifier for each attachment record. Example: 1, 2, 3.
ReferralReqId int This column stores id of a referral through which document has been received or sent out. This column refers to column ReferralId in referral table.
document varbinary This column stores entire document attached to corresponding referral id mentioned in ReferralReqId column in binary format.
filename varchar This column stores name of the file attached to referral.
filelabel varchar This column stores label of document attached to referral, if any.
