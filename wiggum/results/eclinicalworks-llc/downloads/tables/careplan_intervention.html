EHI Export Schema
Table careplan_intervention
Description This table stores information about interventions associated with specific care plans and their status.
Column DataType Description
id (Primary Key) int Unique identifier for each care plan intervention record. Example: 1, 2, 3, etc.
CarePlanid int identifier for the care plan associated with the intervention. This is a foreign key that refers to the `id` column in the `careplan` table. Example: 101, 102, 103, etc.
CarePlanEncDataid int identifier for the care plan encounter data associated with the intervention. This is a foreign key that refers to the `id` column in the `careplan_encounter_data` table. Example: 1001, 1002, 1003, etc.
Intervention text Text description of the intervention. Example: "Physical therapy", "Medication adjustment", "Dietary changes", etc.
Userid int identifier for the user who created the intervention record. This is a foreign key that refers to the `id` column in the `user` table. Example: 501, 502, 503, etc.
Created datetime Date and time when the intervention record was created. Example: '2021-09-01 12:34:56', '2021-09-02 14:23:45', etc.
Status int Integer value representing the status of the intervention (1 for active, 0 for inactive). Example: 1, 0.
