EHI Export Schema
Table tags
Description This table stores the relationships between notes and their associated tags, as well as a delete flag for each relationship.
Column DataType Description
ID (Primary Key) int Unique identifier for each record in the `tags` table. Example: 1, 2, 3.
notesId int The identifier of the note in the `additionalnotes` table. This column has a foreign key relation with the `Id` column in the `additionalnotes` table. Example: 101, 102, 103.
TagsId int The identifier of the tag in the `notes_tag` table. This column has a foreign key relation with the `TagID` column in the `notes_tag` table. Example: 201, 202, 203.
deleteflag tinyint A flag indicating whether the relationship between the note and the tag has been deleted (1) or not (0). Example: 0, 1.
