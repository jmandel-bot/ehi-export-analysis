EHI Export Schema
Table edi_patientstatementdate
Description This table stores information about patient statement dates, their associated practice, and billing facility details.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
pid int Unique identifier for the patient. This could be a foreign key referring to a `patient` table with a column `id`. Example: 101, 102, 103.
Practiceid int Unique identifier for the medical practice. This could be a foreign key referring to a `practice` table with a column `id`. Example: 201, 202, 203.
StmtDate datetime The date of the patient statement. Example: '2021-08-01', '2021-08-02', '2021-08-03'.
Grid int This column store the guarantor id for the patient when patient statement generated.
billingFacilityOfPayToProvider int Unique identifier for the billing facility or pay-to provider. This could be a foreign key referring to a `billing_facility` or `pay_to_provider` table with a column `id`. The default value is '0'. Example: 401, 402, 403.
