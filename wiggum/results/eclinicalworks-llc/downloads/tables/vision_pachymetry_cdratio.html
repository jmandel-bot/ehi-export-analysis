EHI Export Schema
Table vision_pachymetry_cdratio
Description This table stores information about vision pachymetry and cup-to-disc ratio (CD ratio) measurements for eye exams.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer value. It uniquely identifies each record in the table. Example: 1, 2, 3, etc.
Encounterid int identifier for the encounter associated with the eye exam. This is a foreign key that refers to the `encounterid` column in the `enc` table. Example: 101, 102, 103.
ExamType int Integer value representing the type of eye exam conducted. Example: 1 (routine checkup), 2 (glaucoma screening), 3 (diabetic retinopathy screening).
ODHorz varchar Horizontal cup-to-disc ratio measurement for the right eye (OD). Stored as a string. Example: "0.6", "0.7", "0.8".
ODVert varchar Vertical cup-to-disc ratio measurement for the right eye (OD). Stored as a string. Example: "0.5", "0.6", "0.7".
OSHorz varchar Horizontal cup-to-disc ratio measurement for the left eye (OS). Stored as a string. Example: "0.4", "0.5", "0.6".
OSVert varchar Vertical cup-to-disc ratio measurement for the left eye (OS). Stored as a string. Example: "0.3", "0.4", "0.5".
