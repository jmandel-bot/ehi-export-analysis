EHI Export Schema
Table derm_lesionlist
Description This table stores information about various types of dermatological lesions, their descriptions, images, and other related details.
Column DataType Description
lesionid (Primary Key) int This is auto-incremented primary key in This column .
lesionname varchar This column stores name of lesion.
lesiondescription text This column stores description for lesion.
status tinyint This column stores status of lesion. 1 for active and 0 for inactive.
imagename varchar This column stores icon name for lesion.
adminicon varchar This column stores admin icon name for lesion. This column is not in use.
type int This column stored type of lesion. It stores 1 for findings, 2 for procedures and 3 for diagnosis.
lesionorder int This column stores display order for lesion.
title varchar This column stores tool tip title of lesion. This column is not in use.
abbreviation varchar This column stores tool tip title of lesion. This column is not in use.
parentid int This column stores lesion id of parent lesion.
isdeleted tinyint Indicates whether the lesion entry has been deleted (1) or not (0). Example: 1, 0.
