Table ccmr_groupvisitptenc
Description This table stores information about group visits and their associated patient encounters.
Column DataType Description
id (Primary Key) int This is the primary key column and contains a unique integer value for each record in the table. It is used to uniquely identify each row. Example: 1, 2, 3, etc.
groupid int This column contains an integer value representing the group visit id. It is used to associate a patient encounter with a specific group visit. Example: 101, 102, 103, etc.
ptEncid int This column contains an integer value representing the patient encounter id. It is used to associate a patient encounter with a specific group visit. Example: 1001, 1002, 1003, etc.
delflag tinyint This column contains a tinyint value (either 0 or 1) representing the deletion flag. It is used to indicate whether a record has been deleted or not. A value of 0 means the record is not deleted, and a value of 1 means the record is deleted. Example: 0, 1.
