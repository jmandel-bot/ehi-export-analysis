Table ip_rh_druginteraction
Description This table is used to store interaction override details like interaction override reason, severity for interaction and acknowledge flag for acknowledged interaction
Column DataType Description
interactionId (Primary Key) bigint This is the unique identifier for each interaction record.
severityCode tinyint This column is used to store severity code of interaction. The default value is 0. Possible values are: 1 - minor 2 - moderate 3 - severe
encounterId int This column is used to store encounter ID and is foreign key references encounterId column of enc table.
description varchar This column is used to store interaction messages.
overrideFlag tinyint This column is used to store override flag it shows whether interaction is override. 1 - represents interaction is override 0 - represents interaction is not overriden Default value is 1 as record is added when the interaction is override.
overrideReason int This column is used to store override reason id and is foreign key references id column of ip_rh_drugFormulary_reasondictionaryitems table.
interactionCategory int This column is used to store is category of interaction. Default value is 0 and possible values are: 1 - drug to drug 2 - drug to allergy 3 - drug to alcohol/food 6 - duplicate therapy
overrideBy int This column is used to store override by Id and is foreign key references uid column of users table.
overrideDateTime datetime This column is used to store date and time of override interaction in yyyy-mm-dd hh:mm:ss format.
acknowledgeFlag tinyint This column is used to store acknowledge flag it shows whether interaction is acknowledged.1 - interaction is acknowledged 0 - interaction is not acknowledged. Default value is 0 as record is added when interaction is override.
acknowledgeBy int This column is used to store acknowledge Id.
acknowledgeDateTime datetime This column is used to store date and time of acknowledge interaction in yyyy-mm-dd hh:mm:ss format.
