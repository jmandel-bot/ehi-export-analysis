EHI Export Schema
Table vision_hirschberg
Description This table stores information related to the Hirschberg and Kappa measurements of a patient's eyes during an encounter.
Column DataType Description
id (Primary Key) int This column is an auto incremental primary key.
encounterid int This column stores the encounter id and has a foreign key constraint on the "encounterid" column of the "enc" table.
r_hirschberg decimal This column stores decimal (3,2) precision value, since it only accepts number between -6.00 to +6.00 and by default is set to NULL.
l_hirschberg decimal This column stores decimal (3,2) precision value, since it only accepts number between -6.00 to +6.00 and by default is set to NULL.
r_kappa decimal This column stores decimal (3,2) precision value, since it only accepts number between -6.00 to +6.00 and by default is set to NULL.
l_kappa decimal This column stores decimal (3,2) precision value, since it only accepts number between -6.00 to +6.00 and by default is set to NULL.
comments varchar This column stores Hirscberg Tests notes and varchar (255) is choosen to save comments.
