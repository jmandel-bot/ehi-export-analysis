EHI Export Schema
Table rxhub_scriptlog
Description This table stores logs of prescriptions.
Column DataType Description
id (Primary Key) int This column is an auto incremented Primary Key.
Doc_FName varchar This column stores doctor's first name who prescribes the medication which could be a string with maximum 50 characters in it.
Doc_LName varchar This column stores doctor's last name who prescribes the medication which could be a string with maximum 50 characters in it.
Doc_DeaNo varchar This column stores a DEA number (alphanumeric) an identifier assigned to a health care provider by the United States Drug Enforcement Administration allowing them to write prescriptions for controlled substances.
ScriptDate varchar This column stores the date in format YYYYMMDD when the medication was prescribed by the Provider.
Pt_FName varchar This column stores patient's first name who prescribes the medication which could be a string with maximum 50 characters in it.
Pt_LName varchar This column stores patient's last name who prescribes the medication which could be a string with maximum 50 characters in it.
Pt_PBMid varchar This column stores PBMid which is coming from Rx Eligibility.
PayorSourceName varchar This column stores Payor's source name which is coming from Rx Eligibility.
Formularyid varchar This column stores Formulary id coming from Rx Eligibility.
Scriptid varchar This column stores unique id generated for that DEA as Script id.
ESIMail char not in use.
PDHViewed char not in use.
FormularyStatus char This column stores the status of Formulary applicable. This value comes from Formulary Check (internal formulary hub server).
NDCCode varchar This column stores NDC code of the prescribed medication. The code has hyphen '-' in it.
DrugName varchar This column stores name of the prescribed Drug.
dosage varchar This column stores an information about the dosage of the prescribed medicine.
Formulation varchar This column stores an information about the form of the prescribed medicine.
Quantity varchar This column stores dispense data of the prescribed medication.
Generic char This column stores a value 'G' or 'B'. G - if the prescribed medication is generic, B - when it is branded.
OriginalScript char This column stores a flag 'Y' or 'N'. Y - if medication prescribed is for first time for that encounter, N - Otherwise.
Refills varchar This column stores maximum number of refills for the prescribed medication.
DAW char This column stores a flag 'y' or 'N' for Dispense As Written flag. 'Y' - DAW , 'N' - else substitute allowed.
RxChannel varchar This column stores the action taken on the eprescription. It could be one of the below : Faxed, Printed, ePrescripIpad, Fax:Mobile, ePrescription, eRx:Mobile.
XMLReceived text This column stores xml format for this record.
Patientid int This column stores Patient's id for whom the medication is prescribed. This column refers column UID from user table.
doctorid int This column stores doctor's id who prescribes the medication. This column refers column UID from user table.
MailOrderPharmacy tinyint This column stores a value 1 or 0. 1 - if Pharmacy is a Mail Order, 0 - otherwise
Alternativeid varchar This column stores alternative id which populates from Rx Eligibility.
Coverageid varchar This column stores coverage id and populates from Rx Eligibility.
Copayid varchar This column stores copay id which populates from Rx Eligibility.
NPI varchar This column stores the provider’s National Provider identification number.
NCPDPid varchar This column store the unique, national identification number assigned by the National Council for Prescription Drug Programs (NCPDP) to every licensed pharmacy in the United States and its territories.
PayorSourceCode varchar This column stores the code for the Payor source from PayorSourceName column, this value populates from Rx Eligibility.
HealthPlanGroupid varchar This column stores Health Plan Group id which is populated from Rx Eligibility.
HealthPlanid varchar This column stores Health Plan id which is populated from Rx Eligibility.
Encounterid int This column stores the encounter id to which this eprescription is tied to. This column refers to column encounterid from enc table.
Itemid int This column stores the item is corresponding to the prescribed medication. This column refers column itemid from items table.
LoginUserid int This column stores the logged in user's id, this column refers column UID from user table.
TransactionTime varchar This column is to store the time stamp of the transaction when it happens. The format used is YYYYMMDDHHMMSS.
encType int This column stores type of the encounter which this eprescription is tied to. It could be one of the below values: 1 - Office Visit, 2 - Tel Encounter, Virtual Visits, 3 - Out of Office, 4 - Claim, 5 - Lab, 6 - Web Encounter, 7 - ePrescription refills, 8 - PTDASH, 9 - Orderset with patientid as 8663.
LogBatchid int This column stores batch id, this column is used for reporting for Ohio practices.
LogStatus varchar This column stores status of the log (Logged, Reviewed) and is used for reporting for Ohio practices.
Take varchar Instructions for taking the medication (e.g., "Take 1 tablet daily").
Freq varchar Frequency of taking the medication (e.g., "Once daily").
Route varchar Route of administration for the medication (e.g., "Oral").
Duration varchar Duration of the medication treatment (e.g., "30 days").
PharmacyName varchar Name of the pharmacy where the prescription was sent.
OTCCode char Over-the-counter code for the medication (Y for yes, N for no).
CSASchedule char Controlled Substance Act schedule for the medication (e.g., "2" for Schedule II).
IpAddress varchar IP address of the user who logged the prescription.
rxorderno varchar This column stores order number for the set of medications prescribed during a specific encounter. This is application generated number which always begins with 'S' character.
SuprPrid int Unique identifier for the supervising provider, if applicable.
pharmacyid int This column stores the Pharmacy id to which the eprescription is sent.
FaxLogid int Unique identifier for the fax log entry associated with the prescription. Foreign key relation to `faxlogs(Logid)`.
sendToRxLogid int Unique identifier for the send to Rx log entry associated with the prescription. Foreign key relation to `send_to_rx_logs(id)`.
