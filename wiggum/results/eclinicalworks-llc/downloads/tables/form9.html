EHI Export Schema
Table form9
Description This table stores information related to Massachusetts Division of Medical Assistance - Medical services Claim provided to patients by healthcare providers.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table (e.g., 1, 2, 3).
InvId int Invoice ID associated with the billing record, can be NULL (e.g., 101, 102, 103).
ProviderName varchar Name of the healthcare provider (e.g., "John Smith", "Jane Doe").
ProviderAddressLine1 varchar First line of the provider's address (e.g., "123 Main St").
ProviderAddressLine2 varchar Second line of the provider's address, if applicable (e.g., "Apt 4B").
ProviderTeleNo varchar Telephone number of the healthcare provider (e.g., "555-123-4567").
PayToProviderNo varchar Pay-to provider number (e.g., "12345678").
BillingAgentNo varchar Billing agent number (e.g., "87654321").
PriorAuthNo varchar Prior authorization number, if applicable (e.g., "123456789").
ServProviderName varchar Name of the service provider (e.g., "Michael Johnson").
ServProviderNo varchar Service provider number (e.g., "23456789").
RefProviderName varchar Name of the referring provider, if applicable (e.g., "Samantha Brown").
RefProviderNo varchar Referring provider number, can be NULL (e.g., "345678912").
MemberName varchar Name of the member or patient (e.g., "Emily Williams").
RecipientIdNo varchar Recipient ID number (e.g., "456789123").
DOBDD char Day of birth of the patient (e.g., "01").
DOBMM char Month of birth of the patient (e.g., "12").
DOBYY char Year of birth of the patient (e.g., "90").
Sex char Sex of the patient (e.g., "M" for male, "F" for female).
OtherInsurance char Indicates if the patient has other insurance (e.g., "Y" for yes, "N" for no).
PatientAccNo varchar Patient account number (e.g., "567891234").
POS char Place of service code (e.g., "11").
IsMemberAccidentN char Indicates if the member's condition is not related to an accident. Value is X if member is not related to an accident else blank
IsMemberAccidentY char Indicates if the member's condition is not related to an accident. Value is X if member is related to an accident else blank
MemberAccidentY char If IsMemberAccidentY column value is X then this column value is &
DOADD char Day of the accident, if applicable (e.g., "15").
DOAMM char Month of the accident, if applicable (e.g., "06").
DOAYY char Year of the accident, if applicable (e.g., "20").
IsMemberEPSDTN char Indicates if the member is not part of the Early and Periodic Screening, Diagnostic, and Treatment (EPSDT) program (e.g., "X" no).
IsMemberEPSDTY char Indicates if the member is part of the Early and Periodic Screening, Diagnostic, and Treatment (EPSDT) program (e.g., "X" for yes).
LOF varchar Level of care code (e.g., "1234").
PatientStatus char Patient status code (e.g., "001").
DisDateDD char Day of discharge, if applicable (e.g., "30").
DisDateMM char Month of discharge, if applicable (e.g., "09").
DisDateYY char Year of discharge, if applicable (e.g., "21").
DiagnosisCode1 varchar First diagnosis code (e.g., "J45.909").
DiagnosisName1 varchar First diagnosis name (e.g., "Asthma, unspecified").
DiagnosisCode2 varchar Second diagnosis code, if applicable (e.g., "E11.9").
DiagnosisName2 varchar Second diagnosis name, if applicable (e.g., "Type 2 diabetes mellitus without complications").
