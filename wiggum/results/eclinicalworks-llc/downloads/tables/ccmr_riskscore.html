EHI Export Schema
Table ccmr_riskscore
Description This table is used to store final risk score for patient. This score can be system calculated or a custom score entered by the user.
Column DataType Description
id (Primary Key) int The column stores an id of the risk score of a patient. This is an auto incremental primary key of This column .
patientid int This column stores identifier number of a patient, for whom the score is saved. It refers to column uid from user table
score varchar This column contains score value. The value can be whole number or decimal number.
invalid tinyint This column is used to mark score is invalid or not. (0=Valid, 1- Invalid)
modifiedbyid int This column stores identifier number of the provider has stored/updated the score for patient. It refers to column uid from user table.
modifiedon datetime This column stores the date and time when the score was created. The format used is YYYY-MM-DD 00:00:00.000.
riskcategory varchar This column is not in use.
reason varchar This column used to specify the reason while storing/updating score value fora patient.
templateId int This column stores identifier number of a template for whom the score is saved. It refers to column id from ccmr_riskscoretemplate table.
