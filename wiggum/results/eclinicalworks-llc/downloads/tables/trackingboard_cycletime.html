EHI Export Schema
Table trackingboard_cycletime
Description This table stores information about the cycle time of patient encounters, including their status changes and timestamps.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
encounterid int Unique identifier for a patient encounter. Example: 101, 102, 103.
patientid int Unique identifier for a patient. Example: 1001, 1002, 1003.
encDate varchar Date of the encounter as a string. Example: '2021-09-01', '2021-09-02', '2021-09-03'.
prevstatus varchar stores the previous status of the encounter when there is a visit status change
currentStatus varchar stores the current status of the encounter when there is a visit status change
prevTime varchar stores the previous time of the encounter when there is a visit status change
nextTime varchar stores the current time of the encounter when there is a visit status change
userid int stores the user id of the user who changes the visit status of an encounter
timestamp varchar stores the time stamp when the visit status is changed
UTCNextTime datetime stores the current time in UTC format of the encounter when there is a visit status change
UTCPrevTime datetime stores the Previous time in UTC format of the encounter when there is a visit status change
