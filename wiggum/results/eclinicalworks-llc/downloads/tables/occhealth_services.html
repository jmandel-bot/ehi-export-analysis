EHI Export Schema
Table occhealth_services
Description This table stores information related to occupational health services details including the mapped cpt, and letter and templates details.
Column DataType Description
id (Primary Key) int This column stores a unique identifier for each occupational health service record.
servicename varchar This column stores Name of the occupational health service. (e.g., "Physical Examination", "Vaccination")
description varchar This column stores description of the occupational health service. (e.g., "Annual employee physical examination", "Flu vaccination for employees")
cptid varchar This column stores CPT code associated with the service.
letterid varchar This Column stores identifier for the letter template associated with the service.
templateid int This column stores identifier for the document template associated with the service.
ordersetid int This column stores identifier for the order set associated with the service.
estimatedtime int This column stores estimated time in minutes to complete the service.
note text This column stores additional notes or comments about the service.
delflag tinyint This column stores a flag indicating if the service record has been deleted (1) or not (0). (e.g., 0, 1)
cptitemid int This column stores identifier for the CPT item associated with the service.
cptFeeSchid int This column stores identifier for the CPT fee schedule associated with the service.
isActive tinyint This column stores A flag indicating if the service is currently active (1) or not (0). (e.g., 0, 1)
serviceStartDate varchar This column stores Start date of the service availability in the format YYYY-MM-DD.
serviceEndDate varchar This column stores end date of the service availability in the format YYYY-MM-DD.
