EHI Export Schema
Table surescript_dur_details
Description This table is used to save drug use details which will be in the refill request and change Rx request.
Column DataType Description
id (Primary Key) int This column is auto generated Primary Key of the table.
Messageid varchar This column stores and indicates Message id of the refill/changerx request
ServiceReasonCode varchar This column stores Service reason code for DrugUseEvaluation. This column refers column 'Code' from surescript_servicereasoncode table.
ServiceResultCode varchar This column stores Service Result Code for DrugUseEvaluation. This column refers column 'Code' from surescript_serviceresultcode table.
ProfessionalServiceCode varchar This column stores Professional Service Code for DrugUseEvaluation. This column refers column 'Code' from surescript_professionalservicecode table.
ClinicalSignificanceCode varchar This column stores Clinical Significance Code for DrugUseEvaluation. This column refers column 'Code' from surescript_clinicalsignificancecode table.
AcknowledgementReason varchar This column stores Acknowledgement Reason for DrugUseEvaluation.
CoAgentid varchar This column stores Co agent identifier for DrugUseEvaluation.
CoAgentQualifier varchar This column stores Co agent Qualifier for DrugUseEvaluation.
