EHI Export Schema
Table echeckinmedicaldata
Description This table stores medical data related to a patient's electronic check-in, including surgery history, hospitalization history, medication list, and an allergy list.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the echeckinmedicaldata table. Example: 1, 2, 3.
uid int Unique identifier for the user (patient) associated with the medical data. Foreign key relation: `users` table, `uid` column. Example: 101, 102, 103.
encid int Unique identifier for the encounter associated with the medical data. Foreign key relation: `enc` table, `encounterID` column. Example: 1001, 1002, 1003.
surglist varchar A string containing the patient's surgery history, with each surgery separated by a delimiter. Example: "Appendectomy;Gallbladder removal".
hosplist varchar A string containing the patient's hospitalization history, with each hospitalization separated by a delimiter. Example: "St. Mary's Hospital;General Hospital".
medlist varchar A string containing the patient's medication list, with each medication separated by a delimiter. Example: "Aspirin;Lisinopril;Metformin".
allergylist varchar A string containing the patient's allergy list, with each allergy separated by a delimiter. Example: "Penicillin;Peanuts;Shellfish".
delflag tinyint A tiny integer indicating whether the record has been deleted (1) or not (0). Example: 0, 1.
modifieddate datetime A datetime value indicating the last time the record was modified. Example: '2022-01-01 12:34:56'.
