EHI Export Schema
Table dentalpttoothchartinfo
Description This table stores some patient-level details with respect to the dental module. It stores the list of teeth marked primary, missing, along with patient annotation notes and coordinates that correspond to free-hand drawing performed on the tooth chart.
Column DataType Description
id (Primary Key) int This is the auto-increment primary key column for the table.
patientid int This column stores the patient identifier for who the information is being stored. It is a reference to 'UID' column of the table users.
drawingsegments text This column stores JSON text which can be interpreted by the Raphael SketchPad JavaScript library that represents the free-hand drawings done on the tooth chart.
primaryteeth varchar This column stores a comma-separated list of PERMANENT TEETH that are marked as primary. Please note that no primary teeth are stored in this column, but the corresponding permanent teeth that are SET AS PRIMARY.
missingteeth varchar This column stores a comma-separated list of primary or permanent teeth that are marked missing.
notes text This column stores the patient annotation notes that are displayed on the tooth chart section of Dental Examination screen.
dentitionManuallySet tinyint This column is set to 1 if the dentition is manually set by the user. If programmatically set (example - based on patient age), the value is 0.
