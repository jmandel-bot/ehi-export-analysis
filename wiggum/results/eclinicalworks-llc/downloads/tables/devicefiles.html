EHI Export Schema
Table devicefiles
Description This table stores information about files associated with different devices, such as their names, types, and related test IDs.
Column DataType Description
id (Primary Key) int This is the primary key column that uniquely identifies each record in the table. It is an integer value that is automatically incremented with each new record. Example: 1, 2, 3, etc.
Testid int This column stores the test id associated with the device file. It is an integer value and has a default value of 0. This column can be used as a foreign key to reference another table with test information. Example: 101, 102, 103, etc.
FileName varchar This column stores the name of the device file. It is a variable character string with a maximum length of 50 characters and has a default value of an empty string. Example: "file1.jpg", "document2.pdf", "image3.png", etc.
FileType varchar This column stores the type of the device file. It is an enumerated type with three possible values: 'Fax', 'Image', and 'Print'. The default value is 'Image'. Example: 'Fax', 'Image', 'Print'.
