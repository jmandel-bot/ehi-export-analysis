EHI Export Schema
Table interventions
Description This table stores information about interventions performed during encounters.
Column DataType Description
encounterid int This columns stores the encounter id of enc table as reference.
itemid int This column stores the unique identifier for an intervention item. It is an integer value (e.g., 67890) and serves as a foreign key that refers to the items table's itemid column.
notes text This column stores any additional notes or comments about the intervention. It is a text field and can store any string value (e.g., Administered medication as prescribed).
