Table ip_edi_inv_pcscodes
Description This table stores information about invoice PCS (Procedure Coding System) codes, including the code itself, a description, and whether it is the primary code for the invoice.
Column DataType Description
id (Primary Key) int The unique identifier for each record in the table (e.g., 1, 2, 3).
invoiceid int The identifier for the invoice that this PCS code is associated with. Foreign key relation: `invoices.id`. Examples include 101, 102, 103, etc..
pCode varchar The PCS code itself. Example: 'A1234', 'B5678', 'C9012'.
description varchar A description of the PCS code (e.g., 'Knee replacement surgery', 'Appendectomy', 'Cataract surgery', etc.).
isPrimary tinyint A boolean flag indicating whether this PCS code is the primary code for the invoice (1) or not (0). Example: 1, 0.
pDate datetime The date when the procedure was performed (e.g., '2021-01-01', '2021-02-15', '2021-03-30').
displayIndex int The order in which the PCS codes should be displayed for the invoice (e.g., 1, 2, 3).
createdBy int The identifier for the user who created the record. Foreign key relation: `users.id` (e.g., 1, 2, 3).
createdOn datetime The date and time when the record was created (e.g., '2021-01-01 12:34:56', '2021-02-15 14:56:23', '2021-03-30 09:12:45').
deleteFlag tinyint A boolean flag that indicates whether the record has been deleted or not, represented by the values of 1 (deleted) and 0 (not deleted).
