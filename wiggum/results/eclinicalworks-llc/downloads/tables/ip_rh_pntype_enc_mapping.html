EHI Export Schema
Table ip_rh_pntype_enc_mapping
Description This table store the versions for encounter of progressnotes type.
Column DataType Description
id (Primary Key) int Auto increment primary key of the table. Data type defined as: INT with size 11
pnTypeId int This field stores progressnote type id from ip_rh_pntypes table.the pntype id stores the type of progress note configured in the admin with details of progress note links to be used for This encounter
encounterId int This field stores encounter id from enc table.This field stores the encounter id for which the linked pntype id is mapped. This maps the encounter with the configured progress note type.
versionMapId int This field stores version map id from ip_rh_pnsection_version_map table.This stores the latest version id for This progress note configured in the Admin.
