Table cmpgn_surveypatientresponse
Description This table stores the responses of patients to all questions in a survey.
Column DataType Description
id (Primary Key) int This is an auto increment id and primary key of table.
UID int User identifier of the patient who provided the response, foreign key relation to `users` table and `UID` column (e.g., 101, 102, 103).
apUID int This column stores apUID
surveyid int id of survey which patient has responded to.
campaignid int id of the campaign thru which survey has been sent to a patient
initialNotificationMessageid int This is id of message sent to a patient
msgUniqueidentifier varchar Unique identifier for the message (e.g., 'msg123', 'msg456', 'msg789').
pageid int id of survey page where question belongs which patient has responded
questionid int id of question which patient has responded
subQuestionid int id of sub question which patient has responded
optionid int id of option which patient has selected while responding to question in survey
responseText nvarchar Text of the patient's response, stored in utf8mb4 character set (e.g., 'Yes', 'No', 'Maybe').
commentText nvarchar Additional comment text provided by the patient, stored in utf8mb4 character set (e.g., 'I liked the service', 'It was too crowded', NULL).
deleteFlag tinyint This flag indicates if the configuration data stored in the table need to be honored or not. Possible values are 0 and 1.
inserttime datetime time stamp when the response was inserted into the table (e.g., '2021-01-01 12:34:56', '2021-02-02 23:45:01', '2021-03-03 14:30:00').
responseUniqueidentifier varchar Unique identifier for the patient's response to survey. Example: 'xyz123', 'uvw456', 'rst789'.
