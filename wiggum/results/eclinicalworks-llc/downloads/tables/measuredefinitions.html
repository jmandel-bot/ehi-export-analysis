EHI Export Schema
Table measuredefinitions
Description This table stores the definitions for CDSS measures.
Column DataType Description
id (Primary Key) int This column is auto incremented and primary key of This column .
Dictionary tinyint This column indicates an id of the dictionary to which this measure is linked. Possible values are: [ 1 - TCNY (Take Care New York - the default set of CDSS Measures) ], [ 2 - New York City Department of Corrections ], [ 3 - Rhode Island ], [ 4 - MCMP (No longer used) ], [ 5 - LMM ( No longer used) ], [ 6 - Baylor ], [ 7 - PQRI (Physician Quality Reporting Initiative - no longer used) ], [ 8 - CGHN (Central Georgia Health Network) ], [ 9 - eHearts ], [ 10 - Meaningful Use (No longer used) ], [ 11 - Point of Care (healow Insights) ]
Measureid varchar The unique id used by the application to identify the Measure (e.g. "101-OI").
MeasureName varchar The name of the Measure (e.g. "Depression Screening").
MeasureDesc text A description of the Measure.
MeasureNum text A description of the Measure's numerator (i.e. the criteria by which a patient is judged to be compliant with the Measure).
MeasureDen text A description of the Measure's denominator (i.e. the criteria by which the Measure is deemed relevant to a patient).
Deleteflag tinyint 1 if the Measure is deleted, 0 otherwise.
CDSSEnabled tinyint (Column no longer in use)
measureidlabel varchar A label for the Measure.
NumMeasurePeriodDays int (Column no longer in use)
DenMeasurePeriodDays int (Column no longer in use)
NumInclMeasurePeriodDays int (Column no longer in use)
groupid int (Column no longer in use)
measureref varchar The name of the reference file attached to the Measure.
citation varchar The bibliographic citation for the reference study associated with the Measure.
developer varchar The creator of the Measure.
fundingSource varchar The source of funding for the reference study associated with the Measure.
releaseVersion varchar The version number/date of the Measure.
exclusions varchar A description of the Measure's exclusions (i.e. criteria by which the Measure is deemed irrelevant to a patient).
