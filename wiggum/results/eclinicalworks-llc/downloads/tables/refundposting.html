EHI Export Schema
Table refundposting
Description This table stores information about refund postings to the claim CPT level.
Column DataType Description
Id (Primary Key) int Unique identifier for each refund posting. (e.g., 1, 2, 3)
ClaimRefundId int Identifier for the claim refund associated with the refund posting. Foreign key relation to the `ClaimRefund` table's `Id` column. (e.g., 101, 102, 103)
RefundID int This is the identifier for the refund associated with this CPT level refund. This column has a foreign key reference to 'refunds' table, 'RefundId' column
InvId int This column stores the Invoice Id associated with this line level refund. This column has a foreign key reference to 'edi_invoice' table, 'Id' column
LineItemId int This column stores the Invoice line item Id associated with this line level refund. This column has a foreign key reference to 'edi_inv_cpt' table, 'Id' column
amount decimal The amount of the refund posting in decimal format. (e.g., 100.50, 200.75, 300.25)
DeleteFlag int A flag indicating if the refund posting has been deleted (1) or not (0). (e.g., 0, 1)
AddedBy int Identifier for the user who added the refund posting. Foreign key relation to the `users` table's `uid` column
AddedTime datetime Date and time when the refund posting was added. (e.g., '2022-01-01 12:34:56', '2022-01-02 14:30:00')
UpdatedBy int Identifier for the user who last updated the refund posting. Foreign key relation to the `users` table's `uid` column
UpdatedTime datetime Date and time when the refund posting was last updated. (e.g., '2022-01-03 16:45:30', '2022-01-04 18:00:00')
closingId int This column denotes the hard close Id associated with this line item refund if the refund was hard closed. It has a foreign key reference of 'closingtransactions' table, 'closingId' column
