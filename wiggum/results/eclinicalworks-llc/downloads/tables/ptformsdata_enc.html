EHI Export Schema
Table ptformsdata_enc
Description This table stores the status details of the forms assigned to the patient by the third party vendor Chadis and Rothman.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key of this column.
encounterid int This column stores the encounter id which is the primary key (encounterid column) of enc table.
ecwpropid int This column stores the community id of the questionnaire form which is the primary key (ecwPropid column) of ecwproperties table.
remainingquestions varchar This column stores the status of the questionnaire form. 0 - Complete, Any other value - Pending
deleteflag tinyint This column stores value 0/1 indicating if the patient forms data is deleted, this is soft delete. 0 - Not Deleted , 1 - Deleted.
