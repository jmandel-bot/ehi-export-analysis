EHI Export Schema
Table vitalshistory
Description This table stores the history of vital sign measurements for patients during their encounters.
Column DataType Description
id (Primary Key) int This is auto-incremented primary key in This column .
encounterid int identifier for the encounter during which the vital sign was measured. This is a foreign key that refers to the `enc` table and its `encounterid` column. Example: 101, 102, 103.
vitalid int This column stores itemid associated with name of the vitals being documented during the encounter. This column refers to itemid column in items table. Mapping of vital names and itemids are stored in vitaltypes table.
propid int This columns stores itemid to get unit of measurement (UOM). This column refers to itemid column in items table. Mapping of itemid and propid is stored in itemdetail table.
Value text This column stores values with/without HTML tags for different vitals provided by end users during the encounter.
UpdatedBy int This column stores user id of the provider/staff who updated/added vitals information for the encounter. This column refers to UID column in user table.
UpdatedTime datetime This column stores time stamp when each individual vital was added and updated by provider/staff for an encounter. time stamp is stored in 'YYYY-MM-DD HH:mm:ss.sss' format.
displayorder int This column stores order in which vitals are shown in the user interface.
deleteflag tinyint This columns stores 0/1 values to indicate whether the vital was deleted or not. 0 - Not Deleted vital 1- Deleted Vital.
value2 varchar This column stores values for different vitals provided by end users during the encounter. This column does not store HTML tags with values in this column.
nottaken tinyint Tiny integer value to indicate if the vital sign was not taken (1) or taken (0). Example: 0, 1.
exceptionid int identifier for any exception related to the vital sign measurement. This is a foreign key that refers to the `vital_exceptionlist` table and its `id` column. Example: 1 (patient refused), 2 (equipment malfunction).
updatedTimeinUTC datetime time stamp in UTC of when the record was last updated. Example: "2021-09-01 08:34:56".
