Table ip_rh_ptStagingADTOrders
Description The ip_rh_ptStagingADTOrders table relates to the staging of ADT (Admission, Discharge, Transfer) orders in a healthcare context. Below is a description of its columns:
Column DataType Description
id (Primary Key) int This is an auto-incrementing integer column, serving as the primary key. It uniquely identifies each record in the table.
stagingOrderId int This is an integer FK column of table ip_rh_ptstagingorders PK column stagingOrderId. It relates to the ID of the staging order associated with this ADT order.
attendingProviderId int This FK that refers uid column in users.
dischargeDispositionId int This is an integer column with a default value of NULL. It represents the ID of the discharge disposition for the patient. It's FK that refers id column in ip_rh_dict_disposition table.
dischargeProviderId int This is an integer column with a default value of NULL. It represents the ID of the healthcare provider who discharged the patient. FK that refers to uid in users table.
patientWaitingFor text This is a VARCHAR(255) column. It holds information about what the patient is waiting for.
dischargeContingencies text This is a VARCHAR(255) column. It may store descriptive information about any contingencies or special considerations related to the patient's discharge.
admittingProviderId int This is an integer column with a default value of NULL. It represents the ID of the healthcare provider who admitted the patient. This is FK and refer column uid from users.
reasonForTransfer text This is a VARCHAR(255) column. It may store descriptive information about the reason for the patient's transfer.
doNotCreateSubEncFlag tinyint This is a tinyint column with a default value of 0 and it cannot be NULL. This is a flag to indicate whether or not to create a sub-encounter for this order.
admitDateTime datetime This field is a datetime data type and can be NULL. It represents admit DateTime.
anticipatedDischargeDate date This field is a date data type and can be NULL. It represents anticipatedDischarge Date.
dischargeDateTime datetime This field is a datetime data type and can be NULL. It represents discharge DateTime.
deleteFlag tinyint This is a tinyint, a smaller range integer. It cannot be NULL and has a default value of 0. This column indicates whether a staging ADT order has been marked for deletion or not. deleteFlag value 0 mark as not deleted and value 1 mark as deleted.
