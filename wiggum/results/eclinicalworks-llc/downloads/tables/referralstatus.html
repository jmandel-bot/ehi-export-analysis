EHI Export Schema
Table referralstatus
Description This table stores a master list of different statuses available for referral.
Column DataType Description
id (Primary Key) int This is auto-incremented primary key in This column .
status varchar This is a varchar(50) column that stores the status description of the referral. It has a default value of an empty string (''). Example: 'Pending', 'Approved', 'Rejected', etc.
deleteFlag tinyint This column stores flag to enable and disable substatus. 0 - status is enabled, 1 - status is disabled
statusType varchar This is an enum column that stores the type of referral status. It can have one of the following values: 'Open', 'Consult Pending', 'Insurance Auth', 'Internal Review', 'Addressed', or 'To Be Reconciled'. Example: 'Open', 'Consult Pending', etc.
