EHI Export Schema
Table crc_claimalertfiles
Description This table stores information about claim alert files, including their import date, the user who imported them, and the contents of the files in JSON format.
Column DataType Description
fileid (Primary Key) int This is the primary key and an auto-incrementing integer that uniquely identifies each claim alert file. Example: 1, 2, 3, etc.
filetimefield varchar This is a varchar column that stores the time field associated with the claim alert file. Example: "2021-08-01 12:34:56".
importdate datetime This is a datetime column that stores the date and time when the claim alert file was imported into the system. Example: "2021-08-01 12:34:56".
importedby int This is an integer column that stores the id of the user who imported the claim alert file. It can be a foreign key referring to the user's id in a separate user table. Example: 1, 2, 3, etc.
jsonstring text This is a longtext column that stores the contents of the claim alert file in JSON format. Example: '{"claim_id": 123, "alert_type": "high_risk", "details": "This claim has a high risk of fraud."}'
