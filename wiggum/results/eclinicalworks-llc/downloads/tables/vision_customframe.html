Table vision_customframe
Description This table stores frame information used in the spectacle pair.
Column DataType Description
id (Primary Key) int Unique identifier for each custom frame entry. (e.g., 1, 2, 3)
pairid int This column is an identifier for the pair of frames (e.g., 10, 25, 42).
product_type int identifier for the type of product (e.g., 1, 2, 3).
frame_type int identifier for the type of frame. Foreign key relation with `inventory_vframe_type` table's `id` column. (e.g., 1, 2, 3).
color varchar This column contains the color of the custom frame (e.g., 'Black', 'Blue', 'Red').
gender tinyint Gender category for the custom frame, stored as an unsigned tiny integer (e.g., 0 for unisex, 1 for male, 2 for female).
description text Long text description of the custom frame.
width decimal Width of the custom frame in decimal format (e.g., 12.50, 15.25, 18.00).
height decimal Height of the custom frame in decimal format (e.g., 10.50, 12.25, 14.00)
maxlength decimal Maximum length of the custom frame in decimal format (e.g., 20.00, 22.50, 25.00).
distbetweenlens decimal Distance between the lenses of the custom frame (e.g., 10, 12, 14).
Materialid int This column is an identifier for the material of the custom frame. Foreign key relation with `inventory_vframe_mat` table's `id` column. (e.g., 1, 2, 3)
Style varchar This column contains the style of the custom frame (e.g., 'Classic', 'Modern', 'Vintage').
Temple int This column contains te length of the temple (arm) of the custom frame (e.g., 120, 130, 140).
isdeleted bit This column is a bit value indicating whether the custom frame entry has been deleted (1) or not (0).
isInventoryAttr bit This column is a bit value indicating whether the custom frame entry is an inventory attribute (1) or not (0).
