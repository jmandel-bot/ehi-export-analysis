EHI Export Schema
Table ordersetlabinstructions
Description This table stores the lab instructions for each order set item in a specific encounter.
Column DataType Description
id int Unique identifier for each record in the table. Example: 1, 2, 3
Encounterid int identifier for the encounter associated with the order set lab instructions. This is a foreign key that refers to the `Encounter` table's `id` column. Example: 101, 102, 103
Itemid int identifier for the order set item associated with the lab instructions. This is a foreign key that refers to the `OrderSetItem` table's `id` column. Example: 1001, 1002, 1003
OrderSetEncid int identifier for the order set encounter associated with the lab instructions. This is a foreign key that refers to the `OrderSetEncounter` table's `id` column. Example: 5001, 5002, 5003
LabInstructions text Text column containing the lab instructions for the order set item in the encounter. Example: "Collect blood sample in a red-top tube.", "Fasting required for 12 hours before test."
