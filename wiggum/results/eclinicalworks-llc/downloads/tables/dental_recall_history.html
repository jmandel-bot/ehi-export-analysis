EHI Export Schema
Table dental_recall_history
Description This table stores historical information of dental recalls and this information is used to display in the Progress Notes > CDSS > Dental tab.
Column DataType Description
id (Primary Key) bigint This column is auto-incremented primary key in This column . Unique id is assigned to each Dental Recalls historical data when user add history for any recall on CDSS> Dental Tab. It is defined as NOT NULL constraint.
Patientid int This column stores information of patient id for patient specific dental recalls which is used in CDSS> Dental tab screen.
Recallid int This column stores identifier number of Dental Recalls. It refers to column id from dental_recall table.
Procedureid bigint This column stores identifier number of Procedure id. It refers to column procNum from ProcedureHistory table.
LastDoneDate datetime This column stores Last Done Date of Dental Recall.
LastDoneBy int This column stores user id value which indicates user details who Added historical information of Dental recall.
UpdatedDate datetime This column stores Updated Date of Dental Recall historical data.
DeleteFlag tinyint A flag indicating whether the dental recall history record has been deleted. It is of tinyint(1) data type and has a default value of '0'. Example: 0 (not deleted), 1 (deleted).
RecallStatus tinyint This column stores Dental Recall status value. SUPPRESSED = -1 COMPLIANT = 1 PENDING = 2; NONCOMPLIANT = 3 PENDING_OTHER = 4; ALL = 5;
Itemid int This column stores itemid of Dental Recall (Associated itemid column value on dentalitems table).
DentalItemid int This column stores dentalitemid of Dental Recall (Associated id column value on dentalitems table).
