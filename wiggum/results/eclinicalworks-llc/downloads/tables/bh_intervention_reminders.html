Table bh_intervention_reminders
Description This table stores information about intervention reminders for patients, including the task, due date, and status of the reminder.
Column DataType Description
interReminderid (Primary Key) int Unique identifier for each intervention reminder. Example: 1, 2, 3.
structMappingid int identifier for the structure mapping of the intervention reminder. Foreign key relation with `bh_treatment_plan_pgoid_mapping` table's `id` column. Example: 101, 102, 103.
structid int identifier for the structure of the intervention reminder. Foreign key relation with `bh_treatment_plan_dictionary` table's `id` column. Example: 201, 202, 203.
patientid int identifier for the patient associated with the intervention reminder. Foreign key relation with `users` table's `UID` column. Example: 301, 302, 303.
reminderTask varchar Task description for the intervention reminder. Example: "Take medication", "Attend therapy session", "Complete exercise routine".
reminderDueDate datetime Due date for the intervention reminder. Example: '2022-12-01', '2022-12-15', '2023-01-01'.
reminderDueTime datetime Due time for the intervention reminder. Example: '08:00:00', '14:30:00', '18:45:00'.
reminderStatus tinyint Status of the intervention reminder, where 0 is not completed and 1 is completed. Example: 0, 1.
addedByUserid int identifier for the user who added the intervention reminder. Foreign key relation with `users` table's `UID` column. Example: 401, 402, 403.
addedDateTime datetime Date and time when the intervention reminder was added. Example: '2022-11-01 10:30:00', '2022-11-15 14:45:00', '2022-12-01 09:00:00'.
deleteFlag tinyint Flag indicating whether the intervention reminder has been deleted, where 0 is not deleted and 1 is deleted. Example: 0, 1.
