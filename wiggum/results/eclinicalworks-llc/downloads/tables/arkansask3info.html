EHI Export Schema
Table arkansask3info
Description This table stores information related to medical records of patients, including tumor size, stage, and treatment indicators.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
InvCptId int Unique identifier for the inventory capture. Example: 101, 102, 103.
InvId int Unique identifier for the inventory. Example: 1001, 1002, 1003.
ResultCodeBP char A 2-character code representing the result of a specific test or procedure related to the patient's condition. Example: 'A1', 'B2', 'C3'.
ResultCodeCP char Another 2-character code representing the result of a different test or procedure related to the patient's condition. Example: 'D1', 'E2', 'F3'.
RecommdtnCodeBP char A 2-character code representing a recommendation based on the ResultCodeBP. Example: 'G1', 'H2', 'I3'.
RecommdtnCodeCP char A 2-character code representing a recommendation based on the ResultCodeCP. Example: 'J1', 'K2', 'L3'.
ShortTermFollowup char A 2-character code indicating whether a short-term follow-up is required for the patient. Example: 'Y1', 'N2'.
TumorSize char A 3-character code representing the size of the tumor. Example: 'S01', 'M02', 'L03'.
TumorStage char A 1-character code representing the stage of the tumor. Example: 'A', 'B', 'C'.
TreatmntInd char A 1-character code indicating whether treatment is indicated for the patient. Example: 'Y', 'N'.
ReasonNoTreatmnt char A 1-character code representing the reason for not providing treatment, if applicable. Example: '1', '2', '3'.
PapAdequacyCode char A 1-character code representing the adequacy of a Pap test, if applicable. Example: 'A', 'B', 'C'.
deleteFlag smallint A small integer (2) representing whether the record has been deleted or not. '0' means not deleted, and '1' means deleted. Example: 0, 1.
