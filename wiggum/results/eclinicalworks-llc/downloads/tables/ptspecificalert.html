EHI Export Schema
Table ptspecificalert
Description This table stores Patient-Specific Alerts.
Column DataType Description
id (Primary Key) int This column is an auto incremented Primary Key for This column .
patientid int This column stores patient's id for whom alert is created. This column refers column UID from user table.
Itemid int This column stores itemid for the corresponding lab/DI/Procedure/Billing/Dental.This column refers column itemid from items table. If Other is selected int the user interfaceit inserts 0 as a value into this column.
Name varchar This column stores the item name selected to create a alert. In that case it refers to column name from items table. If Other is selected while creating alert in that case it stores the name (free text) given by the end user.
endDt datetime This column stores the date till the alert is complaint. The format used is YYYY-MM-DD HH:MM:SS.
frequency int This column stores the frequency of the alert created.
freqFlag varchar This column stores one of the below values indicating the frequency unit. Y - Years, M - Months, W - Weeks and D - Days.
alertType varchar This column stores one of the below values indicating type of the alert. Values: BillingAlert, Immunizations, Labs, Other, Procedures, XRays.
Providerid int This column is not used anymore.
ProviderName varchar This column is not used anymore.
IsRecurrent tinyint This column stores a value 0 or 1. 1 - indicating a recurring alert or 0 - Otherwise.
Recurrencefrequency int This column stores the frequency of the alert created in case if isRecurrent column is set to 1.
RecurrenceFreqFlag varchar This column stores one of the below values indicating the frequency unit in case if isRecurrent column is set to 1. Y - Years, M - Months, W - Weeks and D - Days.
additionalitems varchar This column stores itemid for the corresponding lab/DI/Procedure/Immunization additional item added while configuring a patient specific alert. This column refers column itemid from items table.
alert_description varchar This column stores description (free text) added by end user for this alert.
orderSetEncid int This column is not used anymore.
orderSetsMeasureid varchar This column is not used anymore.
