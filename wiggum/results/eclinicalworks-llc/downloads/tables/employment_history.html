Table employment_history
Description This table stores the employment history records of users, including information about projects, job sites, job categories, job titles, and hourly rates.
Column DataType Description
id (Primary Key) int The column stores an id of the employment history. This column is an auto incremental primary key.
Project varchar This column stores the department name as a project for added employer to patient information.
JobSite varchar This column stores the job title location as a job site for an added employer to patient information.
JobCategory varchar This column stores the category as a job category for an added employer to patient information.
JobTitle varchar This column stores the title as a job title for an added employer to patient information.
deleteFlag int This column stores integer value of 0 or 1 for delete flag. A "0" means the record is active, and a "1" means the record is deleted.
userid int This column stores the identifier number of a logged-in user. It refers to column UID from the user table.
insuranceid int It is used to store insurance ids. This column refers to the ins id column in the insurance table.
Hourly_Rate varchar This column stores the rate of hourly working from an employer to patient information.
projectid int This column stores the project id as an integer for an added employer to patient information.
jobSiteid int This column stores the site id as an integer for an added employer to patient information.
jobCategoryid int This column stores the category id as an integer for an added employer to patient information.
jobTitleid int This column stores the title id as an integer for an added employer to patient information.
