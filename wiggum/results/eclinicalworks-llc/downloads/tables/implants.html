EHI Export Schema
Table implants
Description This table stores patient's implants/explants related information.
Column DataType Description
id (Primary Key) int This is auto-incremented primary key in This column .
encounterid int This column stores id of an encounter for which implant is being documented. This column refers to encounterid column in enc table.
implantType varchar This column stores type of surgery being performed during the encounter. Two possible values are Implant and Explant.
udi varchar This column stores Unique Device identifier (UDI) number documented by end user on User Interface. UDI number is from Global Unique Device identification Database approved by FDA.
mfgCatid varchar This column stores Manufacturer Category identifier. This is distinct identification code for a human cell, tissue, or cellular and tissue-based product(HCT/P) regulated as a device.
modelNo varchar The column store manufacturer device model number of being implanted/explanted.
lotNo varchar This column stores Lot or batch number in which the device being implanted/explanted was manufactured by the manufacturer.
deviceSnomedCode varchar This column stores Systematized Nomenclature of Medicine -- Clinical Terms (SNOMED CT) code mapped to the GMDN PT Name for the device being implanted/explanted. SNOMED CT is the most comprehensive clinical terminology in use around the world.
cptCode varchar This column stores Current Procedural Terminology (CPT) code for the procedure being performed to implant/explant the device.
size varchar This column stores size of the implant/explant.
quantity int This column stores number of devices being implanted/explanted during current encounter.
itemDesc text This column stores description documented by end user of an item being implanted/explanted.
mfgName varchar This column stores name of the manufacturer of the device being implanted/explanted.
serialNo varchar This column stores serial number of the device being implanted/explanted.
mfgDate datetime This column stores date when the device being implanted/explanted was manufactured.
expiryDate datetime This column stores date when device being implanted/explanted might expire.
implantedDate datetime This column stores date when device was implanted.
mriSafetyInfo text This column stores end user documented information on MRI related safety from the device label.
siteDesc text This column stores details on the site on the body of the implant/explant for the patient.
comments text This column stores free text notes documented by end user under comments section in the User Interface.
isSMDA tinyint This column stores values 0 or 1 to indicate if device is marked for Safe Medical Device Act (SDMA) or not. 0 - If not marked for SDMA, 1 - if marked for SDMA.
isLatexInfo tinyint This column stores values 0 or 1 to indicate whether device is labeled as containing natural rubber latex or dry natural rubber. 0 - Latex Info is not Labeled, 1 - Latex Info is Labeled for the device.
isExpiryDateNA tinyint This column stores values 0/1 to indicate whether expiry date is applicable or not for the device being implanted/explanted. 0 - If Expiry Date is Applicable, 1 - If Expiry Date is Not Applicable.
gmdnPtName varchar This column stores attribute code associated with the device identifier (UDI), obtained from GUDid. Global Medical Device Nomenclature (GMDN) is a system of internationally agreed descriptors used to identify medical device products and is managed by GMDN Agency.
deleteFlag tinyint This column stores values 0 or 1 to indicate whether the implant/explant was deleted. 0 - If implant/explant is not deleted, 1 -If implant/explant is deleted.
withUdi tinyint This column stores values 0/1 to indicate whether Implant is with or without UDI registry. 0 - Without UDI Registry, 1 - with UDI Registry
barcode varchar This column stores UDI of the device being implanted using barcode when "UDI Registry" radio button on user interfaceis selected.
snomedCTName varchar This column stores Systematized Nomenclature of Medicine -- Clinical Terms (SNOMED CT) Name mapped to the GMDN PT Name for the device being implanted/explanted. SNOMED CT is the most comprehensive clinical terminology in use around the world.
explantDate datetime This column stores date when device was explanted.
brandName varchar This column stores brand name of the device being implanted/explanted.
issuingAgency varchar This column stores abbreviation of the agency which issued the UDI for the device being implanted/explanted. Three issuing agencies are: GS1 Issuing Agency, Health Industry Business Communications Council (HIBCC), or International Council for Community in Blood Banking Automation (ICCBBA).
donationid varchar This column stores donation id in case the device being implanted is a donation.
