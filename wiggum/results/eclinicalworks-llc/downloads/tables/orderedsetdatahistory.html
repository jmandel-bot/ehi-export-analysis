EHI Export Schema
Table orderedsetdatahistory
Description This table contains order set data history for order set encounters
Column DataType Description
id (Primary Key) int This column stores the auto-incremented primary key.
OrderSetEncid int This column stores the encounter ID assigned to the Order Set created by the end user. This column refers to the encounterid column from the enc table.
PtEncid int This column stores an ID of the encounter of a patient. This column refers to the encounterid column from the enc table.
Ptid int This column stores the identifier number of the patient for whom the appointment is created. It refers to the UID column from the user table.
Itemid int This column stores numeric type data. It refers to itemid column of items table.
OrderType int This column stores the type of the order (e.g., 1 for lab, 2 for DI, 7 for immunization).
Status varchar This column stores the order status selected from the UI (e.g., Refusal, Contraindication, etc.).
Comments text This column stores the comments added while selecting the reason for an order.
OrdDate datetime This column stores the order date in YYYY-MM-DD format.
Result varchar This column stores the result value in string format.
deleteFlag tinyint This column stores an integer indicating whether the record has been deleted (1) or not (0).
ItemOrdered tinyint This column stores an integer indicating whether the item has been ordered (1) or not (0).
virtualReportid int An integer representing the virtual report id associated with the order. This column value can be set when historical immunization is created for order.
suppressed int This column stores an integer indicating whether the record has been suppressed (1) or not (0).
ComponentItemid int This column stores the order's item ID and refers to item id column in the item table.
modifiedTime datetime This column stores the date and time when the record was last modified (Example: 2021-09-01 12:34:56, 2021-09-02 14:56:23).
statusCode varchar This column stores the order’s status code (e.g., RES, HIS, etc.).
