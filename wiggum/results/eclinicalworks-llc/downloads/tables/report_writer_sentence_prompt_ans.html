Table report_writer_sentence_prompt_ans
Description This table stores the answers for sentence prompts in a report writer, along with their associated sentence, template, and report information.
Column DataType Description
id (Primary Key) int This is the primary key, an auto-incrementing integer that uniquely identifies each answer in the table. Example: 1, 2, 3, etc.
ans varchar This column is used to store answers of prompt for given sentence. The default value of this column is empty because this field is not mandatory in UI. As per requirement, user can add up to 2000 characters from UI. Hence, varchar(2000) is suitable here.
sentenceid int This column is used to store the id of report_writer_sentence table. Each prompt ans should be associated with particular sentence. Hence, not NULL constraint is suitable here. This column is a foreign key that references to report_writer_sentence (sentenceid). Sentenceid column of report_writer_sentence is INT type. Hence, INT datatype is suitable here.
templateid int This column is used to store the id of report_writer_template table. Each prompt ans should be associated with particular template. Hence, not NULL constraint is suitable here. This column is a foreign key that references to report_writer_template (templateid). Templateid column of report_writer_template is INT type. Hence, INT datatype is suitable here.
reportid int This column is used to store reportid of labdata table. Report writer result can associate with other entities apart from reportid. Hence, added this column as NULL. This column is a foreign key that references to labdata (reportid). Reportid column of labdata is INT type. Hence, INT datatype is suitable here.
promptid int This column is used to store the id of smartnote_prompt table.  Each prompt ans should be associated with particular prompt. Hence, the default value of this column is not NULL. This column is a foreign key that references to smartnote_prompt (id). id column of smartnote_prompt is INT type. Hence, INT datatype is suitable here.
