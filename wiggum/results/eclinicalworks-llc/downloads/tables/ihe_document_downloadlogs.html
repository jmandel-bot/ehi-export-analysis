EHI Export Schema
Table ihe_document_downloadlogs
Description This table stores information about the download logs of IHE (Integrating the Healthcare Enterprise) documents, including the document ID, encounter ID, and the time of download.
Column DataType Description
id (Primary Key) int This is the primary key, an auto-incrementing integer that uniquely identifies each download log entry. Example: 1, 2, 3, etc.
DocId int This is an integer that represents the ID of the IHE document being downloaded. Example: 101, 102, 103, etc.
encounterId int This is an integer that represents the ID of the encounter associated with the document download. It can be a foreign key referring to the encounter table's primary key (e.g., `encounters.id`). Example: 1001, 1002, 1003, etc.
downloadTime datetime This is a time stamp that records the exact date and time when the document was downloaded. Example: '2021-09-01 12:34:56'
