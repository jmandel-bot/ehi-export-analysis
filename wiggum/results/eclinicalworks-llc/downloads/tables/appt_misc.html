Table appt_misc
Description This table stores miscellaneous information related to appointments, such as additional labels and their values.
Column DataType Description
id (Primary Key) int This is the primary key for the table, which uniquely identifies each record. It is an integer with a maximum length of 11 digits and is auto-incremented. Example: 1, 2, 3, etc.
Encounterid int This is a foreign key that refers to the `encounterID` column in the `enc` table. It represents the unique identifier of the encounter associated with appt_misc. Example: 101, 102, 103, etc.
Labelid int This column stores the id forappt_miscinfo_labels table. It is an integer with a maximum length of 11 digits and has a default value of 0. Example: 1, 2, 3, etc.
LabelValue varchar This column stores the value of the label associated with the miscellaneous information. It is a variable character string with a maximum length of 50 characters and has a default value of an empty string. Example: "Urgent", "Follow-up", "Routine", etc.
DeleteFlag int This column indicates whether the record has been deleted or not. It is an unsigned integer with a maximum length of 11 digits and has a default value of 0. A value of 0 means the record is not deleted, while a value of 1 means the record is deleted. Example: 0, 1.
