EHI Export Schema
Table vascular_finding
Description This table stores added, updated, or deleted findings for the Vascular module.
Column DataType Description
id (Primary Key) int This column is auto-incremented primary key in This column .
findingiconid int This column stores identifier number of a Finding icon. It refers to column id from vascular_finding_icon table.
findingname varchar This column stores Name for the Finding.
findingtype tinyint This column stores View type value 1(Vein) , 2(Skin), 0(Both).
createdby int This column stores identifier number of a user who created the finding. It refers to column UID from user table.
createddate datetime This column stores the current time stamp when the Finding is created. The format used is YYYY-MM-DD HH:MM:SS.
modifiedby int This column stores identifier number of a user who modified the finding. It refers to column UID from user table.
modifieddate datetime This column stores the current time stamp when the Finding is modified. The format used is YYYY-MM-DD HH:MM:SS.
color varchar This column stores configured color for the Finding.
deleteflag tinyint This column stores value 0 / 1 indicating this finding is deleted. This is soft delete. 0 - Not Deleted , 1 -Deleted
displayIndex int This column stores display Index for configured Finding. This index will be used to sort finding while displaying Finding List.
abberivation varchar This column stores abbreviation for Finding.
STATUS tinyint This column stores value 0 / 1 indicating this Finding is activate or Inactive. 0 - Active , 1 -Inactive
