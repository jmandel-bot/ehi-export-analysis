Table ptorders_ack_mapping
Description This table stores the acknowledgment mapping information for orders.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
ackid int The acknowledgment id associated with an order. Example: 101, 102, 103.
ackBy int The user id of the person who acknowledged the order. This can be a foreign key referring to the user's id in the `users` table (if exists). Example: 1001, 1002, 1003.
ackFlag tinyint A boolean flag indicating whether the order has been acknowledged or not. '0' means not acknowledged, and '1' means acknowledged. Example: 0, 1.
ackDateTime datetime The date and time when the order was acknowledged. Example: '2021-09-15 14:30:00', '2021-09-16 10:15:00'.
deleteFlag tinyint A boolean flag indicating whether the record has been deleted or not. '0' means not deleted, and '1' means deleted. Example: 0, 1.
