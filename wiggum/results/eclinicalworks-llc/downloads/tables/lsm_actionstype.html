EHI Export Schema
Table lsm_actionstype
Description This table stores information about different types of actions that can be performed in the system, along with their display order, deletion status, and other related attributes.
Column DataType Description
Id (Primary Key) int This column is a primary key for table.
ActionType varchar This column stores and indicates patient action type.
DisplayIndex int This column stores index number value to display.
DelFlag tinyint This column indicates flag for action type record is deleted (1) or not (0).
DentalFlag tinyint Column stores flag to indicate whether the action stored in this master table is related to Dental module or not. 0 - Not related to Dental, 1 - Action is for Dental module
ActionTypeColor varchar This column stores a string representing the color code associated with the action type. For example, "#FF0000" for red, "#00FF00" for green, etc.
