EHI Export Schema
Table itemdetail
Description This table stores the details of items with their properties and values.
Column DataType Description
itemID (Primary Key) int This is an integer column that represents the unique identifier for an item. It is a primary key and cannot be NULL. Example values: 1, 2, 3, etc.
propID (Primary Key) int This is an integer column that represents the unique identifier for a property associated with an item. It is a primary key and cannot be NULL. Example values: 1, 2, 3, etc.
VALUE varchar This is a varchar(1000) column that stores the value of the property associated with the item. It can be NULL. Example values: "Red", "Large", "100 Watts", etc.
