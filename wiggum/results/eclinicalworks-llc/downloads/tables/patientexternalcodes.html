EHI Export Schema
Table patientexternalcodes
Description This table stores information about external patient codes and their relationships with interfaces and patients in the system.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto incrementing integer value. It uniquely identifies each record in the table. Example: 1, 2, 3, etc.
interfaceid int This is an integer value representing the id of the interface associated with the external patient code. Example: 101, 102, 103, etc.
eCWPatientid int This is an integer value representing the id of the patient in the system. It refers to the primary key of another table that stores patient information (e.g., patients.id). Example: 1001, 1002, 1003, etc.
externalPatientid varchar This is a varchar(30) value representing the external patient id from the interface. It stores the unique identifier of the patient in the external system. Example: EXT123, EXT456, EXT789, etc.
interfaceType int This is an integer value representing the type of interface being used. Example: 1 for HL7, 2 for FHIR, etc.
