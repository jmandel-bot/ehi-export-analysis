Table asc_discharge_medication_mapping
Description This table is used to store discharge medication mapping with an existing home medication status. When the provider clicks continue, the table will store existing home medication status with rxorderno.
Column DataType Description
id (Primary Key) int This is the primary key of the table, an integer value that uniquely identifies each record in the table. It is an auto-incrementing value, meaning it will automatically increase for each new record added. Example: 1, 2, 3, etc.
rxorderno varchar This column stores order number for discharge medications. if existing home medication is added in discharge medication, then same rx order no is used. if new drug is added, then new rx order number is generated
encounterid int This column stores id of an encounter for which medications are being documented in This column . This column refers to encounterid column in enc table.
homemedstatus smallint This column store home medication while performing discharge reconciliation from home mediation. Status id will be added as Continue: 2, Stop: 0,Hold: 100, Refill: from itemkey-Refillid
