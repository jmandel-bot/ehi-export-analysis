EHI Export Schema
Table rpm_device_measure
Description This table stores information about different device measurements in the RPM (Remote Patient Monitoring) system.
Column DataType Description
id (Primary Key) int Contains auto incremented id, datatype is INT (11). This column stores integer value which could grow with number of transactions over the years hence integer(Range: 0-2,147,483,647) is selected. It is AUTO_INCREMENT column and has PRIMARY KEY index
measure_name varchar This column contains measurement names like kg, lbs, cm which is primarily used to denote device readings with its units(measurement type). We have chosen datatype as VARCHAR(30). This column contains the name of the device measures.This column stores characters which cannot exceed 30 characters hence varchar(30) is selected. This field will always have a value hence defined as NOT NULL .
description varchar This column contains in depth description of measurement_name. The column chosen is datatype VARCHAR(255). This column contains the description of the measures defined. This column stores characters which cannot exceed 255 characters hence varchar(255) is selected. This field will always have a value hence defined as NOT NULL.
deleteFlag tinyint Flag indicates if this measurement is deleted or not.The datatype chosen is tinyint. This column stores binary flag i.e 0/1. 0: not deleted 1: deleted - By default the measure is always available for setup.Hence default 0 is chosen for this column
