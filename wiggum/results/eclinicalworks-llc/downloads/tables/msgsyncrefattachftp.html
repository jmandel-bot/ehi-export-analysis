EHI Export Schema
Table msgsyncrefattachftp
Description This table stores information about attachments related to medical records, such as medical summaries, Progress Notes, lab reports, and other documents, along with their remote directory locations and reference request IDs.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
refReqId int This column stores id of am incoming referral. This column refer to refreqid column in referral table.
remoteparentdirectory varchar Remote parent directory where the attachments are stored. Example: '/parent_directory/', '/medical_records/'.
remoteuserdirectory varchar Remote user-specific directory where the attachments are stored. Example: '/user_directory/', '/patient_files/'.
medicalsummary varchar File name of the medical summary attachment. Example: 'summary1.pdf', 'summary2.docx'.
progressnotes varchar File name of the progress notes attachment. Example: 'progress1.pdf', 'progress2.docx'.
labreports varchar File name of the lab reports attachment. Example: 'labreport1.pdf', 'labreport2.docx'.
xrays varchar File name of the X-ray attachment. Example: 'xray1.jpg', 'xray2.png'.
documents varchar File name of other documents attachment. Example: 'document1.pdf', 'document2.docx'.
progressnoteImages varchar File name of the progress note images attachment. Example: 'progress_image1.jpg', 'progress_image2.png'.
labattachments varchar File name of the lab attachments. Example: 'labattachment1.pdf', 'labattachment2.docx'.
xrayattachments varchar File name of the X-ray attachments. Example: 'xrayattachment1.jpg', 'xrayattachment2.png'.
ccr varchar File name of the Continuity of Care Record (CCR) attachment. Example: 'ccr1.xml', 'ccr2.xml'.
c32doc varchar File name of the C32 document attachment. Example: 'c32doc1.xml', 'c32doc2.xml'.
injuries varchar File name of the injuries attachment. Example: 'injury1.pdf', 'injury2.docx'.
careplans varchar File name of the care plans attachment. Example: 'careplan1.pdf', 'careplan2.docx'.
specialtyforms varchar File name of the specialty forms attachment. Example: 'specialtyform1.pdf', 'specialtyform2.docx'.
reqAPUID int Request APU ID associated with the attachments. Foreign key relation to the table and column it refers to is not provided in the given schema. Example: 201, 202, 203.
attachmentscount varchar Not Available
sourceccdaid varchar Source CCDA ID associated with the attachments. Example: 'ccda1', 'ccda2', 'ccda3'.
