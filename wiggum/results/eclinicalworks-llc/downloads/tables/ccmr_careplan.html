Table ccmr_careplan
Description This table contains all the information related to ccmrid. Ccmrid is a unique ID created for all patients used for CCMR modules.
Column DataType Description
ccmrID (Primary Key) int This column in used to generate the ccmrId for every patient.
PatientID int Identifier for the patient associated with the care plan. This is a foreign key that refers to the `PatientID` column in another table (e.g., `patients`). Example: 101, 102, 103.
Discharge_date datetime This column is not used anymore.
CPStat char This column is not used anymore.
LastScreen int This column is not used anymore.
Lock_Ind int This column is not used anymore.
DelStat int Indicator for whether the care plan is deleted (1) or not deleted (0). Example: 0, 1.
Uname varchar This column is not used anymore.
Upd_Date datetime time stamp of the last update made to the care plan. Example: '2021-08-15 12:34:56', '2021-08-16 14:23:45', '2021-08-17 09:15:30'.
AsmtID int This column is not used anymore.
AsmtValue varchar This column is not used anymore.
