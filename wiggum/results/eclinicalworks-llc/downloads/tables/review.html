EHI Export Schema
Table review
Description This table stores patient's Review of Systems related information.
Column DataType Description
encounterid int This column stores id of an encounter for which Review of Systems (ROS) is being documented. This column refers to enocunterid column in enc table.
catid int This column stores id of a category which is being documented by end user for the current visit. This column refers to itemid column in items table.
itemid int This column stores id of a symptom which is being documented by the end user during current visit. This column refers to itemid column in items table.
propid int This column stores id for the properties associated with Notes and Options sections on the user interface. This column refers to propid column in properties table.
value text This column stores free text or pre-configured options selected by end user to document each symptom on user interface.
value2 text This column stores free text or pre-configured options selected by end user to document each symptom on user interfacewith HTML tags.
displayIndex smallint This column stores order in which symptoms are listed on user interface.
