EHI Export Schema
Table hl7labdatadetail
Description This table stores detailed information about laboratory data in the HL7 format, including test results, units, ranges, flags, and other related information.
Column DataType Description
Reportid int This column stores the reportid of the lab for which the result is received from interface or documented through EMR for inhouse labs. This column refers column reportid from labdata table.
NAME varchar This column stores an item name of the attribute. This column refers to the column itemname from items table.
value varchar This column stores a value of test attributes for which the result is received from interface or documented through EMR for inhouse labs.
units varchar This column stores a unit of test attribute values for which the result is received from interface or documented through EMR for inhouse labs.
range varchar This column stores a reference range of test attribute values for which the result is received from interface or documented through EMR for inhouse labs.
flag varchar This column stores a flag of test attribute values for which the result is received from interface or documented through EMR for inhouse labs.
notes text This column stores the notes of test attribute values for which the result is received from interface or documented through EMR for inhouse labs.
resultStatus varchar This column stores the attribute result status of the test attribute received in the Results Inbound HL7 message through the interface.
labid int This column stores the labid of the interface through which the result is received. This column refers column id from the lablist table.
hl7id varchar This column stores the attribute code of the test attribute received in the Results Inbound HL7 message through the interface.
profileid varchar This column stores the profile id (also called as Accession id) of the test received in the Results Inbound HL7 message through the interface.
serialno int This column stores the serial no. (integer value in ascending order used to maintain the order for display) of the test attribute received in the Results Inbound HL7 message through the Interface. This is the number that is generated by the interface not sent by the vendor.
obrcode varchar This column is not used anymore.
performingSiteid varchar This column stores the site id of the performing lab where the testing was performed for that test attribute. This value is received in the HL7 message from the vendor through the interface.
prevvalue varchar This column stores the previous value of test attribute for which the result is received from interface when the new value is sent.
corrreportno int This column stores the Report No. (i.e. integer value in ascending order used to maintain the order for display in case of corrected reports for the same test) of the test result received in the Results Inbound HL7 message through the interface. This is the number that is generated by the Interface not sent by the vendor.
AttrResultDate datetime This column stores the result date of the test attribute received in the Results Inbound HL7 message through the interface. The format used is YYYY-MM-DD HH:MM:SS.
AttrResultTime datetime This column stores the result time of the test attribute received in the Results Inbound HL7 message through the Interface. The format used is YYYYY-MM-DD HH:MM:SS.
obsubid varchar This column stores an id received in case of Culture tests to identify parent/child for results received through the interface.
itemid int This column is not used anymore.
hl7itemid int This column stores an itemid of the attribute. This column refers column itemid from items table.
