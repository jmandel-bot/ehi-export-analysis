EHI Export Schema
Table pharmacy
Description This table stores information about various pharmacies, including their contact details, service levels, and e-prescribing capabilities.
Column DataType Description
pmcid (Primary Key) int This column refers to pmcid column of Pharmacy table in EMR. It is id of pharmacy
pharmacyname varchar Name of the pharmacy. Example: "ABC Pharmacy", "XYZ Pharmacy".
pharmacyaddress varchar Primary address of the pharmacy. Example: "123 Main St", "456 Elm St".
pharmacyaddress2 varchar Secondary address of the pharmacy, if applicable. Example: "Suite 100", "Apt 4B".
pharmacycity varchar City where the pharmacy is located. Example: "New York", "Los Angeles".
pharmacystate varchar State where the pharmacy is located. Example: "NY", "CA".
pharmacyzip varchar ZIP code of the pharmacy's location. Example: "10001", "90001".
pharmacyphone varchar Phone number of the pharmacy. Example: "(555) 123-4567", "(555) 987-6543".
pharmacyfax varchar Fax number of the pharmacy. Example: "(555) 123-4568", "(555) 987-6544".
pharmacyemail varchar Email address of the pharmacy. Example: "info@abcpharmacy.com", "contact@xyzpharmacy.com".
vmid varchar Unique identifier for the pharmacy's virtual machine. Example: "vm123", "vm456".
delFlag int Flag to indicate if the pharmacy record is deleted (1) or not (0). Example: 0, 1.
pharmacyfaxprefix varchar Prefix for the pharmacy's fax number. Example: "1", "001".
NCPDPid varchar National Council for Prescription Drug Programs (NCPDP) identifier for the pharmacy. Example: "1234567", "9876543".
MailOrderPharmacy tinyint Flag to indicate if the pharmacy is a mail-order pharmacy (1) or not (0). Example: 0, 1.
EPrescribeEnabled tinyint Flag to indicate if the pharmacy is e-prescribing enabled (1) or not (0). Example: 0, 1.
EPrescribeEnabledBy varchar identifier for the user who enabled e-prescribing for the pharmacy. Example: "user123", "user456".
Enterpriseid int identifier for the enterprise associated with the pharmacy. Example: 1001, 2002.
StoreName varchar Name of the store where the pharmacy is located. Example: "Store A", "Store B".
ServiceLevel varchar This column indicates what service this pharmacy supports e.g. NewRx, Refills, ChangeRx
EPCSEnabled tinyint Flag to indicate if the pharmacy is Electronic Prescribing of Controlled Substances (EPCS) enabled (1) or not (0). Example: 0, 1.
PillPassLocation tinyint Flag to indicate if the pharmacy is a PillPass location (1) or not (0). Example: 0, 1.
ss_pharmacyfax varchar This column stores the pharmacy fax number coming from Surescripts pharmacy downloads.
pharmacyPhoneExt varchar Phone extension for the pharmacy's phone number. Example: "101", "102".
NPI varchar National Provider identifier (NPI) for the pharmacy. Example: "1234567890", "0987654321".
CrossStreet varchar Cross street near the pharmacy's location. Example: "Main St & Elm St", "Broadway & 5th Ave".
faxDialingCountryid int identifier for the country code used when dialing the pharmacy's fax number. Foreign key relation to `countrycodes` table's `id` column. Example: 1, 2.
