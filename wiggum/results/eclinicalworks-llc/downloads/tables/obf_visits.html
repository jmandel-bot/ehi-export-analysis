EHI Export Schema
Table obf_visits
Description This table stores information about visits related to pregnancies, including visit details, user information, and timestamps.
Column DataType Description
id (Primary Key) int This column stored primary key of the table as numeric value,  This column stored  Flowsheet Add new visit section data based on Pregnancy id
Pregid int This column stores pregnancy id as numeric value of the obf_pregnnacy table
visitid int This column stores visit id as numeric value, its unique value based of pregnancy id
Itemid int This column stored item id as numeric value of items table
VALUE varchar This column stored value as varchar value of documented
Lock_Ind int This column stored value as numeric value like ( 0 or 1), if particular visit is lock the value will be 1 , if unlock the value is 0
DelStat int Deletion status of the record. 0 for not deleted, 1 for deleted. Example: 0, 1.
Uname varchar Username of the user who created or updated the record. Example: "user1", "user2", "user3".
Upd_Date datetime time stamp for when the record was last updated. Example: "2021-08-01 12:34:56".
encounterid int This column stores encounter id as numeric value from the enc table
timeofentry varchar This column stores time of entry as varchar value , when user documented the visit.
preobweight varchar Pre-pregnancy weight of the patient. Example: "60", "65", "70".
