Table ccmrproblemreviewsetup
Description This table stores the configuration settings for the review process of CCMR problems.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
ccmrid int Foreign key referring to the CCMR problem's unique identifier. Example: 101, 102, 103. (Foreign key relation: `ccmrproblem.id`)
ReviewDuration int The duration of the review process in the unit specified by `providerDurationType`. Example: 30, 60, 90.
isRecurrent int A flag indicating whether the review process is recurrent (1) or not (0). Example: 0, 1.
providerDurationType int The unit of time for the review duration, where 0 represents days and 1 represents weeks. Example: 0, 1.
deleteFlag int A flag indicating whether the record is deleted (1) or not (0). Example: 0, 1.
userid int Foreign key referring to the user's unique identifier who is responsible for the review process. Example: 1001, 1002, 1003. (Foreign key relation: `user.id`)
Programid int Foreign key referring to the program's unique identifier associated with the CCMR problem. Example: 2001, 2002, 2003. (Foreign key relation: `program.id`)
reviewCalBasedOn tinyint The basis for calculating the review duration, where 0 represents the start date and 1 represents the end date. Example: 0, 1.
recurringDuration smallint The duration of the recurring review process in the unit specified by `recurringDurationType`. Example: 7, 14, 30.
recurringDurationType tinyint The unit of time for the recurring review duration, where 0 represents days and 1 represents weeks. Example: 0, 1.
