Table bh_outcome
Description This table stores information about different outcomes, their status, display index, and deletion flag.
Column DataType Description
id (Primary Key) int This is the primary key for the table, an integer that uniquely identifies each outcome. It is an auto-incrementing value. Example: 1, 2, 3, etc.
outcomeName varchar This is a varchar(255) column that stores the name of the outcome. Example: "Success", "Failure", "In Progress", etc.
outcomeStatus tinyint This is a tinyint(1) column that stores the status of the outcome. It has a default value of '0'. Example: 0 (Inactive), 1 (Active).
displayIndex int This is an int(11) column that stores the display index of the outcome, which determines the order in which outcomes are displayed. It has a default value of '0'. Example: 0, 1, 2, 3, etc.
deleteFlag tinyint This is a tinyint(1) column that stores a flag indicating whether the outcome has been deleted or not. It has a default value of '0'. Example: 0 (Not Deleted), 1 (Deleted).
