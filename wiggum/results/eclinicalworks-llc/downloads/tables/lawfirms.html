Table lawfirms
Description This table stores the complete information of the attorney.
Column DataType Description
id (Primary Key) int Unique identifier for each law firm (e.g., 1, 2, 3). This is the primary key for the table.
FirmName varchar The name of the law firm (e.g., "Smith & Associates", "Johnson Law Group").
Address1 varchar The first line of the law firm's street address (e.g., "123 Main St").
Address2 varchar The second line of the law firm's street address, if applicable (e.g., "Suite 200").
City varchar The city where the law firm is located (e.g., "New York", "Los Angeles").
State varchar The state where the law firm is located (e.g., "New York", "California").
Zip varchar The zip code of the law firm's address (e.g., "10001", "90001").
ContactPerson varchar The name of the primary contact person at the law firm (e.g., "John Smith", "Jane Doe").
Phone1 varchar The primary phone number for the law firm (e.g., "(555) 123-4567").
Phone2 varchar The secondary phone number for the law firm, if applicable (e.g., "(555) 987-6543").
ContactDetails varchar Additional contact information for the law firm, such as an email address (e.g., "info@smithlaw.com").
Notes text A blob field for storing any additional notes or comments about the law firm.
