EHI Export Schema
Table coc
Description This table stores information about the Circle of Care (CoC) for patients, including details about the care manager, contact information, and related programs.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key of This column .
name varchar This column stores name of the group to which the providers or staff members will be added, such as Specialists.
createdby int This column stores id of the provider or staff member who added patient's care info. This column refers to column UID from user table.
createdon datetime This column stores the created datetime of the patient's care info.
patientid int This column stores id of the patient for whom care info is added. This column refers to column UID from user table.
description varchar This column stores the description of the group in this field.
primarycontactid int This column stores a digit to indicate order in which primary contact of the patient is listed in user interface.
delflag int This column stores value 0 / 1 indicating if the Patient care Info is deleted, this is soft delete. 0 - Not Deleted , 1 -Deleted
isccmrcontact tinyint This column stores value 0 / 1 indicating if the Patient care contact is CCMR contact. 0 - Not CCMR contact , 1 -CCMR contact
hasCareManager tinyint This column stores value 0 / 1 indicating if the Patient's care contact is care manager. 0 - Not care manager , 1 -Care manager
ccmrdispname varchar This column stores ccmr display name of the patient's care contact person.
ccmrprogid int This column stores ccmr program id. This column refers to acoprograms.id.
activeFlag tinyint Flag indicating if the CoC is active (1) or not (0). (e.g., 0, 1)
