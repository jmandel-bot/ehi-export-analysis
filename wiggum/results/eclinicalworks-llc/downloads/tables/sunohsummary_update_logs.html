EHI Export Schema
Table sunohsummary_update_logs
Description This table stores the updated summary versions of the original AI-generated summary stored in the main table sunohsummary. It will maintain a new record for each update of the summary.
Column DataType Description
ID (Primary Key) int It is the primary key column for table sunohsummary_update_logs.
summaryId int This column is a foreign key that references the 'sunohsummary' table's 'ID' column. It is not null because every entry should have a sunohsummary record linked.
summary nvarchar Stores the updated summary of the conversation.
updatedDate datetime Stores the last modified date and time of the entry, not null because every entry should have a modification timestamp. Default value is current timestamp.
updatedBy int Foreign key, references the 'users' table's 'uid' column, not null because every entry should have an updatedBy user id linked. User id is a provider or staff who updated the sunoh conversation summary.
