Table allergyrefillrequest
Description This table stores information about allergy refill requests made by patients, including the request status, reasons, and related details.
Column DataType Description
id (Primary Key) int Unique identifier for each allergy refill request (e.g., 1, 2, 3).
Patientid int identifier for the patient making the refill request, foreign key relation to `patients` table (`pid` column) (e.g., 101, 102, 103).
Queueid int identifier for the queue where the refill request is placed, foreign key relation to `antigen_recipe_queue` table (`id` column) (e.g., 201, 202, 203).
SourceQueueid int identifier for the source queue from which the refill request is made, foreign key relation to `antigen_recipe_queue` table (`id` column) (e.g., 301, 302, 303).
Formulationid int identifier for the allergy formulation related to the refill request, foreign key relation to `allergy_formulation` table (`id` column) (e.g., 401, 402, 403).
Encounterid int identifier for the patient encounter related to the refill request, foreign key relation to `enc` table (`encounterid` column) (e.g., 501, 502, 503).
Recipeid varchar identifier for the recipe associated with the refill request (e.g., 'R001', 'R002', 'R003').
BottleName varchar Name of the bottle associated with the refill request (e.g., 'BottleA', 'BottleB', 'BottleC').
REQUESTTYPE varchar Type of the refill request (e.g., 'Standard', 'Urgent', 'Emergency').
AssignedToid int identifier for the user assigned to handle the refill request, foreign key relation to `users` table (`UID` column) (e.g., 1001, 1002, 1003).
RefillStatus varchar Status of the refill request, can be 'Open', 'Declined', or 'Completed'.
RequestReason varchar Reason for the refill request (e.g., 'Regular Refill', 'Lost Bottle', 'Travel').
DeclineReason varchar Reason for declining the refill request, if applicable (e.g., 'Invalid Prescription', 'Expired Insurance', 'Incomplete Information').
ShotFacilityid int identifier for the facility where the allergy shot is administered, if applicable (e.g., 2001, 2002, 2003).
MixingFacilityid int identifier for the facility where the allergy formulation is mixed, foreign key relation to `edi_facilities` table (`id` column) (e.g., 3001, 3002, 3003).
RequestedByid int identifier for the user who requested the refill, foreign key relation to `users` table (`UID` column) (e.g., 4001, 4002, 4003).
RequestDate datetime Date and time when the refill request was made (e.g., '2022-01-01 12:00:00', '2022-01-02 14:30:00', '2022-01-03 16:45:00').
isHighPriority tinyint A flag indicating if the refill request is high priority (1) or not (0).
Note varchar Additional notes or comments related to the refill request, if any (e.g., 'Patient needs refill before vacation', 'Check insurance coverage before processing').
