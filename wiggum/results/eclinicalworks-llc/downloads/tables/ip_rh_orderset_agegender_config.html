EHI Export Schema
Table ip_rh_orderset_agegender_config
Description The ip_rh_orderset_agegender_config table is designed to store information related to age and gender configurations for order sets. It contains columns for the starting and ending ages in years and months, the gender associated with the configuration, and a flag indicating whether the configuration has been deleted. This information is used to tailor order sets to specific age and gender demographics.
Column DataType Description
Id (Primary Key) int This is Primary Key for the table. This column is a unique identifier for each row, automatically generated when a new row is inserted.
orderSetId int This column stores a foreign key that references the?id?column in the?ip_rh_orderset?table. This maps the order set with age gender configuration.
startYearAge tinyint This column stores the starting age in years for the age/gender configuration. This column contains TINYINTtype values which range between 0 to 255.
endYearAge tinyint This column stores the ending age in years for the age/gender configuration. This column contains TINYINTtype values which range between 0 to 255.
startMonthAge tinyint This column stores the starting age in months for the age/gender configuration. This column contains TINYINTtype values which range between 0 to 255.
endMonthAge tinyint This column stores the ending age in months for the age/gender configuration. This column contains TINYINTtype values which range between 0 to 255.
gender char This column stores the gender associated with the age/gender configuration. It stores Char (1) type value. By default, it stores ‘U'-Unknown. The possible values can be ‘M'-Male,'F'-Female.
deleteFlag tinyint This column stores a flag that indicates whether the age/gender configuration has been deleted. The possible value for the column is either 0 or 1. 0 meant for active order and 1 meant for inactive order.
