EHI Export Schema
Table edi_inv_deleted_adj
Description This table stores information about deleted adjustments related to invoices.
Column DataType Description
id (Primary Key) int Unique identifier for each deleted adjustment entry. Example: 1, 2, 3.
Refid int identifier for the transactions table associated with the deleted adjustment. Example: 1001, 1002, 1003.
Invid int identifier for the invoice associated with the deleted adjustment. Example: 1001, 1002, 1003.
Code varchar A code representing the type or reason for the deleted adjustment. Example: 'DISCOUNT', 'REFUND', 'PENALTY'.
Amount float The amount of the deleted adjustment, stored as a floating-point number with 5 digits before the decimal point and 2 digits after. Example: 100.00, 50.25, 25.99.
Userid int identifier for the user who performed the deletion of the adjustment, stored as an unsigned integer. Example: 1, 2, 3.
modifiedDate datetime time stamp for when the adjustment was deleted or modified, automatically updated to the current time stamp. Example: '2022-01-01 12:34:56', '2022-02-15 09:30:00'.
