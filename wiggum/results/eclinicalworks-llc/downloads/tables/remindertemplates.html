EHI Export Schema
Table remindertemplates
Description This table stores information about reminder templates, including the file name, custom name, description, and a delete flag.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an integer that auto-increments. It uniquely identifies each reminder template. Example: 1, 2, 3, etc.
fileName varchar This is a varchar(255) column that stores the file name of the reminder template. It cannot be NULL and has a default value of an empty string. Example: "template1.txt", "reminder2.docx", etc.
customName varchar This is a varchar(50) column that stores a custom name for the reminder template. It cannot be NULL and has a default value of an empty string. Example: "Birthday Reminder", "Meeting Alert", etc.
description text This is a text column that stores a description of the reminder template. It can be NULL if no description is provided. Example: "A template for birthday reminders", "A template for meeting alerts", etc.
deleteFlag int This is an int(2) column that stores a flag to indicate if the reminder template has been deleted or not. It cannot be NULL and has a default value of '0'. A value of '0' means the template is not deleted, and a value of '1' means the template is deleted. Example: 0, 1.
