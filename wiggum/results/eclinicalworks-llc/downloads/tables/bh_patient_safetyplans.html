Table bh_patient_safetyplans
Description This table stores information about patient safety plans, including their status, modification details, and relationships with encounters and templates.
Column DataType Description
id (Primary Key) int Unique identifier for each patient safety plan record. Example: 1, 2, 3.
encounterid int identifier for the encounter associated with the patient safety plan. Foreign key relation: `enc.encounterid`. Example: 101, 102, 103.
templateid int identifier for the template used in the patient safety plan. Foreign key relation: `items.itemid`. Example: 1001, 1002, 1003.
deleteflag tinyint Indicates whether the patient safety plan record is deleted (1) or not (0). Example: 0, 1.
isLocked tinyint Indicates whether the patient safety plan record is locked (1) or not (0). Example: 0, 1.
modifiedDate datetime Date and time when the patient safety plan record was last modified. Example: '2021-09-01 12:34:56'.
modifiedBy int identifier for the user who last modified the patient safety plan record. Foreign key relation: `users.UID`. Example: 10001, 10002, 10003.
lockedData varchar Text data containing information about the locked state of the patient safety plan record. Example: 'Locked by user 10001 on 2021-09-01 12:34:56'.
createdDate datetime Date and time when the patient safety plan record was created. Example: '2021-08-01 12:34:56'.
