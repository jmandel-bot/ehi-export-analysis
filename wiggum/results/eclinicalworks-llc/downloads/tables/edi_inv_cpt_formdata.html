EHI Export Schema
Table edi_inv_cpt_formdata
Description This table stores information related to responses for specific questions in an inventory capture form.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
InvCptFormId int Foreign key referring to the inventory capture form's unique identifier. Example: 101, 102, 103. (Foreign key relation: `InvCptForm.id`)
QuestionId int Foreign key referring to the unique identifier of the question in the form. Example: 1001, 1002, 1003. (Foreign key relation: `Question.id`)
Refidentification varchar A reference identification for the response, which can be a code or a short description. Example: "REF123", "REF456".
Responsedate datetime The date when the response was provided. Example: '2021-09-01', '2021-09-02'.
Responsepercent decimal The percentage value of the response, if applicable. Example: 75.00, 50.00.
Refyesnoresponse char A single character representing a Yes or No response. Example: 'Y', 'N'.
deleteFlag tinyint A flag indicating whether the record has been deleted or not. Example: 0 (not deleted), 1 (deleted).
