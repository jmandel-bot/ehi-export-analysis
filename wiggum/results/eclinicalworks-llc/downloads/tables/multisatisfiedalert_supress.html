Table multisatisfiedalert_supress
Description This table stores a suppression of a multi-satisfied alerts for a patient.
Column DataType Description
alert_id int This column stores id of the suppressed alert. This column refers column id from multisatisfiedalert table.
suppress_date datetime This column stores the date on which the alert is suppressed. The format used is YYYY-MM-DD 00:000:00.000 hence datetime chosen.
notes text This column stores notes added for the suppressed alert. It is free text hence text chosen.
patient_id int This column stores id of the patient for whom this alert is suppressed. This column refers column UID from user table.
isPermanent tinyint This column indicates if the alert is suppressed permanently or temporarily. 1 - permanent and 0 - temporary.
suppressToDate datetime This column stores the date till which the alert is temporarily suppressed. The format used is YYYY-MM-DD 00:000:00.000 hence datetime chosen.
