EHI Export Schema
Table obf_ptcaretemplate
Description This table stores information about patient care templates, including the patient ID, template name, user ID, and the date and time the template was created or updated.
Column DataType Description
pid int The patient id (integer) that uniquely identifies a patient. This column is a foreign key that refers to the `pid` column in the `patients` table. Example: 1024
TemplateName varchar The name (varchar) of the patient care template. Example: "Diabetes Care Plan"
UID int The user id (integer) that uniquely identifies the user who created or updated the template. This column is a foreign key that refers to the `UID` column in the `users` table. Example: 35
datetime datetime The date and time (time stamp) when the template was created or last updated. This column is automatically set to the current time stamp when a new record is inserted and updated to the current time stamp when an existing record is updated. Example: "2021-09-15 14:23:45"
