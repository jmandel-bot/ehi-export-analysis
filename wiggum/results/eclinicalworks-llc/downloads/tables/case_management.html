EHI Export Schema
Table case_management
Description This table is used to store the information about the patient's case.
Column DataType Description
CaseId (Primary Key) int This is the primary key for the table and uniquely identifies each case. It is an integer value that is automatically incremented for each new case. Example: 1, 2, 3, etc.
CaseName varchar Case_management_CaseName Column store Case Label value from Billing (Band) -> Case Management -> View Case Detail Screen. Example : "Motor Vehicle Case" OR "Accident Case"
CreatedBy int This is an integer column that stores the user ID of the person who created the case. It is a required field and has a default value of 0. This column can be a foreign key referring to the user ID in a `users` table. Example: 1, 2, 3, etc.
AssignedTo int This column stores the id of the Provider/Staff to whom case is assigned. This column refers to column uid from user table.
deleteflag int This is an integer column that indicates whether the case has been deleted or not. It is a required field and has a default value of 0. A value of 0 means the case is not deleted, and a value of 1 means the case is deleted. Example: 0, 1.
Status int This column stores the value 0/1 to indicate if this is an ongoing case. 0 - Case is Open , 1 - Case is Closed.
