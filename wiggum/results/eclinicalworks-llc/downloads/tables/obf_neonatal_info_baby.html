Table obf_neonatal_info_baby
Description This table stores information about neonatal babies, including their associated pregnancy, item, type, choice, lock indicator, deletion status, username, comments, and last update time.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. (Example: 1, 2, 3)
Babyid int identifier for the baby, referring to a specific baby in another table. (Example: 101, 102, 103)
Pregid int identifier for the pregnancy, referring to a specific pregnancy in another table. (Example: 201, 202, 203)
Itemid int identifier for the item, referring to a specific item in another table. (Example: 301, 302, 303)
TYPE varchar A short text field representing the type of information stored in the record. (Example: 'A', 'B', 'C')
Choice varchar A text field representing the choice or option related to the item. (Example: 'Option1', 'Option2', 'Option3')
Lock_Ind int An integer indicating whether the record is locked (1) or not (0). (Example: 0, 1)
DelStat int An integer indicating the deletion status of the record, where 0 means not deleted and 1 means deleted. (Example: 0, 1)
uname varchar A text field representing the username of the person who created or modified the record. (Example: 'user1', 'user2', 'user3')
Comments text A text field for storing any additional comments related to the record. (Example: 'Comment1', 'Comment2', 'Comment3')
lastupdatetime datetime A time stamp indicating the last time the record was updated. (Example: '2021-09-01 12:34:56')
