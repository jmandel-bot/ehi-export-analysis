EHI Export Schema
Table milestone
Description This table stores information about various milestones achieved by patients.
Column DataType Description
Milestoneid (Primary Key) int The unique identifier for each milestone record. Examples include: 1, 2, 3, etc.
MilestoneName varchar The name of the milestone (e.g., 'First Steps', 'Speech Development', etc.).
MilestoneDesc varchar The description of the milestone. Examples include: 'Patient took their first steps', 'Patient started speaking words', etc.
Patientid int Unique identifier for the patient associated with the milestone. This is a foreign key that refers to the `pid` column in the `patient` table. Example: 101, 102, 103, etc.
Created datetime The timestamp of when the milestone record was created. Examples include: '2021-08-01 12:34:56'.
Userid int Unique identifier for the user who created the milestone record. This is a foreign key that refers to the `UID` column in the `users` table. Example: 1001, 1002, 1003, etc.
