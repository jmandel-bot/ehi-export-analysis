EHI Export Schema
Table laceration_location
Description This table stores list of location for laceration documentations
Column DataType Description
id (Primary Key) int This is an auto increment column. This column stores the item names which is preshipped with the itemid as 0. Once the lacerationSqlMigration.jsp is executed, it will add all the preshipped entries in the items table and the generated itemids will be updated in the itemid column.
name varchar This column will store the name of the item which is preshipped and will created after running lacerationSqlMigration.jsp. The size is varchar(100)
itemid int This column is preshipped as 0 . Once the lacerationSqlMigration.jsp is executed, it will add all the preshipped entries( in the name column) in items table and the generated itemids will be updated in the itemid column.
