EHI Export Schema
Table intcodes_enc
Description This table stores information about encounters associated with specific integer codes, including their accession IDs and a flag indicating if the record has been deleted.
Column DataType Description
id (Primary Key) int This column is the primary key column of the table used to uniquely identify each record in table.
encounterid int This column stores the encouterid from the encounter table and is used to map the accession ID with the encounter.
accessionid varchar This column stores the accession ID string received from the HL7 message and is used to map the encounter ID with the accession ID.
deleteflag tinyint This column stores tinyint values (0 or 1) indicating if the record has been deleted. A value of 0 means the record is not deleted, and a value of 1 means the record is deleted. Example: 0, 1.
intid int This column stores the intid from the intlist table and is used to identify to which custom interface the entry is made.
