EHI Export Schema
Table careplan_triggers
Description This table stores information about triggers related to care plans, such as review days, patient sign-off days, and utilization days.
Column DataType Description
id (Primary Key) int Unique identifier for each care plan trigger entry. Example: 1, 2, 3
CarePlanid int Unique identifier for the care plan associated with the trigger. This is a foreign key that refers to the `id` column in the `careplans` table. Example: 101, 102, 103
ReviewDays int Number of days after which the care plan should be reviewed. Example: 30, 60, 90
ReviewRepeat int Indicates whether the review process should be repeated (1) or not (0). Example: 0, 1
PtSignOffDays int Number of days after which the patient should sign off on the care plan. Example: 7, 14, 21
PtSignOffRepeat int Indicates whether the patient sign-off process should be repeated (1) or not (0). Example: 0, 1
UtilizationDays int Number of days after which the care plan utilization should be checked. Example: 15, 30, 45
UtilizationRepeat int Indicates whether the utilization check process should be repeated (1) or not (0). Example: 0, 1
