Table employment_jobcategory
Description This table stores information about different job categories and their associated employers.
Column DataType Description
id (Primary Key) int This is the primary key column, which uniquely identifies each record in the table. It is an integer value that is automatically incremented with each new record. Example values: 1, 2, 3, etc.
JobCategory varchar This column stores the name of the job category as a string with a maximum length of 250 characters. Example values: "Software Developer", "Data Analyst", "Marketing Manager", etc.
deleteFlag tinyint This column is a tiny integer value (1 byte) that indicates whether the record has been deleted or not. A value of 1 means the record is deleted, and a value of 0 means the record is active. Example values: 0, 1.
Employerid int This column stores the foreign key relation to the "insurance" table, specifically the "insid" column. It is an integer value that represents the unique identifier of the employer associated with the job category. Example values: 101, 102, 103, etc. Foreign key relation: "insurance(insid)".
