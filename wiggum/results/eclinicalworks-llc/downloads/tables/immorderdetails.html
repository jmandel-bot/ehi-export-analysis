EHI Export Schema
Table immorderdetails
Description This table stores immunization order details
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
Immid int Unique identifier for the immunization. Example: 101, 102, 103.
Immunizationid int Unique identifier for the specific immunization type. Example: 1001, 1002, 1003.
orderid int This column stores order id of immunization orders. It is a foreign key of Messageid in immorders table.
encid int The column stores an id of the encounter of a patient.
patientid int This column stores identifier number of a patient, for whom the appointment is created. It refers to column UID from user table.
ptlname varchar This column stores patient last name.
ptfname varchar This column stores patient first name.
ptdob datetime Date of birth of the patient. Example: '1990-01-01', '1985-05-15', '2000-12-31'.
givendate datetime Date when the immunization was given. Example: '2021-01-01', '2021-02-15', '2021-03-31'.
ProviderName varchar This column stores the Provider Name.
vaccinename varchar This column stores name of vaccine
location varchar This column is used to store location where an Immunization has been given
facility varchar This column stores name of facility
cvxcode varchar This column stores the cvx code
history varchar This flag is used to save (Y/N) based on Immunization’s history to know whether it was given in past
facilityid int This column is used to store facility id which is primary key id of edi_facilies table
controlno varchar This column refers to controlNo column from patients table
mvx_code varchar This column stores codes that indicate the manufacturer of a vaccine.
