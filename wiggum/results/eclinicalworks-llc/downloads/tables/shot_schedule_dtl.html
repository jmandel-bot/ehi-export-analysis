Table shot_schedule_dtl
Description This table stores the details of the shot schedules for different bottles and templates, including the dose, sequence index, and shot interval.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
QUEUEid int identifier for the queue associated with the shot schedule. Example: 101, 102, 103.
TEMPLATEid int identifier for the template associated with the shot schedule. Example: 201, 202, 203.
BOTTLEid int identifier for the bottle associated with the shot schedule. Example: 301, 302, 303.
CONid int identifier for the container associated with the shot schedule. Example: 401, 402, 403.
DOSE varchar Dose of the shot in the schedule. Example: '5ml', '10ml', '15ml'.
SEQINDEX int Sequence index of the shot in the schedule. Example: 1, 2, 3.
freq_id int identifier for the frequency of the shot schedule. Example: 501, 502, 503. Foreign key relation: References `allergy_structdata` table, `id` column.
SHOTINTERVAL int Interval between shots in the schedule, in minutes. Example: 30, 60, 90.
SHOTINTERVALNAME varchar Name or description of the shot interval. Example: 'Every 30 minutes', 'Every hour', 'Every 90 minutes'.
