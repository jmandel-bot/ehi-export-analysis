EHI Export Schema
Table formpatientresponses
Description This table stores the responses of patients to all questions in a survey.
Column DataType Description
id (Primary Key) int This is an auto increment id and primary key of table.
UID int Unique identifier for the user who submitted the response. Example: 101, 102, 103.
apUID int This column stores apUID
surveyid int id of survey which patient has responded to.
campaignid int This is a id of campaign thru which Survey has been sent to patient and to which patient has responsed.
initialNotificationMessageid int This is id of message sent to a patient
pageid int id of survey page where question belongs which patient has responded
questionid int id of question which patient has responded
subQuestionid int id of sub question which patient has responded
optionid int id of option which patient has selected while responding to question in survey
responseText text Text of the patient's response, stored in utf8mb4 character set (e.g., 'Yes', 'No', 'Maybe').
commentText varchar Additional comment text provided by the patient, stored in utf8mb4 character set (e.g., 'I liked the service', 'It was too crowded', NULL).
inserttime datetime time stamp of when the response was inserted into the table. Example: "2021-08-01 12:34:56".
