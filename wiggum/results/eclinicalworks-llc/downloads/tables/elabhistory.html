EHI Export Schema
Table elabhistory
Description This table store logs for electronically received and transmitted lab results.
Column DataType Description
Reportid int This column stores id of a Lab/DI which was transmitted or for which results were received. This column refer to reportid column in labdata table.
Type char This column stores letters T/R to indicate what kind of record is being stored. T - When e-transmit is selected from Common Send screen by end user and Lab/DI is transmitted R - When Lab/DI result was received over inbound interface.
Userid int This column stores id of a user who transmitted Lab/DI over interface.
labid int This column stores id for lab vendors for this database. This column refers to id column in lablist table.
Clienttimestamp datetime This column stores date and time on client end when Lab/DI was transmitted or Lab/DI result was received. Date and time is stored in 'YYYY-MM-DD hh:mm:ss.000' format.
Servertimestamp datetime This column stores date and time on server end when Lab/DI was transmitted or Lab/DI result was received. Date and time is stored in 'YYYY-MM-DD hh:mm:ss.000' format.
