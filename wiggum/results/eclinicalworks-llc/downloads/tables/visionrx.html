EHI Export Schema
Table visionrx
Description This table stores patients' vision prescription data documented during their vision encounters.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key.
Encounterid int This column stores the encounter id for which vision section is getting documented. This column refers column encounterid from enc table.
ExamType int This column stores type of the exam selected, it could be any one one of the below values: UNAidED_ACUTIES = 1 ; AUTO_REFRACTION = 2;RETINOSCOPY = 3; PRESENTING_SPECTACLE_RX = 4;MANIFEST = 5;CYCLOPLEGIC = 6;FINAL_SPECTACLE_RX = 7;PD = 8; PRESENTING_CONTACT_RX = 9; TRIAL_RX = 10;ORX = 11; FINAL_CONTACT_RX = 12; CAS_CONFRONTATION = 30;CAS_AMSLER_STUDIES = 31;AidED_ACUTIES = 32; and AUTO_REFRACTION_WET = 33.
ExamTabName varchar This column stores the tab name for the prescribed data and when the ExamType column has one of the below values: 4, 7, 9, 10, 11 and 12.
DisplayIndex int This column stores the sequence and indexing of displaying tabs.
SphOD varchar This column stores sphere for right eye.
CylOD varchar This column stores cylinder value for right eye.
AxisOD varchar This column stores axis value for right eye.
HPrismOD varchar This column stores horizontal prism value for right eye.
VPrismOD varchar This column stores vertical prism value for right eye.
AddOD varchar This column stores adduction value for right eye.
DvaOD varchar This column stores distance visual acuity of right eye.
NvaOD varchar This column stores neovascularization of the angle of right eye.
PhOD varchar This column stores pinhole of right eye.
BcOD varchar This column stores base curve of the right eye.
DiaOD varchar This column stores diameter of the right eye.
ProductNameOD varchar This column stores the product name for the vision prescription for right eye. It could be manually entered.
SphOS varchar This column stores sphere for left eye.
CylOS varchar This column stores cylinder value of the left eye.
AxisOS varchar This column stores axis value of left eye.
HPrismOS varchar This column stores horizontal prism value for a left eye.
VPrismOS varchar This column stores vertical prism value for a left eye.
AddOS varchar This column stores adduction value for the left eye.
DvaOS varchar This column stores distance visual acuity of the left eye.
NvaOS varchar This column stores neovascularization of the angle of the left eye.
PhOS varchar This column stores pinhole of left eye.
BcOS varchar This column stores base curve of the left eye.
DiaOS varchar This column stores diameter of the left eye.
ProductNameOS varchar This column stores the product name for the vision prescription for left eye. It could be manually entered.
SphOU varchar This column stores cylinder value for both the eyes.
CylOU varchar This column stores cylinder value for both the eyes.
AxisOU varchar This column stores axis value for both the eyes.
HPrismOU varchar This column is not in use.
VPrismOU varchar This column is not in use.
AddOU varchar This column stores adduction value for both the eyes.
DvaOU varchar This column stores distance visual acuity of both the eyes.
NvaOU varchar This column stores neovascularization of the angle of both the eyes.
PhOU varchar This column stores pinhole of both the eyes.
BcOU varchar This column stores base curve of both the eyes.
DiaOU varchar This column stores diameter of both the eyes.
ProductNameOU varchar This column stores the product name for the vision prescription for both the eyes. It could be manually entered.
ExamTabKey varchar This column stores a unique key associated with the exam tab and is used to link it to the parent exam tab.
ExamTabParentKey varchar This column stores a key of a parent exam tab if this prescription is linked to it.
IsGPL_OD tinyint This column stores a value of 0 or 1. A "1" means a gas permeable or rigid gas permeable contact lens for the right eye, and a "0"is otherwise.
IsGPL_OS tinyint This column stores a value of 0 or 1. A "1" means a gas permeable or rigid gas permeable contact lens for the right eye, and a "0"is otherwise.
notes text This column is not in use.
dilation tinyint This column is not in use.
dilationtime datetime This column is not in use.
distHPrismOD varchar This column is not in use.
distVPrismOD varchar This column is not in use.
interHPrismOD varchar This column is not in use.
interVPrismOD varchar This column is not in use.
interAddOD varchar This column is not in use.
distHPrismOS varchar This column is not in use.
distVPrismOS varchar This column is not in use.
interHPrismOS varchar This column is not in use.
interVPrismOS varchar This column is not in use.
interAddOS varchar This column is not in use.
ocod varchar This column is not in use.
ocos varchar This column is not in use.
TintOD varchar This column stores tint of the contact lens for the right eye. This column refers to column tint from clproduct. This value can be modified in the prescription by the end user.
TintOS varchar This column stores tint of the contact lens for the left eye. This column refers to column tint from clproduct. This value can be modified in the prescription by the end user.
TintOU varchar This column stores tint of the contact lens for the both the eyes. This column refers to the column tint from clproduct. This value can be modified in the prescription by the end user.
delFlag tinyint This column contains a boolean indicating if the record has been deleted. (e.g., 0, 1).
websync int This column contains a boolean indicating if the record has been synced to the web. (e.g., 0, 1).
