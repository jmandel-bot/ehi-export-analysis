Table ccmr_intervention_modality_freq_mapping
Description This table is used to store linking between intervention and modality for the problems section in care plan.
Column DataType Description
Id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
encounterId int Column stores problem / Care plan Encounterid. Refers to encounterid column in enc table.
goalMappingId int unique id from ccmr_goalmapping table for intervention
bhId int unique id from structdatadetail table
TYPE char column stores character 'I' indicating intervention.
modalityId int Stores id from structdatadetail for modality id
frequencyId int id from structdatadetail for frequency id
deleteFlag tinyint A tiny integer representing whether the record is deleted or not. A value of 0 means the record is not deleted, and a value of 1 means the record is deleted. Example: 0, 1.
