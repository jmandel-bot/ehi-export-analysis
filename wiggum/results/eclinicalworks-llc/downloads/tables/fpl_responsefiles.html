EHI Export Schema
Table fpl_responsefiles
Description This table stores information about response files related to the FPL (Fantasy Premier League) system, including the file name, import date, and vendor.
Column DataType Description
id (Primary Key) int This is the primary key column, which uniquely identifies each record in the table. It is an integer with a maximum length of 11 digits and is automatically incremented. Example: 1, 2, 3, etc.
responsefilename varchar Response file name should starts with FinHealthScoreRequest and ends with results.txt
importteddate datetime It stores current date while importing response files
vendorid int Storing vendor id reference of vendor table
