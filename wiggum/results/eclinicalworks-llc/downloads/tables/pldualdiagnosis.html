EHI Export Schema
Table pldualdiagnosis
Description This table stores dual diagnosis (ICD9 and ICD10) for a patient's problem. (enable itemkey = SaveDualDiagnosis)
Column DataType Description
id (Primary Key) int This column is an auto incremented Primary Key.
deleteflag smallint Indicates if the record has been deleted. A value of 0 means not deleted, and a value of 1 means deleted. Example: 0, 1.
itemid int This column stores dual diagnosis item id. This column refers column itemId from items table.
icd9itemid int This column stores item id of the code in icd9code column. This column refers column itemId from items table.
icd9code varchar ICD-9 code for the diagnosis. Example: '250.00', '401.9', '414.01'.
icd9desc varchar Description of the ICD-9 diagnosis. Example: 'Diabetes mellitus', 'Essential hypertension', 'Coronary atherosclerosis'.
icd10itemid int This column stores item id of the code in icd10code column. This column refers column itemId from items table.
icd10code varchar This column stores ICD10 code of the item in icd10itemid column.
icd10desc varchar This column stores name of the item from icd10itemid column. This column refers column itemName from items table.
patientid int This column stores patient id for whom the dual diagnosis codes are added. This column refers column uid from user table.
originalslno int This column stores original serial number. This column refers column SlNo from problemlist table.
