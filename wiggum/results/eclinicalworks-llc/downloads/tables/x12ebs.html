EHI Export Schema
Table x12ebs
Description This table stores information related to electronic billing and insurance claims, including details about providers, patients, encounters, and various codes and identifiers.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. (Example: 1, 2, 3)
Submissionid int identifier for the submission, default value is 0. submission id is id of edi_online_requests table(Example: 101, 102, 103)
TnsRowid int identifier for the TnsRow, default value is 0. (Example: 1001, 1002, 1003)
ClearingHouseid int identifier for the clearing house, default value is 0. (Example: 1, 2, 3)
Providerid int identifier for the provider, default value is 0. (Example: 10001, 10002, 10003)
Patientid int identifier for the patient, default value is 0. (Example: 20001, 20002, 20003)
Encounterid int identifier for the encounter, default value is 0. (Example: 30001, 30002, 30003)
TnsType varchar Type of Tns, stored as a varchar with a default value of an empty string. (Example: 'TypeA', 'TypeB', 'TypeC')
Loopid varchar identifier for the loop, stored as a varchar with a default value of an empty string. (Example: 'Loop1', 'Loop2', 'Loop3')
Elementid varchar identifier for the element, stored as a varchar with a default value of an empty string. (Example: 'Elem1', 'Elem2', 'Elem3')
EBInfo char Information about EB, stored as a char with a default value of an empty string. (Example: 'A', 'B', 'C')
CoverageLevelCode char Code for the coverage level, stored as a char with a default value of an empty string. (Example: 'AAA', 'BBB', 'CCC')
ServiceTypeCode varchar Code for the service type, stored as a varchar with a default value of NULL. (Example: 'STC1', 'STC2', 'STC3')
InsuranceTypeCode char Code for the insurance type, stored as a char with a default value of an empty string. (Example: 'ITC', 'ITD', 'ITE')
PlanCoverageDescription varchar Description of the plan coverage, stored as a varchar with a default value of an empty string. (Example: 'Basic Plan', 'Premium Plan', 'Family Plan')
TimePeriodQualifier char Qualifier for the time period, stored as a char with a default value of an empty string. (Example: 'TP', 'TQ', 'TR')
BenefitAmount varchar Amount of the benefit, stored as a varchar with a default value of an empty string. (Example: '1000', '2000', '3000')
BenefitPercent varchar Percentage of the benefit, stored as a varchar with a default value of an empty string. (Example: '10%', '20%', '30%')
QuantityQualifier char Qualifier for the quantity, stored as a char with a default value of an empty string. (Example: 'QQ', 'QR', 'QS')
BenefitQuantity varchar Quantity of the benefit, stored as a varchar with a default value of an empty string. (Example: '5', '10', '15')
AuthOrCertIndicator char Indicator for authorization or certification, stored as a char with a default value of 'U'. (Example: 'A', 'C', 'U')
InPlanNetworkIndicator char Indicator for in-plan network, stored as a char with a default value of 'U'. (Example: 'Y', 'N', 'U')
ProductOrServiceidQualifier varchar Qualifier for the product or service id, stored as a varchar with a default value of NULL. (Example: 'Prod1', 'Prod2', 'Prod3')
ProcedureCode varchar Code for the procedure, stored as a varchar with a default value of an empty string. (Example: 'PCode1', 'PCode2', 'PCode3')
ProcedureModifier1 char First modifier for the procedure, stored as a char with a default value of an empty string. (Example: 'M1', 'M2', 'M3')
ProcedureModifier2 char Second modifier for the procedure, stored as a char with a default value of an empty string. (Example: 'N1', 'N2', 'N3')
ProcedureModifier3 char Third modifier for the procedure, stored as a char with a default value of an empty string. (Example: 'O1', 'O2', 'O3')
ProcedureModifier4 char Fourth modifier for the procedure, stored as a char with a default value of an empty string. (Example: 'P1', 'P2', 'P3')
timestamp datetime time stamp of the record creation or update, stored as a time stamp with a default value of the current time stamp. (Example: '2022-01-01 12:34:56', '2022-01-02 14:56:23')
