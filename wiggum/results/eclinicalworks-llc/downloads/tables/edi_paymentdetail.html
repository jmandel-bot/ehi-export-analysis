EHI Export Schema
Table edi_paymentdetail
Description This table contains payment related details
Column DataType Description
PmtDetailId (Primary Key) int Unique identifier for each payment detail record. (e.g., 1, 2, 3)
paymentId int It stores id related to edi_inspayments table.
encounterId int It stores encounterid related to enc table.
invoiceId int It stores invoiceid related to edi_invoices table
adjustment decimal Adjustment amount applied to the payment detail. (e.g., 10.00, 5.50, 0.00)
allowed decimal Allowed amount for the payment detail. (e.g., 100.00, 50.00, 25.00)
deduct decimal Deductible amount for the payment detail. (e.g., 10.00, 5.00, 0.00)
coins decimal Coinsurance amount for the payment detail. (e.g., 20.00, 10.00, 5.00)
memberBalance decimal Member's remaining balance for the payment detail. (e.g., 80.00, 40.00, 20.00)
paid decimal Amount paid for the payment detail. (e.g., 90.00, 45.00, 25.00)
withheld decimal Withheld amount for the payment detail. (e.g., 10.00, 5.00, 0.00)
MsgCode varchar Message code associated with the payment detail. (e.g., 'MC001', 'MC002', 'MC003')
timestamp datetime time stamp of the payment detail record creation or update. (e.g., '2022-01-01 12:34:56')
eraAdjustment decimal Electronic remittance advice (ERA) adjustment amount for the payment detail. (e.g., 5.00, 2.50, 0.00)
closingId int Identifier for the closing associated with this payment detail. (e.g., 3001, 3002, 3003)
claimTaxAmt decimal Claim tax amount for the payment detail. (e.g., 10.00, 5.00, 0.00)
CalculatedAdjustment decimal Calculated adjustment amount for the payment detail. (e.g., 10.00, 5.00, 0.00)
createdBy int Identifier for the user who created the payment detail record. (e.g., 1, 2, 3)
createdDateTime datetime Date and time when the payment detail record was created. (e.g., '2022-01-01 12:34:56')
modifiedBy int Identifier for the user who last modified the payment detail record. (e.g., 1, 2, 3)
modifiedDateTime datetime Date and time when the payment detail record was last modified. (e.g., '2022-01-01 12:34:56')
ClaimInsId int It stores insureceid associated with insurance table
seqno int Sequence number for the payment detail record. (e.g., 1, 2, 3)
RecreatedFlag tinyint Flag indicating whether the payment detail record has been recreated (1) or not (0). (e.g., 0, 1)
ERACASPostedAsIs tinyint Flag indicating whether the ERA claim adjustment is posted as is (1) or not (0). (e.g., 0, 1)
ERAClaimStatusCLP02 int ERA claim status code for the payment detail. (e.g., 1, 2, 3)
hl7id varchar Identifier for the HL7 message associated with this payment detail. (e.g., 'HL7001', 'HL7002', 'HL7003')
isreviewed smallint Flag indicating whether the payment detail record has been reviewed (1), not reviewed (0), or pending review (-1). (e.g., 0, 1, -1)
isexception tinyint Flag indicating whether the payment detail record has an exception (1) or not (0). (e.g., 0, 1)
SubscriberSecondaryInvRef varchar Subscriber secondary invoice reference for the payment detail. (e.g., 'SSIR001', 'SSIR002', 'SSIR003')
