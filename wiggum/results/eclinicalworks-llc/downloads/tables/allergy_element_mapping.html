EHI Export Schema
Table allergy_element_mapping
Description This table stores the mapping information of allergy elements, including their categories and parent items.
Column DataType Description
id (Primary Key) int The column stores an id of the elements (allergen items) mapping with forms. This is an auto incremental primary key of This column .
FORMid int This column stores identifier number of a form, which is associated with elements (allergen items). It refers to column formid from allergy_form table.
CATid int This column stores id of the category which is associated with the form. It refers to column itemid from items table.
CATNAME varchar This column stores name of the category which is associated with the form. It refers to column itemName from items table.
ITEMid int This column stores id of the item which is associated with the category. It refers to column itemid from items table.
ITEMNAME varchar This column stores name of the item which is associated with the category. It refers to column itemName from items table.
PARENTITEMid int This column stores parentid of the item (in case of child allergens) which is associated with the child item. It refers to column itemid from items table.
