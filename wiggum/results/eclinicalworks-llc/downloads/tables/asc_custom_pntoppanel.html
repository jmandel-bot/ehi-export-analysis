EHI Export Schema
Table asc_custom_pntoppanel
Description This table is used to store custom section admin-side configurations, such as section name, abbreviation, link, expression name, etc.
Column DataType Description
sectionid (Primary Key) int This is an auto incremented primary key of This column .
sectionName varchar This column used to store section name. Column does not except NULL value and default value is ''.
displayName varchar This column used to store section display name on ASC progress note side. Column does not except NULL value and default value is ''.
iconClass varchar This Column represent css class attribute for a section icon. Column contains NULL value and default value is NULL.
encType int This column indicate for which encType we are creating Custom section. Column does not except NULL value and DEFAULT value is '0'.
displayOrder int This column stores order in which application display section. Column does not except NULL value and DEFAULT value is '0'.
deleteflag int This column stores value 0 / 1 indicating if the section is deleted or not, this is soft delete. 0 - Not Deleted , 1 -Deleted.
hideSection int This column stores value 0 / 1 indicating if the section is hidden or not. 0 - Not Hidden, 1 -Hidden.
itemid int This column represent to id column. Map to the id column of items table
masterSectionid int This column represent id mapped to the id column of asc_pn_custom_sectionmaster table.
isStandard int This column stores value 0 / 1 indicating if the section is standard. 0 - Not standard, 1 -Standard.
abbreviation varchar This column used to store abbreviation for section. Column contains NULL value and default value is ''
isLinkExpression tinyint This column stores value 0 / 1 indicating if the section has linked expression. 0 - Not linked expression, 1 -Linked expression.
linkExpressionName varchar This column stores linked expression name if isLinkExpression logical flag is true. Column does not except NULL value and default value is ''.
defaultTab tinyint This column stores value 0 / 1 indicating if the section has default Tab. 0 - Not default Tab, 1 -default Tab.
