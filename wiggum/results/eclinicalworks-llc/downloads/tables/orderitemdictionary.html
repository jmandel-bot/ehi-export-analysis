EHI Export Schema
Table orderitemdictionary
Description This table stores information about order items, their attributes, and their relationships with devices.
Column DataType Description
id (Primary Key) decimal A unique identifier for each record in the table. It is a double data type and is auto-incremented. For example, 1, 2, 3, etc.
Itemid int An integer representing the identifier of the item. It has a default value of 0. For example, 101, 102, 103, etc.
name varchar A varchar(50) column that stores the name of the item. For example, "Item A", "Item B", "Item C", etc.
isPlotable int An integer(1) column indicating whether the item is plotable or not. It can have values like 0 (not plotable) or 1 (plotable).
pnDisplay smallint A smallint(3) column that stores a display order value for the item. It has a default value of 0. For example, 1, 2, 3, etc.
delteflag smallint A smallint(3) column that indicates whether the record is deleted or not. It has a default value of 0. For example, 0 (not deleted) or 1 (deleted).
Deviceid int An integer(11) column representing the identifier of the device associated with the item. It has a default value of 0. For example, 1001, 1002, 1003, etc.
xmlpath varchar A varchar(100) column that stores the XML path of the item. For example, "/path/to/itemA.xml", "/path/to/itemB.xml", etc.
DefaultAttrValue varchar A varchar(11) column that stores the default attribute value of the item. For example, "10", "20", "30", etc.
