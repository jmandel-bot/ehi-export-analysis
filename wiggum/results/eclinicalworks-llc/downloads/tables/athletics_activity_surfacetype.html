Table athletics_activity_surfacetype
Description This table gives the details of the Surface Type sections to the Activity Admin section of the Athletics Module. In this table, the application stores information about sub-category of surface depending on its types.
Column DataType Description
CODE (Primary Key) int Unique identifier for each surface type. Example: 1, 2, 3.
surfaceCode int This column stores surface code for surface type.
description varchar This column stores the description of surface type.
subCat varchar This column stores the sub category of the surface selected while creation like game, surface_cat etc.
subCatCode int Numeric code representing the subcategory of the surface type. Example: 201, 202, 203.
deleteFlag bit This column stores values 0 or 1 to indicate whether the surface type has been marked as deleted or not. 0 - If surface type is NOT marked as deleted, 1 - If surface type is marked deleted.
