Table occupation
Description This table stores information about different occupations, including whether the occupation involves driving or not, and if the occupation has been deleted.
Column DataType Description
id (Primary Key) int This is the primary key column, which stores a unique integer value for each occupation. It is automatically incremented as new occupations are added. Example values: 1, 2, 3, etc.
occupationName varchar This column stores the name of the occupation as a string with a maximum length of 256 characters. Example values: "Software Engineer", "Nurse", "Truck Driver", etc.
deleteFlag tinyint This column stores a tiny integer value (0 or 1) to indicate whether the occupation has been deleted or not. A value of 0 means the occupation is not deleted, while a value of 1 means the occupation is deleted. Example values: 0, 1.
isDriver tinyint This column stores a tiny integer value (0 or 1) to indicate whether the occupation involves driving or not. A value of 0 means the occupation does not involve driving, while a value of 1 means the occupation involves driving. Example values: 0, 1.
