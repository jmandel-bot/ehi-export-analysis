EHI Export Schema
Table inventory_vendorcredentials
Description This table stores only vendor details like name of the vendor etc.credentils are stored in inventory_vendorcredconfig with encrypted values
Column DataType Description
id (Primary Key) int Unique identifier for each vendor credential record. (Example: 1, 2, 3)
supplier_id int The identifier of the supplier associated with the vendor credentials. This could be a foreign key referring to the `id` column in a `suppliers` table. (Example: 101, 102, 103)
userName varchar The username for the vendor's account. (Example: 'vendor123', 'supplier456')
PASSWORD varchar The password for the vendor's account. (Example: 'p@ssw0rd', 's3cur3p@ss')
urlName varchar A descriptive name for the vendor's URL. (Example: 'Vendor Portal', 'Supplier Dashboard')
urlLink varchar The actual URL link to the vendor's website or portal. (Example: 'https://vendorportal.com', 'https://supplierdashboard.com')
serverDeployed varchar Information about the server where the vendor's system is deployed. (Example: 'AWS', 'Azure')
isdeleted int A flag indicating whether the vendor credential record has been deleted (1) or not (0). (Example: 0, 1)
createdBy int The identifier of the user who created the vendor credential record. This could be a foreign key referring to the `id` column in a `users` table. (Example: 1, 2, 3)
createdOn datetime The date and time when the vendor credential record was created. (Example: '2021-08-01 12:34:56')
updatedBy int The identifier of the user who last updated the vendor credential record. This could be a foreign key referring to the `id` column in a `users` table. (Example: 1, 2, 3)
updatedOn datetime The date and time when the vendor credential record was last updated. (Example: '2021-08-15 14:23:45')
NAME varchar The name of the vendor associated with the credentials. (Example: 'Vendor A', 'Supplier B')
productSupported tinyint A flag indicating whether the vendor supports the product (1) or not (0). (Example: 0, 1)
isVision tinyint A flag indicating whether the vendor is part of the Vision system (1) or not (0). (Example: 0, 1)
