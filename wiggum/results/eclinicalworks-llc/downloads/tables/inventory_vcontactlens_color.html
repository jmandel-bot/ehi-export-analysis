Table inventory_vcontactlens_color
Description This table stores information about various contact lens colors available in the inventory.
Column DataType Description
id (Primary Key) int Unique identifier for each contact lens color. Example: 1, 2, 3, etc.
NAME varchar The name of the contact lens color. Example: "Blue", "Green", "Hazel", etc.
color_code varchar A unique code representing the contact lens color. Example: "#0000FF" for Blue, "#008000" for Green, etc.
isdeleted tinyint A flag indicating whether the contact lens color record has been deleted or not. '0' means not deleted, '1' means deleted.
createdby int The identifier of the user who created the contact lens color record. This could be a foreign key referring to the `id` column in a `users` table.
createdon datetime The date and time when the contact lens color record was created. Example: "2021-09-01 12:34:56".
updatedby int The identifier of the user who last updated the contact lens color record. This could be a foreign key referring to the `id` column in a `users` table.
updatedon datetime The date and time when the contact lens color record was last updated. Example: "2021-09-15 14:23:45".
