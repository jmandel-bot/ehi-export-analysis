EHI Export Schema
Table acoprograms
Description This table stores a master list of programs available for patients to enroll in for different features, such as Chronic Care Management (CCM) and Transition Care Management (TCM).
Column DataType Description
id (Primary Key) int The column stores an id of the acoprograms. This is an auto incremental primary key of This column .
program varchar This column saves the program name created by the user. The type of program created is identified by the programtype column
deleteflag tinyint This column stores value 0/1 indicating this program is deleted. This is soft delete. 0 - Not Deleted and 1 -Deleted.
localflag tinyint This column stores value 0/1 indicating a local flag. By default, it is saved as 0.
ProgramType int This column stores the program type for which the program is created by the user. This refers to the id column from the programtype table.
programterm varchar This column stores the duration of the program created. Values that can be stored in varchar (e.g., 2 months, 1 year).
displayIndex int This column saves the incremental value called as display index. This value denotes the order in which the records are shown on the front end.
