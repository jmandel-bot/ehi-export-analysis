EHI Export Schema
Table structsocialhistory
Description This table stores structured data related to a patient's social history.
Column DataType Description
encounterid int This column stores id of patient's encounter for which structured data related to patient's Social History is being documented. This column refer to encounterid column in enc table.
catid int This column stores id of a category for Social History. This column refers to itemid column in items table. This column always has value 10619 which is an itemid for 'Social History'.
itemid int This column stores id for a structured item or screening being documented related to patient's Social History. i.e Alcohol Screening. This column refers to itemid column in items table.
detailid int This column stores id form each questions or item inside a structured screening. i.e Alcohol Screening related questions. This column refers to id column in structdatadetail table.
value varchar This column stores answer documented by end user for each question or item regarding patient's structured Social History.
valueid varchar This column stores id for values stored in corresponding "value" column. This column refer to optionid column in structdataoptions table.
notes varchar This column stores free text notes or pre-configured keywords selected by end for each corresponding item represented by id in detailid column.
chartdisplay tinyint This column stores values 0/1 to indicate whether record should be displayed on patient's chart (Progress Note) or not. 0 - Record should NOT be displayed on patient's chart, 1 - Record should be displayed on patient's chart.
displayindex int This column stores number to indicate order in which each record must be listed on User Interface.
importFromPortal int This column stores values 0/1 to indicate whether patient's Social History has been imported from Patient Portal or not. 0 - Social History is NOT imported from Patient Portal, 1 - Social History is imported from Patient Portal.
