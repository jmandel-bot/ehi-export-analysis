Table vision_master
Description This table stores information about various vision types, including names, codes, and details about their creation and updates.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. It is an integer value that auto-increments with each new entry. For example, 1, 2, 3, etc.
type int Integer value representing the type of vision. For example, 0, 1, 2, etc.
name varchar A varchar (255) column that stores the name of the vision type. For example, 'Night Vision', 'Thermal Vision', etc.
code varchar Not used anymore.
isdeleted tinyint A tinyint (3) unsigned column that indicates whether the record has been deleted or not. It has a default value of '0'. '0' means not deleted, and '1' means deleted.
createdby int An integer column that stores the identifier of the user who created the record. For example, 1, 2, 3, etc.
createdon datetime A datetime column that stores the date and time when the record was created. For example, '2021-09-01 12:34:56'.
updatedby int An integer column that stores the identifier of the user who last updated the record. For example, 1, 2, 3, etc.
updatedon datetime A datetime column that stores the date and time when the record was last updated. For example, '2021-09-02 14:56:32'.
