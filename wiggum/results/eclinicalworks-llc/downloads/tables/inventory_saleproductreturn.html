Table inventory_saleproductreturn
Description This table stores information about product returns related to sales in the inventory system.
Column DataType Description
id (Primary Key) int Unique identifier for each product return entry. (e.g., 1, 2, 3)
Product_id int identifier for the product being returned. Foreign key relation to the `products` table's `id` column. (e.g., 101, 102, 103)
SaleReturn_id int identifier for the sale return transaction. Foreign key relation to the `sale_returns` table's `id` column. (e.g., 201, 202, 203)
SaleProduct_id int identifier for the sale product associated with the return. Foreign key relation to the `sale_products` table's `id` column. (e.g., 301, 302, 303)
Price decimal Price of the returned product. (e.g., 10.0000, 15.5000, 20.2500)
Qty decimal Quantity of the returned product. (e.g., 1.0000, 2.0000, 3.0000)
TaxAmt decimal Tax amount applied to the returned product. (e.g., 0.5000, 1.0000, 1.5000)
UniqueString varchar Unique string associated with the product return entry. (e.g., "abc123", "def456", "ghi789")
IsDeleted tinyint Flag to indicate if the product return entry is deleted (1) or not (0). (e.g., 0, 1)
UpdatedBy int identifier for the user who updated the product return entry. Foreign key relation to the `users` table's `id` column. (e.g., 1001, 1002, 1003)
UpdatedOn datetime Date and time when the product return entry was last updated. (e.g., '2021-01-01 12:34:56', '2021-02-02 23:45:01')
DiscountPerLine decimal Discount applied per line item for the returned product. (e.g., 0.0000, 1.0000, 2.0000)
DiscReason_id int identifier for the discount reason. Foreign key relation to the `discount_reasons` table's `id` column. (e.g., 401, 402, 403)
DiscountType int identifier for the type of discount applied. Foreign key relation to the `discount_types` table's `id` column. (e.g., 1, 2, 3)
DiscValue decimal Discount value applied to the returned product. (e.g., 0.0000, 0.5000, 1.0000)
stock_id int identifier for the stock associated with the returned product. Foreign key relation to the `stocks` table's `id` column. (e.g., 501, 502, 503)
encid int Encounter identifier associated with the product return entry. (e.g., 10001, 10002, 10003)
CountryTax decimal Country tax applied to the returned product. (e.g., 0.0000, 0.1000, 0.2000)
CityTax decimal City tax applied to the returned product. (e.g., 0.0000, 0.0500, 0.1000)
CosmeticTax decimal Cosmetic tax applied to the returned product. (e.g., 0.0000, 0.0300, 0.0600)
OtherTax decimal Other tax applied to the returned product. (e.g., 0.0000, 0.0200, 0.0400)
CountryTaxCode varchar Tax code for the country tax applied. (e.g., "CT001", "CT002", "CT003")
CityTaxCode varchar Tax code for the city tax applied. (e.g., "CIT001", "CIT002", "CIT003")
CosmeticTaxCode varchar Tax code for the cosmetic tax applied. (e.g., "COS001", "COS002", "COS003")
OtherTaxCode varchar Tax code for the other tax applied. (e.g., "OTH001", "OTH002", "OTH003")
Reason varchar Reason for the product return. (e.g., "Damaged", "Wrong item", "Changed mind")
Note varchar Additional notes related to the product return. (e.g., "Box was damaged", "Customer prefers different color", "Return within 30 days")
Currency varchar Currency used for the product return transaction. (e.g., "USD", "EUR", "GBP")
