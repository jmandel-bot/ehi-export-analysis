Table allergy_extract_billing_notes
Description This table stores billing notes related to allergy extracts for each patient.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
PATIENTid int identifier for the patient associated with the billing note. This could be a foreign key referring to the `id` column in a `patients` table. Example: 101, 102, 103.
NOTES varchar Text containing the billing notes related to the allergy extract for the patient. Example: "Patient has a mild reaction to the extract", "Insurance covers 80% of the cost".
LAST_UPDATED datetime time stamp indicating when the record was last updated. Example: '2021-09-15 14:30:00'.
LAST_UPDATED_BY varchar Name of the user who last updated the record. Example: 'John Doe', 'Jane Smith'.
LAST_UPDATED_BYid int identifier for the user who last updated the record. This could be a foreign key referring to the `id` column in a `users` table. Example: 201, 202, 203.
