EHI Export Schema
Table pe_muscular_exams_new
Description This table stores information about different muscular exams, their categories, and some additional properties.
Column DataType Description
id (Primary Key) int This is a primary key which increments itself automatically.
examName varchar This column stores the new muscular exam name as defined by the user.
catid int This column stores a category identifier of new muscular exam. It refers to the itemid column of items table.
DispIndex int This column stores the index value of the examination name according to which it has to be displayed.
deleteflag int This column stores soft delete data. 1: Soft Deleted;0: Not Deleted.
isCountable tinyint This column stores a flag value which indicates whether measurement counter should be displayed on joint table screen. 0: Would not display measurement counter. 1: would display measurement counter.
