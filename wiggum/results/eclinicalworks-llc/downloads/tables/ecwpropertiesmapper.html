EHI Export Schema
Table ecwpropertiesmapper
Description This table stores the mapping information between properties and their corresponding eCW community properties.
Column DataType Description
id (Primary Key) int Table stores community mapping of HPI and Examination properties and Social History, Physical Examination, Preventive Medicine line items. id is auto increment and primary key of the table.
propid int This column stores id of the property or item mapped to Community item. Refers to column propid (primary key) of properties table if the tblname column value is 'properties', else if column itemid (primary key) of items table if the tblname column value is 'items'
ecwPropid int This column stores Community item id mapped to specific property or item. Refers ecwPropid (primary) column of ecwproperties table.
tblname varchar This column stores table name used to refer propid column. Please check propid description for more details.
