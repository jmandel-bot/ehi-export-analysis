EHI Export Schema
Table prisma_patient_identifier
Description This table stores the patient identifier information, which includes various identifier types and their corresponding values for each patient from document received from external sources.
Column DataType Description
ID (Primary Key) int Unique identifier for each record in the prisma_patient_identifier table. Example: 1, 2, 3.
PATIENT_DATA_ID bigint Foreign key that refers to the patient_data table's ID column. It represents the unique identifier of the patient to whom the identifier belongs. Example: 101, 102, 103.
SYSTEM_URI varchar The URI of the system which represent the identiifier value's system. It can be NULL if not available. Example: "http://example.com/identifier-system", NULL.
IDENTIFIER_VALUE varchar The actual value of the patient identifier. It cannot be NULL and has a default value of an empty string. Example: "123456789", "A987654321".
IDENTIFIER_CODE varchar The code representing the type of identifier. It can be NULL if not available. Example: "SSN", "MRN", NULL.
IDENTIFIER_DISPLAY varchar A human-readable display of the identifier. It can be NULL if not available. Example: "Social Security Number", "Medical Record Number", NULL.
IDENTIFIER_SYSTEM varchar The name of the system where the identifier is used. It can be NULL if not available. Example: "National ID System", "Hospital Management System", NULL.
IDENTIFIER_TEXT varchar A textual description of the identifier. It can be NULL if not available. Example: "Patient's social security number", "Patient's medical record number", NULL.
