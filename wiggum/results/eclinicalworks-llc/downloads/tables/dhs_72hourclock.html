Table dhs_72hourclock
Description This table stores is used to store details of the inmate for the 72 hours discharge from one facility to another.
Column DataType Description
Id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
AlienNo varchar Unique identifier for the detainee being tracked. Example: 'A12345', 'A67890'.
clockEnabled tinyint Indicates if the 72-hour clock is enabled (1) or disabled (0). Example: 0, 1.
xferFrom int The identifier of the facility location from which the detainee is being transferred.
xferTo int The identifier of the facility location from which the detainee is being transferred.
clockStartTime datetime The date and time when the 72-hour clock starts. Example: '2021-09-01 12:00:00', '2021-09-02 15:30:00'.
clockStopTime datetime The date and time when the 72-hour clock stops. Example: '2021-09-04 12:00:00', '2021-09-05 15:30:00'.
deleteflag tinyint Indicates if the record has been deleted (1) or not (0). Example: 0, 1.
patientId int The unique identifier of the patient associated with the alien. Example: 101, 102.
