EHI Export Schema
Table occhealth_encinfo
Description This table stores information related to occupational health encounters, including employer details, costs, and contact information.
Column DataType Description
id (Primary Key) int This is the primary key for the table, which is an integer value that uniquely identifies each record. It is an auto-incrementing value. Example: 1, 2, 3, etc.
employerName varchar Name of the employer associated with the encounter. Example: "Acme Corp", "Tech Solutions".
encounterId int This column stores encounter Id from enc table
certificatEmpId varchar This column stores employer id which referes to insurance table.
orderByName varchar Name of the person who placed the order. Example: "John Doe", "Jane Smith".
orderByEmail varchar Email address of the person who placed the order. Example: "john.doe@example.com", "jane.smith@example.com".
selfPay tinyint Indicates whether the payment was made by the individual (1) or by the employer (0). Example: 0, 1.
phoneno varchar Phone number of the person who placed the order. Example: "+1-555-123-4567", "+1-555-987-6543".
exVatCost decimal Cost of the encounter excluding VAT (Value Added Tax). Example: 100, 150, 200.
incVatCost decimal Cost of the encounter including VAT (Value Added Tax). Example: 120, 180, 240.
orderNo int Unique identifier for the order. Example: 101, 102, 103.
certificateAction varchar It stores selected values for Certificate action for Appt notes
createdOver varchar The method used to create the record, such as "Phone", "Email", or "Online". Example: "Phone", "Email".
