Table corrections_infirmary_console
Description This table stores information about patients admitted to the infirmary in a correctional facility, including their admission and discharge dates, diagnosis, and follow-up details.
Column DataType Description
id (Primary Key) int This is primary key column.
bedtype int This column store the Bed Type used in infirmary admission.
ID_Corrections_infirmary_beds int This column store the Bed id used in infirmary admission.
ID_patients_correctional int This column store the patient's correctional id used in infirmary admission. It is referrring to patients_correctional.id column.
Infirmary_Admission_Date datetime This column store the admission date of patient's infirmary admission.
Followup text This column store the Follow up status of patient's infirmary admission.
Provider_Discharge_Date datetime This column store the date and time of Provider's discharge in infirmary admission.
Physical_Discharge_Date datetime This column store the date and time of Physical discharge in infirmary admission.
Admitting_Diagnosis int This column store the admitting diagnosis in infirmary admission.
Notes text This column store all notes added by provider during infirmary admission.
patientid int This column store patient's id in infirmary admission. It refers to patients.pid column.
AttnPrId int This column store attending provider's id in infirmary admission. It refers to users.uid column.
InfEncID int This column store encounter created in infirmary admission. It refers to enc.encounterid column.
FollowupId int This column store Followup itemid in infirmary admission. It refers to items.itemid column.
FollowupDate datetime This column store date and time of Followup added/updated in infirmary admission.
FacilityId int This column store facilityid mapped to infirmary admission.
