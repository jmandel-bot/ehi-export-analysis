EHI Export Schema
Table vision_vergence
Description This table stores information about vision vergence tests performed during an eye examination.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer value. It uniquely identifies each record in the table. Example: 1, 2, 3, etc.
Encounterid int identifier for the specific encounter during which the vision vergence test was performed. This can be a foreign key referring to the `encounterid` column in the `enc` table. Example: 101, 102, 103.
ExamType int Integer value representing the type of eye examination performed. Example: 1 (Routine Checkup), 2 (Follow-up), 3 (Emergency).
DistBlur varchar A string value representing the distance at which the patient's vision starts to blur during the test. Example: "20 feet", "30 feet", "40 feet".
DistBreak varchar A string value representing the distance at which the patient's vision breaks or loses focus during the test. Example: "25 feet", "35 feet", "45 feet".
DistRecv varchar A string value representing the distance at which the patient's vision recovers or regains focus during the test. Example: "22 feet", "32 feet", "42 feet".
NearBlur varchar A string value representing the near distance at which the patient's vision starts to blur during the test. Example: "10 inches", "12 inches", "14 inches".
NearBreak varchar A string value representing the near distance at which the patient's vision breaks or loses focus during the test. Example: "8 inches", "10 inches", "12 inches".
NearRecv varchar A string value representing the near distance at which the patient's vision recovers or regains focus during the test. Example: "6 inches", "8 inches", "10 inches".
