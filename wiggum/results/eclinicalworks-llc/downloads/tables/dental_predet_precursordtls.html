EHI Export Schema
Table dental_predet_precursordtls
Description This table stores information about dental pre-determination requests and their associated details.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
encid int Foreign key referring to the encounter id from enc table. Example: 101, 102, 103.
Patientid int Foreign key referring to the patient id from user table. Example: 1001, 1002, 1003.
preDetDate datetime Date when the pre-determination request was made. Example: '2021-09-01', '2021-09-02', '2021-09-03'.
STATUS varchar Status of the pre-determination request. Example: 'OPEN', 'CLOSED', 'PENDING'.
billToProvider int Foreign key referring to the provider id from users who will be billed. Example: 201, 202, 203.
insid int Foreign key referring to the insurance id from insurance table. Example: 301, 302, 303.
facid int Foreign key referring to the facility id from edi_facilities table. Example: 401, 402, 403.
preDetXmlid int Foreign key referring to the pre-determination XML id from dental_predet_printxml table. Example: 501, 502, 503.
dentalFormXML text XML data containing the dental form information. Example: '...'.
preDetNumber varchar Unique pre-determination number for the request. Example: 'PDN-001', 'PDN-002', 'PDN-003'.
tpid int Foreign key referring to the treatment plan id from dentaltreatmentplan table. Example: 601, 602, 603.
procCodes varchar Comma-separated list of procedure codes associated with the request. Example: 'D1110,D1120,D1208'.
notes varchar Additional notes or comments about the pre-determination request. Example: 'Patient needs extra care', 'Follow up in 3 months'.
deleteFlag int Flag indicating if the record is deleted (1) or not (0). Example: 0, 1.
isInActive tinyint Flag indicating if the record is inactive (1) or active (0). Example: 0, 1.
