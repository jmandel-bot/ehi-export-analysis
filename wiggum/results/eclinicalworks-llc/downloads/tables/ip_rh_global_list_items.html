Table ip_rh_global_list_items
Description This table appears to store items that belong to a global list, with each item having a description, an association with a list, and flags for deletion and activity status.
Column DataType Description
Id (Primary Key) int Serves as the primary key of the table. This column is unique to each item and does not refer to any other table.
ItemDescription varchar Stores the description of the list item. This column can contain null values, indicating that an item description is not mandatory.
ListId int Indicates the list to which the item belongs. Typically, this column would reference an Id column in another table that stores list information.
DeleteFlag tinyint Indicates whether the item has been marked for deletion. A non-zero value indicates that the item is considered deleted, even if not physically removed from the table.
Inactive tinyint Indicates whether the item is currently inactive. A non-zero value indicates that the item is inactive and should not be used or displayed in the UI.
