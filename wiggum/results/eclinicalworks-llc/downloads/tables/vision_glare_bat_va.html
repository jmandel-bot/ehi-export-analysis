EHI Export Schema
Table vision_glare_bat_va
Description This table stores information about vision glare test results for both eyes under different light conditions.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer value. It uniquely identifies each record in the table. Example: 1, 2, 3, etc.
ENCOUNTERid int Unique identifier for the encounter associated with the vision glare test. This can be a foreign key referring to the `enc` table's `encounterid` column. Example: 101, 102, 103.
LOWOD varchar Vision glare test result for the right eye (OD) under low light conditions. Example: "20/40", "20/60", "20/80".
LOWOS varchar Vision glare test result for the left eye (OS) under low light conditions. Example: "20/40", "20/60", "20/80".
MEDIUMOD varchar Vision glare test result for the right eye (OD) under medium light conditions. Example: "20/30", "20/50", "20/70".
MEDIUMOS varchar Vision glare test result for the left eye (OS) under medium light conditions. Example: "20/30", "20/50", "20/70".
HIGHOD varchar Vision glare test result for the right eye (OD) under high light conditions. Example: "20/20", "20/40", "20/60".
HIGHOS varchar Vision glare test result for the left eye (OS) under high light conditions. Example: "20/20", "20/40", "20/60".
COMMENTS text Additional comments or notes related to the vision glare test. Example: "Patient experienced difficulty in low light conditions", "Test results improved with corrective lenses".
