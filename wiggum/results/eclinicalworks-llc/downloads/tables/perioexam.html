EHI Export Schema
Table perioexam
Description This table stores information about periodontal examinations for patients.
Column DataType Description
PerioExamNum (Primary Key) bigint Unique identifier for each periodontal examination record. Example: 1, 2, 3.
PatNum bigint This column stores identifier number of a patient, for whom the appointment is created. It refers to column pid from patients table. It is defined as NOT NULL constraint.
ExamDate datetime Date when the periodontal examination was conducted. Example: '2021-09-01', '2021-09-02', '2021-09-03'.
ProvNum bigint This column stores identifier number of a provider while appointment is created. It refers to column UID from user table. It is defined as NOT NULL constraint.
delflag tinyint A flag indicating whether the periodontal examination record has been deleted (1) or not (0). Example: 0, 1.
Encounterid bigint identifier for the encounter associated with the periodontal examination. This is a foreign key referring to the `Encounterid` column in the `encounter` table. Example: 2001, 2002, 2003.
CreatedDateTime datetime Date and time when the periodontal examination record was created. Example: '2021-09-01 10:00:00', '2021-09-02 11:00:00', '2021-09-03 12:00:00'.
addedbyutility tinyint This column stores value 0 / 1 indicating this Perio Exam is created via Perio data recovery utility or not. Default Value = 0 which indicated this exam is not created via Perio data recovery utility, Value = 1 which indicated this exam is created via Perio data recovery utility.
truserid int This column stores identifier number of a login user who create or modified Perio Exam. It refers to column UID from user table. It is defined as NOT NULL constraint.
