EHI Export Schema
Table labreports
Description This table stores information about laboratory reports for various medical encounters.
Column DataType Description
Encounterid int This column stores the unique identifier for a specific medical encounter. It is an integer value and cannot be NULL. Example: 12345.
Itemid int This column stores the unique identifier for a specific laboratory test item. It is an integer value and cannot be NULL. Example: 6789.
Propid int This column stores the unique identifier for a specific property of the laboratory test item. It is an integer value and cannot be NULL. Example: 9876.
Value varchar This column stores the value of the specific property for the laboratory test item. It is a string with a maximum length of 255 characters and can be NULL. Example: Positive.
