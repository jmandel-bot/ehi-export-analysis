EHI Export Schema
Table externalalerts
Description This table stores the externally provided alerts associated with each patient, such as healow Insights alerts.
Column DataType Description
id (Primary Key) int This column is an auto incremented primary key.
camid int This column is not used anymore.
patientid int This column stores patient's ID for whom the alert is created. This column refers column UID from users.
measureid varchar This column stores the ID of selected measure to create an external alert. This column refers to column externalalertmeasureid from externalalert_details table.
isCompliantPerSource bit This column stores a value of 0 or 1. 1 is when the measure is complaint as per the source, 0 is otherwise.
compliantUntilDate datetime This column stores date time till the alert is complaint. The format used is YYYY-MM-DD HH:MM:SS. This value comes from payers while fetching Member Insights.
lastComplianceValueCategory varchar This value comes from payers while fetching Member Insights. (No information on data type, size and default value)
lastComplianceValueItem varchar This value comes from payers while fetching Member Insights. (No information on data type, size and default value)
lastComplianceValue varchar This value comes from payers while fetching Member Insights. (No information on data type, size and default value)
lastComplianceDate datetime This column stores a date till which the measure was complaint as per source. The format used is YYYY-MM-DD HH:MM:SS.
receivedFromSourceDate datetime This column stores a date when the source sends an alert to EMR. The format used is YYYY-MM-DD HH:MM:SS.
deleteFlag bit This column stores a value of 0 or 1. 1 is when the alert is marked as deleted, and 0 is otherwise.
payorname varchar This column stores Healow Insight payor name associated with a patient.
assignTo int This column stores the ID of the person to whom the alert is assigned. This column refers column UID from user table.
status varchar This column stores the status of the alert.
actiondetails varchar This column stores an action taken by an end user or source for this alert.
encounterid int This column stores an encounter ID for which this alert is created. This column refers to column encounterid from enc table.
emrgUID varchar This column stores a row identifier for this alert.
encstatuscode varchar This column stores a status code of the encounter while an alert is created.
enclockstatus smallint This column stores a value of 0 or 1. A 1 is for a locked encounter, and a 0 is for an unlocked encounter.
patientaccnumber varchar This column stores an account number of the patient from the patientid column. This column refers to column controlNumber from the patients table.
providernpi varchar This column stores the NPI (National Provider Identifier) of a provider from the Providerid column. This column refers to column NPI from the doctors table.
measurename varchar This column stores the name of the measure selected for this alert.
encounterdate datetime This column stores the created date of an encounter for this alert. This column refers column date from enc table.
providertin varchar This column stores the Tax identification Number (TIN) of the provider from the Providerid column. This column refers to column Taxid from the doctors table.
providerid int This column stores the ID of the provider to whom this alert is assigned. This column refers to column UID from user table.
patientdob varchar This column stores patient's date of birth. The format used is MM/DD/YYYY.
MatchedIns_SubscriberNo varchar This column stores subscriber ID given by an external source.
facilitytin varchar This column stores Federal Tax ID of the facility.
payorid varchar This column stores the ID of the external source.
createdby int This column stores the ID of the user who created this alert. This column refers column UID from user table.
createdbyname varchar This column stores the name of the user who created this alert.
createddate datetime This column stores the date on which this alert was created. The format used is YYYY-MM-DD HH:MM:SS.
updatedby int This column stores the ID of the user who modifies this alert. This column refers to column UID from the user table.
updatedbyname varchar This column stores the name of the user who modified this alert.
updateddate datetime This column stores a date on which this alert was modified. The format used is YYYY-MM-DD HH:MM:SS.
datagUID varchar Unique identifier for the data, stored as a string (e.g., "DG001", "DG002", "DG003").
