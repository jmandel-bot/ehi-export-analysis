Table prisma_careplan_contributor
Description This table stores information about contributors to a care plan, including their practitioner and care team IDs.
Column DataType Description
ID (Primary Key) int This is the primary key for the table and is an auto-incrementing integer value. It uniquely identifies each care plan contributor. Example: 1, 2, 3, etc.
CAREPLAN_ID int This is a foreign key that refers to the `prisma_careplan_details` table's `id` column. It represents the care plan to which the contributor is associated. Example: 101, 102, 103, etc.
PRACTITIONER_ID int This is an integer value that represents the ID of the practitioner who is contributing to the care plan. It can be NULL if there is no specific practitioner associated with the care plan. Example: 1001, 1002, 1003, etc.
CARETEAM_ID int This is an integer value that represents the ID of the care team that is contributing to the care plan. It can be NULL if there is no specific care team associated with the care plan. Example: 2001, 2002, 2003, etc.
