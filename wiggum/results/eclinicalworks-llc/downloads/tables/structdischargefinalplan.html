EHI Export Schema
Table structdischargefinalplan
Description This table stores structured data related to a patient's Discharge Final Plan for an encounter.
Column DataType Description
encounterId int This column stores the encounter id for which structured DischargeFinalPlan data have been documented. This column refers to column encounterid from enc table.
catId int This column stores the itemid of the category for which structured DischargeFinalPlan data have been documented. This column refers to column itemid from items table.
itemId int This column stores the id of the Symptom displayed in the User Interface. This column refers to column propid of properties table.
detailId int This column stores the id of the structured question associated with the DischargeFinalPlan of the patient. This column refers to column id from structdatadetail table.
value text This column stores value of an answer to the structured question documented by end user. This column refers to column name from structdataoptions table.
valueId varchar This column stores the id of an answer selected by the end user while documenting structured questions. This column refers to column optionId from structdataoptions table.
notes text This column stores the note added by the end user for the respective structured question.
chartdisplay tinyint This column stores 0 / 1 , 0 : not to display on chart and 1 : display on chart.
displayIndex tinyint Stores order in which structured questions are listed on the user interface.
