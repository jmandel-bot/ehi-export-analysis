Table surescript_professionalservicecode
Description This table stores professional service codes and their corresponding descriptions.
Column DataType Description
Fieldid (Primary Key) int This is the primary key column, which contains a unique integer value for each record in the table. It is automatically incremented as new records are added. Example values: 1, 2, 3, etc.
Code varchar This column stores the professional service code, which is a 5-character long string. Example values: 'A1234', 'B5678', 'C9012', etc.
Description varchar This column stores a brief description of the professional service code, which is a string with a maximum length of 150 characters. Example values: 'General medical examination', 'Dental cleaning', 'Physical therapy session', etc.
