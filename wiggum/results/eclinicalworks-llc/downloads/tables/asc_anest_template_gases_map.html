EHI Export Schema
Table asc_anest_template_gases_map
Description This table stores the mapping between anesthesia templates and gases items, along with their default status and deletion flag.
Column DataType Description
id (Primary Key) int Unique identifier for each mapping record. Example: 1, 2, 3.
templateid int identifier for the anesthesia template. This is a foreign key that refers to the `id` column in the `asc_anest_template_master` table. Example: 101, 102, 103.
gasesItemid int identifier for the gases item. This is a foreign key that refers to the `id` column in the `asc_ecw_gases_standardlist` table. Example: 201, 202, 203.
isDefault tinyint A boolean flag indicating if the gases item is the default for the anesthesia template.NOT NULL DEFAULT '0'. 1 represents true (default), and 0 represents false (not default). Example: 0, 1.
deleteFlag tinyint A boolean flag indicating if the mapping record is deleted.NOT NULL DEFAULT '0'. 1 represents true (deleted), and 0 represents false (not deleted). Example: 0, 1.
