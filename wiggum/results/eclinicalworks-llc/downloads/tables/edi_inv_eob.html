EHI Export Schema
Table edi_inv_eob
Description This table stores EOB(Explanation of Benefits) invoice payments information including payment details, adjustments, and other financial information.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. (e.g., 1, 2, 3)
PaymentDetailid int id referes to edi_paymentdetail table
InvCptid int id refres to edi_inv_cpt table
Allowed decimal Allowed amount for the invoice line item. (e.g., 100.00, 200.00, 300.00)
deduct decimal Deductible amount for the invoice line item. (e.g., 10.00, 20.00, 30.00)
coins decimal Coinsurance amount for the invoice line item. (e.g., 5.00, 10.00, 15.00)
MemResp decimal Member responsibility amount for the invoice line item. (e.g., 50.00, 100.00, 150.00)
Paid decimal Paid amount for the invoice line item. (e.g., 90.00, 180.00, 270.00)
Adjustment decimal Adjustment amount for the invoice line item. (e.g., 2.00, 4.00, 6.00)
WithHeld decimal Withheld amount for the invoice line item. (e.g., 1.00, 2.00, 3.00)
MsgCode varchar Message code associated with the EOB. (e.g., 'A01', 'B02', 'C03')
eraAdjustment decimal ERA (Electronic Remittance Advice) adjustment amount for the invoice line item. (e.g., 0.50, 1.00, 1.50)
closingid int identifier for the closing associated with the EOB. (e.g., 10001, 10002, 10003)
lineTaxAmt decimal Line tax amount for the invoice line item. (e.g., 0.25, 0.50, 0.75)
CalculatedAdjustment decimal Calculated adjustment amount for the invoice line item. (e.g., 1.25, 2.50, 3.75)
createdBy int identifier for the user who created the record. (e.g., 1, 2, 3)
createdDateTime datetime Date and time when the record was created. (e.g., '2021-01-01 12:00:00', '2021-02-01 12:00:00', '2021-03-01 12:00:00')
modifiedBy int identifier for the user who last modified the record. (e.g., 1, 2, 3)
modifiedDateTime datetime Date and time when the record was last modified. (e.g., '2021-01-02 12:00:00', '2021-02-02 12:00:00', '2021-03-02 12:00:00')
timestamp datetime time stamp for the record. (e.g., '2021-01-01 12:00:00', '2021-02-01 12:00:00', '2021-03-01 12:00:00')
RecreatedFlag tinyint Flag indicating whether the record has been recreated (1) or not (0). (e.g., 0, 1)
