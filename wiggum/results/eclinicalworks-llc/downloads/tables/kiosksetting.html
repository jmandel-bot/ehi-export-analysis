EHI Export Schema
Table kiosksetting
Description This table stores settings and configurations for kiosks, including information about consent forms, content types, and other related details.
Column DataType Description
name varchar This column stores value of consent form name.
value text This column stores HTML content for the consent form in English.
id (Primary Key) int The unique identifier for each kiosk setting. Example: 1.
createdOn varchar The date and time when the kiosk setting was created. Example: "2021-08-25 14:30:00".
createdBy int This column stores id of consent form creator. This column refers to UID column from user table.
modifiedBy int This column stores id of consent form modifier. This column refers to UID column from user table.
modifiedOn varchar The date and time when the kiosk setting was last modified. Example: "2021-08-26 10:15:00".
del_flag tinyint A flag indicating whether the kiosk setting has been deleted. Example: 0 (not deleted), 1 (deleted).
activeConsent char A flag indicating whether the consent form is active. Example: '0' (inactive), '1' (active).
displayIndex int The display order of the kiosk setting. Example: 3.
deletedBy int The user id of the person who deleted the kiosk setting. Example: 103.
deletedOn varchar The date and time when the kiosk setting was deleted. Example: "2021-08-27 09:45:00".
facilityid int The id of the facility associated with the kiosk setting. Example: 5.
isArchieved tinyint A flag indicating whether the kiosk setting is archived. Example: 0 (not archived), 1 (archived).
recurringForm char A flag indicating whether the consent form is recurring. Example: '0' (non-recurring), '1' (recurring).
value_spanish text The value of the kiosk setting in Spanish. Example: "80%".
consentFormType int This column stores value to specify consent form type. 0 - General, 1 - Voice/Text, 2 - ROI consent, 3 - PHI disclosure.
letterid_en int This column stores an id of pdf letter of English consent form. This column refers to id column from letterTemplates table.
letterid_es int This column stores an id of pdf letter of Spanish consent form. This column refers to id column from letterTemplates table.
contentType char This column stores content type of consent form: 0 - Text and 1 - PDF or Letter
visitTypeBased tinyint A flag indicating whether the kiosk setting is based on visit type. Example: 0 (not visit type-based), 1 (visit type-based).
