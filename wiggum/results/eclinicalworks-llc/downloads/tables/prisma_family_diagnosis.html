EHI Export Schema
Table prisma_family_diagnosis
Description This table stores information about family medical diagnoses, including details about the diagnosis, age of the family member at the time of diagnosis, and whether the diagnosis caused death.
Column DataType Description
id (Primary Key) bigint A unique identifier for each record in the table. Example: 1, 2, 3.
familyHxDetailId bigint A foreign key that refers to the `id` column in the `prisma_familyhistory_details` table. It represents the family history detail associated with this diagnosis. Example: 101, 102, 103.
typeCode varchar A short code representing the type of diagnosis. Example: "TYP1", "TYP2", "TYP3".
type varchar A more descriptive text of the diagnosis type. Example: "Type 1 Diabetes", "Type 2 Diabetes", "Type 3 Diabetes".
diagnosisCode varchar A short code representing the specific diagnosis. Example: "DIA1", "DIA2", "DIA3".
diagnosis varchar A more descriptive text of the specific diagnosis. Example: "Diabetes Mellitus", "Hypertension", "Asthma".
diagnosisText varchar Additional text information about the diagnosis. Example: "Controlled with medication", "Severe case", "Mild symptoms".
recordedTime varchar The time when the diagnosis was recorded, stored as a string. Example: "2021-08-01 12:34:56", "2021-08-02 14:23:45".
ageValue varchar The age of the family member at the time of diagnosis, stored as a string. Example: "25", "30", "45".
ageUnit varchar The unit of measurement for the age value. Example: "years", "months", "days".
ageText varchar A more descriptive text of the age at the time of diagnosis. Example: "25 years old", "30 months old", "45 days old".
diagnosisCausedDeathCode varchar A short code representing whether the diagnosis caused death. Example: "Y", "N", "U" (where Y = Yes, N = No, U = Unknown).
diagnosisCausedDeath varchar A more descriptive text of whether the diagnosis caused death. Example: "Yes", "No", "Unknown".
diagnosisCausedDeathText varchar Additional text information about whether the diagnosis caused death. Example: "Died due to complications", "Survived with treatment", "Unknown cause of death".
