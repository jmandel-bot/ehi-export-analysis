Table occhealth_encresultlog
Description This table stores the occupational health encounter result logs, which include information about the encounter, package, service, result status, certificate restrictions, validity, expiration, and modification details.
Column DataType Description
id (Primary Key) int This is an auto incremental primary key column of This column .
encounterid int This column stores the encounter id.This Column refers to encounterid column in enc table.
packageid int This column stores the package id.This Column refers to id column in occhealth_packages table.
serviceid int This column stores the Service id.This Column refers to id column in occhealth_services table.
resultStatus varchar The status of the result, stored as a string. Example: 'Completed', 'Pending', 'Failed'
certificateRestrictionid int This column stores the certificate restriction id.This Column refers to certificateRestrictionid column in certificateRestriction table.
validFor int The number of days the result is valid for. Example: 30, 60, 90.
expiredOn datetime The date and time when the result expires. Example: '2022-12-31 23:59:59', '2023-01-31 23:59:59'.
addedBy int This column stores the user id .This Column refers to UID column in users table.
deletedBy int This column stores the user id .This Column refers to UID column in users table.
deleteFlag tinyint A flag indicating whether the record has been deleted (1) or not (0). Example: 0, 1
modifiedDate datetime The date and time when the record was last modified. Example: '2022-11-30 12:34:56', '2022-12-01 14:56:23'
deletedDate datetime The date and time when the record was deleted. Example: '2022-12-31 23:59:59', '2023-01-31 23:59:59'.
