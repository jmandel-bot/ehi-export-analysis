Table edi_pmd_msgcode
Description This table stores information related to payment and invoice codes, including various cascodes and numcodes.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
paymentid int identifier for the payment associated with the record. Foreign key relation with the `edi_payments` table's `id` column. Example: 101, 102, 103.
invoiceid int identifier for the invoice associated with the record. Foreign key relation with the `edi_invoice` table's `id` column. Example: 201, 202, 203.
cascode varchar First cascode value associated with the record. Example: 'CAS001', 'CAS002', 'CAS003'.
cascode2 varchar Second cascode value associated with the record. Example: 'CAS004', 'CAS005', 'CAS006'.
cascode3 varchar Third cascode value associated with the record. Example: 'CAS007', 'CAS008', 'CAS009'.
cascode4 varchar Fourth cascode value associated with the record. Example: 'CAS010', 'CAS011', 'CAS012'.
numcode varchar First numcode value associated with the record. Example: 'NUM001', 'NUM002', 'NUM003'.
numcode2 varchar Second numcode value associated with the record. Example: 'NUM004', 'NUM005', 'NUM006'.
numcode3 varchar Third numcode value associated with the record. Example: 'NUM007', 'NUM008', 'NUM009'.
numcode4 varchar Fourth numcode value associated with the record. Example: 'NUM010', 'NUM011', 'NUM012'.
customcode varchar First custom code value associated with the record. Example: 'CUS001', 'CUS002', 'CUS003'.
customcode2 varchar Second custom code value associated with the record. Example: 'CUS004', 'CUS005', 'CUS006'.
cascode5 varchar Fifth cascode value associated with the record. Example: 'CAS013', 'CAS014', 'CAS015'.
cascode6 varchar Sixth cascode value associated with the record. Example: 'CAS016', 'CAS017', 'CAS018'.
numcode6 varchar Sixth numcode value associated with the record. Example: 'NUM016', 'NUM017', 'NUM018'.
numcode5 varchar Fifth numcode value associated with the record. Example: 'NUM013', 'NUM014', 'NUM015'.
