EHI Export Schema
Table asc_anesthesia_plan_options
Description This table stores anesthesia plan options against an individual anesthesia plan. It is a plan option master table.
Column DataType Description
id (Primary Key) int This is a primary key which increments itself automatically.
anestPlanDetailid int This column stores anesthesia plan detail identifier. It is a foreign key and refers to id column of asc_anesthesia_plan_details table.
value varchar This column stores Anesthesia plan options value.
displayIndex tinyint This column stores display index which represents the display order of the record when grouped by certain record identifier.
deleteFlag tinyint This column stores soft delete data. 0: Not deleted; 1: Soft Deleted
