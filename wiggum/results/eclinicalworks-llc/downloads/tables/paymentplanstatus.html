EHI Export Schema
Table paymentplanstatus
Description This table stores stores list of status for payment plan. (PENDING, OPEN, APPROVED, DENIED etc.)
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer. It uniquely identifies each payment plan status record. Example: 1, 2, 3, etc.
statusid int This is an integer that represents the status id of the payment plan. It has a default value of '0'. Example: 1 (Active), 2 (Inactive), 3 (Expired), etc.
STATUS varchar This column stores list of status for payment plan. (PENDING, OPEN, APPROVED, DENIED etc.) There is no UI to create new status.
createdate datetime This is a date column that stores the date when the payment plan status record was created. Example: '2021-09-01', '2021-09-02', etc.
deleteflag tinyint This is a tinyint(2) column that indicates whether the payment plan status record has been deleted or not. It has a default value of '0'. Example: 0 (Not Deleted), 1 (Deleted).
