Table kidmed
Description This table stores information about children's medical, vision, hearing, dental, nutritional, developmental, abuse, psychological, and speech care, as well as referrals for suspected conditions.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table (e.g., 1, 2, 3).
encounterid int It stores encounter id associated with enc table.
