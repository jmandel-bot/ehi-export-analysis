Table surescript_requestlog
Description This table stores information about prescription requests and their related data, such as patient, doctor, and pharmacy details.
Column DataType Description
Reqid (Primary Key) int This column is an auto incremented Primary Key for This column .
RxType varchar This column stores type of the medication for which rx response is sent.
Messageid varchar This column stores an identifying string for the refill response sent.
RelatedMessageid varchar This column stores the message id of the previous related transaction's message id.
SentTo varchar In case of incoming request this column stores SPI of the provider else it stores NCPDPid for the outgoing reponses.
SentFrom varchar In case of incoming request this column stores NCPDPid of the pharmacy else it stores SPI of the provider for the outgoing reponses.
SentTime varchar This column stores the date on which the refill response is sent. The format used is YYYYMMDD.
MessageText varchar This column is no longer used.
RxReferenceNo varchar This column stores Rx reference record from the pharmacy of the medication prescribed.
AttemptNo int This column stores the attempt number of the refill response sent by the Provider and hence this is set to 1 always .
LastAttempttimestamp datetime This column stores the current time stamp of the last attempt done by the Provider. The format used is YYYY-MM-DD HH:MM:SS.
statusDesc varchar This column stores status of the refill response sent.
bPostedToDoc char This column is no longer used.
Encounterid int This column stores an id of the encounter for which refill is responded. This column refers column encounterid from enc table.
Patientid int This column stores an id of the patient. This column refers UID column from user table.
Fileid varchar This column stores the file id generated which is a combination of patientid and rx order number.
Patient_FName varchar This column stores first name of the patient for whom refill response is sent.
Patient_LName varchar This column stores last name of the patient for whom refill response is sent.
Patient_MName varchar This column stores middle name of the patient for whom refill response is sent.
DateOfBirth varchar This column stores date of birth of the patient for whom refill response is sent. The format used is YYYYMMDD.

State varchar This column stores state of the patient for whom refill response is sent.

patient_primaryphone varchar This column stores patient's primary phone number for whom refill response is sent. This could include '-' in it.
Patient_CellPhone varchar This column stores patient's cell phone number for whom refill response is sent. This could include '-' in it.
Doctorid int This column stores an id of the Doctor who sends refill reponse. This column refers to column UID from users.
SPI varchar This column stores Surescripts Provider identifier (SPI) number for the provider from Doctorid column who is activated for eprescritions.

doc_ssn varchar This column stores the SSN of the Provider who sends out the refill response.

docA_FName varchar This column stores first name of the provider's assistant who sends the refill response.
docA_LName varchar This column stores last name of the provider's assistant who sends the refill response.
Speciality varchar This column stores speciality of the provider who sends the refill response. This column refers column specialityname from doctor_specialities table.
SpecialityCode char This column stores speciality code of the specialty from speciality column. This column refers column specialitycode from doctor_specialities table.
doc_AddressLine1 varchar This column stores address of the provider who sends the refill response.
doc_AddressLine2 varchar This column stores address of the provider who sends the refill response.
doc_City varchar This column stores city of the provider who sends the refill response.
doc_State varchar This column stores state of the provider who sends the refill response.

doc_Pagerno varchar This column stores pager number of the provider who sends the refill response.

pharmacyname varchar This column stores name of the Pharmacy .

Pharmacy_Phone varchar This column stores phone number of the pharmacy documented in PharmacyName column. This ccould have '-' hyphen in it.

Pharmacy_Email varchar This column stores emal id of the pharmacy documented in PharmacyName column.

PhA_FName varchar This column stores Pharmacist's assistant's first name from the received incoming request.

XMLData text This column stores entire xml used for sending a refill response.
