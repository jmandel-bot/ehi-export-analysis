Table occhealth_hs_result_names
Description This table stores the occupational health surveillance result names along with their modification details.
Column DataType Description
id (Primary Key) int This is the primary key column, containing a unique integer value for each record in the table. For example: 1, 2, 3, etc.
hsResultName varchar This column stores the name of the health surveillance result as a string with a maximum length of 25 characters. For example: "Risk Assessment", "Safety Audit", etc.
modifiedBy int This column stores the user id (integer) of the person who last modified the record. It has a foreign key relation with the `users` table's `UID` column. For example: 101, 102, 103, etc.
modifiedDate datetime This column stores the date and time when the record was last modified. For example: "2021-09-01 12:34:56", "2021-09-02 14:23:45", etc.
delFlag tinyint This column stores a tiny integer value (1 or 0) to indicate whether the record has been deleted or not. '0' means the record is not deleted, and '1' means the record is deleted. For example: 0, 1.
