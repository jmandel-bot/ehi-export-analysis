EHI Export Schema
Table edi_278_cpt
Description This table stores information aboutCPT (Current Procedural Terminology) codes related to EDI (Electronic Data Interchange) 278 transactions, which are used for healthcare services authorization requests and responses.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
SAid int Foreign key referring to the Service Authorization (SA) table's primary key. It represents the service authorization associated with the CPT code. Example: 101, 102, 103.
Referralid int Foreign key referring to the Referral table's primary key. It represents the referral associated with the CPT code. Example: 201, 202, 203.
CPTid int Foreign key referring to the CPT table's primary key. It represents the specific CPT code related to the EDI 278 transaction. Example: 1001, 1002, 1003.
UnitsRequested int The number of units requested for the CPT code in the EDI 278 transaction. Example: 5, 10, 15.
UnitsApproved int The number of units approved for the CPT code in the EDI 278 transaction. Example: 4, 8, 12.
deleteFlag tinyint A flag indicating if the record has been deleted (1) or not (0). Example: 0, 1.
