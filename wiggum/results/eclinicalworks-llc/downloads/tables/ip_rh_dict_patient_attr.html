EHI Export Schema
Table ip_rh_dict_patient_attr
Description This table stores the patient attributes.
Column DataType Description
Id (Primary Key) int This is an auto-incremented primary key column in this table. . It is used to uniquely identify the patient attribute. The data type is int as per business requirement and the column cannot be NULL.
attrName varchar This column stores the attribute name. The datatype is varchar(256) as the expected value will not exceed 256 characters and the column cannot be NULL.
modiBy int Stores the user Id of the person who last modified the patient attribute. It refers to uid column in users table. Datatype is int. The default value for this column is NULL.
modiTime datetime Stores the date and time the patient attribute was last modified. The datetime data type allows for precise storage of date and time values. The default value is NULL.
deleteFlag tinyint A flag to indicate if the patient attribute is deleted(1) or not(0). The data type is tinyint and the default value is 0. The column cannot be NULL.
displayorder int This column stores a number to indicate in which order patient attributes should be listed on user facing screen. The data type is smallint as compared to int is because the expected patient attributes display will not exceed the limitation provided by the smallInt datatype. The column is NOT NULL and the default value is 0.
