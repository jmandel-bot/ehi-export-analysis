EHI Export Schema
Table prisma_advance_directive_details
Description This table stores information about advance directive details, including entry details, author information, and reference text.
Column DataType Description
id (Primary Key) bigint Unique identifier for each record in the table. Example: 1, 2, 3.
sectionId bigint Foreign key that refers to the `id` column in the `prisma_ccd_document_section` table. It represents the section to which the advance directive detail belongs. Example: 101, 102, 103.
entryId varchar Unique identifier for each entry in the advance directive details. Example: "E001", "E002", "E003".
entryCode varchar Code representing the type of entry in the advance directive details. Example: "C001", "C002", "C003".
entryCodeName varchar Name of the entry code in the advance directive details. Example: "Do Not Resuscitate", "Living Will", "Healthcare Proxy".
entryStatus varchar Status of the entry in the advance directive details. Example: "Active", "Inactive", "Completed".
referenceText varchar Text that provides additional information or reference for the entry in the advance directive details. Example: "See attached document", "Refer to medical record", "Patient's request".
authorId varchar Unique identifier for the author of the advance directive detail. Example: "A001", "A002", "A003".
authorName varchar Name of the author of the advance directive detail. Example: "John Doe", "Jane Smith", "Alice Johnson".
authorAddressLine1 varchar First line of the author's address. Example: "123 Main St", "456 Oak St", "789 Elm St".
authorAddressLine2 varchar Second line of the author's address (optional). Example: "Apt 4B", "Suite 300", "Building 5".
authorCity varchar City where the author resides. Example: "New York", "Los Angeles", "Chicago".
authorState varchar State where the author resides. Example: "NY", "CA", "IL".
authorPostalCode varchar Postal code of the author's address. Example: "10001", "90001", "60601".
authorCountry varchar Country where the author resides. Example: "USA", "Canada", "UK".
