EHI Export Schema
Table playerstatuses
Description This table is used to store the meta data of current, projected and game day statuses
Column DataType Description
id (Primary Key) int This is the primary key - auto incremented value of This column
status varchar This stores the value of the status that needs to be displayed
deleteFlag tinyint This column is a boolean flag indicating whether the record has been deleted or not. It is a tiny integer with a default value of 0 (not deleted). Example: 0 (not deleted), 1 (deleted).
Code varchar This column stores a unique code associated with the player's status. It is a variable character field with a maximum length of 25 characters. Example: 'ACT', 'INACT', 'SUSP', etc.
type int stores the type of status(1- Current status, 2- projected status, 3- game day status)
