EHI Export Schema
Table surescript_eprescription
Description This table stores electronic prescription information, including patient, doctor, and pharmacy details, as well as prescription details and message-related data.
Column DataType Description
Reqid (Primary Key) int This column is an auto incremented Primary Key for This column denoting the eprescription id.
RxType varchar This column stores type of the medication for which eprescription is sent out. This is always set to NewRx value.
Messageid varchar This column stores an unique identifying string for the ePrescription sent out.
RelatedMessageid varchar This column stores the message id of the previous related transaction's message id. In case of new Rx generated independently this will be assigned a default value.
SentTo varchar This column stores an id of a vendor to whom the erprescription is sent. This column refers column NCPDPid from Pharmacy table.
SentFrom varchar This column store SPI of the Provider who sends out an eprescription.
SentTime varchar This column stores the date on which the eprescription is sent. The format used is YYYYMMDD.
MessageText varchar This column is no longer used.
StatusDesc varchar This column stores status of the eprescription sent. success - if it is successfully sent to the pharmacy, failed - if it fails or pending - intermediate status till the eprescrition is successfully sent or failed.
AttemptNo int This column stores the Number of attempts made to send the eprescription out (e.g., 1, 2, 3).
LastAttempttimestamp datetime This column stores the current time stamp of the last attempt done by the Provider. The format used is YYYY-MM-DD HH:MM:SS.MMM.
Patientid int This column stores an id of the patient for whom the eprescription is sent. This column refers to column UID from user table.
encounterid int This column stores an id of the encounter on which the medication is added for which eprescription is sent. This column refers to column encounterid from enc table.
Patient_FName varchar This column stores Patient's first name for whom the eprescription is sent. This column refers ufname column from user table.
Patient_LName varchar This column stores Patient's last name for whom the eprescription is sent. This column refers ulname column from user table.
Patient_MName varchar This column stores Patient's middle name for whom the eprescription is sent. This column refers uminitial column from user table.
DateOfBirth varchar This column stores Patient's date of birth for whom the eprescription is sent. The format used is YYYYMMDD.
Sex varchar This column stores gender of the patient for whom eprescription is sent.
Ssn varchar This column stores SSN of the patient for whom eprescription is sent. This column refers column ssn column from user table.
AddressLine1 varchar This column stores the address of the patient for whom eprescription is sent.
AddressLine2 varchar This column stores the address of the patient for whom eprescription is sent.
City varchar This column stores the city name of the patient for whom eprescription is sent.
State varchar This column stores the State name of the patient for whom eprescription is sent.
ZipCode varchar This column stores zip code of the patient for whom eprescription is sent.
patient_primaryphone varchar This column stores patient's primary phone number for whom eprescription is sent. This could include '-' in it.
Patient_CellPhone varchar This column stores patient's cell phone number for whom eprescription is sent. This could include '-' in it.
Patient_Fax varchar This column stores patient's fax number for whom eprescription is sent. This could include '-' in it.
Patient_Email varchar This column stores patient's email address for whom eprescription is sent.
Doctorid int This column stores an id of the Doctor who sends an ePrescription. This column refers to column UID from users.
SPI varchar This column stores Surescripts Provider identifier (SPI) number for the provider from Doctorid column who is activated for eprescritions.
StateLicenseNumber varchar This column stores state license number of the provider who sends the prescription.
doc_ssn varchar This column stores the SSN of the Provider who sends out the eprescription.
DEANo varchar This column stores the assigned DEA number of the Provider who sends out the eprescription.
doc_FName varchar This column stores first name of the provider who sends the prescription.
doc_LName varchar This column stores last name of the provider who sends the prescription.
doc_MName varchar This column stores middle name of the provider who sends the prescription.
doc_suffix varchar This column stores assigned suffix of the provider who sends the prescription.
Speciality varchar This column stores speciality of the provider who sends the precription. This column refers column specialityname from doctor_specialities table.
SpecialityCode char This column stores speciality code of the specialty from speciality column. This column refers column specialitycode from doctor_specialities table.
doc_AddressLine1 varchar This column stores address of the provider who sends the prescription.
doc_AddressLine2 varchar This column stores address of the provider who sends the prescription.
doc_City varchar This column stores state of the provider who sends the prescription.
doc_State varchar This column stores state of the provider who sends the prescription.
doc_Zipcode varchar This column stores zip code of the provider who sends the prescription.
doc_Primaryphone varchar This column stores primary phone number of the provider who sends the prescription.
doc_Pagerno varchar This column stores pager number of the provider who sends the prescription.
doc_Mobilephone varchar This column stores mobile number of the provider who sends the prescription.
Pharmacyid int This column stores id of the pharmacy to which eprescription is sent. This column refers column pmcid from pharmacy table.
NCPDPid varchar This column stores unique, national identifier number assigned to the pharmacies to which eprescription is sent. This column refers column NCPDPid from pharmacy table.
PharmacyName varchar This column stores name of the Pharmacy to which eprescription is sent.
Pharmacy_AddressLine1 varchar This column stores address of the pharmacy.
Pharmacy_City varchar This column stores city name of the pharmacy documented in PharmacyName column.
Pharmacy_State varchar This column stores State name of the pharmacy documented in PharmacyName column.
Pharmacy_Zipcode varchar This column stores zip code of the pharmacy documented in PharmacyName column.
Pharmacy_Phone varchar This column stores phone number of the pharmacy documented in PharmacyName column. This ccould have '-' hyphen in it.
Pharmacy_Fax varchar This column stores fax number of the pharmacy documented in PharmacyName column. This ccould have '-' hyphen in it.
Pharmacy_Email varchar This column stores emal id of the pharmacy documented in PharmacyName column.
DrugDescription varchar This column stores description of the drug for which eprescription is sent.
ProductCode varchar This column stores ndc code of the product which comes from the associated drug database. This column refers column ndc_code from OldRxMain table.
ProductCodeQualifier varchar This column stores the qualifier for the ndc code from product code column.
DosageForm varchar This column stores code of the formulation for the drug from drugdescription column. .This column refers nctlcode from surescript_ncpdpcodes table.
Strength varchar This column stores the prescribed strength of the medication.
StrengthUnits varchar This column stores the unit of strength of the prescribed medication.
DrugDBCode varchar This column stores id of a medication in the associated drug database sent for the sent eprescription.
DrugDBCodeQualifier varchar This column stores the qualifier for Drug database associated. MD- Medispan, - Multum
Frequency varchar Frequency of drug administration (e.g., 'Every 8 hours').
duration varchar Duration of the prescription (e.g., '10 days').
Quantity varchar Quantity of the prescribed drug (e.g., '30').
Take varchar Instructions for taking the prescribed drug (e.g., 'Take 1 tablet every 8 hours').
Refills int This column stores the maximum number of refills for the prescribed medication by the Provider.
SubsMedAllowed int This column stores a flag 0 or 1. 1 - If substitution is is allowed, 0 - otherwise.
comments varchar This column stores additional comments from Provider about ePrescription to the Pharmacy.
XMLData text This column is to hold the entire eprescription data in a xml format
XMLResponseData text This column is to store the success / failed response received for the eprescription sent in a xml format.
RefillsQualifier char This column stores qualifier for a refill. R - refill , PRN - as per the need.
bAsynMsgPostedToDoc char Indicates if an asynchronous message was posted to the doctor (e.g., 'Y' for Yes, 'N' for No).
bVerified char Indicates if the prescription has been verified (e.g., 'Y' for Yes, 'N' for No).
ScriptVersion varchar This column stores values denoting Surescript script standard used. Old version used: 4x, 10x, Current version: 2017x.
SentBy varchar Name of the person who sent the prescription (e.g., 'Nurse C').
Rxhubid varchar RxHub identifier for the prescription (e.g., 'RXHUB12345').
RouteSource varchar This column stores the information about where the eprescription is routed through: surescript or it could be a interface.
Addressed tinyint This column stores flag 0 or 1. 1 - when the eprescription is failed and user takes an action for the failed eprescription to address it. 0 - otheriwse.
AddressedBy varchar This column stores username of the logged in user who addresses the failed eprescription.
PrescriberOrderNo varchar This column stores unique order number which is a combination of encounterid,APUID and itemid. This is used to match drug when application receive refill request. .
SentByid int This column stores an id of the provider from SentBy column. This column refers to UID column from user table.
IpAddress varchar This column stores a IPv4 address of the system used to send out a prescription.
isVoiceMsgSent tinyint This column stores flag 0 or 1. 1- if voice message is sent. 0 - otheriwse.
isTextMsgSent tinyint This column stores flag 0 or 1. 1- if text message is sent. 0 - otheriwse.
rxOrderNo varchar This column stores order number for the set of medications prescribed electronically. This column refers column rxOrderNo from oldrxmain table.
DEASchedule varchar DEA schedule of the prescribed drug (e.g., 'II', 'III', 'IV').
TxMode varchar This column stores the transmission mode of the ePrescription (e.g., 'eRx', 'Sent as Fax').
structuredCodifiedSigSent tinyint This column stores the structured and codified sig sent in an ePrescription
StatusCode varchar This column stores the status code of the ePrescription (e.g., '010', '900', '600').
