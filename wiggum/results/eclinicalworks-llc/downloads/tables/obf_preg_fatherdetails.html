EHI Export Schema
Table obf_preg_fatherdetails
Description This table stores information about the father associated with a pregnancy record.
Column DataType Description
id (Primary Key) int This column is auto-incremented primary key in This column .
Pregid int This column stores unique id created for each pregnancy of a patient. This column refers to pregid column in obf_pregnancy table.
Patientid int This column stores id of a patient for whom a pregnancy has been created in the system. This column refers to UID column in user table.
FirstName varchar This column stores first name of the father of the baby.
mInitial varchar This column stores middle name initial of the father of the baby.
LastName varchar This column stores last name of the father of the baby.
DOB varchar This column stores Date of Birth of the father of the baby.
Phone varchar This column stores phone number of the father of the baby.
CellPhone varchar This column stores cell phone number of the father of the baby.
Occupation varchar This column stores occupation of the father of the baby as documented by provider/staff.
Ethnicity varchar This column stores Ethnicity of the father of the baby.
Race varchar This column stores race of the father of the baby.
