Table vision_eligibility_policy_info
Description This table stores information about vision eligibility policies for individuals, including their personal details, policy information, and addresses.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1
eligid int Unique identifier for the eligibility policy. Example: 12345
LastName varchar Last name of the individual. Example: "Smith"
FirstName varchar First name of the individual. Example: "John"
mi varchar Middle initial of the individual. Example: "A"
dob datetime Date of birth of the individual in datetime format. Example: "1985-08-15 00:00:00"
gender varchar Gender of the individual, with possible values being 'male', 'female', or 'unknown'. Example: "male"
policy varchar Policy number or identifier. Example: "POL12345"
address varchar First line of the individual's address. Example: "123 Main St"
address1 varchar Second line of the individual's address (e.g., apartment number). Example: "Apt 4B"
city varchar City of the individual's address. Example: "New York"
state varchar State of the individual's address. Example: "NY"
zipcode varchar Zip code of the individual's address. Example: "10001"
country varchar Country of the individual's address. Example: "USA"
Insuredid varchar Unique identifier for the insured individual. Example: "INS12345"
Medicareid varchar Unique identifier for the individual's Medicare policy, if applicable. Example: "MED98765"
deleteflag int Flag indicating if the record has been deleted (1) or not (0). Example: 0
status tinyint Status of the policy, represented as a tiny integer. Example: 1
rel varchar Relationship of the individual to the policyholder. Example: "spouse"
