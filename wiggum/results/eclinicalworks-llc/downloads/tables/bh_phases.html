Table bh_phases
Description This table stores information about different phases in a business process.
Column DataType Description
id (Primary Key) int This is the primary key column, which stores a unique integer value for each phase. Example: 1, 2, 3, etc.
phaseName varchar This column stores the name of the phase as a string. The names must be unique. Example: "Planning", "Execution", "Review", etc.
delflag tinyint This column stores a tiny integer value (either 0 or 1) to indicate whether the phase has been deleted or not. A value of 0 means the phase is not deleted, and a value of 1 means the phase is deleted. Example: 0, 1.
