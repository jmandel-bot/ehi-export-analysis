EHI Export Schema
Table careplan_notes
Description This table stores the notes related to care plans, along with the user who created the note and the time it was last modified.
Column DataType Description
id (Primary Key) int Unique identifier for each care plan note. Example: 1, 2, 3, etc.
CarePlanid int identifier for the care plan that the note is related to. This is a foreign key that refers to the `id` column in the `careplan` table. Example: 10, 25, 30, etc.
Userid int identifier for the user who created the note. This is a foreign key that refers to the `id` column in the `users` table. Example: 5, 15, 20, etc.
Notes text Text content of the care plan note. Example: "Patient needs to increase daily water intake.", "Schedule a follow-up appointment in 2 weeks.", etc.
ModifiedTime datetime Date and time when the care plan note was last modified. Example: '2021-08-25 14:30:00', '2021-09-01 10:15:00', etc.
