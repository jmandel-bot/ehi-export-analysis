EHI Export Schema
Table dental_overridingfees
Description This table stores information about overriding fees for specific patients andCPT (Current Procedural Terminology) groups in a dental practice.
Column DataType Description
id (Primary Key) int • This column will store the id of the dental_overridingfees • The datatype of the column would be INT to accommodate many different override fees of procedures data that can be added. • This would be unique and auto incremented and so PRIMARYKEY constraint is added
patientid int • This column stores the id of the patient. • This is a reference to the column pid of table patients.
CPTGroupid int • This column stores cpt code group id entered on UI. • This is a reference to the column id of table groupcodes.
overrideamt decimal • This column stores overridden fee amount applied on cpt group procedures exist in treatment plans. • The datatype of the column would be decimal as it can have 0 to 9999999.99 values.
