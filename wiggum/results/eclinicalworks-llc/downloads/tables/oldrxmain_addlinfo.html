Table oldrxmain_addlinfo
Description This table stores additional information related to old prescriptions.
Column DataType Description
OldRxId (Primary Key) int The unique identifier for the old prescription. This is the primary key for the table. Example: 12345
EncounterId int The unique identifier for the encounter associated with the old prescription. This is a foreign key that refers to the `encounterID` column in the `enc` table. Example: 67890
Rxorderno varchar The order number of the prescription. Example: 'RX123456'
DoctorsFlag tinyint This columns stores Rx status like stop, start, increase decrease etc.
PharmacyNCPDP varchar The National Council for Prescription Drug Programs (NCPDP) identifier for the pharmacy. Example: '1234567'
DURMsg varchar The Drug Utilization Review (DUR) message associated with the prescription. Example: 'Potential drug interaction'
DURServiceCode varchar The service code for the Drug Utilization Review (DUR). Example: 'DUR123'
Deleteflag tinyint A flag indicating whether the record has been deleted (1) or not (0). Example: 0
DAW tinyint The Dispense as Written (DAW) code, which indicates whether a generic substitution is allowed (0) or not (1). Example: 0
AdditionalInstructions varchar Additional instructions related to the prescription. Example: 'Take with food'
RxNotes varchar Notes related to the prescription. Example: 'Patient has a history of allergies'
pharmacyId int The unique identifier for the pharmacy associated with the prescription. This is a foreign key that refers to the `pmcid` column in the `pharmacy` table. Example: 98765
