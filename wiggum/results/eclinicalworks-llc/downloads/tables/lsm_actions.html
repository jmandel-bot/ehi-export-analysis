EHI Export Schema
Table lsm_actions
Description This table stores information about providers actions, such as creating and assigning patient-specific tasks, scheduling surgery, calling a PCP for a referral, etc.
Column DataType Description
Id (Primary Key) int This is an auto incremented primary key of This column . This id uniquely identify each action being created by end user.
PatientId int This column stores the ID of the patient for whom action is created. This column refers to pid column in patients table.
AssignedToId int This column store the ID of the Staff / Provider to whom the action is assigned. This column refers to uid column in user table.
Subject varchar This column stores the value from the pre-defined list. From subcategory of an action type.
Status tinyint This column stores the value from the pre-defined list. Not Started / In Progress / Completed / Overdue / Cancelled.
CreationDate datetime This column stores the date and time when the action is created.
StartDate datetime This column stores the date and time when the action is started.
DueDate datetime This column stores the date and time when the action is due.
CreatedBy varchar This column stores the name of the Staff / Provider who created the action.
Priority tinyint This column stores the priority of the action. 1- Low / 2- Normal / 3 - High.
Notes text This column stores free text note of the action.
IsRecurrent tinyint This column stores the value 0 / 1 indicating if the action is recur at regular intervals. 0 - Not recurrence , 1 - Recurrence
Frequency int This column stores the frequency of the recurrence of the action.
LastDue datetime This column stores the date time on which the previous date the recurring action was due.
LastDone datetime This column stores the date time when the most recent date the recurring action was actually completed.
DateCompleted datetime This column stores the date time on which the action is completed.
ActionType int This column stores the Id of the action type. This column refers to id column in lsm_actionsType table.
UseAttachment tinyint This column stores value 0 /1 indicating to use existing attachments for recurrent action. 0 - Do not use existing attachments , 1 - Use existing attachments.
FrequencyType varchar This column stores the frequency type of the recurrent action. Hours / Days / Weeks / Months / Years.
FacilityId int This column store the id of the facility of the action. This columns refers to column id from edi_facilities.
RecurrenceRange_Option int This column stores value 0 / 1 / 2 indicating action recurrent range. 0 - No End date, 1 - End After Number of occurrence , 2 - End by date.
RecurrenceRange_EndAfter int This column stores value of the recurrent range after the action completes.
RecurrenceRange_EndDate datetime This column stores the end date of the recurrent action.
EncounterId int This column stores the ID of an encounter for which action is created. This column refers column encounterId from enc table.
TemplateName varchar This column stores the template name which is merged for the action created.
delflag tinyint This column stores value 0 / 1 indicating if the action is deleted, this is soft delete. 0 - Not Deleted , 1 -Deleted
hl7id varchar This column is not in use.
modifiedDate datetime This column stores the date time of the action modified.
ptacoid int Column refers to ptacoprograms.id
providerId int This column store the ID of the Provider to whom the action is assigned.
stateId int This column store the ID of the state with the provider is associated.
payerId int This column stores the ID of the payor insurance information.
isEscalated tinyint This column stores the value 0/1 indicating if the action is escalated. 0 - Not escalated, 1 - Escalated.
