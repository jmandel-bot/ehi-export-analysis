EHI Export Schema
Table form_icdcodes
Description This table is used in the Allergy Formulation tab. This table stores information about different ICD codes and their descriptions, along with their associated queueId and item ID.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3, etc.
QUEid int This column stores queueid for Formulation where icdcodes are being mapped. This refers to id column in antigen_recipe_queue table
CODENAME varchar The ICD code name. Example: 'A00.0', 'A00.1', 'A00.2', etc.
DESCRIPTION varchar A brief description of the ICD code. Example: 'Cholera due to Vibrio cholerae 01, biovar cholerae', 'Cholera due to Vibrio cholerae 01, biovar eltor', etc.
ITEMid int The item id associated with the ICD code.
