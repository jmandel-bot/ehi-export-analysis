Table ip_rh_ptorders_intd
Description This table is used to store interdisciplinary orders schedule status healthcare setting.
Column DataType Description
id (Primary Key) int This column serves as the primary key for the table, providing a unique identifier for each row or record.
orderId int ip_rh_ptorders is a table that stores order details, this column linked to the orderId of that table Constraints: NOT NULL. This constraint ensures that each record must have an order ID; it cannot be left blank.
scheduleStatus int Stores the status of each order's schedule. ip_rh_orderstatus_master is a table that maps status codes to their meanings, this column refers to that table. The values this column can take, and their meanings would need to be documented for clarity in ip_rh_orderstatus_master.
