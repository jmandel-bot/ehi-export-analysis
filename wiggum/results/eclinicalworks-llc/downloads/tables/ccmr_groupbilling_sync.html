EHI Export Schema
Table ccmr_groupbilling_sync
Description This table stores information related to group billing synchronization for medical encounters.
Column DataType Description
encounterid int This is an integer column that stores the unique identifier for each medical encounter. Example values: 1, 2, 3, etc.
deleteFlag tinyint This is a tinyint column that indicates whether the record has been marked for deletion or not. It has a default value of '0'. Example values: 0 (not marked for deletion), 1 (marked for deletion).
