Table pronouns_options
Description This table stores the different pronoun options and their descriptions.
Column DataType Description
id (Primary Key) int This is the primary key for the table, and it is an auto-incrementing integer. It uniquely identifies each pronoun option. Example: 1, 2, 3, etc.
pronouns_name varchar This is a varchar(150) column that stores the name of the pronoun option. Example: "He/Him", "She/Her", "They/Them", etc.
pronouns_description varchar This is a varchar(200) column that stores a brief description of the pronoun option. Example: "Masculine pronouns", "Feminine pronouns", "Gender-neutral pronouns", etc.
pronouns_code varchar This is a varchar(50) column that stores an optional code for the pronoun option. It can be used for internal reference or categorization. Example: "MASC", "FEM", "NEUT", etc. This column can also be NULL if no code is provided.
deleteflag tinyint This is a tinyint(2) column with a default value of '0'. It is used to mark a record as deleted without actually removing it from the table. A value of '0' means the record is active, while a value of '1' means the record is deleted.
