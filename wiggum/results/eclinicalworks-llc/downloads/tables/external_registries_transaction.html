EHI Export Schema
Table external_registries_transaction
Description This table stores external registries (Healtheintent) transaction ID and responses of patients.
Column DataType Description
id (Primary Key) int This column is primary key id of transaction records for external registries
patientid varchar This column holds patient id of transaction records for external registries
trUserid varchar This column holds user id of transaction records for external registries
externalRegistriesResponse text This column holds JSON response of transaction records for external registries
transactionid varchar This column holds system generated transaction id gUID of transaction records for external registries
SourceName varchar This column holds source name (i.e. Healtheintent) of transaction records for external registries
deletedFlag tinyint This is a tinyint column that indicates whether the transaction record has been deleted or not. It cannot be NULL and has a default value of '0'. A value of '0' means the record is not deleted, and a value of '1' means the record is deleted.
createdDate datetime This column represented created date of the record
lastUpdatedDate datetime This column represented record last updated date.
