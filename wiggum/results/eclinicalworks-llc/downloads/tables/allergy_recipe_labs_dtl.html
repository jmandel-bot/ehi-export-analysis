Table allergy_recipe_labs_dtl
Description This table stores detailed information about allergy recipes in a laboratory, including their concentrations, diluents, and expiration dates.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
QUEUE_id int identifier for the queue that the allergy recipe belongs to. Example: 101, 102, 103.
BOTTLE_id int identifier for the bottle containing the allergy recipe. Example: 1001, 1002, 1003.
CONCENTRATION_id int identifier for the concentration level of the allergy recipe. This may have a foreign key relation to another table containing concentration details. Example: 10, 20, 30.
IS_COMPLETED tinyint A boolean flag indicating whether the allergy recipe has been completed (1) or not (0). Example: 0, 1.
EXPIRATIONDATE datetime The expiration date of the allergy recipe. Example: '2022-12-31', '2023-01-01'.
LOTNO varchar The lot number of the allergy recipe. Example: 'L12345', 'L67890'.
DILUENT_TYPE varchar The type of diluent used in the allergy recipe. Example: 'Saline', 'Glycerin'.
DILUENT_CONC varchar The concentration of the diluent used in the allergy recipe. Example: '1%', '2%'.
DILUENT_LOTNO varchar The lot number of the diluent used in the allergy recipe. Example: 'DL123', 'DL456'.
DILUENT_EXPDATE datetime The expiration date of the diluent used in the allergy recipe. Example: '2022-06-30', '2022-07-31'.
