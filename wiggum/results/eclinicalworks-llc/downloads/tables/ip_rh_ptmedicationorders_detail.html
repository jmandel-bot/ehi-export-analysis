EHI Export Schema
Table ip_rh_ptmedicationorders_detail
Description This table store the details of order like formulation, strength that are placed for a patient after being finalized.
Column DataType Description
Id (Primary Key) int Unique identifier for each interaction record.
ptmedorderId int This column is used to store ptmedorderId id and is foreign key references medorderId column ip_rh_ptmedicationorders table.
ipitemid int This column is used to store item id and is foreign key references itemID column of items table.
order_strength varchar This column is used to store the order strength like 1 MG, 100MG etc.
order_strength_UOM varchar This column is used to store units like ml/hr, ml/min.
order_formulation varchar This column is used to store order formulation like Tablet, IV fluid.
orderDispense varchar This column is used to store order dispense like 1, 1.5 ml etc.This data will be populated from dispenseSize column of ip_rh_drugformulary so added same datatype and size.
order_ndc_code varchar This column is used to store order NDC code. This data will be populated from ndcCode column of mmw_drug_pack table of medispan database so added same datatype and size.
order_ddid int This column is used to store order ddid value. This data will be populated from ddid column of mmw_drug_disp table of medispan database so added same datatype and size.
deleteflag tinyint This column is used to store in delete flag it to show whether order is deleted or not. 1 - represents order is deleted and 0 - represents order is not deleted Default value is 0 ensures order is not deleted.
DrugFormularyId int This column is used to store drug formulary id and is foreign key references id column ip_rh_drugformulary table.
ivvolume int This column is to store IV volume like 100, 200 etc. This data will be populated from volume column of ip_rh_drugformulary so added same datatype and size.
Ivdiluentid int This column is used to store diluent values like 25, 26 etc.
ingredientCount int This column is used to store ingredients count e.g. 2.
isAdditive tinyint This column is used to store additive or not.1 - represents additive and 0 - represents not additive
formularyDispenseSize varchar This column is used to store dispense sizes like 1ml, 1 ea. This data will be populated from dispenseSize column of ip_rh_drugformulary so added same datatype and size.
dispenseQty varchar This column is used to store dispense quantity like 200mg.
actualDispense varchar This column is used to store actual dispense like 200mg.
isDiluent tinyint This column is used to store if diluent or not.1 - represents diluent 0 - represents not diluent Default value is 0 for order record.
displayIndex tinyint This column is used to store display index or not.1 - represents to display and 0 - represents not to display Default value is 0 for order record
formularyDose varchar This column is used to store formulary dose for every order like 30ml. This data will be populated from doseSize column of ip_rh_drugformulary so added same datatype and size.
formularyDoseUOM varchar This column is used to store formulary dose units for every order like ml, mg. This data will be populated from doseSizeUom column of ip_rh_drugformulary so added same datatype and size.
orderDose varchar This column is used to store order dose units for every order like 5ml, 50mg
formularyDispenseUOM varchar This column is used to store formulary dispense units for every order like ml, mg. This data will be populated from dispenseSizeUom column of ip_rh_drugformulary so added same datatype and size.
ivvolume_uom varchar This column is used to store ivvolume units for every order like ml, mg. This data will be populated from volumeuom column of ip_rh_drugformulary so added same datatype and size.
rxnorm varchar This column is used to store Rx norm for orders.
