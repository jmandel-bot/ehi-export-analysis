Table prisma_socialhistory_author
Description This table stores information about the authors of social history records, including their personal and organizational details.
Column DataType Description
id (Primary Key) bigint Unique identifier for each author record (e.g., 1, 2, 3).
socialhistoryDetailId bigint The identifier of the social history detail this author is associated with. Foreign key relation to `prisma_socialhistory_details` table, `id` column.
userId varchar The user identifier associated with the author (e.g., "user123").
providerTaxonomyCode varchar The taxonomy code of the healthcare provider (e.g., "207Q00000X").
providerTaxonomyCodeName varchar The taxonomy code name of the healthcare provider (e.g., "Family Medicine").
addressLine1 varchar The first line of the author's address (e.g., "123 Main St").
addressLine2 varchar The second line of the author's address, if applicable (e.g., "Apt 4B").
city varchar The city where the author is located (e.g., "New York").
county varchar The county where the author is located (e.g., "New York County").
state varchar The state where the author is located (e.g., "NY").
postalCode varchar The postal code of the author's address (e.g., "10001").
country varchar The country where the author is located (e.g., "United States").
homePhone varchar The author's home phone number (e.g., "555-123-4567").
workPhone varchar The author's work phone number (e.g., "555-987-6543").
faxnumber varchar The author's fax number, if applicable (e.g., "555-111-2222").
personFName varchar The author's first name (e.g., "John").
personLName varchar The author's last name (e.g., "Doe").
personSufix varchar The author's name suffix, if applicable (e.g., "Jr.").
organizationId varchar The identifier of the organization the author is associated with (e.g., "org001").
organizationName varchar The name of the organization the author is associated with (e.g., "Healthcare Clinic").
organizationPhone varchar The phone number of the organization the author is associated with (e.g., "555-222-3333").
organizationFax varchar The fax number of the organization the author is associated with, if applicable (e.g., "555-444-5555").
organizationAddLine1 varchar The first line of the organization's address (e.g., "456 Market St").
organizationAddLine2 varchar The second line of the organization's address, if applicable (e.g., "Suite 100").
organizationCity varchar The city where the organization is located (e.g., "Los Angeles").
organizationState varchar The state where the organization is located (e.g., "CA").
organizationPostalCode varchar The postal code of the organization's address (e.g., "90001").
organizationCountry varchar The country where the organization is located (e.g., "United States").
practitionerId int The identifier of the practitioner associated with the author. Foreign key relation to `prisma_practitioner_details` table, `ID` column.
displayName varchar The display name of the author (e.g., "Dr. John Doe, MD").
