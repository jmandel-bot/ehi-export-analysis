EHI Export Schema
Table devicetestdetail
Description This table stores information about device test details for patients, including test status, operator initials, and other relevant data.
Column DataType Description
id (Primary Key) int Unique identifier for each device test detail record. Example: 1, 2, 3.
Patientid int identifier for the patient associated with the device test. It can be a foreign key referring to the `id` column in the `Patient` table. Example: 101, 102, 103.
Reportid int identifier for the report associated with the device test. It can be a foreign key referring to the `id` column in the `Report` table. Example: 1001, 1002, 1003.
ExtOrderid varchar External order identifier for the device test. Example: 'ORD123', 'ORD124', 'ORD125'.
ExtTestid int External test identifier for the device test. Example: 1, 2, 3.
ReportDate datetime Date when the report was generated. Example: '2021-09-01 00:00:00', '2021-09-02 00:00:00', '2021-09-03 00:00:00'.
ReportTime datetime Time when the report was generated. Example: '2021-09-01 14:30:00', '2021-09-02 15:45:00', '2021-09-03 16:20:00'.
OperatorInitials char Initials of the operator who performed the device test. Example: 'A', 'B', 'C'.
Deviceid int identifier for the device used in the test. It can be a foreign key referring to the `id` column in the `Device` table. Example: 10001, 10002, 10003.
Interpretations text Text containing interpretations of the device test results. Example: 'Normal', 'Abnormal', 'Inconclusive'.
Comments text Text containing any additional comments about the device test. Example: 'Patient was cooperative', 'Test was interrupted', 'Retest recommended'.
TestStatus varchar Status of the device test. Example: 'Completed', 'Pending', 'Cancelled'.
Reviewerid int identifier for the reviewer of the device test. Example: 1, 2, 3.
technician varchar Name of the technician who performed the device test. Example: 'John Doe', 'Jane Smith', 'Alice Johnson'.
delFlag int Flag indicating whether the record has been deleted (1) or not (0). Example: 0, 1.
ModifiedDate datetime Date when the record was last modified. Example: '2021-09-01 00:00:00', '2021-09-02 00:00:00', '2021-09-03 00:00:00'.
ModifiedTime datetime Time when the record was last modified. Example: '2021-09-01 14:30:00', '2021-09-02 15:45:00', '2021-09-03 16:20:00'.
RequestedBy varchar Name of the person who requested the device test. Example: 'Dr. James', 'Dr. Lisa', 'Dr. Robert'.
SupervisingPhysician varchar Name of the supervising physician for the device test. Example: 'Dr. Williams', 'Dr. Brown', 'Dr. Taylor'.
ReviewedBy varchar Name of the person who reviewed the device test. Example: 'Dr. Johnson', 'Dr. Lee', 'Dr. Patel'.
ftpdirpath varchar Path to the directory where the test files are stored on an FTP server. Example: '/path/to/test/files'.
sdkversion varchar Version of the SDK used for the device test. Example: '1.0', '1.1', '1.2'.
