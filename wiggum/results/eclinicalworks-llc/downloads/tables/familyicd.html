EHI Export Schema
Table familyicd
Description This table stores practice-specific ICD configurations, such as custom name and SNOMED code for the Family History section.
Column DataType Description
icdItemid int Stores ICD's itemid, It refers to items table's itemid column.
icdCode varchar Stores ICD code, It refers to itemsdetails table's value column
icdDesc varchar Stores ICD description which is mainly from items table's itemName column.
snomedCode varchar Stores SNOMED code.
custom_name varchar Stores customized names for ICD which can be used to display instead of ICD description.
displayindex tinyint stores the sequence number of the customized ICD when it is added. Based on which list of ICD can be displayed.
showcustom_name varchar Used to mark whether customized should be displayed or not. Valid values are 0 and 1. 0 means not display and 1 means display customized name.
snomeddesc varchar Stores SNOMED description.
