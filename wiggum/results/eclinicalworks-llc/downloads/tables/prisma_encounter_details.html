Table prisma_encounter_details
Description This table stores information about various encounters, including their types, classes, service types, priorities, and other related details.
Column DataType Description
id (Primary Key) bigint Unique identifier for each record in the table (e.g., 1, 2, 3).
sectionId bigint Identifier for the section to which the encounter belongs. Foreign key relation to `prisma_ccd_document_section.id`.
encounterId varchar Unique identifier for the encounter (e.g., "E123", "E456").
encounterStartTime varchar Start time of the encounter in a string format (e.g., "2021-01-01 12:00:00").
encounterEndTime varchar End time of the encounter in a string format (e.g., "2021-01-01 14:00:00").
encounterTypeCode varchar Code representing the type of encounter (e.g., "123", "456").
encounterTypeName varchar Name of the encounter type (e.g., "Inpatient", "Outpatient").
encounterTypeText varchar Text description of the encounter type (e.g., "Inpatient encounter for care").
encounterStatus varchar Status of the encounter (e.g., "planned", "completed").
encounterClassCode varchar Code representing the class of the encounter (e.g., "AMB", "IMP").
encounterClassName varchar Name of the encounter class (e.g., "Ambulatory", "Inpatient").
encounterClassSystem varchar System used for the encounter class (e.g., "http://hl7.org/fhir/v3/ActCode").
encounterTypeSystem varchar System used for the encounter type (e.g., "http://hl7.org/fhir/v3/ActCode").
encounterServiceTypeCode varchar Code representing the service type of the encounter (e.g., "123", "456").
encounterServiceTypeName varchar Name of the encounter service type (e.g., "Cardiology", "Neurology").
encounterServiceTypeSystem varchar System used for the encounter service type (e.g., "http://hl7.org/fhir/v3/ActCode").
encounterServiceTypeText varchar Text description of the encounter service type (e.g., "Cardiology consultation").
encounterPriorityCode varchar Code representing the priority of the encounter (e.g., "1", "2").
encounterPriorityName varchar Name of the encounter priority (e.g., "High", "Low").
encounterPrioritySystem varchar System used for the encounter priority (e.g., "http://hl7.org/fhir/v3/ActCode").
encounterPriorityText varchar Text description of the encounter priority (e.g., "High priority encounter").
partOfId varchar Identifier for the parent encounter, if the current encounter is part of another encounter (e.g., "E789").
fhirId int Stores id of fhir_source_table.
isException tinyint Flag indicating whether the record is an exception (0 for false, 1 for true).
exceptionMessage varchar Message describing the exception, if any. Empty if there is no exception.
sourcePriority int Priority of the source system for the encounter (e.g., 1, 2, 3).
codeSystemPriority int Priority of the code system used for the encounter (e.g., 1, 2, 3).
duplicatedBy int Identifier of the record that duplicates the current record, if any (e.g., 4, 5, 6).
