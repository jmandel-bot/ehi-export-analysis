EHI Export Schema
Table bulk_batch_details_resource_files_mapping
Description This table stores the mapping between bulk batch details and bulk resources file details.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer. It uniquely identifies each mapping record. Example: 1, 2, 3, etc.
bulk_batch_details_id int This is a foreign key column that refers to the `id` column in the `bulk_batch_details` table. It represents the bulk batch detail associated with the mapping. Example: 101, 102, 103, etc.
bulk_resources_file_details_id int This is a foreign key column that refers to the `id` column in the `bulk_resources_file_details` table. It represents the bulk resources file detail associated with the mapping. Example: 201, 202, 203, etc.
