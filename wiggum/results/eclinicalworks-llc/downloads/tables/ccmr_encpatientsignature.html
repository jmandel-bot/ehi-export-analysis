EHI Export Schema
Table ccmr_encpatientsignature
Description This table stores information about patient signatures taken during their medical encounters.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
patientEncid int identifier for the patient encounter. This is a foreign key that refers to the `id` column in the `patientEncounters` table. Example: 101, 102, 103.
signTakenTime datetime time stamp for when the patient's signature was taken. Example: '2021-09-15 14:30:00', '2021-09-16 10:15:00'.
patientSignature varbinary Binary data representing the patient's signature as a longblob. Example: (Binary data not shown).
signaturePadid tinyint identifier for the signature pad used to capture the patient's signature. Example: 1, 2, 3.
delFlag tinyint A flag indicating if the record has been deleted (1) or not (0). Example: 0, 1.
signatureTakenBy int identifier for the staff member who took the patient's signature. This is a foreign key that refers to the `id` column in the `staffMembers` table. Example: 1001, 1002, 1003.
