Table prisma_labresult_details
Description This table stores detailed information about lab results, including the procedure activity, organization, and various timestamps related to the lab result.
Column DataType Description
id (Primary Key) bigint Unique identifier for each lab result detail record. Example: 1, 2, 3.
sectionId bigint Foreign key relation to the `prisma_ccd_document_section` table's `id` column. It represents the section in which the lab result detail is present. Example: 101, 102, 103.
entryId varchar Unique identifier for the lab result entry. Example: "ENTRY001", "ENTRY002", "ENTRY003".
currentStatus varchar Current status of the lab result. Example: "Completed", "Pending", "Cancelled".
procedureActivityCode varchar Code representing the procedure activity. Example: "PAC001", "PAC002", "PAC003".
procedureActivityCodeName varchar Name of the procedure activity code. Example: "Blood Test", "Urine Test", "X-Ray".
resultObtainedOn varchar time stamp when the lab result was obtained. Example: "2021-09-01 10:30:00", "2021-09-02 14:45:00".
documentationTime varchar time stamp when the lab result was documented. Example: "2021-09-01 11:00:00", "2021-09-02 15:15:00".
organizationName varchar Name of the organization that conducted the lab test. Example: "LabCorp", "Quest Diagnostics", "BioReference Laboratories".
referenceText varchar Long text containing reference information about the lab result. Example: "This is a reference text for the lab result."
collectionTime varchar time stamp when the sample was collected for the lab test. Example: "2021-09-01 09:30:00", "2021-09-02 13:45:00".
sourceSiteCode varchar Code representing the source site where the lab test was conducted. Example: "SSC001", "SSC002", "SSC003".
sourceSiteName varchar Name of the source site where the lab test was conducted. Example: "Main Lab", "Downtown Lab", "Uptown Lab".
entityCode varchar Code representing the entity related to the lab result. Example: "EC001", "EC002", "EC003".
entityname varchar Name of the entity related to the lab result. Example: "Entity A", "Entity B", "Entity C".
collectionEndTime datetime time stamp when the sample collection for the lab test ended. Example: "2021-09-01 09:45:00", "2021-09-02 14:00:00".
reviewedTime datetime time stamp when the lab result was reviewed. Example: "2021-09-01 12:00:00", "2021-09-02 16:30:00".
externalEncounterId varchar Unique identifier for the external encounter related to the lab result. Example: "EEID001", "EEID002", "EEID003".
diagnosticReportId varchar Unique identifier for the diagnostic report related to the lab result. Example: "DRID001", "DRID002", "DRID003".
