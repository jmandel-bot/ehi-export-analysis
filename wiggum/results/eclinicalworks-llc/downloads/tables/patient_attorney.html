Table patient_attorney
Description This table stores the relationship between patients and their attorneys, along with the attorney's office information and display preferences.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the patient_attorney table. Example: 1, 2, 3
Pid int Foreign key referring to the patient's unique identifier in the `patient` table. Example: 101, 102, 103
Attid int Foreign key referring to the attorney's unique identifier in the `attorney` table. Example: 201, 202, 203
AttOffid int Foreign key referring to the attorney's office unique identifier in the `attorney_office` table. Example: 301, 302, 303
DisplayIndex int An integer value representing the display order of the attorney information for a specific patient. Example: 1, 2, 3
Deleted int A flag indicating if the record has been deleted (1) or not (0). Example: 0, 1
Inactive int A flag indicating if the record is inactive (1) or active (0). Example: 0, 1
