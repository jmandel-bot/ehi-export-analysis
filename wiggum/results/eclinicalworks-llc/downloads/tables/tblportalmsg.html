EHI Export Schema
Table tblportalmsg
Description This table stores messages related to Web Encounters created on a portal or kiosk.
Column DataType Description
MsgId (Primary Key) int This column stores the Id of the message (primary key) and also it is autoincremented.
uid int User ID of the message owner. Example: 101, 102, 103.
RefMsgId int Reference message ID for threaded messages or replies. If the message is not a reply, the default value is 0. Example: 0, 5, 7.
MsgFrom int User ID of the sender. Example: 101, 102, 103.
MsgTo int User ID of the recipient. Example: 104, 105, 106.
MsgType int Integer representing the type of message. Example: 0 (normal), 1 (urgent), 2 (system).
Priority int Integer representing the priority level of the message. Example: 0 (low), 1 (medium), 2 (high).
FolderId int Folder ID where the message is stored. Example: 1 (inbox), 2 (sent), 3 (archive).
Subject varchar Subject of the message, limited to 100 characters. Example: "Meeting Reminder", "Project Update".
Body text Text content of the message. Example: "Please join the meeting at 3 PM", "The project is on track and will be completed on time".
dtSent datetime Date and time when the message was sent. Example: '2022-01-01 12:30:00'.
dtReceived datetime Date and time when the message was received. Example: '2022-01-01 12:31:00'.
dtRead datetime Date and time when the message was read by the recipient. Example: '2022-01-01 12:35:00'.
dtArchived datetime Date and time when the message was archived. Example: '2022-01-02 10:00:00'.
dtDeleted datetime Date and time when the message was deleted. Example: '2022-01-03 15:00:00'.
