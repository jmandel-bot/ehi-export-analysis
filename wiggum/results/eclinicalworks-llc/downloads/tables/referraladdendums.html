EHI Export Schema
Table referraladdendums
Description This table stores addendums related to referrals after they are marked as addressed., including the user who created the addendum and the date and time it was created.
Column DataType Description
id (Primary Key) int This is auto-incremented primary key in This column which uniquely identify each record in This column .
referralid int This column stores id of a referral for which addendum is being added by the end user. This column refers to referralid column in referral table.
userid int This column stores id of a user who has added addendum to a referral. This column refers to UID column in user table.
date datetime This column stores date on which addendum was added to a referral by end user. Date is stored in "YYYY-MM-DD 00:00:00.000" format.
time datetime This column stores time at which addendum was added to a referral by end user. Time is stored in "1900-01-01 hh:mm:ss.000" format.
addendum text This column stores text of an addendum added by end user to a referral.
