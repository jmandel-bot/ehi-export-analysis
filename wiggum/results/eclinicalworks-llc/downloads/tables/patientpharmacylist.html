EHI Export Schema
Table patientpharmacylist
Description This table is used to store the pharmacy list for patient's through the Patient Demographics screen
Column DataType Description
id (Primary Key) int This is auto-incremented primary key in This column .
Patientid int This column stores id of a patient whose pharmacy is being documented by end user. This column refers to UID column in user table.
Pharmacyid int This column stores id of a pharmacy which is being documented for the patient by end user. This column refers to pmcid column in pharmacy table.
IsPrimary tinyint This column stores the pharmacy if selected as primary by the end user
DisplayIndex int This column is used to store the sequence of the pharmacies selected by the end user
