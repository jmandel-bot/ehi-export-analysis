EHI Export Schema
Table vision_motility_ductions
Description This table stores information about patients' vision motility and ductions, which are related to eye movement and muscle function.
Column DataType Description
id (Primary Key) int This column is an auto incremental primary key.
patientid int This column refers to the UID column in the user table. It stores the patient id for patients.
encid int This column stores an encounter id of a patient. This column refers to column encounterid from the enc table.
l_sr varchar This column stores Left superior rectus muscle value. Value can be stored from the range between -5 to +100 with interval of 0.5. Column has been declared as VARCHAR because this column store "+" sign to display on UI.
l_lr varchar This column stores Left lateral rectus muscle value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
l_ir varchar This column stores Left inferior rectus muscle value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
l_mr varchar This column stores values for Left medial rectus muscle value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
l_io varchar This column stores Left inferior oblique muscle value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
l_so varchar This column stores Left superior oblique muscle value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
l_elevation varchar This column stores Left eye elevation value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
l_depression varchar This column stores Left eye depression value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
l_upshoot varchar This column stores Left eye upshoot value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
l_downshoot varchar This column stores Left eye downshoot value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
r_sr varchar This column stores Right superior rectus muscle value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
r_lr varchar This column stores Right lateral rectus muscle value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
r_ir varchar This column stores Right inferior rectus muscle value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
r_mr varchar This column stores Right medial rectus muscle value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
r_io varchar This column stores Right inferior oblique muscle value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
r_so varchar This column stores Right superior oblique muscle value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
r_elevation varchar This column stores Right eye elevation value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
r_depression varchar This column stores Right eye depression value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
r_upshoot varchar This column stores Right eye upshoot value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
r_downshoot varchar This column stores Right eye downshoot value. Value can be stored from the range between -5 to +100 with interval of 0.5. Default value is NULL. This column length is varchar(5).
comments varchar This column stores Motility Duction Exam notes and varchar (255) is chosen to save comments.
created_date datetime This column stores date and time when duction was created. Datetime is stored in 'YYYY-MM-DD hh:mm:ss.000' format.
last_updated_date datetime This column stores date and time when duction was last updated. Datetime is stored in 'YYYY-MM-DD hh:mm:ss.000' format.
deleteflag tinyint This column stores a value of 0 or 1. A "0" means not deleted, and a "1" means deleted.
