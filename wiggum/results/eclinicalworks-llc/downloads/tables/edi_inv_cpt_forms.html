EHI Export Schema
Table edi_inv_cpt_forms
Description This table stores information about the relationship between invoice CPTs and forms.
Column DataType Description
Id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
InvCPTId int Identifier for the invoice CPT. This column can be a foreign key referring to the `Id` column in the `invoice_cpts` table (if it exists). Example: 101, 102, 103.
FormId int Identifier for the form. This column can be a foreign key referring to the `Id` column in the `forms` table (if it exists). Example: 201, 202, 203.
deleteflag tinyint A flag indicating whether the record is deleted or not. 0 means not deleted, and 1 means deleted. Example: 0, 1.
