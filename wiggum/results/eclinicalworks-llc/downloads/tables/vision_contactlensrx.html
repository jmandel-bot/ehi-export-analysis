Table vision_contactlensrx
Description This table stores information about vision contact lens prescriptions, including details about the lenses, products, and related orders.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key.
Encounterid int This column stores the encounter id for which contact lens are getting documented. This column refers column encounterid from enc table.
visionrxid int This column indicates the prescribed product for this contact lens. This column refers column id from visionrx table.
applytotrial tinyint This column stores a value 0 or 1. A "1" is when the over refraction data for contact lens needs to be applied in case of a trial fpr the prescription, and "0" is otherwise.
rlensfromstock int This column stores the number of contact lens for a right eye given to a customer for a trial from stock.
llensfromstock int This column stores the number of contact lens for a left eye given to a customer for a trial from stock.
unsuccessfulFit tinyint This column is not in use.
productidOD int This column indicates the prescribed product for the right eye. This column refers column id from inventory_product table.
productidOS int This column indicates the prescribed product for the left eye. This column refers column id from inventory_product table.
AddTypeOD varchar This column is not in use.
AddTypeOS varchar This column is not in use.
lenstype int This column stores a values 0 or 1. A "1" is when it is of type 'Other', and a "0" is when it is of type 'Soft'.
orderflag tinyint This column stores a values 0 or 1. A "1" is when the lens are not available in stock and ordered from dispense for a trial, and a "0" is otherwise.
rlensorder int This column stores number of lenses ordered for right eye.
llensorder int This column stores number of lenses ordered for left eye.
contactlensidod int This column is not being used any more. It was initially used as to match extact contact lens product combination to validate SKU data.
contactlensidos int This column is not being used any more. It was initially used as to match extact contact lens product combination to validate SKU data.
coloridOD int This column indicates the color of prescribed product for the right eye. This column refers column id from inventory_vcontactlens_color table.
coloridOS int This column indicates the color of prescribed product for the left eye. This column refers column id from inventory_vcontactlens_color table.
