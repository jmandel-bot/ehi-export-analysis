EHI Export Schema
Table edi_277ca_cptresponse
Description This table stores information aboutCPT (Current Procedural Terminology) responses in the EDI 277CA (Claim Acknowledgement) format.
Column DataType Description
CPTResponseid (Primary Key) int Unique identifier for each CPT response. (Example: 1, 2, 3)
ClaimResponseid int identifier for the associated claim response, foreign key referring to the claim response table. (Example: 101, 102, 103)
Invid int identifier for the associated invoice, foreign key referring to the invoice table. (Example: 1001, 1002, 1003)
InvCptid int identifier for the associated invoice CPT, foreign key referring to the invoice CPT table. (Example: 2001, 2002, 2003)
CPT_2220D_SVC01 varchar CPT code for the service provided. (Example: '99213', '99214', '99215')
CPTCharges_2220D_SVC02 decimal Charges for the CPT service in decimal format. (Example: 100.00, 150.00, 200.00)
LineItemControlNumber varchar Control number for the line item. (Example: 'LICN001', 'LICN002', 'LICN003')
StartDate datetime Start date of the service period in date format. (Example: '2021-01-01', '2021-01-15', '2021-02-01')
EndDate datetime End date of the service period in date format. (Example: '2021-01-31', '2021-02-14', '2021-02-28')
Status_2220D_STC01_1 varchar First part of the status code for the CPT response. (Example: 'A1', 'A2', 'A3')
Status_2220D_STC01_2 varchar Second part of the status code for the CPT response. (Example: 'B1', 'B2', 'B3')
Status_2220D_STC01_3 char Third part of the status code for the CPT response. (Example: '001', '002', '003')
Status_2220D_STC01_4 char Fourth part of the status code for the CPT response. (Example: '004', '005', '006')
ActionCode_2220D_STC03 varchar Action code for the CPT response. (Example: 'AC01', 'AC02', 'AC03')
Message varchar Message related to the CPT response, up to 1024 characters. (Example: 'Service approved', 'Service denied', 'Service pending')
deleteFlag tinyint Flag to indicate if the record is deleted (1) or not (0). (Example: 0, 1)
