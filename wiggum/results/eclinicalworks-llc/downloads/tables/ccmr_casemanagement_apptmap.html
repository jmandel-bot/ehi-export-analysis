EHI Export Schema
Table ccmr_casemanagement_apptmap
Description This table stores information about appointments related to case management.
Column DataType Description
id (Primary Key) int Unique identifier for each appointment mapping record. Example: 1, 2, 3.
caseId int This column stores caseid which is generated when patient gets enrolled for case program. This column refers to caseId column from ccmr_casemanagement_case table.
refEncId int This column stores id of an encounter during which patient got enrolled to case program. This column refers to encounterid column in enc table. These encounters have encType set to 17 which is used for case management encounters.
tempId int This column stores id of a care plan template mapped with case program. This column refers to id column from "ccmr_problemtemplatemapping" table.
delflag tinyint This column stores values 0/1 to indicate whether the record is marked as deleted or not. 0 - Record is NOT marked as deleted, 1 - Record is marked as deleted.
STATUS tinyint Status of the appointment mapping record. 0 means inactive, and 1 means active. Example: 0, 1.
customStatus int Custom status identifier for the appointment mapping record. Example: 10, 20, 30.
note text Text field for storing additional notes or comments about the appointment mapping record. Example: "Appointment rescheduled", "Client did not show up".
