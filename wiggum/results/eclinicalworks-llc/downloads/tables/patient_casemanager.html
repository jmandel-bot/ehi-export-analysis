Table patient_casemanager
Description This table stores the relationship between patients and their case managers, including information about whether the relationship is active or deleted.
Column DataType Description
Pid int This is the patient ID, which is a unique identifier for each patient. It is an integer value, and it cannot be NULL. Example: 12345
CaseManid int This is the case manager id, which is a unique identifier for each case manager. It is an integer value, and it cannot be NULL. Example: 67890
deleteflag tinyint This is a flag to indicate whether the relationship between the patient and the case manager is active or deleted. It is a tiny integer value (either 0 or 1), where 0 means the relationship is active, and 1 means it is deleted. It cannot be NULL and has a default value of 0. Example: 0 (active)
