EHI Export Schema
Table paymentplanprepayment
Description This table stores information about prepayments made for payment plans associated with specific claims.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the paymentplanprepayment table. Example: 1, 2, 3.
planid int identifier for the payment plan associated with the prepayment. This can be a foreign key referring to the paymentplandetail.planid. Example: 101, 102, 103.
claimid int identifier for the claim associated with the prepayment. This can be a foreign key referring to the edi_invoice.id column. Example: 1001, 1002, 1003.
deleteflag tinyint A flag indicating whether the record has been deleted (1) or not (0). Example: 0, 1.
claimamount decimal The total amount of the claim associated with the prepayment. Example: 500.00, 1000.00, 1500.00.
claimbal decimal The remaining balance of the claim after the prepayment has been applied. Example: 250.00, 750.00, 1250.00.
ptbal decimal The remaining balance of the payment plan after the prepayment has been applied. Example: 200.00, 800.00, 1200.00.
TransferFlag tinyint A flag indicating whether the prepayment has been transferred (1) or not (0). Example: 0, 1.
