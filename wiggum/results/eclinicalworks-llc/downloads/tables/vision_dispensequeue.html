EHI Export Schema
Table vision_dispensequeue
Description This table stores information about the dispense queue for vision products, such as glasses or contact lenses, including the status of the dispense process, assignment details, and related encounter and order information.
Column DataType Description
id (Primary Key) int This column is an auto incremented primary key.
encounterid int This column stores an encounter id for which dispense record is added. This column refers to column encouner id from the enc table. Also, enctype = 16 indicates dispense encounter.
disp_status int This column stores different statuses when the dispense record is created using dispense module. This column refers to the column id from the vision_master table.
assignedto int This column stores an identifier for the user to whom the dispense queue record is assigned. (e.g., 1001, 1002, 1003)
context tinyint This column stores one of the below context for this dispense record. A "1" is created based on a previous sight exam, a "2" is for a remake, a "3" is for outside Rx, a "4" is for contact lens from previous previous sight tests, and a "5" is outside Rx for contact lens, and "6" is for over the counter.
isdeleted tinyint This column contains a flag to indicate whether the record is deleted, where 0 is the default value (e.g., 0, 1).
createdby int This column stores the id of the staff or provider who creates this dispense record. This column refers to the column UID from the user table.
createdon datetime This column stores the time stamp when this dispense record was created. The format used is YYYY-MM-DD HH:MM:SS.
updatedby int This column stores the id of the staff or provider who updates this dispense record. This column refers to the column UID from the user table.
updatedon datetime This column stores the time stamp when this dispense record was created. The format used is YYYY-MM-DD HH:MM:SS.
pnencounterid int This column stores the id of the encounter, where the original prescription was documented. This will be same as of the encounterid column when the context = 3. This column refers to the column encounterid from the enc table.
pairid int This column stores the id of the pair whenever it is added to this dispense record. This column refers to the column id from the vision_order_pair table.
orderid int This column stores order id whenever a order is placed. This column refers to column id from the vision_orders table.
indispense_status_log datetime This column stores the time stamp when last time the status was updated. The format used is YYYY-MM-DD HH:MM:SS.
notes text This column stores additional notes (free text) for this dispense record.
visionrxid int This column stores an identifier for the vision prescription associated with the dispense queue record (e.g., 10001, 10002, 10003).
