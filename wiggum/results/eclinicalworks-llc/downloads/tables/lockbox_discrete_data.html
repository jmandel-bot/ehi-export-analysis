EHI Export Schema
Table lockbox_discrete_data
Description This table stores information related to lockbox transactions, including payment details, status, and associated images.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table (e.g., 1, 2, 3).
Batchid int identifier for the batch that the transaction belongs to (e.g., 101, 102, 103).
DateCreated datetime Date and time when the record was created (e.g., '2021-08-01 12:34:56').
DateModified datetime Date and time when the record was last modified (e.g., '2021-08-02 14:56:23').
AccountNumber varchar Account number associated with the transaction (e.g., '1234567890').
AmountDue varchar Amount due for the transaction (e.g., '100.00').
PmtAmt varchar Payment amount for the transaction (e.g., '100.00').
PatientName varchar Name of the patient associated with the transaction (e.g., 'John Doe').
PmtMethod varchar Payment method used for the transaction (e.g., 'Credit Card', 'Check').
PmtType varchar Payment type for the transaction (e.g., 'Full', 'Partial').
CheckNum varchar Check number, if applicable (e.g., '12345678').
Status int Status code for the transaction (e.g., 0 for pending, 1 for completed, 2 for failed).
reason text Text description of the reason for the transaction status (e.g., 'Payment successful', 'Insufficient funds').
Transid varchar Unique identifier for the transaction (e.g., 'T123456789').
TransDate varchar Date of the transaction (e.g., '2021-08-01').
Paymentid int Unique identifier for the payment (e.g., 1001, 1002, 1003).
img_reason varchar Text description of the reason for the image status (e.g., 'Image uploaded successfully', 'Image not found').
img_status int Status code for the image associated with the transaction (e.g., 0 for pending, 1 for completed, 2 for failed).
vendorUser varchar Username of the vendor user associated with the transaction (e.g., 'vendorUser1').
processFlow text Text description of the process flow for the transaction (e.g., 'Step 1: Payment received, Step 2: Payment processed').
