EHI Export Schema
Table claimstatuscodes
Description This table stores information about claim status codes, including their short descriptions, display indexes, and various flags related to their usage.
Column DataType Description
id (Primary Key) int Unique identifier for each claim status code record. Example: 1, 2, 3.
code varchar The claim status code itself, represented as a string. Example: "A01", "B02", "C03".
ShortDesc varchar A short description of the claim status code. Example: "Approved", "Rejected", "Pending".
displayIndex int An integer representing the display order of the claim status codes. Example: 0, 1, 2.
deleteFlag int A flag indicating whether the claim status code has been deleted (1) or not (0). Example: 0, 1.
UID int A unique identifier for the user who created or modified the claim status code record. Example: 1234567890123.
ShortCutKey varchar A shortcut key associated with the claim status code for quick access. Example: "Ctrl+A", "Ctrl+B", "Ctrl+C".
skipclaimopenvalidation tinyint A flag indicating whether to skip claim open validation (1) or not (0) for this claim status code. Example: 0, 1.
excludepatientstmts tinyint A flag indicating whether to exclude this claim status code from patient statements (1) or not (0). Example: 0, 1.
