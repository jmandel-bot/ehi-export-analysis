EHI Export Schema
Table encounters
Description This table stores the patient's examination data for an encounter.
Column DataType Description
id (Primary Key) int This is auto-incremented and Primary Key of This column .
Encounterid int This column stores id of the encounter for which Examination section is documented. This column refers column encounterid from enc table.
Examid int This column stores id of the Category selected while documenting Examination section of the patient. This column refers to column itemid from items table.
Propid int This column stores id of the Field selected in the user interfacewhile documenting Examination section of the patient. This column refers to column propid from properties table.
PropDetailid int This column stores property detail id and not actively used hence it is always assigned default value 0.
Selection int This column is not used anymore.
Notes varchar This column stores Notes documented in the Examination section of the Patient. This could be selected structured question along with answer or free text entered by the end user.
createDate datetime This column stores date when the exam was added. The format used is YYYY-MM-DD HH:MM:SS. This column is not actively used.
displayseq int This column stores sequence of the items to be displayed.
notes2 varchar This column stores the same value as in Notes column but along with the HTML tags i.e. in HTML format.
displaycatseq int This column stores sequence number of the documented categories to be displayed in the Progress Note. The sequence starts from 0.
drawingnotes varchar This column stores additional notes documented on the attached drawing.
