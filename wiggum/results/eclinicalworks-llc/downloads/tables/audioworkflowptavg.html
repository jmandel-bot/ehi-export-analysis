EHI Export Schema
Table audioworkflowptavg
Description This table stores the average audio workflow values for the right and left ears of a patient during an encounter.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the audioworkflowptavg table. Example: 1, 2, 3, etc.
Encid int The encounter identifier to which the audio workflow values are related. This is a foreign key that refers to the `encounterid` column in the `enc` table. Example: 101, 102, 103, etc.
PTAvgRt varchar The average audio workflow value for the patient's right ear. Stored as a string. Example: "25.5", "30.2", "22.8", etc.
PTAvgLt varchar The average audio workflow value for the patient's left ear. Stored as a string. Example: "24.3", "29.1", "21.7", etc.
