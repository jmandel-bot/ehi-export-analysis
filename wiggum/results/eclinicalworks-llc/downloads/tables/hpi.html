EHI Export Schema
Table hpi
Description This table stores patient's History of Illness.
Column DataType Description
Encounterid int Stores encounter id of a encounter for which HPI is being documented. Column refers to encounterid column in enc table.
Hpiid int This column stores HPI category selected for documentation by provider/staff. Column refers to itemid in items table.
Propid int Stores id for property documented for associated category on HPI screen. Column refers to propid in properties table.
Selected int Stores int value for "Care of (c/o)" or "Denies" options for each HPI item. 0 - Denies, 1 - Care of. If none of the option is selected by end user, thi column is set to -1 by default from the application. Note: Default value from database id not honored as application always inserts -1 in case nothing is selected.
Notes text Stores free text notes documented by provider/staff from HPI screen and Non-structured item screen.
Duration varchar This column stores duration of the illness/symptom being documented by end user. Duration is a combination can be in Days,Weeks, Months or Years. Duration field also records free text data from the end user.
createDate datetime A datetime field for storing the date and time when the HPI record was created. Example: "2021-10-01 12:34:56"
displayseq int This column stores order in which categories are listed on the user interface.
notes2 text This column store notes documented by provider/staff from HPI screen and Non-structured item screen in HTML-like tags.
displayIndex smallint This column stores order in which hpi records under a category are listed in the UI.
