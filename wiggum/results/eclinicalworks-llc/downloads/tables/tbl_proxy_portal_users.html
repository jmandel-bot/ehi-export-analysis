Table tbl_proxy_portal_users
Description This table stores information about proxy portal users, their credentials, and related details.
Column DataType Description
id (Primary Key) int Unique identifier for each proxy portal user (e.g., 1, 2, 3).
username varchar The username of the proxy portal user (e.g., "john_doe").
name varchar The full name of the proxy portal user (e.g., "John Doe").
pwd varchar The Bcrypted password of the proxy portal user (e.g., "$2a$10$TDl").
encpwd varchar The encrypted password of the proxy portal user (e.g., "a1b2c3d4e5").
email varchar The email address of the proxy portal user (e.g., "john.doe@example.com").
duration datetime The date when the user's access duration ends (e.g., "2022-12-31").
UID int The unique identifier of the related user in another table (e.g., 1, 2, 3).
relationship varchar The relationship between the proxy portal user and the primary user (e.g., "spouse", "child").
STATUS tinyint The status of the proxy portal user, where 1 is active and 0 is inactive (e.g., 1).
proxyUID int The unique identifier of the related proxy user in another table (e.g., 1, 2, 3).
locked tinyint Indicates if the proxy portal user account is locked, where 1 is locked and 0 is unlocked (e.g., 0).
primary_user tinyint Indicates if the proxy portal user is a primary user, where 1 is primary and 0 is not primary (e.g., 1).
createdby int The unique identifier of the user who created the proxy portal user (e.g., 1, 2, 3).
modifiedby int The unique identifier of the user who last modified the proxy portal user (e.g., 1, 2, 3).
createddate datetime The date when the proxy portal user was created (e.g., "2021-01-01").
modifieddate datetime The date when the proxy portal user was last modified (e.g., "2021-12-01").
deleteflag int Indicates if the proxy portal user is deleted, where 1 is deleted and 0 is not deleted (e.g., 0).
websync tinyint Indicates if the proxy portal user's data has been synchronized with the web, where 1 is synchronized and 0 is not synchronized (e.g., 1).
sentemail tinyint Indicates if an email has been sent to the proxy portal user, where 1 is sent and 0 is not sent (e.g., 1).
preset tinyint Indicates if the proxy portal user has a preset configuration, where 1 is preset and 0 is not preset (e.g., 0).
access_type tinyint The access type of the proxy portal user, where 1 is full access and 0 is limited access (e.g., 1).
relatedptid int The unique identifier of the related patient in another table (e.g., 1, 2, 3).
expiry datetime The date when the proxy portal user's access expires (e.g., "2023-01-01").
