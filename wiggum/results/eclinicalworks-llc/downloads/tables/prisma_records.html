EHI Export Schema
Table prisma_records
Description This table stores records related to Prisma and their indexing status, along with other metadata.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table (int). Example: 1, 2, 3.
recordType tinyint Type of the record, represented as a tinyint. Example: 0, 1.
pid int Unique identifier for the user associated with the record (int). Foreign key relation: `users.uid`. Example: 101, 102, 103.
indexedStatus tinyint Status of the record indexing, represented as a tinyint. Example: 0 (not indexed), 1 (indexed).
indexedId varchar Unique identifier for the indexed record (varchar). Example: "IDX123", "IDX124".
indexedDate datetime Date and time when the record was indexed (datetime). Example: "2021-09-01 12:30:45".
lastRetrievalDate datetime Date and time when the record was last retrieved (datetime). Example: "2021-09-01 12:30:45".
createdDate datetime Date and time when the record was created (datetime). Example: "2021-09-01 12:30:45".
modifiedDate datetime Date and time when the record was last modified (datetime). Example: "2021-09-01 12:30:45".
deleteFlag tinyint Flag indicating whether the record is deleted or not, represented as a tinyint. Example: 0 (not deleted), 1 (deleted).
sourceRecordType tinyint Type of the source record, represented as a tinyint. Example: 0, 1.
ptDocId int Unique identifier for the document associated with the record (int). Foreign key relation: `document.docID`. Example: 1001, 1002, 1003.
sourceId int Unique identifier for the source of the record (int). Example: 1, 2, 3.
indexStatusCode smallint Status code for the record indexing, represented as a smallint. Example: 0 (not indexed), 1 (indexed).
