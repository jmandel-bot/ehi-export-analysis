EHI Export Schema
Table eptstmtsautopayclaims
Description This table stores the data for the claims that have been considered in patient balance statements for the card-on-file feature.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
origencounterid int Original encounter id associated with the claim. Example: 1001, 1002, 1003.
encounterid int Encounter id associated with the claim. Example: 2001, 2002, 2003.
origclaimid int Original claim id associated with the auto-pay claim. Example: 3001, 3002, 3003.
claimid int Claim id associated with the auto-pay claim. Example: 4001, 4002, 4003.
servicedt datetime Date of service for the claim. Example: '2021-01-01', '2021-01-02', '2021-01-03'.
invoicedt datetime Date of invoice for the claim. Example: '2021-01-10', '2021-01-11', '2021-01-12'.
DOSproviderid int Provider id for the date of service. Example: 101, 102, 103.
PayToProviderid int Provider id to whom the payment should be made to. Example: 201, 202, 203.
RenProviderid int Rendering provider id. Example: 301, 302, 303.
facilityid int Facility id where the service was provided. Example: 401, 402, 403.
statementid int Statement id associated with the auto-pay claim. Example: 501, 502, 503.
statementdate datetime Date of the statement. Example: '2021-01-15', '2021-01-16', '2021-01-17'.
totalBalance decimal Total balance of the claim. Example: 1000.00, 2000.00, 3000.00.
chargeBalance decimal Patient balance at the time of processing card on file.
amountpaid decimal Amount paid for the claim. Example: 250.00, 350.00, 450.00.
createdtime datetime Time stamp of when patient's statement was cerated.
ePtStmtSlNo int Serial number for the electronic patient statement. Example: 10001, 10002, 10003.
StatementType int Type of statement (integer value). Example: 1, 2, 3.
pmtdetailid int Payment detail id associated with the claim. Example: 11001, 11002, 11003.
postedamt decimal Posted amount for the claim. Example: 150.00, 250.00, 350.00.
inserttime datetime time stamp when the record was inserted into the table. Example: '2021-01-01 12:30:00', '2021-01-02 12:30:00', '2021-01-03 12:30:00'.
deleteflag int Flag to indicate if the record is deleted (0 for not deleted, 1 for deleted). Example: 0, 1.
