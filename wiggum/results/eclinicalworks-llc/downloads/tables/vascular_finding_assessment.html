EHI Export Schema
Table vascular_finding_assessment
Description This table stores assessments configured with Vascular findings.
Column DataType Description
id (Primary Key) int This column is auto-incremented primary key in This column .
userid int This column stores identifier number of a User. It refers to column UID from user table.
findingid int This column stores identifier number of a Finding. It refers to column id from vascular_finding table.
itemid int Stores assessment selected by user for the Finding. List of assessments are stored in items table. Column refers to itemid column in items table.
icdversion tinyint This column stores Version information about ICD code.
isdefault tinyint This column stores is default assessment or not 0- default 1- not default.
createdby int This column stores identifier number of a user who created the finding. It refers to column UID from user table.
createddate datetime This column stores the current time stamp when the Finding is created. The format used is YYYY-MM-DD HH:MM:SS.
