EHI Export Schema
Table vision_fittingmeasurements
Description This table stores the fitting measurements related to vision, including horizontal visible iris diameter, vertical palpebral aperture, and pupil diameter for both eyes, and any additional comments.
Column DataType Description
id (Primary Key) int This column is auto-incremented Primary Key for This column .
encid int This column contains the encounterid value. It refers to enc table `encounterid` column. Example: 101, 102, 103.
hvidOD varchar Horizontal visible iris diameter for the right eye (OD). For example: '11.5', '12.0', '11.0'.
hvidOS varchar Horizontal visible iris diameter for the left eye (OS). For example: '11.5', '12.0', '11.0'.
vpaOD varchar Vertical palpebral aperture for the right eye (OD). Example: '9.0', '8.5', '10.0'.
vpaOS varchar Vertical palpebral aperture for the left eye (OS). Example: '9.0', '8.5', '10.0'.
pupilDiaOD varchar Pupil diameter for the right eye (OD). For example: '4.5', '5.0', '4.0'.
pupilDiaOS varchar Pupil diameter for the left eye (OS). For example: '4.5', '5.0', '4.0'.
comments text Additional comments related to the fitting measurements. Example: 'Patient has dry eyes', 'Measurements taken after dilation', 'Patient wears contact lenses'.
