EHI Export Schema
Tableip_rh_structobhistory
DescriptionThis table stores the OB history for the patient under specific encounter id.
ColumnDataTypeDescription
id(primaryKey) int Stores the auto increment id for this table
encounterId int Stores the encounter id FK from Enc table
catId int This stores the itemid for the OB History item from items table. This is static ItemName.
itemId int This stores the itemid for the Question for which the answer is documented in the encounter. This also gets stored in Items table
detailId int This stores the id of the mapped entry stored in structdatadetail table which links the catid with itemid.
value varchar The documented value is stored in this column
valueId varchar If the data stored has a structured id then is gets stored in this column. Default value is blank ('')
notes varchar This column stores any notes attached with the question getting answered.
chartdisplay tinyint This column specifies if the data is to be displayed on the chart. 1 - Yes, 0 - No
displayIndex tinyint This stores the display index of the questions answered. Lower index specifies shown at first in order.
deleteFlag tinyint Soft delete flag for this entry, 1 - Deleted, 0 - Not Deleted
createdBy int User who created this entry in this table, refers to uid from Users table
createdDateTime datetime Time stamp of when this created action was performed.
modifiedBy int User who modified this entry in this table, refers to uid from Users table
modifiedDateTime datetime Time stamp of when this modify action was performed.
