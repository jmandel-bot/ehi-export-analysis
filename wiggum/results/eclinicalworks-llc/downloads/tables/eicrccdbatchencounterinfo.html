EHI Export Schema
Table eicrccdbatchencounterinfo
Description This table stores information about batch encounters related to eICR CCD (Electronic Initial Case Report - Continuity of Care Document) generation, including the associated doctor, patient, and encounter details.
Column DataType Description
Id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
batch_id int Identifier for the batch to which the encounter belongs. Foreign key relation to `eicrccdbatch.batch_id`. Example: 101, 102, 103.
encounter_id int Identifier for the encounter associated with the EICR CCD generation. Foreign key relation to `enc.encounterID`. Example: 1001, 1002, 1003.
ccd_generated tinyint A flag indicating whether the CCD has been generated for the encounter Where 0 is for not generated, 1 is for generated and 2 is for ccr generation error.
doctor_id int Identifier for the doctor associated with the encounter. Foreign key relation to `users.uid`. Example: 501, 502, 503.
patient_id int Identifier for the patient associated with the encounter. Foreign key relation to `users.uid`. Example: 601, 602, 603.
resend tinyint A flag indicating whether the EICR CCD needs to be resent (1) or not (0). Example: 0, 1.
eicr_transaction_log_id int Identifier for the EICR transaction log associated with the encounter. It has the foreign key reference to the eicr_transaction_logs.id
date_created datetime time stamp for when the record was created. Example: '2022-01-01 12:34:56', '2022-01-02 14:56:32'.
