EHI Export Schema
Table progressnotes
Description This table stores the Progress Notes locked summary in encrypted form.
Column DataType Description
encounterid int This column stores id of the encounter for which patient progress note is created. Column refers to encounterid column in enc table.
summary text This column stores detail of patient's progress note sections in xml format.
xslid int This column stores id of the xslFileName. This column refers to xslid column in PrNotesXSLRefs table.
unlocked smallint This column stores value 0 / 1 indicating if the Progress note is locked. 0 - Locked , 1 - Unlocked.
ModifyDate datetime This column stores the Progress note modified date.
ModifyDate2 varchar This column stores the Progress note modified date which would be used to decrypt the xml present in summary column
regionName varchar Region/Zone name appointment facility
