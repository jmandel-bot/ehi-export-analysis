Table allergy_shot_global_alerts
Description This table stores information about global alerts related to allergy shots, including their names, colors, values, and whether they have been deleted or not.
Column DataType Description
id (Primary Key) int This is the primary key column, which stores a unique integer identifier for each allergy shot global alert. For example, the value could be 1, 2, 3, etc.
NAME varchar This column stores a string (up to 50 characters) representing the name of the allergy shot global alert. For example, the value could be "Pollen High Alert" or "Dust Mite Warning".
COLOR varchar This column stores a string (up to 10 characters) representing the color associated with the allergy shot global alert. This could be used for display purposes. For example, the value could be "red", "yellow", or "green".
VALUE varchar This column stores a text value containing additional information or details about the allergy shot global alert. For example, the value could be "Pollen levels are extremely high today, take necessary precautions."
DELETE_FLAG int This column stores an integer value (either 0 or 1) indicating whether the allergy shot global alert has been deleted or not. A value of 0 means the alert is active, while a value of 1 means the alert has been deleted.
