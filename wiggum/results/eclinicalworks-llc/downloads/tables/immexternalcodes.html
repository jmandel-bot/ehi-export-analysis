EHI Export Schema
Table immexternalcodes
Description This table stores information about external codes associated with immunization items.
Column DataType Description
Immid int This column stores the id of the immlist table as reference.
itemid int This is an integer column representing the item id. It is a unique identifier for each item associated with an immunization. Example: 67890
eCWCode varchar This is a varchar(20) column representing the eClinicalWorks code. It is a unique code assigned by the eClinicalWorks system for each immunization item. Example: eCW123.
externalCode varchar This is a varchar(20) column representing the external code. It is a unique code assigned by an external system for each immunization item. Example: EXT456.
flag (Primary Key) varchar This is a varchar(5) column representing the flag status. It is used to indicate the status of the immunization item, with M being the default value. Example: M.
