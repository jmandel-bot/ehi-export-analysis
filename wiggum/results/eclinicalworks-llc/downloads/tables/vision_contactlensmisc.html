Table vision_contactlensmisc
Description This table stores information related to contact lens dispense orders, including the order facility, notes, etc.
Column DataType Description
id (Primary Key) int This is the primary key for the table, which uniquely identifies each record. It is an integer value that is automatically incremented as new records are added. Example: 1, 2, 3, etc.
encounterid int This column represents the id of the encounter associated with the contact lens. It is a foreign key that refers to the encounter table's primary key. It is an integer value and cannot be NULL. Example: 101, 102, 103, etc.
reasonid int This column represents the id of the reason for the contact lens encounter. It is a foreign key that refers to the reason table's primary key. It is an integer value and cannot be NULL. Example: 1, 2, 3, etc.
reasonnote varchar This column indicates notes for product not suitable for this contact lens.
clnotsuitable tinyint This column will determine whether product is suitable to user or not. The data type of this column in tinyint. The default value for CL not suitable will be 0. Acceptable values are 0 or 1.
