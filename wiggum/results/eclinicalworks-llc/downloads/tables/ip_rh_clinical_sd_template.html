EHI Export Schema
Table ip_rh_clinical_sd_template
Description This table stores the templates created for Progress Notes Section on the clinical side.
Column DataType Description
id (Primary Key) int This is an auto-incremented primary key column in this table. The data type is int because the templates are dynamically created and the templates created could be huge in number. The column cannot be NULL as it will always have an id.
tempName varchar This column stores the template name. The default value is NULL.
categoryId int This column refers to the itemId column in ip_rh_items table. It stores the itemid of the progress notes section for which the template is created.
enableScore tinyint This column stores the value 0/1. Value 1 indicates the score is enabled for the corresponding template and 0 means the score is not enabled.
forGender tinyint This column is used to identify if the template is created for Male, Female or for both the genders. It stores the values 0/1/2. Value 1 indicates the template is created for Male. Value 2 indicates the template is created for Female. Value 0 indicates the template is created for both Male and Female.
ageRangeFrom tinyint This column stores the minimum age for which the template is applied.
ageRangeTo tinyint This column stores the maximum age for which the template is applied.
ageIn varchar This column stores the values MON/YRS/DAY. It indicates if the age range set using the columns ageRangeFrom and ageRangeTo are in MONTHS/YEARS/DAYS respectively.
displayIndex smallint This column stores a number to indicate in which order templates should be listed on user facing screen. The data type is smallint and the value can range from -32,768 to 32,767. The data type is smallint as compared to int is because the expected templates displayed will not exceed the limitation provided by the smallInt datatype.
createdBy int Stores the user id of the person who created the template. It refers to uid column in users table.
createdTime datetime Stores the date and time the template was created. The datetime data type allows for precise storage of date and time values. This field will always have a value hence defined as NOT NULL.
modiBy int Stores the user Id of the person who last modified the template. It refers to uid column in users table.
modiTime datetime Stores the date and time the template was last modified.
deleteFlag tinyint A flag indicating if the template is deleted (1) or not (0). The tinyint data type allows for efficient storage of binary values.
partialScoring tinyint This column stores the values 0/1. Value 1 indicates partial scoring is allowed for the corresponding template. Value 0 indicates partial scoring is not allowed for the corresponding template.
groupName varchar This column stores the name of the group to which the template belongs.
