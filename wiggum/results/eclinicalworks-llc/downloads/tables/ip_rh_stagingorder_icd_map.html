Table ip_rh_stagingorder_icd_map
Description This table stores order mapping that belongs to icd.
Column DataType Description
id (Primary Key) int Acts as the primary key for the table, uniquely identifying each record.
stagingOrderId int Foreign key that refers to a staging order record. This column refers to the ip_rh_ptstagingorders table's id column.
assessmentId int Foreign key that refers to an assessment record. NOT NULL ensures that every record must have an associated assessment.
deleteFlag tinyint Indicates whether a record is marked for deletion. NOT NULL ensures that the delete status must be explicitly set; DEFAULT ((0)) automatically sets the flag to 0 (not deleted) if no value is provided, which is useful for new records. This column is used for soft deletes, where records are flagged instead of being physically deleted from the database.
