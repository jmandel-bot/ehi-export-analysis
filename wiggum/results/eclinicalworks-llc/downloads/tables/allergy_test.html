EHI Export Schema
Table allergy_test
Description This table stores information about allergy tests performed on patients, including test details, results, and billing information.
Column DataType Description
TESTid (Primary Key) int The column stores an id of the allergy test. This is an auto incremental primary key of This column .
FORMid int This column stores id of a form, which is associated with allergy test. It refers to column formid from allergy_form table.
FORMNAME varchar This column stores name of a form, which is associated with allergy test.
ENCOUNTERid int This column stores id of an enc, which is associated with allergy test. It refers to column encounterid from enc table.
PATIENTid int This column stores identifier number of a patient, associated with allergy test. It refers to column UID from user table.
PERFORMED_BY varchar This column stores id of the user who has perfomed allergy test. It refers to column UID from user table.
READ_BY varchar This column stores id of the user who has reviewed allergy test. It refers to column UID from user table.
