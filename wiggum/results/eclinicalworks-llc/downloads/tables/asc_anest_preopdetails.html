EHI Export Schema
Table asc_anest_preopdetails
Description This table stores the preoperative details of anesthesia cases, including the encounter, category, property, and property detail information, as well as any notes and drawings associated with the case.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
Encounterid int identifier for the encounter associated with the anesthesia case. NOT NULL DEFAULT '0'. Example: 101, 102, 103.
catid int identifier for the category associated with the anesthesia case.NOT NULL DEFAULT '0'. Example: 1, 2, 3.
Propid int identifier for the property associated with the anesthesia case. NOT NULL DEFAULT '0'. Example: 10, 20, 30.
PropDetailid int identifier for the property detail associated with the anesthesia case. NOT NULL DEFAULT '0'. Example: 100, 200, 300.
Selection int Integer value representing the selection made for the property detail.DEFAULT NULL. Example: 1, 2, 3.
Notes text Text field for storing any notes related to the anesthesia case. Example: "Patient has a history of hypertension."
createDate datetime Datetime value representing the date and time the record was created. Example: '2021-09-01 12:34:56'
displayseq int Integer value representing the display sequence of the property detail.UNSIGNED DEFAULT NULL. Example: 1, 2, 3.
notes2 text Text field for storing additional notes related to the anesthesia case. Example: "Patient is allergic to penicillin."
displaycatseq int Integer value representing the display sequence of the category.NOT NULL DEFAULT '0'. Example: 1, 2, 3.
drawingnotes text Text field for storing any notes related to drawings associated with the anesthesia case.NOT NULL. Example: "See attached diagram for nerve block placement."
