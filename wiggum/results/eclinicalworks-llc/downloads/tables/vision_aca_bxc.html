EHI Export Schema
Table vision_aca_bxc
Description This table stores the AC ratio and BXC14B values for Vision encounters.
Column DataType Description
id (Primary Key) int This is the primary key for the table, which is an integer that uniquely identifies each record in the table. It is auto-incremented, meaning that it automatically increases by 1 for each new record added. Example: 1, 2, 3, etc.
encounterid int This is a foreign key column that refers to the `encounterid` column in the `enc` table. It is an integer that uniquely identifies the encounter associated with the vision record. Example: 101, 102, 103, etc.
ac_ratio varchar This is a varchar column that stores the AC ratio value as a string. The maximum length of the string is 5 characters. Example: '0.75', '1.25', '2.00', etc.
bxc14b decimal This is a decimal column that stores the BXC14B value as a decimal number with a maximum of 5 digits and 2 decimal places. Example: 12.34, 56.78, 90.12, etc.
