EHI Export Schema
Table asc_opnote_section
Description This table is used to store operative note sections.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key of This column .
opNoteid int This column stores operative note id. Map to id column of asc_opnote. Column does not expect NULL . Default value is '0'.
sectionid int This column stores section id value. Map to sectionid column of asc_custom_pntoppanel table. Column does not expect NULL . Default value is '0'.
displayIndex int This column is used to represent the display order of the record when grouped by certain record identifier.
deleteFlag tinyint This column stores value 0 / 1 indicating if the Operative note section is deleted or not, this is soft delete. 0 - Not Deleted , 1 -Deleted.
modifiedBy int This column stores id to represent who modified the record. Maps to the UID column in user table.
createdDateTime datetime This column stores Date and Time when this record was created.
modifiedDateTime datetime This column stores Date and Time when this record was modified.
sortingInPN tinyint This column stores value 0 / 1 indicating if the Operative note section records are sorted or not. 0 - Not sorted , 1 -Sorted .
latestOnTop tinyint This column stores value 0 / 1 indicating if the Operative note section latest records are on top or not. 0 - Not on top, 1 -On top .
latestOnTopWhenSortingDisabled tinyint A flag indicating whether the latest record should be displayed on top (1) or not (0) when sorting is disabled. Example: 0, 1.
