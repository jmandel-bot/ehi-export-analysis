Table unc_demographics
Description This table stores demographic information about students at a university, including their enrollment status, academic details, and eligibility for various programs.
Column DataType Description
Id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
PatientId int Unique identifier for the student as a patient. Example: 101, 102, 103.
StudentId varchar Unique identifier for the student. Example: "S001", "S002", "S003".
EthnicCode varchar Code representing the student's ethnicity. Example: "HISP", "ASIAN", "WHITE".
WithdrawalDate datetime Date when the student withdrew from the university, if applicable. Example: '2021-09-01', '0000-00-00' (if not withdrawn).
EnrollmentStatus char Character representing the student's enrollment status. Example: 'A' (Active), 'W' (Withdrawn), 'G' (Graduated).
MatriculationDate datetime Date when the student first enrolled in the university. Example: '2019-08-15', '2020-01-10'.
MatriculationYear varchar Year when the student first enrolled in the university. Example: "2019", "2020".
ForeignStatus char Character representing whether the student is a foreign student or not. Example: 'Y' (Yes), 'N' (No).
AcdmDiv varchar Code representing the student's academic division. Example: "ENG", "SCI", "ARTS".
AcdmClass char Character code representing the student's academic class. Example: 'FR' (Freshman), 'SO' (Sophomore), 'JR' (Junior), 'SR' (Senior).
AcdmLevel varchar Code representing the student's academic level. Example: "UGRD" (Undergraduate), "GRAD" (Graduate).
FullTimeStatus char Character representing whether the student is a full-time student or not. Example: 'Y' (Yes), 'N' (No).
CreditHours varchar Number of credit hours the student is enrolled in. Example: "12", "15", "18".
Degree varchar Code representing the student's degree program. Example: "BS", "BA", "MS", "PhD".
Eligibility varchar Code representing the student's eligibility for various programs. Example: "NCAA", "FINAID", "HONORS".
EligibilityDetails varchar Additional details about the student's eligibility for various programs. Example: "Academic scholarship", "Athletic scholarship", "Needs-based financial aid".
EligibilityModDate datetime Date when the student's eligibility information was last modified. Example: '2021-08-01', '2021-09-15'.
NCAACode varchar Code representing the student's NCAA eligibility status. Example: "ELIG", "INELIG", "PEND".
CountryOfOrigin varchar Country where the student is originally from. Example: "USA", "Canada", "India".
CHSCode varchar Code representing the student's high school. Example: "HS001", "HS002", "HS003".
InoculationCode varchar Code representing the student's inoculation status. Example: "IMMUN", "EXEMPT", "PEND".
TermCode varchar Code representing the academic term for which the record is applicable. Example: "FALL2021", "SPRING2022".
