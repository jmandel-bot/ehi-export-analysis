EHI Export Schema
Table social
Description This table stores patient's social history documented during encounters.
Column DataType Description
encounterid int This column stores encounter id of a visit for which Social History is being documented. This column refers to encounterid column in enc table.
catid int This column store itemid of a category for which Social History have been added by the end user. This column refers to itemid column in items table.
itemid int This column stores itemid of items documented under Social Info section on the user interfaceduring the encounter. This column refers to itemid column in items table.
propid int This column stores propids from properties table to indicate properties associated with the data stored in this column. 83 is stored when value stored is Notes, 91 is stored when value stored in this column represent option selected on the User Interface. This column refers to propid column in properties table.
value varchar This column stores free text note documented by end user for Details section on Social History User Interface.
displayIndex smallint This column stores order in which Social History are displayed in the user interface.
