EHI Export Schema
Table vision_order_items
Description This table stores items associated with patient spectacle orders.
Column DataType Description
id (Primary Key) int This column is an auto incremented Primary Key.
pair_id int This column indicates the pair to which this item belongs. This column refers column id from vision_order_pair table.
product_id int This column stores the product id from inventory_product table.
unitprice decimal This column stores the unit amount in decimal format for this repair order.
is_patientframe tinyint not in use.
unique_string varchar not in use.
taxratio_code_id int not in use.
taxamount decimal This column stores the tax amount in decimal format for this repair order.
totalprice decimal This column stores the final amount in decimal format for this repair order.
item_type int This column indicates what item is this. 0 - Frame, 1 - Lens and 2 - Options.
side tinyint This column indicates which side it is. 0 -Both, 1 - Right and 2 - Left.
isdeleted tinyint This column indicates if this item is deleted or not. 1 - item deleted or 0 - Otherwise.
createdby int This column stores the logged in user id who creates this item. This column refers column UID from user table.
createdon datetime This column stores the time stamp when this item is added. The format used is YYYY-MM-DD HH:MM:SS.
updatedby int This column stores the logged in user id who modifies this item. This column refers column UID from user table.
updatedon datetime This column stores the time stamp when this item is modified. The format used is YYYY-MM-DD HH:MM:SS.
dispense_type varchar not in use.
supplierlens varchar not in use.
source_type tinyint This column indicates the source type for this item. 0 - Lab, 1 - Practice and 2 - Patient.
fac_location int This column indicates the facility location fro this item. This column refers column id from inventory_sku table.
