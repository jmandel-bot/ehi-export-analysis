EHI Export Schema
Table claimstatuscategorycodes
Description This table stores information about Claim status Category codes and other related information
Column DataType Description
id (Primary Key) int This column stores height of the patient.
CLMSTS char This column stores the date when height was recorded. Date is stored in 'YYYY-MM-DD 00:00:00.000' format.
CODE char This column stores weight of the patient.
FieldName varchar This column stores the date when weight was recorded. Date is stored in 'YYYY-MM-DD 00:00:00.000' format.
ClaimStatusId varchar This column stores BMI of the patient.
EffectiveDate datetime This column stores the date when BMI was recorded. Date is stored in 'YYYY-MM-DD 00:00:00.000' format.
DeactivateDate datetime This column stores Heart Rate of the patient.
LastModifiedDate datetime This column stores the date when heart rate was recorded. Date is stored in 'YYYY-MM-DD 00:00:00.000' format.
Notes text This column stores head circumference of the patient.
