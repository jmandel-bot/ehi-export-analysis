Table bh_cptusertype
Description This table stores information about different user types in the system along with their creation and modification details.
Column DataType Description
id (Primary Key) int This is the primary key column, which stores a unique identifier for each user type. Example: 1, 2, 3.
UserType varchar This column stores the name of the user type. Example: 'Admin', 'Editor', 'Viewer'.
CreatedBy int This column stores the user id of the person who created the user type. It has a foreign key relation with the `users` table's `UID` column. Example: 101, 102, 103.
CreatedDate datetime This column stores the date and time when the user type was created. Example: '2021-08-01 12:34:56'.
ModifiedBy int This column stores the user id of the person who last modified the user type. It has a foreign key relation with the `users` table's `UID` column. Example: 101, 102, 103.
ModifiedDate datetime This column stores the date and time when the user type was last modified. Example: '2021-08-02 14:56:32'.
DelFlag tinyint This column stores a tiny integer value (0 or 1) indicating whether the user type has been deleted (1) or not (0). Example: 0, 1.
