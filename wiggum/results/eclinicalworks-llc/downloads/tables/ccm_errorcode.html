EHI Export Schema
Table ccm_errorcode
Description This table stores information about error codes and description related to PHM Billing claim / encounter generation.
Column DataType Description
id (Primary Key) int Contains auto incremented id, datatype is INT (11). This column stores integer value which could grow with number of transactions over the years hence integer(Range: 0-2,147,483,647) is selected. It is AUTO_INCREMENT column and has PRIMARY KEY index.
errCode varchar Contains unique error code to identify an error occurred while phm billing batch processing of patients. The data type of this field is varchar(10). This field can have nul value.
errDesc text Contains description of th error code to identify an error occurred while phm billing batch processing of patients. The data type of this field is text. This field can have nul value.
delFlag int Flag indicates if this data is deleted or not, datatype is TINYINT. This column stores binary flag i.e 0/1. 0: not deleted 1: deleted - By default the record must not be in deleted status hence default 0 is chosen.
isRCM int Flag indicates if this error is for RCM client or not, datatype is INT. This column stores binary flag i.e 0/1. 0: not RCM 1: RCM - By default the record must not be RCM hence default 0 is chosen.
