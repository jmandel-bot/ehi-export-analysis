EHI Export Schema
Table adminsites
Description This tables stores a list of Drug Administration sites created by end users.
Column DataType Description
id (Primary Key) int This is an auto-incremented primary key in this column.
SiteCode varchar This columns stores user defined alphanumeric code for the Drug Administration site.
SiteDesc varchar This column stores free text description of the site entered by the end user while creating the Drug Administration site.
status smallint This column stores values 0/1 to indicate whether the drug administration site is active or not.
modifiedBy int This column stores an id of a user who modified the existing Drug Administration site. This column refers to the UID column in the user table.
modifiedDateTime datetime This column stores date and time when the Drug Administration site was modified by the end user. Date and time is stored in YYYY-MM-DD hh:mm:ss.sss format.
createdBy int This column stores an id of a user who created the Drug Administration site. This column refers to the UID column in the user table.
createdDateTime datetime This column stores date and time when the Drug Administration site was created by the end user. Date and time is stored in YYYY-MM-DD hh:mm:ss.sss format.
