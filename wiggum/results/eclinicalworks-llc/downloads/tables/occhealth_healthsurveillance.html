Table occhealth_healthsurveillance
Description This table stores information about health surveillance records in occupational health.
Column DataType Description
hsid (Primary Key) int This is the primary key for the table and is an integer that uniquely identifies each health surveillance record. Example: 1, 2, 3, etc.
hsname varchar This is a varchar column that stores the name of the health surveillance. Example: "Hearing Test", "Vision Test", etc.
mdate datetime This is a time stamp column that stores the date and time when the health surveillance record was last modified. Example: "2021-09-01 12:34:56".
delflag tinyint This is a tinyint column that indicates whether the health surveillance record has been deleted or not. A value of '0' means the record is not deleted, and a value of '1' means the record is deleted. Example: 0, 1.
