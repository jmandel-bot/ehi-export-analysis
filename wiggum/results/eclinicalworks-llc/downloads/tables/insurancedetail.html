EHI Export Schema
Table insurancedetail
Description This table contains information regarding patient insurances like patient id, insurance id, copays etc...
Column DataType Description
Id (Primary Key) int It is the primary key column for the table and it is used to uniquely identify the table row
pid int It is use to store patient Id. This column refers to uid column in user table.
insid int It is use to store insurance Id. This column refers to insId column in insurance table.
startDate varchar Start date of the insurance coverage, in varchar(30) format. Example: "2021-01-01".
endDate varchar End date of the insurance coverage, in varchar(30) format. Example: "2021-12-31".
groupNo varchar Group number of the insurance plan, in varchar(51) format. Example: "GRP12345".
subscriberNo varchar Subscriber number of the insurance plan, in varchar(30) format. Example: "SUB98765".
copays varchar Copay amount for the insurance plan, in varchar(30) format. Example: "20".
copayType varchar Copay type, either in dollar amount or percentage, in varchar(5) format. Example: "$" or "%".
insOrder int Order of the insurance plan for the patient, in int(6) format. Example: 1 for primary insurance, 2 for secondary insurance.
DeleteFlag smallint Flag to indicate if the insurance detail record is deleted, in smallint(6) format. Example: 0 for not deleted, 1 for deleted.
GrId int This column stores guarantor Id for insurance.
GrRel int This column stores guarantor relation for insurance.
IsGrPt int This column stores flag to indicate is guarantor a patient.
PaymentSource varchar Payment source for the insurance plan, in varchar(5) format. Example: "PPO".
groupName varchar Name of the insurance group, in varchar(60) format. Example: "Acme Insurance Group".
InsType varchar Type of insurance, in varchar(10) format. Example: "Medical" or "Dental".
AssignBenefits char Flag to indicate if benefits are assigned to the patient, in char(1) format. Example: "Y" for yes, "N" for no.
SeqNo int Sequence number for the insurance detail record, in int(4) format.
PtSigSource varchar Patient signature source, in varchar(5) format. Example: "Paper" or "Digital".
PPOIndex varchar PPO index for the insurance plan, in varchar(5) format. Example: "PPO1".
PPOId varchar PPO identifier for the insurance plan, in varchar(15) format. Example: "PPO12345".
PayerClaimOfficeNo varchar Payer claim office number, in varchar(10) format. Example: "OFF9876".
SuppInsInd char Supplementary insurance indicator, in char(1) format. Example: "Y" for yes, "N" for no.
MedicaidId varchar Medicaid identifier for the patient, in varchar(25) format. Example: "MCD123456789".
patSubscriberNo varchar Patient subscriber number, in varchar(30) format. Example: "PATSUB12345".
KenPAC int KenPAC identifier, in int(2) format. Example: 1.
InsuranceClass varchar Insurance class, in varchar(10) format. Example: "Standard" or "Premium".
MedicaidSeqNo char Medicaid sequence number, in char(3) format. Example: "001".
EncEligibilityStatus char Encounter eligibility status, in char(3) format. Example: "V" for valid, "I" for invalid.
EligibilityMessage varchar Eligibility message, in varchar(255) format. Example: "Eligible for coverage".
InsuredAltLName varchar Insured alternate last name, in varchar(35) format. Example: "Doe".
InsuredAltFName varchar Insured alternate first name, in varchar(25) format. Example: "John".
InsuredAltMiddleInitial varchar Insured alternate middle initial, in varchar(25) format. Example: "A".
notes text Notes related to the insurance detail record, in text format.
PatientAltLName varchar Patient alternate last name, in varchar(35) format. Example: "Smith".
PatientAltFName varchar Patient alternate first name, in varchar(25) format. Example: "Jane".
PatientAltMiddleInitial varchar Patient alternate middle initial, in varchar(20) format. Example: "B".
SubscriberSecondaryIdentifier varchar Subscriber secondary identifier, in varchar(30) format. Example: "SSID12345".
SubscriberIdentificationQualifier varchar Subscriber identification qualifier, in varchar(4) format. Example: "IDQ1".
DentalIns int Flag to indicate if the insurance is dental insurance, in int(1) format. Example: 1 for yes, 0 for no.
MedicaidSubId varchar Medicaid sub-identifier, in varchar(20) format. Example: "MSUB12345".
SubscriberSecondaryInvRef varchar Subscriber secondary invoice reference, in varchar(30) format. Example: "INVREF98765".
SubscriberSecondaryInvRefQualifier varchar Subscriber secondary invoice reference qualifier, in varchar(4) format. Example: "INVQ1".
InsuranceImgName varchar Insurance image name, in varchar(255) format. Example: "insurance_card.jpg".
MultipleCoPay tinyint Flag to indicate if there are multiple copays, in tinyint(1) format. Example: 1 for yes, 0 for no.
PCPCoPay varchar Primary care physician copay amount, in varchar(10) format. Example: "10".
SpecialistCoPay varchar Specialist copay amount, in varchar(10) format. Example: "20".
OtherCoPay varchar Other copay amount, in varchar(10) format. Example: "15".
CoInsurance varchar Coinsurance percentage, in varchar(10) format. Example: "20%".
hl7id varchar HL7 identifier, in varchar(25) format. Example: "HL712345".
FtpDirPath varchar FTP directory path for insurance-related files, in varchar(255) format. Example: "/mobiledoc".
ScannedInsName varchar Scanned insurance name, in varchar(255) format. Example: "scanned_insurance.pdf".
IsFreshScan char Flag to indicate if the insurance document is a fresh scan, in char(3) format. Example: "yes" or "no".
ptInsStartDate date Patient insurance start date, in date format. Example: "2021-01-01".
ptInsEndDate date Patient insurance end date, in date format. Example: "2021-12-31".
