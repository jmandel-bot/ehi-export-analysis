EHI Export Schema
Table enc_utctimes
Description This table stores the UTC times related to different encounters.
Column DataType Description
id (Primary Key) int This is the primary key for the table, an automatically incrementing integer that uniquely identifies each record in the table. Example: 1, 2, 3, etc.
encounterid int This is a foreign key that refers to the `encounterid` column in the `enc` table. It is an integer that represents the id of the encounter associated with the UTC times. Example: 101, 102, 103, etc.
UTCTimeOut datetime This is a datetime column that stores the UTC time when the encounter started. Example: '2022-01-01 12:00:00', '2022-01-02 14:30:00', etc.
UTCDepTime datetime This is a datetime column that stores the UTC departure time during the encounter. Example: '2022-01-01 12:15:00', '2022-01-02 14:45:00', etc.
UTCArrTime datetime This is a datetime column that stores the UTC arrival time during the encounter. Example: '2022-01-01 13:30:00', '2022-01-02 15:30:00', etc.
UTCTimeIn datetime This is a datetime column that stores the UTC time when the encounter ended. Example: '2022-01-01 14:00:00', '2022-01-02 16:00:00', etc.
