EHI Export Schema
Table pcm_icd_mapping
Description This table stores information related to the ICD codes selected while enrolling a patient in Principal Care Management program.
Column DataType Description
id (Primary Key) int This column is used to identify unique records. It is an auto incremented primary key of This column . Data type is int(11).This field doesn't allow NULL value hence defined as default NOT NULL.
ptacoid int This column is used to identify enrollment details of patient. This is foreign key column. It refers to primary key of ptacoprograms table. Data type is int(11).
icdCode varchar This column used to store chronic condition code  i.e icdCode for which patient is being enrolled. This code are aplhanumeric like E09.00 hence data type is varchar(20). It doesn't allow NULL value.
delflag tinyint This column is used to soft delete data.  Data type is tinyint(1). (0-not deleted, 1- deleted). Default Value is 0.
addedBy int This column is used to store userid to identify which user added chronic condition at enrollment. This is foreign key column. It refers to primary key of user table. Data type is int(11).
addedDateTime datetime This column store datetime when user add chronic condition for patient enrollment. Datatype is Datetime and doesnt allow NULL value.
modifiedBy int This column is used to store user id to identify which user modify chronic condition after  enrollment. This is foreign key column. It refers to primary key of user table. Data type is int(11).
modifiedDateTime datetime This column store datetime when user modify chronic condition after patient enrollment. Datatype is Datetime and doesnt allow NULL value.
