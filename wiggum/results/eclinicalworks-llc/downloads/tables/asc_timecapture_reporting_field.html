Table asc_timecapture_reporting_field
Description This table stores information about reporting fields for a time capture system, including their names, keywords, and flags for start/end and deletion.
Column DataType Description
id (Primary Key) int This is an integer column that serves as the primary key and is auto-incremented.NOT NULL. It uniquely identifies each reporting field in the table. Example: 1, 2, 3, etc.
NAME varchar This is a varchar(50) column that stores the name of the reporting field. It cannot be NULL and has a default value of an empty string. Example: "Work Hours", "Break Time", "Overtime".
keyword varchar This is a varchar(50) column that stores a unique keyword associated with the reporting field. It cannot be NULL and has a default value of an empty string. Example: "work_hours", "break_time", "overtime".
startEndFlag int This is an int(1) column that indicates whether the reporting field is a start (1) or end (0) field. It cannot be NULL and has a default value of 1. Example: 0 or 1.
deleteFlag tinyint This is a tinyint(1) column that indicates whether the reporting field has been deleted (1) or not (0). It has a default value of 0. Example: 0 or 1.
