EHI Export Schema
Table questionnairedata
Description This table stores Questionnaire data records, including information about the document name,patient's Questionnaire questions filled out from the Patient Portal, submitted user details,source details,questionnaire type,action taken details and import details.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key of this column.
UID int This column stores the id of the user who gets the questionniare response. This column refers to the column UID from the user table.
DocName varchar This column stores the name for the questionnaire.
value text Text data containing the value of the questionnaire response submitted from the portal side or kiosk.
pagenum int Integer representing the page number in the document. Example: 0, 1, 2.
dataImported int Integer (2) indicating whether the data has been imported (1) or not (0). Example: 0, 1.
tempUID int This column is not in use.
dtStatusChanged datetime This column is not in use.
encid int This column stores the id of an encounter for which questionniare is documented. This column refers to the column encounterid from the enc table.
completed char This column stores the value 0/1 indicating if the questionnaire has been completed by the user on the kiosk or not. 0 - Not Completed, 1 - Completed.
lastSubmittedQuestion varchar This column saves information of the last submitted question used in the kiosk.
uploadDateTime datetime Datetime representing when the questionnaire data was uploaded. Example: 2021-09-01 12:34:56.
source tinyint This column stores the value 0/1 indicating the questionnaire was submitted from the kiosk or not. 0 - Not submitted, 1 - Submitted.
logtimestamp datetime This column stores the date and time when staff import the patient submitted questionnaire into the Progress Notes.
importedBy int This column stores the id of the Provider/Staff who has imported questionniare data from the portal.
dtimported datetime This column stores the date and time when the staff import the patient submitted questionnaire into the Progress Notes.
actionTaken varchar This column stores the value indicating whether the questionnaire is imported, reviewed, rejected and skipped.
proxyUID int This column stores id of the user who gets the questionniare response. This column refers to column UID from the user table.
ptSubmittimestamp datetime Datetime representing when the questionnaire data was submitted by the user can be NULL. Example: 2021-09-01 12:34:56.
msggUID varchar Varchar (40) containing a message GUID can be NULL. Example: 123e4567-e89b-12d3-a456-426614174000.
questType varchar This is questionnaire type like ASC or general questionnaire.
