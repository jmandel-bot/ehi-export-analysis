EHI Export Schema
Table asc_enc_custom_tpl
Description This table is used to render macro questions and record answers for particular questions.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key of This column .
Encounterid int This column stores enc id value. Map to encounterid column of enc table. Column does not expect NULL . Default value is '0'.
catid int This column stores category item id. Map to itemid column of items table. Column does not expect NULL . Default value is '0'.
Propid int This column stores prop id for macro. Maps to propid column of table properties. Column does not expect NULL . Default value is '0'.
PropDetailid int This column is not in use.
Selection int This column is used to store selection value like number.
Notes text This column is used to stores free text note for macro questions.
createDate datetime This column stores Date and Time when this record was created.
displayseq int This column is used to represent the display order of the record when grouped by certain record identifier.
notes2 text This column is not in use.
displaycatseq int This column is used to represent the display order of the category.
drawingnotes text This column represent note on images while used for label.
copiedFromSentenceid int This column stores id value. Map to id column of asc_pnsmartmacrostnotes table.
sentenceNote text This column stores free text notes for sentence.
sentenceOrder int This column represent order of sentence in macro user side.
childSentence tinyint This column stores value 0 / 1 indicating if the Child sentence used in multi level question or not. 0 - Not used, 1 -Used.
parentid int This column stores parent question id. Map to id column of table asc_enc_custom_tpl.
triggerOptionid int This column represent option id. Map to optionid column of structdataoptions table.
triggerQuestionid int This column represent question id. Map to id column of structdatadetail table
mandatory tinyint This column stores value 0 / 1 indicating if the question is mandatory to answer or not. 0 - Not mandatory, 1 -Mandatory.
deleteFlag tinyint This column stores value 0 / 1 indicating if the record is deleted or not, this is soft delete. 0 - Not Deleted , 1 -Deleted.
createdBy int This column stores id to represent who created the record. Maps to the UID column in user table.
modiBy int This column stores id to represent who modified the record. Maps to the UID column in user table.
modiTime datetime This column stores Date and Time when this record was modified.
versionid int This column represent version id column. Map to id column of asc_enc_custom_tpl_version table.
isUnAnswered tinyint This column stores value 0 / 1 indicating if the question is answered or not. 0 - Not answered, 1 -Answered .
