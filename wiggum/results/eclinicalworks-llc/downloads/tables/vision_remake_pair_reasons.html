Table vision_remake_pair_reasons
Description This table stores information about reasons for remaking vision pairs, including details about the pair, reason, and encounter associated with the remake.
Column DataType Description
id (Primary Key) int This column is an auto incremented primary key.
pairid int This column indicates the pair to which this reason belongs. This column refers column id from vision_order_pair table
reasonid int Identifier for the reason associated with the vision pair remake. This could be a foreign key referring to the reason table's primary key. Example: 201, 202, 203.
isdeleted tinyint A flag indicating whether the record has been deleted (1) or not (0). Example: 0, 1.
createdby int Identifier for the user who created the record. This could be a foreign key referring to the user table's primary key. Example: 301, 302, 303.
createdon datetime Time stamp for when the record was created. Example: '2021-09-01 12:34:56'.
updatedby int Identifier for the user who last updated the record. This could be a foreign key referring to the user table's primary key. Example: 301, 302, 303.
updatedon datetime Time stamp for when the record was last updated. Example: '2021-09-02 14:56:12'.
encounterid int This column indicates the encounter ID of the order to which this reason belongs. This column refers column encounterid from enc table(foreign key reference).
