Table bh_treatment_plan_problem_icd
Description This table stores information about the ICD (International Classification of Diseases) codes associated with the problems in a patient's treatment plan.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
treatmentPlanid int identifier for the treatment plan associated with the problem. Foreign key relation with `bh_treatment_plan.id`. Example: 101, 102, 103.
groupProblemMapid int identifier for the group problem mapping. Foreign key relation with `bh_treatment_plan_group_problem_mapping.id`. Example: 201, 202, 203.
patientid int identifier for the patient associated with the treatment plan and problem. Foreign key relation with `users.UID`. Example: 301, 302, 303.
asmtItemid int identifier for the assessment item associated with the problem. Foreign key relation with `items.itemid`. Example: 401, 402, 403.
icdcode varchar The ICD code associated with the problem. Example: 'A00.0', 'A00.1', 'A00.2'.
userid int identifier for the user who created or modified the record. Example: 501, 502, 503.
deleteFlag tinyint A flag indicating if the record has been deleted (1) or not (0). Example: 0, 1.
