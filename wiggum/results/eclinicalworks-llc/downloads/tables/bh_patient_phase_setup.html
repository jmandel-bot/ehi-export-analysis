Table bh_patient_phase_setup
Description This table stores information about the setup of patient phases in a behavioral health program.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table (e.g., 1, 2, 3).
patientid int Unique identifier for the patient associated with the phase setup, foreign key relation to `users` table on `UID` column (e.g., 101, 102, 103).
bhProgramid int Unique identifier for the behavioral health program associated with the phase setup, foreign key relation to `acoprograms` table on `id` column (e.g., 201, 202, 203).
bhPhasesid int Unique identifier for the behavioral health phase associated with the phase setup, NULLable (e.g., 301, 302, 303).
phaseName varchar Name of the phase associated with the phase setup, NULLable (e.g., "Phase 1", "Phase 2", "Phase 3").
isPhaseEnabled tinyint Flag indicating if the phase is enabled (1) or not (0), default value is 1.
isVisitTypeEnabled tinyint Flag indicating if the visit type is enabled (1) or not (0), default value is 0.
visitTypeid int Unique identifier for the visit type associated with the phase setup, foreign key relation to `visitcodes` table on `Codeid` column, NULLable (e.g., 401, 402, 403).
isSignOffEnabled tinyint Flag indicating if the sign-off is enabled (1) or not (0), default value is 1.
initialSignOff smallint Initial sign-off value, default value is 0 (e.g., 0, 1, 2).
initialSignOffFormat tinyint Format of the initial sign-off, default value is 0 (e.g., 0, 1, 2).
recurringSignOff smallint Recurring sign-off value, default value is 0 (e.g., 0, 1, 2).
recurringSignOffFormat tinyint Format of the recurring sign-off, default value is 0 (e.g., 0, 1, 2).
nextSignOffCalculation tinyint Flag indicating the calculation method for the next sign-off, default value is 0 (e.g., 0, 1).
isRecurring tinyint Flag indicating if the phase setup is recurring (1) or not (0), default value is 0.
deleteFlag tinyint Flag indicating if the record is deleted (1) or not (0), default value is 0.
modifiedBy int Unique identifier for the user who modified the record, foreign key relation to `users` table on `UID` column (e.g., 501, 502, 503).
modifiedDate datetime Date and time when the record was last modified (e.g., "2021-09-01 12:34:56").
activeInitialSignOff smallint Active initial sign-off value, default value is 0 (e.g., 0, 1, 2).
activeRecurringSignOff smallint Active recurring sign-off value, default value is 0 (e.g., 0, 1, 2).
activeInitialSignOffFormat tinyint Active format of the initial sign-off, NULLable (e.g., 0, 1, 2).
activeRecurringSignOffFormat tinyint Active format of the recurring sign-off, NULLable (e.g., 0, 1, 2).
isSignRoutingEnabled tinyint Flag indicating if the sign routing is enabled (1) or not (0), default value is 0.
