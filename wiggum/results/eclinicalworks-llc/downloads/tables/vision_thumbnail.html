EHI Export Schema
Table vision_thumbnail
Description This table stores information about thumbnail images associated with specific encounters.
Column DataType Description
id (Primary Key) int This is the primary key for the table and is an auto-incrementing integer value. It uniquely identifies each record in the table. Example: 1, 2, 3, etc.
encounterid int Identifier for the encounter associated with the thumbnail. This is a foreign key that refers to the `enc` table's `encounterid` column. Example: 101, 102, 103, etc.
document varbinary Binary Large Object (BLOB) data type that stores the actual thumbnail image. Example: Binary data representing the image.
filename varchar Name of the file associated with the thumbnail. Example: "thumbnail1.jpg", "thumbnail2.png", etc.
docid int Identifier for the document associated with the thumbnail. This is a foreign key that refers to the `document` table's `docid` column. Example: 201, 202, 203, etc.
displayindex smallint The order in which the thumbnail should be displayed. Example: 0, 1, 2, etc.
deleteflag tinyint A flag indicating whether the thumbnail has been deleted (1) or not (0). Example: 0 or 1.
catid int Identifier for the category associated with the thumbnail. Example: 301, 302, 303, etc.
itemid int Identifier for the item associated with the thumbnail. Example: 401, 402, 403, etc.
