Table prisma_document_reference_category_details
Description This table stores the category details of document references in the Prisma system.
Column DataType Description
id (Primary Key) int This is the primary key column, which uniquely identifies each record in the table. It is an integer with a maximum length of 11 digits and is auto-incremented. Example: 1, 2, 3, etc.
docRefId int This column stores the document reference ID associated with the category details. It is an integer with a maximum length of 11 digits. Example: 101, 102, 103, etc. This column has a foreign key relation with the `prisma_document_reference_details` table, specifically the `id` column.
categoryCode varchar This column stores the category code of the document reference. It is a variable character string with a maximum length of 200 characters. Example: "CAT001", "CAT002", "CAT003", etc. This column can have NULL values.
categoryDisplay varchar This column stores the display name of the category associated with the document reference. It is a variable character string with a maximum length of 200 characters. Example: "Category 1", "Category 2", "Category 3", etc. This column can have NULL values.
