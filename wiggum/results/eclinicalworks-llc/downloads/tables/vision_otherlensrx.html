Table vision_otherlensrx
Description This table stores information about the vision prescription for other lenses, including right and left eye measurements, categories, and materials.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
Encounterid int Identifier for the encounter associated with the prescription. Example: 101, 102, 103.
bozdOD varchar Base curve value for the right eye (OD). Example: "8.6", "8.4", "8.8".
bozdOS varchar Base curve value for the left eye (OS). Example: "8.6", "8.4", "8.8".
VisionRxid int Identifier for the vision prescription. Example: 1001, 1002, 1003.
categoryOD varchar Category of the lens for the right eye (OD). Example: "Single Vision", "Bifocal", "Progressive".
categoryOS varchar Category of the lens for the left eye (OS). Example: "Single Vision", "Bifocal", "Progressive".
oadOD varchar Overall diameter value for the right eye (OD). Example: "14.0", "14.2", "14.4".
oadOS varchar Overall diameter value for the left eye (OS). Example: "14.0", "14.2", "14.4".
pc1OD varchar First power curve value for the right eye (OD). Example: "-1.00", "-1.25", "-1.50".
pc1OS varchar First power curve value for the left eye (OS). Example: "-1.00", "-1.25", "-1.50".
pc2OD varchar Second power curve value for the right eye (OD). Example: "-1.75", "-2.00", "-2.25".
pc2OS varchar Second power curve value for the left eye (OS). Example: "-1.75", "-2.00", "-2.25".
pc3OD varchar Third power curve value for the right eye (OD). Example: "-2.50", "-2.75", "-3.00".
pc3OS varchar Third power curve value for the left eye (OS). Example: "-2.50", "-2.75", "-3.00".
pc4OD varchar Fourth power curve value for the right eye (OD). Example: "-3.25", "-3.50", "-3.75".
pc4OS varchar Fourth power curve value for the left eye (OS). Example: "-3.25", "-3.50", "-3.75".
ctOD varchar Center thickness value for the right eye (OD). Example: "0.08", "0.10", "0.12".
ctOS varchar Center thickness value for the left eye (OS). Example: "0.08", "0.10", "0.12".
scOD varchar This is a small integer column that stores the pachymetry measurement for the left eye. It can be NULL if the measurement is not available. Example: 540, 550, 560, ...
scOS varchar This is a varchar (255) column that stores any additional comments or notes related to the vision pachymetry measurement. It has a default value of an empty string (''). Example: 'Measurement taken after LASIK surgery', 'Patient has dry eyes', ...
tcOD varchar Integer value representing the type of eye exam conducted. Example: 1 (routine checkup), 2 (glaucoma screening), 3 (diabetic retinopathy screening).
tcOS varchar Horizontal cup-to-disc ratio measurement for the right eye (OD). Stored as a string. Example: "0.6", "0.7", "0.8".
materialOD varchar Vertical cup-to-disc ratio measurement for the right eye (OD). Stored as a string. Example: "0.5", "0.6", "0.7".
materialOS varchar Horizontal cup-to-disc ratio measurement for the left eye (OS). Stored as a string. Example: "0.4", "0.5", "0.6".
colorOD varchar Color of the lens for the right eye (OD). Example: "Clear", "Blue", "Green".
colorOS varchar Color of the lens for the left eye (OS). Example: "Clear", "Blue", "Green".
