EHI Export Schema
Table prisma_ccd_document_section
Description This table stores information about document sections in the Prisma CCD (Continuity of Care Document) system, including section details, HTML text, errors, and related identifiers.
Column DataType Description
id (Primary Key) bigint Unique identifier for each record in the table. Example: 1, 2, 3.
ccdId bigint Stores id of ihe_document table.
section varchar Section identifier within the CCD document. Example: "allergies", "medications", "problems".
sectionTitle varchar Title of the section within the CCD document. Example: "Allergies and Adverse Reactions", "Medications", "Problem List".
htmlText varchar HTML formatted text of the section content. Example: "

Allergies: None

Medications: Aspirin

".
error varchar Error message related to the section, if any.
adtId int Identifier for the ADT (Admission, Discharge, and Transfer) document. Foreign key relation to `adtinbounddocuments.Id`. Example: 201, 202, 203.
patientid int Identifier for the patient associated with the CCD document. Foreign key relation to `users.uid`. Example: 301, 302, 303.
payorId int Identifier for the payor associated with the CCD document. Example: 401, 402, 403.
ccrId int Identifier for the CCR (Continuity of Care Record) document. Example: 501, 502, 503.
dpcId int Identifier for the DPC (Data Processing Center) document. Example: 601, 602, 603.
ccrtimestamp datetime time stamp for the CCR document. Example: "2022-01-01 12:34:56".
sourceId int Identifier for the source of the CCD document. Foreign key relation to `prisma_ccda_parsing_source_config.id`. Example: 1, 2, 3.
fhirId int Stores id of fhir_source_table.
docRefAttchId int Identifier for the document reference attachment. Foreign key relation to `prisma_document_reference_content_element_details.id`. Example: 801, 802, 803.
preprocessTxId varchar Identifier for the preprocessing transaction. Example: "TX123", "TX124", "TX125".
isTxnFailed smallint Flag indicating if the transaction failed (-1) or not (0). Example: -1, 0.
