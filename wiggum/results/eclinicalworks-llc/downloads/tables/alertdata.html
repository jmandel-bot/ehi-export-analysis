EHI Export Schema
Table alertdata
Description This table stores the CDSS Alerts calculated for patients.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key for this column.
Patientid int This column stores an id of the patient who is eligible for the CDSS enabled alert from the Measureid column. This column refers to the column UID from the user table.
Measureid varchar This column stores id of the measure/alert for which that particular patient is eligible. This column refers to the column Measureid from the measureconfiguration table.
MeasureName varchar This column stores the name of the measure from the Measureid column. This column refers to the MeasureName column from the measuredefinitions table.
Itemid int This column stores an id of the item selected which is used to render the patient's compliant with the alert. This column refers to the column itemid from the items table.
Frequency tinyint This column stores the frequency of the alert. This column refers to the column paraValue from the measureparaconfig table.
LastDone varchar This column stores the date when the test was last ordered for the alert in Measureid column. The format used is MM/DD/YYYY.
DueDate varchar This column stores the date when the alert is next due for the patient. The format used is YYYY-MM-DD.
Value varchar This column stores the results of the ordered test.
controlstatus tinyint This column stores the current status of the alert for the patient: 1 - Compliant, 2 - Pending Results, 3 - Noncompliant or Suppressed, 4 - Excluded.
RegistryAlert tinyint This column stores flags 0 or 1.
labReceived tinyint This column stores flags 0 or 1.
mdate datetime This column stores the current time stamp when the CDSS alert is calculated for the eligible patient. The format used is YYYY-MM-DD HH:MM:SS.
isExternalAlert bit This column stores flag 0 or 1. 1 - alert was received from an external source through healow insights, and 0 - alert was not received from an external source.
isMergedToCdssAlert bit This column stores flags 0 or 1. 1 - alert is an external alert that was merged with a CDSS alert, and 0 - otherwise.
