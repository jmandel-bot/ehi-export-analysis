Table prisma_procedure_details
Description This table stores information about medical procedures, their details, and associated metadata.
Column DataType Description
id (Primary Key) bigint Unique identifier for each record in the table (e.g., 1, 2, 3).
sectionId bigint Identifier for the section to which the procedure belongs, foreign key relation to `prisma_ccd_document_section` table's `id` column (e.g., 101, 102, 103).
procedureId varchar Unique identifier for the procedure
codeid varchar Identifier for the code associated with the procedure (e.g., "C001", "C002", "C003").
codeValue varchar Value of the code associated with the procedure (e.g., "12345", "67890", "24680").
statusCode varchar Status code for the procedure (e.g., "active", "completed", "cancelled").
recordedDate varchar Date when the procedure was recorded, stored as a string (e.g., "2021-09-01", "2021-09-02", "2021-09-03").
TargetSiteCode varchar Code for the target site of the procedure (e.g., "TSC001", "TSC002", "TSC003").
TargetSiteValue varchar Value of the target site code (e.g., "left arm", "right leg", "head").
deviceCode varchar Code for the device used in the procedure (e.g., "DC001", "DC002", "DC003").
deviceName varchar Name of the device used in the procedure (e.g., "Surgical Knife", "Syringe", "Stethoscope").
referenceText text Long text containing reference information about the procedure.
authorId varchar Identifier for the author of the procedure (e.g., "A001", "A002", "A003").
authorName varchar Name of the author of the procedure (e.g., "Dr. John Doe", "Dr. Jane Smith", "Dr. Alice Johnson").
outcome varchar Outcome of the procedure
procedureCode varchar Code for the procedure
procedureCodingSystem varchar Coding system used for the procedure (e.g., "ICD-10", "CPT", "SNOMED CT").
encounterReference varchar Reference to the encounter associated with the procedure (e.g., "E001", "E002", "E003").
isException tinyint A flag indicating if the problem code is an exception (1) or not (0). Example: 0, 1.
exceptionMessage varchar Comma separated ids, each id pointing to a message in table "prisma_ccd_preprocessor_error_messages". Empty if there is no exception.
sourcePriority int Priority of the source of the procedure information, stored as an integer (e.g., 1, 2, 3).
codeSystemPriority int Priority of the code system used for the procedure, stored as an integer (e.g., 1, 2, 3).
duplicatedBy int Identifier for the record that duplicates this record, stored as an integer (e.g., -1 for no duplicate, 2 for duplicated by record 2).
