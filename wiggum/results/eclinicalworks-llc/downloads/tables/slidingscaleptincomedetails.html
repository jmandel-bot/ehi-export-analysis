Table slidingscaleptincomedetails
Description This table stores the income details of patients on a sliding scale payment plan, including their gross income, bi-weekly rates, spouse's income, and other income sources.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. (e.g., 1, 2, 3)
Ptid int Unique identifier for the patient, foreign key referring to the patient's table. (e.g., 101, 102, 103)
GrHrRate varchar Hourly rate of the patient's gross income. (e.g., '15.00', '20.00', '25.00')
GrHrPerWeek varchar Number of hours the patient works per week. (e.g., '40', '30', '20')
GrGrossAmt varchar Total gross amount of the patient's income. (e.g., '600.00', '800.00', '1000.00')
GrBiRate varchar Bi-weekly rate of the patient's gross income. (e.g., '1200.00', '1600.00', '2000.00')
GrBiGrossAmt varchar Total bi-weekly gross amount of the patient's income. (e.g., '2400.00', '3200.00', '4000.00')
SpHrRate varchar Hourly rate of the patient's spouse's income. (e.g., '10.00', '15.00', '20.00')
SpHrPerWeek varchar Number of hours the patient's spouse works per week. (e.g., '20', '30', '40')
SpGrossAmt varchar Total gross amount of the patient's spouse's income. (e.g., '400.00', '600.00', '800.00')
SpBiRate varchar Bi-weekly rate of the patient's spouse's income. (e.g., '800.00', '1200.00', '1600.00')
SpBiGrossAmt varchar Total bi-weekly gross amount of the patient's spouse's income. (e.g., '1600.00', '2400.00', '3200.00')
OtherIncomeTypes varchar Types of other income sources for the patient. (e.g., 'rental', 'freelance', 'investment')
OtherIncomeAmt varchar Amount of other income sources for the patient. (e.g., '200.00', '300.00', '400.00')
OtherIncomeGrossAmt varchar Total gross amount of other income sources for the patient. (e.g., '400.00', '600.00', '800.00')
ProofOfIncome varchar Indicator of whether the patient has provided proof of income. (e.g., 'yes', 'no')
Userid int Unique identifier for the user who created or updated the record, foreign key referring to the user's table. (e.g., 1, 2, 3)
UserName varchar Name of the user who created or updated the record. (e.g., 'John Doe', 'Jane Smith', 'Alice Johnson')
timestamp datetime Date and time when the record was created or updated. (e.g., '2021-08-01 12:34:56', '2021-08-02 14:56:23')
OtherIncomeReason varchar Reason for the other income sources. (e.g., 'rental property', 'freelance work', 'investment income')
MemberNotes varchar Additional notes about the patient's income details. (e.g., 'recently changed jobs', 'spouse on maternity leave', 'received inheritance')
