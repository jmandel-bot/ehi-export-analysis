Table prisma_problemlist_details
Description This table stores information about problem list details in a medical record system, including problem names, types, status, and related observations.
Column DataType Description
id (Primary Key) bigint Unique identifier for each record in the table (bigint).
sectionId bigint Foreign key referring to the `prisma_ccd_document_section` table's `id` column, representing the section in which the problem list detail is included (bigint).
entryId varchar Unique identifier for each entry within a section (varchar, 100).
addedOn varchar Date when the problem was added to the list, in the format "YYYY-MM-DD" (varchar, 50).
statusCode varchar Code representing the status of the problem (varchar, 20).
statusName varchar Name of the status corresponding to the `statusCode` (varchar, 50).
problemname varchar Name or description of the problem (varchar, 500).
problemTypeCode varchar Code representing the type of problem (varchar, 50).
problemType varchar Name or description of the problem type corresponding to the `problemTypeCode` (varchar, 250).
onset varchar Date when the problem started, in the format "YYYY-MM-DD" (varchar, 50).
resolvedOn varchar Date when the problem was resolved, in the format "YYYY-MM-DD" (varchar, 50).
observationCode varchar Code representing the observation related to the problem (varchar, 50).
observationName varchar Name or description of the observation corresponding to the `observationCode` (varchar, 255).
problemCondition varchar Condition of the problem (varchar, 20). Example values: "active", "inactive".
problemWuCondition varchar This column represent the verification status of the problem. Ex: Confirmed, Provisional
observationCodeSystem varchar Code system used for the `observationCode` (varchar, 25). Example value: "LOINC".
verificationStatusCode varchar Code representing the verification status of the problem (varchar, 100).
encounterReference bigint Reference to the encounter where the problem was identified (bigint).
recorderReference int Reference to the person or system that recorded the problem (int).
asserterReference int Reference to the person or system that asserted the problem (int).
severityCode varchar Code representing the severity of the problem (varchar, 100).
severityDisplay varchar Name or description of the severity corresponding to the `severityCode` (varchar, 255).
conditionCategoryDisplay varchar Name or description of the condition category (varchar, 255).
fhirId int Identifier for the problem in the FHIR (Fast Healthcare Interoperability Resources) format (int).
deathDiagnosis tinyint The Flag indicates whether the diagnosis for a patient identified as part of death diagnosis. If 1 means the problem identified as part of death diagnosis.
