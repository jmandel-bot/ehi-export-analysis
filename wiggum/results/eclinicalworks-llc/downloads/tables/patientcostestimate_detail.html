EHI Export Schema
Table patientcostestimate_detail
Description This table stores CPT data for cost estimator for patient.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
estid int Identifier for the cost estimate, referring to a specific patient cost estimate. Example: 101, 102, 103.
linenumber tinyint Line number of the cost estimate detail. Example: 1, 2, 3.
cpt char Current Procedural Terminology (CPT) code for the medical procedure. Example: '99213', '99214', '99215'.
mod1 char First modifier for the CPT code. Example: '25', '59', '76'.
mod2 char Second modifier for the CPT code. Example: '25', '59', '76'.
mod3 char Third modifier for the CPT code. Example: '25', '59', '76'.
mod4 char Fourth modifier for the CPT code. Example: '25', '59', '76'.
cptunits int Number of units for the CPT code. Example: 1, 2, 3.
estprovchg decimal Estimated provider charge for the procedure. Example: 100.00, 200.00, 300.00.
provchgoverridden char Indicates if the provider charge has been overridden. Example: 'Y' (Yes) or 'N' (No).
estfacchg decimal Estimated facility charge for the procedure. Example: 100.00, 200.00, 300.00.
facchgoverridden char Indicates if the facility charge has been overridden. Example: 'Y' (Yes) or 'N' (No).
estaneschg decimal Estimated anesthesia charge for the procedure. Example: 100.00, 200.00, 300.00.
aneschgoverridden char Indicates if the anesthesia charge has been overridden. Example: 'Y' (Yes) or 'N' (No).
esttotalchg decimal Estimated total charge for the procedure. Example: 100.00, 200.00, 300.00.
itemid int Identifier for the item related to the procedure. Example: 1001, 1002, 1003.
deleteflag tinyint Indicates if the record has been deleted. Example: 0 (Not deleted) or 1 (Deleted).
estfacunitfee decimal Estimated facility unit fee for the procedure. Example: 100.00, 200.00, 300.00.
estprovunitfee decimal Estimated provider unit fee for the procedure. Example: 100.00, 200.00, 300.00.
NDCCode varchar National Drug Code for the medication related to the procedure. Example: '12345-6789', '98765-4321'.
ICD1 varchar First International Classification of Diseases (ICD) code related to the procedure. Example: 'M54.5', 'E11.9', 'J45.909'.
ICD2 varchar Second International Classification of Diseases (ICD) code related to the procedure. Example: 'M54.5', 'E11.9', 'J45.909'.
