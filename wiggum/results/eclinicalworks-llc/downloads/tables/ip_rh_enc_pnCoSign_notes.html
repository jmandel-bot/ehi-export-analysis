EHI Export Schema
Table ip_rh_enc_pnCoSign_notes
Description This table store the encounter's cosign notes data.
Column DataType Description
id (Primary Key) int Auto increment primary key of the table. Data type defined as: INT with size 11
pnCoSignId int This field stores cosignId from ip_rh_enc_pnCoSign table. This field stores the cosign id for which cosign notes is being mapped.
notes varchar Stores Cosign notes in this column.
deleteFlag tinyint Stores delete the flag of the column. Default value is defined as zero which represents the row is active and not deleted and 1 represents the row is not active and deleted.
createdBy int Column stores the id of a user by whom Cosign notes data is being Created. createdBy is reference column storing uid field value of user's table.
createdDatetime datetime Stores the timestamp of Cosign notes are being Created.
modifiedBy int Column stores the id of a user by whom Cosign notes data is being modified. modifiedBy is reference column storing uid field value of user's table.
modifiedDatetime datetime Stores the timestamp of Cosign notes are being modified.
