Table prisma_advance_directive_observation
Description This table stores information about observations related to advance directives, including details about the observation type, author, verifier, and care agent.
Column DataType Description
id (Primary Key) bigint Unique identifier for each observation record (e.g., 1, 2, 3).
detailId bigint Foreign key relation to the `prisma_advance_directive_details` table, referencing the `id` column. It represents the identifier for the advance directive detail associated with the observation.
observationTypeCode varchar Code representing the type of observation (e.g., "OBS001", "OBS002").
observationType varchar Description of the observation type (e.g., "Medical Condition", "Treatment Preference").
observationCode varchar Code representing the specific observation (e.g., "OBS01A", "OBS02B").
observation varchar Description of the specific observation (e.g., "Diabetes", "No resuscitation").
obsStatus varchar Status of the observation (e.g., "Active", "Inactive").
referencetext varchar Reference text related to the observation (e.g., "Patient has a history of diabetes").
referenceDocName varchar Name of the reference document associated with the observation (e.g., "Medical History.pdf").
authorId varchar Unique identifier for the author of the observation (e.g., "A123", "A456").
authorName varchar Name of the author of the observation (e.g., "Dr. John Doe").
authorAddressLine1 varchar First line of the author's address (e.g., "123 Main St").
authorAddressLine2 varchar Second line of the author's address (e.g., "Apt 4B").
authorCity varchar City of the author's address (e.g., "New York").
authorState varchar State of the author's address (e.g., "NY").
authorpostalcode varchar Postal code of the author's address (e.g., "10001").
authorCountry varchar Country of the author's address (e.g., "USA").
verfierId varchar Unique identifier for the verifier of the observation (e.g., "V123", "V456").
verifierTaxonomyNumberCode varchar Taxonomy number code for the verifier (e.g., "TAX001", "TAX002").
verifierSpeciality varchar Speciality of the verifier (e.g., "Cardiology", "Neurology").
verifirerRelationCode varchar Code representing the relation between the verifier and the observation (e.g., "REL001", "REL002").
verifirerRelation varchar Description of the relation between the verifier and the observation (e.g., "Primary Physician", "Consulting Doctor").
verifirerDocumentedTime varchar time stamp when the verifier documented the observation (e.g., "2021-08-01 12:34:56").
verfierName varchar Name of the verifier (e.g., "Dr. Jane Smith").
verifierAddressLine1 varchar First line of the verifier's address (e.g., "456 Elm St").
verifierAddressLine2 varchar Second line of the verifier's address (e.g., "Suite 300").
verifierCity varchar City of the verifier's address (e.g., "Los Angeles").
verifierState varchar State of the verifier's address (e.g., "CA").
verifierpostalcode varchar Postal code of the verifier's address (e.g., "90001").
verifierCountry varchar Country of the verifier's address (e.g., "USA").
verifierOrgName varchar Name of the organization the verifier is associated with (e.g., "LA Medical Center").
careAgentId varchar Unique identifier for the care agent related to the observation (e.g., "C123", "C456").
careAgentTaxonomyCode varchar Taxonomy code for the care agent (e.g., "TAX003", "TAX004").
careAgentSpeciality varchar Speciality of the care agent (e.g., "Oncology", "Pediatrics").
careAgentRelationCode varchar Code representing the relation between the care agent and the observation (e.g., "REL003", "REL004").
careAgentRelation varchar Description of the relation between the care agent and the observation (e.g., "Care Coordinator", "Nurse Practitioner").
careAgentDocumentedTime varchar time stamp when the care agent documented the observation (e.g., "2021-08-02 14:30:00").
careAgentName varchar Name of the care agent (e.g., "Nurse Mary Johnson").
careAgentAddressLine1 varchar First line of the care agent's address (e.g., "789 Oak St").
careAgentAddressLine2 varchar Second line of the care agent's address (e.g., "Bldg 2").
careAgentCity varchar City of the care agent's address (e.g., "Chicago").
careAgentState varchar State of the care agent's address (e.g., "IL").
careagentpostalcode varchar Postal code of the care agent's address (e.g., "60601").
careAgentCountry varchar Country of the care agent's address (e.g., "USA").
careAgentOrgName varchar Name of the organization the care agent is associated with (e.g., "Chicago Health Clinic").
