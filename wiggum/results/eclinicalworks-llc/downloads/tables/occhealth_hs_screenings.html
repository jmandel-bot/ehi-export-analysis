EHI Export Schema
Table occhealth_hs_screenings
Description This table stores information about occupational health screenings.
Column DataType Description
screeningid (Primary Key) int Unique identifier for each screening record. Example: 1, 2, 3
hsid int identifier for the health screening. It can be NULL. Example: 101, 102, 103
seq int Sequence number for the screening. It can be NULL. Example: 1, 2, 3
screeningname varchar Name of the screening. It can be NULL. Example: "Blood Test", "Vision Test", "Hearing Test"
cptid varchar identifier for the Current Procedural Terminology (CPT) code related to the screening. It can be NULL. Example: "12345", "67890", "11111"
servicename varchar Name of the service related to the screening. It can be NULL. Example: "Lab Test", "Eye Exam", "Audiometry"
action tinyint Indicates whether an action is required for the screening (1) or not (0). Default value is '0'. Example: 0, 1
delflag tinyint Indicates whether the screening record is deleted (1) or not (0). Default value is '0'. Example: 0, 1
serviceid int identifier for the service related to the screening. Default value is '0'. Example: 1, 2, 3
