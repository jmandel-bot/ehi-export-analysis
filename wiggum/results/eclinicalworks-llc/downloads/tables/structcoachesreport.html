EHI Export Schema
Table structcoachesreport
Description This tables stores data for structured questions from the athletics module of coaches' report column information.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3, etc.
injuryid int identifier for the injury associated with the report. This could be a foreign key referring to another table containing injury information. Example: 101, 102, 103, etc.
catid int identifier for the category of the injury. This could be a foreign key referring to another table containing category information. Example: 1, 2, 3, etc.
itemid int identifier for the item related to the injury. This could be a foreign key referring to another table containing item information. Example: 1001, 1002, 1003, etc.
detailid int identifier for the specific details of the injury. This could be a foreign key referring to another table containing detail information. Example: 5001, 5002, 5003, etc.
VALUE text Text value containing additional information about the injury. Example: "Fractured wrist", "Sprained ankle", etc.
valueid varchar A string identifier for the value column. Example: "val_001", "val_002", "val_003", etc.
notes text Text field for storing any additional notes or comments about the injury. Example: "Player fell during practice", "Injury occurred during the game", etc.
chartdisplay tinyint A tinyint(1) value indicating whether the record should be displayed on a chart (1) or not (0). Example: 1, 0.
displayIndex tinyint A tinyint(1) value indicating the display order of the record. Example: 0, 1, 2, etc.
encounterid int identifier for the encounter during which the injury occurred. This could be a foreign key referring to another table containing encounter information. Example: 2001, 2002, 2003, etc.
