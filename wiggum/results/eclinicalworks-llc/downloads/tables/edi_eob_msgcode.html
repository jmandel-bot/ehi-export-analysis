EHI Export Schema
Table edi_eob_msgcode
Description This table stores information related to EDI (Electronic Data Interchange) EOB (Explanation of Benefits) message codes for payments.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
paymentid int identifier for the payment associated with the EOB message code. Example: 1001, 1002, 1003.
eobid int identifier for the EOB associated with the message code. Example: 2001, 2002, 2003.
cascode varchar First CAS (Claim Adjustment Reason) code associated with the EOB. Example: 'CO45', 'PR50', 'OA23'.
cascode2 varchar Second CAS code associated with the EOB. Example: 'CO12', 'PR32', 'OA45'.
cascode3 varchar Third CAS code associated with the EOB. Example: 'CO23', 'PR21', 'OA12'.
cascode4 varchar Fourth CAS code associated with the EOB. Example: 'CO34', 'PR43', 'OA34'.
numcode varchar First numeric code associated with the EOB. Example: '001', '002', '003'.
numcode2 varchar Second numeric code associated with the EOB. Example: '004', '005', '006'.
numcode3 varchar Third numeric code associated with the EOB. Example: '007', '008', '009'.
numcode4 varchar Fourth numeric code associated with the EOB. Example: '010', '011', '012'.
customcode varchar First custom code associated with the EOB. Example: 'C001', 'C002', 'C003'.
customcode2 varchar Second custom code associated with the EOB. Example: 'C004', 'C005', 'C006'.
cascode5 varchar Fifth CAS code associated with the EOB. Example: 'CO56', 'PR54', 'OA56'.
cascode6 varchar Sixth CAS code associated with the EOB. Example: 'CO67', 'PR65', 'OA67'.
numcode6 varchar Sixth numeric code associated with the EOB. Example: '013', '014', '015'.
numcode5 varchar Fifth numeric code associated with the EOB. Example: '016', '017', '018'.
