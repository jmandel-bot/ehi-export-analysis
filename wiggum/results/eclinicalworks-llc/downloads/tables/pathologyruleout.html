EHI Export Schema
Table pathologyruleout
Description This table stores information about pathology rule-out criteria for patients.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key of this column.
itemid int This column stores id of the item. This column refers to column itemid from items table.
deleteflag tinyint This column stores 0/1. 0 - lab order is not deleted. 1- lab order is marked deleted, this is soft delete.
ruleout varchar This column stores the rule out name in context of pathology rule.
displayIndex int This column stores the sequence of the row, it is used to display the same comment in the user interface.
patientid int This column stores identifier number of a patient. It refers to column UID from user table.
