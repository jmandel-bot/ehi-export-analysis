Table phm_outreach
Description This table stores information about outreach activities for patient care in PHM, including Date of contact, type of contact, and related notes.
Column DataType Description
id (Primary Key) int Unique identifier for each outreach record. Example: 1, 2, 3.
ptAcoId int The identifier of the patient ACO program that the outreach is related to. Foreign key relation: `ptacoprograms.id`. Example: 101, 102, 103.
contactDate datetime The date when the outreach contact was made. Example: '2021-08-25', '2021-08-26', '2021-08-27'.
typeOfContact int The identifier of the type of contact made during the outreach. Foreign key relation: `tcmgeneralsettings.id`. Example: 1 (Phone Call), 2 (Email), 3 (In-person visit).
notes varchar Additional notes or information about the outreach. Example: 'Patient was receptive to the call', 'Left a voicemail', 'Discussed care plan'.
deleteFlag tinyint A flag indicating whether the record has been deleted (1) or not (0). Example: 0, 1.
addedBy int The identifier of the user who added the outreach record. Foreign key relation: `users.uid`. Example: 1001, 1002, 1003.
addedDtTime datetime The date and time when the outreach record was added. Example: '2021-08-25 14:30:00', '2021-08-26 10:15:00', '2021-08-27 16:45:00'.
lastmodifiedBy int The identifier of the user who last modified the outreach record. Foreign key relation: `users.uid`. Example: 1001, 1002, 1003.
lastmodifiedDtTime datetime The date and time when the outreach record was last modified. Example: '2021-08-25 15:00:00', '2021-08-26 11:00:00', '2021-08-27 17:30:00'.
