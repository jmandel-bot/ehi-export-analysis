EHI Export Schema
Table struct_asc_keyword_map
Description This table stores the mapping between keywords, encounters, properties, and details, along with their display order, deletion status, and version information.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
keywordid int identifier for the keyword associated with the record. NOT NULL DEFAULT '0'. Example: 101, 102, 103.
encounterid int identifier for the encounter associated with the record.NOT NULL DEFAULT '0'. Example: 201, 202, 203.
propid int identifier for the property associated with the record.NOT NULL DEFAULT '0'. Example: 301, 302, 303.
detailid int identifier for the detail associated with the record.NOT NULL DEFAULT '0'. Example: 401, 402, 403.
displayOrder int Integer value representing the display order of the record.NOT NULL DEFAULT '0'. Example: 1, 2, 3.
deleteFlag int Integer value representing the deletion status of the record. A value of 0 means not deleted, and a value of 1 means deleted. NOT NULL DEFAULT '0'.Example: 0, 1.
sentenceid int identifier for the sentence associated with the record.NOT NULL DEFAULT '0'. Example: 501, 502, 503.
modiBy int identifier for the user who last modified the record.NOT NULL DEFAULT '0'. Example: 1001, 1002, 1003.
modiTime datetime time stamp representing the date and time when the record was last modified. Example: '2022-01-01 12:34:56'.
versionid int identifier for the version of the record.DEFAULT '0'. Example: 1, 2, 3.
