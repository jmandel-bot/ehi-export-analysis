EHI Export Schema
Table voicemsglogs
Description This table stores voice message logs, including information about the message, message status, and related details.
Column DataType Description
ID (Primary Key) int This column is auto-incremented primary key in This column .
ToName varchar This column stores the name to whom Voice message is sent.
ContactType varchar This column specifies the type of contact( home, cell, work).
ToPhone varchar This column stores the number to which Voice message is sent.
Retry int This column is no longer in use
voiceMessageStatus varchar This column stores the Voice message status, status could be 'Logged', 'Uploaded', 'Sending', 'Pending', 'Completed', 'Failed', 'TTSReady', 'TTSPending', 'TTSFailed', 'Cancelled', 'Facility exclude', 'Provider exclude', 'VisitType exclude', 'Resource exclude', 'Family Appt', 'Multiple Appt', 'FailedRetry', 'no-answer', 'busy', 'in-progress', 'queued', 'ringing', 'Futuremsg exclude', 'RxFax exclude', 'Rx-ePresc exclude'
SentDate varchar This column stores the date on which Voice message has been sent.
Priority int This column stores values 0/1 to indicate whether message priority is high or regular. 0 - regular priority, 1 - high priority.
VoiceMessageType varchar This column stores the type of Voice Message. The type can be 'appointment', 'labs', 'PtStatements', 'General', 'unknown', 'Rx', 'TelEnc', 'HealthMaintenance', 'No-Show', 'Birthday', 'Messenger Consent', 'Referral'
VoiceMessageLiveText text This column stores live voice template that is used to send notification based on type.
VoiceMessageLiveAudio text This column is no longer in use
VoiceMessageMachineText text This column stores Machine voice template that is used to send notification based on type.
VoiceMessageMachineAudio text This column is no longer in use
InitialGreetingAudio text This column is no longer in use
VoiceMessageId varchar This column stores reference id indicating records on the messenger server.
Date_Created datetime This column stores the date and time of message creation.
Date_Modified varchar This column stores the date and time of message modification.
isDeleted tinyint This column stores values 0/1 to indicate whether the Voice Message has been marked as deleted or not. 0 - If it is NOT marked as deleted, 1 - If it is marked as deleted.
StatusMsg text This column stores status of the voice message.
CallerIDNumber varchar This column stores the caller number which will be displayed to the patient when they receive the voice message.
InitialGreetingAudioText text This column stores greeting message template which will always play before the actual message.
OtherContactType varchar This column stores email id of the patient if exist.
EncounterID int This column stores the encounter id against which Voice message has be been sent. This column refers to encounterID column in enc table.
MaxLiveReplays smallint This column is no longer in use
OperatorNumber varchar This column stores the operator number, which is used as callback number.
ConfirmationStatus varchar This column stores the confirmation status provided by the patient, status could be 'Confirmed', 'Cancelled', 'Unknown', 'Rescheduled'
DeliveryType varchar This column stores the delivery status, status could be 'Fail', 'Live', 'LiveAndPartial', 'VoiceMail', 'VoiceMailAndPartial', 'LineDown', 'NoMessage', 'Busy', 'NoAnswer', 'Timeout', 'FaxMachine', 'InvalidFile', 'Unknown'.
ErrorMsg text This column stores the error message which is been shown to user in case of any failure.
LabID int This column stores the lab id against which Voice message has be been sent. This column refers to ReportId column in labdata table.
ConfirmCancelEnabled varchar A flag indicating whether the confirmation and cancellation features are enabled (1) or not (0).
PatientId int This column stores the patient id to whom message has been sent. This column refers to pid column in patients table.
PtStatementId int This column is no longer in use
Language varchar This column stores the language in which Voice message has been sent. The language value can be English, Spanish etc.
TTSVoice varchar This column stored the voice type in which the message is to be played on patient's device, value can be 'Male', 'Female'.
VMsgUser varchar This column stored the username of the messenger server where the message is being uploaded.
VMsgPwd varchar This column stored the password of the messenger server where the message is being uploaded.
AppServMachineName varchar This column stores the hostname of the messenger job server through message has be sent.
AppServMachineIP varchar This column stores the ip address of the messenger job server through message has be sent.
ScheduledDate datetime This column stores the scheduler date on which the Voice message is to be send.
ScheduledTime datetime This column stores the scheduler time on which the Voice message is to be send.
SentByLastName varchar This column stores the last name who has initiated Voice message.
SentByFirstName varchar This column stores the first name who has initiated Voice message.
SentBy int This column stores the user id who has initiated Voice message. This column refers to uid column in user table.
FacilityId int This column stores the id of the facility from where the message has been sent. This column refers to Id column in edi_facilities table.
SentTime datetime This column stores the time on which Voice message has been sent.
uploadLocation varchar This column stored the URL of messenger server on which message has ben uploaded.
parentid int This column stores guarantor for the patient.
recordurl text This column is no longer in use
StatusDetail varchar This column is no longer in use
ResourceId int This column stores ID of resource such as anyone on the staff (except a physician), devices, or even rooms.
recordedMessageId varchar This column is no longer in use
ecwVersion varchar This column stores the EMR version through which the Voice message has been sent.
ecmVersion varchar This column stores the eClinicalMessenger version through which the Voice message has been sent.
timeZone varchar This column stores time zone value of the facility.
retryfailedmsg int This column is no longer in use
ApptStatus varchar This column stores the appointment status for the encounter. This column refers to Status column in enc table.
ivrvalue tinyint This column stores value to play ivr menu options after the voice message.
AutoUpgradeKey varchar This column stores the APU id of the practice.
OtherContactNumbers varchar This column stores other contact number to reach patient, incase of message failure for the primary contact.
ErrorCode int This column stores the error code which sent from the messenger server in case of any failure.
MsgExpireTime datetime This column store the expire date and time for the particular Voice message.
PtPin varchar This column stores patient pin for secure voice messages.
PtPinType varchar This column stores patient pin type for secure voice messages.
isMsgSecured tinyint This column stores values 0/1 to indicate whether message is secured of not. 0 - not secured, 1 - secured.
providerid int This column stores provider id.
msgSource varchar This column stores the source of the message creation like Manual Message etc.
providerGreetingsUid varchar This column stores the id of the provider to play greeting audio file.
ClientResponse int This column stores patients response for the voice message.
messageJobType tinyint This column stores values 0/1 to indicate whether it is a first reminder or not. 0 - first reminder, 1 - not first reminder.
FaxLogId int This column stored fax log id. This column refers to FaxLogId column in FaxLogs table.
eRxReqID int This column stores the eRx Request ID. This column refers to ReqID column in surescript_eprescription or surescript_requestLog table.
eRxReFillReqID int This column stores the eRx refill Request ID. This column refers to ReqID column in surescript_eprescription or surescript_requestLog table.
campaignid int This column stores the campaign id for appointment reminder, Rx reminder, Lab reminder etc.
campaign varchar This column stores campaign name.
referralId int This column stores referral id.
customreportid int This column is no longer in use
templateid int This column stores the id of the Voice message template used send Voice message. This column refers to id column in voicemsgTemplateText table.
ScheduledDateTime datetime This column stores schedule date time for sending voice message.
policyid int This column stores policy id for all the reminders like appointment, rx, lab etc.
measureid varchar The identifier for the measure associated with the Voice message.
combinedencids varchar This column is used for family based messages.
