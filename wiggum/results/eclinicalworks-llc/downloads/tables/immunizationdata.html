EHI Export Schema
Table immunizationdata
Description This table stores immunization data records, including information about the document name,patient's immunization questionnaire filled out from the Patient Portal, submitted user details and import details.
Column DataType Description
id (Primary Key) int This is auto-incremented primary key in This column .
UID int This column stores UID of a patient who has submitted immunization questionnaire from Patient Portal. This column refers to UID column in user table.
DocName varchar This column stores name of immunization Questionnaire that patient has filled out form Patient Portal.
value text This column stores response from Patient Portal in XML format.
pagenum int This column stores page number of an immunization form patient has filled out.
dataImported int This column stores values 0/1 to indicate of whether data received from Patient Portal has been imported in to EMR or not. Default value (0) - When staff has not yet imported immunization data. 1 - When staff has imported immunization data.
importedBy int This column stores id of a Staff who has imported immunization data into EMR. This column refers to UID column in user table.
dtimported datetime This column stores date when staff imported immunization data into EMR. Datetime is stored in 'YYYY-MM-DD hh:mm:ss.000' format.
actionTaken varchar This column stores action taken by end user related to immunization data received from Patient Portal. Default (''); - When no action has been performed by end user, 'imported' - when data is imported into EMR.
ptSubmittimestamp datetime This column stores a datetime value representing the time stamp when the record was submitted by the patient. It can be NULL if not submitted. Example: '2021-01-01 12:34:56', '2021-02-01 14:56:23', etc.
