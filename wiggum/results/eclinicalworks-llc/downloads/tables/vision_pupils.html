EHI Export Schema
Table vision_pupils
Description This table stores information about patients' pupils documented during their encounters, including details about their pupil size, brightness, and any abnormalities.
Column DataType Description
id (Primary Key) int This column is an auto incremental primary key.
encounterid int This column stores the encounter ID and has foreign key constraint on "encounterid" column of "enc" table.
isperrla bit This column will indicate whether pupils are equal, round, and reactive to light and accommodation. This column is a bit with a default value of 0. Value is 1 when check box is checked in UI.
isra bit This column will indicate whether pupils are reactive to accommodation. This column is a bit with a default value 0. Value is 1 when checkbox is checked in UI.
islgao bit This column will indicate whether pupils have Limbal Glow Angles Open. This column is a bit with a default value of 0. Value is 1 when checkbox is checked in UI.
apd varchar This column will indicate what type of Afferent Pupillary Defect the pupils have. It can only hold one of the following values - Right, Left, Negative and Both. Default value is NULL indicating non of the radio buttons are selected. It is of type Enum in mysql. In MSSQL it is varchar with a check constraint on it.
r_dim tinyint This column will store values for Dim for the right eye pupil. Values that can be stored are from 1 - 10. Default value is NULL. This column is optional and hence when user does not enter any value, NULL will be stored in them.
l_dim tinyint This column will store values for Dim for left eye pupil. Values that can be stored are from 1 - 10. Default value is NULL. This column is optional and hence when user does not enter any value, NULL will be stored in them.
r_bright tinyint This column will store values for Bright for right eye pupil. Values that can be stored are from 1 - 10. Default value is NULL. This column is optional and hence when user does not enter any value, NULL will be stored in them.
l_bright tinyint This column will store values for Bright for left eye pupil. Values that can be stored are from 1 - 10. Default value is NULL. This column is optional and hence when user does not enter any value, NULL will be stored in them.
