EHI Export Schema
Table audioworkflowspeech
Description This table stores information related to audio workflow speech data, including various SRT, WRData, MCL, and UCL values, as well as test reliability, comments, and equipment ID.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table (e.g., 1, 2, 3, etc.).
Encid int The encounter id associated with the audio workflow speech data. This is a foreign key that refers to the `encounterid` column in the `enc` table.
SRT1 varchar The first SRT (Speech Reception Threshold) value (e.g., "25", "30", "35").
SRT2 varchar The second SRT value (e.g., "25", "30", "35").
SRT3 varchar The third SRT value (e.g., "25", "30", "35").
WRData1 varchar The first WRData (Word Recognition Data) value (e.g., "80", "85", "90").
WRData2 varchar The second WRData value (e.g., "80", "85", "90").
WRData3 varchar The third WRData value (e.g., "80", "85", "90").
WRData4 varchar The fourth WRData value (e.g., "80", "85", "90").
WRData5 varchar The fifth WRData value (e.g., "80", "85", "90").
WRData6 varchar The sixth WRData value (e.g., "80", "85", "90").
WRData7 varchar The seventh WRData value (e.g., "80", "85", "90").
WRData8 varchar The eighth WRData value (e.g., "80", "85", "90").
WRData9 varchar The ninth WRData value (e.g., "80", "85", "90").
WRData10 varchar The tenth WRData value (e.g., "80", "85", "90").
WRData11 varchar The eleventh WRData value (e.g., "80", "85", "90").
WRData12 varchar The twelfth WRData value (e.g., "80", "85", "90").
WRData13 varchar The thirteenth WRData value (e.g., "80", "85", "90").
WRData14 varchar The fourteenth WRData value (e.g., "80", "85", "90").
WRData15 varchar The fifteenth WRData value (e.g., "80", "85", "90").
WRData16 varchar The sixteenth WRData value (e.g., "80", "85", "90").
WRData17 varchar The seventeenth WRData value (e.g., "80", "85", "90").
WRData18 varchar The eighteenth WRData value (e.g., "80", "85", "90").
MCL1 varchar The first MCL (Most Comfortable Level) value (e.g., "50", "55", "60").
MCL2 varchar The second MCL value (e.g., "50", "55", "60").
MCL3 varchar The third MCL value (e.g., "50", "55", "60").
UCL1 varchar The first UCL (Uncomfortable Level) value (e.g., "90", "95", "100").
UCL2 varchar The second UCL value (e.g., "90", "95", "100").
UCL3 varchar The third UCL value (e.g., "90", "95", "100").
TestReliability varchar The reliability of the test (e.g., "High", "Medium", "Low").
Comments text Additional comments related to the audio workflow speech data (e.g., "Patient was cooperative", "Background noise present").
Equipmentid tinyint The id of the equipment used for the test (e.g., 0, 1, 2).
