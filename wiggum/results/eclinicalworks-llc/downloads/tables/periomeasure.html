EHI Export Schema
Table periomeasure
Description This table stores information about periodontal measurements for dental examinations.
Column DataType Description
PerioMeasureNum (Primary Key) bigint Unique identifier for each periodontal measurement record. Example: 1, 2, 3.
PerioExamNum bigint identifier for the periodontal examination that the measurement belongs to. This is a foreign key referring to the `PerioExam` table's primary key. Example: 101, 102, 103.
SequenceType tinyint This column stores value betweek 0 to 6 indicating below things. 0 - Mobility , 1-Furcation, 2-Ging Marg , 3-MGJ , 4-Probing (PD) , 5-Skip Tooth , 6-Bleeding
IntTooth smallint This column stores value 1 to 32 indicating initial tooth number value.
ToothValue smallint This column stores value like 0, -1 ,1 etc...indicating teeth is missing or not or the value of Mobility which is depended on sequence type column value. Column default value = 0. when sequence type =0 then it indicates Mobility value. when sequence type =5 then it indicates skip tooth value (missing teeth). value =-1 indicates that teeth is not missing and 0 indicates that missing teeth.
MBvalue smallint This column stores Probing Depth value for tooth number which is store in upper right side in perio exam.
Bvalue smallint This column stores Probing Depth value for tooth number which is store in upper middle side in perio exam.
DBvalue smallint This column stores Probing Depth value for tooth number which is store in upper left side in perio exam.
MLvalue smallint This column stores Probing Depth value for tooth number which is store in lower right side in perio exam.
Lvalue smallint This column stores Probing Depth value for tooth number which is store in lower middle side in perio exam.
DLvalue smallint This column stores Probing Depth value for tooth number which is store in lower left side in perio exam.
delflag tinyint This column stores value 0 / 1 indicating the Perio measure data is deleted or not. This is soft delete action. Value 0 = Not Deleted , Value 1 =Deleted. This column allows 0 as default value.
