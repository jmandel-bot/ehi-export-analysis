EHI Export Schema
Table kioskdemographicsettinglogdetails
Description This table stores information about the display and mandatory demographic field logs for a specific facility.
Column DataType Description
id int This is an auto incremented primary key of This column .
userid int This column stores default user id of Kiosk.
facilityid int This column stores an id of facility for mapped demographic settings. This column refers to id column from edi_facilities table.
patientType char This column stores value for patient type. 0-Existing Patient, 1- New Patient.
ts datetime time stamp of when the demographic settings were changed. Example: '2021-09-01 12:34:56'.
displayRace char This column stores display value for Race field. 0-Hide, 1-Show.
displayLangauge char A single character indicating whether to display the language field ('1' for yes, '0' for no).
displayEthnicity char A single character indicating whether to display the ethnicity field ('1' for yes, '0' for no).
displayBirthSex char A single character indicating whether to display the birth sex field ('1' for yes, '0' for no).
displaySO char A single character indicating whether to display the sexual orientation field ('1' for yes, '0' for no).
displayGI char A single character indicating whether to display the gender identity field ('1' for yes, '0' for no).
displayProfilePhoto char A single character indicating whether to display the profile photo field ('1' for yes, '0' for no).
displayAddr1 char A single character indicating whether to display the address line 1 field ('1' for yes, '0' for no).
displayAddr2 char A single character indicating whether to display the address line 2 field ('1' for yes, '0' for no).
displayCity char A single character indicating whether to display the city field ('1' for yes, '0' for no).
displayState char A single character indicating whether to display the state field ('1' for yes, '0' for no).
displayZip char A single character indicating whether to display the zip code field ('1' for yes, '0' for no).
displayEmail char A single character indicating whether to display the email field ('1' for yes, '0' for no).
displayCellPhone char A single character indicating whether to display the cell phone field ('1' for yes, '0' for no).
displayHomePhone char A single character indicating whether to display the home phone field ('1' for yes, '0' for no).
displayWorkPhone char A single character indicating whether to display the work phone field ('1' for yes, '0' for no).
displayContact char A single character indicating whether to display the contact field ('1' for yes, '0' for no).
displayPreferredName tinyint A tiny integer indicating whether to display the preferred name field (1 for yes, 0 for no).
