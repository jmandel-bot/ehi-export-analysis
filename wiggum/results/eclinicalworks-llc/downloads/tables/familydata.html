EHI Export Schema
Table familydata
Description This table stores a patient's family member additional details, such as status, DOB, and age at encounter level.
Column DataType Description
encounterid int Stores encounter id of a visit for which Family History is being documented. Column refers to encounterid column in enc table.
itemid int Stores itemid for a relationship selected for documentation by the end user. This column refers to itemid column in items table.
NAME varchar This column stores name of the relationship selected by end user for documentation. This column refers to itemname column in items table.
STATUS varchar This column stores status whether relative of the patient ,whose information is being documented, are alive, deceased or unknown as selected by end user.
age varchar This column stores age of the relative of the patient whose information is being documented.
dob varchar This column stores date of birth of the relative of the patient whose information is being documented.
