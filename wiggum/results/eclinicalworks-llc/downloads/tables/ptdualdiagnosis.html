EHI Export Schema
Table ptdualdiagnosis
Description This table stores information about dual diagnoses for patients, including ICD-9 and ICD-10 codes, patient and encounter IDs, and other related details.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
deleteflag smallint Indicates whether the record has been deleted or not. A value of 0 means not deleted, and a value of 1 means deleted. Example: 0, 1.
icd9itemid int The identifier for the ICD-9 item related to the diagnosis. Example: 12345, 67890.
icd9code varchar The ICD-9 code for the diagnosis. Example: '250.00', '401.9'.
icd10itemid int The identifier for the ICD-10 item related to the diagnosis. Example: 98765, 43210.
icd10code varchar The ICD-10 code for the diagnosis. Example: 'E10.9', 'I10'.
patientid int The identifier for the patient associated with the dual diagnosis. This can be a foreign key referring to the `id` column in the `patient` table. Example: 1001, 1002.
encounterid int The identifier for the encounter associated with the dual diagnosis. This can be a foreign key referring to the `id` column in the `encounter` table. Example: 2001, 2002.
originalitemid int The identifier for the original item associated with the dual diagnosis. Example: 3001, 3002.
originaldisplayindex int The display index of the original item associated with the dual diagnosis. Example: 0, 1, 2.
itype varchar The type of the item associated with the dual diagnosis. Example: 'Diagnosis', 'Procedure'.
