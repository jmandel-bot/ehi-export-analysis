Table asc_pntoppanel
Description This table stores information about different sections in a panel, including their display properties and order.
Column DataType Description
sectionid (Primary Key) int An integer that uniquely identifies each section in the table. It is the primary key and is auto-incremented. Example: 1, 2, 3.
sectionName varchar A varchar(100) that stores the name of the section. It has a NOT NULL constraint and a default value of an empty string. Example: 'dashboard', 'settings'.
displayName varchar A varchar(100) that stores the display name of the section, which is shown to the user. It has a NOT NULL constraint and a default value of an empty string. Example: 'Dashboard', 'Settings'.
iconClass varchar A varchar(30) that stores the CSS class name for the icon associated with the section. It can be NULL. Example: 'icon-dashboard', 'icon-settings'.
encType int An integer that stores the encryption type of the section. It has a NOT NULL constraint and a default value of 0. Example: 0, 1, 2.
displayOrder int An integer that stores the display order of the section in the panel. Sections with lower displayOrder values will appear before those with higher values. It has a NOT NULL constraint and a default value of 0. Example: 0, 1, 2.
deleteflag int An integer that indicates whether the section has been deleted or not. It has a NOT NULL constraint and a default value of 0. A value of 0 means the section is not deleted, and a value of 1 means the section is deleted. Example: 0, 1.
hideSection int An integer that indicates whether the section should be hidden or not. It has a NOT NULL constraint and a default value of 0. A value of 0 means the section is not hidden, and a value of 1 means the section is hidden. Example: 0, 1.
