EHI Export Schema
Table rxhub_rxhistory
Description This column stores a patient's retrieved medication history.
Column DataType Description
RxHubRxHistoryid (Primary Key) int This is a Primary Key of the table and is set to auto-increment.
Patientid int This column stores id of the Patient whose medication history is documented. This column refers column UID from user table.
Encounterid int This column stores id of the encounter of the patient for which External Rx History is retrieved. This column refers to column encounterid from enc table.
PvdFirstName varchar This column stores first name of the provider who retrieves external medication history of the patient.
PvdMiddleName varchar This column stores middle name of the provider who retrieves external medication history of the patient.
PvdLastName varchar This column stores last name of the provider who retrieves external medication history of the patient.
PvdSuffix varchar This column stores assigned suffix of the provider who retrieves medication history of the patient.
PvdPrefix varchar This column stores assigned prefix of the provider who retrieves medication history of the patient.
NDCCode varchar This column stores National Drug Code(NDC) of the documented drug.
DrugName varchar This column stores the name of the Drug added as an external medication.
Strength varchar This column stores strength of the medication from external medication history of the patient.
StrengthUnits varchar This column stores unit of strength of the medication from external medication history of the patient.
Formulation varchar This column stores an information about the form of the medicine in the retrieved external medication history of the patient.
Quantity1 varchar This column stores dispense data of the medication retrieved in the external medication history.
Quantity2 varchar This column stores the second quantity of the medication if present otherwise it is assigned default value.
QuantityUnit1 varchar This column stores the unit of the quantity from quantity1 column.
QuantityUnit2 varchar This column stores the unit of the quantity from quantity2 column.
QuantityCode1 varchar This column stores the code of the quantity from the quantity1 column.
QuantityCode2 varchar This column stores the code of the quantity from the quantity2 column.
Refills varchar This column stores the number of refills of the medication retrieved in the external medication history.
Direction varchar This column stores an information about how the medication was taken from the retrieved external medication history.
LongDrugName1 varchar This column stores name of the drug 1 from the retrieved external medication history.
LongDrugName2 varchar This column stores name of the drug 2 from the retrieved external medication history..
LongDrugName3 varchar This column stores name of the drug 3 from the retrieved external medication history..
LastFillDate varchar This columns stores the date when pharmacy fills the medication last time from the retrieved external medication history.
MoreHistoryAvailable char This column stores a flag 0 or 1. 1- when more medication history is available of a patient, 0 - otherwise
RxHistoryReqDate varchar This column stores request date of the medication history of a patient. The format used is YYYYMMDD.
NCPDPid varchar This column stores unique, national identifier number assigned to the pharmacies. This column refers column NCPDPid from pharmacy table.
PharmacyName varchar This column stores name of the Pharmacy who dispensed the medication.
PharmacyCity varchar This column store city name of the pharmacy documented in PharmacyName column.
DaysSupply varchar This column stores duration of the medication retrieved.
SourceName varchar This column stores name of the source, it could be : Pharmacy or PBM.
writtendate datetime Date the prescription was written. (e.g., "2021-01-01", "2021-06-15")
primaryDiagnosisCode varchar Primary diagnosis code associated with the prescription. (e.g., "I10", "E11.9", "J06.9")
primaryDiagnosisQualifier varchar Qualifier for the primary diagnosis code. (e.g., "ICD-10", "ICD-9")
primaryDiagnosisDescription varchar Description of the primary diagnosis. (e.g., "Essential (primary) hypertension", "Type 2 diabetes mellitus without complications")
secondaryDiagnosisCode varchar Secondary diagnosis code associated with the prescription. (e.g., "F32.9", "G47.9")
secondaryDiagnosisQualifier varchar Qualifier for the secondary diagnosis code. (e.g., "ICD-10", "ICD-9")
secondaryDiagnosisDescription varchar Description of the secondary diagnosis. (e.g., "Major depressive disorder, single episode, unspecified", "Sleep disorder, unspecified")
refillRemaining int Number of refills remaining for the prescription. (e.g., 0, 1, 2)
pvdFacility varchar Name of the facility where the prescribing provider works. (e.g., "General Hospital", "City Clinic", "Medical Center")
pharmacyaddress varchar Address of the pharmacy where the prescription was filled. (e.g., "123 Main St", "456 Elm St")
pharmacystate varchar State where the pharmacy is located. (e.g., "NY", "CA", "IL")
pharmacyzip varchar ZIP code of the pharmacy location. (e.g., "10001", "90001", "60601")
