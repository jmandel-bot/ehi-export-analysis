EHI Export Schema
Table edi_invcpt_supplementalinformation
Description This table stores supplemental information related to invoices and invoice line items.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
Invoiceid int identifier for the invoice that this supplemental information is related to. Foreign key relation to the `Invoice` table's `id` column. Example: 101, 102, 103.
InvCptid int identifier for the invoice line item that this supplemental information is related to. Foreign key relation to the `InvoiceLineItem` table's `id` column. Example: 1001, 1002, 1003.
ReportTypeCode varchar A short code representing the type of report associated with this supplemental information. Example: 'RPT1', 'RPT2', 'RPT3'.
ReportTransmissionCode varchar A short code representing the transmission method of the report associated with this supplemental information. Example: 'EMAIL', 'FAX', 'POST'.
identificationCode varchar A longer code or identifier for the supplemental information. Example: 'id123456789', 'id987654321', 'id246813579'.
isLineLevel tinyint A boolean flag indicating if the supplemental information is related to a line item level (1) or invoice level (0). Example: 0, 1.
deleteFlag tinyint A boolean flag indicating if the record has been marked for deletion (1) or not (0). Example: 0, 1.
FormatidentificationCode varchar A short code representing the format of the supplemental information. Example: 'PDF', 'TXT', 'CSV'.
documentid int identifier for the document associated with this supplemental information. Foreign key relation to the `Document` table's `id` column. Example: 10001, 10002, 10003.
