EHI Export Schema
Table holterreports
Description This table stores information about Holter monitor reports for patients, including file details, report dates, and associated medical professionals.
Column DataType Description
reportId (Primary Key) int This is an auto-incremented primary key in This column .
encId int This column stores id of the encounter for which lab is added. This column refers column encounterId from enc table.
labId int This column stores lab id associated with Holter report
labType int This column stores the type of lab data for the given holter report
patientId int This column stores the id of a patient related with the Holter report.
filePath varchar This column stores the file path of the Holter report.
fileName1 varchar This column stores file name for the Holter report
fileName2 varchar This column stores file name for the Holter report
reportDate datetime This column stores creation date of the Holter report
referredBy varchar This column stores information regarding referrals for the Holter report.
reviewedBy varchar This column stores information of reviewer of the Holter reports
reviewDate datetime This column stores the review date of the holter report
technician varchar This column stores the technician information associated with the Holter report.
usesPaceMaker int Indicates whether the patient uses a pacemaker (1 for yes, 0 for no). (e.g., 0)
reasonForTest varchar Reason for conducting the Holter test. (e.g., "Irregular heartbeat")
instName varchar Name of the institution where the Holter test was performed. (e.g., "Heart Institute")
delFlag int Indicates whether the Holter report has been deleted (1 for yes, 0 for no). (e.g., 0)
FaxReportFileName1 varchar Name of the fax report file associated with the Holter report. (e.g., "fax_report_123.pdf")
comments text Additional comments about the Holter report. (e.g., "Patient experienced dizziness during the test.")
interpretation text Interpretation of the Holter report by the reviewing medical professional. (e.g., "Mild arrhythmia detected.")
ftpdirpath varchar FTP directory path where the Holter report is stored. (e.g., "/ftp/reports/holter/123/")
pdfuploadsuccess int Indicates whether the PDF upload was successful (1 for yes, 0 for no). (e.g., 1)
sdkversion varchar Version of the SDK used to generate the Holter report. (e.g., "1.0.0")
