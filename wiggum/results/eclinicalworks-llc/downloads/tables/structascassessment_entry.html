EHI Export Schema
Table structascassessment_entry
Description This table stores nursing struct ASC assessment notes at template level.
Column DataType Description
entryid (Primary Key) int used to store Auto increment id for the table.
encounterid int used for storing patient's encounter id from enc master table.
templateid int used for storing templateid, which maps to id from ip_struct_data_template table.
isLocked tinyint Its value can be 0 or 1,used to store if template is locked or not.1 indicates template is locked.0 Indicates template is not locked.
reportedBy int used for storing user id ,which maps to UID from user table.
reportedTime datetime used for storing reported date time.Example: '2022-01-01 12:34:56'.
modifiedBy int used for storing user id ,which maps to UID from user table.It is used to stored modified user id.
modifiedTime datetime used for storing modified date time.Example: '2022-01-01 12:34:56'.
notes text This columns stores template notes.
deleteFlag tinyint its value can be 0 or 1, which indicates if item is deleted or not.1 indicates the item is deleted.0 indicates the item exists.
totalScore float used for storing template score.
interpretation text used for storing template interpretation of a ASC nursing assessment.
formulaid int Used for storing formula id , which maps to id from ip_struct_data_template_formula table.
formulaText text Used for storing formula, as the formula can be modified at master entry, this column captures the formula text at assessment entry time.
isCompleted tinyint Not Available
