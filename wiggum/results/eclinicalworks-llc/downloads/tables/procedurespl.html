EHI Export Schema
Table procedurespl
Description This table stores information about medical procedures performed during patient encounters.
Column DataType Description
encounterid int This column stores the encounter id of the patient which the Procedure being documented is associated to. This column refers to column encounterid from enc table.
catid int This column stores the category id of the category associated with the Procedure being documented of the patient. This column refers to column itemid from items table.
itemid int This column stores the id of the item documented in the User Interface. This column refers to column itemid of items table.
propid int This column is not used in the application hence it is assigned its default value.
value varchar This column stores question along with answer to the structured data question. It could be date, Boolean (Yes / No) , Year or Numeric selected in the User Interface. This column could also refer to the column 'name' of structdataoptions table depending upon what category and Organ is selected in the User Interface.
displayIndex smallint This column stores the number in order the structured data question are being displayed on user interfaceor other items.
VALUE2 varchar This column stores the HTML format of the value from value column.
noofwounds tinyint This column stores number of wounds identified for which the procedure is conducted on a patient.
