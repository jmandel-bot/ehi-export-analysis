Table wcv_patient_details
Description This table stores the details of patients, including their Well Visit status and whether the status or related message should be shown to them.
Column DataType Description
id (Primary Key) int This is the primary key for the table, which is an auto-incrementing integer value. It uniquely identifies each record in the table. Example: 1, 2, 3, etc.
patientid int This is a foreign key that refers to the `UID` column in the `users` table. It represents the unique identifier for the patient. Example: 101, 102, 103, etc. Foreign Key Relation: `users` (`UID`).
showMessage tinyint This is a tinyint (1) unsigned column that indicates whether a message should be shown to the patient or not. It can have a value of 0 (do not show message) or 1 (show message). Example: 0, 1
STATUS varchar This is a varchar(10) column that stores the status of the Well Visit of the patient. It has a default value of an empty string. Example: 'Due Soon', 'Past Due', 'WCV scheduled','wcv already done','Current'
