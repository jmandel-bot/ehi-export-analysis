EHI Export Schema
Table vision_eligibility
Description This table stores information about the eligibility of patients for vision-related services based on their encounters and case IDs.
Column DataType Description
id (Primary Key) int The column stores an id of the Vision Eligibility. This is an auto incremental primary key of This column .
encounterid int The column stores an id of the encounter of a patient.
BOSSEncid varchar The column stores an id of the encounter of a patient for vendor system called BOSS.
deleteFlag smallint This column stores values 0/1 to indicate whether the Vision Eligibility has been marked as deleted or not. 0 - If it is NOT marked as deleted, 1 - If it is marked deleted.
patientid int This column stores identifier number of a patient, for whom the vision eligibility is created. It refers to column UID from user table.
STATUS tinyint This column stores values 0/1 to indicate whether status is success or failed.
Caseid int This columns store the number that indicate case associated with vision eligibility
