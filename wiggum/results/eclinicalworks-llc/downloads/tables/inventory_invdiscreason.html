EHI Export Schema
Table inventory_invdiscreason
Description This table stores inventory discount reasons, which are used in the checkout window.
Column DataType Description
id (Primary Key) int Unique identifier for each inventory discrepancy reason. Example: 1, 2, 3, etc.
Name varchar This column stores discount reason for inventory transactions
IsDeleted tinyint A boolean flag indicating if the inventory discrepancy reason has been deleted (1) or not (0). Example: 0 (not deleted), 1 (deleted).
UpdatedBy int The unique identifier of the user who last updated the inventory discrepancy reason. This is a foreign key that refers to the `id` column in the `users` table. Example: 1, 2, 3, etc.
UpdatedOn datetime The time stamp when the inventory discrepancy reason was last updated. Example: '2021-09-01 12:34:56'.
CreatedBy int The unique identifier of the user who created the inventory discrepancy reason. This is a foreign key that refers to the `id` column in the `users` table. Example: 1, 2, 3, etc.
CreatedOn datetime The time stamp when the inventory discrepancy reason was created. Example: '2021-08-01 12:34:56'.
