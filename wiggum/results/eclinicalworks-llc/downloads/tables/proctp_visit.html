Table proctp_visit
Description This table stores information about dental procedures performed during a patient's visit, including details about the procedure, tooth number, surface, and fees.
Column DataType Description
encProcTPNum (Primary Key) int This is the auto-increment primary key column for the table.
encid int This column references the encounterid column of enc table and identifies the encid where the instance of this procedure entry was added/updated.
procNum int This is a reference to the column ProcNum of the table procedurelog.
patientid int This column stores the patient identifier reference for whom the procedure is being documented. It is a reference to the column 'UID' of table 'users'.
deleteFlag tinyint This column identifies whether a procedure is flagged as deleted (soft-deleted) or not for appointment treatment plan. When set to 1, the procedure is not shown for the patient appointment Treatment plan and is treated as a non-existing entry. When 0, the procedure is treated as being present and valid.
Priority bigint This column represents the priority of the procedure and is a reference to the column priorityNo of table dentalpriority.
ProcStatus tinyint this column references the id column of dentalentrystatus table and identifies the entry status of the procedure.
ToothNumTP varchar This column stores the tooth number (if applicable) for which the procedure is applied.
Surf varchar This column stores the surfaces (if applicable) for which the procedure is applied. It's a combination of all applicable surfaces stored as a single string (example - MOD, DL, MODBL, etc).
CodeNum bigint This column references the itemid column of items table and identifies the CPT code associated to the procedure.
procfee decimal This column stores the fee associated to the procedure.
ToothRange varchar If the procedure applied is for multiple teeth, or a specific treatment area, such as quadrant, arch, sextant, or full mouth, that information is stored in this column.
createdByUserid int This column references the UID column of user table and identifies the id of the user who added the procedure.
modifiedByUserid int This column references the UID column of user table and identifies the id of the user who modified the procedure last.
createdDateTime datetime This column stores the date of when the procedure was added. DATE value is NOT NULL
modifiedDateTime datetime This column stores the date of when the procedure was last modified. DATE value is NOT NULL.
procTPNum int This column stores the procedure treatment plan number identifier reference. It is a reference to the column 'procTPNum' of table 'proctp'.
