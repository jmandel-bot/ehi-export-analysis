Table ptparentinfo
Description This table stores information about the parents and other guardians of a patient.
Column DataType Description
id (Primary Key) int auto incremented primary key of the table
Ptid int stores the id of the patient that refers to UID of user table
Mom1 varchar stores name of the mother
Mom1Ph varchar stores phone number of the mother
Mom1Email varchar stores email of the mother
Mom2 varchar stores name of the mother(alternate)
Mom2Ph varchar stores phone number of the mother(alternate)
Mom2Email varchar stores email of the mother(alternate)
Dad1 varchar stores name of the father
Dad1Ph varchar stores phone number of the father
Dad1Email varchar stores email of the father
Dad2 varchar stores name of the mother(alternate)
Dad2Ph varchar stores phone number of the father(alternate)
Dad2Email varchar stores email of the father(alternate)
Other varchar stores name of any other parental relation
OtherPh varchar stores phone number of the any other parental relation
OtherEmail varchar stores email of the any other parental relation
deleteFlag tinyint sets the delete flag for each record
