Table ip_rh_ptorders_ack
Description The table 'ip_rh_ptorders_ack' contains ack orders.
Column DataType Description
id (Primary Key) int This is the primary key for the table.
orderId int This is an Integer data type, cannot be NULL. This field holds the identifier for the order associated with the acknowledgement. It references the orderId column in the ip_rh_ptorders table.
actionType varchar This is a varchar of size 20 and CANNOT be NULL. The default value is empty string. It holds the type of the action or acknowledgement performed on the order.
performedBy int This is an Integer data type, can be NULL. This field hold the identifier of the user or entity who performed the action or acknowledgement. It references the uId column in the users table.
performedDateTime datetime This field is a datetime data type and can be NULL. It represents the date and time the action or acknowledgement was performed.
deleteFlag tinyint This is a tinyint, a smaller range integer. It cannot be NULL and has a default value of 0. This column indicates whether an patient order's acknowledgement has been marked for deletion or not. deleteFlag value 0 mark as not deleted and value 1 mark as deleted.
orderScheduleId int This is an Integer data type, can be NULL. This field is FK and refers to orderScheduleId column in table ip_rh_ptorderschedule.
