EHI Export Schema
Table inventory_vaccount
Description This table stores information about accounts related to a specific vision type inventory service.
Column DataType Description
id (Primary Key) int This is the primary key column, which contains a unique integer value for each record in the table. It is automatically incremented as new records are added. Example values: 1, 2, 3, etc.
serviceName varchar This column contains a string value representing the name of the service associated with the virtual account. Example values: "StreamingService", "OnlineStorage", "CloudComputing", etc.
subscriber varchar This column contains a string value representing the name or identifier of the subscriber associated with the virtual account. Example values: "JohnDoe", "JaneSmith", "CompanyA", etc.
zipcode varchar This column contains a string value representing the postal code of the subscriber's location. Example values: "12345", "A1B2C3", "90210", etc.
