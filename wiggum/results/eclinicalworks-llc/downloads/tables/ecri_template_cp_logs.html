EHI Export Schema
Table ecri_template_cp_logs
Description This table stores logs of clinical notes copied or merged into a set of Progress Notes from either a template or caret encounters.
Column DataType Description
id (Primary Key) int Contains auto incremented id, datatype is INT. This column stores integer value which could grow with every copy or merge activity of each section in Progress Note. It is AUTO_INCREMENT column and has PRIMARY KEY index
module varchar The name of the module where the copy or merge operation was performed in Pt. Specific template or carets, stored as a varchar. For example: "PN", etc.
featureid int It stores source of copy/merge. It refers to id column in table ecri_cp_qualifiers.
copyMergeFlag tinyint A tinyint flag indicating whether the operation was a copy (1) or a merge (0). For example: 0 or 1.
srcid int It stores source encounter id. This column has a foreign key relation with the `enc` table's `encounterid` column.
destid int It stores destination encounter id. This column has a foreign key relation with the `enc` table's `encounterid` column.
userid int The identifier of the user who performed the copy or merge operation, with integer values. This column has a foreign key relation with the `users` table's `UID` column. For example: 501, 502, 503, etc.
entrydatetime datetime It stores date and time when notes are copy/merged through Pt. Specific Template or carets.
