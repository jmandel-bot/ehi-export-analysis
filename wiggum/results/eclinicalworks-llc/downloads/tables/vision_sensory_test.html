EHI Export Schema
Table vision_sensory_test
Description This table stores information about vision sensory tests performed during patient encounters.
Column DataType Description
id (Primary Key) int This is a varchar column with a maximum length of 255 characters that stores any additional comments about the vision sensory test. It has a default value of an empty string. Example: "Patient has astigmatism", "Patient needs reading glasses", etc.
encounterid int This is the primary key column and contains a unique integer value for each record in the table. It is an auto-incrementing column, so the value increases automatically with each new record. Example values: 1, 2, 3, etc.
examtype tinyint This column stores the name of the vision service type as a string with a maximum length of 250 characters. Example values: "Eye Examination", "Contact Lens Fitting", "LASIK Surgery", etc.
rx_type tinyint This column is a small integer that indicates whether a record has been deleted or not. A value of '0' means the record is active, while a value of '1' means the record has been deleted. Example values: 0, 1.
comments varchar This column stores an integer value representing a unique code for each service type. Example values: 1001, 1002, 1003, etc.
