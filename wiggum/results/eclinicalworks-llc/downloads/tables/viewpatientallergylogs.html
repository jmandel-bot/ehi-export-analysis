EHI Export Schema
Table viewpatientallergylogs
Description This table is used to store access logs for the Allergy module, including when an allergy is retrieved/viewed by any user, the allergy name, date/time, and for which encounter or patient.
Column DataType Description
id (Primary Key) int Auto increment identifier to uniquely identify the row
patientid int The columns contains patient id whose allergy information was viewed. This is a foreign key referring to the `pid` column in a `patients` table.
encounterid int The column contains an encounter id of the allergy information which was viewed. This is a foreign key referring to the `encounterid` column in a `enc` table.
userid int The column contains an user id of user who has viewed the allergy information. This is a foreign key referring to the `UID` column in a `users` table.
data text It stores whatever allergy is viewed by the user. Multiple allergies would be concatenated and stored as a one-string value
user_name varchar Stores full name of the user who has accessed the allergy, it's the combination of ufname, uminitial, and ulname from the user table.
viewdate datetime the date on which allergy is viewed in yyyy-MM-dd format, this is a mandatory column as it answer of when the allergy is accessed.
viewtime datetime the date on which allergy is viewed in HH:mm:ss format, this is a mandatory column as it answers when the allergy is accessed.
