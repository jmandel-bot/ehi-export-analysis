EHI Export Schema
Table nhxreferral
Description This table stores information about referrals requests received from P2P. Includes healthcare referring and receiving providers details, procedures,diagnosis and has column to store referral xml which contains patient details name, address, date of birth.
Column DataType Description
Reqid (Primary Key) int This is auto-incremented primary key in This column . This column represents unique requests received over P2P.
refFrom int This column stores id of the provider from whom referral has been received. This column refers to UID column in user table.
authNo varchar This column refers to free text string entered into Auth Code filled in the user interface. This code represents Auth Code provided by insurance company for the referral. Authorization Types and codes are created by end user.
authtype varchar This column stores value of field Auth Type (Authorization Type) selected by end user on the user interface. Authorization Types and codes are created by end user.
date datetime This column stores date on which the referral was created for the patient. Date is stored in 'YYYY-MM-DD 00:00:00.000' format.
reason text This column stores reason why the referral was created. Reason could be free text or pre-configured keywords selected by the provider/staff.
diagnosis text This column stores patient's diagnosis for which referral as created. Column has been declared as TEXT because column stores ICD code with the text for diagnosis and may store more than one diagnosis separated by "|". i.e 721.3 - Lumbosacral spondylosis without myelopathy|333.94 - Restless legs syndrome RLS|311 - Depressive disorder, not elsewhere classified|
refStDate datetime This column stores start date of the referral after which referral can be used by the patient for the visit and the date is selected by provider/staff.
refEnddate datetime This column stores end date of the referral after which referral can not be used by the patient and the date is selected by provider/staff.
visitsAllowed int This column stores a number, which represents number of times visit to referred provider will be allowed with the referral.
visitsUsed int This column stores a number, which represents the number of visits to the referred provider patient has used with the referral.
RefTo int This column stores an id of a provider under To section in User Interface. Provider is to whom patient has been referred to. This column refer to UID column in user table.
notes text This column stores free text notes documented by provider/staff under General Notes section under Notes tab in the User Interface.
referralType varchar This column stores a character I/O to indicate type of the referral being stored. I - Incoming Referral, O - Outgoing Referral, PTRECORD_I - Incoming Patient Record, PTRECORD_O - Outgoing Patient Record.
priority int This column stores priority of the referral documented by provider/staff. 0 - Routine, 1 - Urgent, 2 - Stat
assignedTo varchar This column stores name of the provider/staff to whom this referral has been assigned to.
assignedToid int This column stores id of a provider/staff to whom the referral has been assigned to in the user interfaceby end user. This column refers to UID column in user table.
status tinyint This column stores a digit to indicate current status of the referral received over P2P network. 0 - Referral is Open, 1 - Consultation is pending, 2 - Referral is addressed.
fromfacility int This column stores id of a facility selected by the end user under From section in user interfacewhile creating the referral. This column refers to id column in edi_facilities table.
ToFacility int This column stores id of a facility selected by the end user under To section in user interfacewhile creating the referral. This column refers to id column in edi_facilities table.
speciality int This column stores id for a specialty of the practice for which the referral has been created. This column refers to id column in edi_speciality column.
ReferralNumber varchar This column stores Referral Number provided by a payer for prior authorization of a referral. If Referral Number is missing and if insurance 'requires referral' then claim can not be generated.
clinicalNotes text This column stores free text notes documented by provider/staff under Clinical Notes section under Notes tab in the User Interface.
refEncid int This column stores id of the encounter from which outgoing referral has been created. This column refers to encounterid column in enc table.
extRefid int This column stores id of a referral from practice which sent referral.
apptdate datetime This column stores date for an appointment of a patient with referred provider.
ReceivedDate datetime This column stores date from the Received Date field in the User Interface. Date when referral was received.
procedures text This column stores an ids and names of procedures selected by provider/staff in the user interfacewhile creating the referral. Datatype TEXT was selected because provider/staff may select more than one procedures which are separated by "|" while being stored in this column. i.e 11750 - Excision Ingrown Nail|11730 - AVULSION PLATE PART/COMPLT SIM|11721 - Debridement nails 6+|11720 - Debridement nails 1-5|11719 - Nail Trim any number|
apptTime varchar This column stores time of the appointment which was created based on referral. Column is declared varchar as free text time is being stored in this column.
referralXml text This column stores incoming/outgoing referral from P2P system in XML format.
nhxPatientid int This column stores P2P system id of patient for whom referral has been created or received. This column refer to id column in nhxpatient table.
refFromP2pNPI varchar This column stores National Provider identifier (NPI) number of the referring provider. This column refers to NPI column in doctors table.
Reffromname varchar This column stores name of the referring provider.
refToP2pNPI varchar This column stores National Provider identifier (NPI) number of the referred provider. This column refers to NPI column in doctors table.
Reftoname varchar This column stores name of the referred provider.
apUID int This column stores id of a practice which sent the referral with attached telephone encounter.
srcregistrationid varchar This column is not being used.
refCCP2PNPIs varchar This column stores comma separated National Provider identifiers (NPI) of providers added as CC while using 'Send P2P Patient Record' feature.
Refccp2pnames varchar This column stores comma separated names of providers added as CC while using 'Send P2P Patient Record' feature.
nhxMsgTxid int This column stores message transaction id stored on P2P server.
requeststatus tinyint Column stores values 1/2 to indicate whether request process is successful or unsuccessful. 1 - REQUEST PROCESS SUCCESSFUL, 2 - REQUEST PROCESS UNSUCCESSFUL
p2pDeliveryStatus tinyint This column is used to store p2p delivery status of the referral / patient record
referral360id varchar This column is used to store 360xid of the referral / patient record
