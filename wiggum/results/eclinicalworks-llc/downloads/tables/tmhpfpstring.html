EHI Export Schema
Table tmhpfpstring
Description This table stores information related to demographic and family planning data for individuals.
Column DataType Description
Id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
EncId int Unique identifier for the encounter associated with the record. Example: 101, 102, 103.
Race varchar Race of the individual. Example: "White", "Black", "Asian".
Ethnicity varchar Ethnicity of the individual. Example: "Hispanic", "Non-Hispanic".
Marital varchar Marital status of the individual. Example: "Single", "Married", "Divorced".
Cnty char 3-character code representing the county of the individual. Example: "001", "002", "003".
NumPreg char 2-character code representing the number of pregnancies the individual has had. Example: "01", "02", "03".
NumLiveBirths char 2-character code representing the number of live births the individual has had. Example: "01", "02", "03".
NumLivingChild char 2-character code representing the number of living children the individual has. Example: "01", "02", "03".
NeworExisting varchar Indicates if the individual is a new or existing patient. Example: "New", "Existing".
AnIncome varchar Annual income of the individual. Example: "30000", "40000", "50000".
FamSize char 3-character code representing the size of the individual's family. Example: "001", "002", "003".
BefBCM varchar Status of the individual before the birth control method (BCM) was applied. Example: "Yes", "No".
AftBCM varchar Status of the individual after the birth control method (BCM) was applied. Example: "Yes", "No".
PractLevel varchar The level of practice for the individual. Example: "Beginner", "Intermediate", "Advanced".
NoBCM varchar Indicates if the individual has no birth control method (BCM). Example: "Yes", "No".
TitleX varchar Indicates if the individual is a Title X patient. Example: "Yes", "No".
EligDt varchar Date when the individual became eligible for the program. Example: "2021-01-01", "2021-02-01", "2021-03-01".
TitleCod varchar Code representing the Title of the individual. Example: "001", "002", "003".
FPStr varchar Family planning string, a 42-character code representing the individual's family planning data. Example: "A1B2C3D4E5F6G7H8I9J0K1L2M3N4O5P6Q7R8S9T0".
deleteFlag tinyint A flag indicating if the record has been deleted (1) or not (0). Example: 0, 1.
