EHI Export Schema
Table ip_rh_orderset
Description The ip_rh_orderset table is designed to store information related to standardized order sets setting. It contains the name of the order set, associated encountered, associated measure, and whether age validation is required.
Column DataType Description
Id (Primary Key) int This is Primary Key for the table. This column is a unique identifier for each row, automatically generated when a new row is inserted.
osName varchar This column contains varchar type value belongs to the name of the order set.
encounterId int This column stores a foreign key that references the?encounterid?column in the?enc?table. This maps the encounter with orderset.
measure varchar This column stores the measure associated with the order set. By default, it stores blank. The measure is free text field contains 100-character long string in length.
userId int This column stores a foreign key that references the?uid?column in the?users?table. This maps the user with orderset.
ageValidationFlag tinyint This column contains tinyint type value is a flag that indicates whether age validation is required for the order set. The possible value for the column is either 0 or 1.
deleteFlag tinyint This column contains tinyint type value is a flag that indicates whether the order set has been deleted. The possible value for the column is either 0 or 1. 0 meant for active order and 1 meant for inactive order.
