EHI Export Schema
Table attachedppdimmunizations
Description This table stores a list of past PPD tests pulled into current Progress Notes from the RCP (Right Chart Panel). This table stores the ImmunizationId and the EncounterID if a PPD type of Immunization Order has been pulled into an encounter to document readings.
Column DataType Description
id (Primary Key) int This is auto incremented primary key in this column.
ImmInjId int This column stores id of past test/immunization administered to the patient when the record is pulled to current Progress Notes from Right Chart Panel. This column refer to immunizationid column in immunizations table.
AttachedToEncId int This column stores id of an encounter to which patient's past immunization test/immunization is pulled to from Right Chart Panel. This column refers to encounterid column in enc table.
deleteFlag tinyint This column stores values 0/1 to indicate whether the record has been marked as deleted or not. 0 - Immunization test has NOT been removed from Progress Notes, 1 - Immunization test has been removed from Progress Notes.
