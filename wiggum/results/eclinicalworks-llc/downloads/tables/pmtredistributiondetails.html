EHI Export Schema
Table pmtredistributiondetails
Description This table stores the payment redistribution details, including the original payment, new payment, and the redistribution amount between claims.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
PtAcNo varchar Patient account number as a string. Example: "PA123456", "PA789012".
OrigPmtid int The original payment id as an integer. Example: 101, 102, 103.
NewPmtid int The new payment id after redistribution as an integer. Example: 201, 202, 203.
PmtRedisAmt decimal The payment redistribution amount as a decimal with 9 digits and 2 decimal places. Example: 100.00, 150.25.
RedisFromClaimid int The claim id from which the payment is being redistributed as an integer. Example: 301, 302, 303.
RedisToClaimid int The claim id to which the payment is being redistributed as an integer. Example: 401, 402, 403.
RemainingBal decimal The remaining balance after the payment redistribution as a decimal with 9 digits and 2 decimal places. Example: 50.00, 75.25.
CreatedOn datetime The date and time when the record was created as a datetime value. Example: '2021-09-01 12:34:56', '2021-09-02 14:56:23'.
Taskid int The task id associated with the payment redistribution as an integer. Example: 501, 502, 503.
balAfterRedist decimal The balance after the payment redistribution as a decimal with 9 digits and 2 decimal places. Example: 200.00, 225.25.
