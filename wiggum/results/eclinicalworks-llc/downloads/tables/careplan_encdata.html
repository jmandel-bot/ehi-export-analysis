Table careplan_encdata
Description This table stores information about care plans for patients, including problems, goals, objectives, and other related data.
Column DataType Description
id (Primary Key) int Unique identifier for each care plan entry. Example: 1, 2, 3.
Encid int The identifier for the encounter associated with the care plan. Example: 101, 102, 103.
Ptid int The identifier for the patient associated with the care plan. Example: 1001, 1002, 1003.
Problem varchar A description of the patient's problem or issue. Example: "High blood pressure", "Diabetes", "Anxiety".
Goal varchar A description of the goal for addressing the problem. Example: "Reduce blood pressure", "Control blood sugar levels", "Decrease anxiety symptoms".
Objective varchar A description of the specific objective to achieve the goal. Example: "Lower systolic pressure to 120", "Maintain HbA1c below 7%", "Reduce panic attacks to once a month".
Frequency varchar The frequency at which the care plan should be followed or reviewed. Example: "Daily", "Weekly", "Monthly".
Modality varchar The method or approach used to address the problem. Example: "Medication", "Therapy", "Lifestyle changes".
Notes text Additional notes or information about the care plan. Example: "Patient prefers morning appointments", "Monitor side effects of medication", "Encourage regular exercise".
AssignedToid int The identifier for the healthcare professional responsible for the care plan. Example: 201, 202, 203.
AssignedTo varchar The name of the healthcare professional responsible for the care plan. Example: "Dr. Smith", "Nurse Johnson", "Therapist Brown".
ProblemStatus int An integer representing the status of the problem (e.g., active, resolved, etc.). Example: 0 (active), 1 (resolved), 2 (in progress).
ProblemStatusDesc varchar A description of the problem status. Example: "Active", "Resolved", "In progress".
CompletionDate datetime The date when the care plan was completed or the problem was resolved. Example: '2022-01-01 00:00:00', '2022-02-15 00:00:00', '2022-03-30 00:00:00'.
AssociatedICD varchar The ICD-10 code associated with the problem. Example: "I10", "E11.9", "F41.1".
