Table antigen_concentration_patient
Description This table stores the concentrations for a Shot Schedule of a patient when a Shot Schedule is created from the Formulation window.
Column DataType Description
CONCENTRATION_id (Primary Key) int Unique identifier for each concentration record. Example: 1, 2, 3.
BOTTLE_id int identifier for the bottle containing the antigen concentration. Example: 101, 102, 103.
CONCENTRATION_NAME varchar Name of the antigen concentration. Example: "High Concentration", "Medium Concentration", "Low Concentration".
COLOR varchar Color associated with the antigen concentration. Example: "Red", "Blue", "Green".
CONCENTRATION varchar Concentration value of the antigen. Example: "1:10", "1:100", "1:1000".
START_DOSE varchar Starting dose for the antigen concentration. Example: "0.1ml", "0.2ml", "0.3ml".
GOAL_DOSE varchar Target dose for the antigen concentration. Example: "1ml", "2ml", "3ml".
INCREMENT_BY varchar Amount to increment the dose by. Example: "0.1ml", "0.2ml", "0.3ml".
SHOTS int Number of shots required for the antigen concentration. Example: 1, 2, 3.
SHOTINTERVAL int Interval between shots in days. Example: 7, 14, 21.
SHOTINTERVALNAME varchar Name of the shot interval. Example: "Weekly", "Biweekly", "Monthly".
EXP_DATE datetime Expiration date of the antigen concentration. Example: "2022-12-31", "2023-01-31", "2023-02-28".
SCHEDULE_id int identifier for the schedule associated with the antigen concentration. Example: 1001, 1002, 1003.
QUEUE_id int identifier for the queue associated with the antigen concentration. Example: 2001, 2002, 2003.
FREQUENCY varchar Frequency of the antigen concentration administration. Example: "Daily", "Weekly", "Monthly".
TEMPLATE_id int identifier for the template associated with the antigen concentration. Example: 3001, 3002, 3003.
EXP_DURATION int Duration of the antigen concentration's expiration in days. Example: 30, 60, 90.
DISPLAYINDEX int Index for the display order of the antigen concentration. Example: 1, 2, 3.
LOCATION varchar Location where the antigen concentration is administered. Example: "Clinic A", "Clinic B", "Clinic C".
ROUTE varchar Route of administration for the antigen concentration. Example: "Subcutaneous", "Intramuscular", "Intradermal".
SELECTED tinyint Indicates if the antigen concentration is selected (1) or not (0). Example: 1, 0.
TOTAL_MINI_BOTTLE_VOL varchar Total volume of the mini bottle containing the antigen concentration. Example: "10ml", "20ml", "30ml".
TOTAL_ALLERGEN varchar Total amount of allergen in the antigen concentration. Example: "100mg", "200mg", "300mg".
B_TEMPLATE_id int identifier for the bottle template associated with the antigen concentration. Example: 4001, 4002, 4003.
UNITS_BILLED int Number of units billed for the antigen concentration. Example: 1, 2, 3.
EXP_DURATION_TYPE int Type of expiration duration for the antigen concentration. Example: 0 (days), 1 (weeks), 2 (months).
FORMULATIONid int identifier for the formulation associated with the antigen concentration. Example: 5001, 5002, 5003.
IS_MULTIPLE_FREQUENCY tinyint Indicates if multiple frequencies are used for the antigen concentration (1) or not (0). Example: 1, 0.
FREQ_id int identifier for the frequency associated with the antigen concentration. Foreign key relation to `allergy_structdata` table's `id` column. Example: 6001, 6002, 6003.
