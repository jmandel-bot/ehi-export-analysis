Table biometricimage
Description This table stores biometric image data and related information for user authentication and verification purposes.
Column DataType Description
id (Primary Key) int Unique identifier for each biometric image record. (e.g., 1, 2, 3)
LoggedInUser int The id of the user who is currently logged in when the biometric image is captured. Default value is 0. (e.g., 1, 2, 3)
Patientid int The id of the patient associated with the biometric image. Default value is 0. (e.g., 1, 2, 3)
EnrolledBiometricMask int The mask value for the enrolled biometric type. Default value is 1. (e.g., 1, 2, 3)
MaxEnrolledCount int The maximum number of enrolled biometric images allowed for a user. Default value is 1. (e.g., 1, 2, 3)
BiometricDevice varchar The type of biometric device used to capture the image. Possible values are 'Finger Printing', 'Palm Secure', 'Eye-Control', and 'SmartCard'. (e.g., 'Finger Printing', 'Palm Secure')
BinaryImage text The binary data of the captured biometric image. (e.g., binary data of a fingerprint image)
CapturedEventHandlingStatus varchar The status of the biometric image capture event. Possible values are 'Success' and 'Failure'. Default value is 'Failure'. (e.g., 'Success', 'Failure')
VerifiedEventHandlingStatus varchar The status of the biometric image verification event. Possible values are 'NotInitiated', 'Success', and 'Failure'. Default value is 'NotInitiated'. (e.g., 'NotInitiated', 'Success')
LoggedInUserOnVerification int The id of the user who is logged in when the biometric image is verified. Default value is 0. (e.g., 1, 2, 3)
ClientIP varchar The IP address of the client device used to capture the biometric image. (e.g., '192.168.1.1', '10.0.0.2')
DateCreated datetime The date and time when the biometric image record was created. (e.g., '2021-01-01 12:34:56')
DateModified datetime The date and time when the biometric image record was last modified. This value is automatically updated on modification. (e.g., '2021-01-02 12:34:56')
DateVerified datetime The date and time when the biometric image was last verified. (e.g., '2021-01-03 12:34:56')
deleteflag tinyint A flag to indicate whether the biometric image record is deleted or not. Default value is 0 (not deleted). (e.g., 0, 1)
RawData varchar The raw data of the captured biometric image in a string format. (e.g., 'raw data of a fingerprint image')
SecurityPin varchar The security PIN associated with the biometric image. (e.g., '1234', '5678')
