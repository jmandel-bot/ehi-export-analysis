EHI Export Schema
Table structdemographics
Description This tables stores data for structured questions from Patient Demographics module.
Column DataType Description
patientid int This column stores id of a patient whose structured demographics data is being stored. This column refers to UID column in user table.
catid int This column is not being used.
itemid int This column is not being used.
detailid int This column stores id of the question or item being documented by end user. This column refers to id column in structdatadetail table.
value text This column stores value documented by end user for respective question or item.
valueid varchar This column stores id of an option selected from a drop-down for question or item being documented by end user. This column refers to optionid column in structdataoptions table.
notes text This column stores free text notes documented by end user for a question or item.
chartdisplay tinyint This column stores values 0/1 to indicate whether record should be shown on Progress Note or not. 0 - Data should be displayed on Progress Note, 1 - Data Should not be displayed on Progress Note.
displayIndex tinyint This column is not being used.
id (Primary Key) int The primary key and unique identifier for each record in the table. Example: 98765
deleteflag int A flag indicating whether the record has been deleted (1) or not (0). Example: 0
lastUpdate datetime The time stamp of the last update made to the record. Example: "2021-10-01 12:34:56"
websync tinyint A flag indicating whether the record has been synchronized with a web-based system (1) or not (0). Example: 1
