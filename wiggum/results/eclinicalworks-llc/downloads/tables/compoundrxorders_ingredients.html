Table compoundrxorders_ingredients
Description This table stores information about the ingredients used in compound prescription orders, including their strength, formulation, and quantity.
Column DataType Description
id (Primary Key) int It is auto-generated unique primary key in This column .
encounterid int identifier for the encounter associated with the compound prescription order. Foreign key relation: `enc.encounterid`. Example: 101, 102, 103.
rxOrderNo varchar Unique identifier for the compound prescription order. Example: 'RX001', 'RX002', 'RX003'.
oldRxid int identifier for the old prescription associated with the compound prescription order. Foreign key relation: `oldrxmain.oldRxid`. Example: 201, 202, 203.
itemid int identifier for the ingredient item used in the compound prescription order. Foreign key relation: `items.itemid`. Example: 301, 302, 303.
NAME varchar Name of the ingredient used in the compound prescription order. Example: 'Acetaminophen', 'Ibuprofen', 'Aspirin'.
strength varchar Strength of the ingredient used in the compound prescription order. Example: '500mg', '200mg', '325mg'.
formulation varchar Formulation of the ingredient used in the compound prescription order. Example: 'Tablet', 'Capsule', 'LiqUID'.
quantity varchar Quantity of the ingredient used in the compound prescription order. Example: '30', '60', '90'.
quantityunit varchar Unit of measurement for the quantity of the ingredient used in the compound prescription order. Example: 'mg', 'g', 'ml'.
ndc_code varchar National Drug Code for the ingredient used in the compound prescription order. Example: '12345-6789', '98765-4321', '56789-1234'.
displayIndex int Display order index for the ingredient in the compound prescription order. Example: 0, 1, 2.
csaSchedule char Controlled Substance Act schedule classification for the ingredient. Example: '0', '1', '2'.
