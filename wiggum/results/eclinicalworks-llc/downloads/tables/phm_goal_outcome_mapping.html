Table phm_goal_outcome_mapping
Description This table stores the mapping between goals and outcomes for a specific encounter and problem encounter.
Column DataType Description
id (Primary Key) int This is the primary key for the table, an auto-incrementing integer that uniquely identifies each mapping record. Example: 1, 2, 3.
encId int This column stores careplan and Patient Problems encounterId. Column refers to encounterid column in enc table.
problemEncId int This column stores careplan and Patient Problems encounterId. Column refers to encounterid column in enc table. This can be NULL if there is no specific problem encounter
goalId int This is a foreign key that refers to the `Id` column in the `structdatadetail` table.
outcomeId int This is a foreign key that refers to the `Id` column in the `phm_outcome` table.
deleteflag tinyint This is a tinyint column that indicates whether the mapping record has been deleted or not. A value of '0' means the record is not deleted, while a value of '1' means the record is deleted. The default value is '0'. Example: 0, 1.
