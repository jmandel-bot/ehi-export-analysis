EHI Export Schema
Table dentalmovements
Description This table stores any movements applied on the teeth for a patient. Each movement type for each tooth number will be stored as a separate row in the table.
Column DataType Description
pid int This column stores the patient identifier reference for who the movements are being stored. It is a reference to the column 'UID' of table 'users'.
toothnum char This column stores a single tooth number (primary or permanent) for which the movement is stored. Each movement type and each tooth will have a separate entry in the table.
movementtype char This column represents the type of movement being recorded in the row. Currently supported values are M, O, L, and MT for Mesial, Occlusal, Labial and Mesial Tip, respectively.
value smallint This column stores a numeric value representing the movement applied. It supports negative and positive integers.
