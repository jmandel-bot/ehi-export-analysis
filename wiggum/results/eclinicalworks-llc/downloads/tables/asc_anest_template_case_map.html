EHI Export Schema
Table asc_anest_template_case_map
Description This table stores the mapping information of anesthesia templates to cases, including display order, deletion status, category, and other related details.
Column DataType Description
id (Primary Key) int The unique identifier for each record in the table. Example: 1, 2, 3.
templateid int The identifier of the anesthesia template being mapped. NOT NULL DEFAULT '0'. Example: 101, 102, 103.
displayIndex int The display order of the anesthesia template in the case.DEFAULT '0'. Example: 1, 2, 3.
deleteFlag int Indicates whether the record is deleted or not. A value of 0 means not deleted, and a value of 1 means deleted.NOT NULL DEFAULT '0' Example: 0, 1.
catid int The identifier of the category that the anesthesia template belongs to. NOT NULL DEFAULT '0'. Example: 10, 11, 12.
createdBy int The identifier of the user who created the record. NOT NULL DEFAULT '0'. Example: 1001, 1002, 1003.
modifiedBy int The identifier of the user who last modified the record. NOT NULL DEFAULT '0'. Example: 1001, 1002, 1003.
isVerified tinyint Indicates whether the record is verified or not. A value of 0 means not verified, and a value of 1 means verified. DEFAULT '0'Example: 0, 1.
TIME datetime The date and time when the record was created or modified. Example: '2022-01-01 12:34:56'.
comments varchar Any additional comments or notes related to the record. Example: 'This is a sample comment.'.
isDefault tinyint Indicates whether the anesthesia template is the default template for the case. A value of 0 means not default, and a value of 1 means default.NOT NULL DEFAULT '0' Example: 0, 1.
