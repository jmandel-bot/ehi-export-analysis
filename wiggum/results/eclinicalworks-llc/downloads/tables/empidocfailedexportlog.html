EHI Export Schema
Table empidocfailedexportlog
Description This table stores the reason for document export failure error reason.
Column DataType Description
id (Primary Key) int This is the primary key for the table, an automatically incrementing integer value that uniquely identifies each failed export attempt. Example: 1, 2, 3, etc.
docid int This column contains the doc id value for when an export was failed. It refers to the document.docid column.
timestamp datetime This column represents when an entry was made into this time stamp column. It inserts the current date and time.
errorlogs text This column contains an error message for a document export failure.
