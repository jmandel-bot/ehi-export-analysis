Table asc_dischargesummary_optnotesect_map
Description This table stores the mapping between discharge summaries and operation note sections, including information about whether the section is checked, deleted, or standard.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3.
opNoteSecMappingid int The identifier for the operation note section mapping. Example: 101, 102, 103.
encounterid int The identifier for the encounter associated with the discharge summary. Example: 1001, 1002, 1003.
isChecked tinyint A boolean flag indicating whether the operation note section is checked or not. 1 for checked, 0 for not checked.
deleteflag tinyint A boolean flag indicating whether the record is deleted or not. 1 for deleted, 0 for not deleted.
isStandard int An integer flag indicating whether the operation note section is standard or not. 1 for standard, 0 for not standard.
opNoteStdSecMappingid int The identifier for the operation note standard section mapping. Example: 201, 202, 203. Foreign key relation: asc_opnote_standardsection_mapping(id).
