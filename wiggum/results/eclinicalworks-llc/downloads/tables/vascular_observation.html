EHI Export Schema
Table vascular_observation
Description This table stores added, updated, or deleted observations for the Vascular module.
Column DataType Description
id (Primary Key) int This column is auto-incremented primary key in This column .
encounterid int The column stores an id of the encounter of a patient. It refers to column encounterid from enc table.
patientid int This column stores identifier number of a patient, for whom the observation is created. It refers to column UID from user table.
labdataid int This column stores identifier number of a labdata. It refers to column reportid from labdata table.
findingid int This column stores identifier number of a Finding. It refers to column id from vascular_finding table.
anatomyid int This column stores identifier number of a Anatomy. It refers to column id from vascular_Anatomy table.
annotation varchar This column stores annotation for Observation.
size float This column stores size for Finding.
notes varchar This column stores Notes for Observation.
augmentation tinyint This column stores augmentation for Observation.
compression tinyint This column stores compression for Observation.
reflux tinyint This column stores reflux for Observation.
type tinyint This column stores View type value 1(Vein) or 2(Skin).
leg varchar This column stores leg identifier L(Left) or R(Right).
createdby int This column stores identifier number of a user who created the observation. It refers to column UID from user table.
createddate datetime This column stores the current time stamp when observation is created. The format used is YYYY-MM-DD HH:MM:SS.
modifiedby int This column stores identifier number of a user who modified the observation. It refers to column UID from user table.
modifieddate datetime This column stores the current time stamp when observation is modified. The format used is YYYY-MM-DD HH:MM:SS.
deleteflag tinyint This column stores value 0 / 1 indicating this observation is deleted. This is soft delete. 0 - Not Deleted , 1 -Deleted
refluxtime float This column stores refluxtime for Observation.
