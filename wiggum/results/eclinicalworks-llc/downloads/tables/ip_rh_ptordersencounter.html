EHI Export Schema
Table ip_rh_ptordersencounter
Description This table is used for storing patient order encounters related details.
Column DataType Description
Id (Primary Key) int This column is a primary key of this table. It is a unique identifier for each row, automatically generated when a new row is inserted.
encId int This column is used to store the unique virtual encounter id. It contains integer type value. This column stores a foreign key that references the encounterid column in the enc table.
encDate datetime This column is used to store the date from the date column from enc table when the encounter is created.It stores date in YYYY-MM-DD HH:mm:ss format.
documentedBy int This column is used to store the unique id of the user who documented the encounter. This column stores a foreign key that references the uid column in the users table.
