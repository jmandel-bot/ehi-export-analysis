EHI Export Schema
Table asc_anest_ioorders_temp
Description This table is used to store temporary anesthesia chart INTAKE orders from the anesthesia chart.
Column DataType Description
id (Primary Key) int This is a primary key which increments itself automatically.
itemid int This column stores an item id for temporary anesthesia chart INTAKE orders. It refers to id column in items table.
encounterid int This column stores encounter id. It refers to the encounterid column from enc table
givenAmount varchar This column stores an amount of medication in measurable value.
routeid int This column stores the route id of respective medication route.
siteid int This column stores a site id of respective medication.
source varchar This column stores the source of a medication. The source can be, io-meds or meds.
startDateTime datetime This column stores the Datetime when medication has started.
isPaused tinyint This column stores a flag to indicate if the status of medication is paused. 0: No, 1: Yes.
givenByid int This column stores the id of a user who has modified the record. It refers to the UID column in user table.
notes text This column stores the note against the use of medication if a patient has some allergies etc.
oldRxid int This column stores an old medication item id. It refers to id column of OldRxMain table.
deleteFlag tinyint This column stores soft delete data. 0: Not deleted; 1: Soft Deleted
givenAmountUnit varchar This column stores the given amount of medication unit like mg, ml, mcg,% etc.
