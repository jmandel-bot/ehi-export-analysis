EHI Export Schema
Table lablotnumbers
Description This table stores information about lab lot numbers, including details about the manufacturer, expiry date, number of doses, dosage, and doses remaining for each lot number.
Column DataType Description
id (Primary Key) int This is an auto incremented primary key of this column.
lotNumber varchar This column stores the lot number.
manufacturer varchar This column stores manufacturer item.
ExpiryDate datetime The expiry date of the lab item. Example: 2022-12-31, 2023-01-01.
NoOfDoses int The total number of doses available in the lab item. Example: 100, 200.
Dosage varchar The dosage amount for each dose of the lab item. Example: 10mg, 5ml.
DosesRemaining int The number of doses remaining in the lab item. Example: 50, 75.
Facilityid int This column is used to store facilitiy id of the edi_facilities table as reference.
deleteflag tinyint This column stores the delete flag. It’ll be set to 1 if entry is deleted, otherwise 0.
