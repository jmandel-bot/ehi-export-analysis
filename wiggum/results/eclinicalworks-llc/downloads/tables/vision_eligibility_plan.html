Table vision_eligibility_plan
Description This table stores information about vision eligibility plans, including details about the carrier, plan, employer, and other related notes.
Column DataType Description
id (Primary Key) int Unique identifier for each vision eligibility plan record. Example: 1, 2, 3.
eligid int identifier for the eligibility associated with the plan. Example: 101, 102, 103.
CarrierName varchar Name of the insurance carrier providing the vision eligibility plan. Example: "Carrier A", "Carrier B", "Carrier C".
Planname varchar Name of the vision eligibility plan. Example: "Plan A", "Plan B", "Plan C".
PlanNotes varchar Additional notes or information related to the vision eligibility plan. Example: "Covers annual eye exams", "Includes contact lens benefits".
CarrierNotes varchar Additional notes or information related to the insurance carrier. Example: "Nationwide coverage", "Preferred provider network".
EmployerName varchar Name of the employer associated with the vision eligibility plan. Example: "Company A", "Company B", "Company C".
insid int identifier for the insurance record associated with the plan. Example: 201, 202, 203.
deleteflag int Flag to indicate if the record has been deleted (1) or not (0). Example: 0, 1.
