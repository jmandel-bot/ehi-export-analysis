Table vision_cas_test
Description This table stores information about vision tests performed during patient encounters, including the type of test and the specific row and column numbers associated with the test results.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3, etc.
Encounterid int identifier for the patient encounter during which the vision test was performed. This could be a foreign key referring to the `Encounters` table's primary key (e.g., `Encounters.id`). Example: 101, 102, 103, etc.
OD_OS_Type tinyint Indicates whether the test was performed on the right eye (OD), left eye (OS), or both eyes (OU). Example: 0 (OD), 1 (OS), 2 (OU).
TestType tinyint Indicates the type of vision test performed. Example: 0 (Visual Acuity), 1 (Color Vision), 2 (Contrast Sensitivity), etc.
RowNo int The row number associated with the test result. Example: 1, 2, 3, etc.
ColNo int The column number associated with the test result. Example: 1, 2, 3, etc.
