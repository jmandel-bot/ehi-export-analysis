Table patient_census
Description This table stores information about the census data of patients, including their location and other related details.
Column DataType Description
id (Primary Key) int Unique identifier for each record in the table. Example: 1, 2, 3, etc.
Ptid int Unique identifier for each patient, referring to their id in another table (foreign key relation not specified). Example: 101, 102, 103, etc.
Census varchar Census information for the patient. Example: "Census 1", "Census 2", "Census 3", etc.
Unix varchar Unix time stamp representing the date and time of the census data. Example: "1633024800", "1633111200", "1633197600", etc.
CVTCode varchar Code related to the census data. Example: "CVT001", "CVT002", "CVT003", etc.
Type varchar Type of the location where the patient resides. Example: "Residential", "Commercial", "Rural", etc.
Place varchar Name of the place where the patient resides. Example: "New York", "Los Angeles", "Chicago", etc.
Number varchar Street number of the patient's residence. Example: "123", "456", "789", etc.
Direction varchar Direction of the street where the patient resides. Example: "North", "South", "East", "West", etc.
Street varchar Name of the street where the patient resides. Example: "Main St", "Broadway", "Elm St", etc.
MailCity varchar City where the patient receives mail. Example: "New York", "Los Angeles", "Chicago", etc.
CensusZip varchar Zip code of the patient's census location. Example: "10001", "90001", "60601", etc.
deleteFlag tinyint Flag indicating if the record has been deleted (1) or not (0). Example: 0, 1.
