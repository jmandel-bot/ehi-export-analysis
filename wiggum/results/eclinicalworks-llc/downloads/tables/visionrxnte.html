Table visionrxnte
Description This table stores the log information related to updates in the VisionRx system for patients and their encounters.
Column DataType Description
id (Primary Key) int This is the primary key of the table, which is an integer value that uniquely identifies each record in the table. It is auto-incremented, meaning that the value increases automatically for each new record added. Example: 1, 2, 3, etc.
encounterId int This is a foreign key column that refers to the `id` column in the `encounters` table. It is an integer value that represents the unique identifier of the encounter associated with the vision prescription. Example: 101, 102, 103, etc.
formattedString varchar This is a varchar(80) column that stores a formatted string representation of the vision prescription. It has a maximum length of 80 characters and cannot be NULL. Example: "OD: -1.00, OS: -1.25, PD: 62"
deleteFlag tinyint This is a tinyint(2) column that indicates whether the record has been deleted or not. It has a default value of '0', which means the record is not deleted. If the value is '1', the record is considered deleted. Example: 0 (not deleted), 1 (deleted)
