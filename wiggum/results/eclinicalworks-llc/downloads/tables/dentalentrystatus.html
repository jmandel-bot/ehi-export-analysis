EHI Export Schema
Table dentalentrystatus
Description This is the dictionary table for dental entry statuses. Every dental procedure documented will be in one of the statuses defined in this table.
Column DataType Description
id (Primary Key) int This is the primary key column, which uniquely identifies each dental entry status. It is an integer with a maximum length of 11 digits and is automatically incremented. Example: 1, 2, 3, etc.
entryStatus varchar This column stores the status of the dental entry as a string with a maximum length of 20 characters. Example: 'Completed', 'Pending', 'Cancelled', etc.
colorCode varchar This column stores the color code associated with the dental entry status as a string with a maximum length of 10 characters. Example: '#FF5733', '#C70039', '#900C3F', etc.
icon varchar This column is currently unused and will be removed in a future build.
displayImg varchar This column is currently unused and will be removed in a future build.
orderStatus smallint This column represents the order of precedence given to the Entry Status when displaying procedures on the Progress Notes. The lower the value, the higher is the status (Example - status 1 is given precedence over status 3).
