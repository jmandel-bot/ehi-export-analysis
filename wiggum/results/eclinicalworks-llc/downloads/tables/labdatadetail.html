EHI Export Schema
Table labdatadetail
Description This table stores detailed information about laboratory data reports, including the properties and values associated with each report.
Column DataType Description
Reportid int This column stores the ID of the received lab. This column refers to the reportid column from the labdata table.
Propid int This column stores an ID for each attribute in the ordered test. This column refers to the itemid column from the items table.
Value varchar This column stores the value for the attribute from the Propid column in the result received.
importFlag int Column is not in use
id (Primary Key) int This column stores the auto-incremented primary key.
UpdatedBy int This column stores the ID of the provider/staff who manually updated the received results. This column refers to the UID column from the user table. If the results are received through an interface, then the value in this column is 0.
UpdatedTime datetime This column stores the current time stamp when the results are received or updated once received. The format used is YYYY-MM-DD HH:MM:SS.
Testid int This column stores the ID of the test from a device.
flag varchar This column stores data in varchar (15) format, indicating any specific flags or statuses associated with the record. Example: verified.
resultEntrySource tinyint This column stores a tinyint (4) representing the source of the result entry, such as manual entry (0) or automated import (1). Example: 0.
