Table prisma_goal_address
Description This table stores information about goal addresses, including their relationship with goal details and conditions, as well as the display of resources.
Column DataType Description
ID (Primary Key) int This is the primary key for the table and is an auto-incrementing integer value. It uniquely identifies each goal address. Example: 1, 2, 3, etc.
GOAL_DETAILS_ID int This is a foreign key column that refers to the `prisma_goal_details` table's `id` column. It represents the relationship between a goal address and its corresponding goal details. Example: 1, 2, 3, etc. Foreign Key Relation: `prisma_goal_details` (`id`)
CONDITION_ID int This is an integer column that stores the condition ID associated with a goal address. It can have a NULL value if there is no specific condition associated with the goal address. Example: 1, 2, 3, NULL, etc.
RESOURCE_DISPLAY varchar This is a varchar(255) column that stores the display text for the resources related to a goal address. It can have a NULL value if there is no specific resource display text. Example: "Resource 1", "Resource 2", NULL, etc.
