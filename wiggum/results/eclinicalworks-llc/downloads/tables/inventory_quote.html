EHI Export Schema
Table inventory_quote
Description This table stores information about inventory quotes, including details about the proposal, pricing, discounts, and related entities such as patients, providers, and facilities.
Column DataType Description
id (Primary Key) int Unique identifier for each inventory quote record. Example: 1, 2, 3.
ProposalDate datetime time stamp of when the quote was proposed. Example: '2021-09-15 12:34:56'.
Expires datetime time stamp of when the quote expires. Example: '2021-10-15 12:34:56'.
Patient_id int Foreign key referring to the `id` column in the `Patient` table. Represents the patient associated with the quote.
Coordinator_id int Foreign key referring to the `id` column in the `Coordinator` table. Represents the coordinator associated with the quote.
Provider_id int Foreign key referring to the `id` column in the `Provider` table. Represents the provider associated with the quote.
Anes_Facility_id int Foreign key referring to the `id` column in the `Anes_Facility` table. Represents the anesthesia facility associated with the quote.
Facility_id int Foreign key referring to the `id` column in the `Facility` table. Represents the facility associated with the quote.
Facility_Time varchar The time spent at the facility. Example: '2 hours'.
Anesthesia_Time varchar The time spent under anesthesia. Example: '1 hour 30 minutes'.
Status_id int Foreign key referring to the `id` column in the `Status` table. Represents the status of the quote.
Notes text Additional notes or comments about the quote. Example: 'Patient requires extra care during anesthesia.'
IsDeleted tinyint A flag indicating if the quote has been deleted (1) or not (0). Example: 0, 1.
UpdatedBy int Foreign key referring to the `id` column in the `User` table. Represents the user who last updated the quote.
UpdatedOn datetime time stamp of when the quote was last updated. Example: '2021-09-16 14:45:23'.
CreatedBy int Foreign key referring to the `id` column in the `User` table. Represents the user who created the quote.
CreatedOn datetime time stamp of when the quote was created. Example: '2021-09-15 12:34:56'.
ProcedureName varchar The name of the procedure associated with the quote. Example: 'Knee Replacement'.
Package_id int Foreign key referring to the `id` column in the `Package` table. Represents the package associated with the quote.
SubPackage_id int Foreign key referring to the `id` column in the `SubPackage` table. Represents the subpackage associated with the quote.
ChargesAmt decimal The total amount of charges for the quote. Example: 10000.0000.
DiscountAmt decimal The total amount of discounts applied to the quote. Example: 1000.0000.
TaxesAmt decimal The total amount of taxes applied to the quote. Example: 500.0000.
TotalAmt decimal The total amount payable for the quote after applying discounts and taxes. Example: 9500.0000.
Discount_id int Foreign key referring to the `id` column in the `Discount` table. Represents the discount associated with the quote.
Discount_Value decimal The value of the discount applied to the quote. Example: 10.0000.
DiscReason_id int Foreign key referring to the `id` column in the `DiscReason` table. Represents the reason for the discount.
isProdNonBillable tinyint A flag indicating if the product is non-billable (1) or billable (0). Example: 0, 1.
OutsideFees decimal The total amount of fees from outside sources. Example: 200.0000.
Quote_Facility_id int Foreign key referring to the `id` column in the `Quote_Facility` table. Represents the facility associated with the quote.
DiscReason_id_Surg int Foreign key referring to the `id` column in the `DiscReason` table. Represents the reason for the discount on the surgery.
DiscountType_Surg int The type of discount applied to the surgery. Example: 1 (percentage), 2 (fixed amount).
DiscValue_Surg decimal The value of the discount applied to the surgery. Example: 5.0000.
DiscountPerLine_Surg decimal The discount per line item for the surgery. Example: 50.0000.
DiscReason_id_Anes int Foreign key referring to the `id` column in the `DiscReason` table. Represents the reason for the discount on anesthesia.
DiscountType_Anes int The type of discount applied to anesthesia. Example: 1 (percentage), 2 (fixed amount).
DiscValue_Anes decimal The value of the discount applied to anesthesia. Example: 5.0000.
DiscountPerLine_Anes decimal The discount per line item for anesthesia. Example: 25.0000.
OutsideDiscAmt decimal The total amount of discounts from outside sources. Example: 100.0000.
CptNProductPrice decimal The price of the CPT (Current Procedural Terminology) and product. Example: 1500.0000.
FacFees decimal The total amount of facility fees. Example: 3000.0000.
AnesFees decimal The total amount of anesthesia fees. Example: 2000.0000.
IsFacilityBillSep tinyint A flag indicating if the facility bill is separate (1) or not (0). Example: 0, 1.
IsAnesBillSep tinyint A flag indicating if the anesthesia bill is separate (1) or not (0). Example: 0, 1.
CoreTotal decimal The core total amount of the quote. Example: 8000.0000.
IsNewQuoteDisplay tinyint A flag indicating if the quote is new and should be displayed (1) or not (0). Example: 0, 1.
