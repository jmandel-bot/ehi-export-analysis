EHI Export Schema
Table specialty_types
Description This table stores information about various specialty types, their display names, and display order.
Column DataType Description
id (Primary Key) int This is the primary key column, containing unique integer values to identify each specialty type (e.g., 1, 2, 3, etc.).
specialty varchar This column contains unique varchar(255) values representing the specialty type (e.g., cardiology, neurology, orthopedics, etc.).
displayname varchar This column contains varchar(255) values representing the display name of the specialty type. These names are used for displaying purposes in the user interface (e.g., Cardiology, Neurology, Orthopedics, etc.).
displayindex tinyint This column contains tinyint(4) values representing the display order of the specialty types. Lower values will be displayed first (e.g., 1, 2, 3, etc.).
deleteflag bit This column contains bit(1) values indicating whether or not the specialty type has been deleted. A value of b0 means the specialty type is not deleted, while a value of b1 means it is deleted.
