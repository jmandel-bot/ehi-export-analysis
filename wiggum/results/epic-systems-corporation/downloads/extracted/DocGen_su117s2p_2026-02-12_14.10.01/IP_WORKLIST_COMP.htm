IP_WORKLIST_COMP
Description:
This table provides information on work list tasks.

Primary Key
Column Name Ordinal Position
TASK_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 TASK_ID VARCHAR No
The unique identifier for the task record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 COMP_ON_TIME DATETIME (Local) No
The instant a work list task was completed.
4 COMP_USER_ID VARCHAR No
The unique ID of the user completing a work list task. For automatically completed episode tasks, this will be blank.
5 COMP_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
6 SCHED_TIME DATETIME (Local) No
The instant the work list task was scheduled.
7 CANC_USER_ID VARCHAR No
The unique ID of the user canceling a work list task.
8 CANC_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
9 INSTANCE_STATUS_C_NAME VARCHAR No
The current instance status category ID for the task record.
May contain organization-specific values: No
Category Entries:
Incomplete
Completed
Canceled
Removed
Forwarded
Skipped
10 ANCHOR_SCHED_I_DTTM DATETIME (Local) No
If this task was created from an anchored order, this is the due time of the anchor order related to this task due time
11 NOTIFICATION_GRP_C_NAME VARCHAR No
The notification group category ID for the task.
May contain organization-specific values: Yes
Category Entries:
Morning
Around Noon
Afternoon
Evening
Bedtime
As Needed
Anytime
Late Morning
None
Other
12 SCHED_TIME_UTC_DTTM DATETIME (UTC) No
The instant the work list task was scheduled in UTC.
13 COMPLETED_IN_PAT_ENC_CSN_ID NUMERIC No
The unique contact serial number (CSN) of the patient contact in which the task was completed. If you use IntraConnect, this is the Unique Contact Identifier (UCI).
14 TASK_COMP_DATE DATETIME No
The date the user indicates the task was completed
15 OUTCOME_C_NAME VARCHAR No
The task outcome category ID for the task.
May contain organization-specific values: Yes
Category Entries:
Positive
Negative
Not Done
16 REMOVED_IN_PAT_ENC_CSN_ID NUMERIC No
The unique contact serial number (CSN) of the patient contact from which a user removed the task. If you use IntraConnect, this is the Unique Contact Identifier (UCI).
17 UNCOMPLETED_IN_PAT_ENC_CSN_ID NUMERIC No
The unique contact serial number (CSN) of the patient contact from which a user undid the completion action for the task. If you use IntraConnect, this is the Unique Contact Identifier (UCI).
18 TASK_NOTE_ID VARCHAR No
The unique identifier of the note record for the comment associated with a specific task instance.
19 TASK_RESPONSIBLE_USER_ID VARCHAR No
The responsible user assigned to an instance of a care plan task.
20 TASK_RESPONSIBLE_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
21 TASK_RESPONSIBLE_IB_POOL_ID NUMERIC No
The responsible pool assigned to an instance of a care plan task.
22 TASK_RESPONSIBLE_IB_POOL_ID_REGISTRY_NAME VARCHAR No
The name of the In Basket registry in the HIP master file.
23 EDIT_SOURCE_LINE INTEGER No
If this task instance was created by editing an existing instance, this value is the line number of the previous version of this task instance.
24 EDIT_NEXT_LINE INTEGER No
If this task instance was subsequently edited, this line is the location of the next version of this task instance.
25 ORIG_SCHED_DTTM DATETIME (Local) No
The instant a task instance was first scheduled.
26 MOVE_COUNT INTEGER No
The number of times a task instance has been moved relative to this version of the instance.
27 LAST_MOVE_USER_ID VARCHAR No
The user attributed to the last move of this task instance.
28 LAST_MOVE_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
29 COMP_TIME_UTC_DTTM DATETIME (UTC) No
The instant a work list task was completed in UTC.
30 PLANNED_COMP_DATE DATETIME No
Date when a task is planned to be performed, as distinct from when the task is due.
31 TASK_SKIP_RSN_C_NAME VARCHAR No
The category value reason for skipping the task.
May contain organization-specific values: Yes

35 COMP_LATENCY_SECONDS INTEGER No
The number of seconds late that a safety check task was completed. If the task was completed before its due time, no value will be set.
36 COMP_LATENCY_MINUTES INTEGER No
The number of minutes late that a safety check task was completed. This is the difference in minutes of the row's due time and completion time. If the due time falls after the completion time, no value will be recorded.
