MODALITY_DATA
Description:
This table contains basic information about imaging data from the performing modality. It also contains cumulative radiation dose information.

Primary Key
Column Name Ordinal Position
IMY_ID 1

Column Information
Name Type Discontinued?
1 IMY_ID NUMERIC No
The unique identifier (.1 item) for the modality data record.
2 PAT_ID VARCHAR No
The unique ID of the patient record to which the modality data pertains.
3 SOP_INSTANCE_UID VARCHAR No
The DICOM SOP Instance unique identifier of the message from modality. This ID is uniquely assigned for every image in a study.
4 AET_FDD_ID NUMERIC No
The unique ID of the Application Entity Title record from the FDD master file which specifies DICOM definitions for the performing modality.
5 AET_FDD_ID_DICOM_NAME VARCHAR No
The name of an Application Entity (modality). This is not the AE Title.
6 PPS_STATUS_C_NAME VARCHAR No
The PPS status category ID for the DICOM instance record.
May contain organization-specific values: No
Category Entries:
IN PROGRESS
DISCONTINUED
COMPLETED
7 PPS_START_DTTM DATETIME (Local) No
The DICOM Performed Procedure Step Start Date and Time.
8 PPS_END_DTTM DATETIME (Local) No
The DICOM Performed Procedure Step End Date and Time.
9 SOP_CLASS_FDD_ID NUMERIC No
The DICOM SOP Class of the message from the performing modality. DICOM capabilities are all expressed as Service-Object Pair Classes, identified by SOP Class names and unique identifiers.
10 SOP_CLASS_FDD_ID_DICOM_NAME VARCHAR No
The name of an Application Entity (modality). This is not the AE Title.
11 RDSR_MFR VARCHAR No
The manufacturer of the machine sending the data.
12 RDSR_MFR_MODEL_NAME VARCHAR No
The model name of the machine sending the data.
13 RDSR_STATION_NAME VARCHAR No
The station sending the data.
14 RDSR_PROC_RPTD_C_NAME VARCHAR No
The procedure reported category ID for the procedure reported to be performed for the DICOM instance record.
May contain organization-specific values: No
Category Entries:
Computed Tomography X-Ray
Projection X-Ray
Mammography
15 RDSR_IRRD_STRT_DTTM DATETIME (Local) No
The start time of irradiation.
16 RDSR_IRRD_STOP_DTTM DATETIME (Local) No
The end time of irradiation.
17 RDSR_SCP_ACCUMLTN_C_NAME VARCHAR No
The scope of accumulation category ID for the DICOM instance record.
May contain organization-specific values: Yes
Category Entries:
Study
Series
Performed Procedure Step
Irradiation Event
18 RDSR_TOTAL_EFF_DOSE NUMERIC No
The aggregate effective dose.
19 RDSR_TTL_MSR_MTHD_C_NAME VARCHAR No
The total dose measurement method category ID for the DICOM instance.
May contain organization-specific values: Yes
Category Entries:
DLP to E conversion via MC computation
CTDIfreeair to E conversion via MC computation
DLP to E conversion via measurement
CTDIfreeair to E conversion via measurement
20 RDSR_PAT_MODEL VARCHAR No
The effective does patient model.
21 RDSR_REF_AUTH VARCHAR No
The effective dose reference authority.
22 RDSR_TTL_DOSE_PHNTM VARCHAR No
The total effective dose phantom type.
23 RDSR_DOSIMETER_TYPE VARCHAR No
The total effective dose dosimeter type.
24 RDSR_DOSE_CMT VARCHAR No
The comment associated with the accumulated radiation dose.
25 RDSR_PROC_INTENT_C_NAME VARCHAR No
The procedure intent category ID for the DICOM instance record.
May contain organization-specific values: Yes
Category Entries:
Diagnostic Intent
Therapeutic Intent
Combined Diagnostic and Therapeutic Procedure
26 PPS_DESC VARCHAR No
The DICOM Performed Procedure Step Description.
