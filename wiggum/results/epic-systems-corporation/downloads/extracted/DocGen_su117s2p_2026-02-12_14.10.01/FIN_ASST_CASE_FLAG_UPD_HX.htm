FIN_ASST_CASE_FLAG_UPD_HX
Description:
This table contains the updates made to case flags of a financial assistance case record.
Whenever any of the following columns are changed, one or more rows are added to this table based on the number of case flags present on the financial assistance case record.
1. Case status
2. Case flags
3. Assigned user
4. Follow up date
5. Application provided date
6. Application signed date.

Primary Key
Column Name Ordinal Position
FIN_ASST_CASE_ID 1
GROUP_LINE 2
VALUE_LINE 3

Column Information
Name Type Discontinued?
1 FIN_ASST_CASE_ID NUMERIC No
The unique identifier for the financial assistance case record.
2 GROUP_LINE INTEGER No
The line number of the associated update made to one of the following columns in financial assistance case record. 1. Case status 2. Case flags 3. Assigned user 4. Follow up date 5. Application provided date 6. Application signed date Together with FIN_ASST_CASE_ID, this forms the foreign key to table FIN_ASST_CASE_UPDATE_HX.
3 VALUE_LINE INTEGER No
The line number of one of the multiple case flags associated with a financial assistance case after one of the following have been changed: 1. Case status 2. Case flags 3. Assigned user 4. Follow up date 5. Application provided date 6. Application signed date
4 UPDATE_FLAG_C_NAME VARCHAR No
The category ID of the one of the case flags that were on the financial assistance case when they were added to update history.
May contain organization-specific values: Yes
