SDD_WANTS_ASSISTANCE
Description:
Stores information about whether the patient would like resource recommendations for SDOH domains.

Primary Key

SDOH_DATA_ID 1
LINE 2

Name Type Discontinued?

4 WANTS_ASST_INST_UTC_DTTM DATETIME (UTC) No
The instant a patient was documented as wanting or not wanting assistance. This instant stores when the data was written to SDD. If the data was filed via a flowsheet, it may not match the taken or recorded instant. The intent of this item is to indicate when a user could have first viewed this status in SDOH views for the patient. To find when the flowsheet was filed, items 904 and 905 can be used to find the linked flowsheet data.
