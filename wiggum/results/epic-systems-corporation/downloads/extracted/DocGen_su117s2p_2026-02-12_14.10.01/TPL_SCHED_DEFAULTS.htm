TPL_SCHED_DEFAULTS
Description:
This table maps visit kinds to a provider and department to be used as the default when requesting a visit of that kind.

Primary Key
Column Name Ordinal Position
TREATMENT_PLAN_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 TREATMENT_PLAN_ID NUMERIC No
The unique ID of the treatment plan.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 SCHED_RES_VST_KND_C_NAME VARCHAR No
The visit kind category number to which the default provider and department are mapped.
May contain organization-specific values: Yes
4 SCHED_RES_PROV_ID_PROV_NAME VARCHAR No
The name of the service provider. This item may be hidden in a public view of the CLARITY_SER table.
5 SCHED_RES_DEPT_ID_EXTERNAL_NAME VARCHAR No
The external name of the department record. This is often used in patient correspondence such as reminder letters.
