EPI_ANTICOAG_PROV
Description:
When the anticoagulation functionality is used, this table will contain the providers who initiated the enrollment for the patient.
Ater the new functionalities added in 2010 version, the anticoag providers can be specified a role. The role can be obtained by joining the EPI_AC_PROV_INFO table, with the line number and summary block ID.

Primary Key
Column Name Ordinal Position
SUMMARY_BLOCK_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 SUMMARY_BLOCK_ID NUMERIC No
The unique ID of the episode of care record.
2 LINE INTEGER No
The Line Count
3 INITIATING_SER_ID_PROV_NAME VARCHAR No
The name of the service provider. This item may be hidden in a public view of the CLARITY_SER table.
