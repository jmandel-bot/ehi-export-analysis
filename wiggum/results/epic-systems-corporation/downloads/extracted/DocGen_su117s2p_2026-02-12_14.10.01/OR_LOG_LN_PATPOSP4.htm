OR_LOG_LN_PATPOSP4
Description:
This table contains the line IDs (ORM) for the Panel 4 Patient Positioning Information of the Surgical Log (ORL).

Primary Key
Column Name Ordinal Position
LOG_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 LOG_ID VARCHAR No
The unique ID of the procedural log record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 PAN_4_PATPOS_ID VARCHAR No
List of line record IDs, each of which stores the information for a single procedure and its related diagnoses.
