ORD_NLP_TEXT_RECOM
Description:
This table contains information about general recommendations that were extracted from radiology reports by a natural language processing model. This table captures the resolved rate of the recommendation before the recommendation was discretely added.

Primary Key
Column Name Ordinal Position
ORDER_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 ORDER_ID NUMERIC No
The unique identifier (.1 item) for the order record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 TEXT_REC_REGION_C_NAME VARCHAR No
The region of the recommendation in the text.
May contain organization-specific values: No
Category Entries:
Chest
Abdomen
Pelvis
Thyroid
Head
Neck
Breast
Heart
4 TEXT_REC_MODALITY_C_NAME VARCHAR No
The modality of the recommendation in the text.
May contain organization-specific values: No
Category Entries:
Computed Tomography
Ultrasound
Magnetic Resonance Imaging
X-Ray
PET/CT
5 TEXT_REC_TIMEFR_C_NAME VARCHAR No
The time frame of the recommendation in the text.
May contain organization-specific values: No
Category Entries:
1 year
6 months
3 months
6 TEXT_REC_STATUS_C_NAME VARCHAR No
The status of the recommendation in the text.
May contain organization-specific values: Yes
Category Entries:
Needs Follow-up
Closed
Resolved
7 REC_ORDER_ID NUMERIC No
The order placed for the recommendation in the text.
8 REC_PAT_ENC_CSN_ID NUMERIC No
The appointment scheduled for the recommendation in the text.
