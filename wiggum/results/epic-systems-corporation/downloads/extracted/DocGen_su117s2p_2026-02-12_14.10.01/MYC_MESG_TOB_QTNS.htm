MYC_MESG_TOB_QTNS
Description:
This table contains patient answers to tobacco use questions submitted in a history questionnaire.

Primary Key
Column Name Ordinal Position
MESSAGE_ID 1
LINE 2

Column Information
Name Type Discontinued?
1 MESSAGE_ID VARCHAR No
The unique identifier for the message record.
2 LINE INTEGER No
The line number for the information associated with this record. Multiple pieces of information can be associated with this record.
3 TOB_QUESN_ID VARCHAR No
The ID of the tobacco use question answered by the patient.
4 TOB_QUESN_ID_RECORD_NAME VARCHAR No
The name of the Visit Navigator (VN) History Template Definition (LQH) record.
5 TOB_QUESN_RESP_C_NAME VARCHAR No
The patient's response to the tobacco use question.
May contain organization-specific values: No
Category Entries:
Yes
Never
Not Asked
Quit
Passive
6 TOB_QUESN_PPD VARCHAR No
The number of packs smoked per day entered by the patient.
7 TOB_QUESN_YRS VARCHAR No
The number of years of tobacco use entered by the patient.
8 TOB_QUESN_QUIT_DT DATETIME No
The date quit smoking entered by the patient.
9 TOB_QUESN_CMT VARCHAR No
The comment entered by the patient for the tobacco use question.
10 TOB_QUESN_FILED_YN VARCHAR No
The filed status of the tobacco use question--has the provider filed it to the patient's record.
May contain organization-specific values: No
Category Entries:
No
Yes
11 SMOKELESS_TOB_USE_C_NAME VARCHAR No
The patient's response to the smokeless tobacco use question.
May contain organization-specific values: No
Category Entries:
Current
Former
Never
Unknown
12 SMOKELESS_QUIT_DATE DATETIME No
The date quit smokeless tobacco entered by the patient.
13 SMOKING_TOB_USE_C_NAME VARCHAR No
The patient's response to the smoking tobacco use question.
May contain organization-specific values: No
Category Entries:
Every Day
Some Days
Smoker, Current Status Unknown
Former
Never
Never Assessed
Passive Smoke Exposure - Never Smoker
Unknown
Heavy Smoker
Light Smoker
14 SMOKING_QUIT_DATE DATETIME No
The date quit smoking entered by the patient.
15 TOBACCO_READY_QT_C_NAME VARCHAR No
The patient's response to the ready to quit question.
May contain organization-specific values: No
Category Entries:
Not Asked
Yes
No
16 SMOKING_START_DATE DATETIME No
The start smoking date entered by the patient.
17 CURRENT_PACKS_PER_DAY VARCHAR No
The number of packs currently smoked per day entered by the patient.
18 CURRENT_YEARS VARCHAR No
The current number of years of tobacco use entered by the patient.
19 HAS_CUR_CIG_PER_DAY_QUESN_YN VARCHAR No
Used to know if we are asking a current cigarettes per day question. Stores 1 if we are 0 if it's a current packs per day question.
The category values for this column were already listed for column: TOB_QUESN_FILED_YN
20 HAS_AVG_CIG_PER_DAY_QUESN_YN VARCHAR No
Used to know if we are asking a average cigarettes per day question. Stores 1 if we are 0 if it's a average packs per day question.
The category values for this column were already listed for column: TOB_QUESN_FILED_YN
21 TOB_QUESN_FILED_USER_ID VARCHAR No
Stores the ID of the user who reconciled the tobacco data on this submission.
22 TOB_QUESN_FILED_USER_ID_NAME VARCHAR No
The name of the user record. This name may be hidden.
23 TOB_QUESN_FILED_LOCAL_DTTM DATETIME (Local) No
Stores the instant (in local time) this tobacco history data was filed.
