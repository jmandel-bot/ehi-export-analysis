MYC_CONVO
Description:
MyChart conversations.

Primary Key
Column Name Ordinal Position
THREAD_ID 1

Column Information
Name Type Discontinued?
1 THREAD_ID NUMERIC No
The unique identifier (.1 item) for the thread record.
2 PAT_ID VARCHAR No
The patient this conversation is about. Currently only ever expected to be one, but that could change in the future.
3 COMM_ID NUMERIC No
Contains a link the communication record created for a customer service conversation. Not all customer service conversations will have this link, its creation is controlled by settings.
4 SUBJECT VARCHAR No
Subject of the conversation.
5 MYC_MSG_TYP_C_NAME VARCHAR No
The MyChart message type of this conversation. This is the primary message type that will be used for messages on this conversation.
May contain organization-specific values: Yes
