ClinicalCustomFinding

Table Definition  Parent Tables  Child Tables

Table Definition

Description: This table stores the custom findings used by the Medcin system. This replaces ODSCustomFinding and ODSCustomFindingAttr in Intergy 4.00.

Field Datatype Default Null Option Comment
MedcinID INTEGER ?   OPTIONAL Medcin unique identifier. Generated by the CustomFinding sequence (as of 3.50.00).
AttachMedcinID INTEGER ?   OPTIONAL User entered. Could be another ClinicalCustomFinding record or point to MedcinFinding record. MedcinID that this finding is attached to.
RecordStatus CHAR(1)   OPTIONAL Codes for lookup type 'RecordStatus' are not user definable. Valide codes are (A)ctive and (I)nactive.
AttachType CHAR(1)   OPTIONAL Method of Attachment: "P", "S", "R".
BrowseListCaption CHAR(80)   OPTIONAL Caption for browser lists (Known as 'nomin' in Medcin UI Developer Docs).
PhrasePositive CHAR(80)   OPTIONAL Phrase for positive finding.
PhraseNegative CHAR(80)   OPTIONAL Phrase for negative finding.
SentencePositive CHAR(120)   OPTIONAL Sentence for positive finding.
SentenceNegative CHAR(120)   OPTIONAL Sentence for negative finding.
CPT CHAR(5)   OPTIONAL CPT Code.
CPTModifier CHAR(2)   OPTIONAL CPT Modifier.
ExtraSearch CHAR(240)   OPTIONAL Additional Search Words.
FemaleHighRange DECIMAL(10,2)   OPTIONAL Normal Range high for females.
FemaleLowRange DECIMAL(10,2)   OPTIONAL Normal Range low for females.
ICD2 CHAR(6)   OPTIONAL Second ICD9 Code.
ICD9 CHAR(6)   OPTIONAL ICD9 Code.
ICDF CHAR(6)   OPTIONAL ICD9 family history code.
ICDH CHAR(6)   OPTIONAL ICD9 History code.
MaleHighRange DECIMAL(10,2)   OPTIONAL Normal Range high for males.
MaleLowRange DECIMAL(10,2)   OPTIONAL Normal Range low for males.
SearchListCaption CHAR(60)   OPTIONAL Caption for prompt lists. (Known as 'search' in the Medcin UI Development Docs).
ApplicationTag CHAR(50)   OPTIONAL Application defined tag.
EditorFlags INTEGER   OPTIONAL Flag bits for UDF editor use.
UnitOfMeasure CHAR(16)   OPTIONAL Standard unit for values.
AdditionalData CHAR(28000)   OPTIONAL
TimeStamp CHARACTER(14)   OPTIONAL System generated date/time stamp record was last written.
LastUser INTEGER ?   OPTIONAL User ID of user that last wrote the record.
CreateStamp CHARACTER(14)   OPTIONAL System generated date/time stamp record was created.
CreateUser INTEGER ?   OPTIONAL User ID of user that created the record.
AttachTimeStamp CHARACTER(14)   OPTIONAL Identifies the time stamp of the User defined findings when the User Defined Finding AttachId changes.
Nodekey CHAR(20)   OPTIONAL Uniquely identifies the location of the finding with the Medcin hierarchy. The nodekey can change with each version of the Medcin knowledgebase.
EffectiveMedcinID INTEGER ?   OPTIONAL This is the resulting Medcin ID based on the attachment type. The EffectiveMedcinID is same as the AttachMedcinID when the current finding is a 'Sub' or a 'Parallel' (AttachType of 'S' or 'P') finding. When current finding is replacement type, this is the AttachMedcinID of the parent finding.
YesIsNormalFlag CHAR(1) No_Value__N_   MANDATORY Flag indicating whether answering "Yes" to a finding is equivalent to a medically normal result. Possible Values (Y)es, (N)o, or (U)nknown. Default Value is 'N'.
ICD10-1 CHAR(10) Set_To_Empty_String   OPTIONAL ICD-10 Code
ICD10-2 CHAR(10) Set_To_Empty_String   OPTIONAL ICD-10 Code
ICD10-3 CHAR(10) Set_To_Empty_String   OPTIONAL ICD-10 Code
ICD10-4 CHAR(10) Set_To_Empty_String   OPTIONAL ICD-10 Code
LOINCCode CHAR(30)   OPTIONAL LOINC Code.
SNOMEDCode CHAR(30)   OPTIONAL SNOMED Code.
