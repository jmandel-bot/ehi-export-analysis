Table Definition

Description: This table stores the reaction codes corresponding to each record in the PersonAllergyHistory table.

Field Datatype Default Null Option Comment
PersonID INTEGER ?   MANDATORY FK - From PersonAllergyHistory
AllergyCode CHARACTER(8)   MANDATORY FK - From PersonAllergyHistory
HistoryCount INTEGER   MANDATORY FK - From PersonAllergyHistory
ReactionCode CHARACTER(8)   MANDATORY User enter allergy reaction code. This code is validated by the 'AllergyReaction' lookup type.
TimeStamp CHARACTER(14)   OPTIONAL System generated date/time stamp record was last written.
LastUser INTEGER ?   OPTIONAL User ID of user that last wrote the record.
CreateStamp CHARACTER(14)   OPTIONAL System generated date/time stamp record was created.
CreateUser INTEGER ?   OPTIONAL User ID of user that created the record.
ReactionSeverity CHAR(4)   OPTIONAL Severity of allergic reaction. This code is defined by the 'Severity' lookup type.
