Table Definition  Parent Tables  Child Tables

Table Definition

Description: This table stores the medcin finding data.

Field Datatype Default Null Option Comment
MedcinID INTEGER 0   MANDATORY Medcin ID that uniquely identifies the finding.
FullDescription CHARACTER(60)   MANDATORY The description of the finding independent of its position in the Medcin hierarchy
PartialDescription CHAR(80)   MANDATORY The description of the finding relative to its parent finding's full description
NodeKey CHARACTER(20)   MANDATORY Uniquely identifies the location of the finding with the Medcin hierarchy. The nodekey can change with each version of the Medcin knowledgebase.
TermType INTEGER 0   MANDATORY Numeric code representing the type of medical term as represented in the Medcin hierarchy. Allowed values must be defined in the MedcinTermType lookup type.
RetiredFlag CHARACTER(1) No_Value__N_   MANDATORY Flag indicating whether the finding is retired. A retired finding does not display in the Medcin hierarchy, though it still exists in the knowledgebase. Default value is 'N'.
YesIsNormalFlag CHARACTER(1) No_Value__N_   MANDATORY Flag indicating whether answering "Yes" to a finding is equivalent to a medically normal result. Default value is 'N'.
NonMedicalFlag CHARACTER(1) No_Value__N_   MANDATORY Flag indicating when a finding is not available in the Medcin hierarchy because it exists solely for proprietary reasons. Defaule value is 'N'.
HasChildrenFlag CHARACTER(1) No_Value__N_   MANDATORY Flag indicating that a finding has children in the Medcin hierarchy. Default value is 'N'.
Gender CHARACTER(1)   OPTIONAL Indicates when a finding is gender-specific. Valid values are 'M'ale, 'F'emale.
Specialty CHARACTER(8)   OPTIONAL Contains multiple 1-character codes representing specialties relevant to the finding.
MedcinProcedureCode CHARACTER(6)   OPTIONAL Medcin supplied Procedure Code. Note: May not exist in Intergy Procedure Code.
MedcinDiagnosisCode CHAR(8)   OPTIONAL Medcin supplied Diagnosis Code. Note: May not exist in Intergy Diagnosis Code.
StandardUnit CHARACTER(16)   OPTIONAL Standard unit of measure for the finding.
MinAge INTEGER 0   OPTIONAL Minimum age (in minutes) relevant to the finding.
MaxAge INTEGER 0   OPTIONAL Maximum age (in minutes) relevant to the finding.
TimeStamp CHARACTER(14)   OPTIONAL System generated date/time stamp record was last written.
LastUser INTEGER ?   OPTIONAL User ID of user that last wrote the record.
CreateStamp CHARACTER(14)   OPTIONAL System generated date/time stamp record was created.
CreateUser INTEGER ?   OPTIONAL User ID of user that created the record.
ProcedureModifier CHAR(2)   OPTIONAL The procedure modifier associated with this finding.
