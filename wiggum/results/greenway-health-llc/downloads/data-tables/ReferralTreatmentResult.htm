ReferralTreatmentResult

Table Definition  Parent Tables  Child Tables

Table Definition

Description: This table holds referral result details. The referral results items may be procedures or procedure groups.

Field Datatype Default Null Option Comment
ReferralSID INTEGER ?   MANDATORY FK
ResultNumber INTEGER 0   MANDATORY Result line number. This field is system generated and is incremented any time a new referral result is created.
ChargeSID INTEGER ?   OPTIONAL System generated id using the Charge sequence.
TreatmentCodeType CHARACTER(1)   MANDATORY The treatment plan procedure type. Possible values: (P)rocedure (G)roup
TreatmentCode CHARACTER(10)   MANDATORY User defined code that identifies a collection of procedure code ranges.
StartDate DATE   OPTIONAL
EndDate DATE   OPTIONAL
Units DECIMAL(10,2)   OPTIONAL Number of units used.
Visits DECIMAL(10,2)   OPTIONAL The number of used visits for the procedure or group.
Amount DECIMAL(10,2) 0   OPTIONAL Used amount for procedure or group.
Memo CHARACTER(32000)   OPTIONAL
TimeStamp CHARACTER(14)   OPTIONAL System generated date/time stamp record was last written.
LastUser INTEGER ?   OPTIONAL User ID of user that last wrote the record.
CreateStamp CHARACTER(14)   OPTIONAL System generated date/time stamp record was created.
CreateUser INTEGER ?   OPTIONAL User ID of user that created the record.
TreatmentNumber INTEGER 0   OPTIONAL FK
VisitIncurred CHARACTER(1)   OPTIONAL Flag to determine if a visit was incurred when the procedure was performed. Possible values: (Y)es (N)o Note: This field is reserved for future use.
