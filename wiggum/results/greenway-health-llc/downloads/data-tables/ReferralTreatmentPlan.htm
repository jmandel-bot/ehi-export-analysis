Table Definition

Description: This table lists the procedures that make up a referral detail treatment plan definiton.

Field Datatype Default Null Option Comment
ReferralSID INTEGER ?   MANDATORY
TreatmentNumber INTEGER 0   MANDATORY
TreatmentCodeType CHARACTER(1)   MANDATORY The treatment plan procedure type. Possible values: (P)rocedure (G)roup
TreatmentCode CHARACTER(10)   MANDATORY Procedure code which represents a procedure performed. User entered.
Units DECIMAL(10,2)   OPTIONAL Units allowed for procedure or group.
Visits INTEGER 0   OPTIONAL The number of allowed visits for the procedure or group.
Amount DECIMAL(10,2) 0   OPTIONAL Allowed amount for procedure or group.
Memo CHARACTER(32000)   OPTIONAL Narrative text associated with treatment.
TimeStamp CHARACTER(14)   OPTIONAL System generated date/time stamp record was last written.
LastUser INTEGER ?   OPTIONAL User ID of user that last wrote the record.
CreateStamp CHARACTER(14)   OPTIONAL System generated date/time stamp record was created.
CreateUser INTEGER ?   OPTIONAL User ID of user that created the record.
